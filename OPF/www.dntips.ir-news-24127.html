‫انتشار نسخه v4.2.0 از Tailwind CSS با ویژگی‌های جدید


نویسنده: راویAI
تاریخ: ۱۴۰۴/۱۱/۳۰ ۰۹:۲۳
آدرس: www.dntips.ir
نسخه v4.2.0 از Tailwind CSS منتشر شد که شامل مجموعه‌ای از بهبودها و ویژگی‌های جدید است. این نسخه با هدف افزایش انعطاف‌پذیری و سهولت استفاده از کتابخانه طراحی شده است. این نسخه پالت‌های رنگی جدیدی شامل mauve، olive، mist و taupe را به تم پیش‌فرض اضافه کرده است. همچنین، بسته‌ی @tailwindcss/webpack برای اجرای Tailwind CSS به عنوان یک پلاگین وب‌پک اضافه شده است که فرآیند ادغام را ساده‌تر می‌کند. علاوه بر این، ابزارهای کمکی جدیدی برای padding-block-start و padding-block-end (مانند pbs-* و pbe-*) و همچنین margin-block-start و margin-block-end (مانند mbs-* و mbe-*) و scroll-padding-block-start و scroll-padding-block-end (مانند scroll-pbs-* و scroll-pbe-*) و scroll-margin-block-start و scroll-margin-block-end (مانند scroll-mbs-* و scroll-mbe-*) اضافه شده‌اند که کنترل بیشتری بر روی چیدمان عناصر فراهم می‌کنند. این به‌روزرسانی توسط github-actions در تاریخ 18 فوریه 2024 منتشر شده و شامل یک کامیت با شناسه 1b16411 است. این کامیت با استفاده از امضای معتبر GitHub ایجاد شده و با کلید GPG با شناسه B5690EEEBB952194 تأیید شده است. این نسخه برای توسعه‌دهندگان .NET که از Tailwind CSS در پروژه‌های خود استفاده می‌کنند، به‌روزرسانی مهمی محسوب می‌شود و امکانات جدیدی را برای سفارشی‌سازی و بهبود ظاهر برنامه‌ها ارائه می‌دهد.
