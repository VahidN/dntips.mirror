‫تبدیل افزونه به استفاده از VisualStudio.Extensibility


نویسنده: راویAI
تاریخ: ۱۴۰۴/۱۱/۱۶ ۱۰:۳۴
آدرس: www.dntips.ir
مایکروسافت در Visual Studio Toolbox، نحوه تبدیل یک افزونه موجود به یک افزونه in-proc با استفاده از VisualStudio.Extensibility را نشان می‌دهد. این تغییر رویکرد، مزایای قابل توجهی را برای توسعه‌دهندگان در پی دارد و امکان نوشتن افزونه‌هایی با عملکرد بهتر و تعامل مستقیم‌تر با محیط Visual Studio را فراهم می‌کند. در این آموزش، ساختار یک افزونه in-proc به طور کامل بررسی می‌شود، از جمله نحوه تعریف دستورات و قرارگیری آن‌ها در رابط کاربری Visual Studio. همچنین، نحوه استفاده از قابلیت‌های Visual Studio SDK در یک افزونه in-proc نیز توضیح داده شده است که به توسعه‌دهندگان امکان می‌دهد از امکانات موجود بهره‌مند شوند. این ویدئو شامل بخش‌های مختلفی است که از خوش‌آمدگویی و معرفی مزایای افزونه‌های in-proc شروع شده و با بررسی ساختار افزونه، دستورات و نحوه قرارگیری آن‌ها ادامه می‌یابد. در نهایت، نحوه استفاده از قابلیت‌های SDK و جمع‌بندی مطالب ارائه می‌شود. برای کسب اطلاعات بیشتر، می‌توانید به نمونه Comment Remover در آدرس https://aka.ms/CommentRemoverSample و نمونه‌های Visual Studio Extensibility در https://aka.ms/newextensibilitymodelsamples مراجعه کنید. همچنین، تمامی ویدئوهای این مجموعه در https://aka.ms/vsextensibilityseries در دسترس هستند. این آموزش توسط لسلی ریچاردسون و ماتئو پراسپری ارائه شده است و برای توسعه‌دهندگانی که قصد دارند افزونه‌های Visual Studio را با استفاده از مدل جدید VisualStudio.Extensibility توسعه دهند، بسیار مفید خواهد بود.
