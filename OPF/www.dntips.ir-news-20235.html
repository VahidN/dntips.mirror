Щ…ЩӮШҜЩ…ЩҮ
ШҜШұ ШҜЩҶЫҢШ§ЫҢ ШӘЩҲШіШ№ЩҮ ЩҶШұЩ…вҖҢШ§ЩҒШІШ§Шұ ШЁШ§ Ш§ШіШӘЩҒШ§ШҜЩҮ Ш§ШІ ЩҒШұЫҢЩ…вҖҢЩҲШұЪ© Microsoft .NETШҢ Ш§ШЁШІШ§ШұЩҮШ§ ЩҲ Ъ©ШӘШ§ШЁШ®Ш§ЩҶЩҮвҖҢЩҮШ§ЫҢЫҢ Щ…Ш§ЩҶЩҶШҜ MediatR ШЁЩҮ ШҜЩ„ЫҢЩ„ ШіШ§ШҜЪҜЫҢ ЩҲ Ш§ЩҶШ№Ш·Ш§ЩҒвҖҢЩҫШ°ЫҢШұЫҢ ШҜШұ Щ…ШҜЫҢШұЫҢШӘ ШҜШұШ®ЩҲШ§ШіШӘвҖҢЩҮШ§ (Requests) ЩҲ ЩҫШ§ШіШ®вҖҢЩҮШ§ (Responses) ШҜШұ Ш§Щ„ЪҜЩҲЫҢ CQRS (Command Query Responsibility Segregation) ЩҲ Щ…Ш№Щ…Ш§ШұЫҢ Щ…ШЁШӘЩҶЫҢ ШЁШұ ЩҲШ§ШіШ·ЩҮ (Mediator Pattern) ШЁШіЫҢШ§Шұ Щ…ШӯШЁЩҲШЁ ШҙШҜЩҮвҖҢШ§ЩҶШҜ. ШЁШ§ Ш§ЫҢЩҶ ШӯШ§Щ„ШҢ ШҜШұ ШЁШұШ®ЫҢ ЩҫШұЩҲЪҳЩҮвҖҢЩҮШ§ Щ…Щ…Ъ©ЩҶ Ш§ШіШӘ ЩҶЫҢШ§ШІ ШЁЩҮ Ш¬Ш§ЫҢЪҜШІЫҢЩҶЫҢ MediatR ШЁШ§ ШұЩҲШҙвҖҢЩҮШ§ ЫҢШ§ Ш§ШЁШІШ§ШұЩҮШ§ЫҢ ШҜЫҢЪҜШұ Ш§ШӯШіШ§Ші ШҙЩҲШҜШӣ ЪҶЩҮ ШЁЩҮ ШҜЩ„ЫҢЩ„ ЩҫЫҢЪҶЫҢШҜЪҜЫҢвҖҢЩҮШ§ЫҢ ШәЫҢШұШ¶ШұЩҲШұЫҢШҢ ЩҶЫҢШ§ШІ ШЁЩҮ Ш№Щ…Щ„Ъ©ШұШҜ ШЁЩҮШӘШұШҢ ЫҢШ§ ШӘЩ…Ш§ЫҢЩ„ ШЁЩҮ Ш§ШіШӘЩҒШ§ШҜЩҮ Ш§ШІ ШұШ§ЩҮЪ©Ш§ШұЩҮШ§ЫҢ ШӘЩҲШІЫҢШ№вҖҢШҙШҜЩҮ (Distributed Systems). ШҜШұ Ш§ЫҢЩҶ Щ…ЩӮШ§Щ„ЩҮШҢ ШЁЩҮ ШЁШұШұШіЫҢ ЪҜШІЫҢЩҶЩҮвҖҢЩҮШ§ЫҢ Ш¬Ш§ЫҢЪҜШІЫҢЩҶ ШЁШұШ§ЫҢ MediatR ШҜШұ ЩҫШұЩҲЪҳЩҮвҖҢЩҮШ§ЫҢ .NET Щ…ЫҢвҖҢЩҫШұШҜШ§ШІЫҢЩ… ЩҲ ШЁШ§ ШӘШӯЩ„ЫҢЩ„ Щ…ШІШ§ЫҢШ§ ЩҲ Щ…Ш№Ш§ЫҢШЁ ЩҮШұ ШұЩҲШҙШҢ ШұШ§ЩҮЩҶЩ…Ш§ЫҢЫҢ Ш¬Ш§Щ…Ш№ЫҢ ШЁШұШ§ЫҢ ШӘЩҲШіШ№ЩҮвҖҢШҜЩҮЩҶШҜЪҜШ§ЩҶ Ш§ШұШ§ШҰЩҮ Щ…ЫҢвҖҢШҜЩҮЫҢЩ…. ЩҮШҜЩҒ Ш§ЫҢЩҶ Ш§ШіШӘ Ъ©ЩҮ ШҙЩ…Ш§ ШЁШӘЩҲШ§ЩҶЫҢШҜ ШЁШ§ ШӘЩҲШ¬ЩҮ ШЁЩҮ ЩҶЫҢШ§ШІЩҮШ§ЫҢ ЩҫШұЩҲЪҳЩҮ Ш®ЩҲШҜШҢ ШЁЩҮШӘШұЫҢЩҶ ШӘШөЩ…ЫҢЩ… ШұШ§ ШЁЪҜЫҢШұЫҢШҜ.

1. ШӘШІШұЫҢЩӮ Щ…ШіШӘЩӮЫҢЩ… ЩҲШ§ШЁШіШӘЪҜЫҢвҖҢЩҮШ§ (Direct Dependency Injection)
ЫҢЪ©ЫҢ Ш§ШІ ШіШ§ШҜЩҮвҖҢШӘШұЫҢЩҶ ШұЩҲШҙвҖҢЩҮШ§ЫҢ Ш¬Ш§ЫҢЪҜШІЫҢЩҶ ШЁШұШ§ЫҢ MediatRШҢ Ш§ШіШӘЩҒШ§ШҜЩҮ Ш§ШІ ШӘШІШұЫҢЩӮ Щ…ШіШӘЩӮЫҢЩ… ЩҲШ§ШЁШіШӘЪҜЫҢвҖҢЩҮШ§ (Dependency Injection) Ш§ШіШӘ Ъ©ЩҮ ШЁЩҮ ШөЩҲШұШӘ ЩҫЫҢШҙвҖҢЩҒШұШ¶ ШҜШұ ЩҒШұЫҢЩ…вҖҢЩҲШұЪ© .NET Core Ш§ШұШ§ШҰЩҮ Щ…ЫҢвҖҢШҙЩҲШҜ. ШҜШұ Ш§ЫҢЩҶ ШұЩҲШҙШҢ ШЁЩҮвҖҢШ¬Ш§ЫҢ Ш§ШіШӘЩҒШ§ШҜЩҮ Ш§ШІ ЫҢЪ© ЩҲШ§ШіШ·ЩҮ Щ…ШұЪ©ШІЫҢ Щ…Ш§ЩҶЩҶШҜ MediatR ШЁШұШ§ЫҢ Ш§ШұШіШ§Щ„ ЩҲ Щ…ШҜЫҢШұЫҢШӘ ШҜШұШ®ЩҲШ§ШіШӘвҖҢЩҮШ§ШҢ ШіШұЩҲЫҢШівҖҢЩҮШ§ ЩҲ ЩҲШ§ШЁШіШӘЪҜЫҢвҖҢЩҮШ§ Щ…ШіШӘЩӮЫҢЩ…Ш§ЩӢ ШҜШұ Ъ©Щ„Ш§ШівҖҢЩҮШ§ЫҢЫҢ Ъ©ЩҮ ШЁЩҮ ШўЩҶвҖҢЩҮШ§ ЩҶЫҢШ§ШІ ШҜШ§ШұЩҶШҜ ШӘШІШұЫҢЩӮ Щ…ЫҢвҖҢШҙЩҲЩҶШҜ.

Щ…ШІШ§ЫҢШ§:
  • Ъ©Ш§ЩҮШҙ ЩҫЫҢЪҶЫҢШҜЪҜЫҢ: ШӯШ°ЩҒ Щ„Ш§ЫҢЩҮ ЩҲШ§ШіШ·ЩҮ ШЁШ§Ш№Ш« Щ…ЫҢвҖҢШҙЩҲШҜ Ъ©ШҜ Ш®ЩҲШ§ЩҶШ§ШӘШұ ЩҲ ЩӮШ§ШЁЩ„вҖҢЩҒЩҮЩ…вҖҢШӘШұ ШҙЩҲШҜ.
  • Ш№Щ…Щ„Ъ©ШұШҜ ШЁЩҮШӘШұ: Ш№ШҜЩ… ЩҶЫҢШ§ШІ ШЁЩҮ ЩҫШұШҜШ§ШІШҙ ШҜШұШ®ЩҲШ§ШіШӘвҖҢЩҮШ§ Ш§ШІ Ш·ШұЫҢЩӮ ЫҢЪ© ЩҲШ§ШіШ·ЩҮШҢ ШІЩ…Ш§ЩҶ Ш§Ш¬ШұШ§ ШұШ§ Ъ©Ш§ЩҮШҙ Щ…ЫҢвҖҢШҜЩҮШҜ.
  • Ъ©ЩҶШӘШұЩ„ ШЁЫҢШҙШӘШұ: ШӘЩҲШіШ№ЩҮвҖҢШҜЩҮЩҶШҜЪҜШ§ЩҶ Щ…ЫҢвҖҢШӘЩҲШ§ЩҶЩҶШҜ ШЁЩҮ ШөЩҲШұШӘ Щ…ШіШӘЩӮЫҢЩ… ШЁШ§ ШіШұЩҲЫҢШівҖҢЩҮШ§ ШӘШ№Ш§Щ…Щ„ ШҜШ§ШҙШӘЩҮ ШЁШ§ШҙЩҶШҜ.
Щ…Ш№Ш§ЫҢШЁ:
  • Ш§ЩҒШІШ§ЫҢШҙ ЩҲШ§ШЁШіШӘЪҜЫҢвҖҢЩҮШ§: ШҜШұ ЩҫШұЩҲЪҳЩҮвҖҢЩҮШ§ЫҢ ШЁШІШұЪҜШҢ ШӘШ№ШҜШ§ШҜ ЩҲШ§ШЁШіШӘЪҜЫҢвҖҢЩҮШ§ЫҢ ШӘШІШұЫҢЩӮвҖҢШҙШҜЩҮ Щ…Щ…Ъ©ЩҶ Ш§ШіШӘ Щ…ШҜЫҢШұЫҢШӘ Ъ©ШҜ ШұШ§ ШҜШҙЩҲШ§Шұ Ъ©ЩҶШҜ.
  • Ш№ШҜЩ… Ш¬ШҜШ§ШіШ§ШІЫҢ Щ…ШіШҰЩҲЩ„ЫҢШӘвҖҢЩҮШ§: ШЁШұШ®Щ„Ш§ЩҒ MediatR Ъ©ЩҮ ШЁЩҮ Ш·ЩҲШұ Ш°Ш§ШӘЫҢ Ш§ШІ Ш§Щ„ЪҜЩҲЫҢ CQRS ЩҫШҙШӘЫҢШЁШ§ЩҶЫҢ Щ…ЫҢвҖҢЪ©ЩҶШҜШҢ ШҜШұ Ш§ЫҢЩҶ ШұЩҲШҙ ШЁШ§ЫҢШҜ Ш§ЫҢЩҶ Ш¬ШҜШ§ШіШ§ШІЫҢ ШЁЩҮ ШөЩҲШұШӘ ШҜШіШӘЫҢ ЩҫЫҢШ§ШҜЩҮвҖҢШіШ§ШІЫҢ ШҙЩҲШҜ.

Щ…Ш«Ш§Щ„ Ъ©ШҜ:
public class OrderController : ControllerBase
{
    private readonly IOrderService _orderService;

    public OrderController(IOrderService orderService)
    {
        _orderService = orderService;
    }

    [HttpPost]
    public IActionResult CreateOrder(OrderDto order)
    {
        _orderService.CreateOrder(order);
        return Ok();
    }
}
ШҜШұ Ш§ЫҢЩҶ Щ…Ш«Ш§Щ„ШҢ ШЁЩҮ Ш¬Ш§ЫҢ Ш§ШіШӘЩҒШ§ШҜЩҮ Ш§ШІ MediatR ШЁШұШ§ЫҢ Ш§ШұШіШ§Щ„ ЫҢЪ© ШҜШіШӘЩҲШұ (Command)ШҢ ШіШұЩҲЫҢШі Щ…ЩҲШұШҜЩҶШёШұ Щ…ШіШӘЩӮЫҢЩ…Ш§ЩӢ ШӘШІШұЫҢЩӮ ШҙШҜЩҮ ЩҲ ЩҒШұШ§Ш®ЩҲШ§ЩҶЫҢ Щ…ЫҢвҖҢШҙЩҲШҜ.

2. MassTransit: ЪҶШ§ШұЪҶЩҲШЁЫҢ ШЁШұШ§ЫҢ ШіЫҢШіШӘЩ…вҖҢЩҮШ§ЫҢ ШӘЩҲШІЫҢШ№вҖҢШҙШҜЩҮ
MassTransit ЫҢЪ© Ъ©ШӘШ§ШЁШ®Ш§ЩҶЩҮ ЩӮШҜШұШӘЩ…ЩҶШҜ ЩҲ Щ…ШӘЩҶвҖҢШЁШ§ШІ (Open-Source) Ш§ШіШӘ Ъ©ЩҮ ШЁШұШ§ЫҢ ШӘЩҲШіШ№ЩҮ ШЁШұЩҶШ§Щ…ЩҮвҖҢЩҮШ§ЫҢ ШӘЩҲШІЫҢШ№вҖҢШҙШҜЩҮ Ш·ШұШ§ШӯЫҢ ШҙШҜЩҮ ЩҲ ЩӮШ§ШЁЩ„ЫҢШӘвҖҢЩҮШ§ЫҢЫҢ Щ…Ш§ЩҶЩҶШҜ Ш§ШұШіШ§Щ„ ЩҫЫҢШ§Щ… (Messaging)ШҢ Щ…ШҜЫҢШұЫҢШӘ ШұЩҲЫҢШҜШ§ШҜЩҮШ§ (Event Handling) ЩҲ ЩҫШҙШӘЫҢШЁШ§ЩҶЫҢ Ш§ШІ ШөЩҒвҖҢЩҮШ§ЫҢ ЩҫЫҢШ§Щ… (Message Queues) Щ…Ш§ЩҶЩҶШҜ RabbitMQ ЩҲ Azure Service Bus ШұШ§ Ш§ШұШ§ШҰЩҮ Щ…ЫҢвҖҢШҜЩҮШҜ. Ш§ЫҢЩҶ Ш§ШЁШІШ§Шұ Щ…ЫҢвҖҢШӘЩҲШ§ЩҶШҜ Ш¬Ш§ЫҢЪҜШІЫҢЩҶ Щ…ЩҶШ§ШіШЁЫҢ ШЁШұШ§ЫҢ MediatR ШҜШұ ЩҫШұЩҲЪҳЩҮвҖҢЩҮШ§ЫҢЫҢ ШЁШ§ШҙШҜ Ъ©ЩҮ ЩҶЫҢШ§ШІ ШЁЩҮ Ш§ШұШӘШЁШ§Ш· ШЁЫҢЩҶ ШіШұЩҲЫҢШівҖҢЩҮШ§ ЫҢШ§ Щ…Ш№Щ…Ш§ШұЫҢ Щ…ЫҢЪ©ШұЩҲШіШұЩҲЫҢШі ШҜШ§ШұЩҶШҜ.

Щ…ШІШ§ЫҢШ§:
  • ЩҫШҙШӘЫҢШЁШ§ЩҶЫҢ Ш§ШІ Щ…Ш№Щ…Ш§ШұЫҢ ШӘЩҲШІЫҢШ№вҖҢШҙШҜЩҮ: MassTransit ШЁШұШ§ЫҢ ШіЩҶШ§ШұЫҢЩҲЩҮШ§ЫҢЫҢ Ъ©ЩҮ ЩҶЫҢШ§ШІ ШЁЩҮ Ш§ШұШіШ§Щ„ ЩҫЫҢШ§Щ… ШЁЫҢЩҶ ШіШұЩҲЫҢШівҖҢЩҮШ§ ЫҢШ§ ШіЫҢШіШӘЩ…вҖҢЩҮШ§ ШҜШ§ШұЩҶШҜШҢ Ш§ЫҢШҜЩҮвҖҢШўЩ„ Ш§ШіШӘ.
  • Ш§ЩҶШ№Ш·Ш§ЩҒвҖҢЩҫШ°ЫҢШұЫҢ: Ш§Щ…Ъ©Ш§ЩҶ Ш§ШіШӘЩҒШ§ШҜЩҮ Ш§ШІ ЩҫШұЩҲШӘЪ©Щ„вҖҢЩҮШ§ЫҢ Щ…Ш®ШӘЩ„ЩҒ ЩҲ Ш§ШЁШІШ§ШұЩҮШ§ЫҢ ЩҫЫҢШ§Щ…вҖҢШұШіШ§ЩҶЫҢ ШұШ§ ЩҒШұШ§ЩҮЩ… Щ…ЫҢвҖҢЪ©ЩҶШҜ.
  • ЩӮШ§ШЁЩ„ЫҢШӘ ШӘЩҲШіШ№ЩҮ: ЩҲЫҢЪҳЪҜЫҢвҖҢЩҮШ§ЫҢЫҢ Щ…Ш§ЩҶЩҶШҜ Saga ШЁШұШ§ЫҢ Щ…ШҜЫҢШұЫҢШӘ ШӘШұШ§Ъ©ЩҶШҙвҖҢЩҮШ§ЫҢ Ш·ЩҲЩ„Ш§ЩҶЫҢвҖҢЩ…ШҜШӘ (Long-Running Transactions) ШұШ§ ШҜШ§ШұШҜ.
Щ…Ш№Ш§ЫҢШЁ:
  • ЩҫЫҢЪҶЫҢШҜЪҜЫҢ ШЁЫҢШҙШӘШұ: ЩҶШіШЁШӘ ШЁЩҮ MediatRШҢ ШұШ§ЩҮвҖҢШ§ЩҶШҜШ§ШІЫҢ ЩҲ ЩҫЫҢЪ©ШұШЁЩҶШҜЫҢ ШўЩҶ ШІЩ…Ш§ЩҶвҖҢШЁШұШӘШұ Ш§ШіШӘ.
  • Щ…ЩҶШ§ШіШЁ ЩҶШЁЩҲШҜЩҶ ШЁШұШ§ЫҢ ЩҫШұЩҲЪҳЩҮвҖҢЩҮШ§ЫҢ Ъ©ЩҲЪҶЪ©: Ш§ЪҜШұ ЩҫШұЩҲЪҳЩҮ ШҙЩ…Ш§ ЩҶЫҢШ§ШІЫҢ ШЁЩҮ Щ…Ш№Щ…Ш§ШұЫҢ ШӘЩҲШІЫҢШ№вҖҢШҙШҜЩҮ ЩҶШҜШ§ШұШҜШҢ Ш§ШіШӘЩҒШ§ШҜЩҮ Ш§ШІ MassTransit Щ…Щ…Ъ©ЩҶ Ш§ШіШӘ ШЁЫҢШҙ Ш§ШІ ШӯШҜ ЩҫЫҢЪҶЫҢШҜЩҮ ШЁШ§ШҙШҜ.

Щ…Ш«Ш§Щ„ Ъ©ШҜ:
public class OrderConsumer : IConsumer<CreateOrderCommand>
{
    public Task Consume(ConsumeContext<CreateOrderCommand> context)
    {
        // Щ…ЩҶШ·ЩӮ ЩҫШұШҜШ§ШІШҙ ШіЩҒШ§ШұШҙ
        return Task.CompletedTask;
    }
}

// ЩҫЫҢЪ©ШұШЁЩҶШҜЫҢ MassTransit
services.AddMassTransit(x =>
{
    x.AddConsumer<OrderConsumer>();
    x.UsingRabbitMq((context, cfg) =>
    {
        cfg.Host("localhost");
        cfg.ConfigureEndpoints(context);
    });
});
ШҜШұ Ш§ЫҢЩҶШ¬Ш§ШҢ MassTransit ШЁЩҮ Ш№ЩҶЩҲШ§ЩҶ ЫҢЪ© ШіЫҢШіШӘЩ… ЩҫЫҢШ§Щ…вҖҢШұШіШ§ЩҶ Ш№Щ…Щ„ Щ…ЫҢвҖҢЪ©ЩҶШҜ ЩҲ ШҜШіШӘЩҲШұШ§ШӘ (Commands) ШұШ§ ШЁЩҮ Щ…ШөШұЩҒвҖҢЪ©ЩҶЩҶШҜЩҮвҖҢЩҮШ§ (Consumers) Ш§ШұШіШ§Щ„ Щ…ЫҢвҖҢЪ©ЩҶШҜ.

3. CAP: ШұШ§ЩҮЪ©Ш§ШұЫҢ ШіШЁЪ© ШЁШұШ§ЫҢ ШұЩҲЫҢШҜШ§ШҜЩҮШ§
CAP (Щ…Ш®ЩҒЩҒ C# Advanced Publisher) ЫҢЪ© Ъ©ШӘШ§ШЁШ®Ш§ЩҶЩҮ ШіШЁЪ© ЩҲ Ъ©Ш§ШұШўЩ…ШҜ Ш§ШіШӘ Ъ©ЩҮ ШЁШұШ§ЫҢ Ш§ЩҶШӘШҙШ§Шұ ЩҲ Ш§ШҙШӘШұШ§Ъ© ШұЩҲЫҢШҜШ§ШҜЩҮШ§ (Event Publishing and Subscribing) ШҜШұ ШіЫҢШіШӘЩ…вҖҢЩҮШ§ЫҢ ШӘЩҲШІЫҢШ№вҖҢШҙШҜЩҮ Ш·ШұШ§ШӯЫҢ ШҙШҜЩҮ Ш§ШіШӘ. Ш§ЫҢЩҶ Ш§ШЁШІШ§Шұ Ш§ШІ Ш§Щ„ЪҜЩҲЫҢ ШұЩҲЫҢШҜШ§ШҜЩ…ШӯЩҲШұ (Event-Driven) ЩҫШҙШӘЫҢШЁШ§ЩҶЫҢ Щ…ЫҢвҖҢЪ©ЩҶШҜ ЩҲ ШЁШ§ Ш§ШЁШІШ§ШұЩҮШ§ЫҢЫҢ Щ…Ш§ЩҶЩҶШҜ Kafka ЩҲ Redis ШіШ§ШІЪҜШ§Шұ Ш§ШіШӘ.

Щ…ШІШ§ЫҢШ§:
  • ШіШ§ШҜЪҜЫҢ: ЩҶШіШЁШӘ ШЁЩҮ MassTransitШҢ ШұШ§ЩҮвҖҢШ§ЩҶШҜШ§ШІЫҢ ЩҲ Ш§ШіШӘЩҒШ§ШҜЩҮ Ш§ШІ CAP ШіШ§ШҜЩҮвҖҢШӘШұ Ш§ШіШӘ.
  • Ш№Щ…Щ„Ъ©ШұШҜ ШЁШ§Щ„Ш§: ШЁЩҮ ШҜЩ„ЫҢЩ„ ШіШЁЪ© ШЁЩҲШҜЩҶШҢ ШЁШұШ§ЫҢ ЩҫШұЩҲЪҳЩҮвҖҢЩҮШ§ЫҢЫҢ ШЁШ§ ШӯШ¬Щ… ШӘШұШ§Ъ©ЩҶШҙ ШЁШ§Щ„Ш§ Щ…ЩҶШ§ШіШЁ Ш§ШіШӘ.
  • ЩӮШ§ШЁЩ„ЫҢШӘ Ш§Ш·Щ…ЫҢЩҶШ§ЩҶ: Ш§ШІ Щ…Ъ©Ш§ЩҶЫҢШІЩ…вҖҢЩҮШ§ЫҢЫҢ Щ…Ш§ЩҶЩҶШҜ "ШӯШҜШ§ЩӮЩ„ ЫҢЪ©вҖҢШЁШ§Шұ ШӘШӯЩҲЫҢЩ„" (At-Least-Once Delivery) ЩҫШҙШӘЫҢШЁШ§ЩҶЫҢ Щ…ЫҢвҖҢЪ©ЩҶШҜ.
Щ…Ш№Ш§ЫҢШЁ:
  • Щ…ШӯШҜЩҲШҜЫҢШӘ ШҜШұ ЩӮШ§ШЁЩ„ЫҢШӘвҖҢЩҮШ§: ЩҶШіШЁШӘ ШЁЩҮ MassTransitШҢ Ш§Щ…Ъ©Ш§ЩҶШ§ШӘ Ъ©Щ…ШӘШұЫҢ ШЁШұШ§ЫҢ Щ…ШҜЫҢШұЫҢШӘ ЩҫЫҢЪҶЫҢШҜЪҜЫҢвҖҢЩҮШ§ЫҢ ЩҫЫҢШҙШұЩҒШӘЩҮ ШҜШ§ШұШҜ.
  • ШӘЩ…ШұЪ©ШІ ШЁШұ ШұЩҲЫҢШҜШ§ШҜЩҮШ§: ШЁЫҢШҙШӘШұ ШЁШұШ§ЫҢ ШіЩҶШ§ШұЫҢЩҲЩҮШ§ЫҢ ШұЩҲЫҢШҜШ§ШҜЩ…ШӯЩҲШұ Щ…ЩҶШ§ШіШЁ Ш§ШіШӘ ШӘШ§ ШҜШіШӘЩҲШұШ§ШӘ (Commands).

Щ…Ш«Ш§Щ„ Ъ©ШҜ:
public class OrderController : ControllerBase
{
    private readonly ICapPublisher _capPublisher;

    public OrderController(ICapPublisher capPublisher)
    {
        _capPublisher = capPublisher;
    }

    [HttpPost]
    public async Task<IActionResult> CreateOrder(OrderDto order)
    {
        await _capPublisher.PublishAsync("order.created", order);
        return Ok();
    }
}
ШҜШұ Ш§ЫҢЩҶ Щ…Ш«Ш§Щ„ШҢ CAP ШЁШұШ§ЫҢ Ш§ЩҶШӘШҙШ§Шұ ЫҢЪ© ШұЩҲЫҢШҜШ§ШҜ Ш§ШіШӘЩҒШ§ШҜЩҮ ШҙШҜЩҮ Ъ©ЩҮ Щ…ЫҢвҖҢШӘЩҲШ§ЩҶШҜ ШӘЩҲШіШ· ШіШұЩҲЫҢШівҖҢЩҮШ§ЫҢ ШҜЫҢЪҜШұ Ш§ШҙШӘШұШ§Ъ© ШҙЩҲШҜ.

4. Brighter: ЩҫШұШҜШ§ШІШҙЪҜШұ ШҜШіШӘЩҲШұШ§ШӘ Щ…ШҙШ§ШЁЩҮ MediatR
Brighter ЫҢЪ© Ъ©ШӘШ§ШЁШ®Ш§ЩҶЩҮ ШҜЫҢЪҜШұ Ш§ШіШӘ Ъ©ЩҮ Ш§Щ„ЪҜЩҲЫҢ ЩҫШұШҜШ§ШІШҙ ШҜШіШӘЩҲШұШ§ШӘ (Command Processor) ЩҲ Ш§ШұШіШ§Щ„ (Dispatching) ШұШ§ ЩҫЫҢШ§ШҜЩҮвҖҢШіШ§ШІЫҢ Щ…ЫҢвҖҢЪ©ЩҶШҜ ЩҲ ШҙШЁШ§ЩҮШӘ ШІЫҢШ§ШҜЫҢ ШЁЩҮ MediatR ШҜШ§ШұШҜ. Ш§ЫҢЩҶ Ш§ШЁШІШ§Шұ ШЁШұ ШіШ§ШҜЪҜЫҢ ЩҲ Ш№Щ…Щ„Ъ©ШұШҜ ШӘЩ…ШұЪ©ШІ ШҜШ§ШұШҜ ЩҲ ЩӮШ§ШЁЩ„ЫҢШӘвҖҢЩҮШ§ЫҢЫҢ Щ…Ш§ЩҶЩҶШҜ ШІЩҶШ¬ЫҢШұЩҮ Щ…ШіШҰЩҲЩ„ЫҢШӘ (Chain of Responsibility) ШұШ§ Ш§ШұШ§ШҰЩҮ Щ…ЫҢвҖҢШҜЩҮШҜ.

Щ…ШІШ§ЫҢШ§:
  • ШҙШЁШ§ЩҮШӘ ШЁЩҮ MediatR: Ш§ЪҜШұ ШЁШ§ MediatR ШўШҙЩҶШ§ ЩҮШіШӘЫҢШҜШҢ Ш§ЩҶШӘЩӮШ§Щ„ ШЁЩҮ Brighter ШўШіШ§ЩҶ Ш®ЩҲШ§ЩҮШҜ ШЁЩҲШҜ.
  • ЩӮШ§ШЁЩ„ЫҢШӘвҖҢЩҮШ§ЫҢ Ш§Ш¶Ш§ЩҒЫҢ: Ш§Щ…Ъ©Ш§ЩҶШ§ШӘЫҢ Щ…Ш§ЩҶЩҶШҜ Retry ЩҲ Circuit Breaker ШұШ§ ШЁЩҮ ШөЩҲШұШӘ ШҜШ§Ш®Щ„ЫҢ ЩҫШҙШӘЫҢШЁШ§ЩҶЫҢ Щ…ЫҢвҖҢЪ©ЩҶШҜ.
  • Ш§ЩҶШ№Ш·Ш§ЩҒвҖҢЩҫШ°ЫҢШұЫҢ: Щ…ЫҢвҖҢШӘЩҲШ§ЩҶШҜ ШЁШ§ ШіЫҢШіШӘЩ…вҖҢЩҮШ§ЫҢ ЩҫЫҢШ§Щ…вҖҢШұШіШ§ЩҶ Щ…Ш§ЩҶЩҶШҜ RabbitMQ Ш§ШҜШәШ§Щ… ШҙЩҲШҜ.
Щ…Ш№Ш§ЫҢШЁ:
  • Ш¬Ш§Щ…Ш№ЩҮ Ъ©Ш§ШұШЁШұЫҢ Ъ©ЩҲЪҶЪ©ШӘШұ: ЩҶШіШЁШӘ ШЁЩҮ MediatRШҢ Щ…ЩҶШ§ШЁШ№ ЩҲ ЩҫШҙШӘЫҢШЁШ§ЩҶЫҢ Ъ©Щ…ШӘШұЫҢ ШҜШ§ШұШҜ.
  • ЩҫЫҢЪҶЫҢШҜЪҜЫҢ ШҜШұ ШіЩҶШ§ШұЫҢЩҲЩҮШ§ЫҢ Ш®Ш§Шө: Щ…Щ…Ъ©ЩҶ Ш§ШіШӘ ШЁШұШ§ЫҢ ЩҫШұЩҲЪҳЩҮвҖҢЩҮШ§ЫҢ ШЁШіЫҢШ§Шұ ШЁШІШұЪҜ ШЁЩҮ Ш§ЩҶШҜШ§ШІЩҮ Ъ©Ш§ЩҒЫҢ Щ…ЩӮЫҢШ§ШівҖҢЩҫШ°ЫҢШұ ЩҶШЁШ§ШҙШҜ.

Щ…Ш«Ш§Щ„ Ъ©ШҜ:
public class CreateOrderCommand : IRequest
{
    public OrderDto Order { get; set; }
}

public class CreateOrderHandler : RequestHandler<CreateOrderCommand>
{
    public override void Handle(CreateOrderCommand command)
    {
        // Щ…ЩҶШ·ЩӮ Ш§ЫҢШ¬Ш§ШҜ ШіЩҒШ§ШұШҙ
    }
}

// ЩҫЫҢЪ©ШұШЁЩҶШҜЫҢ Brighter
var dispatcher = new CommandDispatcher();
dispatcher.Dispatch(new CreateOrderCommand { Order = order });
Ш§ЫҢЩҶ Ъ©ШҜ ЩҶШҙШ§ЩҶ Щ…ЫҢвҖҢШҜЩҮШҜ Ъ©ЩҮ Brighter ЪҶЪҜЩҲЩҶЩҮ Щ…ЫҢвҖҢШӘЩҲШ§ЩҶШҜ ШҜШіШӘЩҲШұШ§ШӘ ШұШ§ Щ…ШҙШ§ШЁЩҮ MediatR Щ…ШҜЫҢШұЫҢШӘ Ъ©ЩҶШҜ.

ЩҶШӘЫҢШ¬ЩҮвҖҢЪҜЫҢШұЫҢ
Ш§ЩҶШӘШ®Ш§ШЁ Ш¬Ш§ЫҢЪҜШІЫҢЩҶ Щ…ЩҶШ§ШіШЁ ШЁШұШ§ЫҢ MediatR ШЁЩҮ ЩҶЫҢШ§ШІЩҮШ§ЫҢ Ш®Ш§Шө ЩҫШұЩҲЪҳЩҮ ШҙЩ…Ш§ ШЁШіШӘЪҜЫҢ ШҜШ§ШұШҜ. Ш§ЪҜШұ ШЁЩҮ ШҜЩҶШЁШ§Щ„ ШіШ§ШҜЪҜЫҢ ЩҲ Ш№Щ…Щ„Ъ©ШұШҜ ШЁШ§Щ„Ш§ ЩҮШіШӘЫҢШҜШҢ ШӘШІШұЫҢЩӮ Щ…ШіШӘЩӮЫҢЩ… ЩҲШ§ШЁШіШӘЪҜЫҢвҖҢЩҮШ§ ЪҜШІЫҢЩҶЩҮвҖҢШ§ЫҢ Ш№Ш§Щ„ЫҢ Ш§ШіШӘ. ШЁШұШ§ЫҢ ЩҫШұЩҲЪҳЩҮвҖҢЩҮШ§ЫҢ ШӘЩҲШІЫҢШ№вҖҢШҙШҜЩҮ ЩҲ ЩҫЫҢЪҶЫҢШҜЩҮвҖҢШӘШұШҢ MassTransit ЩҲ CAP Ш§ШЁШІШ§ШұЩҮШ§ЫҢ ЩӮШҜШұШӘЩ…ЩҶШҜЫҢ ЩҮШіШӘЩҶШҜ Ъ©ЩҮ ЩҮШұ Ъ©ШҜШ§Щ… ЩҶЩӮШ§Ш· ЩӮЩҲШӘ Ш®ЩҲШҜ ШұШ§ ШҜШ§ШұЩҶШҜ: MassTransit ШЁШұШ§ЫҢ ШіЩҶШ§ШұЫҢЩҲЩҮШ§ЫҢ ЩҫЫҢШҙШұЩҒШӘЩҮвҖҢШӘШұ ЩҲ CAP ШЁШұШ§ЫҢ ШұШ§ЩҮЪ©Ш§ШұЩҮШ§ЫҢ ШіШЁЪ©вҖҢШӘШұ. ШҜШұ ЩҶЩҮШ§ЫҢШӘШҢ Brighter Щ…ЫҢвҖҢШӘЩҲШ§ЩҶШҜ ЪҜШІЫҢЩҶЩҮвҖҢШ§ЫҢ Щ…ЫҢШ§ЩҶЫҢ ШЁШ§ШҙШҜ Ъ©ЩҮ ШӘШ№Ш§ШҜЩ„ЫҢ ШЁЫҢЩҶ ШіШ§ШҜЪҜЫҢ MediatR ЩҲ ЩӮШ§ШЁЩ„ЫҢШӘвҖҢЩҮШ§ЫҢ Ш§Ш¶Ш§ЩҒЫҢ Ш§ШұШ§ШҰЩҮ Щ…ЫҢвҖҢШҜЩҮШҜ.
ШӘЩҲШөЫҢЩҮ Щ…ЫҢвҖҢШҙЩҲШҜ ЩҫЫҢШҙ Ш§ШІ Ш§ЩҶШӘШ®Ш§ШЁШҢ ЩҶЫҢШ§ШІЩ…ЩҶШҜЫҢвҖҢЩҮШ§ЫҢ ЩҫШұЩҲЪҳЩҮ (Щ…Ш§ЩҶЩҶШҜ Щ…ЩӮЫҢШ§ШівҖҢЩҫШ°ЫҢШұЫҢШҢ ЩҫЫҢЪҶЫҢШҜЪҜЫҢ ЩҲ ЩҶЫҢШ§ШІ ШЁЩҮ Щ…Ш№Щ…Ш§ШұЫҢ ШӘЩҲШІЫҢШ№вҖҢШҙШҜЩҮ) ШұШ§ ШЁЩҮ ШҜЩӮШӘ ШЁШұШұШіЫҢ Ъ©ЩҶЫҢШҜ ЩҲ ШЁШ§ ШўШІЩ…Ш§ЫҢШҙ ЩҮШұ Ш§ШЁШІШ§Шұ ШҜШұ ЫҢЪ© ЩҫШұЩҲЪҳЩҮ ЩҶЩ…ЩҲЩҶЩҮШҢ ШЁЩҮШӘШұЫҢЩҶ ЪҜШІЫҢЩҶЩҮ ШұШ§ Ш§ЩҶШӘШ®Ш§ШЁ Ъ©ЩҶЫҢШҜ. Ш§ЫҢЩҶ ШұЩҲЫҢЪ©ШұШҜ ШЁЩҮ ШҙЩ…Ш§ Ъ©Щ…Ъ© Щ…ЫҢвҖҢЪ©ЩҶШҜ ШӘШ§ Ш¶Щ…ЩҶ ШӯЩҒШё Ъ©ЫҢЩҒЫҢШӘ Ъ©ШҜШҢ ШЁЩҮШұЩҮвҖҢЩҲШұЫҢ ШӘЫҢЩ… ШӘЩҲШіШ№ЩҮ ШұШ§ ЩҶЫҢШІ Ш§ЩҒШІШ§ЫҢШҙ ШҜЩҮЫҢШҜ.

    Ш§ШҙШӘШұШ§Ъ©вҖҢЩҮШ§ШўЫҢШ§ Ш§ШіШӘЩҒШ§ШҜЩҮ Ш§ШІ MediatR ШҜШұ ЩҮШұ ЩҫШұЩҲЪҳЩҮвҖҢШ§ЫҢ Ш¶ШұЩҲШұЫҢ ЫҢШ§ ШӯШӘЫҢ ШЁЩҮЫҢЩҶЩҮ Ш§ШіШӘШҹШ§ШҙШӘШұШ§Ъ©вҖҢЩҮШ§ШұШ§ЩҮЩҶЩ…Ш§ЫҢ Ш§ЩҶШӘШ®Ш§ШЁ ШЁЫҢЩҶ MediatR ЩҲ FastEndpoints ШЁШұШ§ЫҢ ЩҫЫҢШ§ШҜЩҮвҖҢШіШ§ШІЫҢ Ш§Щ„ЪҜЩҲЫҢ Ш·ШұШ§ШӯЫҢ CQRSШ§ШҙШӘШұШ§Ъ©вҖҢЩҮШ§ЩҫЫҢШ§ШҜЩҮ ШіШ§ШІЫҢ Ш§Щ„ЪҜЩҲЫҢ Mediator ШҜШұ ШҜШ§ШӘвҖҢЩҶШӘ: ШЁШұШұШіЫҢ ЩҲ Щ…ЩӮШ§ЫҢШіЩҮ ШҜЩҲ Ш§ШЁШІШ§Шұ ЩӮШҜШұШӘЩ…ЩҶШҜШ§ШҙШӘШұШ§Ъ©вҖҢЩҮШ§Ш¬Ш§ЫҢЪҜШІЫҢЩҶЫҢ MediatR ШЁШ§ System.Threading.Channels Ш§ШҙШӘШұШ§Ъ©вҖҢЩҮШ§ШӘШәЫҢЫҢШұ Щ…ШҜЩ„ ШӘШ¬Ш§ШұЫҢ MassTransitШ§ШҙШӘШұШ§Ъ©вҖҢЩҮШ§ШЁШ§ШІЩҶЪҜШұЫҢ ШҜШұ Ш§ШіШӘЩҒШ§ШҜЩҮ Ш§ШІ MediatR ШҜШұ ШӘЩҲШіШ№ЩҮ ШЁШұЩҶШ§Щ…ЩҮвҖҢЩҮШ§ЫҢ ASP.NET CoreШ§ШҙШӘШұШ§Ъ©вҖҢЩҮШ§ЩҲЩ„ЩҲШұЫҢЩҶ: Ш¬Ш§ЫҢЪҜШІЫҢЩҶЫҢ ЩӮШҜШұШӘЩ…ЩҶШҜ ШЁШұШ§ЫҢ MediatR ШҜШұ ШӘЩҲШіШ№ЩҮ ЩҶШұЩ…вҖҢШ§ЩҒШІШ§ШұЩҮШ§ЫҢ ШҜШ§ШӘвҖҢЩҶШӘШ§ШҙШӘШұШ§Ъ©вҖҢЩҮШ§ШӘШ¬Ш§ШұЫҢвҖҢШіШ§ШІЫҢ Ъ©ШӘШ§ШЁШ®Ш§ЩҶЩҮвҖҢЩҮШ§ЫҢ Щ…ШӘЩҶвҖҢШЁШ§ШІ ШҜШ§ШӘвҖҢЩҶШӘ ЩҲ ШӘШЈШ«ЫҢШұ ШўЩҶ ШЁШұ ШӘЩҲШіШ№ЩҮвҖҢШҜЩҮЩҶШҜЪҜШ§ЩҶ: ЩҶЪҜШ§ЩҮЫҢ ШЁЩҮ ЩҫШ§ЫҢШҜШ§ШұЫҢ ЩҫШұЩҲЪҳЩҮвҖҢЩҮШ§ЫҢ Щ…ШӘЩҶвҖҢШЁШ§ШІЩ…Ш·Ш§Щ„ШЁDomain-Driven Design (DDD)Шӣ ШЁШ®Шҙ ШўШ®Шұ: ЫҢЪ© ЩҫШұЩҲЪҳЩҮЩ…Ш·Ш§Щ„ШЁШЁШұШұШіЫҢ Ш§Щ„ЪҜЩҲЫҢ SAGA ШҜШұ Щ…ЫҢЪ©ШұЩҲШіШұЩҲЫҢШівҖҢЩҮШ§: ШұЩҲЫҢЪ©ШұШҜЩҮШ§ЫҢ Choreography ЩҲ Orchestration ЩҲ Ш§ШіШӘЩҒШ§ШҜЩҮ Ш§ШІ MassTransit Automatonymous
