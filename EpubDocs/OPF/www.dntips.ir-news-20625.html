‫خلاصه گزارش Octoverse 2025 GitHub


نویسنده: وحید نصیری
تاریخ: ۱۴۰۴/۰۸/۱۴ ۱۲:۳۸
آدرس: www.dntips.ir
گزارش سالانه Octoverse 2025 GitHub، که بر اساس داده‌های سپتامبر ۲۰۲۴ تا اوت ۲۰۲۵ تهیه شده، بر رشد بی‌سابقه جامعه توسعه‌دهندگان، نقش هوش مصنوعی (AI) و تغییرات در زبان‌های برنامه‌نویسی تمرکز دارد. این گزارش نشان‌دهنده بزرگ‌ترین تغییرات در توسعه نرم‌افزار طی بیش از یک دهه است.

رشد جامعه توسعه‌دهندگان:
  • هر ثانیه بیش از یک توسعه‌دهنده جدید به GitHub می‌پیوندد، که منجر به اضافه شدن بیش از ۳۶ میلیون توسعه‌دهنده جدید در سال گذشته شده است (رشد ۲۳ درصدی نسبت به سال قبل). اکنون بیش از ۱۸۰ میلیون توسعه‌دهنده روی GitHub فعالیت می‌کنند.
  • راه‌اندازی GitHub Copilot Free در اواخر ۲۰۲۴ باعث جهش در ثبت‌نام‌ها شد. حدود ۸۰ درصد توسعه‌دهندگان جدید در هفته اول از Copilot استفاده می‌کنند.
  • رشد جهانی: هند با اضافه کردن بیش از ۵ میلیون توسعه‌دهنده (۱۴ درصد کل جدیدها)، رتبه دوم را دارد و پیش‌بینی می‌شود تا ۲۰۳۰ یک سوم توسعه‌دهندگان جدید را تشکیل دهد. کشورهای برتر: ایالات متحده (۲۸ میلیون)، هند (۲۲ میلیون)، چین، برزیل و غیره.

نقش هوش مصنوعی و ابزارهای عامل‌محور (Agentic):
  • AI اکنون استاندارد توسعه است: بیش از ۱.۱ میلیون مخزن عمومی از SDKهای مدل‌های زبانی بزرگ (LLM) استفاده می‌کنند (رشد ۱۷۸ درصدی). ۴.۳ میلیون پروژه AI وجود دارد (دو برابر از ۲۰۲۳).
  • فعالیت‌ها رکورد زد: ۵۱۸.۷ میلیون pull request ادغام‌شده (رشد ۲۹ درصدی)، ۹۸۶ میلیون commit (رشد ۲۵ درصدی). ابزارهایی مانند Copilot Autofix و Copilot Code Review به بهبود امنیت و کارایی کمک کرده‌اند.
  • عامل‌ها (Agents) وارد جریان اصلی شده‌اند: بیش از ۱ میلیون pull request توسط Copilot Coding Agent ایجاد شده.

زبان‌های برنامه‌نویسی:
  • TypeScript برای اولین بار در اوت ۲۰۲۵ به عنوان پراستفاده‌ترین زبان GitHub پیشی گرفت (بیش از Python و JavaScript)، با رشد ۶۶ درصدی (۱ میلیون توسعه‌دهنده جدید). این نشان‌دهنده گرایش به زبان‌های تایپ‌دار برای کدهای قابل اعتمادتر با AI است.
  • رتبه‌بندی برتر: ۱. TypeScript، ۲. Python (رشد ۴۸ درصدی، غالب در AI و داده‌کاوی)، ۳. JavaScript، ۴. Java، ۵. C#.
  • زبان‌های سریع‌رشد: Luau (۱۹۴ درصد)، Typst (۱۰۸ درصد)، Astro (۷۸ درصد).

منبع‌باز (Open Source):
  • ۳۹۵ میلیون مخزن عمومی (رشد ۱۹ درصدی)، ۱.۱۲ میلیارد contribution (رشد ۱۳ درصدی).
  • ۶۰ درصد از ۱۰ پروژه برتر توسط مشارکت‌کنندگان، AI-محور هستند (مانند vLLM، Hugging Face Transformers).
  • پروژه‌های سریع‌رشد: Zen Browser، Cline، vLLM.

امنیت:
  • زمان رفع آسیب‌پذیری‌های بحرانی ۳۰ درصد سریع‌تر شده (از ۳۷ به ۲۶ روز). استفاده از Dependabot دو برابر شده (۸۴۶ هزار مخزن).
  • آسیب‌پذیری برتر: Broken Access Control (رشد ۱۷۲ درصدی)، که اغلب به دلیل کدهای AI-تولیدشده است.

نتیجه‌گیری:
گزارش تأکید می‌کند که AI نه تنها سرعت کدزنی را افزایش می‌دهد، بلکه انتخاب زبان‌ها و ابزارها را تغییر می‌دهد. GitHub به عنوان مرکز همگرایی توسعه‌دهندگان جهانی عمل می‌کند و پیش‌بینی می‌شود هند تا ۲۰۳۰ بزرگ‌ترین جامعه را داشته باشد. این تغییرات نشان‌دهنده ورود به عصر AI-محور توسعه نرم‌افزار است.
