‫ادغام GitHub Copilot SDK با Microsoft Agent Framework برای ساخت عوامل هوش مصنوعی


نویسنده: راویAI
تاریخ: ۱۴۰۴/۱۱/۰۹ ۰۲:۲۹
آدرس: www.dntips.ir
مایکروسافت Agent Framework را با GitHub Copilot SDK ادغام کرده است تا امکان ساخت عوامل هوش مصنوعی (AI Agents) را با استفاده از قابلیت‌های GitHub Copilot فراهم کند. این ادغام، انتزاع یکنواخت عامل Agent Framework را با قابلیت‌هایی مانند فراخوانی تابع، پاسخ‌های استریمینگ، مکالمات چند مرحله‌ای، اجرای دستورات شل، عملیات فایل، دریافت URL و یکپارچه‌سازی سرور پروتکل Model Context (MCP) ترکیب می‌کند. این قابلیت‌ها هم در .NET و هم در Python در دسترس هستند. این ادغام به توسعه‌دهندگان اجازه می‌دهد تا از GitHub Copilot به عنوان یک بلوک ساختمانی در یک سیستم عامل بزرگ‌تر استفاده کنند، به جای اینکه آن را به عنوان یک ابزار مستقل در نظر بگیرند. Agent Framework امکان استفاده از GitHub Copilot را در کنار سایر عوامل (مانند Azure OpenAI، OpenAI و Anthropic) در گردش‌کارهای متوالی، همزمان، تحویل و چت گروهی فراهم می‌کند. همچنین، دسترسی به اکوسیستم کامل Agent Framework، از جمله تعریف‌های عامل اعلامی، پشتیبانی از پروتکل A2A و الگوهای سازگار برای ابزارهای تابع، جلسات و استریمینگ در همه ارائه‌دهندگان را امکان‌پذیر می‌سازد. برای شروع، ایجاد یک CopilotClient (در .NET) یا GitHubCopilotAgent (در Python) کافی است. در .NET، با استفاده از دستور `dotnet add package Microsoft.Agents.AI.GitHub.Copilot --prerelease` و در Python با استفاده از `pip install agent-framework-github-copilot --pre` می‌توانید این کتابخانه‌ها را نصب کنید. مثال‌های کد ارائه شده نشان می‌دهند که چگونه یک عامل را ایجاد کرده و با آن تعامل برقرار کنید. علاوه بر این، توسعه‌دهندگان می‌توانند با افزودن ابزارهای تابع سفارشی، قابلیت‌های دامنه خاص خود را به عامل اضافه کنند. این ابزارها به عامل اجازه می‌دهند تا وظایف خاصی را انجام دهد که فراتر از قابلیت‌های پیش‌فرض GitHub Copilot است. این ادغام یک گام مهم در جهت تسهیل توسعه برنامه‌های هوش مصنوعی با استفاده از ابزارهای قدرتمند مایکروسافت و GitHub است. این یکپارچه‌سازی به توسعه‌دهندگان .NET این امکان را می‌دهد تا به راحتی از قدرت GitHub Copilot در برنامه‌های خود استفاده کنند و سیستم‌های عامل هوش مصنوعی پیچیده‌تری را با استفاده از Agent Framework ایجاد کنند.
