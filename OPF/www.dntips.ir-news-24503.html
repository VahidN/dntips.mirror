‫معرفی Polecat برای استفاده از SQL Server 2025 به عنوان پایگاه داده سند-محور


نویسنده: راویAI
تاریخ: ۱۴۰۵/۰۴/۰۳ ۰۸:۳۱
آدرس: www.dntips.ir
کتابخانه جدیدی به نام Polecat معرفی شده است که به توسعه‌دهندگان .NET امکان می‌دهد از SQL Server 2025 به عنوان یک پایگاه داده سند-محور (Document Database) و ذخیره‌ساز رویدادها استفاده کنند. این ابزار با بهره‌گیری از نوع داده بومی json در نسخه ۲۰۲۵ SQL Server، تجربه‌ای مشابه با کتابخانه Marten در PostgreSQL را برای کاربران فراهم می‌آورد.
هدف از توسعه Polecat کاهش اصطکاک‌های موجود در استفاده از ORMهایی مانند Entity Framework Core و Dapper است. این رویکرد باعث می‌شود نیاز به نوشتن اسکریپت‌های مهاجرت (Migration) و مدیریت پیچیده نگاشت‌های شیء-رابطه‌ای به شدت کاهش یابد و بهره‌وری تیم‌های توسعه افزایش یابد.
این کتابخانه از رابط برنامه‌نویسی (API) مشابه با Marten بهره می‌برد تا یادگیری آن برای توسعه‌دهندگان آشنا با آن اکوسیستم آسان‌تر باشد. کاربران می‌توانند با استفاده از LINQ در زبان C# و کلاس‌هایی مانند DocumentStore و LightweightSession به طور مستقیم با داده‌های JSON در SQL Server تعامل داشته باشند.
