ЩҶЩҲЫҢШіЩҶШҜЪҜШ§ЩҶ ЫІЫ·Ы¶
ЪҜШұЩҲЩҮвҖҢЩҮШ§ЫҢ Щ…Ш·Ш§Щ„ШЁ ЫұЫ°ЫІЫҙ
ЩҶЩӮШҙЩҮвҖҢЩҮШ§ЫҢ ШұШ§ЩҮ ЫұЫұЫ№
ШҜЩҲШұЩҮвҖҢЩҮШ§ ЫұЫҙ
Ш§ШҙШӘШұШ§Ъ©вҖҢЩҮШ§ ЫұЫ·Ы№ЫұЫІ
→ вҖ«Ъ©Ш§ЩҮШҙ ШӘШ№ШҜШ§ШҜ ШЁШ§Шұ ШӘШ№ШұЫҢЩҒ using ЩҮШ§ ШҜШұ C# 10.0 ЩҲ NET 6.0.
вҖ«Щ…ШӘШҜ Ш¬ШҜЫҢШҜ Chunk ШҜШұ ШҜШ§ШӘ ЩҶШӘ 6 ←
Ш№ЩҶЩҲШ§ЩҶ:

вҖ«ШұЩҲШҙ ШЁШ§ШІЪҜШҙШӘ ШЁЩҮ ЩӮШ§Щ„ШЁвҖҢЩҮШ§ЫҢ Ъ©Щ„Ш§ШіЫҢЪ© ЩҫШұЩҲЪҳЩҮвҖҢЩҮШ§ ШҜШұ ШҜШ§ШӘ ЩҶШӘ 6


ЩҶЩҲЫҢШіЩҶШҜЩҮ: ЩҲШӯЫҢШҜ ЩҶШөЫҢШұЫҢ
ШӘШ§ШұЫҢШ®: ЫұЫҙЫ°Ы°/Ы°Ы·/ЫІЫ¶ ЫұЫұ:ЫұЫ°
ШўШҜШұШі: www.dntips.ir
ЩҶЪҜШ§ШұШҙ ЩҶЩҮШ§ЫҢЫҢ ШҜШ§ШӘ ЩҶШӘ 6ШҢ ШӯШҜЩҲШҜ ЫҢЪ© Щ…Ш§ЩҮ ШҜЫҢЪҜШұ Щ…ЩҶШӘШҙШұ Щ…ЫҢвҖҢШҙЩҲШҜ ЩҲ Ш§ЪҜШұ ШЁШұШ§ЫҢ ЩҶЩ…ЩҲЩҶЩҮ RC2 ШўЩҶвҖҢШұШ§ ЩҶШөШЁ Ъ©ШұШҜЩҮ ШЁШ§ШҙЫҢШҜШҢ ШЁШ§ Ш§ЫҢШ¬Ш§ШҜ ЫҢЪ© ЩҫШұЩҲЪҳЩҮвҖҢЫҢ Ъ©ЩҶШіЩҲЩ„ Ш¬ШҜЫҢШҜ Щ…ШЁШӘЩҶЫҢ ШЁШұ ШўЩҶ ... ШҙЪҜЩҒШӘ ШІШҜЩҮ Ш®ЩҲШ§ЩҮЫҢШҜ ШҙШҜ!  ШҙШ§ЫҢШҜ Ш§ЩҶШӘШёШ§Шұ ШҜШ§ШҙШӘЩҮ ШЁШ§ШҙЫҢШҜ Ъ©ЩҮ ШЁШ§ ЪҶЩҶЫҢЩҶ ЩҒШ§ЫҢЩ„ЫҢ Щ…ЩҲШ§Ш¬ЩҮ ШҙЩҲЫҢШҜ:
using System; 
 
namespace MyVerboseApp 
{ 
    public class Program 
    { 
        public static void Main(string[] args) 
        { 
            Console.WriteLine("Hello World!"); 
        } 
    } 
}
Ш§Щ…Ш§ ЫҢЪ© ЪҶЩҶЫҢЩҶ Ш®ШұЩҲШ¬ЫҢ ШӘЩҲЩ„ЫҢШҜ Щ…ЫҢвҖҢШҙЩҲШҜ:
 // See https://aka.ms/new-console-template for more information
Console.WriteLine("Hello, World!");
Ш§ЫҢЩҶ Щ…ЩҲШұШҜ ЩӮШ§ШЁЩ„ЫҢШӘЫҢ Ш§ШіШӘ Ъ©ЩҮ ШЁЩҮ ЩҮЩ…ШұШ§ЩҮ C# 9.0 ШЁЩҮ ЩҶШ§Щ… В«Top Level ProgramsВ» Ш§ШұШ§ШҰЩҮ ШҙШҜ ЩҲ Ш§Ъ©ЩҶЩҲЩҶ ШҜШұ ШӘЩ…Ш§Щ… ЩӮШ§Щ„ШЁвҖҢЩҮШ§ЫҢ ЩҫЫҢШҙвҖҢЩҒШұШ¶ ЩҫШұЩҲЪҳЩҮвҖҢЩҮШ§ЫҢ Щ…ШЁШӘЩҶЫҢ ШЁШұ ШҜШ§ШӘ ЩҶШӘ 6ШҢ Ш§ШіШӘЩҒШ§ШҜЩҮ ШҙШҜЩҮвҖҢШ§ШіШӘ. Ш§ЫҢЩҶ ЩӮШ§Щ„ШЁ ШҙШ§ЫҢШҜ ШЁШұШ§ЫҢ ШӘШ§ШІЩҮвҖҢЪ©Ш§ШұЩҮШ§ШҢ Ш¬Ш§Щ„ШЁ ШЁШ§ШҙШҜ ЩҲ Ъ©Щ… ШӯШ¬Щ… ЩҲ Ъ©Щ… ШіШ·ШұШҢ Ш§Щ…Ш§ В«Щ…Ш§ ШўЩҶвҖҢШұШ§ ШҜШұШ®ЩҲШ§ШіШӘ ЩҶШҜШ§ШҜЩҮ ШЁЩҲШҜЫҢЩ…!В».


ШұЩҲШҙ ШЁШ§ШІЪҜШҙШӘ ШЁЩҮ ЩӮШ§Щ„ШЁвҖҢЩҮШ§ЫҢ ЩӮШЁЩ„ЫҢ

ШҜШұ ШӯШ§Щ„ ШӯШ§Ш¶Шұ ЩҲ ШҜШұ ЩҶЪҜШ§ШұШҙ ЩҒШ№Щ„ЫҢ ЩҲ ШӯШӘЫҢ ШұШіЩ…ЫҢ ШҜШ§ШӘ ЩҶШӘ 6ШҢ ШұЩҲШҙЫҢ ШЁШұШ§ЫҢ ШЁШ§ШІЪҜШҙШӘ ШЁЩҮ ШӯШ§Щ„ШӘ ЩӮШЁЩ„ЫҢ ЩҲШ¬ЩҲШҜ ЩҶШҜШ§ШұШҜ Ъ©ЩҮ ШЁЩҮ Ш§ШӯШӘЩ…Ш§Щ„ ШІЫҢШ§ШҜ ШҜШұ ЩҶЪҜШ§ШұШҙвҖҢЩҮШ§ЫҢ ЩҫШі Ш§ШІ RTM Щ„ШӯШ§Шё Ш®ЩҲШ§ЩҮШҜ ШҙШҜ (Щ…ЫҢвҖҢШӘЩҲШ§ЩҶЫҢШҜ ШҜШұ Ш§ЫҢЩҶШ¬Ш§ ^ ЩҲ ^ ШЁЩҮ ШўЩҶ ШұШ§ЫҢ ШҜЩҮЫҢШҜ). ШӘЩҶЩҮШ§ ШұШ§ЩҮ ШӯЩ„ Щ…ЩҲШ¬ЩҲШҜШҢ Ш§ШіШӘЩҒШ§ШҜЩҮ Ш§ШІ ШҜШіШӘЩҲШұ ШІЫҢШұ Ш§ШіШӘ:
dotnet new console --framework net5.0 --target-framework-override net6.0
Ш§ЫҢЩҶ ШҜШіШӘЩҲШұ ШҜШұ Ш§ШөЩ„ ШЁЩҮ Ш§ЫҢЩҶ Щ…Ш№ЩҶШ§ Ш§ШіШӘ Ъ©ЩҮ ЩҫШұЩҲЪҳЩҮвҖҢЫҢ Щ…ЩҶ ШұШ§ ШЁШұ Ш§ШіШ§Ші ЩӮШ§Щ„ШЁ ЩҫШұЩҲЪҳЩҮвҖҢЩҮШ§ЫҢ NET 5.0. ШӘЩҲЩ„ЫҢШҜ Ъ©ЩҶШӣ Ш§Щ…Ш§ ШҜШұ ЩҒШ§ЫҢЩ„ csproj ШўЩҶШҢ ШЁШ¬Ш§ЫҢ net5.0 Ш§ШІ net6.0 ШЁЩҮ Ш№ЩҶЩҲШ§ЩҶ target framework Ш§ШіШӘЩҒШ§ШҜЩҮ ШҙЩҲШҜ:
<Project Sdk="Microsoft.NET.Sdk">
<PropertyGroup>
    <OutputType>Exe</OutputType>
-    <TargetFramework>net5.0</TargetFramework>
+    <TargetFramework>net6.0</TargetFramework>
  </PropertyGroup>
</Project>
ШҜШұ Ш§ЫҢЩҶШ¬Ш§ ШіШ·Шұ net5.0 ШұШ§ ШӯШ°ЩҒ ЩҲ ШЁШ§ net6.0 Ш¬Ш§ЫҢЪҜШІЫҢЩҶ Ъ©ЩҶЫҢШҜ.
Щ…Ш·Ш§Щ„ШЁ Щ…ШҙШ§ШЁЩҮ
    Щ…Ш·Ш§Щ„ШЁШ§ШІ NET Standard. ШЁЩҮ NET 5.Щ…Ш·Ш§Щ„ШЁЪ©Ш§ЩҮШҙ ШӘШ№ШҜШ§ШҜ ШЁШ§Шұ ШӘШ№ШұЫҢЩҒ using ЩҮШ§ ШҜШұ C# 10.0 ЩҲ NET 6.0.Щ…Ш·Ш§Щ„ШЁШЁЩҮШЁЩҲШҜЩҮШ§ЫҢ ШӘЩҲЩ„ЫҢШҜ ШЁШұЩҶШ§Щ…ЩҮвҖҢЩҮШ§ЫҢ Ш§Ш¬ШұШ§ЫҢЫҢ ШӘЪ© ЩҒШ§ЫҢЩ„ЫҢ ШҜШұ NET 6x.Щ…Ш·Ш§Щ„ШЁШ§ШұШӘЩӮШ§ШЎ ЩҒШ§ЫҢЩ„вҖҢЩҮШ§ЫҢ ШўШәШ§ШІЫҢЩҶ ШЁШұЩҶШ§Щ…ЩҮвҖҢЩҮШ§ЫҢ ASP.NET Core 5x ШЁЩҮ 6xЩ…Ш·Ш§Щ„ШЁЪ©Ш§Щ…ЩҫШ§ЫҢЩ„Шұ C# 9.0ШҢ Ш®Ш·Ш§ЩҮШ§ ЩҲ Ш§Ш®Ш·Ш§ШұЩҮШ§ЫҢ ШЁЫҢШҙШӘШұЫҢ ШұШ§ ЩҶЩ…Ш§ЫҢШҙ Щ…ЫҢвҖҢШҜЩҮШҜЩ…Ш·Ш§Щ„ШЁШЁЩҮ ШұЩҲШІ ШұШіШ§ЩҶЫҢ ЩӮШ§Щ„ШЁ ЩҒШ§ЫҢЩ„вҖҢЩҮШ§ЫҢ csproj ЩҫШұЩҲЪҳЩҮвҖҢЩҮШ§ЫҢ ЩӮШҜЫҢЩ…ЫҢЩ…Ш·Ш§Щ„ШЁШӘЩҶШёЫҢЩ…Ш§ШӘ Щ…ЩҲШұШҜ ЩҶЫҢШ§ШІ Ш¬ЩҮШӘ ШҙШұЩҲШ№ ШЁЩҮ Ъ©Ш§Шұ ШЁШ§ C# 9.0Щ…Ш·Ш§Щ„ШЁШЁЩҮШЁЩҲШҜ Ъ©Ш§ШұШўЫҢЫҢ Reflection ШҜШұ ШҜШ§ШӘ ЩҶШӘ 7ЩҶШёШұШ§ШӘ Щ…Ш·Ш§Щ„ШЁШўШІЩ…ЩҲЩҶвҖҢЩҮШ§ЫҢ ЫҢЪ©ЩҫШ§ШұЪҶЪҜЫҢ ШҜШұ ШЁШұЩҶШ§Щ…ЩҮвҖҢЩҮШ§ЫҢ ASP.NET CoreЩҶШёШұШ§ШӘ Щ…Ш·Ш§Щ„ШЁЩ…ШіШӘЩҶШҜ ШіШ§ШІЫҢ ASP.NET Core 2x API ШӘЩҲШіШ· OpenAPI Swagger - ЩӮШіЩ…ШӘ ШҜЩҲЩ… - ШҙШұЩҲШ№ ШЁЩҮ Щ…ШіШӘЩҶШҜ ШіШ§ШІЫҢ ЫҢЪ© API
