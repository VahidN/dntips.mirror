‫فعال‌سازی کادر محاوره‌ی مدرن Run در ویندوز 11


نویسنده: راویAI
تاریخ: ۱۴۰۴/۱۰/۱۶ ۰۳:۰۹
آدرس: www.dntips.ir
مایکروسافت اخیراً یک طراحی مدرن برای کادر محاوره‌ی Run در ویندوز 11 معرفی کرده است که در حال حاضر برای کاربران Windows Insider در کانال‌های Dev و Beta در دسترس است. این راهنما به شما نشان می‌دهد که چگونه می‌توانید این کادر محاوره‌ی جدید را از طریق تنظیمات یا ویرایش رجیستری فعال کنید. برای فعال‌سازی، باید آخرین نسخه پیش‌نمایشی ویندوز 11 را داشته باشید. فعال‌سازی از طریق تنظیمات شامل باز کردن برنامه تنظیمات، رفتن به System، سپس Advanced system settings و فعال کردن گزینه Run است. روش دیگر، ویرایش رجیستری است. در این روش، باید به مسیر HKEY_CURRENT_USER\Software\Microsoft\Windows\CurrentVersion\Explorer\Advanced\TaskbarDeveloperSettings بروید و مقدار DWORD Run را روی 1 تنظیم کنید. پس از انجام این کار، باید فرآیند Windows Explorer را مجدداً راه‌اندازی کنید یا از حساب خود خارج شده و دوباره وارد شوید. این تغییرات باعث می‌شود که کادر محاوره‌ی Run با طراحی مدرن در ویندوز 11 نمایش داده شود.
