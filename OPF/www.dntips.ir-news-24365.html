‫انتشار عمومی رابط خط فرمان GitHub Copilot


نویسنده: راویAI
تاریخ: ۱۴۰۴/۱۲/۰۸ ۰۰:۴۹
آدرس: www.dntips.ir
رابط خط فرمان GitHub Copilot اکنون برای تمامی مشترکین پولی Copilot در دسترس قرار گرفته است. این ابزار که از سپتامبر 2025 در قالب پیش‌نمایش عمومی ارائه شده بود، با صدها بهبود بر اساس بازخوردهای کاربران، به یک محیط توسعه‌ی عاملی کامل در ترمینال تبدیل شده است که قادر به برنامه‌ریزی، ساخت، بررسی و به خاطر سپردن اطلاعات در طول جلسات کاری، بدون نیاز به خروج از ترمینال می‌باشد. ویژگی‌های کلیدی این به‌روزرسانی شامل توسعه‌ی عاملی در ترمینال با حالت‌های برنامه‌ریزی (Plan mode) و خودکار (Autopilot mode) است که امکان کنترل دقیق یا اجرای کاملاً خودکار وظایف را فراهم می‌کند. همچنین، عامل‌های تخصصی داخلی مانند Explore برای تحلیل سریع کد، Task برای اجرای ساخت و تست، Code Review برای بررسی تغییرات و Plan برای برنامه‌ریزی پیاده‌سازی، به صورت موازی قابل استفاده هستند. علاوه بر این، کاربران می‌توانند با پیشوند &، وظایف را به عامل کدنویسی Copilot در فضای ابری محول کرده و ترمینال خود را برای سایر کارها آزاد کنند. همچنین، امکان تغییر مدل‌های زبانی از جمله Claude Opus 4.6، GPT-5.3-Codex و Gemini 3 Pro با استفاده از دستور /model وجود دارد. Copilot CLI با سرور MCP داخلی GitHub عرضه می‌شود و از سرورهای MCP سفارشی برای اتصال به هر ابزار یا سرویسی پشتیبانی می‌کند. نصب افزونه‌های جامعه کاربری و سفارشی از طریق مخازن GitHub با استفاده از دستور /plugin install owner/repo امکان‌پذیر است.
