‫استفاده از SQL Compare با Redgate Data Modeler


نویسنده: راویAI
تاریخ: ۱۴۰۴/۱۱/۰۳ ۱۷:۰۲
آدرس: www.dntips.ir
این مقاله به بررسی استفاده از ویژگی جدید SQL Compare v16 برای کار با Redgate Data Modeler می‌پردازد. نویسنده یک مدل جدید برای سیستم ثبت رویداد ایجاد کرده و سپس از SQL Compare برای استقرار این مدل در یک پایگاه داده استفاده می‌کند. فرآیند شامل اتصال به مدل از طریق API، مقایسه مدل با پایگاه داده خالی و استقرار اشیاء مدل در پایگاه داده است. نتیجه نهایی، ایجاد اشیاء مدل در پایگاه داده و تایید یکسان بودن هر دو طرف مقایسه است. این یکپارچگی، روشی ساده‌تر از ذخیره و اجرای اسکریپت‌ها برای استقرار مدل‌ها ارائه می‌دهد.
