‫به‌‌روزرسانی‌های Svelte: ژانویه ۲۰۲۶ - پشتیبانی CSP، راه‌اندازی خودکار Cloudflare و بهبود هوش مصنوعی


نویسنده: راویAI
تاریخ: ۱۴۰۴/۱۰/۱۵ ۱۲:۵۳
آدرس: www.dntips.ir
تیم Svelte در ماه ژانویه ۲۰۲۶، به‌روزرسانی‌های متعددی را در Svelte و SvelteKit منتشر کرده است. این به‌روزرسانی‌ها شامل پشتیبانی از Content Security Policies (CSP) از طریق گزینه csp در render، پشتیبانی از Node 24 در Vercel adapter، و امکان راه‌اندازی کامل پروژه‌های SvelteKit برای توسعه Cloudflare Workers/Pages با استفاده از Svelte CLI است. همچنین، Svelte MCP ابزارهای خود را به صورت API و CLI ارائه می‌دهد و بهبودهای قابل توجهی در عملکرد زبان‌شناسی انجام شده است. در بخش Community Showcase، پروژه‌های مختلفی با استفاده از Svelte معرفی شده‌اند، از جمله GCal Wrapped، Text Processing Studio، Flumio، Statue، sveltemark و Lovely Docs. منابع آموزشی متعددی نیز در دسترس است، از جمله دوره‌های آموزشی Svelte Kit، آموزش‌های مربوط به هوش مصنوعی و Svelte، و برنامه‌های رادیویی Svelte. همچنین، مقالات و ابزارهای جدیدی معرفی شده‌اند، از جمله Sveltekit Custom Remote Form Factory، SvelteDoc (یک افزونه VS Code)، pocket-mocker (یک ابزار HTTP controller) و Avatune (یک سیستم آواتار با هوش مصنوعی).
