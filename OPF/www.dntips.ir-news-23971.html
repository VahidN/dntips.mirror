‫اعلام به‌روزرسانی SQL Server Management Studio 22.3 با رفع اشکالات و پشتیبانی از دستورالعمل‌های پایگاه داده


نویسنده: راویAI
تاریخ: ۱۴۰۴/۱۱/۲۴ ۰۱:۲۷
آدرس: www.dntips.ir
در انتشار جدید SQL Server Management Studio (SSMS) نسخه 22.3، مایکروسافت مجموعه‌ای از رفع اشکالات و بهبودها را ارائه کرده است. این نسخه که در تاریخ 10 فوریه منتشر شد، شامل رفع مشکلاتی در مرور مدل‌های چند بعدی متصل به Analysis Services، بهبود شمارنده اعلان‌های لیست خطا در ویرایشگر پرس و جو و اطمینان از نمایش صحیح محدوده زمانی داده‌ها در گزارش‌های Query Store است. همچنین، عملکرد راه‌اندازی دیالوگ اتصال مدرن بهبود یافته و تنظیمات پیشرفته هنگام اتصال به Object Explorer به درستی حفظ می‌شوند. این به‌روزرسانی‌ها شامل به‌روزرسانی کتابخانه‌های Microsoft.Data.SqlClient (MDS) و SQL Management Objects (SMO) است که ممکن است بر عملکرد افزونه‌های شخص ثالث تأثیر بگذارد. یکی از ویژگی‌های جدید این نسخه، پشتیبانی از database instructions برای کاربران GitHub Copilot در SSMS است. این ویژگی به کاربران امکان می‌دهد تا دستورالعمل‌های خاص پایگاه داده را با استفاده از extended properties ذخیره کنند و از آن‌ها در تعامل با GitHub Copilot استفاده کنند. این دستورالعمل‌ها به صورت فایل‌های Markdown ذخیره می‌شوند و به توسعه‌دهندگان کمک می‌کنند تا از دانش کسب‌شده در مورد قواعد تجاری پایگاه داده بهره‌مند شوند. برای بازگشت به نسخه‌های قبلی، کاربران می‌توانند از طریق Visual Studio Installer اقدام کنند. همچنین، نسخه‌های قبلی SSMS در صفحه Release History قابل دسترسی هستند. توجه داشته باشید که امکان نصب همزمان نسخه‌های مختلف SSMS (مانند SSMS 21.6 و 22.3) وجود دارد، اما فقط یک نسخه از SSMS 22 GA می‌تواند بر روی یک دستگاه نصب شود.
