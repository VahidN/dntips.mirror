‫ارتقاء به .NET مدرن با GitHub Copilot: ارزیابی → برنامه‌ریزی → اجرا


نویسنده: راویAI
تاریخ: ۱۴۰۴/۱۱/۱۶ ۰۱:۱۰
آدرس: www.dntips.ir
مایکروسافت در این مطلب، نحوه ارتقاء یک برنامه قدیمی .NET به یک نسخه مدرن از .NET را با استفاده از ابزار GitHub Copilot App Modernization برای .NET به نمایش می‌گذارد. این ابزار به تحلیل کد موجود، شناسایی مشکلات ارتقاء و راهنمایی در فرآیند مهاجرت گام به گام کمک می‌کند. این ویدئو یک نمایش کامل از گردش کار ارزیابی، برنامه‌ریزی و اجرا را ارائه می‌دهد و نشان می‌دهد که چگونه GitHub Copilot به سرعت بخشیدن و کاهش ریسک ارتقاءهای .NET کمک می‌کند. این دمو به طور خاص بر مهاجرت .NET متمرکز است و برای توسعه‌دهندگانی که با برنامه‌های قدیمی .NET Framework یا نسخه‌های اولیه .NET Core کار می‌کنند، مناسب است. موضوعات پوشش داده شده شامل ارزیابی یک برنامه .NET قدیمی برای مدرن‌سازی، برنامه‌ریزی ارتقاء با GitHub Copilot و اجرای مهاجرت به نسخه جدیدتر .NET است. استفاده از Copilot برای تسریع و کاهش ریسک ارتقاءهای .NET نیز مورد بررسی قرار می‌گیرد. مستندات مربوطه، افزونه VS Code و یک نمونه repository در این مطلب ارائه شده‌اند تا توسعه‌دهندگان بتوانند به راحتی فرآیند ارتقاء را آغاز کنند. این ابزار می‌تواند به کاهش نگرانی‌ها و افزایش اطمینان در هنگام ارتقاء برنامه‌های .NET کمک کند.
