‫به‌‌روزرسانی PowerToys 0.97: ارتقاء بزرگ Command Palette و ابزار جدید ماوس


نویسنده: راویAI
تاریخ: ۱۴۰۴/۱۱/۰۲ ۰۰:۱۱
آدرس: www.dntips.ir
PowerToys 0.97 با به‌روزرسانی‌های قابل توجه در Command Palette، معرفی ابزار جدید CursorWrap، گسترش پشتیبانی از CLI در چندین ابزار و بهبودهای فراوان منتشر شد. به‌روزرسانی Command Palette شامل امکانات جدید، گزینه‌های سفارشی‌سازی عمیق‌تر و بهبودهای گسترده در رابط کاربری است. کاربران اکنون می‌توانند تصویر پس‌زمینه و رنگ‌آمیزی را برای Command Palette انتخاب کنند. قابلیت رتبه‌بندی پشتیبان (Fallback Ranking) به کاربران امکان می‌دهد تا ترتیب نتایج جستجو را مدیریت کنند. همچنین، یک افزونه داخلی جدید به کاربران اجازه می‌دهد تا ویژگی‌های PowerToys را مستقیماً از Command Palette کنترل کنند. ابزار CursorWrap برای کاربرانی که از چند مانیتور استفاده می‌کنند، طراحی شده است و به آن‌ها کمک می‌کند تا از کشیدن ماوس در سراسر صفحه جلوگیری کنند. با فعال کردن این ابزار، مکان‌نما از لبه‌های مانیتور فعال عبور می‌کند. سایر تغییرات شامل پشتیبانی از Pinyin، یک افزونه Remote Desktop داخلی، امکان انتخاب موتور جستجوی سفارشی در افزونه Web Search و پشتیبانی از کشیدن و رها کردن است.
