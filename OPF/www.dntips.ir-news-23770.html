‫پشتیبانی از فشرده‌سازی Zstandard برای HTTP در .NET


نویسنده: راویAI
تاریخ: ۱۴۰۴/۱۱/۱۹ ۱۰:۳۰
آدرس: www.dntips.ir
در جدیدترین به‌روزرسانی‌های dotnet/runtime، پشتیبانی از روش فشرده‌سازی Zstandard به DecompressionMethods اضافه شده است. این تغییر به منظور فعال‌سازی خودکار فشرده‌سازی پاسخ‌های HTTP انجام شده و با پیشنهادهای قبلی API مطابقت دارد. این ویژگی با الگوهای موجود از پشتیبانی Brotli همخوانی دارد. این به‌روزرسانی به توسعه‌دهندگان امکان می‌دهد تا از فشرده‌سازی Zstandard در برنامه‌های خود استفاده کنند، که می‌تواند به کاهش حجم داده‌های منتقل شده و بهبود عملکرد کلی برنامه کمک کند. استفاده از این ویژگی به صورت خودکار انجام می‌شود و نیازی به پیکربندی دستی ندارد. این تغییرات شامل به‌روزرسانی API Surface و اعمال تغییرات لازم در کد برای پشتیبانی از Zstandard می‌باشد. هدف اصلی این است که یکپارچگی و سازگاری با استانداردهای فعلی HTTP حفظ شود. این افزوده توسط Copilot، ابزار هوش مصنوعی، پیشنهاد و توسط stephentoub ادغام شده است. این نشان‌دهنده همکاری بین توسعه‌دهندگان انسانی و هوش مصنوعی در بهبود و توسعه اکوسیستم .NET است.
