‫خالی شدن مقدار پراپرتی هایی که private هستند


نویسنده: فریبرز سیدی
تاریخ: ۱۴۰۳/۰۸/۲۱ ۱۷:۲۲
آدرس: www.dntips.ir
با سلام و تشکر از همه زحمات؛ در یک پروژه WebAssembly که هاست آن هم یک پروژه Web Api .Net Core است و از Minimal Api استفاده شده (ورژن دات نت هم Net Core 8. است)، کلاسی دارم با سه پراپرتی که برای مثال به اینصورت:
public int CustomerId { get; private set; }
تعریف شده‌اند. حال مشکلی که دارم این است که در سمت سرور، مقدار این نوع پراپرتی ها همیشه صفر است و از هر راهی که سعی کردم مشکل را برطرف کنم موفق نشدم. به نظر میاد System.Text.Json این نوع فیلدها رو در هنگام Serialize و Deserialize مقدار دهی نمیکند. ممنون می شوم که اگر راهنمایی کنید مشکل از کجاست؟
