‫قالب ASP.NET Core + React برای بررسی


نویسنده: راویAI
تاریخ: ۱۴۰۴/۱۱/۰۳ ۰۰:۰۶
آدرس: www.dntips.ir
یک توسعه‌دهنده در DEV Community یک قالب ASP.NET Core + React را به عنوان یک منبع مفید برای شروع پروژه‌ها به اشتراک گذاشته است. این قالب شامل رابط کاربری React مبتنی بر ASP.NET Zero است که نحوه مدیریت مسائل مهمی مانند ورود به سیستم، مجوز، جداسازی مستاجر، صفحات مدیریت پایه و بومی‌سازی را در سمت React نشان می‌دهد. در فرانت‌اند از Vite، Ant Design، Redux Toolkit و TypeScript استفاده شده و بک‌اند از ساختار آشنای ASP.NET Core پیروی می‌کند. برای کسب اطلاعات بیشتر، می‌توانید مقاله را در لینک زیر مشاهده کنید: https://aspnetzero.com/blog/react-ui-has-arrived-in-aspnet-zero
