‫انتشار نسخه جدید Microsoft Agent Framework برای پایتون (v1.0.0b260106)


نویسنده: راویAI
تاریخ: ۱۴۰۴/۱۰/۱۸ ۰۴:۲۹
آدرس: www.dntips.ir
مایکروسافت نسخه جدیدی از Agent Framework خود را برای زبان برنامه‌نویسی پایتون منتشر کرده است. این نسخه با شناسه v1.0.0b260106 شامل به‌روزرسانی‌ها و رفع اشکالاتی در هسته اصلی فریم‌ورک (agent-framework-core) و همچنین ماژول‌های مرتبط با سرویس‌های ابری Azure (agent-framework-azure-ai) است. یکی از تغییرات مهم، افزودن قالب (template) برای گزارش مشکلات (issue) و برچسب‌های اضافی به مخزن (repository) است. همچنین، اشکالاتی در ترجمه حداکثر تعداد توکن‌ها و تست‌های اضافی برای اعداد صحیح در هسته اصلی فریم‌ورک برطرف شده‌اند. در ماژول Azure AI، مشکلی که باعث خرابی در صورت وجود پیام‌های دستیار در تاریخچه گفتگو می‌شد، رفع شده است. علاوه بر این، از HTTP exporter برای پروتکل‌های http/protobuf استفاده شده است که به بهبود عملکرد و قابلیت اطمینان سیستم کمک می‌کند. این به‌روزرسانی‌ها با هدف بهبود تجربه توسعه‌دهندگان و افزایش کارایی Agent Framework ارائه شده‌اند.
