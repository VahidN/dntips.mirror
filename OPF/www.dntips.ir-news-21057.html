‫Oluṣọ: سرور هویت متن‌باز برای .NET با پشتیبانی از OAuth 2.0 و OpenID Connect


نویسنده: راویAI
تاریخ: ۱۴۰۴/۱۰/۱۷ ۰۲:۱۲
آدرس: www.dntips.ir
توسعه‌دهنده یک سرور هویت متن‌باز به نام Oluṣọ را برای پلتفرم .NET معرفی کرده است. این سرور با هدف ارائه یک ارائه‌دهنده هویت خودمیزبان (self-hosted) با قابلیت‌های چندمستاجری (multi-tenancy) و رابط کاربری گرافیکی برای مدیریت جریان‌های احراز هویت (auth flows) طراحی شده است. Oluṣọ از ASP.NET Core 8، EF Core و پایگاه داده‌های SQL Server یا SQLite استفاده می‌کند. مجوز این پروژه Fair-source است که امکان استفاده رایگان برای شرکت‌هایی با درآمد کمتر از ۱ میلیون دلار را فراهم می‌کند و سطوح پولی برای سازمان‌های بزرگتر ارائه می‌شود. توسعه‌دهنده به دنبال بازخورد اولیه در مورد تجربه توسعه‌دهنده (DX) است و از کاربران دعوت می‌کند تا پروژه را کلون کرده، نمونه را اجرا کرده و مشکلات احتمالی را گزارش دهند. Oluṣọ از پروتکل‌های OIDC، SAML، LDAP و همچنین ورود اجتماعی (Social Logins) پشتیبانی می‌کند. در آینده، بسته‌های NuGet و تنظیمات Docker همراه با مستندات تعاملی ارائه خواهند شد. اطلاعات بیشتر و کد منبع در GitHub و وب‌سایت رسمی پروژه قابل دسترسی است.
