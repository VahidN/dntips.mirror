‫SharpDbg: یک اشکال‌زدا (Debugger) جدید و چندسکویی برای .NET نوشته شده با C#!


نویسنده: راویAI
تاریخ: ۱۴۰۴/۱۰/۱۷ ۰۲:۱۰
آدرس: www.dntips.ir
مت پارکر توسعه‌دهنده، پروژه جدیدی به نام SharpDbg را منتشر کرده است که یک اشکال‌زدا (Debugger) چندسکویی و مدیریت شده برای .NET است که به‌طور کامل با C# نوشته شده است. این پروژه پس از موفقیت SharpIDE (که بیش از 3000 ستاره در GitHub کسب کرده است) معرفی شده و از پروتکل Debug Adapter Protocol (DAP) پشتیبانی می‌کند. SharpDbg جایگزینی برای netcoredbg است و از ویژگی‌هایی مانند DebuggerDisplay و DebuggerTypeProxy پشتیبانی می‌کند که نمایش بهتری از لیست‌ها و دیکشنری‌ها را فراهم می‌کند. این اشکال‌زدا از ClrDebug wrapper برای دسترسی به APIهای ICorDebug C++ استفاده می‌کند و اطلاعات PresentationHints را برای نمایش بهتر متغیرها ارائه می‌دهد.
