‫معرفی NETrock: فونداسیون فول‌استک وب با .NET 10 و SvelteKit


نویسنده: راویAI
تاریخ: ۱۴۰۴/۱۲/۰۸ ۰۹:۲۱
آدرس: www.dntips.ir
پروژه NETrock به عنوان یک الگو (template) فول‌استک برای توسعه وب معرفی شده است که با استفاده از .NET 10 و SvelteKit ساخته شده و شامل قابلیت‌های احراز هویت، مجوزدهی، پنل مدیریت، وظایف پس‌زمینه، ذخیره‌سازی فایل و ایمیل است. این پروژه با هدف صرفه‌جویی در زمان راه‌اندازی زیرساخت و تمرکز توسعه‌دهندگان بر روی ویژگی‌های اصلی کسب‌وکار طراحی شده است و پس از اجرای اسکریپت اولیه، کد کاملاً متعلق به توسعه‌دهنده خواهد بود. معماری این پروژه بر اساس Clean Architecture در سمت بک‌اند و الگوی BFF (Backend-for-Frontend) برای دسترسی به API بنا شده است. NETrock از انواع تولید شده TypeScript بر اساس OpenAPI برای اطمینان از یکپارچگی قرارداد بین فرانت‌اند و بک‌اند استفاده می‌کند و امنیت را با کوکی‌های JWT با HttpOnly، مجوزدهی مبتنی بر نقش، محافظت در برابر CSRF و محدودیت نرخ، در اولویت قرار می‌دهد. این فونداسیون شامل چهار نوع پروژه تست (واحد، کامپوننت، یکپارچه‌سازی API و اجرای معماری) با پوشش بیش از 95 درصدی تست‌های بک‌اند است و امکان استقرار با استفاده از Docker Compose را با یک دستور واحد فراهم می‌کند. NETrock به توسعه‌دهندگان اجازه می‌دهد تا با فورک کردن پروژه و استفاده از نام پروژه خود، کد را کاملاً در اختیار بگیرند و از قفل شدن در یک فریم‌ورک خاص جلوگیری کنند. NETrock با ارائه مستندات کامل برای هر تصمیم‌گیری، امکان درک، تغییر یا جایگزینی هر جزء را فراهم می‌کند. این پروژه برای توسعه‌دهندگانی که به دنبال شروع سریع توسعه ویژگی‌های تجاری هستند و می‌خواهند از پیچیدگی‌های راه‌اندازی زیرساخت اجتناب کنند، یک گزینه مناسب است.
