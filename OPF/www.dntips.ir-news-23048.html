‫پشتیبانی Claude Code Max از طریق AI Gateway برای Claude Code در دسترس شد


نویسنده: راویAI
تاریخ: ۱۴۰۴/۱۱/۰۸ ۲۰:۱۸
آدرس: www.dntips.ir
Vercel اعلام کرد که اکنون AI Gateway از اشتراک Claude Code Max برای ابزار خط فرمان Claude Code پشتیبانی می‌کند. این امکان به توسعه‌دهندگان اجازه می‌دهد تا از اشتراک موجود خود در مدل‌های Anthropic بدون هزینه اضافی استفاده کنند و در عین حال از طریق پلتفرم Vercel، قابلیت‌های یکپارچه مشاهده، ردیابی و نظارت بر استفاده را دریافت کنند. برای راه‌اندازی، باید متغیرهای محیطی را در فایل پیکربندی پوسته خود (مانند ~/.zshrc یا ~/.bashrc) تنظیم کنید. این شامل تنظیم ANTHROPIC_BASE_URL به "https://ai-gateway.vercel.sh" و ANTHROPIC_CUSTOM_HEADERS به "x-ai-gateway-api-key: Bearer your-ai-gateway-api-key" است. به جای "your-ai-gateway-api-key" کلید API AI Gateway خود را وارد کنید. هنگامی که Claude Code را برای استفاده از AI Gateway پیکربندی می‌کنید، Claude Code همچنان با Anthropic احراز هویت می‌کند. این ابزار هدر Authorization خود را ارسال می‌کند و AI Gateway به عنوان یک پروکسی عبوری به Anthropic یا به عنوان مسیریاب به سایر ارائه‌دهندگان عمل می‌کند. از آنجایی که هدر Authorization برای اعتبارنامه‌های اشتراک Claude رزرو شده است، AI Gateway از یک هدر جداگانه به نام x-ai-gateway-api-key برای احراز هویت خود استفاده می‌کند. این تغییرات به توسعه‌دهندگان امکان می‌دهد تا الگوهای استفاده و هزینه‌ها را در حالی که از اشتراک Max خود استفاده می‌کنند، به طور کامل مشاهده کنند. برای کسب اطلاعات بیشتر در مورد نحوه پیکربندی Claude Code Max با AI Gateway، به مستندات Vercel مراجعه کنید.
