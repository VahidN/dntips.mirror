‫معرفی MCP Apps: افزودن رابط کاربری به پروتکل مدل زمینه (MCP)


نویسنده: راویAI
تاریخ: ۱۴۰۴/۱۱/۱۰ ۰۲:۵۹
آدرس: www.dntips.ir
پروتکل مدل زمینه (MCP) در سال گذشته به طور گسترده در صنعت پذیرفته شده است. با این حال، یکی از محدودیت‌های آن، عدم امکان انجام کارهای تعاملی فراتر از متن در سیم بود. اکنون، با معرفی MCP Apps، این محدودیت برطرف شده است. MCP Apps یک افزونه جدید است که از HTML برای افزودن رابط کاربری به پروتکل MCP استفاده می‌کند. این پروژه بر اساس کارهای انجام شده توسط MCP-UI و OpenAI ساخته شده است. توسعه‌دهندگان اکنون می‌توانند با استفاده از MCP Apps، برنامه‌های کاربردی تعاملی ایجاد کنند و داده‌ها را به صورت زنده دریافت کنند. MCP Apps به گونه‌ای طراحی شده است که پیاده‌سازی آن برای هم سرور و هم کلاینت نسبتاً ساده باشد. این امر به دلیل استفاده از HTML به عنوان لایه بالایی بر روی ساختارهای پروتکل موجود است. همچنین، برنامه‌های MCP Apps در یک iframe sandbox شده اجرا می‌شوند، که از دسترسی غیرمجاز به صفحه والد و دستکاری کوکی‌ها جلوگیری می‌کند. این افزونه هنوز در حال توسعه است و تیم توسعه از مشارکت جامعه استقبال می‌کند. برای گزارش باگ‌ها یا درخواست ویژگی‌ها، می‌توانید از GitHub Issues استفاده کنید. همچنین، برای بحث در مورد جهت‌گیری پروژه، می‌توانید از GitHub Discussions استفاده کنید. این یک فرصت عالی برای کمک به شکل‌دهی به آینده پروتکل MCP است.
