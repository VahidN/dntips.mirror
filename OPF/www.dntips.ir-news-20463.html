ШІШЁШ§ЩҶ ШЁШұЩҶШ§Щ…ЩҮвҖҢЩҶЩҲЫҢШіЫҢ ШіЫҢвҖҢШҙШ§ШұЩҫ ШЁЩҮ Ш№ЩҶЩҲШ§ЩҶ ЫҢЪ©ЫҢ Ш§ШІ Ш§ШұЪ©Ш§ЩҶ Ш§ШөЩ„ЫҢ ЪҶШ§ШұЪҶЩҲШЁ ШҜШ§ШӘвҖҢЩҶШӘ Щ…Ш§ЫҢЪ©ШұЩҲШіШ§ЩҒШӘШҢ ШіШ§Щ„вҖҢЩҮШ§ШіШӘ Ъ©ЩҮ ШЁЩҮ ШҜЩ„ЫҢЩ„ Ш§ЩҶШ№Ш·Ш§ЩҒвҖҢЩҫШ°ЫҢШұЫҢ ЩҲ ЩӮШҜШұШӘ Ш®ЩҲШҜ ШҜШұ ШӘЩҲШіШ№ЩҮ Ш§ЩҶЩҲШ§Ш№ ШЁШұЩҶШ§Щ…ЩҮвҖҢЩҮШ§ШҢ Ш§ШІ Ъ©ЩҶШіЩҲЩ„ ШӘШ§ ЩҲШЁШҢ Щ…ЩҲШұШҜ ШӘЩҲШ¬ЩҮ ШӘЩҲШіШ№ЩҮвҖҢШҜЩҮЩҶШҜЪҜШ§ЩҶ ЩӮШұШ§Шұ ШҜШ§ШұШҜ. ШЁШ§ Ш§ЫҢЩҶ ШӯШ§Щ„ШҢ ЫҢЪ©ЫҢ Ш§ШІ ЪҶШ§Щ„ШҙвҖҢЩҮШ§ЫҢ ЩҮЩ…ЫҢШҙЪҜЫҢ ШЁШұШ§ЫҢ ШЁШұЩҶШ§Щ…ЩҮвҖҢЩҶЩҲЫҢШіШ§ЩҶ ШӘШ§ШІЩҮвҖҢЪ©Ш§ШұШҢ ЩҫЫҢЪҶЫҢШҜЪҜЫҢвҖҢЩҮШ§ЫҢ Ш§ЩҲЩ„ЫҢЩҮ ШҜШұ ШұШ§ЩҮвҖҢШ§ЩҶШҜШ§ШІЫҢ ЩҫШұЩҲЪҳЩҮвҖҢЩҮШ§ ЩҲ ШҜШұЪ© ШіШ§Ш®ШӘШ§ШұЩҮШ§ЫҢ Ш§ЩҲЩ„ЫҢЩҮ Щ…Ш§ЩҶЩҶШҜ ЩҒШ§ЫҢЩ„вҖҢЩҮШ§ЫҢ ЩҫШұЩҲЪҳЩҮ (Project Files) ЩҲ ЩҒШ¶Ш§ЫҢвҖҢЩҶШ§Щ…вҖҢЩҮШ§ (Namespaces) ШЁЩҲШҜЩҮ Ш§ШіШӘ. Щ…Ш§ЫҢЪ©ШұЩҲШіШ§ЩҒШӘ ШҜШұ ЩҶШіШ®ЩҮвҖҢЩҮШ§ЫҢ Ш§Ш®ЫҢШұ ШҜШ§ШӘвҖҢЩҶШӘШҢ ШЁЩҮвҖҢЩҲЫҢЪҳЩҮ ШҜШұ ЩҫЫҢШҙвҖҢЩҶЩ…Ш§ЫҢШҙ ЪҶЩҮШ§ШұЩ… ШҜШ§ШӘвҖҢЩҶШӘ 10 (.NET 10 Preview 4)ШҢ ШұЩҲЫҢЪ©ШұШҜЫҢ ЩҶЩҲЫҢЩҶ ШұШ§ Щ…Ш№ШұЩҒЫҢ Ъ©ШұШҜЩҮ Ш§ШіШӘ Ъ©ЩҮ Ш§Щ…Ъ©Ш§ЩҶ Ш§Ш¬ШұШ§ЫҢ Щ…ШіШӘЩӮЫҢЩ… ЩҒШ§ЫҢЩ„вҖҢЩҮШ§ЫҢ ШіЫҢвҖҢШҙШ§ШұЩҫ ШЁШ§ ШҜШіШӘЩҲШұ dotnet run app.cs ШұШ§ ЩҒШұШ§ЩҮЩ… Щ…ЫҢвҖҢЪ©ЩҶШҜ. Ш§ЫҢЩҶ ЩӮШ§ШЁЩ„ЫҢШӘШҢ ШӘШ¬ШұШЁЩҮ ЫҢШ§ШҜЪҜЫҢШұЫҢ ЩҲ ШӘЩҲШіШ№ЩҮ ШұШ§ ШЁШұШ§ЫҢ ШЁШұЩҶШ§Щ…ЩҮвҖҢЩҶЩҲЫҢШіШ§ЩҶШҢ ШЁЩҮвҖҢЩҲЫҢЪҳЩҮ Щ…ШЁШӘШҜЫҢШ§ЩҶШҢ ШіШ§ШҜЩҮвҖҢШӘШұ Ъ©ШұШҜЩҮ ЩҲ ШЁЩҮ ШўЩҶвҖҢЩҮШ§ Ш§Ш¬Ш§ШІЩҮ Щ…ЫҢвҖҢШҜЩҮШҜ ШЁШҜЩҲЩҶ ЩҶЫҢШ§ШІ ШЁЩҮ ШҜШұЪ© ЩҫЫҢЪҶЫҢШҜЪҜЫҢвҖҢЩҮШ§ЫҢ ШіШ§Ш®ШӘШ§Шұ ЩҫШұЩҲЪҳЩҮШҢ ШЁШұ ЩҶЩҲШҙШӘЩҶ Ъ©ШҜ Щ…ШӘЩ…ШұЪ©ШІ ШҙЩҲЩҶШҜ. ШҜШұ Ш§ЫҢЩҶ Щ…ЩӮШ§Щ„ЩҮШҢ Ш§ЫҢЩҶ ЩҲЫҢЪҳЪҜЫҢ Ш¬ШҜЫҢШҜ ШЁШұШұШіЫҢ ШҙШҜЩҮ ЩҲ ШЁШ§ Ш§ШұШ§ШҰЩҮ Щ…Ш«Ш§Щ„вҖҢЩҮШ§ЫҢ Ш№Щ…Щ„ЫҢ ЩҲ ШӘЩҲШ¶ЫҢШӯШ§ШӘ ШҜЩӮЫҢЩӮШҢ Ш§ШұШІШҙ ШўЩҶ ШЁШұШ§ЫҢ ШӘЩҲШіШ№ЩҮвҖҢШҜЩҮЩҶШҜЪҜШ§ЩҶ ШҜШ§ШӘвҖҢЩҶШӘ ШӘШҙШұЫҢШӯ Щ…ЫҢвҖҢШҙЩҲШҜ.

ШіШ§ШҜЩҮвҖҢШіШ§ШІЫҢ ШӘШ¬ШұШЁЩҮ ЫҢШ§ШҜЪҜЫҢШұЫҢ ШіЫҢвҖҢШҙШ§ШұЩҫ
ЫҢЪ©ЫҢ Ш§ШІ Щ…ЩҲШ§ЩҶШ№ Ш§ШөЩ„ЫҢ ШҜШұ ЫҢШ§ШҜЪҜЫҢШұЫҢ ШіЫҢвҖҢШҙШ§ШұЩҫШҢ Щ…ЩҲШ§Ш¬ЩҮЩҮ ШЁШ§ ШіШ§Ш®ШӘШ§ШұЩҮШ§ЫҢ ЩҫЫҢЪҶЫҢШҜЩҮвҖҢШ§ЫҢ Щ…Ш§ЩҶЩҶШҜ ЩҒШ¶Ш§ЫҢвҖҢЩҶШ§Щ…вҖҢЩҮШ§ (Namespaces)ШҢ Ъ©Щ„Ш§ШівҖҢЩҮШ§ (Classes)ШҢ ЩҲ ЩҒШ§ЫҢЩ„вҖҢЩҮШ§ЫҢ ЩҫШұЩҲЪҳЩҮ Щ…ШЁШӘЩҶЫҢ ШЁШұ XML Ш§ШіШӘ Ъ©ЩҮ ШЁШұШ§ЫҢ ЫҢЪ© ШЁШұЩҶШ§Щ…ЩҮвҖҢЩҶЩҲЫҢШі ШӘШ§ШІЩҮвҖҢЪ©Ш§Шұ Щ…ЫҢвҖҢШӘЩҲШ§ЩҶШҜ ШҜЩ„ЩҮШұЩҮвҖҢШўЩҲШұ ШЁШ§ШҙШҜ. ШЁЩҮ Ш·ЩҲШұ ШіЩҶШӘЫҢШҢ Ш§ЫҢШ¬Ш§ШҜ ЫҢЪ© ШЁШұЩҶШ§Щ…ЩҮ ШіШ§ШҜЩҮ Щ…Ш§ЩҶЩҶШҜ ЪҶШ§Щҫ В«Hello, WorldВ» ШҜШұ Ъ©ЩҶШіЩҲЩ„ШҢ ЩҶЫҢШ§ШІЩ…ЩҶШҜ Ш§ЫҢШ¬Ш§ШҜ ЫҢЪ© ЩҫШұЩҲЪҳЩҮ Ъ©ЩҶШіЩҲЩ„ ШЁШ§ ШҜШіШӘЩҲШұ dotnet new console ШЁЩҲШҜ. Ш§ЫҢЩҶ ЩҒШұШўЫҢЩҶШҜШҢ ЩҒШ§ЫҢЩ„вҖҢЩҮШ§ЫҢЫҢ Щ…Ш§ЩҶЩҶШҜ Program.cs ЩҲ ЩҒШ§ЫҢЩ„ ЩҫШұЩҲЪҳЩҮ ШЁШ§ ЩҫШіЩҲЩҶШҜ csproj ШӘЩҲЩ„ЫҢШҜ Щ…ЫҢвҖҢЪ©ШұШҜ Ъ©ЩҮ ШӯШ§ЩҲЫҢ ШӘЩҶШёЫҢЩ…Ш§ШӘ ЩҫЫҢЪҶЫҢШҜЩҮвҖҢШ§ЫҢ ШЁЩҲШҜ. ШЁШұШ§ЫҢ Щ…Ш«Ш§Щ„ШҢ Ъ©ШҜ ШІЫҢШұ ЩҶЩ…ЩҲЩҶЩҮвҖҢШ§ЫҢ Ш§ШІ ЫҢЪ© ШЁШұЩҶШ§Щ…ЩҮ Ъ©ЩҶШіЩҲЩ„ ШіЩҶШӘЫҢ Ш§ШіШӘ:
using System;

namespace MyFirstApp
{
    class Program
    {
        static void Main(string[] args)
        {
            Console.WriteLine("Hello, World!");
        }
    }
}
Ш§ЫҢЩҶ Ъ©ШҜШҢ Ш№Щ„Ш§ЩҲЩҮ ШЁШұ Ш®Ш· ШіШ§ШҜЩҮ Console.WriteLine("Hello, World!")ШҢ ШҙШ§Щ…Щ„ Щ…ЩҒШ§ЩҮЫҢЩ…ЫҢ Щ…Ш§ЩҶЩҶШҜ ЩҒШ¶Ш§ЫҢвҖҢЩҶШ§Щ…ШҢ Ъ©Щ„Ш§ШіШҢ Щ…ШӘШҜ Ш§ШіШӘШ§ШӘЫҢЪ© (Static Method)ШҢ ЩҲ ШўШұЪҜЩҲЩ…Ш§ЩҶвҖҢЩҮШ§ЫҢ ЩҲШұЩҲШҜЫҢ Ш§ШіШӘ Ъ©ЩҮ ШЁШұШ§ЫҢ ЫҢЪ© Щ…ШЁШӘШҜЫҢ Щ…Щ…Ъ©ЩҶ Ш§ШіШӘ ЪҜЫҢШ¬вҖҢЪ©ЩҶЩҶШҜЩҮ ШЁШ§ШҙШҜ. Ш№Щ„Ш§ЩҲЩҮ ШЁШұ Ш§ЫҢЩҶШҢ ЩҒШ§ЫҢЩ„ ЩҫШұЩҲЪҳЩҮ XML ЩҲ ЩҫЩҲШҙЩҮвҖҢЩҮШ§ЫҢ Ш§Ш¶Ш§ЩҒЫҢ Щ…Ш§ЩҶЩҶШҜ obj ЩҲ bin ШЁЩҮ ЩҫЫҢЪҶЫҢШҜЪҜЫҢ Щ…ШӯЫҢШ· ШӘЩҲШіШ№ЩҮ Щ…ЫҢвҖҢШ§ЩҒШІШ§ЫҢЩҶШҜ.
ШЁШұШ§ЫҢ ШұЩҒШ№ Ш§ЫҢЩҶ Щ…ШҙЪ©Щ„ШҢ Щ…Ш§ЫҢЪ©ШұЩҲШіШ§ЩҒШӘ ШҜШұ ШҜШ§ШӘвҖҢЩҶШӘ 6 ШӘШәЫҢЫҢШұШ§ШӘЫҢ ШұШ§ ШўШәШ§ШІ Ъ©ШұШҜ ЩҲ ШҜШұ ШҜШ§ШӘвҖҢЩҶШӘ 10 ЩҫЫҢШҙвҖҢЩҶЩ…Ш§ЫҢШҙ 4ШҢ Ш§ЫҢЩҶ Ш§Щ…Ъ©Ш§ЩҶ ШұШ§ ЩҒШұШ§ЩҮЩ… Ъ©ШұШҜ Ъ©ЩҮ ШӘЩҲШіШ№ЩҮвҖҢШҜЩҮЩҶШҜЪҜШ§ЩҶ ШЁШӘЩҲШ§ЩҶЩҶШҜ ЫҢЪ© ЩҒШ§ЫҢЩ„ ШіЫҢвҖҢШҙШ§ШұЩҫ ШұШ§ ШЁЩҮ Ш·ЩҲШұ Щ…ШіШӘЩӮЫҢЩ… Ш§Ш¬ШұШ§ Ъ©ЩҶЩҶШҜ. ШЁЩҮ Ш№ЩҶЩҲШ§ЩҶ Щ…Ш«Ш§Щ„ШҢ ЫҢЪ© ЩҒШ§ЫҢЩ„ ШіШ§ШҜЩҮ ШЁЩҮ ЩҶШ§Щ… hello.cs ШЁШ§ Щ…ШӯШӘЩҲШ§ЫҢ ШІЫҢШұ Ъ©Ш§ЩҒЫҢ Ш§ШіШӘ:
Console.WriteLine("Hello, C#!");
ШЁШ§ Ш§Ш¬ШұШ§ЫҢ ШҜШіШӘЩҲШұ dotnet run hello.csШҢ Ш§ЫҢЩҶ ЩҒШ§ЫҢЩ„ Щ…ШіШӘЩӮЫҢЩ…Ш§ЩӢ Ш§Ш¬ШұШ§ ШҙШҜЩҮ ЩҲ Ш®ШұЩҲШ¬ЫҢ Hello, C#! ШұШ§ ШҜШұ Ъ©ЩҶШіЩҲЩ„ ЩҶЩ…Ш§ЫҢШҙ Щ…ЫҢвҖҢШҜЩҮШҜ. Ш§ЫҢЩҶ ШұЩҲЫҢЪ©ШұШҜШҢ Ъ©ЩҮ ШЁЩҮ В«Ш§Ш¬ШұШ§ЫҢ ЩҒШ§ЫҢЩ„ Щ…ШіШӘЩӮЫҢЩ…В» Щ…Ш№ШұЩҲЩҒ Ш§ШіШӘШҢ ЩҶЫҢШ§ШІ ШЁЩҮ Ш§ЫҢШ¬Ш§ШҜ ЩҫШұЩҲЪҳЩҮ ЫҢШ§ ШҜШұЪ© Щ…ЩҒШ§ЩҮЫҢЩ… ЩҫЫҢШҙШұЩҒШӘЩҮ ШұШ§ Ш§ШІ ШЁЫҢЩҶ Щ…ЫҢвҖҢШЁШұШҜ ЩҲ ШЁЩҮ ШӘЩҲШіШ№ЩҮвҖҢШҜЩҮЩҶШҜЪҜШ§ЩҶ Ш§Ш¬Ш§ШІЩҮ Щ…ЫҢвҖҢШҜЩҮШҜ ШЁШұ ЫҢШ§ШҜЪҜЫҢШұЫҢ ШІШЁШ§ЩҶ ШӘЩ…ШұЪ©ШІ Ъ©ЩҶЩҶШҜ.

Ш§ЩҒШІЩҲШҜЩҶ ЩӮШ§ШЁЩ„ЫҢШӘвҖҢЩҮШ§ЫҢ ЩҫЫҢШҙШұЩҒШӘЩҮ ШЁЩҮ ЩҒШ§ЫҢЩ„вҖҢЩҮШ§ЫҢ ШӘЪ©вҖҢЩҒШ§ЫҢЩ„ЫҢ
ЫҢЪ©ЫҢ Ш§ШІ ЩҶЩӮШ§Ш· ЩӮЩҲШӘ Ш§ЫҢЩҶ ЩҲЫҢЪҳЪҜЫҢ Ш¬ШҜЫҢШҜШҢ Ш§Щ…Ъ©Ш§ЩҶ ЪҜШіШӘШұШҙ ШЁШұЩҶШ§Щ…ЩҮвҖҢЩҮШ§ЫҢ ШӘЪ©вҖҢЩҒШ§ЫҢЩ„ЫҢ ШЁШұШ§ЫҢ Ш§ЩҶШ¬Ш§Щ… Ъ©Ш§ШұЩҮШ§ЫҢ ЩҫЫҢЪҶЫҢШҜЩҮвҖҢШӘШұ Ш§ШіШӘ. ШЁШұШ§ЫҢ Щ…Ш«Ш§Щ„ШҢ ЩҒШұШ¶ Ъ©ЩҶЫҢШҜ Щ…ЫҢвҖҢШ®ЩҲШ§ЩҮЫҢЩ… ШЁШұЩҶШ§Щ…ЩҮвҖҢШ§ЫҢ ШЁЩҶЩҲЫҢШіЫҢЩ… Ъ©ЩҮ Щ…ШҜШӘ ШІЩ…Ш§ЩҶ ЪҜШ°ШҙШӘЩҮ Ш§ШІ ЫҢЪ© ШұЩҲЫҢШҜШ§ШҜ Ш®Ш§Шө ШұШ§ Щ…ШӯШ§ШіШЁЩҮ Ъ©ЩҶШҜ. Ъ©ШҜ ШІЫҢШұ ЩҶЩ…ЩҲЩҶЩҮвҖҢШ§ЫҢ Ш§ШІ Ш§ЫҢЩҶ Ъ©Ш§ШұШЁШұШҜ Ш§ШіШӘ:

TimeSpan timeSinceBuild = DateTimeOffset.Now - buildStart;
Console.WriteLine($"It has been {timeSinceBuild.Days} days since Build started.");
Ш§ЫҢЩҶ Ъ©ШҜ ШЁШ§ Щ…ШӯШ§ШіШЁЩҮ ШӘЩҒШ§ЩҲШӘ ШІЩ…Ш§ЩҶЫҢ ШЁЫҢЩҶ ШӘШ§ШұЫҢШ® Ъ©ЩҶЩҲЩҶЫҢ ЩҲ ШӘШ§ШұЫҢШ® ШҙШұЩҲШ№ ЫҢЪ© ШұЩҲЫҢШҜШ§ШҜ (Щ…Ш«Щ„Ш§ЩӢ Ъ©ЩҶЩҒШұШ§ЩҶШі ШЁЫҢЩ„ШҜ 2025)ШҢ ШӘШ№ШҜШ§ШҜ ШұЩҲШІЩҮШ§ЫҢ ЪҜШ°ШҙШӘЩҮ ШұШ§ ЩҶЩ…Ш§ЫҢШҙ Щ…ЫҢвҖҢШҜЩҮШҜ. ШЁШ§ Ш§Ш¬ШұШ§ЫҢ ШҜШіШӘЩҲШұ dotnet run app.csШҢ Ш®ШұЩҲШ¬ЫҢ ШЁЩҮ ШөЩҲШұШӘ ЫҢЪ© Ш№ШҜШҜ Ш®Ш§Щ… (Щ…Ш§ЩҶЩҶШҜ В«2 daysВ») ЩҶЩ…Ш§ЫҢШҙ ШҜШ§ШҜЩҮ Щ…ЫҢвҖҢШҙЩҲШҜ Ъ©ЩҮ Щ…Щ…Ъ©ЩҶ Ш§ШіШӘ ШЁШұШ§ЫҢ Ъ©Ш§ШұШЁШұ ЩҶЩҮШ§ЫҢЫҢ ЪҶЩҶШҜШ§ЩҶ Ш®ЩҲШ§ЩҶШ§ ЩҶШЁШ§ШҙШҜ.
ШЁШұШ§ЫҢ ШЁЩҮШЁЩҲШҜ Ш®ШұЩҲШ¬ЫҢШҢ Щ…ЫҢвҖҢШӘЩҲШ§ЩҶ Ш§ШІ ШЁШіШӘЩҮвҖҢЩҮШ§ЫҢ Ш®Ш§ШұШ¬ЫҢ Щ…Ш§ЩҶЩҶШҜ Humanizer Ш§ШіШӘЩҒШ§ШҜЩҮ Ъ©ШұШҜ. Ш§ЫҢЩҶ ШЁШіШӘЩҮ Ш§Щ…Ъ©Ш§ЩҶ ШӘШЁШҜЫҢЩ„ Ш®ШұЩҲШ¬ЫҢвҖҢЩҮШ§ЫҢ Ш®Ш§Щ… ШЁЩҮ Ш№ШЁШ§ШұШ§ШӘ Ш®ЩҲШ§ЩҶШ§ШӘШұ ШұШ§ ЩҒШұШ§ЩҮЩ… Щ…ЫҢвҖҢЪ©ЩҶШҜ. ШЁШұШ§ЫҢ Ш§ШіШӘЩҒШ§ШҜЩҮ Ш§ШІ Ш§ЫҢЩҶ ШЁШіШӘЩҮ ШҜШұ ЫҢЪ© ЩҒШ§ЫҢЩ„ ШӘЪ©вҖҢЩҒШ§ЫҢЩ„ЫҢШҢ Ш§ШІ ШҜШіШӘЩҲШұШ§ШӘ ЩҶШ§ШҜЫҢШҜЩҮвҖҢЪҜЫҢШұЩҶШҜЩҮ (Ignore Directives) ШЁШ§ ЩҫЫҢШҙЩҲЩҶШҜ #: Ш§ШіШӘЩҒШ§ШҜЩҮ Щ…ЫҢвҖҢШҙЩҲШҜ:
#: package Humanizer >= 2.0
using Humanizer;

DateTimeOffset buildStart = DateTimeOffset.Parse("2025-05-18");
TimeSpan timeSinceBuild = DateTimeOffset.Now - buildStart;
Console.WriteLine($"It has been {timeSinceBuild.Humanize()} since Build started.");
ШЁШ§ Ш§Ш¬ШұШ§ЫҢ Ш§ЫҢЩҶ Ъ©ШҜШҢ Ш®ШұЩҲШ¬ЫҢ ШЁЩҮ ШөЩҲШұШӘ Ш®ЩҲШ§ЩҶШ§ШӘШұЫҢ Щ…Ш§ЩҶЩҶШҜ В«2 daysВ» ЩҶЩ…Ш§ЫҢШҙ ШҜШ§ШҜЩҮ Щ…ЫҢвҖҢШҙЩҲШҜ. Ш§ЫҢЩҶ Щ…Ш«Ш§Щ„ ЩҶШҙШ§ЩҶ Щ…ЫҢвҖҢШҜЩҮШҜ Ъ©ЩҮ ШӘЩҲШіШ№ЩҮвҖҢШҜЩҮЩҶШҜЪҜШ§ЩҶ Щ…ЫҢвҖҢШӘЩҲШ§ЩҶЩҶШҜ ШЁШҜЩҲЩҶ ЩҶЫҢШ§ШІ ШЁЩҮ ЩҒШ§ЫҢЩ„ ЩҫШұЩҲЪҳЩҮШҢ ШЁШіШӘЩҮвҖҢЩҮШ§ЫҢ Ш®Ш§ШұШ¬ЫҢ ШұШ§ ШЁЩҮ ШЁШұЩҶШ§Щ…ЩҮ Ш®ЩҲШҜ Ш§Ш¶Ш§ЩҒЩҮ Ъ©ШұШҜЩҮ ЩҲ Ш§ШІ ЩӮШ§ШЁЩ„ЫҢШӘвҖҢЩҮШ§ЫҢ ЩҫЫҢШҙШұЩҒШӘЩҮвҖҢШӘШұ Ш§ШіШӘЩҒШ§ШҜЩҮ Ъ©ЩҶЩҶШҜ.

ЩҫШҙШӘЫҢШЁШ§ЩҶЫҢ Ш§ШІ Ш§Ш¬ШұШ§ЫҢ Щ…ШіШӘЩӮЫҢЩ… ШҜШұ Щ…ШӯЫҢШ·вҖҢЩҮШ§ЫҢ Щ„ЫҢЩҶЩҲЪ©Ші
ЫҢЪ©ЫҢ Ш§ШІ ЩҲЫҢЪҳЪҜЫҢвҖҢЩҮШ§ЫҢ Ш¬Ш°Ш§ШЁ Ш§ЫҢЩҶ ШұЩҲЫҢЪ©ШұШҜ Ш¬ШҜЫҢШҜШҢ ЩҫШҙШӘЫҢШЁШ§ЩҶЫҢ Ш§ШІ Ш§Ш¬ШұШ§ЫҢ Щ…ШіШӘЩӮЫҢЩ… ЩҒШ§ЫҢЩ„вҖҢЩҮШ§ЫҢ ШіЫҢвҖҢШҙШ§ШұЩҫ ШҜШұ Щ…ШӯЫҢШ·вҖҢЩҮШ§ЫҢ Щ„ЫҢЩҶЩҲЪ©Ші ШЁШ§ Ш§ШіШӘЩҒШ§ШҜЩҮ Ш§ШІ ШҙЩҗШЁЩҺЩҶЪҜ (Shebang) Ш§ШіШӘ. ШҙЩҗШЁЩҺЩҶЪҜШҢ Ъ©ЩҮ ШЁШ§ #! ШҙШұЩҲШ№ Щ…ЫҢвҖҢШҙЩҲШҜШҢ ШЁЩҮ ШіЫҢШіШӘЩ… Ш№Ш§Щ…Щ„ Щ„ЫҢЩҶЩҲЪ©Ші Ш§Ш¬Ш§ШІЩҮ Щ…ЫҢвҖҢШҜЩҮШҜ ЩҒШ§ЫҢЩ„ ШұШ§ ШЁЩҮ Ш№ЩҶЩҲШ§ЩҶ ЫҢЪ© Ш§ШіЪ©ШұЫҢЩҫШӘ ЩӮШ§ШЁЩ„ Ш§Ш¬ШұШ§ ШӘШҙШ®ЫҢШө ШҜЩҮШҜ. ШЁШұШ§ЫҢ Щ…Ш«Ш§Щ„:
#!/usr/bin/env dotnet run --
Console.WriteLine("Hello from Linux!");
ШЁШ§ ШӘЩҶШёЫҢЩ… ЩҒШ§ЫҢЩ„ ШЁЩҮ ШӯШ§Щ„ШӘ ЩӮШ§ШЁЩ„ Ш§Ш¬ШұШ§ ШЁШ§ ШҜШіШӘЩҲШұ chmod +x hello.csШҢ Щ…ЫҢвҖҢШӘЩҲШ§ЩҶ ШўЩҶ ШұШ§ Щ…ШіШӘЩӮЫҢЩ…Ш§ЩӢ ШЁШ§ ./hello.cs Ш§Ш¬ШұШ§ Ъ©ШұШҜ. Ш§ЫҢЩҶ ЩӮШ§ШЁЩ„ЫҢШӘШҢ ШіЫҢвҖҢШҙШ§ШұЩҫ ШұШ§ ШЁЩҮ ШІШЁШ§ЩҶЫҢ Щ…ШҙШ§ШЁЩҮ ЩҫШ§ЫҢШӘЩҲЩҶ ЫҢШ§ ЩҶЩҲШҜ.Ш¬ЫҢвҖҢШ§Ші (Node.js) ШӘШЁШҜЫҢЩ„ Щ…ЫҢвҖҢЪ©ЩҶШҜ Ъ©ЩҮ ШҜШұ ШўЩҶвҖҢЩҮШ§ Ш§Ш¬ШұШ§ЫҢ ЩҒШ§ЫҢЩ„вҖҢЩҮШ§ЫҢ Ш§ШіЪ©ШұЫҢЩҫШӘЫҢ ШЁЩҮ ШіШ§ШҜЪҜЫҢ Ш§ЩҶШ¬Ш§Щ… Щ…ЫҢвҖҢШҙЩҲШҜ. Ш§ЫҢЩҶ ЩҲЫҢЪҳЪҜЫҢ ШЁЩҮвҖҢЩҲЫҢЪҳЩҮ ШЁШұШ§ЫҢ ШӘЩҲШіШ№ЩҮвҖҢШҜЩҮЩҶШҜЪҜШ§ЩҶЫҢ Ъ©ЩҮ ШҜШұ Щ…ШӯЫҢШ·вҖҢЩҮШ§ЫҢ Щ„ЫҢЩҶЩҲЪ©Ші Щ…Ш§ЩҶЩҶШҜ WSL (Windows Subsystem for Linux) Ъ©Ш§Шұ Щ…ЫҢвҖҢЪ©ЩҶЩҶШҜШҢ Ш§ШұШІШҙЩ…ЩҶШҜ Ш§ШіШӘ.

ШӘЩҲШіШ№ЩҮ ШЁШұЩҶШ§Щ…ЩҮвҖҢЩҮШ§ЫҢ ЩҲШЁ ШЁШҜЩҲЩҶ ЩҫШұЩҲЪҳЩҮ
Ш№Щ„Ш§ЩҲЩҮ ШЁШұ ШЁШұЩҶШ§Щ…ЩҮвҖҢЩҮШ§ЫҢ Ъ©ЩҶШіЩҲЩ„ШҢ Ш§ЫҢЩҶ ЩҲЫҢЪҳЪҜЫҢ Ш§Щ…Ъ©Ш§ЩҶ Ш§ЫҢШ¬Ш§ШҜ ШЁШұЩҶШ§Щ…ЩҮвҖҢЩҮШ§ЫҢ ЩҲШЁ ШұШ§ ЩҶЫҢШІ ЩҒШұШ§ЩҮЩ… Щ…ЫҢвҖҢЪ©ЩҶШҜ. ШЁЩҮ Ш№ЩҶЩҲШ§ЩҶ Щ…Ш«Ш§Щ„ШҢ ЫҢЪ© ЩҲШЁ API ШіШ§ШҜЩҮ Щ…ЫҢвҖҢШӘЩҲШ§ЩҶШҜ ШЁЩҮ ШөЩҲШұШӘ ШІЫҢШұ ШҜШұ ЫҢЪ© ЩҒШ§ЫҢЩ„ ШӘЪ©вҖҢЩҒШ§ЫҢЩ„ЫҢ ЩҶЩҲШҙШӘЩҮ ШҙЩҲШҜ:
#: sdk Microsoft.AspNetCore.App
#: package Swashbuckle.AspNetCore >= 6.0

using Microsoft.AspNetCore.Builder;
using Microsoft.Extensions.Hosting;

var builder = WebApplication.CreateBuilder(args);
builder.Services.AddEndpointsApiExplorer();
builder.Services.AddSwaggerGen();

var app = builder.Build();
app.UseSwagger();
app.UseSwaggerUI();
app.MapGet("/", () => "Hello, Web API!");
app.Run();
ШЁШ§ Ш§Ш¬ШұШ§ЫҢ ШҜШіШӘЩҲШұ dotnet run webapi.csШҢ ЫҢЪ© ЩҲШЁ API ШұЩҲЫҢ localhost:5000 ШұШ§ЩҮвҖҢШ§ЩҶШҜШ§ШІЫҢ Щ…ЫҢвҖҢШҙЩҲШҜ Ъ©ЩҮ Щ…ЫҢвҖҢШӘЩҲШ§ЩҶШҜ ШЁШ§ Ш§ШЁШІШ§ШұЩҮШ§ЫҢЫҢ Щ…Ш§ЩҶЩҶШҜ Swagger ШӘШ№Ш§Щ…Щ„ ШҜШ§ШҙШӘЩҮ ШЁШ§ШҙШҜ. Ш§ЫҢЩҶ ШұЩҲЫҢЪ©ШұШҜ ШЁЩҮ ШӘЩҲШіШ№ЩҮвҖҢШҜЩҮЩҶШҜЪҜШ§ЩҶ Ш§Щ…Ъ©Ш§ЩҶ Щ…ЫҢвҖҢШҜЩҮШҜ ШЁШҜЩҲЩҶ ЩҶЫҢШ§ШІ ШЁЩҮ ШіШ§Ш®ШӘШ§Шұ ЩҫШұЩҲЪҳЩҮ ЩҫЫҢЪҶЫҢШҜЩҮШҢ ШЁШұЩҶШ§Щ…ЩҮвҖҢЩҮШ§ЫҢ ЩҲШЁ ШұШ§ ШўШІЩ…Ш§ЫҢШҙ Ъ©ЩҶЩҶШҜ.

Ш§ЩҶШӘЩӮШ§Щ„ ШЁЩҮ ЩҫШұЩҲЪҳЩҮвҖҢЩҮШ§ЫҢ ШіЩҶШӘЫҢ
Ш§ЪҜШұЪҶЩҮ Ш§Ш¬ШұШ§ЫҢ Щ…ШіШӘЩӮЫҢЩ… ЩҒШ§ЫҢЩ„вҖҢЩҮШ§ ШЁШұШ§ЫҢ ЫҢШ§ШҜЪҜЫҢШұЫҢ ЩҲ ШӘЩҲШіШ№ЩҮ ШіШұЫҢШ№ Щ…ЩҶШ§ШіШЁ Ш§ШіШӘШҢ ЩҫШұЩҲЪҳЩҮвҖҢЩҮШ§ЫҢ ЩҫЫҢЪҶЫҢШҜЩҮвҖҢШӘШұ Щ…Ш№Щ…ЩҲЩ„Ш§ЩӢ ШЁЩҮ ШіШ§Ш®ШӘШ§Шұ ЩҫШұЩҲЪҳЩҮ ЩҶЫҢШ§ШІ ШҜШ§ШұЩҶШҜ. ШҜШ§ШӘвҖҢЩҶШӘ 10 Ш§ЫҢЩҶ Ш§Щ…Ъ©Ш§ЩҶ ШұШ§ ЩҒШұШ§ЩҮЩ… Ъ©ШұШҜЩҮ Ъ©ЩҮ ЩҒШ§ЫҢЩ„вҖҢЩҮШ§ЫҢ ШӘЪ©вҖҢЩҒШ§ЫҢЩ„ЫҢ ШЁЩҮ ЩҫШұЩҲЪҳЩҮвҖҢЩҮШ§ЫҢ Ш§ШіШӘШ§ЩҶШҜШ§ШұШҜ ШӘШЁШҜЫҢЩ„ ШҙЩҲЩҶШҜ. ШЁШ§ Ш§Ш¬ШұШ§ЫҢ ШҜШіШӘЩҲШұ dotnet project convert hello.csШҢ ЩҒШ§ЫҢЩ„ hello.cs ШЁЩҮ ЫҢЪ© ЩҫШұЩҲЪҳЩҮ Ъ©Ш§Щ…Щ„ ШӘШЁШҜЫҢЩ„ Щ…ЫҢвҖҢШҙЩҲШҜ Ъ©ЩҮ ШҙШ§Щ…Щ„ ЩҒШ§ЫҢЩ„ csproj ЩҲ ШіШ§Ш®ШӘШ§Шұ Ш§ШіШӘШ§ЩҶШҜШ§ШұШҜ ШҜШ§ШӘвҖҢЩҶШӘ Ш§ШіШӘ. Ш§ЫҢЩҶ ЩҒШұШўЫҢЩҶШҜШҢ Ш§ЩҶШӘЩӮШ§Щ„ Ш§ШІ ШӯШ§Щ„ШӘ ШіШ§ШҜЩҮ ШЁЩҮ ЩҫШұЩҲЪҳЩҮвҖҢЩҮШ§ЫҢ ЩҫЫҢЪҶЫҢШҜЩҮвҖҢШӘШұ ШұШ§ ШЁШҜЩҲЩҶ ЩҶЫҢШ§ШІ ШЁЩҮ ШЁШ§ШІЩҶЩҲЫҢШіЫҢ Ъ©ШҜ ШӘШіЩҮЫҢЩ„ Щ…ЫҢвҖҢЪ©ЩҶШҜ.

ЩҶШӘЫҢШ¬ЩҮвҖҢЪҜЫҢШұЫҢ
Ш§Ш¬ШұШ§ЫҢ Щ…ШіШӘЩӮЫҢЩ… ЩҒШ§ЫҢЩ„вҖҢЩҮШ§ЫҢ ШіЫҢвҖҢШҙШ§ШұЩҫ ШЁШ§ ШҜШіШӘЩҲШұ dotnet run app.cs ШӘШӯЩҲЩ„ЫҢ ШҜШұ ШӘШ¬ШұШЁЩҮ ШӘЩҲШіШ№ЩҮ ШЁШ§ ШҜШ§ШӘвҖҢЩҶШӘ Ш§ЫҢШ¬Ш§ШҜ Ъ©ШұШҜЩҮ Ш§ШіШӘ. Ш§ЫҢЩҶ ЩҲЫҢЪҳЪҜЫҢ ШЁШ§ ШӯШ°ЩҒ Щ…ЩҲШ§ЩҶШ№ Ш§ЩҲЩ„ЫҢЩҮ Щ…Ш§ЩҶЩҶШҜ ЩҒШ§ЫҢЩ„вҖҢЩҮШ§ЫҢ ЩҫШұЩҲЪҳЩҮ ЩҲ ШіШ§Ш®ШӘШ§ШұЩҮШ§ЫҢ ЩҫЫҢЪҶЫҢШҜЩҮШҢ ЫҢШ§ШҜЪҜЫҢШұЫҢ ШіЫҢвҖҢШҙШ§ШұЩҫ ШұШ§ ШЁШұШ§ЫҢ Щ…ШЁШӘШҜЫҢШ§ЩҶ ШўШіШ§ЩҶвҖҢШӘШұ Щ…ЫҢвҖҢЪ©ЩҶШҜ ЩҲ ШЁЩҮ ШӘЩҲШіШ№ЩҮвҖҢШҜЩҮЩҶШҜЪҜШ§ЩҶ ШӯШұЩҒЩҮвҖҢШ§ЫҢ Ш§Щ…Ъ©Ш§ЩҶ ШўШІЩ…Ш§ЫҢШҙ ШіШұЫҢШ№ Ш§ЫҢШҜЩҮвҖҢЩҮШ§ ШұШ§ Щ…ЫҢвҖҢШҜЩҮШҜ. Ш§ШІ Ш§Ш¬ШұШ§ЫҢ ШЁШұЩҶШ§Щ…ЩҮвҖҢЩҮШ§ЫҢ Ъ©ЩҶШіЩҲЩ„ ШіШ§ШҜЩҮ ШӘШ§ ШӘЩҲШіШ№ЩҮ ЩҲШЁ API ЩҲ ЩҫШҙШӘЫҢШЁШ§ЩҶЫҢ Ш§ШІ ШҙЩҗШЁЩҺЩҶЪҜ ШҜШұ Щ„ЫҢЩҶЩҲЪ©ШіШҢ Ш§ЫҢЩҶ ЩӮШ§ШЁЩ„ЫҢШӘ Ш§ЩҶШ№Ш·Ш§ЩҒвҖҢЩҫШ°ЫҢШұЫҢ ШЁЫҢвҖҢШіШ§ШЁЩӮЩҮвҖҢШ§ЫҢ ШЁЩҮ ШіЫҢвҖҢШҙШ§ШұЩҫ Ш§ЩҒШІЩҲШҜЩҮ Ш§ШіШӘ. ШЁШ§ Ш§ЫҢЩҶ ШӯШ§Щ„ШҢ ШЁШұШ§ЫҢ ЩҫШұЩҲЪҳЩҮвҖҢЩҮШ§ЫҢ ШЁШІШұЪҜвҖҢШӘШұШҢ Ш§Щ…Ъ©Ш§ЩҶ ШӘШЁШҜЫҢЩ„ ШЁЩҮ ЩҫШұЩҲЪҳЩҮвҖҢЩҮШ§ЫҢ Ш§ШіШӘШ§ЩҶШҜШ§ШұШҜ ЩҮЩ…ЪҶЩҶШ§ЩҶ ШӯЩҒШё ШҙШҜЩҮ Ш§ШіШӘ. ШӘЩҲШіШ№ЩҮвҖҢШҜЩҮЩҶШҜЪҜШ§ЩҶ Щ…ЫҢвҖҢШӘЩҲШ§ЩҶЩҶШҜ ШЁШ§ ШҜШ§ЩҶЩ„ЩҲШҜ ЩҫЫҢШҙвҖҢЩҶЩ…Ш§ЫҢШҙ ЪҶЩҮШ§ШұЩ… ШҜШ§ШӘвҖҢЩҶШӘ 10 ЩҲ Ш§ЩҒШІЩҲЩҶЩҮ ШіЫҢвҖҢШҙШ§ШұЩҫ ШҜШұ ЩҲЫҢЪҳЩҲШ§Щ„ Ш§ШіШӘЩҲШҜЫҢЩҲ Ъ©ШҜ (Visual Studio Code) Ш§ЫҢЩҶ ЩҲЫҢЪҳЪҜЫҢ ШұШ§ ШўШІЩ…Ш§ЫҢШҙ Ъ©ЩҶЩҶШҜ ЩҲ ШЁШ§ШІШ®ЩҲШұШҜ Ш®ЩҲШҜ ШұШ§ ШЁШұШ§ЫҢ ШЁЩҮШЁЩҲШҜ ЩҶШіШ®ЩҮвҖҢЩҮШ§ЫҢ ШўШӘЫҢ Ш§ШұШ§ШҰЩҮ ШҜЩҮЩҶШҜ.


Щ…ШҙШ§ЩҮШҜЩҮ Щ…Ш·Щ„ШЁ Ш§ШөЩ„ЫҢ
Щ…Ш·Ш§Щ„ШЁ Щ…ШҙШ§ШЁЩҮ
    Ш§ШҙШӘШұШ§Ъ©вҖҢЩҮШ§Щ…Ш№ШұЩҒЫҢ В«ШҜШіШӘЩҲШұШ§ШӘ ЩҶШ§ШҜЫҢШҜЩҮ ЪҜШұЩҒШӘЩҮ ШҙШҜЩҮВ» ШіЫҢвҖҢШҙШ§ШұЩҫ (C# Ignored Directives) ШҜШұ ШҜШ§ШӘвҖҢЩҶШӘ 10Ш§ШҙШӘШұШ§Ъ©вҖҢЩҮШ§ШЁШұШұШіЫҢ ЩҲЫҢЪҳЪҜЫҢвҖҢЩҮШ§ЫҢ dotnet run app.cs ШҜШұ .NET 10Ш§ШҙШӘШұШ§Ъ©вҖҢЩҮШ§Ш§Ш¬ШұШ§ЫҢ ШіШұЫҢШ№ ЩҒШ§ЫҢЩ„вҖҢЩҮШ§ЫҢ ШіЫҢвҖҢШҙШ§ШұЩҫ ШҜШұ ЩҲЫҢвҖҢШ§ШівҖҢЪ©ШҜ ШЁШ§ ШЁЩҮШұЩҮвҖҢЪҜЫҢШұЫҢ Ш§ШІ ШҜШ§ШӘвҖҢЩҶШӘ ЫұЫ°Ш§ШҙШӘШұШ§Ъ©вҖҢЩҮШ§Ш®ШҜШ§ШӯШ§ЩҒШёЫҢ ШЁШ§ Dockerfile ШҜШұ ШҜШ§ШӘвҖҢЩҶШӘ ЫұЫ° ШЁШ§ Ъ©Ш§ЩҶШӘЫҢЩҶШұШіШ§ШІЫҢ ШЁЩҲЩ…ЫҢ (Native Containerization)Ш§ШҙШӘШұШ§Ъ©вҖҢЩҮШ§Ш§ШіШӘЩҒШ§ШҜЩҮ Ш§ШІ ШЁШұЩҶШ§Щ…ЩҮвҖҢЩҮШ§ЫҢ Щ…ШЁШӘЩҶЫҢ ШЁШұ ЩҒШ§ЫҢЩ„ ШҜШұ .NET ШЁШұШ§ЫҢ ЩҶЩ…ЩҲЩҶЩҮвҖҢШіШ§ШІЫҢ API: ЩҶЪ©ШӘЩҮвҖҢШ§ЫҢ Ъ©ЩҮ ШҜШұ Ш§ЩҲЩ„ЫҢЩҶ Ш§Ш¬ШұШ§ ШЁШ§ ШўЩҶ Щ…ЩҲШ§Ш¬ЩҮ ШҙШҜЩ…Ш§ШҙШӘШұШ§Ъ©вҖҢЩҮШ§ШұЩҒШ№ Щ…ШҙЪ©Щ„Ш§ШӘ ШұШ§ЩҮвҖҢШ§ЩҶШҜШ§ШІЫҢ .NET SDK ШұЩҲЫҢ macOSЩ…ШіЫҢШұШұШ§ЩҮвҖҢЩҮШ§ШЁШұШұШіЫҢ ЩҲЫҢЪҳЪҜЫҢвҖҢвҖҢЩҮШ§ЫҢ Ш¬ШҜЫҢШҜ C# 14 ЩҲ NET 10.Щ…Ш·Ш§Щ„ШЁЩ„ЫҢЩҶЩҲЪ©Ші ШЁШұШ§ЫҢ ШӘЩҲШіШ№ЩҮвҖҢШҜЩҮЩҶШҜЪҜШ§ЩҶ ШҜШ§ШӘвҖҢЩҶШӘ: Ш§ШІ ЩҲЫҢЩҶШҜЩҲШІ ШӘШ§ Ъ©ШұШ§Ші-ЩҫЩ„ШӘЩҒШұЩ…Шӣ Ш®ЩҲШҜШўШІЩ…Ш§ЫҢЫҢЩ…Ш·Ш§Щ„ШЁШұШ§ЩҮЩҶЩ…Ш§ЫҢ Щ…ЩҮШ§Ш¬ШұШӘ ШЁЩҮ xUnit v3Щ…Ш·Ш§Щ„ШЁЩ„ЫҢЩҶЩҲЪ©Ші ШЁШұШ§ЫҢ ШӘЩҲШіШ№ЩҮвҖҢШҜЩҮЩҶШҜЪҜШ§ЩҶ ШҜШ§ШӘвҖҢЩҶШӘ: Ш§ШІ ЩҲЫҢЩҶШҜЩҲШІ ШӘШ§ Ъ©ШұШ§Ші-ЩҫЩ„ШӘЩҒШұЩ…Шӣ ШЁШ®Шҙ ШіЩҲЩ…: ШҜЫҢЩҫЩ„ЩҲЫҢ ЩҲ Ш§Ш¬ШұШ§ЫҢ ШЁШұЩҶШ§Щ…ЩҮвҖҢЩҮШ§ЫҢ ШҜШ§ШӘвҖҢЩҶШӘ ШұЩҲЫҢ Щ„ЫҢЩҶЩҲЪ©Ші
