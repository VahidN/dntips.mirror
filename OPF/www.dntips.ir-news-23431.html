‫کتابخانه Pri.ProductivityExtensions.Source: امکانات مدرن C# در .NET Standard 2.0


نویسنده: راویAI
تاریخ: ۱۴۰۴/۱۱/۱۴ ۱۰:۱۴
آدرس: www.dntips.ir
این بسته NuGet، کتابخانه منبعی را ارائه می‌دهد که امکان استفاده از ویژگی‌های زبانی مدرن C# را در کتابخانه‌های .NET Standard 2.0 فراهم می‌کند. این ویژگی‌ها شامل Ranges و Indices، توابع کمکی برای پرتاب ArgumentException، و نشانه‌های [DoesNotReturn] و [NotNullWhen] هستند. این کتابخانه به عنوان یک بسته منبع-فقط یا محتوا-فقط عمل می‌کند و هیچ فایل باینری اضافی را معرفی نمی‌کند که نیاز به مدیریت یا استقرار داشته باشد. برای استفاده از این کتابخانه، ابتدا یک پروژه کتابخانه کلاسی .NET Standard 2.0 ایجاد کنید. سپس، با استفاده از دستور `dotnet add client package Pri.ProductivityExtensions.Source`، به پروژه خود ارجاع دهید. این کار به طور خودکار فایل‌های لازم را برای فعال کردن ویژگی‌های زبانی C# در پروژه شما اضافه می‌کند. این کتابخانه با .NET Standard 2.0 هدف‌گیری شده است و با فریم‌ورک‌های بالاتر سازگار است. این بدان معناست که می‌توانید از آن در پروژه‌هایی که از .NET 5.0، .NET 6.0، .NET 7.0 و .NET 8.0 استفاده می‌کنند، استفاده کنید. همچنین، این کتابخانه از طیف گسترده‌ای از پلتفرم‌ها از جمله ویندوز، اندروید، iOS، macOS و tvOS پشتیبانی می‌کند. این بسته از طریق NuGet Gallery قابل دسترسی است و می‌توان آن را با استفاده از ابزارهای مختلفی مانند Package Manager Console، .NET CLI، PMC، Paket CLI، Script & Interactive، File-based Apps و Cake نصب کرد. برای اطلاعات بیشتر و دریافت پشتیبانی، می‌توانید با توسعه‌دهندگان این کتابخانه تماس بگیرید. این کتابخانه یک راه حل عالی برای توسعه‌دهندگانی است که می‌خواهند از ویژگی‌های زبانی مدرن C# در پروژه‌های .NET Standard 2.0 خود استفاده کنند، بدون اینکه نیاز به افزودن وابستگی‌های باینری اضافی داشته باشند.
