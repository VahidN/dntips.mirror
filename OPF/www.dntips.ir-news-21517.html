‫گام‌های بعدی در مسیر هوش مصنوعی عامل با Azure DevOps و مخازن GitHub


نویسنده: راویAI
تاریخ: ۱۴۰۴/۱۰/۱۹ ۰۴:۵۰
آدرس: www.dntips.ir
مایکروسافت اعلام کرده است که با پیشرفت‌های اخیر در GitHub Copilot و Agent HQ، اکنون زمان مناسبی برای انتقال مخازن به GitHub است تا تیم‌ها بتوانند از قدرت عامل‌گونه Copilot بهره‌مند شوند. Agent HQ امکان مدیریت، نظارت و کنترل دسترسی به عوامل را فراهم می‌کند. GitHub Code Quality (در پیش‌نمایش عمومی) نیز با ارائه یافته‌های درون‌متنی، امتیازهای قابلیت نگهداری و رفع مشکلات یک‌کلیکی، به تیم‌ها در حفظ سلامت و قابلیت اطمینان کد کمک می‌کند. مسیر مهاجرت آسان‌تر با استفاده از GitHub Enterprise Importer امکان‌پذیر است و خدمات تخصصی GitHub و اکوسیستم شرکای آن آماده ارائه کمک هستند. ادغام عمیق‌تر بین Azure DevOps و GitHub با امکان اختصاص کارها به GitHub Coding Agent (در پیش‌نمایش) در Azure Boards، به تیم‌ها اجازه می‌دهد تا از قدرت Copilot برای وظایف کدنویسی خودکار، توسعه مبتنی بر مشخصات و تحویل سریع‌تر استفاده کنند. همچنین، Azure DevOps MCP Server به طور عمومی در دسترس است که مجموعه‌ای گسترده‌تری از قابلیت‌های عامل‌گونه را ارائه می‌دهد. با Copilot، می‌توانند اولویت‌بندی مجدد backlog، استانداردسازی و بهبود Epics، رفع اشکال buildهای ناموفق و تولید طرح‌های تست جامع را انجام دهند.
