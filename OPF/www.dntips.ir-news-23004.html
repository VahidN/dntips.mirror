Ш§ЫҢЩҶ Щ…ЩӮШ§Щ„ЩҮ ШЁЩҮ ШЁШұШұШіЫҢ ЩҫШұЩҲЪҳЩҮвҖҢЩҮШ§ЫҢ Щ…ШӘЩҶвҖҢШЁШ§ШІ ШіЫҢШіШӘЩ… Щ…ШҜЫҢШұЫҢШӘ Щ…ШӯШӘЩҲШ§ (CMS) Щ…ШЁШӘЩҶЫҢ ШЁШұ .NET Щ…ЫҢвҖҢЩҫШұШҜШ§ШІШҜ Ъ©ЩҮ ШЁШұШ§ЫҢ Ш§ШіШӘЩҒШ§ШҜЩҮ ШҜШұ ШіШ§Щ„ ЫІЫ°ЫІЫ¶ Щ…ЩҶШ§ШіШЁ ЩҮШіШӘЩҶШҜ. ЩҶЩҲЫҢШіЩҶШҜЩҮ ШҜШұ Ш§ЫҢЩҶ ШЁШұШұШіЫҢШҢ ЩҫШұЩҲЪҳЩҮвҖҢЩҮШ§ЫҢЫҢ ШұШ§ ШҜШұ ЩҶШёШұ ЪҜШұЩҒШӘЩҮ Ъ©ЩҮ Ъ©Ш§Щ…Щ„Ш§ЩӢ Щ…ШӘЩҶвҖҢШЁШ§ШІ ШЁШ§ШҙЩҶШҜ (ЩҶЩҮ ЩҒЩӮШ· ЩҮШіШӘЩҮ Щ…ШӘЩҶвҖҢШЁШ§ШІ)ШҢ ШЁШұ ШұЩҲЫҢ Щ„ЫҢЩҶЩҲЪ©Ші Ш§Ш¬ШұШ§ ШҙЩҲЩҶШҜ (ЩҲ ШӘШұШ¬ЫҢШӯШ§ЩӢ Ш§ШІ ЩҫШ§ЫҢЪҜШ§ЩҮ ШҜШ§ШҜЩҮ PostgreSQL ЩҫШҙШӘЫҢШЁШ§ЩҶЫҢ Ъ©ЩҶЩҶШҜ)ШҢ ШЁЩҮ Ш·ЩҲШұ ЩҒШ№Ш§Щ„ ШҜШұ ШӯШ§Щ„ ШӘЩҲШіШ№ЩҮ ШЁШ§ШҙЩҶШҜ ЩҲ Ш®Ш·Шұ ШұЩҮШ§ ШҙШҜЩҶ ШўЩҶвҖҢЩҮШ§ Ъ©Щ… ШЁШ§ШҙШҜ. Щ…Ш№ЫҢШ§Шұ Ш§ШөЩ„ЫҢ ШЁШұШ§ЫҢ Ш§ЩҶШӘШ®Ш§ШЁ ЩҫШұЩҲЪҳЩҮвҖҢЩҮШ§ШҢ ШҜШ§ШҙШӘЩҶ ШӯШҜШ§ЩӮЩ„ ЪҶЩҶШҜ Щ…ШҙШ§ШұЪ©ШӘвҖҢЪ©ЩҶЩҶШҜЩҮ ШҜШұ Ш·ЩҲЩ„ ШіШ§Щ„ ЪҜШ°ШҙШӘЩҮ ШЁЩҲШҜЩҮ Ш§ШіШӘ. ЩҫШұЩҲЪҳЩҮвҖҢЩҮШ§ЫҢ Ш§ШөЩ„ЫҢ Щ…ЩҲШұШҜ ШЁШұШұШіЫҢ ШҙШ§Щ…Щ„ Orchard CoreШҢ UmbracoШҢ Squidex ЩҲ Oqtane ЩҮШіШӘЩҶШҜ. Orchard Core ШЁЩҮ ШҜЩ„ЫҢЩ„ Ш¬Ш§Щ…Ш№ЩҮ ШЁШІШұЪҜШҢ Щ…Ш§ЪҳЩҲЩ„Ш§Шұ ШЁЩҲШҜЩҶ ЩҲ ЩӮШ§ШЁЩ„ЫҢШӘ ШӘЩҲШіШ№ЩҮ ШўШіШ§ЩҶШҢ ЪҜШІЫҢЩҶЩҮ Ш§ЫҢЩ…ЩҶвҖҢШӘШұЫҢ ШЁЩҮ ЩҶШёШұ Щ…ЫҢвҖҢШұШіШҜШҢ Ш§Щ…Ш§ ШҜШ§ШұШ§ЫҢ Щ…ЩҶШӯЩҶЫҢ ЫҢШ§ШҜЪҜЫҢШұЫҢ ШӘЩҶШҜЫҢ ЩҲ Щ…Ш№Щ…Ш§ШұЫҢ ЩҫЫҢЪҶЫҢШҜЩҮвҖҢШ§ЫҢ Ш§ШіШӘ. Umbraco ЫҢЪ© CMS ШЁШІШұЪҜ ШҜЫҢЪҜШұ Ш§ШіШӘ Ъ©ЩҮ ШҜШұ ШӯШ§Щ„ ШӯШ§Ш¶Шұ ЩҒЩӮШ· Ш§ШІ MS SQL Server ЩҫШҙШӘЫҢШЁШ§ЩҶЫҢ Щ…ЫҢвҖҢЪ©ЩҶШҜШҢ Ш§Щ…Ш§ ШЁШұЩҶШ§Щ…ЩҮвҖҢЩҮШ§ЫҢЫҢ ШЁШұШ§ЫҢ Щ…ЩҮШ§Ш¬ШұШӘ ШЁЩҮ EF Core ШҜШұ ШіШ§Щ„ ЫІЫ°ЫІЫ¶ ЩҲ Ш§ЩҒШІЩҲШҜЩҶ ЩҫШҙШӘЫҢШЁШ§ЩҶЫҢ Ш§ШІ ЩҫШ§ЫҢЪҜШ§ЩҮвҖҢЩҮШ§ЫҢ ШҜШ§ШҜЩҮ ШҜЫҢЪҜШұ ШҜШ§ШұШҜ. Squidex ЫҢЪ© CMS ШЁШҜЩҲЩҶ ШіШұ (Headless CMS) Ш§ШіШӘ Ъ©ЩҮ ЩҶШіШЁШӘШ§ЩӢ Ш¬ШҜЫҢШҜШӘШұ Ш§ШіШӘШҢ Ш§Щ…Ш§ ШЁЩҮ ЩҶШёШұ Щ…ЫҢвҖҢШұШіШҜ Ш§ШІ Ш·ШұЫҢЩӮ SaaS ЩҲ Ш®ШҜЩ…Ш§ШӘ Щ…ШҜЫҢШұЫҢШӘ ШҙШҜЩҮШҢ ШЁЩҮ Ш®ЩҲШЁЫҢ ШӘШ§Щ…ЫҢЩҶ Щ…Ш§Щ„ЫҢ ШҙШҜЩҮ ЩҲ Ш®Ш·Шұ ШұЩҮШ§ ШҙШҜЩҶ ШўЩҶ Ъ©Щ… Ш§ШіШӘ. Oqtane ЩҶЫҢШІ ШӘЩҲШіШ· ШӘЩҲШіШ№ЩҮвҖҢШҜЩҮЩҶШҜЩҮ Ш§ШөЩ„ЫҢ DNN (DotNetNuke) ШӘЩҲШіШ№ЩҮ ЫҢШ§ЩҒШӘЩҮ ЩҲ ШЁЩҮ Ш№ЩҶЩҲШ§ЩҶ ЫҢЪ© ЪҜШІЫҢЩҶЩҮ Ш¬Ш§Щ„ШЁ Щ…Ш№ШұЩҒЫҢ ШҙШҜЩҮ Ш§ШіШӘ. ЩҶЩҲЫҢШіЩҶШҜЩҮ Ш§ШІ ШіШ§ЫҢШұ ШӘЩҲШіШ№ЩҮвҖҢШҜЩҮЩҶШҜЪҜШ§ЩҶ Щ…ЫҢвҖҢШ®ЩҲШ§ЩҮШҜ ШӘШ¬ШұШЁЫҢШ§ШӘ Ш®ЩҲШҜ ШұШ§ ШЁШ§ Ш§ЫҢЩҶ ЩҫШұЩҲЪҳЩҮвҖҢЩҮШ§ ШЁЩҮ Ш§ШҙШӘШұШ§Ъ© ШЁЪҜШ°Ш§ШұЩҶШҜ ЩҲ ЩҶШёШұШ§ШӘ Ш®ЩҲШҜ ШұШ§ ШЁЫҢШ§ЩҶ Ъ©ЩҶЩҶШҜ. Ш№Щ„Ш§ЩҲЩҮ ШЁШұ ЩҫШұЩҲЪҳЩҮвҖҢЩҮШ§ЫҢ Ш§ШөЩ„ЫҢШҢ ЩҫШұЩҲЪҳЩҮвҖҢЩҮШ§ЫҢ ШҜЫҢЪҜШұЫҢ ЩҶЫҢШІ Щ…Ш№ШұЩҒЫҢ ШҙШҜЩҮвҖҢШ§ЩҶШҜ Ъ©ЩҮ Щ…Щ…Ъ©ЩҶ Ш§ШіШӘ ЩҮЩҶЩҲШІ ШЁЩҮ Ш§ЩҶШҜШ§ШІЩҮ Ъ©Ш§ЩҒЫҢ Ш§Ш«ШЁШ§ШӘ ЩҶШҙШҜЩҮ ШЁШ§ШҙЩҶШҜ ЫҢШ§ ШӘЩҲШіШ· ЫҢЪ© ЩҶЩҒШұ ШӘЩҲШіШ№ЩҮ ШҜШ§ШҜЩҮ ШҙЩҲЩҶШҜ. Ш§ЫҢЩҶ ЩҫШұЩҲЪҳЩҮвҖҢЩҮШ§ ШҙШ§Щ…Щ„ Piranha CMSШҢ CofoundryШҢ FluentCMSШҢ cloudscribe SimpleContentШҢ FormCMSШҢ Raytha ЩҮШіШӘЩҶШҜ. ЩҶЩҲЫҢШіЩҶШҜЩҮ Ш§ШІ Ш®ЩҲШ§ЩҶЩҶШҜЪҜШ§ЩҶ Щ…ЫҢвҖҢШ®ЩҲШ§ЩҮШҜ ШҜШұ ШөЩҲШұШӘ ШҜШ§ШҙШӘЩҶ ШӘШ¬ШұШЁЩҮ ШЁШ§ Ш§ЫҢЩҶ ЩҫШұЩҲЪҳЩҮвҖҢЩҮШ§ШҢ ШӘШ¬ШұШЁЫҢШ§ШӘ Ш®ЩҲШҜ ШұШ§ ШЁЩҮ Ш§ШҙШӘШұШ§Ъ© ШЁЪҜШ°Ш§ШұЩҶШҜ. ШҜШұ ЩҫШ§ЫҢШ§ЩҶ Щ…ЩӮШ§Щ„ЩҮШҢ ШЁЩҮ Щ…ЩҶШ§ШЁШ№ ШўЩ…ЩҲШІШҙЫҢ C# Ш§ШҙШ§ШұЩҮ ШҙШҜЩҮ Ш§ШіШӘ Ъ©ЩҮ ШҙШ§Щ…Щ„ Щ…ШіШӘЩҶШҜШ§ШӘ ШұШіЩ…ЫҢ Щ…Ш§ЫҢЪ©ШұЩҲШіШ§ЩҒШӘ ЩҲ Щ…ШіЫҢШұЩҮШ§ЫҢ ЫҢШ§ШҜЪҜЫҢШұЫҢ ШӘШ№Ш§Щ…Щ„ЫҢ ШҜШұ Microsoft Learn Щ…ЫҢвҖҢШЁШ§ШҙШҜ.

    Ш§ШҙШӘШұШ§Ъ©вҖҢЩҮШ§вҖ«Ш§ШіШӘЩҒШ§ШҜЩҮ Ш§ШІ ЫҢЪ© Web Application ШЁЩҮ Ш№ЩҶЩҲШ§ЩҶ ЫҢЪ© Module ШҜШұ Orchard CMSШ§ШҙШӘШұШ§Ъ©вҖҢЩҮШ§ЫұЫ¶ ЩҫШұЩҲЪҳЩҮ Щ…ШӘЩҶвҖҢШЁШ§ШІ Ъ©ЩҮ ШҜШұ ШӯШ§Щ„ Щ…ШӘШӯЩҲЩ„ Ъ©ШұШҜЩҶ ЩҮЩҲШҙ Щ…ШөЩҶЩҲШ№ЫҢ ЩҲ ЫҢШ§ШҜЪҜЫҢШұЫҢ Щ…Ш§ШҙЫҢЩҶ ЩҮШіШӘЩҶШҜШ§ШҙШӘШұШ§Ъ©вҖҢЩҮШ§ШӘШӯЩҲЩ„Ш§ШӘ ШҜШұ ШөШҜЩҲШұ Щ…Ш¬ЩҲШІ ЩҶШұЩ…вҖҢШ§ЩҒШІШ§Шұ Щ…ШӘЩҶвҖҢШЁШ§ШІШ§ШҙШӘШұШ§Ъ©вҖҢЩҮШ§ШӘШәЫҢЫҢШұ Щ…ШҜЩ„ ШӘШ¬Ш§ШұЫҢ MassTransitШ§ШҙШӘШұШ§Ъ©вҖҢЩҮШ§ЩҒШұШө Щ…ШҙШ§ШұЪ©ШӘ ШҜШұ ЩҫШұЩҲЪҳЩҮвҖҢЩҮШ§ЫҢ Щ…ШӘЩҶвҖҢШЁШ§ШІ ШЁШ§ ШЁШұЩҶШ§Щ…ЩҮ ЪҜЩҲЪҜЩ„ ШіШ§Щ…Шұ ШўЩҒ Ъ©ШҜ (GSoC)Ш§ШҙШӘШұШ§Ъ©вҖҢЩҮШ§ШўЫҢШ§ Ш§Ъ©ЩҲШіЫҢШіШӘЩ… NET. ШҜШұ ШЁШӯШұШ§ЩҶ Ш§ШіШӘШҹШ§ШҙШӘШұШ§Ъ©вҖҢЩҮШ§ШЁШұШұШіЫҢ ЩҫШұЩҲЪҳЩҮвҖҢЩҮШ§ЫҢ Щ…ШӘЩҶвҖҢШЁШ§ШІ .NET ЩҲ Ш§ШЁШІШ§ШұЩҮШ§ЫҢ ЩҫШҙШӘЫҢШЁШ§ЩҶЫҢвҖҢЪ©ЩҶЩҶШҜЩҮШ§ШҙШӘШұШ§Ъ©вҖҢЩҮШ§ШўЫҢШ§ Ш§ЫҢЩҶ ЩҫШұЩҲЪҳЩҮ ЩҲШ§ЩӮШ№Ш§ Щ…ШӘЩҶвҖҢШЁШ§ШІ (FOSS) Ш§ШіШӘШҹЩ…Ш·Ш§Щ„ШЁЩ…Ш№ШұЩҒЫҢ ЩҫШұЩҲЪҳЩҮ OrchardЩҶШёШұШ§ШӘ Щ…Ш·Ш§Щ„ШЁЩ…Ш№ШұЩҒЫҢ ЩҫШұЩҲЪҳЩҮ Orchard
