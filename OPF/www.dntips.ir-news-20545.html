Щ…ШҜЫҢШұЫҢШӘ ШҜЩӮЫҢЩӮ ШӘШәЫҢЫҢШұШ§ШӘ ЫҢЪ© ЩҫШұЩҲЪҳЩҮ ЫҢЪ©ЫҢ Ш§ШІ Щ…ЩҮЩ…вҖҢШӘШұЫҢЩҶ ЩҲ ШҜШұ Ш№ЫҢЩҶ ШӯШ§Щ„ ЪҶШ§Щ„ШҙвҖҢШЁШұШ§ЩҶЪҜЫҢШІШӘШұЫҢЩҶ ЩҲШёШ§ЫҢЩҒ Ш§ШіШӘ. ЫҢЪ© Changelog ЫҢШ§ "ШіЩҶШҜ Ш«ШЁШӘ ШӘШәЫҢЫҢШұШ§ШӘ" Ш§ШЁШІШ§ШұЫҢ ЩӮШҜШұШӘЩ…ЩҶШҜ Ш§ШіШӘ Ъ©ЩҮ ШЁЩҮ ШҙЩ…Ш§ Ъ©Щ…Ъ© Щ…ЫҢвҖҢЪ©ЩҶШҜ ШӘШ§ Ш§ЫҢЩҶ ЩҒШұШўЫҢЩҶШҜ ШұШ§ ШЁЩҮвҖҢШҙЪ©Щ„ЫҢ ШіШ§ШІЩ…Ш§ЩҶвҖҢЫҢШ§ЩҒШӘЩҮ ЩҲ ШҙЩҒШ§ЩҒ Щ…ШҜЫҢШұЫҢШӘ Ъ©ЩҶЫҢШҜ. Ш§ЫҢЩҶ Щ…ЩӮШ§Щ„ЩҮШҢ ШҜЩ„Ш§ЫҢЩ„ Ш§ЩҮЩ…ЫҢШӘ ШҜШ§ШҙШӘЩҶ ЫҢЪ© Changelog ЩҲ ЩҶЪ©Ш§ШӘЫҢ Ш№Щ…Щ„ЫҢ ШЁШұШ§ЫҢ ЩҶЪҜЩҮШҜШ§ШұЫҢ ШўЩҶ ШҜШұ ЩҫШұЩҲЪҳЩҮвҖҢЩҮШ§ЫҢ ШҜШ§ШӘвҖҢЩҶШӘ ШұШ§ ШЁШұШұШіЫҢ Щ…ЫҢвҖҢЪ©ЩҶШҜ.

Changelog ЪҶЫҢШіШӘШҹ

Changelog ЫҢЪ© ШұЪ©ЩҲШұШҜ ШҜЫҢШ¬ЫҢШӘШ§Щ„ЫҢ ЩҲ ШІЩ…Ш§ЩҶЫҢ Ш§ШІ ШӘЩ…Ш§Щ… ШӘШәЫҢЫҢШұШ§ШӘЫҢ Ш§ШіШӘ Ъ©ЩҮ ШҜШұ ЫҢЪ© ЩҶШұЩ…вҖҢШ§ЩҒШІШ§Шұ ЫҢШ§ ШіЫҢШіШӘЩ… Ш§ЫҢШ¬Ш§ШҜ Щ…ЫҢвҖҢШҙЩҲШҜ. ШЁШұШ®Щ„Ш§ЩҒ Release Notes Ъ©ЩҮ ШЁЫҢШҙШӘШұ ШЁШұ ШұЩҲЫҢ ЩҲЫҢЪҳЪҜЫҢвҖҢЩҮШ§ЫҢ ШЁШІШұЪҜ ЩҲ ЩҶШіШ®ЩҮвҖҢЩҮШ§ЫҢ Ш§ШөЩ„ЫҢ ШӘЩ…ШұЪ©ШІ ШҜШ§ШұЩҶШҜШҢ Changelog Ъ©ЩҲЪҶЪ©вҖҢШӘШұЫҢЩҶ ШӘШәЫҢЫҢШұШ§ШӘ ЩҒЩҶЫҢШҢ ШұЩҒШ№ ШЁШ§ЪҜвҖҢЩҮШ§ШҢ ЩҲ ШЁЩҮвҖҢШұЩҲШІШұШіШ§ЩҶЫҢвҖҢЩҮШ§ЫҢ Ш§Щ…ЩҶЫҢШӘЫҢ ШұШ§ ЩҶЫҢШІ Ш«ШЁШӘ Щ…ЫҢвҖҢЪ©ЩҶШҜ.

Щ…ШІШ§ЫҢШ§ЫҢ ШҜШ§ШҙШӘЩҶ Changelog ШЁШұШ§ЫҢ ШЁШұЩҶШ§Щ…ЩҮвҖҢЩҶЩҲЫҢШіШ§ЩҶ ШҜШ§ШӘвҖҢЩҶШӘ

ШЁЩҮШЁЩҲШҜ Ш§ШұШӘШЁШ§Ш·Ш§ШӘ ШӘЫҢЩ…ЫҢ ЩҲ Щ…ШҙШӘШұЫҢШ§ЩҶ
ЫҢЪ© Changelog ШҙЩҒШ§ЩҒ ШЁЩҮ ШӘЫҢЩ… ШҙЩ…Ш§ Ъ©Щ…Ъ© Щ…ЫҢвҖҢЪ©ЩҶШҜ ШӘШ§ ЩҮЩ…ЫҢШҙЩҮ Ш§ШІ ЩҲШ¶Ш№ЫҢШӘ ЩҫШұЩҲЪҳЩҮ ШўЪҜШ§ЩҮ ШЁШ§ШҙЩҶШҜ. Ш§ЪҜШұ ЫҢЪ© ШЁШ§ЪҜ Ш¬ШҜЫҢШҜ ЪҜШІШ§ШұШҙ ШҙЩҲШҜШҢ ШӘЫҢЩ… ЩҫШҙШӘЫҢШЁШ§ЩҶЫҢ Щ…ЫҢвҖҢШӘЩҲШ§ЩҶШҜ ШЁЩҮ ШіШұШ№ШӘ Changelog ШұШ§ ШЁШұШұШіЫҢ Ъ©ЩҶШҜ ШӘШ§ ШЁШЁЫҢЩҶШҜ ШўЫҢШ§ ШӘШәЫҢЫҢШұ Ш®Ш§ШөЫҢ ЩӮШЁЩ„ Ш§ШІ ШЁШұЩҲШІ Щ…ШҙЪ©Щ„ Ш§ЫҢШ¬Ш§ШҜ ШҙШҜЩҮ Ш§ШіШӘ ЫҢШ§ Ш®ЫҢШұ. Ш№Щ„Ш§ЩҲЩҮ ШЁШұ Ш§ЫҢЩҶШҢ Щ…ШҙШӘШұЫҢШ§ЩҶ ЩҲ Ъ©Ш§ШұШЁШұШ§ЩҶ ЩҶЫҢШІ ШЁШ§ Щ…ШҙШ§ЩҮШҜЩҮ Changelog Щ…ШӘЩҲШ¬ЩҮ Щ…ЫҢвҖҢШҙЩҲЩҶШҜ Ъ©ЩҮ Щ…ШӯШөЩҲЩ„ ШҙЩ…Ш§ ШЁЩҮ Ш·ЩҲШұ Щ…ЩҶШёЩ… ЩҶЪҜЩҮШҜШ§ШұЫҢ ЩҲ ШЁЩҮШЁЩҲШҜ Щ…ЫҢвҖҢЫҢШ§ШЁШҜ.

ШҙЩҒШ§ЩҒЫҢШӘ ЩҲ ЩҫШ§ШіШ®ЪҜЩҲЫҢЫҢ
Ш§ЩҶШӘШҙШ§Шұ ЫҢЪ© Changelog Ш№Щ…ЩҲЩ…ЫҢ ЩҶШҙШ§ЩҶвҖҢШҜЩҮЩҶШҜЩҮ ШӘШ№ЩҮШҜ ШҙЩ…Ш§ ШЁЩҮ Ъ©ЫҢЩҒЫҢШӘ ЩҲ ШҙЩҒШ§ЩҒЫҢШӘ Ш§ШіШӘ. Ш§ЫҢЩҶ Ъ©Ш§Шұ ШЁЩҮ Ъ©Ш§ШұШЁШұШ§ЩҶ ЩҶШҙШ§ЩҶ Щ…ЫҢвҖҢШҜЩҮШҜ Ъ©ЩҮ ШӯШӘЫҢ ШӘШәЫҢЫҢШұШ§ШӘ Ъ©ЩҲЪҶЪ© Щ…Ш§ЩҶЩҶШҜ ШұЩҒШ№ ШЁШ§ЪҜвҖҢЩҮШ§ ЩҲ ШЁЩҮвҖҢШұЩҲШІШұШіШ§ЩҶЫҢвҖҢЩҮШ§ЫҢ Ш§Щ…ЩҶЫҢШӘЫҢ ЩҶЫҢШІ ШЁШұШ§ЫҢ ШҙЩ…Ш§ Ш§ЩҮЩ…ЫҢШӘ ШҜШ§ШұЩҶШҜ. Ш§ЫҢЩҶ ШҙЩҒШ§ЩҒЫҢШӘШҢ Ш§Ш№ШӘЩ…Ш§ШҜ Ъ©Ш§ШұШЁШұШ§ЩҶ ШЁЩҮ Щ…ШӯШөЩҲЩ„ ШҙЩ…Ш§ ШұШ§ Ш§ЩҒШІШ§ЫҢШҙ Щ…ЫҢвҖҢШҜЩҮШҜ ЩҲ ШўЩҶ ШұШ§ Ш§ШІ ШұЩӮШЁШ§ Щ…ШӘЩ…Ш§ЫҢШІ Щ…ЫҢвҖҢЪ©ЩҶШҜ.

ЫҢЪ© ШұЪ©ЩҲШұШҜ ШӘШ§ШұЫҢШ®ЫҢ ШҜЩӮЫҢЩӮ
Changelog ЫҢЪ© Щ…ШұШ¬Ш№ Ш§ШұШІШҙЩ…ЩҶШҜ ШЁШұШ§ЫҢ ШӘШ§ШұЫҢШ®ЪҶЩҮ ЩҫШұЩҲЪҳЩҮ ШҙЩ…Ш§ШіШӘ. ШЁШ§ Щ…ШіШӘЩҶШҜШіШ§ШІЫҢ ШӘЩ…Ш§Щ… ШӘШәЫҢЫҢШұШ§ШӘШҢ ШӘЫҢЩ… ШҙЩ…Ш§ Щ…ЫҢвҖҢШӘЩҲШ§ЩҶШҜ Ш§ШІ ШӘШ¬ШұШЁЫҢШ§ШӘ ЪҜШ°ШҙШӘЩҮ ШҜШұШі ШЁЪҜЫҢШұШҜ ЩҲ Ш§ШІ ШЁШұЩҲШІ Щ…ШҙЪ©Щ„Ш§ШӘ Щ…ШҙШ§ШЁЩҮ ШҜШұ ШўЫҢЩҶШҜЩҮ Ш¬Щ„ЩҲЪҜЫҢШұЫҢ Ъ©ЩҶШҜ. Ш§ЫҢЩҶ ШұЪ©ЩҲШұШҜ ШЁЩҮ ШҙЩ…Ш§ Ъ©Щ…Ъ© Щ…ЫҢвҖҢЪ©ЩҶШҜ ШӘШ§ ШЁЩҒЩҮЩ…ЫҢШҜ Ъ©ШҜШ§Щ… ШЁШ®ШҙвҖҢЩҮШ§ЫҢ ШіЫҢШіШӘЩ… ШҙЩ…Ш§ ЩҶЫҢШ§ШІ ШЁЩҮ ШӘЩҲШ¬ЩҮ ШЁЫҢШҙШӘШұЫҢ ШҜШ§ШұЩҶШҜ.

ЩҶЪ©Ш§ШӘЫҢ ШЁШұШ§ЫҢ ШіШ§Ш®ШӘ ЩҲ ЩҶЪҜЩҮШҜШ§ШұЫҢ Changelog ШҜШұ ЩҫШұЩҲЪҳЩҮвҖҢЩҮШ§ЫҢ ШҜШ§ШӘвҖҢЩҶШӘ

ШіШ§Ш®ШӘШ§ШұШҜЩҮЫҢ Щ…ЩҶШ§ШіШЁ
ШЁШұШ§ЫҢ Ш§ЫҢЩҶЪ©ЩҮ Changelog ШҙЩ…Ш§ Щ…ЩҒЫҢШҜ ШЁШ§ШҙШҜШҢ ШЁШ§ЫҢШҜ ЫҢЪ© ШіШ§Ш®ШӘШ§Шұ Ш«Ш§ШЁШӘ ШҜШ§ШҙШӘЩҮ ШЁШ§ШҙШҜ. ЩҮШұ ЩҲШұЩҲШҜЫҢ ШЁШ§ЫҢШҜ ШҙШ§Щ…Щ„ Щ…ЩҲШ§ШұШҜ ШІЫҢШұ ШЁШ§ШҙШҜ:
  • ШҙЩ…Ш§ШұЩҮ ЩҶШіШ®ЩҮ (Version Number): ШЁШұШ§ЫҢ Щ…Ш«Ш§Щ„ШҢ "V3.1.5"
  • ШӘШ§ШұЫҢШ® Ш§ЩҶШӘШҙШ§Шұ (Release Date): ШЁШұШ§ЫҢ Щ…Ш«Ш§Щ„ШҢ "2025-06-19"
  • ЩҶЩҲШ№ ШӘШәЫҢЫҢШұ (Change Type): Ш§ШІ ШЁШұЪҶШіШЁвҖҢЩҮШ§ЫҢ Ш§ШіШӘШ§ЩҶШҜШ§ШұШҜ Ш§ШіШӘЩҒШ§ШҜЩҮ Ъ©ЩҶЫҢШҜШҢ Щ…Ш§ЩҶЩҶШҜ:
  • ADDED: ШЁШұШ§ЫҢ ЩҲЫҢЪҳЪҜЫҢвҖҢЩҮШ§ЫҢ Ш¬ШҜЫҢШҜ.
  • CHANGED: ШЁШұШ§ЫҢ ЩҲЫҢЪҳЪҜЫҢвҖҢЩҮШ§ЫҢ ШӘШәЫҢЫҢШұ ЫҢШ§ЩҒШӘЩҮ.
  • FIXED: ШЁШұШ§ЫҢ ШЁШ§ЪҜвҖҢЩҮШ§ЫҢ ШЁШұШ·ШұЩҒ ШҙШҜЩҮ.
  • SECURITY: ШЁШұШ§ЫҢ ШЁЩҮвҖҢШұЩҲШІШұШіШ§ЩҶЫҢвҖҢЩҮШ§ЫҢ Ш§Щ…ЩҶЫҢШӘЫҢ.
  • DEPRECATED: ШЁШұШ§ЫҢ ЩҲЫҢЪҳЪҜЫҢвҖҢЩҮШ§ЫҢЫҢ Ъ©ЩҮ ЩӮШұШ§Шұ Ш§ШіШӘ ШӯШ°ЩҒ ШҙЩҲЩҶШҜ.
  • ШӘЩҲШ¶ЫҢШӯШ§ШӘ (Description): ЫҢЪ© ШӘЩҲШ¶ЫҢШӯ Щ…Ш®ШӘШөШұ ЩҲ Щ…ЩҒЫҢШҜ Ш§ШІ ШӘШәЫҢЫҢШұ.

Ш«ШЁШӘ ШӘЩ…Ш§Щ… ШӘШәЫҢЫҢШұШ§ШӘ
  • ШЁШұШ®Щ„Ш§ЩҒ Release Notes Ъ©ЩҮ ЩҒЩӮШ· ШЁЩҮ ШӘШәЫҢЫҢШұШ§ШӘ Щ…Ш«ШЁШӘ Щ…ЫҢвҖҢЩҫШұШҜШ§ШІЩҶШҜШҢ Changelog ШЁШ§ЫҢШҜ ШӘЩ…Ш§Щ… ШӘШәЫҢЫҢШұШ§ШӘШҢ ШӯШӘЫҢ Щ…ЩҲШ§ШұШҜ Щ…ЩҶЩҒЫҢ Щ…Ш§ЩҶЩҶШҜ ШЁШ§ЪҜвҖҢЩҮШ§ ЩҲ Щ…ШҙЪ©Щ„Ш§ШӘ Ш№Щ…Щ„Ъ©ШұШҜЫҢ ШұШ§ ЩҶЫҢШІ Ш«ШЁШӘ Ъ©ЩҶШҜ. Ш§ШіШӘЩҒШ§ШҜЩҮ Ш§ШІ ШЁШұЪҶШіШЁвҖҢЩҮШ§ЫҢ Ш§ШіШӘШ§ЩҶШҜШ§ШұШҜ ШЁШ§Щ„Ш§ ШЁЩҮ ШҙЩ…Ш§ Ъ©Щ…Ъ© Щ…ЫҢвҖҢЪ©ЩҶШҜ ШӘШ§ ЩҮЩ…ЩҮ ЪҶЫҢШІ ШұШ§ ШЁЩҮ ШөЩҲШұШӘ Щ…ЩҶШіШ¬Щ… Ш«ШЁШӘ Ъ©ЩҶЫҢШҜ.
Ш§ШіШӘЩҒШ§ШҜЩҮ Ш§ШІ ШӘШұШӘЫҢШЁ ШІЩ…Ш§ЩҶЫҢ Щ…Ш№Ъ©ЩҲШі
