‫دوره نحوه ساخت یک برنامه وب کامل با استفاده از ASP.NET Core MVC


نویسنده: وحید نصیری
تاریخ: ۱۴۰۳/۱۲/۰۳ ۰۹:۲۶
آدرس: www.dntips.ir
این دوره رایگان از freeCodeCamp.org در یوتیوب نحوه ساخت یک برنامه وب کامل با استفاده از ASP.NET Core MVC را آموزش می‌دهد.

بررسی اجمالی دوره:
  • کل فرآیند توسعه، از راه‌اندازی پروژه تا ادغام کتابخانه‌های خارجی مانند Chart.js را پوشش می‌دهد.
  • آموزش گام به گام ایجاد مدل‌ها، کنترلرها و مدیریت پایگاه داده را ارائه می‌دهد.
  • بر ساخت یک برنامه وب کاربردی و ایجاد پایه و اساس قوی در توسعه ASP.NET Core تمرکز دارد.

نکات کلیدی یادگیری:
  • راه‌اندازی پروژه: ایجاد و پیکربندی یک پروژه جدید ASP.NET Core MVC.
  • ساخت مدل: تعریف ساختارهای داده (به عنوان مثال، مدل هزینه) و پیاده‌سازی قوانین اعتبارسنجی.
  • تعامل با پایگاه داده: استفاده از Entity Framework Core برای ایجاد و مدیریت پایگاه داده.
  • توسعه کنترلر و متد: ساخت کنترلرها و متدها برای مدیریت درخواست‌های HTTP و منطق برنامه، از جمله عملیات ناهمزمان.
  • پیاده‌سازی لایه سرویس: ایجاد یک لایه سرویس برای مدیریت منطق کسب و کار.
  • ادغام نمودار: افزودن نمودارهای پویا با استفاده از Chart.js برای تجسم داده‌ها.
