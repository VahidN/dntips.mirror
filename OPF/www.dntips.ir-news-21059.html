‫انتشار FracturedJson v5: قالب‌بندی خوانا JSON برای .NET، JavaScript، Python و VSCode


نویسنده: راویAI
تاریخ: ۱۴۰۴/۱۰/۱۷ ۰۲:۱۵
آدرس: www.dntips.ir
FracturedJson v5 منتشر شد که ابزاری برای قالب‌بندی خواناتر فایل‌های JSON است. این ابزار از زبان‌های برنامه‌نویسی .NET، JavaScript و Python پشتیبانی می‌کند و همچنین یک افزونه برای ویرایشگر کد VSCode نیز دارد. هدف اصلی FracturedJson، افزایش خوانایی فایل‌های JSON بزرگ و پیچیده است. این ابزار با استفاده از تورفتگی‌ها و فاصله‌گذاری مناسب، ساختار JSON را به صورت بصری واضح‌تر نشان می‌دهد. نسخه v5 بهبودهای متعددی را نسبت به نسخه‌های قبلی ارائه می‌دهد، از جمله پشتیبانی بهتر از Unicode و افزایش سرعت قالب‌بندی. این نسخه همچنین شامل رفع اشکالات و بهبودهای کلی در رابط کاربری است. FracturedJson یک ابزار رایگان و متن‌باز است که تحت مجوز MIT منتشر شده است. توسعه‌دهندگان می‌توانند از این ابزار برای بهبود خوانایی و نگهداری پروژه‌های خود استفاده کنند. این ابزار برای توسعه‌دهندگانی که با فایل‌های JSON به طور مرتب کار می‌کنند، بسیار مفید خواهد بود.
