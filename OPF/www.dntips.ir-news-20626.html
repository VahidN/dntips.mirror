Ш№ЩҶЩҲШ§ЩҶ:

вҖ«ШҜШіШӘШұШіЫҢ ШўШіШ§ЩҶвҖҢШӘШұ ШЁЩҮ ШЁШ§ШІШӘШ§ШЁ ШЁШ§ [UnsafeAccessorType] ШҜШұ NET 10.


ЩҶЩҲЫҢШіЩҶШҜЩҮ: ЩҲШӯЫҢШҜ ЩҶШөЫҢШұЫҢ
ШӘШ§ШұЫҢШ®: ЫұЫҙЫ°Ыҙ/Ы°Ыё/ЫұЫҙ ЫұЫі:Ы°Ы¶
ШўШҜШұШі: www.dntips.ir
ЪҜШ§ЩҮЫҢ Ш§ЩҲЩӮШ§ШӘ ЩҶЫҢШ§ШІ ШЁЩҮ ШҜШіШӘШұШіЫҢ ШЁЩҮ Ш§Ш№Ш¶Ш§ЫҢ Ш®ШөЩҲШөЫҢ Ъ©Щ„Ш§ШівҖҢЩҮШ§ ЫҢШ§ Ш§ЩҶЩҲШ§Ш№ШҢ Щ…Ш§ЩҶЩҶШҜ ЩҒЫҢЩ„ШҜЩҮШ§ЫҢ ШҜШ§Ш®Щ„ЫҢ ЫҢШ§ Щ…ШӘШҜЩҮШ§ЫҢ ЩҫЩҶЩҮШ§ЩҶШҢ ЩҫЫҢШҙ Щ…ЫҢвҖҢШўЫҢШҜ. Ш§ЫҢЩҶ Ъ©Ш§Шұ Щ…Ш№Щ…ЩҲЩ„Ш§ЩӢ ШЁШұШ§ЫҢ Ш§ЩҮШҜШ§ЩҒ ЩҫЫҢШҙШұЩҒШӘЩҮвҖҢШ§ЫҢ Щ…Ш§ЩҶЩҶШҜ ШӘШіШӘ ЩҲШ§ШӯШҜШҢ instrumentation ЫҢШ§ ШЁЩҮЫҢЩҶЩҮвҖҢШіШ§ШІЫҢ Ш№Щ…Щ„Ъ©ШұШҜ Ш§ЩҶШ¬Ш§Щ… Щ…ЫҢвҖҢШҙЩҲШҜ. ШҜШұ .NET 8ШҢ ЩҲЫҢЪҳЪҜЫҢ [UnsafeAccessor] Щ…Ш№ШұЩҒЫҢ ШҙШҜ ШӘШ§ Ш§ЫҢЩҶ ШҜШіШӘШұШіЫҢ ШұШ§ ШЁШҜЩҲЩҶ ЩҶЫҢШ§ШІ ШЁЩҮ APIЩҮШ§ЫҢ ШЁШ§ШІШӘШ§ШЁ (reflection) ШіЩҶШӘЫҢШҢ ШіШ§ШҜЩҮвҖҢШӘШұ ЩҲ ШіШұЫҢШ№вҖҢШӘШұ Ъ©ЩҶШҜ. ШЁШ§ Ш§ЫҢЩҶ ШӯШ§Щ„ШҢ ШҜШұ .NET 8 ЩҲ 9ШҢ Щ…ШӯШҜЩҲШҜЫҢШӘвҖҢЩҮШ§ЫҢЫҢ ЩҲШ¬ЩҲШҜ ШҜШ§ШҙШӘ Ъ©ЩҮ Щ…Ш§ЩҶШ№ Ш§ШіШӘЩҒШ§ШҜЩҮ Ш§ШІ ШўЩҶ ШҜШұ ШіЩҶШ§ШұЫҢЩҲЩҮШ§ЫҢ ЩҫЫҢЪҶЫҢШҜЩҮ Щ…ЫҢвҖҢШҙШҜ. ШҜШұ Ш§ЫҢЩҶ Щ…ЩӮШ§Щ„ЩҮШҢ ШЁЩҮ ШЁШұШұШіЫҢ ШЁЩҮШЁЩҲШҜЩҮШ§ЫҢ .NET 10 Щ…ЫҢвҖҢЩҫШұШҜШ§ШІЫҢЩ…ШҢ Ш¬Ш§ЫҢЫҢ Ъ©ЩҮ ЩҲЫҢЪҳЪҜЫҢ Ш¬ШҜЫҢШҜ [UnsafeAccessorType] Ш§ЫҢЩҶ Щ…ШӯШҜЩҲШҜЫҢШӘвҖҢЩҮШ§ ШұШ§ ШЁШұШ·ШұЩҒ Щ…ЫҢвҖҢЪ©ЩҶШҜ. Ш§ЫҢЩҶ ШӘШәЫҢЫҢШұШ§ШӘШҢ ШӘЩҲШіШ№ЩҮвҖҢШҜЩҮЩҶШҜЪҜШ§ЩҶ ШұШ§ ЩӮШ§ШҜШұ Щ…ЫҢвҖҢШіШ§ШІШҜ ШӘШ§ ШЁШ§ Ш§ЩҶШ№Ш·Ш§ЩҒ ШЁЫҢШҙШӘШұЫҢ ШЁЩҮ Ш§Ш№Ш¶Ш§ЫҢ Ш®ШөЩҲШөЫҢ ШҜШіШӘШұШіЫҢ ЩҫЫҢШҜШ§ Ъ©ЩҶЩҶШҜШҢ ШЁШҜЩҲЩҶ Ш§ЫҢЩҶЪ©ЩҮ Щ…Ш¬ШЁЩҲШұ ШЁЩҮ ШЁШ§ШІЪҜШҙШӘ ШЁЩҮ ШұЩҲШҙвҖҢЩҮШ§ЫҢ Ъ©ЩҶШҜШӘШұ ШҙЩҲЩҶШҜ. Ш§ЪҜШұ ШЁШ§ .NET ШўШҙЩҶШ§ ЩҮШіШӘЫҢШҜШҢ Ш§ЫҢЩҶ Щ…ЩӮШ§Щ„ЩҮ ШЁЩҮ ШҙЩ…Ш§ Ъ©Щ…Ъ© Щ…ЫҢвҖҢЪ©ЩҶШҜ ШӘШ§ Ш§ШІ Ш§ЫҢЩҶ Ш§ШЁШІШ§ШұЩҮШ§ЫҢ Ш¬ШҜЫҢШҜ ШЁЩҮ Ш·ЩҲШұ Щ…ШӨШ«Шұ Ш§ШіШӘЩҒШ§ШҜЩҮ Ъ©ЩҶЫҢШҜШӣШ§Щ„ШЁШӘЩҮ ШЁШ§ Ш§ШӯШӘЫҢШ§Ш·ШҢ ШІЫҢШұШ§ ШҜШіШӘШұШіЫҢ ШЁЩҮ Ш§Ш№Ш¶Ш§ЫҢ Ш®ШөЩҲШөЫҢ Щ…ЫҢвҖҢШӘЩҲШ§ЩҶШҜ Ъ©ЩҫШіЩҲЩ„ЩҮвҖҢШіШ§ШІЫҢ (encapsulation) ШұШ§ ЩҶЩӮШ¶ Ъ©ЩҶШҜ ЩҲ Ъ©ШҜ ШұШ§ ШҙЪ©ЩҶЩҶШҜЩҮвҖҢШӘШұ ШіШ§ШІШҜ.

ШӘЩҲШ¶ЫҢШӯ Щ…ШіШҰЩ„ЩҮ
ЫҢЪ©ЫҢ Ш§ШІ ЪҶШ§Щ„ШҙвҖҢЩҮШ§ЫҢ Ш§ШөЩ„ЫҢ ШҜШұ Ъ©Ш§Шұ ШЁШ§ Ш§ЩҶЩҲШ§Ш№ ШҜШұ .NETШҢ ШҜШіШӘШұШіЫҢ ШЁЩҮ Ш§Ш№Ш¶Ш§ЫҢ Ш®ШөЩҲШөЫҢ ЫҢШ§ ШҜШ§Ш®Щ„ЫҢ (internal) Ш§ШіШӘ Ъ©ЩҮ visibility ШўЩҶвҖҢЩҮШ§ Щ…ШӯШҜЩҲШҜ ШҙШҜЩҮ. ШЁШұШ§ЫҢ Щ…Ш«Ш§Щ„ШҢ ЩҒШұШ¶ Ъ©ЩҶЫҢШҜ ШЁШ§ ЫҢЪ© Ъ©Щ„Ш§Ші Ш№Щ…ЩҲЩ…ЫҢ (PublicClass) Ъ©Ш§Шұ Щ…ЫҢвҖҢЪ©ЩҶЫҢШҜ Ъ©ЩҮ ЫҢЪ© ЩҒЫҢЩ„ШҜ Ш®ШөЩҲШөЫҢ Ш§ШІ ЩҶЩҲШ№ ШҜШ§Ш®Щ„ЫҢ (PrivateClass) ШҜШ§ШұШҜ. ШҜШұ .NET 8 ЩҲ 9ШҢ ШЁШ§ Ш§ШіШӘЩҒШ§ШҜЩҮ Ш§ШІ [UnsafeAccessor]ШҢ ШЁШ§ЫҢШҜ ШЁШӘЩҲШ§ЩҶЫҢШҜ Щ…ШіШӘЩӮЫҢЩ…Ш§ЩӢ ШЁЩҮ ШӘЩ…Ш§Щ… Ш§ЩҶЩҲШ§Ш№ ШҜШұЪҜЫҢШұ ШҜШұ ШіЫҢЪҜЩҶЪҶШұ Щ…ШӘШҜ Ш§ШұШ¬Ш§Ш№ ШҜЩҮЫҢШҜ. Ш§ЪҜШұ ЩҶЩҲШ№ PrivateClass ШҜШ§Ш®Щ„ЫҢ ШЁШ§ШҙШҜ ЩҲ ЩҶШӘЩҲШ§ЩҶЫҢШҜ ШЁЩҮ ШўЩҶ Ш§ШұШ¬Ш§Ш№ Щ…ШіШӘЩӮЫҢЩ… ШҜЩҮЫҢШҜШҢ [UnsafeAccessor] Ъ©Ш§Шұ ЩҶЩ…ЫҢвҖҢЪ©ЩҶШҜ.
Ш§ЫҢЩҶ Щ…ШӯШҜЩҲШҜЫҢШӘ ШҜЩҲ ШіЩҶШ§ШұЫҢЩҲ Ш§ШөЩ„ЫҢ Ш§ЫҢШ¬Ш§ШҜ Щ…ЫҢвҖҢЪ©ЩҶШҜ:
  • Щ…ШӯШҜЩҲШҜЫҢШӘ visibility: Ш§ЩҶЩҲШ§Ш№ ШҜШ§Ш®Щ„ЫҢ ЫҢШ§ Ш®ШөЩҲШөЫҢ Ъ©ЩҮ ЩҶЩ…ЫҢвҖҢШӘЩҲШ§ЩҶЫҢШҜ ШҜШұ Ъ©ШҜ Ш®ЩҲШҜ import Ъ©ЩҶЫҢШҜ.
  • Щ…ШӯШҜЩҲШҜЫҢШӘ Ъ©Ш§Щ…ЩҫШ§ЫҢЩ„вҖҢШӘШ§ЫҢЩ…: ШІЩ…Ш§ЩҶЫҢ Ъ©ЩҮ Ш§ЩҶЩҲШ§Ш№ ШҜШұ Щ„Ш§ЫҢШЁШұШұЫҢвҖҢЩҮШ§ЫҢ ШҜЫҢЪҜШұ ЩҮШіШӘЩҶШҜ ЩҲ ШЁЩҮ ШҜЩ„ЫҢЩ„ ЩҲШ§ШЁШіШӘЪҜЫҢвҖҢЩҮШ§ЫҢ ЪҶШұШ®ЩҮвҖҢШ§ЫҢ (circular dependencies) ЫҢШ§ Щ…ШіШ§ШҰЩ„ ЩҶШіШ®ЩҮвҖҢШЁЩҶШҜЫҢШҢ ЩҶЩ…ЫҢвҖҢШӘЩҲШ§ЩҶЫҢШҜ ШЁЩҮ ШўЩҶвҖҢЩҮШ§ Ш§ШұШ¬Ш§Ш№ ШҜЩҮЫҢШҜ. Щ…Ш«Щ„Ш§ЩӢ ШҜШұ Щ„Ш§ЫҢШЁШұШұЫҢвҖҢЩҮШ§ЫҢ instrumentation Щ…Ш§ЩҶЩҶШҜ DatadogШҢ ЫҢШ§ ШӯШӘЫҢ ШҜШұ Ш®ЩҲШҜ runtime .NET ШЁШұШ§ЫҢ Щ„Ш§ЫҢШЁШұШұЫҢвҖҢЩҮШ§ЫҢ HTTP ЩҲ Cryptography.
Ш№Щ„Ш§ЩҲЩҮ ШЁШұ Ш§ЫҢЩҶШҢ Ъ©Ш§Шұ ШЁШ§ Ш§ЩҶЩҲШ§Ш№ ЪҳЩҶШұЫҢЪ© ЩҫЫҢЪҶЫҢШҜЩҮвҖҢШӘШұ Щ…ЫҢвҖҢШҙЩҲШҜШҢ ШІЫҢШұШ§ ШЁШ§ЫҢШҜ ШіЫҢЪҜЩҶЪҶШұ ШҜЩӮЫҢЩӮ ШұШ§ ШӯЩҒШё Ъ©ЩҶЫҢШҜ. ШЁШҜЩҲЩҶ ШӯЩ„ Ш§ЫҢЩҶ Щ…ШіШ§ШҰЩ„ШҢ ШӘЩҲШіШ№ЩҮвҖҢШҜЩҮЩҶШҜЪҜШ§ЩҶ Щ…Ш¬ШЁЩҲШұ ШЁЩҮ Ш§ШіШӘЩҒШ§ШҜЩҮ Ш§ШІ reflection ШіЩҶШӘЫҢШҢ System.Reflection.Emit ЫҢШ§ System.Linq.Expressions Щ…ЫҢвҖҢШҙЩҲЩҶШҜ Ъ©ЩҮ ЩҶЩҮ ШӘЩҶЩҮШ§ Ъ©ЩҶШҜШӘШұ ЩҮШіШӘЩҶШҜ (ШӘШ§ ЪҶЩҶШҜЫҢЩҶ ШЁШұШ§ШЁШұ)ШҢ ШЁЩ„Ъ©ЩҮ Ъ©ШҜ ШұШ§ ЩҫЫҢЪҶЫҢШҜЩҮвҖҢШӘШұ Щ…ЫҢвҖҢЪ©ЩҶЩҶШҜ. Ш§ЫҢЩҶ Ъ©ЩҶШҜЫҢ Щ…ЫҢвҖҢШӘЩҲШ§ЩҶШҜ ШҜШұ ШЁШұЩҶШ§Щ…ЩҮвҖҢЩҮШ§ЫҢ Ш№Щ…Щ„Ъ©ШұШҜЩ…ШӯЩҲШұШҢ Щ…Ш§ЩҶЩҶШҜ ШіШұЩҲЫҢШівҖҢЩҮШ§ЫҢ ЩҲШЁ ЫҢШ§ Ш§ШЁШІШ§ШұЩҮШ§ЫҢ Щ…Ш§ЩҶЫҢШӘЩҲШұЫҢЩҶЪҜШҢ Щ…ШҙЪ©Щ„вҖҢШіШ§ШІ ШЁШ§ШҙШҜ.

ШұШ§ЩҮвҖҢШӯЩ„ ЩӮШҜЫҢЩ…ЫҢ: Ш§ШіШӘЩҒШ§ШҜЩҮ Ш§ШІ [UnsafeAccessor] ШҜШұ .NET 8 ЩҲ 9
ШҜШұ .NET 8ШҢ [UnsafeAccessor] ШЁЩҮ Ш№ЩҶЩҲШ§ЩҶ Ш¬Ш§ЫҢЪҜШІЫҢЩҶЫҢ ШіШұЫҢШ№вҖҢШӘШұ ШЁШұШ§ЫҢ reflection Щ…Ш№ШұЩҒЫҢ ШҙШҜ ЩҲ ШҜШұ .NET 9 ШЁШ§ ЩҫШҙШӘЫҢШЁШ§ЩҶЫҢ Ш§ШІ ЪҳЩҶШұЫҢЪ©вҖҢЩҮШ§ ШЁЩҮШЁЩҲШҜ ЫҢШ§ЩҒШӘ. Ш§ЫҢЩҶ ЩҲЫҢЪҳЪҜЫҢ Ш§Ш¬Ш§ШІЩҮ Щ…ЫҢвҖҢШҜЩҮШҜ ШӘШ§ ЩҒЫҢЩ„ШҜЩҮШ§ШҢ Щ…ШӘШҜЩҮШ§ШҢ ШіШ§ШІЩҶШҜЩҮвҖҢЩҮШ§ ЩҲ Ш§Ш№Ш¶Ш§ЫҢ Ш§ШіШӘШ§ШӘЫҢЪ© ШұШ§ ШЁШҜЩҲЩҶ reflection ЩҒШұШ§Ш®ЩҲШ§ЩҶЫҢ Ъ©ЩҶЫҢШҜ. ШЁШұШ§ЫҢ Щ…Ш«Ш§Щ„ШҢ ШҜШіШӘШұШіЫҢ ШЁЩҮ ЩҒЫҢЩ„ШҜ Ш®ШөЩҲШөЫҢ _items ШҜШұ Ъ©Щ„Ш§Ші List<T>:
ШЁШ§ reflection ШіЩҶШӘЫҢ:
var itemsFieldInfo = typeof(List<int>).GetField("_items", BindingFlags.NonPublic | BindingFlags.Instance);
var list = new List<int>(16);
var items = (int[])itemsFieldInfo.GetValue(list);
Console.WriteLine($"{items.Length} items"); // Ш®ШұЩҲШ¬ЫҢ: 16 items
ШӯШ§Щ„Ш§ ШЁШ§ [UnsafeAccessor]:
var list = new List<int>(16);
int[] items = Accessors<int>.GetItems(list);
Console.WriteLine($"{items.Length} items"); // Ш®ШұЩҲШ¬ЫҢ: 16 items

static class Accessors<T>
{
    [UnsafeAccessor(UnsafeAccessorKind.Field, Name = "_items")]
    public static extern ref T[] GetItems(List<T> list);
}
ШӘЩҲШ¬ЩҮ Ъ©ЩҶЫҢШҜ Ъ©ЩҮ ШЁШұШ§ЫҢ Ш§ЩҶЩҲШ§Ш№ ЪҳЩҶШұЫҢЪ© Щ…Ш§ЩҶЩҶШҜ List<T>ШҢ ШЁШ§ЫҢШҜ Ш§ШІ ЫҢЪ© Ъ©Щ„Ш§Ші Ъ©Ш§ЩҶШӘЫҢЩҶШұ Ш§ШіШӘЩҒШ§ШҜЩҮ Ъ©ЩҶЫҢШҜ ЩҲ Щ…ШӘШҜ ШұШ§ extern ШӘШ№ШұЫҢЩҒ Ъ©ЩҶЫҢШҜ. Ш§ЫҢЩҶ ШұЩҲШҙ ЩҶЩҮ ШӘЩҶЩҮШ§ Ш®ЩҲШ§ЩҶШ§ШӘШұ Ш§ШіШӘШҢ ШЁЩ„Ъ©ЩҮ ШіШұЫҢШ№вҖҢШӘШұ Ш§Ш¬ШұШ§ Щ…ЫҢвҖҢШҙЩҲШҜ ШІЫҢШұШ§ runtime Щ…ШіШӘЩӮЫҢЩ…Ш§ЩӢ ШЁЩҮ Ш№Ш¶ЩҲ ШҜШіШӘШұШіЫҢ ЩҫЫҢШҜШ§ Щ…ЫҢвҖҢЪ©ЩҶШҜ.

ШЁШұШ§ЫҢ Щ…ШӘШҜЩҮШ§ЫҢ Ш§ШіШӘШ§ШӘЫҢЪ©ШҢ Щ…Ш«Щ„Ш§ЩӢ IsCompatibleObject ШҜШұList<T>:
bool isCompat = Accessors<int?>.IsCompatibleObject(null, 123); // true

static class Accessors<T>
{
    [UnsafeAccessor(UnsafeAccessorKind.StaticMethod, Name = "IsCompatibleObject")]
    public static extern bool IsCompatibleObject(List<T> instance, object? value);
}
Ш§ЫҢЩҶ ШұЩҲШҙ ШЁШұШ§ЫҢ ЩҒЫҢЩ„ШҜЩҮШ§ ref ШЁШұЩ…ЫҢвҖҢЪҜШұШҜШ§ЩҶШҜШҢ ЩҫШі Щ…ЫҢвҖҢШӘЩҲШ§ЩҶЫҢШҜ ЩҒЫҢЩ„ШҜ ШұШ§ ШӘШәЫҢЫҢШұ ШҜЩҮЫҢШҜ:
var list = new List<int>(16);
Accessors<int>.GetItems(list) = Array.Empty<int>();
Console.WriteLine($"Capacity: {list.Capacity}"); // Ш®ШұЩҲШ¬ЫҢ: 0
ШЁШ§ Ш§ЫҢЩҶ ШӯШ§Щ„ШҢ ЩҮЩ…Ш§ЩҶвҖҢШ·ЩҲШұ Ъ©ЩҮ ЪҜЩҒШӘЫҢЩ…ШҢ Ш§ЪҜШұ ЩҶШӘЩҲШ§ЩҶЫҢШҜ ШЁЩҮ Ш§ЩҶЩҲШ§Ш№ Ш§ШұШ¬Ш§Ш№ ШҜЩҮЫҢШҜ (Щ…Ш«Щ„ PrivateClass ШҜШ§Ш®Щ„ЫҢ)ШҢ ШЁШ§ЫҢШҜ ШЁЩҮ reflection ШЁШұЪҜШұШҜЫҢШҜШӣ Ъ©ЩҮ ЩҶЩӮШ·ЩҮ Ш¶Ш№ЩҒ Ш§ШөЩ„ЫҢ Ш§ШіШӘ.

ШұШ§ЩҮвҖҢШӯЩ„ Ш¬ШҜЫҢШҜ: [UnsafeAccessorType] ШҜШұ .NET 10
.NET 10 ШЁШ§ Щ…Ш№ШұЩҒЫҢ [UnsafeAccessorType]ШҢ Ш§ЫҢЩҶ Щ…ШӯШҜЩҲШҜЫҢШӘ ШұШ§ ШЁШұШ·ШұЩҒ Щ…ЫҢвҖҢЪ©ЩҶШҜ. Ш§ЫҢЩҶ attribute Ш§Ш¬Ш§ШІЩҮ Щ…ЫҢвҖҢШҜЩҮШҜ ШӘШ§ Ш§ЩҶЩҲШ§Ш№ ШұШ§ ШЁЩҮ ШөЩҲШұШӘ string Щ…ШҙШ®Шө Ъ©ЩҶЫҢШҜШҢ ШЁШҜЩҲЩҶ ЩҶЫҢШ§ШІ ШЁЩҮ Ш§ШұШ¬Ш§Ш№ Щ…ШіШӘЩӮЫҢЩ…. Ш§ЫҢЩҶ Ъ©Ш§Шұ ЩҮШұ ШҜЩҲ ШіЩҶШ§ШұЫҢЩҲ visibility ЩҲ Ъ©Ш§Щ…ЩҫШ§ЫҢЩ„вҖҢШӘШ§ЫҢЩ… ШұШ§ ШӯЩ„ Щ…ЫҢвҖҢЪ©ЩҶШҜ.

Щ…Ш«Ш§Щ„: ШҜШіШӘШұШіЫҢ ШЁЩҮ SomeValue ШҜШұ PrivateClass Ш§ШІ Ш·ШұЫҢЩӮ PublicClass:
public class PublicClass
{
    private readonly PrivateClass _private = new("Hello world!");
    internal PrivateClass GetPrivate() => _private;
}

internal class PrivateClass(string someValue)
{
    internal string SomeValue { get; } = someValue;
}
accessorЩҮШ§:
[UnsafeAccessor(UnsafeAccessorKind.Method, Name = "GetPrivate")]
[return: UnsafeAccessorType("PrivateClass")]
static extern object GetByMethod(PublicClass instance);

[UnsafeAccessor(UnsafeAccessorKind.Method, Name = "get_SomeValue")]
static extern string GetSomeValue([UnsafeAccessorType("PrivateClass")] object instance);
Ш§ШіШӘЩҒШ§ШҜЩҮ:
var publicClass = new PublicClass();
object privateClass = GetByMethod(publicClass);
string value = GetSomeValue(privateClass);
Console.WriteLine(value); // Ш®ШұЩҲШ¬ЫҢ: Hello world!
ЩҶШ§Щ… ЩҶЩҲШ№ ШЁШ§ЫҢШҜ fully qualified ШЁШ§ШҙШҜШҢ Щ…Ш§ЩҶЩҶШҜ вҖңPrivateLib.Class1, PrivateLibвҖқ ШЁШұШ§ЫҢ assembly-qualified. ШЁШұШ§ЫҢ ЪҳЩҶШұЫҢЪ©вҖҢЩҮШ§ШҢ Ш§ШІ ЩҒШұЩ…ШӘвҖҢЩҮШ§ЫҢЫҢ Щ…Ш§ЩҶЩҶШҜ вҖңList`1[[!0]]вҖқ Ш§ШіШӘЩҒШ§ШҜЩҮ Ъ©ЩҶЫҢШҜ (!0 ШЁШұШ§ЫҢ ЩҫШ§ШұШ§Щ…ШӘШұЩҮШ§ЫҢ ЩҶЩҲШ№ ШЁШ§ШІ).

Щ…Ш«Ш§Щ„ ЩҫЫҢЪҶЫҢШҜЩҮвҖҢШӘШұ ШЁШ§ ЪҳЩҶШұЫҢЪ©вҖҢЩҮШ§ ЩҲ constraints:
[UnsafeAccessor(UnsafeAccessorKind.Method, Name = "GenericWithConstraints")]
public extern static bool CallGenericClassGenericWithConstraints<V, W>(
    [UnsafeAccessorType("PrivateLib.GenericClass`1[[!0]], PrivateLib")] object tgt,
    [UnsafeAccessorType("System.Collections.Generic.List`1[[!0]]")] object a,
    [UnsafeAccessorType("System.Collections.Generic.List`1[[!!0]]")] object b,
    List<W> c,
    [UnsafeAccessorType("System.Collections.Generic.List`1[[PrivateLib.Class1, PrivateLib]]")] object d) where W : T;
Ш§ЫҢЩҶ ШұЩҲШҙ ЩҮЩ…ЪҶЩҶШ§ЩҶ ШіШұЫҢШ№ Ш§ШіШӘ ЩҲ Ш№Щ…Щ„Ъ©ШұШҜ ШЁЩҮШӘШұЫҢ ЩҶШіШЁШӘ ШЁЩҮ reflection ШҜШ§ШұШҜ. ЩҶЪ©ШӘЩҮ Ш§Ш¶Ш§ЩҒЫҢ: ШЁШұШ§ЫҢ Ш§Щ…ЩҶЫҢШӘ ШЁЫҢШҙШӘШұШҢ ЩҮЩ…ЫҢШҙЩҮ ШЁШұШұШіЫҢ Ъ©ЩҶЫҢШҜ Ъ©ЩҮ ШўЫҢШ§ Ш§ШіШӘЩҒШ§ШҜЩҮ Ш§ШІ Ш§ЫҢЩҶ ЩҲЫҢЪҳЪҜЫҢ Ш¶ШұЩҲШұЫҢ Ш§ШіШӘШҢ ШІЫҢШұШ§ Щ…ЫҢвҖҢШӘЩҲШ§ЩҶШҜ ШЁЩҮ ЩҲШ§ШЁШіШӘЪҜЫҢвҖҢЩҮШ§ЫҢ ЩҶШ§Ш®ЩҲШ§ШіШӘЩҮ Щ…ЩҶШ¬Шұ ШҙЩҲШҜ. ЩҮЩ…ЪҶЩҶЫҢЩҶШҢ ШҜШұ Щ…ШӯЫҢШ·вҖҢЩҮШ§ЫҢ ШӘЩҲШІЫҢШ№вҖҢШҙШҜЩҮШҢ Ш§Ш·Щ…ЫҢЩҶШ§ЩҶ ШӯШ§ШөЩ„ Ъ©ЩҶЫҢШҜ Ъ©ЩҮ assemblyЩҮШ§ ШҜШұ ШҜШіШӘШұШі ШЁШ§ШҙЩҶШҜ.

ШЁШ§ Ш§ЫҢЩҶ ШӯШ§Щ„ШҢ Щ…ШӯШҜЩҲШҜЫҢШӘвҖҢЩҮШ§ЫҢЫҢ ШЁШ§ЩӮЫҢ Щ…Ш§ЩҶШҜЩҮ:
  • ЩҶЩ…ЫҢвҖҢШӘЩҲШ§ЩҶЫҢШҜ ШЁШұШ§ЫҢ Ш§ЩҶЩҲШ§Ш№ ЪҳЩҶШұЫҢЪ© Ъ©ЩҮ ЩҫШ§ШұШ§Щ…ШӘШұ ЩҶЩҲШ№ ШәЫҢШұЩӮШ§ШЁЩ„ Ш§ШұШ¬Ш§Ш№ Ш§ШіШӘШҢ instance ШЁШіШ§ШІЫҢШҜ.
  • ЩҒЫҢЩ„ШҜЩҮШ§ ЫҢШ§ Щ…ШӘШҜЩҮШ§ЫҢ ref-returning Ъ©ЩҮ ЩҶЩҲШ№ШҙШ§ЩҶ ЩҶЫҢШ§ШІ ШЁЩҮ [UnsafeAccessorType] ШҜШ§ШұШҜШҢ ЩҫШҙШӘЫҢШЁШ§ЩҶЫҢ ЩҶЩ…ЫҢвҖҢШҙЩҲЩҶШҜ (Ш®Ш·Ш§ЫҢ NotSupportedException).

ЩҶШӘЫҢШ¬ЩҮвҖҢЪҜЫҢШұЫҢ
ЩҲЫҢЪҳЪҜЫҢ [UnsafeAccessor] ШҜШұ .NET 8 ЩҲ 9ШҢ ШҜШіШӘШұШіЫҢ ШЁЩҮ Ш§Ш№Ш¶Ш§ЫҢ Ш®ШөЩҲШөЫҢ ШұШ§ ШіШ§ШҜЩҮвҖҢШӘШұ Ъ©ШұШҜШҢ Ш§Щ…Ш§ Щ…ШӯШҜЩҲШҜЫҢШӘвҖҢЩҮШ§ЫҢ Ш§ШұШ¬Ш§Ш№ ШЁЩҮ Ш§ЩҶЩҲШ§Ш№ШҢ ШўЩҶ ШұШ§ Щ…ШӯШҜЩҲШҜ Щ…ЫҢвҖҢЪ©ШұШҜ. ШЁШ§ [UnsafeAccessorType] ШҜШұ .NET 10ШҢ ШӯШ§Щ„Ш§ Щ…ЫҢвҖҢШӘЩҲШ§ЩҶЫҢШҜ ШӯШӘЫҢ ШЁЩҮ Ш§ЩҶЩҲШ§Ш№ ШәЫҢШұЩӮШ§ШЁЩ„ Ш§ШұШ¬Ш§Ш№ ШҜШіШӘШұШіЫҢ ЩҫЫҢШҜШ§ Ъ©ЩҶЫҢШҜШҢ Ъ©ЩҮ Ш§ЫҢЩҶ ЩҲЫҢЪҳЪҜЫҢ ШұШ§ ШЁШұШ§ЫҢ ШіЩҶШ§ШұЫҢЩҲЩҮШ§ЫҢ ЩҫЫҢШҙШұЩҒШӘЩҮ Щ…Ш§ЩҶЩҶШҜ instrumentation ЫҢШ§ ШӯЩ„ ЩҲШ§ШЁШіШӘЪҜЫҢвҖҢЩҮШ§ЫҢ ЩҫЫҢЪҶЫҢШҜЩҮШҢ Ш§ЫҢШҜЩҮвҖҢШўЩ„ Щ…ЫҢвҖҢШіШ§ШІШҜ. Ш§ЫҢЩҶ ШЁЩҮШЁЩҲШҜ ЩҶЩҮ ШӘЩҶЩҮШ§ Ъ©ШҜ ШұШ§ Ш®ЩҲШ§ЩҶШ§ШӘШұ ЩҶЪҜЩҮ Щ…ЫҢвҖҢШҜШ§ШұШҜШҢ ШЁЩ„Ъ©ЩҮ Ш№Щ…Щ„Ъ©ШұШҜ ШұШ§ ШӯЩҒШё Щ…ЫҢвҖҢЪ©ЩҶШҜ. ШЁШ§ Ш§ЫҢЩҶ ШӯШ§Щ„ШҢ ЩҮЩ…ЫҢШҙЩҮ ШЁШ§ Ш§ШӯШӘЫҢШ§Ш· Ш§ШіШӘЩҒШ§ШҜЩҮ Ъ©ЩҶЫҢШҜ ШӘШ§ Ъ©ЩҫШіЩҲЩ„ЩҮвҖҢШіШ§ШІЫҢ ШұШ§ ЩҶЩӮШ¶ ЩҶЪ©ЩҶЫҢШҜ.


Щ…ШҙШ§ЩҮШҜЩҮ Щ…Ш·Щ„ШЁ Ш§ШөЩ„ЫҢ
Щ…Ш·Ш§Щ„ШЁ Щ…ШҙШ§ШЁЩҮ
    Ш§ШҙШӘШұШ§Ъ©вҖҢЩҮШ§Щ…Ш№ШұЩҒЫҢ UnsafeAccessor ШҜШұ .NET ШЁШұШ§ЫҢ ШҜШіШӘШұШіЫҢ ШіШұЫҢШ№вҖҢШӘШұ ШЁЩҮ Ш§Ш№Ш¶Ш§ЫҢ Ш®ШөЩҲШөЫҢШ§ШҙШӘШұШ§Ъ©вҖҢЩҮШ§Щ…Ш№ШұЩҒЫҢ ШұШ§ЩҮ Ш¬ШҜЫҢШҜЫҢ ШЁШұШ§ЫҢ Reflection ШҜШұ ШҜШ§ШӘвҖҢЩҶШӘ Ыё Ш§ШҙШӘШұШ§Ъ©вҖҢЩҮШ§Щ…ЩӮШ§ЫҢШіЩҮ ЩҫЫҢШ§ШҜЩҮвҖҢШіШ§ШІЫҢ Ш¬ЩҶШұЫҢЪ©вҖҢЩҮШ§ ШҜШұ ШіЫҢвҖҢШҙШ§ШұЩҫ ЩҲ Ш¬Ш§ЩҲШ§Ш§ШҙШӘШұШ§Ъ©вҖҢЩҮШ§ШұЩҲШҙвҖҢЩҮШ§ЫҢЫҢ ШЁШұШ§ЫҢ ШЁЩҮЫҢЩҶЩҮвҖҢШіШ§ШІЫҢ Ш§ШіШӘЩҒШ§ШҜЩҮ Ш§ШІ Reflection ШҜШұ #CШ§ШҙШӘШұШ§Ъ©вҖҢЩҮШ§ЩҲЫҢЪҳЪҜЫҢ Ш¬ШҜЫҢШҜ Extension Members ШҜШұ C# 14.0Ш§ШҙШӘШұШ§Ъ©вҖҢЩҮШ§ШЁШұШұШіЫҢ Ш№Щ…ЫҢЩӮ Ш§Ш№Ш¶Ш§ЫҢ Ш§Щ„ШӯШ§ЩӮЫҢ ШҜШұ C# 14Щ…Ш·Ш§Щ„ШЁStatic ReflectionЩ…Ш·Ш§Щ„ШЁШ§ЫҢШ¬Ш§ШҜ Щ„ЫҢШіШӘЫҢ Ш§ШІ Ъ©Щ„Ш§ШіЫҢ Ш¬ЩҶШұЫҢЪ©Щ…Ш·Ш§Щ„ШЁШЁЩҮШЁЩҲШҜ Ъ©Ш§ШұШўЫҢЫҢ Reflection ШҜШұ ШҜШ§ШӘ ЩҶШӘ 7Щ…Ш·Ш§Щ„ШЁШ§ШіЩ…ШЁЩ„ЫҢвҖҢЩҮШ§ЫҢ ШҜЩҲШіШӘ (Friend Assembly)
