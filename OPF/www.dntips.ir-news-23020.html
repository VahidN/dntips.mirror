‫راهنمای استفاده پیشرونده از PicoServer (نسخه C#)


نویسنده: راویAI
تاریخ: ۱۴۰۴/۱۱/۰۸ ۰۶:۰۷
آدرس: www.dntips.ir
این مقاله راهنمای استفاده از PicoServer به زبان C# را ارائه می‌دهد و به توسعه‌دهندگان کمک می‌کند تا به سرعت یک وب API ساده راه‌اندازی کنند. PicoServer یک سرور سبک و سریع است که برای ساخت برنامه‌های وب ساده و APIها مناسب است. در این راهنما، ابتدا به راه‌اندازی سرور و شروع کار با استفاده از کلاس `WebAPIServer` و متد `StartServer` پرداخته می‌شود. همچنین، نحوه ثبت مسیرها (Routes) با استفاده از متد `AddRoute` و لیست سفید (RouteWhitelist) برای دور زدن پردازش میانی‌افزارها توضیح داده شده است. سپس، سه روش برای نگاشت مسیرها (Mapping Routes) معرفی می‌شود: روش Mapping، تابع بی‌نام (Anonymous Function) و Lambda. روش Mapping به دلیل وضوح و سهولت نگهداری، توصیه می‌شود. در این روش، با استفاده از متد `AddRoute`، مسیرها و توابع مربوطه به سرور اضافه می‌شوند. در نهایت، یک مثال ساده از نحوه پاسخگویی به درخواست‌های HTTP با استفاده از متد `hello` و دریافت پارامترهای URL با استفاده از `request.GetQuery` ارائه شده است. این مثال نشان می‌دهد که چگونه با استفاده از C# raw string literals، ساختار JSON مناسب را برای پاسخ به درخواست‌ها ایجاد کرد.
