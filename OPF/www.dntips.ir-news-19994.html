عنوان:

‫استفاده از NET. در لینوکس


نویسنده: وحید نصیری
تاریخ: ۱۴۰۳/۱۱/۲۳ ۰۸:۳۶
آدرس: www.dntips.ir
این پست وبلاگ تجربه نویسنده در استفاده از NET. در لینوکس را شرح می‌دهد و ابزارها و فرآیندهای مرتبط را برجسته می‌کند.

نکات کلیدی

  • NET. به طور یکپارچه روی توزیع‌های مختلف لینوکس کار می‌کند.
  • نویسنده لینوکس را برای توسعه NET. به ویندوز ترجیح می‌دهد.
  • نصب دستی SDK .NET برای کنترل بهتر نگارش‌های آن، ترجیح داده می‌شود.
  • اکثر بسته‌های NuGet و کامپایل AOT روی لینوکس کار می‌کنند.
  • Maui تنها ویژگی قابل توجهی است که وجود ندارد.

ابزارهای توسعه

  • IDEها: VS Code (اولیه)، Rider
  • ابزارهای پایگاه داده: DBeaver، DataGrip (برای MS SQL Server، Postgres، CockroachDB)
  • کنترل منبع: Git، GitHub CLI

سایر ابزارها و فناوری‌ها

  • پایگاه‌های داده: تصاویر Docker برای MySQL، Postgres، MariaDB، MS SQL Server، RabbitMQ، Redis، Kafka
  • پاورشل: PowerShell Core با تاریخچه دستور و تکمیل اضافه شده
  • سایر موارد: Fiddler، Postman، Firefox، Chrome، Chromium، Edge، Brave، Opera، LibreOffice، Audacity، Inkscape، GIMP

نرم‌افزار‌های گم‌شده (که از ویندوز برای آن‌ها استفاده می‌شود)

  • آفیس 365 (پاورپوینت)
  • مجموعه CorelDraw
  • LightRoom
