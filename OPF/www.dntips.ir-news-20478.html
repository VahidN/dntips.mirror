ШӘЩ„Ш§Шҙ ШЁШұШ§ЫҢ ЩҶЩҲШҙШӘЩҶ Ъ©ШҜЫҢ ШӘЩ…ЫҢШІШҢ ЩӮШ§ШЁЩ„ ЩҶЪҜЩҮШҜШ§ШұЫҢ ЩҲ ШЁШ§ Ъ©Ш§ШұШ§ЫҢЫҢ ШЁШ§Щ„Ш§ ЩҮЩ…ЩҲШ§ШұЩҮ Ш§ШІ Ш§ЩҮШҜШ§ЩҒ Ш§ШөЩ„ЫҢ ШЁШұЩҶШ§Щ…ЩҮвҖҢЩҶЩҲЫҢШіШ§ЩҶ ШЁЩҲШҜЩҮ Ш§ШіШӘ. ШЁШ§ Ш§ЫҢЩҶ ШӯШ§Щ„ШҢ ЪҜШ§ЩҮЫҢ Ш§ШіШӘЩҒШ§ШҜЩҮ ЩҶШ§ШҜШұШіШӘ Ш§ШІ ШЁШұШ®ЫҢ ШіШ§ШІЩҲЪ©Ш§ШұЩҮШ§ЫҢ ШІШЁШ§ЩҶ ШЁШұЩҶШ§Щ…ЩҮвҖҢЩҶЩҲЫҢШіЫҢ Щ…ЫҢвҖҢШӘЩҲШ§ЩҶШҜ Щ…ЩҶШ¬Шұ ШЁЩҮ ШЁШұЩҲШІ Ш¶ШҜШ§Щ„ЪҜЩҲЩҮШ§ЫҢЫҢ (Anti-Patterns) ШҙЩҲШҜ Ъ©ЩҮ Ъ©ЫҢЩҒЫҢШӘ Ъ©ШҜ ШұШ§ ШЁЩҮ ШҙШҜШӘ ШӘШӯШӘ ШӘШЈШ«ЫҢШұ ЩӮШұШ§Шұ Щ…ЫҢвҖҢШҜЩҮЩҶШҜ. ЫҢЪ©ЫҢ Ш§ШІ Ш§ЫҢЩҶ Ш¶ШҜШ§Щ„ЪҜЩҲЩҮШ§ЫҢ ШұШ§ЫҢШ¬ШҢ В«ШӘЩҲШіШ№ЩҮ Щ…ШЁШӘЩҶЫҢ ШЁШұ Ш§ШіШӘШ«ЩҶШ§ШЎВ» (Exception-Driven Development ЫҢШ§ EDD) Ш§ШіШӘ. Ш§ЫҢЩҶ ШұЩҲЫҢЪ©ШұШҜШҢ Ш§ШіШӘШ«ЩҶШ§ЩҮШ§ (Exceptions) ШұШ§ Ш§ШІ ШӯШ§Щ„ШӘ ЩҮШҙШҜШ§ШұЩҮШ§ЫҢ ЩҶШ§ШҜШұ ШЁЩҮ ЪҶШұШ§ШәвҖҢЩҮШ§ЫҢ ШұШ§ЩҮЩҶЩ…Ш§ЫҢЫҢ ШұЩҲШІЩ…ШұЩҮ ШӘШЁШҜЫҢЩ„ Щ…ЫҢвҖҢЪ©ЩҶШҜ Ъ©ЩҮ Щ…ЩҶШ¬Шұ ШЁЩҮ Ъ©ЩҶШҜЫҢ Ъ©ШҜШҢ ЩҫЫҢЪҶЫҢШҜЪҜЫҢ Щ…ЩҶШ·ЩӮ ЩҲ ШҜЩҲШұ ШҙШҜЩҶ Ш§ШІ ЩҶШұЩ…вҖҢШ§ЩҒШІШ§ШұЫҢ ЩҫШ§Ъ© ЩҲ ЩӮШ§ШЁЩ„ ЩҶЪҜЩҮШҜШ§ШұЫҢ Щ…ЫҢвҖҢШҙЩҲШҜ. ШҜШұ Ш§ЫҢЩҶ Щ…ЩӮШ§Щ„ЩҮШҢ ШЁЩҮ ШЁШұШұШіЫҢ Ш№Щ…ЫҢЩӮ Ш§ЫҢЩҶ Ш¶ШҜШ§Щ„ЪҜЩҲШҢ ШҜЩ„Ш§ЫҢЩ„ Щ…Ш¶Шұ ШЁЩҲШҜЩҶ ШўЩҶ ЩҲ Ш§ШұШ§ШҰЩҮ ШұШ§ЩҮЪ©Ш§ШұЩҮШ§ЫҢ ШЁЩҮШӘШұ ШЁШ§ ШӘЩ…ШұЪ©ШІ ШЁШұ ШӘЩҲШіШ№ЩҮвҖҢШҜЩҮЩҶШҜЪҜШ§ЩҶ .NET Ш®ЩҲШ§ЩҮЫҢЩ… ЩҫШұШҜШ§Ш®ШӘ.

ШӘЩҲШіШ№ЩҮ Щ…ШЁШӘЩҶЫҢ ШЁШұ Ш§ШіШӘШ«ЩҶШ§ШЎ (Exception-Driven Development) ЪҶЫҢШіШӘШҹ
ШӘЩҲШіШ№ЩҮ Щ…ШЁШӘЩҶЫҢ ШЁШұ Ш§ШіШӘШ«ЩҶШ§ШЎ (EDD) ЫҢЪ© ШұЩҲЫҢЪ©ШұШҜ ШЁШұЩҶШ§Щ…ЩҮвҖҢЩҶЩҲЫҢШіЫҢ Ш§ШіШӘ Ъ©ЩҮ ШҜШұ ШўЩҶ Ш§ШІ Ш§ШіШӘШ«ЩҶШ§ЩҮШ§ ШЁШұШ§ЫҢ Ъ©ЩҶШӘШұЩ„ Ш¬ШұЫҢШ§ЩҶ Ш№Ш§ШҜЫҢ ШЁШұЩҶШ§Щ…ЩҮ Ш§ШіШӘЩҒШ§ШҜЩҮ Щ…ЫҢвҖҢШҙЩҲШҜШҢ ШЁЩҮ Ш¬Ш§ЫҢ ШўЩҶЪ©ЩҮ ШЁШұШ§ЫҢ ШҙШұШ§ЫҢШ· ЩҲШ§ЩӮШ№Ш§ЩӢ Ш§ШіШӘШ«ЩҶШ§ЫҢЫҢ ЩҲ ШәЫҢШұЩ…ЩҶШӘШёШұЩҮ ШұШІШұЩҲ ШҙЩҲЩҶШҜ. ШҜШұ Ш§ЫҢЩҶ ЩҫШ§ШұШ§ШҜШ§ЫҢЩ…ШҢ Ш§ШіШӘШ«ЩҶШ§ЩҮШ§ ШЁЩҮ ЫҢЪ© ШіШ§Ш®ШӘШ§Шұ Ъ©ЩҶШӘШұЩ„ЫҢ ШҙШЁЫҢЩҮ ШЁЩҮ ШҜШіШӘЩҲШұШ§ШӘ if ЫҢШ§ ШӯЩ„ЩӮЩҮвҖҢЩҮШ§ ШӘШЁШҜЫҢЩ„ Щ…ЫҢвҖҢШҙЩҲЩҶШҜШҢ ЩҶЩҮ Ш§ЫҢЩҶЪ©ЩҮ ШЁЩҮ Ш№ЩҶЩҲШ§ЩҶ ШҙШЁЪ©ЩҮвҖҢЩҮШ§ЫҢ Ш§ЫҢЩ…ЩҶЫҢ ШЁШұШ§ЫҢ ШҙШұШ§ЫҢШ· ЩҲШ§ЩӮШ№Ш§ЩӢ Ш§ШіШӘШ«ЩҶШ§ЫҢЫҢ Ш№Щ…Щ„ Ъ©ЩҶЩҶШҜ Ъ©ЩҮ ШЁЩҮ ЩҶШҜШұШӘ ШЁШ§ЫҢШҜ ШұШ® ШҜЩҮЩҶШҜ. ЩҲЫҢЪҳЪҜЫҢ Ш§ШөЩ„ЫҢ EDDШҢ Ш·ШұШ§ШӯЫҢ Ш№Ш§Щ…ШҜШ§ЩҶЩҮ Ъ©ШҜЫҢ Ш§ШіШӘ Ъ©ЩҮ Ш§ЩҶШӘШёШ§Шұ ШҜШ§ШұШҜ Ш§ШіШӘШ«ЩҶШ§ЩҮШ§ ШҜШұ Ш·ЩҲЩ„ Ш№Щ…Щ„ЫҢШ§ШӘ Ш№Ш§ШҜЫҢ ЩҫШұШӘШ§ШЁ ШҙЩҲЩҶШҜ. ШЁЩҮ Ш¬Ш§ЫҢ ШЁШұШұШіЫҢ ЩҫЫҢШҙвҖҢШҙШұШ·вҖҢЩҮШ§ ЫҢШ§ Ш§ШіШӘЩҒШ§ШҜЩҮ Ш§ШІ Щ…ШӘШҜЩҮШ§ЫҢ ШӘШ®ШөШөЫҢ ШЁШұШ§ЫҢ ШӘШ№ЫҢЫҢЩҶ Щ…ЩҲЩҒЩӮЫҢШӘ ЫҢЪ© Ш№Щ…Щ„ЫҢШ§ШӘШҢ ШӘЩҲШіШ№ЩҮвҖҢШҜЩҮЩҶШҜЪҜШ§ЩҶ ШЁШ§ Ш§ЫҢЩҶ ШұЩҲЫҢЪ©ШұШҜ ШЁЩҮ ШіШ§ШҜЪҜЫҢ Ш№Щ…Щ„ЫҢШ§ШӘ ШұШ§ Ш§ЩҶШ¬Ш§Щ… ШҜШ§ШҜЩҮ ЩҲ ЩҮШұЪҜЩҲЩҶЩҮ Ш§ШіШӘШ«ЩҶШ§ШЎ ЩҶШ§ШҙЫҢ Ш§ШІ ШўЩҶ ШұШ§ ШҜШұЫҢШ§ЩҒШӘ (catch) Щ…ЫҢвҖҢЪ©ЩҶЩҶШҜ.

ЩҫЫҢШҙЫҢЩҶЩҮ ШӘШ§ШұЫҢШ®ЫҢ Ш§ШіШӘШ«ЩҶШ§ЩҮШ§
Щ…Ъ©Ш§ЩҶЫҢШіЩ…вҖҢЩҮШ§ЫҢ Ш§ШіШӘШ«ЩҶШ§ШЎ ШҜШұ ШІШЁШ§ЩҶвҖҢЩҮШ§ЫҢ ШЁШұЩҶШ§Щ…ЩҮвҖҢЩҶЩҲЫҢШіЫҢ ШЁШұШ§ЫҢ Щ…ШҜЫҢШұЫҢШӘ ШҙШұШ§ЫҢШ· ШәЫҢШұЩ…ЩҶШӘШёШұЩҮ ЩҲ Ш§ШіШӘШ«ЩҶШ§ЫҢЫҢ Щ…Ш№ШұЩҒЫҢ ШҙШҜЩҶШҜ Ъ©ЩҮ ШЁЩҮ ЩҶШҜШұШӘ ШҜШұ Ш·ЩҲЩ„ Ш§Ш¬ШұШ§ЫҢ Ш№Ш§ШҜЫҢ ШЁШұЩҶШ§Щ…ЩҮ ШұШ® Щ…ЫҢвҖҢШҜЩҮЩҶШҜ. ШІШЁШ§ЩҶвҖҢЩҮШ§ЫҢЫҢ Щ…Ш§ЩҶЩҶШҜ C++ ЩҲ ШЁШ№ШҜЩҮШ§ #JavaШҢ C ЩҲ ШіШ§ЫҢШұЫҢЩҶШҢ Щ…ШҜЫҢШұЫҢШӘ Ш§ШіШӘШ«ЩҶШ§ШЎ ШұШ§ ШЁШұШ§ЫҢ Ш¬ШҜШ§ШіШ§ШІЫҢ Ъ©ШҜ Щ…ШҜЫҢШұЫҢШӘ Ш®Ш·Ш§ Ш§ШІ Щ…ЩҶШ·ЩӮ Ш§ШөЩ„ЫҢ ШЁШұЩҶШ§Щ…ЩҮ ЩҫЫҢШ§ШҜЩҮвҖҢШіШ§ШІЫҢ Ъ©ШұШҜЩҶШҜ ШӘШ§ Ъ©ШҜ ЩҲШ§Ш¶ШӯвҖҢШӘШұ ЩҲ ЩӮШ§ШЁЩ„ ЩҶЪҜЩҮШҜШ§ШұЫҢвҖҢШӘШұ ШҙЩҲШҜ. ШЁШ§ Ш§ЫҢЩҶ ШӯШ§Щ„ШҢ ЩҮЩ…Ш§ЩҶЩҶШҜ ШЁШіЫҢШ§ШұЫҢ Ш§ШІ ЩҲЫҢЪҳЪҜЫҢвҖҢЩҮШ§ЫҢ ШЁШұЩҶШ§Щ…ЩҮвҖҢЩҶЩҲЫҢШіЫҢШҢ Ш§ШіШӘШ«ЩҶШ§ЩҮШ§ ШЁЩҮ ШұЩҲШҙвҖҢЩҮШ§ЫҢЫҢ Щ…ЩҲШұШҜ Ш§ШіШӘЩҒШ§ШҜЩҮ ЩӮШұШ§Шұ ЪҜШұЩҒШӘЩҶШҜ Ъ©ЩҮ Ш·ШұШ§ШӯШ§ЩҶ ШўЩҶвҖҢЩҮШ§ ШҜШұ ЩҶШёШұ ЩҶШҜШ§ШҙШӘЩҶШҜ. ШЁШұШ®ЫҢ ШӘЩҲШіШ№ЩҮвҖҢШҜЩҮЩҶШҜЪҜШ§ЩҶ ШҙШұЩҲШ№ ШЁЩҮ Ш§ШіШӘЩҒШ§ШҜЩҮ Ш§ШІ Ш§ШіШӘШ«ЩҶШ§ЩҮШ§ ШЁШұШ§ЫҢ Ш¬ШұЫҢШ§ЩҶ Ъ©ЩҶШӘШұЩ„ Ш№Ш§ШҜЫҢ Ъ©ШұШҜЩҶШҜ Ъ©ЩҮ Щ…ЩҶШ¬Шұ ШЁЩҮ ЪҶЫҢШІЫҢ ШҙШҜ Ъ©ЩҮ Ш§Ъ©ЩҶЩҲЩҶ ШЁЩҮ Ш№ЩҶЩҲШ§ЩҶ ШӘЩҲШіШ№ЩҮ Щ…ШЁШӘЩҶЫҢ ШЁШұ Ш§ШіШӘШ«ЩҶШ§ШЎ Щ…ЫҢвҖҢШҙЩҶШ§ШіЫҢЩ….

ЪҶШұШ§ ШӘЩҲШіШ№ЩҮ Щ…ШЁШӘЩҶЫҢ ШЁШұ Ш§ШіШӘШ«ЩҶШ§ШЎ Щ…Ш¶Шұ ШӘЩ„ЩӮЫҢ Щ…ЫҢвҖҢШҙЩҲШҜШҹ
EDD ШЁЩҮ ШҜЩ„Ш§ЫҢЩ„ ЩӮШ§ЩҶШ№вҖҢЪ©ЩҶЩҶШҜЩҮвҖҢШ§ЫҢ Ъ©ЩҮ ШЁШұ Ъ©ЫҢЩҒЫҢШӘ Ъ©ШҜШҢ Ш№Щ…Щ„Ъ©ШұШҜ ЩҲ ЩӮШ§ШЁЩ„ЫҢШӘ ЩҶЪҜЩҮШҜШ§ШұЫҢ ШӘШЈШ«ЫҢШұ Щ…ЫҢвҖҢЪҜШ°Ш§ШұЩҶШҜШҢ ШЁЩҮ Ш·ЩҲШұ ЪҜШіШӘШұШҜЩҮ ЫҢЪ© Ш№Щ…Щ„ ЩҶШ§Щ…ЩҶШ§ШіШЁ ШҜШұ ЩҶШёШұ ЪҜШұЩҒШӘЩҮ Щ…ЫҢвҖҢШҙЩҲШҜ.

ЩҫЫҢШ§Щ…ШҜЩҮШ§ЫҢ Ш№Щ…Щ„Ъ©ШұШҜЫҢ
Ш§ШіШӘШ«ЩҶШ§ЩҮШ§ ШЁШұШ§ЫҢ ШҙШұШ§ЫҢШ· Ш§ШіШӘШ«ЩҶШ§ЫҢЫҢ Ш·ШұШ§ШӯЫҢ ШҙШҜЩҮвҖҢШ§ЩҶШҜ ЩҲ ШЁЩҶШ§ШЁШұШ§ЫҢЩҶ ЩҫЫҢШ§ШҜЩҮвҖҢШіШ§ШІЫҢ ШўЩҶвҖҢЩҮШ§ Ш§Ш·Щ„Ш§Ш№Ш§ШӘ ШӘШҙШ®ЫҢШөЫҢ ШұШ§ ШЁШұ Ш№Щ…Щ„Ъ©ШұШҜ Ш§ЩҲЩ„ЩҲЫҢШӘ Щ…ЫҢвҖҢШҜЩҮШҜ. ЩҮЩҶЪҜШ§Щ…ЫҢ Ъ©ЩҮ ЫҢЪ© Ш§ШіШӘШ«ЩҶШ§ШЎ ЩҫШұШӘШ§ШЁ Щ…ЫҢвҖҢШҙЩҲШҜШҢ ШіЫҢШіШӘЩ… ШІЩ…Ш§ЩҶ Ш§Ш¬ШұШ§ (runtime system) ШЁШ§ЫҢШҜ ЪҶЩҶШҜЫҢЩҶ Ш№Щ…Щ„ЫҢШ§ШӘ ЩҫШұЩҮШІЫҢЩҶЩҮ Ш§ЩҶШ¬Ш§Щ… ШҜЩҮШҜ: ЪҜШұЩҒШӘЩҶ ЩҫШҙШӘЩҮ ЩҒШұШ§Ш®ЩҲШ§ЩҶЫҢ ЩҒШ№Щ„ЫҢ (call stack)ШҢ Ш¬ШіШӘШ¬ЩҲ ШЁШұШ§ЫҢ ЫҢЪ© Ъ©ЩҶШӘШұЩ„вҖҢЪ©ЩҶЩҶШҜЩҮ Ш§ШіШӘШ«ЩҶШ§ШЎ (exception handler) Щ…ЩҶШ§ШіШЁШҢ ШЁШ§ШІ Ъ©ШұШҜЩҶ ЩҫШҙШӘЩҮ ЩҒШұШ§Ш®ЩҲШ§ЩҶЫҢ (unwind the call stack) ЩҲ Ш§ЩҶШӘЩӮШ§Щ„ Ъ©ЩҶШӘШұЩ„ ШЁЩҮ Ъ©ЩҶШӘШұЩ„вҖҢЪ©ЩҶЩҶШҜЩҮ. Ш§ЫҢЩҶ ЩҒШұШўЫҢЩҶШҜ ШЁЩҮ Ш·ЩҲШұ ЩӮШ§ШЁЩ„ ШӘЩҲШ¬ЩҮЫҢ ЪҜШұШ§ЩҶвҖҢШӘШұ Ш§ШІ Щ…Ъ©Ш§ЩҶЫҢШіЩ…вҖҢЩҮШ§ЫҢ Ш¬ШұЫҢШ§ЩҶ Ъ©ЩҶШӘШұЩ„ Ш№Ш§ШҜЫҢ Ш§ШіШӘ. ШҜШұ C#ШҢ ЩҫШұШӘШ§ШЁ ЫҢЪ© Ш§ШіШӘШ«ЩҶШ§ШЎ Щ…ЫҢвҖҢШӘЩҲШ§ЩҶШҜ ШөШҜЩҮШ§ ЫҢШ§ ШӯШӘЫҢ ЩҮШІШ§ШұШ§ЩҶ ШЁШ§Шұ Ъ©ЩҶШҜШӘШұ Ш§ШІ Ш§ШіШӘЩҒШ§ШҜЩҮ Ш§ШІ Щ…ЩҶШ·ЩӮ ШҙШұШ·ЫҢ ШЁШ§ШҙШҜ.

Ъ©Ш§ЩҮШҙ Ш®ЩҲШ§ЩҶШ§ЫҢЫҢ Ъ©ШҜ
Ъ©ШҜЫҢ Ъ©ЩҮ ШЁШұШ§ЫҢ Ш¬ШұЫҢШ§ЩҶ Ъ©ЩҶШӘШұЩ„ ШЁЩҮ Ш§ШіШӘШ«ЩҶШ§ЩҮШ§ Щ…ШӘЪ©ЫҢ Ш§ШіШӘШҢ Щ…Ш№Щ…ЩҲЩ„Ш§ЩӢ Ш®ЩҲШ§ЩҶШҜЩҶ ЩҲ ШҜШұЪ© ШўЩҶ ШҜШҙЩҲШ§ШұШӘШұ Ш§ШіШӘ. Ш¬ШұЫҢШ§ЩҶ ЩҲШ§ЩӮШ№ЫҢ Ш§Ш¬ШұШ§ Щ…ШЁЩҮЩ… Щ…ЫҢвҖҢШҙЩҲШҜ ЩҲ ШӘЩҲШіШ№ЩҮвҖҢШҜЩҮЩҶШҜЪҜШ§ЩҶ ШұШ§ Щ…Щ„ШІЩ… Щ…ЫҢвҖҢЪ©ЩҶШҜ ШӘШ§ ШЁЩҮ Ш¬Ш§ЫҢ ШҜЩҶШЁШ§Щ„ Ъ©ШұШҜЩҶ ЫҢЪ© ШұЩҲШ§ЫҢШӘ Ш®Ш·ЫҢ ЩҲ ЩҲШ§Ш¶ШӯШҢ Щ…ШіЫҢШұЩҮШ§ЫҢ ШЁШ§Щ„ЩӮЩҲЩҮ Ш§ШіШӘШ«ЩҶШ§ШЎ ШұШ§ ШЁЩҮ ШөЩҲШұШӘ Ш°ЩҮЩҶЫҢ ШұШҜЫҢШ§ШЁЫҢ Ъ©ЩҶЩҶШҜ.

ЩҫЫҢЪҶЫҢШҜЪҜЫҢ Ш§ШҙЪ©Ш§Щ„вҖҢШІШҜШ§ЫҢЫҢ (Debugging)
ЩҮЩҶЪҜШ§Щ…ЫҢ Ъ©ЩҮ Ш§ШіШӘШ«ЩҶШ§ЩҮШ§ ШЁЩҮ Ш·ЩҲШұ Щ…Ъ©ШұШұ ШЁЩҮ Ш№ЩҶЩҲШ§ЩҶ ШЁШ®ШҙЫҢ Ш§ШІ Ш№Щ…Щ„ЫҢШ§ШӘ Ш№Ш§ШҜЫҢ ЩҫШұШӘШ§ШЁ Щ…ЫҢвҖҢШҙЩҲЩҶШҜШҢ ШӘШҙШ®ЫҢШө ШЁЫҢЩҶ Ш§ШіШӘШ«ЩҶШ§ЩҮШ§ЫҢ Щ…ЩҲШұШҜ Ш§ЩҶШӘШёШ§Шұ ЩҲ ШўЩҶЩҮШ§ЫҢЫҢ Ъ©ЩҮ ЩҲШ§ЩӮШ№Ш§ЩӢ Щ…ШҙЪ©Щ„вҖҢШіШ§ШІ ЩҮШіШӘЩҶШҜШҢ ШҜШҙЩҲШ§Шұ Щ…ЫҢвҖҢШҙЩҲШҜ. Ш§ЫҢЩҶ Ш§Щ…Шұ Щ…ЫҢвҖҢШӘЩҲШ§ЩҶШҜ ШӘЩ„Ш§ШҙвҖҢЩҮШ§ЫҢ Ш§ШҙЪ©Ш§Щ„вҖҢШІШҜШ§ЫҢЫҢ ШұШ§ ШЁЩҮ Ш·ЩҲШұ ЩӮШ§ШЁЩ„ ШӘЩҲШ¬ЩҮЫҢ ЩҫЫҢЪҶЫҢШҜЩҮ Ъ©ЩҶШҜШҢ ШІЫҢШұШ§ Ш§ШЁШІШ§ШұЩҮШ§ ЩҲ Щ„Ш§ЪҜвҖҢЩҮШ§ ШЁШ§ "ЩҶЩҲЫҢШІ" ЩҶШ§ШҙЫҢ Ш§ШІ Ш§ШіШӘШ«ЩҶШ§ЩҮШ§ЫҢ Щ…ЩҲШұШҜ Ш§ЩҶШӘШёШ§Шұ ЩҫШұ Щ…ЫҢвҖҢШҙЩҲЩҶШҜ.

ЩҫЩҶЩҮШ§ЩҶ Ъ©ШұШҜЩҶ Ш®Ш·Ш§ЩҮШ§ЫҢ Щ…ЩҶШ·ЩӮЫҢ
ШӘЩҲШіШ№ЩҮ Щ…ШЁШӘЩҶЫҢ ШЁШұ Ш§ШіШӘШ«ЩҶШ§ШЎ Щ…ЫҢвҖҢШӘЩҲШ§ЩҶШҜ ШЁШ§ ШӘЩ„ЩӮЫҢ Ш®Ш·Ш§ЩҮШ§ЫҢ Щ…ЩҶШ·ЩӮЫҢ ШЁЩҮ Ш№ЩҶЩҲШ§ЩҶ Ш¬ШұЫҢШ§ЩҶ Ш№Ш§ШҜЫҢ ШЁШұЩҶШ§Щ…ЩҮШҢ ШўЩҶвҖҢЩҮШ§ ШұШ§ ЩҫЩҶЩҮШ§ЩҶ Ъ©ЩҶШҜ. Ш§ЫҢЩҶ Ъ©Ш§Шұ Щ…ШұШІ ШЁЫҢЩҶ ШҙШұШ§ЫҢШ· Щ…ЩҲШұШҜ Ш§ЩҶШӘШёШ§Шұ ЩҲ ШЁШ§ЪҜвҖҢЩҮШ§ЫҢ ЩҲШ§ЩӮШ№ЫҢ ШұШ§ Щ…ШӯЩҲ Щ…ЫҢвҖҢЪ©ЩҶШҜ ЩҲ ШҙЩҶШ§ШіШ§ЫҢЫҢ ЩҲ ШұЩҒШ№ Щ…ШҙЪ©Щ„Ш§ШӘ ЩҲШ§ЩӮШ№ЫҢ ШҜШұ Ъ©ШҜ ШұШ§ ШҜШҙЩҲШ§ШұШӘШұ Щ…ЫҢвҖҢШіШ§ШІШҜ.

ЪҶШ§Щ„ШҙвҖҢЩҮШ§ЫҢ ШӘШіШӘвҖҢЩҶЩҲЫҢШіЫҢ
Ъ©ШҜЫҢ Ъ©ЩҮ ШЁШұШ§ЫҢ Ш¬ШұЫҢШ§ЩҶ Ъ©ЩҶШӘШұЩ„ ШЁЩҮ Ш§ШіШӘШ«ЩҶШ§ЩҮШ§ Щ…ШӘЪ©ЫҢ Ш§ШіШӘШҢ Щ…Ш№Щ…ЩҲЩ„Ш§ЩӢ ШӘШіШӘвҖҢЩҶЩҲЫҢШіЫҢ ШўЩҶ ШҜШҙЩҲШ§ШұШӘШұ Ш§ШіШӘ. ШӘШіШӘвҖҢЩҮШ§ЫҢ ЩҲШ§ШӯШҜ (Unit tests) ШЁШ§ЫҢШҜ Ш§ШіШӘШ«ЩҶШ§ЩҮШ§ЫҢ Щ…ЩҲШұШҜ Ш§ЩҶШӘШёШ§Шұ ШұШ§ ШҜШұ ЩҶШёШұ ШЁЪҜЫҢШұЩҶШҜ Ъ©ЩҮ Ш·ШұШ§ШӯЫҢ ШӘШіШӘ ШұШ§ ЩҫЫҢЪҶЫҢШҜЩҮ Ъ©ШұШҜЩҮ ЩҲ Щ…ЫҢвҖҢШӘЩҲШ§ЩҶШҜ Щ…Ш№ЫҢШ§ШұЩҮШ§ЫҢ ЩҫЩҲШҙШҙ Ъ©ШҜ (code coverage) ШұШ§ Ъ©Щ…вҖҢЩ…Ш№ЩҶШ§ШӘШұ Ъ©ЩҶШҜ.

ШҙЩҶШ§ШіШ§ЫҢЫҢ ШӘЩҲШіШ№ЩҮ Щ…ШЁШӘЩҶЫҢ ШЁШұ Ш§ШіШӘШ«ЩҶШ§ШЎ
ШӘЩҲШіШ№ЩҮ Щ…ШЁШӘЩҶЫҢ ШЁШұ Ш§ШіШӘШ«ЩҶШ§ШЎ ШұШ§ Щ…ЫҢвҖҢШӘЩҲШ§ЩҶ ШЁШ§ ЪҶЩҶШҜЫҢЩҶ ЩҶШҙШ§ЩҶЩҮ ШҜШұ Ъ©ШҜ ШҙЩҶШ§ШіШ§ЫҢЫҢ Ъ©ШұШҜ:
  • Ш§ШіШӘЩҒШ§ШҜЩҮ Щ…Ъ©ШұШұ Ш§ШІ ШЁЩ„ЩҲЪ©вҖҢЩҮШ§ЫҢ try-catch ШЁШұШ§ЫҢ Ш№Щ…Щ„ЫҢШ§ШӘ ШұЩҲШӘЫҢЩҶ.
  • ШЁЩ„ЩҲЪ©вҖҢЩҮШ§ЫҢ catch Ш®Ш§Щ„ЫҢ ЫҢШ§ ШЁЩ„ЩҲЪ©вҖҢЩҮШ§ЫҢЫҢ Ъ©ЩҮ Ш§ШіШӘШ«ЩҶШ§ЩҮШ§ ШұШ§ ШЁШҜЩҲЩҶ Щ…ШҜЫҢШұЫҢШӘ Щ…Ш№ЩҶШ§ШҜШ§Шұ Щ…ЫҢвҖҢШЁЩ„Ш№ЩҶШҜ (swallow exceptions).
  • Ш§ШіШӘЩҒШ§ШҜЩҮ Ш§ШІ Ш§ШіШӘШ«ЩҶШ§ЩҮШ§ ШЁШұШ§ЫҢ Ш§Ш№ШӘШЁШ§ШұШіЩҶШ¬ЫҢ ЫҢШ§ ШҙШұШ§ЫҢШ· Ш®Ш·Ш§ЫҢ Щ…ЩҲШұШҜ Ш§ЩҶШӘШёШ§Шұ.
  • Ш¬ШұЫҢШ§ЩҶ Ъ©ЩҶШӘШұЩ„ЫҢ Ъ©ЩҮ ШЁЩҮ ЩҫШұШӘШ§ШЁ ШҙШҜЩҶ Ш§ШіШӘШ«ЩҶШ§ЩҮШ§ЫҢ Ш®Ш§Шө ШЁШіШӘЪҜЫҢ ШҜШ§ШұШҜ.
  • Щ…ШӘШҜЩҮШ§ ЫҢШ§ APIЩҮШ§ЫҢЫҢ Ъ©ЩҮ ШЁШұШ§ЫҢ ЩҫШұШӘШ§ШЁ Ш§ШіШӘШ«ЩҶШ§ЩҮШ§ ШЁЩҮ Ш№ЩҶЩҲШ§ЩҶ ШЁШ®ШҙЫҢ Ш§ШІ Ш№Щ…Щ„ЫҢШ§ШӘ Ш№Ш§ШҜЫҢ Ш®ЩҲШҜ Ш·ШұШ§ШӯЫҢ ШҙШҜЩҮвҖҢШ§ЩҶШҜ.

Ш§ШЁШІШ§ШұЩҮШ§ЫҢЫҢ Щ…Ш§ЩҶЩҶШҜ ШӘШӯЩ„ЫҢЩ„вҖҢЪҜШұЩҮШ§ЫҢ Ъ©ШҜ Ш§ЫҢШіШӘШ§ (Static code analyzers) Щ…Ш§ЩҶЩҶШҜ SonarQube ЩҲ ReSharper Щ…ЫҢвҖҢШӘЩҲШ§ЩҶЩҶШҜ ШЁШұШ®ЫҢ Ш§ШІ Ш§Щ„ЪҜЩҲЩҮШ§ЫҢ Ш§ШіШӘЩҒШ§ШҜЩҮ ЩҶШ§ШҜШұШіШӘ Ш§ШІ Ш§ШіШӘШ«ЩҶШ§ШЎ ШұШ§ ШҙЩҶШ§ШіШ§ЫҢЫҢ Ъ©ЩҶЩҶШҜ. ЩҮЩ…ЪҶЩҶЫҢЩҶШҢ ЩҫШұЩҲЩҒШ§ЫҢЩ„ШұЩҮШ§ЫҢ Ш№Щ…Щ„Ъ©ШұШҜ (Performance profilers) Щ…Ш§ЩҶЩҶШҜ dotTrace ЫҢШ§ Visual Studio Profiler Щ…ЫҢвҖҢШӘЩҲШ§ЩҶЩҶШҜ Щ…ШҙШ®Шө Ъ©ЩҶЩҶШҜ Ъ©ЩҮ Ш§ШіШӘШ«ЩҶШ§ЩҮШ§ ШҜШұ Ъ©Ш¬Ш§ ШЁЩҮ Ш·ЩҲШұ Щ…Ъ©ШұШұ ШҜШұ Ш·ЩҲЩ„ Ш№Щ…Щ„ЫҢШ§ШӘ Ш№Ш§ШҜЫҢ ЩҫШұШӘШ§ШЁ Щ…ЫҢвҖҢШҙЩҲЩҶШҜ. ШЁШ§ШІШЁЫҢЩҶЫҢ Ъ©ШҜ (Code reviews) Щ…ЩҶШёЩ… ШӘЩҲШіШ· ЩҮЩ…Ъ©Ш§ШұШ§ЩҶ ШЁШ§ ШӘШЈЪ©ЫҢШҜ ШЁШұ Щ…ШҜЫҢШұЫҢШӘ Ш§ШіШӘШ«ЩҶШ§ШЎ ЩҶЫҢШІ Щ…ЫҢвҖҢШӘЩҲШ§ЩҶШҜ ШЁЩҮ ШҙЩҶШ§ШіШ§ЫҢЫҢ EDD Ъ©Щ…Ъ© Ъ©ЩҶШҜ.

Ш¶ШҜШ§Щ„ЪҜЩҲЩҮШ§ЫҢ ШұШ§ЫҢШ¬ EDD ШҜШұ C#
ШЁЫҢШ§ЫҢЫҢШҜ ШЁШұШ®ЫҢ Ш§ШІ Ш§Щ„ЪҜЩҲЩҮШ§ЫҢ ШұШ§ЫҢШ¬ Ъ©ЩҮ ШҜШұ ШўЩҶвҖҢЩҮШ§ ШӘЩҲШіШ№ЩҮвҖҢШҜЩҮЩҶШҜЪҜШ§ЩҶ ШЁЩҮ Ш·ЩҲШұ ШіЩҮЩҲЫҢ ЫҢШ§ Ш№Щ…ШҜЫҢ Ш§ШІ EDD ШҜШұ Ъ©ШҜ C# Ш§ШіШӘЩҒШ§ШҜЩҮ Щ…ЫҢвҖҢЪ©ЩҶЩҶШҜ ШұШ§ ШЁШұШұШіЫҢ Ъ©ЩҶЫҢЩ….

Ш¶ШҜШ§Щ„ЪҜЩҲ Ыұ: Ш§ШіШӘЩҒШ§ШҜЩҮ Ш§ШІ Ш§ШіШӘШ«ЩҶШ§ЩҮШ§ ШЁШұШ§ЫҢ Ш§Ш№ШӘШЁШ§ШұШіЩҶШ¬ЫҢ
ЫҢЪ©ЫҢ Ш§ШІ ШұШ§ЫҢШ¬вҖҢШӘШұЫҢЩҶ ЩҶЩ…ЩҲЩҶЩҮвҖҢЩҮШ§ЫҢ EDD Ш§ШіШӘЩҒШ§ШҜЩҮ Ш§ШІ Ш§ШіШӘШ«ЩҶШ§ЩҮШ§ ШЁШұШ§ЫҢ Ш§Ш№ШӘШЁШ§ШұШіЩҶШ¬ЫҢ ЩҲШұЩҲШҜЫҢ Ш§ШіШӘ. ШЁЩҮ Ш№ЩҶЩҲШ§ЩҶ Щ…Ш«Ш§Щ„ШҢ ШҜШұ Ъ©ШҜ ШІЫҢШұШҢ ШЁЩҮ Ш¬Ш§ЫҢ ШЁШұШұШіЫҢ ШөШұЫҢШӯ Ш§ЫҢЩҶЪ©ЩҮ ШўЫҢШ§ username ШӘЩҮЫҢ (null) Ш§ШіШӘШҢ Ъ©ШҜ ШЁЩҮ ЩҫШұШӘШ§ШЁ ШҙШҜЩҶ NullReferenceException ЩҮЩҶЪҜШ§Щ… ШҜШіШӘШұШіЫҢ ШЁЩҮ Ш®Ш§ШөЫҢШӘ Length Щ…ШӘЪ©ЫҢ Ш§ШіШӘ:
public void ProcessUser(string username)
{
    try
    {
        // Directly try to process without validation
        if (username.Length < 3) // Potential NullReferenceException if username is null
        {
            Console.WriteLine("Username too short");
        }
        // Further processing...
    }
    catch (NullReferenceException)
    {
        Console.WriteLine("Username cannot be null");
    }
}
Ш§ЫҢЩҶ ШұЩҲЫҢЪ©ШұШҜ ШҜШұЪ© Ъ©ШҜ ШұШ§ ШҜШҙЩҲШ§ШұШӘШұ ЩҲ ЩҶШ§Ъ©Ш§ШұШўЩ…ШҜШӘШұ Щ…ЫҢвҖҢЪ©ЩҶШҜ. ЫҢЪ© ШұЩҲЫҢЪ©ШұШҜ ШЁЩҮШӘШұШҢ ШЁШұШұШіЫҢ ШөШұЫҢШӯ ЩҫЫҢШҙвҖҢШҙШұШ·вҖҢЩҮШ§ Ш§ШіШӘ:
public void ProcessUser(string username)

    {
        Console.WriteLine("Username cannot be null");
        return;
    }
    
    if (username.Length < 3)
    {
        Console.WriteLine("Username too short");
    }
    // Further processing...
}

Ш¶ШҜШ§Щ„ЪҜЩҲ ЫІ: Ш№Щ…Щ„ЫҢШ§ШӘ ЩҒШ§ЫҢЩ„ ШЁШҜЩҲЩҶ ШЁШұШұШіЫҢвҖҢЩҮШ§ЫҢ Ш§ЩҲЩ„ЫҢЩҮ
ШіЩҶШ§ШұЫҢЩҲЫҢ ШұШ§ЫҢШ¬ ШҜЫҢЪҜШұШҢ Ш§ЩҶШ¬Ш§Щ… Ш№Щ…Щ„ЫҢШ§ШӘ ЩҒШ§ЫҢЩ„ ШЁШҜЩҲЩҶ ШЁШұШұШіЫҢ Ш§ЩҲЩ„ЫҢЩҮ ЩҲШ¬ЩҲШҜ ЩҒШ§ЫҢЩ„ Ш§ШіШӘ:
public string ReadFileContents(string filePath)
{
    try
    {
        return File.ReadAllText(filePath); // Potential FileNotFoundException
    }
    catch (FileNotFoundException)
    {
        return "File not found";
    }
}
Ъ©ШҜ ЩҒЩҲЩӮ ШЁЩҮ Ш§ШіШӘШ«ЩҶШ§ШЎ ШЁШұШ§ЫҢ ШӘШ№ЫҢЫҢЩҶ ЩҲШ¬ЩҲШҜ ЩҒШ§ЫҢЩ„ Щ…ШӘЪ©ЫҢ Ш§ШіШӘ. ШұЩҲЫҢЪ©ШұШҜ Щ…ШіШӘЩӮЫҢЩ…вҖҢШӘШұ ШЁЩҮ ШҙЪ©Щ„ ШІЫҢШұ Ш§ШіШӘ:
public string ReadFileContents(string filePath)
{
    if (!File.Exists(filePath))
    {
        return "File not found";
    }
    
    return File.ReadAllText(filePath);
}
ШЁШ§ Ш§ЫҢЩҶ ШӯШ§Щ„ШҢ ШҙШ§ЫҢШ§ЩҶ Ш°Ъ©Шұ Ш§ШіШӘ Ъ©ЩҮ ШӯШӘЫҢ Ш§ЫҢЩҶ ЩҶШіШ®ЩҮ ШЁЩҮШЁЩҲШҜ ЫҢШ§ЩҒШӘЩҮ ЩҶЫҢШІ ШҜШ§ШұШ§ЫҢ ЫҢЪ© ЩҲШ¶Ш№ЫҢШӘ ШұЩӮШ§ШЁШӘЫҢ (race condition) ШЁЫҢЩҶ ШЁШұШұШіЫҢ ЩҲ Ш№Щ…Щ„ЫҢШ§ШӘ Ш®ЩҲШ§ЩҶШҜЩҶ Ш§ШіШӘ. ЫҢЪ© ШұШ§ЩҮвҖҢШӯЩ„ ЩҲШ§ЩӮШ№Ш§ЩӢ Щ…ЩӮШ§ЩҲЩ…ШҢ Ш§ШіШӘШ«ЩҶШ§ШЎ ШЁШ§Щ„ЩӮЩҲЩҮвҖҢШ§ЫҢ ШұШ§ Ъ©ЩҮ Щ…Щ…Ъ©ЩҶ Ш§ШіШӘ ШҜШұ ШөЩҲШұШӘ ШӯШ°ЩҒ ЩҒШ§ЫҢЩ„ ШЁЫҢЩҶ ШЁШұШұШіЫҢ ЩҲ Ш®ЩҲШ§ЩҶШҜЩҶ ШұШ® ШҜЩҮШҜШҢ Щ…ШҜЫҢШұЫҢШӘ Щ…ЫҢвҖҢЪ©ЩҶШҜ.

Ш¶ШҜШ§Щ„ЪҜЩҲ Ыі: Ш¬ШіШӘШ¬ЩҲ ШҜШұ ШҜЫҢЪ©ШҙЩҶШұЫҢ (Dictionary Lookups)
Ш§ШіШӘЩҒШ§ШҜЩҮ Ш§ШІ Ш§ШіШӘШ«ЩҶШ§ЩҮШ§ ШЁШұШ§ЫҢ Ш¬ШіШӘШ¬ЩҲ ШҜШұ ШҜЫҢЪ©ШҙЩҶШұЫҢ ЩҶЫҢШІ ЫҢЪ© Ш§Щ„ЪҜЩҲЫҢ ШұШ§ЫҢШ¬ Ш§ШіШӘ:
public string GetValue(Dictionary<string, string> dictionary, string key)
{
    try
    {
        return dictionary[key]; // Potential KeyNotFoundException
    }
    catch (KeyNotFoundException)
    {
        return "Key not found";
    }
}
ШҜЫҢЪ©ШҙЩҶШұЫҢвҖҢЩҮШ§ЫҢ C# ЫҢЪ© Щ…ШӘШҜ Ш®Ш§Шө ШЁШұШ§ЫҢ Ш¬Щ„ЩҲЪҜЫҢШұЫҢ Ш§ШІ Ш§ЫҢЩҶ ШұЩҲЫҢЪ©ШұШҜ Щ…ШЁШӘЩҶЫҢ ШЁШұ Ш§ШіШӘШ«ЩҶШ§ШЎ Ш§ШұШ§ШҰЩҮ Щ…ЫҢвҖҢШҜЩҮЩҶШҜ:
public string GetValue(Dictionary<string, string> dictionary, string key)
{
    if (dictionary.TryGetValue(key, out string value))
    {
        return value;
    }
    return "Key not found";
}

Ш¶ШҜШ§Щ„ЪҜЩҲ Ыҙ: ШӘШЁШҜЫҢЩ„ ЩҶЩҲШ№ (Type Conversions)
ШӘШЁШҜЫҢЩ„ ЩҶЩҲШ№ Щ…ШЁШӘЩҶЫҢ ШЁШұ Ш§ШіШӘШ«ЩҶШ§ШЎ ЩҶЫҢШІ ШұШ§ЫҢШ¬ Ш§ШіШӘ:
public int ConvertToInt(string input)
{
    try
    {
        return int.Parse(input); // Potential FormatException
    }
    catch (FormatException)
    {
        return 0; // Default value on failure
    }
}
ЫҢЪ© ШұЩҲЫҢЪ©ШұШҜ Ъ©Ш§ШұШўЩ…ШҜШӘШұ Ш§ШІ Ш§Щ„ЪҜЩҲЫҢ TryParse Ш§ШіШӘЩҒШ§ШҜЩҮ Щ…ЫҢвҖҢЪ©ЩҶШҜ:
public int ConvertToInt(string input)
{
    if (int.TryParse(input, out int result))
    {
        return result;
    }
    return 0; // Default value on failure
}

ЪҶЩҮ ШІЩ…Ш§ЩҶЫҢ Ш§ШіШӘЩҒШ§ШҜЩҮ Ш§ШІ Ш§ШіШӘШ«ЩҶШ§ЩҮШ§ Щ…ЩҶШ§ШіШЁ Ш§ШіШӘШҹ
Щ…ЩҮЩ… Ш§ШіШӘ Ъ©ЩҮ ШЁЫҢЩҶ EDD ЩҲ Ш§ШіШӘЩҒШ§ШҜЩҮ ШөШӯЫҢШӯ Ш§ШІ Ш§ШіШӘШ«ЩҶШ§ЩҮШ§ ШӘЩ…Ш§ЫҢШІ ЩӮШ§ШҰЩ„ ШҙЩҲЫҢЩ…. Ш§ШіШӘШ«ЩҶШ§ЩҮШ§ ЩҮЩҶЪҜШ§Щ…ЫҢ Ъ©ЩҮ ШЁЩҮ ШҜШұШіШӘЫҢ Ш§ШіШӘЩҒШ§ШҜЩҮ ШҙЩҲЩҶШҜШҢ Ш§ШЁШІШ§ШұЫҢ Ш§ШұШІШҙЩ…ЩҶШҜ ШЁШ§ЩӮЫҢ Щ…ЫҢвҖҢЩ…Ш§ЩҶЩҶШҜ. ШўЩҶвҖҢЩҮШ§ ШЁШұШ§ЫҢ ШҙШұШ§ЫҢШ· ЩҲШ§ЩӮШ№Ш§ЩӢ Ш§ШіШӘШ«ЩҶШ§ЫҢЫҢ Ъ©ЩҮ ЩҶШЁШ§ЫҢШҜ ШҜШұ Ш·ЩҲЩ„ Ш§Ш¬ШұШ§ЫҢ Ш№Ш§ШҜЫҢ ШЁШұЩҶШ§Щ…ЩҮ ШұШ® ШҜЩҮЩҶШҜШҢ Щ…ЩҶШ§ШіШЁ ЩҮШіШӘЩҶШҜШҢ Щ…Ш§ЩҶЩҶШҜ: Ш®ШұШ§ШЁЫҢ ШҙШЁЪ©ЩҮШҢ Щ…ШҙЪ©Щ„Ш§ШӘ Ш§ШӘШөШ§Щ„ ШЁЩҮ ЩҫШ§ЫҢЪҜШ§ЩҮ ШҜШ§ШҜЩҮШҢ Щ…ШҙЪ©Щ„Ш§ШӘ Щ…Ш¬ЩҲШІ ШіЫҢШіШӘЩ… ЩҒШ§ЫҢЩ„ШҢ Ъ©Щ…ШЁЩҲШҜ ШӯШ§ЩҒШёЩҮ ЩҲ Ш®Ш·Ш§ЩҮШ§ЫҢ ШЁШұЩҶШ§Щ…ЩҮвҖҢЩҶЩҲЫҢШіЫҢ Ъ©ЩҮ ЩҶШҙШ§ЩҶвҖҢШҜЩҮЩҶШҜЩҮ ШЁШ§ЪҜ ЩҮШіШӘЩҶШҜ (Щ…Ш§ЩҶЩҶШҜ Ш§ЩҶШҜЫҢШі Ш®Ш§ШұШ¬ Ш§ШІ Щ…ШӯШҜЩҲШҜЩҮ ШўШұШ§ЫҢЩҮ).

ЫҢЪ© ШӘЩ…Ш§ЫҢШІ Ъ©Щ„ЫҢШҜЫҢ ШЁЫҢЩҶ Щ…ЩҲШ§ШұШҜ Ш§ШіШӘШ«ЩҶШ§ЫҢЫҢ ЩҲ Ш®ШұШ§ШЁЫҢвҖҢЩҮШ§ЫҢ Щ…ЩҲШұШҜ Ш§ЩҶШӘШёШ§Шұ ЩҲШ¬ЩҲШҜ ШҜШ§ШұШҜ:
  • Ш®ШұШ§ШЁЫҢ Щ…ЩҲШұШҜ Ш§ЩҶШӘШёШ§Шұ (Expected failure): ШҙШұШ§ЫҢШ·ЫҢ Ъ©ЩҮ Щ…Ш№Щ…ЩҲЩ„Ш§ЩӢ ШҜШұ Ш·ЩҲЩ„ Ш№Щ…Щ„ЫҢШ§ШӘ Ш№Ш§ШҜЫҢ ШұШ® Щ…ЫҢвҖҢШҜЩҮШҜШҢ Щ…Ш§ЩҶЩҶШҜ ЩҲШ§ШұШҜ Ъ©ШұШҜЩҶ ШҜШ§ШҜЩҮ ЩҶШ§Щ…Ш№ШӘШЁШұ ШӘЩҲШіШ· Ъ©Ш§ШұШЁШұ ЫҢШ§ Ш№ШҜЩ… ЩҲШ¬ЩҲШҜ ЫҢЪ© ЩҒШ§ЫҢЩ„.
  • ШҙШұШ§ЫҢШ· Ш§ШіШӘШ«ЩҶШ§ЫҢЫҢ (Exceptional condition): ЫҢЪ© ЩҲШ¶Ш№ЫҢШӘ ЩҶШ§ШҜШұ ЫҢШ§ ШәЫҢШұЩ…ЩҶШӘШёШұЩҮ Ъ©ЩҮ ЩҶШҙШ§ЩҶ Щ…ЫҢвҖҢШҜЩҮШҜ Щ…ШҙЪ©Щ„ЫҢ Ш¬ШҜЫҢ ШұШ® ШҜШ§ШҜЩҮ Ш§ШіШӘ.

ЪҜШ§ЩҮЫҢ Ш§ЩҲЩӮШ§ШӘ ШҙЩ…Ш§ Щ…Ш¬ШЁЩҲШұ ШЁЩҮ Ъ©Ш§Шұ ШЁШ§ APIЩҮШ§ЫҢЫҢ ЩҮШіШӘЫҢШҜ Ъ©ЩҮ ШЁЩҮ Ш№ЩҶЩҲШ§ЩҶ ШЁШ®ШҙЫҢ Ш§ШІ Ш№Щ…Щ„ЫҢШ§ШӘ Ш№Ш§ШҜЫҢ Ш®ЩҲШҜ Ш§ШіШӘШ«ЩҶШ§ШЎ ЩҫШұШӘШ§ШЁ Щ…ЫҢвҖҢЪ©ЩҶЩҶШҜ. ШҜШұ Ш§ЫҢЩҶ Щ…ЩҲШ§ШұШҜШҢ ЪҜШұЩҒШӘЩҶ Ш§ЫҢЩҶ Ш§ШіШӘШ«ЩҶШ§ЩҮШ§ Щ…ЩҶШ§ШіШЁ Ш§ШіШӘШҢ Ш§ЪҜШұЪҶЩҮ Щ…Щ…Ъ©ЩҶ Ш§ШіШӘ ШЁШ®ЩҲШ§ЩҮЫҢШҜ ЪҶЩҶЫҢЩҶ APIЩҮШ§ЫҢЫҢ ШұШ§ ШҜШұ APIЩҮШ§ЫҢ Ш®ЩҲШҜ Ъ©ЩҮ Ш§ШІ ШҙЫҢЩҲЩҮвҖҢЩҮШ§ЫҢ ШЁЩҮШӘШұЫҢ ЩҫЫҢШұЩҲЫҢ Щ…ЫҢвҖҢЪ©ЩҶЩҶШҜШҢ ШЁЩҫЩҲШҙШ§ЩҶЫҢШҜ. ЩҮЩ…ЪҶЩҶЫҢЩҶШҢ ЩҮЩҶЪҜШ§Щ…ЫҢ Ъ©ЩҮ Ъ©ШҜ Щ…ШҙШӘШұЫҢ (client code) ЩӮШұШ§ШұШҜШ§ШҜ ЫҢЪ© Щ…ШӘШҜ ШұШ§ ЩҶЩӮШ¶ Щ…ЫҢвҖҢЪ©ЩҶШҜ (Щ…Ш§ЩҶЩҶШҜ Ш§ШұШіШ§Щ„ null ШІЩ…Ш§ЩҶЫҢ Ъ©ЩҮ Щ…ШӘШҜ null ШұШ§ ЩҶЩ…ЫҢвҖҢЩҫШ°ЫҢШұШҜ)ШҢ ЩҫШұШӘШ§ШЁ ЫҢЪ© Ш§ШіШӘШ«ЩҶШ§ШЎ Щ…Ш§ЩҶЩҶШҜ ArgumentNullException Щ…ЩҶШ§ШіШЁ Ш§ШіШӘ.

ШЁЩҮ Ш№ЩҶЩҲШ§ЩҶ Щ…Ш«Ш§Щ„ШҢ ШҜШұ Ъ©ШҜ ШІЫҢШұШҢ Ш§Ш№ШӘШЁШ§ШұШіЩҶШ¬ЫҢ Ш§ЩҲЩ„ЫҢЩҮ Ш§ЩҶШ¬Ш§Щ… Щ…ЫҢвҖҢШҙЩҲШҜ ЩҲ ШіЩҫШі Ш№Щ…Щ„ЫҢШ§ШӘ ЩҶЩҲШҙШӘЩҶ ЩҒШ§ЫҢЩ„ ШҜШұ ЫҢЪ© ШЁЩ„ЩҲЪ© try-catch ЩӮШұШ§Шұ Щ…ЫҢвҖҢЪҜЫҢШұШҜШҢ ШІЫҢШұШ§ Ш§ЫҢЩҶ Ш№Щ…Щ„ЫҢШ§ШӘ ЩҮЩ…ЪҶЩҶШ§ЩҶ Щ…Щ…Ъ©ЩҶ Ш§ШіШӘ ШЁЩҮ ШҜЩ„Ш§ЫҢЩ„ Ш§ШіШӘШ«ЩҶШ§ЫҢЫҢ (Щ…Ш§ЩҶЩҶШҜ ЩҫШұ ШЁЩҲШҜЩҶ ШҜЫҢШіЪ© ЫҢШ§ Щ…ШҙЪ©Щ„Ш§ШӘ Щ…Ш¬ЩҲШІ) ШЁШ§ ШҙЪ©ШіШӘ Щ…ЩҲШ§Ш¬ЩҮ ШҙЩҲШҜ:
public void SaveDocument(Document document, string path)
{
    if (document == null)
        throw new ArgumentNullException(nameof(document)); // Contract violation
    if (string.IsNullOrEmpty(path))
        throw new ArgumentException("Path cannot be empty", nameof(path)); // Contract violation

    // Normal validation completed, now we can proceed
    try
    {
        // This could still fail for exceptional reasons (disk full, permissions)
        File.WriteAllText(path, document.Content);
    }
    catch (IOException ex)
    {
        // Handle truly exceptional condition
        LogError($"Failed to save document: {ex.Message}");
        throw new DocumentSaveException("Could not save the document", ex);
    }
}

Ш¬Ш§ЫҢЪҜШІЫҢЩҶвҖҢЩҮШ§ЫҢ ШЁЩҮШӘШұ ШЁШұШ§ЫҢ ШӘЩҲШіШ№ЩҮ Щ…ШЁШӘЩҶЫҢ ШЁШұ Ш§ШіШӘШ«ЩҶШ§ШЎ
ЪҶЩҶШҜЫҢЩҶ Ш§Щ„ЪҜЩҲЫҢ Ш·ШұШ§ШӯЫҢ ЩҲ ШӘЪ©ЩҶЫҢЪ© ШЁШұЩҶШ§Щ…ЩҮвҖҢЩҶЩҲЫҢШіЫҢШҢ Ш¬Ш§ЫҢЪҜШІЫҢЩҶвҖҢЩҮШ§ЫҢ ШЁЩҮШӘШұЫҢ ШЁШұШ§ЫҢ EDD Ш§ШұШ§ШҰЩҮ Щ…ЫҢвҖҢШҜЩҮЩҶШҜ.

Ш§Щ„ЪҜЩҲЫҢ TryPattern ШҜШұ C#

Ш§Щ„ЪҜЩҲЫҢ Result
Ш§Щ„ЪҜЩҲЫҢ Result (ЩҶШӘЫҢШ¬ЩҮ) ШұЩҲШҙЫҢ ШіШ§Ш®ШӘШ§ШұЫҢШ§ЩҒШӘЩҮ ШЁШұШ§ЫҢ ШЁШ§ШІЪҜШұШҜШ§ЩҶШҜЩҶ ЫҢШ§ ЫҢЪ© Щ…ЩӮШҜШ§Шұ Щ…ЩҲЩҒЩӮЫҢШӘвҖҢШўЩ…ЫҢШІ ЫҢШ§ Ш§Ш·Щ„Ш§Ш№Ш§ШӘ Ш®Ш·Ш§ Ш§ШұШ§ШҰЩҮ Щ…ЫҢвҖҢШҜЩҮШҜ. 
public class Result<T>
{
    public bool IsSuccess { get; }
    public T Value { get; }
    public string Error { get; }
    
    private Result(bool isSuccess, T value, string error)
    {
        IsSuccess = isSuccess; Value = value; Error = error;
    }
    
    public static Result<T> Success(T value) => new Result<T>(true, value, null);
    public static Result<T> Failure(string error) => new Result<T>(false, default, error);
}

Ш§ЩҶЩҲШ§Ш№ Option/Maybe
Ш§ЩҶЩҲШ§Ш№ Option ЫҢШ§ Maybe ШЁЩҮ ШөШұШ§ШӯШӘ Ш§Щ…Ъ©Ш§ЩҶ Ш№ШҜЩ… ЩҲШ¬ЩҲШҜ ЫҢЪ© Щ…ЩӮШҜШ§Шұ ШұШ§ ЩҶШҙШ§ЩҶ Щ…ЫҢвҖҢШҜЩҮЩҶШҜ. (Ш§ЫҢЩҶ Ш§ЩҶЩҲШ§Ш№ Щ…Ш№Щ…ЩҲЩ„Ш§ЩӢ Ш§ШІ Ш·ШұЫҢЩӮ Ъ©ШӘШ§ШЁШ®Ш§ЩҶЩҮвҖҢЩҮШ§ЫҢ ШҙШ®Шө Ш«Ш§Щ„Ш« ЫҢШ§ ЩҫЫҢШ§ШҜЩҮвҖҢШіШ§ШІЫҢ ШіЩҒШ§ШұШҙЫҢ ШҜШұ ШҜШіШӘШұШі ЩҮШіШӘЩҶШҜ.)

ШЁШ§ШІШіШ§ШІЫҢ (Refactoring) Ъ©ШҜ Щ…ШЁШӘЩҶЫҢ ШЁШұ Ш§ШіШӘШ«ЩҶШ§ШЎ
ШЁЫҢШ§ЫҢЫҢШҜ ШЁШұШұШіЫҢ Ъ©ЩҶЫҢЩ… Ъ©ЩҮ ЪҶЪҜЩҲЩҶЩҮ Ъ©ШҜ Щ…ШЁШӘЩҶЫҢ ШЁШұ Ш§ШіШӘШ«ЩҶШ§ШЎ ШұШ§ ШЁШұШ§ЫҢ ЩҫЫҢШұЩҲЫҢ Ш§ШІ ШҙЫҢЩҲЩҮвҖҢЩҮШ§ЫҢ ШЁЩҮШӘШұ ШЁШ§ШІШіШ§ШІЫҢ Ъ©ЩҶЫҢЩ….

Щ…Ш«Ш§Щ„ Ыұ: ШЁШ§ШІШіШ§ШІЫҢ Щ…ЩҶШ·ЩӮ Ш§Ш№ШӘШЁШ§ШұШіЩҶШ¬ЫҢ
ЩӮШЁЩ„: Ъ©ШҜ Ш§ШІ decimal.Parse ШҜШұ ЫҢЪ© ШЁЩ„ЩҲЪ© try-catch ШЁШұШ§ЫҢ ШӘШЁШҜЫҢЩ„ ШұШҙШӘЩҮ ШЁЩҮ decimal Ш§ШіШӘЩҒШ§ШҜЩҮ Щ…ЫҢвҖҢЪ©ЩҶШҜ ЩҲ ШҜШұ ШөЩҲШұШӘ ШЁШұЩҲШІ FormatException Щ…ЩӮШҜШ§Шұ Ы° ШұШ§ ШЁШ§ШІЩ…ЫҢвҖҢЪҜШұШҜШ§ЩҶШҜ.

Щ…Ш«Ш§Щ„ ЫІ: ШЁШ§ШІШіШ§ШІЫҢ ШҜШіШӘШұШіЫҢ ШЁЩҮ Щ…ЩҶШ§ШЁШ№
ЩӮШЁЩ„: Ъ©ШҜ ШіШ№ЫҢ Щ…ЫҢвҖҢЪ©ЩҶШҜ Щ…ШҙШӘШұЫҢ ШұШ§ Ш§ШІ Щ…Ш®ШІЩҶ (repository) ШҜШұЫҢШ§ЩҒШӘ Ъ©ЩҶШҜ ЩҲ ШҜШұ ШөЩҲШұШӘ ШЁШұЩҲШІ CustomerNotFoundExceptionШҢ ЫҢЪ© Щ…ШҙШӘШұЫҢ ЩҫЫҢШҙвҖҢЩҒШұШ¶ ШЁШ§ ЩҶШ§Щ… "Unknown" ШЁШ§ШІЩ…ЫҢвҖҢЪҜШұШҜШ§ЩҶШҜ.

ШЁШ№ШҜ (ШЁШ§ ШЁШұШұШіЫҢ ЩҲШ¬ЩҲШҜ): Ш§ШЁШӘШҜШ§ ШЁШұШұШіЫҢ Щ…ЫҢвҖҢШҙЩҲШҜ Ъ©ЩҮ ШўЫҢШ§ Щ…ШҙШӘШұЫҢ ЩҲШ¬ЩҲШҜ ШҜШ§ШұШҜ ЩҲ ШіЩҫШі Ш§ЩӮШҜШ§Щ… ШЁЩҮ ШҜШұЫҢШ§ЩҒШӘ ШўЩҶ Щ…ЫҢвҖҢШҙЩҲШҜ.

ШЁШ№ШҜ (ШЁШ§ Ш§ШіШӘЩҒШ§ШҜЩҮ Ш§ШІ Ш§Щ„ЪҜЩҲЫҢ Result): Ш§ЫҢЩҶ ШұЩҲЫҢЪ©ШұШҜ Ш§Ш·Щ„Ш§Ш№Ш§ШӘ ШҜЩӮЫҢЩӮвҖҢШӘШұЫҢ ШҜШұ Щ…ЩҲШұШҜ Щ…ЩҲЩҒЩӮЫҢШӘ ЫҢШ§ ШҙЪ©ШіШӘ Ш№Щ…Щ„ЫҢШ§ШӘ Ш§ШұШ§ШҰЩҮ Щ…ЫҢвҖҢШҜЩҮШҜ ЩҲ ЩҮЩ…ЪҶЩҶЫҢЩҶ Щ…ЫҢвҖҢШӘЩҲШ§ЩҶШҜ Ш®Ш·Ш§ЩҮШ§ЫҢ ШәЫҢШұЩ…ЩҶШӘШёШұЩҮ ШҜЫҢЪҜШұ ШұШ§ ЩҶЫҢШІ Щ…ШҜЫҢШұЫҢШӘ Ъ©ЩҶШҜ.
// Assuming Result<T>, Customer class, and _customerRepository exist
public Result<Customer> GetCustomer(int id)
{
    if (!_customerRepository.Exists(id)) // Assumes Exists method
    {
        return Result<Customer>.Failure($"Customer with id {id} not found");
    }
    
    try
    {
        var customer = _customerRepository.Get(id); // Assumes Get method
        return Result<Customer>.Success(customer);
    }
    catch (Exception ex) // Catching broader exceptions for unexpected issues during retrieval
    {
        // Log the exception (e.g., LogError(ex.Message);)
        return Result<Customer>.Failure($"Error retrieving customer: {ex.Message}");
    }
}

ШӘШЈШ«ЫҢШұ ШӘЩҲШіШ№ЩҮ Щ…ШЁШӘЩҶЫҢ ШЁШұ Ш§ШіШӘШ«ЩҶШ§ШЎ ШЁШұ Ъ©ЫҢЩҒЫҢШӘ ЩҶШұЩ…вҖҢШ§ЩҒШІШ§Шұ
Ш§ШіШӘЩҒШ§ШҜЩҮ Ш§ШІ EDD ЪҶЩҶШҜЫҢЩҶ ШӘШЈШ«ЫҢШұ Щ…ЩҶЩҒЫҢ ШЁШұ Ъ©ЫҢЩҒЫҢШӘ Ъ©Щ„ЫҢ ЩҶШұЩ…вҖҢШ§ЩҒШІШ§Шұ ШҜШ§ШұШҜ:
  • Ъ©Ш§ЩҮШҙ ЩӮШ§ШЁЩ„ЫҢШӘ ЩҶЪҜЩҮШҜШ§ШұЫҢ (Maintainability): ШҜШұЪ© ЩҲ ЩҶЪҜЩҮШҜШ§ШұЫҢ Ъ©ШҜЫҢ Ъ©ЩҮ Ш§ШІ Ш§ШіШӘШ«ЩҶШ§ЩҮШ§ ШЁШұШ§ЫҢ Ш¬ШұЫҢШ§ЩҶ Ъ©ЩҶШӘШұЩ„ Ш§ШіШӘЩҒШ§ШҜЩҮ Щ…ЫҢвҖҢЪ©ЩҶШҜШҢ ШҜШҙЩҲШ§ШұШӘШұ Ш§ШіШӘ. ШӘЩҲШіШ№ЩҮвҖҢШҜЩҮЩҶШҜЪҜШ§ЩҶ Ш¬ШҜЫҢШҜ Щ…Щ…Ъ©ЩҶ Ш§ШіШӘ ШҜШұ ШҜЩҶШЁШ§Щ„ Ъ©ШұШҜЩҶ Щ…ШіЫҢШұ Ш§Ш¬ШұШ§ ШҜЪҶШ§Шұ Щ…ШҙЪ©Щ„ ШҙЩҲЩҶШҜ.
  • Ъ©Ш§ЩҮШҙ Ш№Щ…Щ„Ъ©ШұШҜ (Performance): ЩҮЩ…Ш§ЩҶШ·ЩҲШұ Ъ©ЩҮ ЩӮШЁЩ„Ш§ЩӢ Ш°Ъ©Шұ ШҙШҜШҢ Ш§ШіШӘШ«ЩҶШ§ЩҮШ§ ШЁЩҮ Ш·ЩҲШұ ЩӮШ§ШЁЩ„ ШӘЩҲШ¬ЩҮЫҢ ЪҜШұШ§ЩҶвҖҢШӘШұ Ш§ШІ Ш¬ШұЫҢШ§ЩҶ Ъ©ЩҶШӘШұЩ„ Ш№Ш§ШҜЫҢ ЩҮШіШӘЩҶШҜ ЩҲ Щ…ЫҢвҖҢШӘЩҲШ§ЩҶЩҶШҜ Щ…ЩҶШ¬Шұ ШЁЩҮ Ъ©ЩҶШҜЫҢ Щ…ШӯШіЩҲШі ШҜШұ ШЁШұЩҶШ§Щ…ЩҮвҖҢЩҮШ§ЫҢ ШӯШіШ§Ші ШЁЩҮ Ш№Щ…Щ„Ъ©ШұШҜ ШҙЩҲЩҶШҜ.
  • Ш§ШЁЩҮШ§Щ… ШҜШұ ЩӮШөШҜ (Obscured Intent): Ъ©ШҜ Щ…ШЁШӘЩҶЫҢ ШЁШұ Ш§ШіШӘШ«ЩҶШ§ШЎ Ш§ШәЩ„ШЁ ЩӮШөШҜ Ш§ШөЩ„ЫҢ ШӘЩҲШіШ№ЩҮвҖҢШҜЩҮЩҶШҜЩҮ ШұШ§ ЩҫЩҶЩҮШ§ЩҶ Щ…ЫҢвҖҢЪ©ЩҶШҜ.
  • Ъ©ШҜ ШҙЪ©ЩҶЩҶШҜЩҮ (Brittle Code): EDD Ш§ШәЩ„ШЁ Щ…ЩҶШ¬Шұ ШЁЩҮ Ъ©ШҜЫҢ ШҙЪ©ЩҶЩҶШҜЩҮ Щ…ЫҢвҖҢШҙЩҲШҜ Ъ©ЩҮ ШЁШ§ ШӘШәЫҢЫҢШұ ЩҫЫҢШ§ШҜЩҮвҖҢШіШ§ШІЫҢвҖҢЩҮШ§ЫҢ ШІЫҢШұШЁЩҶШ§ЫҢЫҢ ШЁЩҮ ШұЩҲШҙвҖҢЩҮШ§ЫҢ ШәЫҢШұЩ…ЩҶШӘШёШұЩҮвҖҢШ§ЫҢ Щ…ЫҢвҖҢШҙЪ©ЩҶШҜ. ШЁШұШ§ЫҢ Щ…Ш«Ш§Щ„ШҢ Ш§ЪҜШұ ЫҢЪ© API ШҜЫҢЪҜШұ ЫҢЪ© Ш§ШіШӘШ«ЩҶШ§ШЎ Ш®Ш§Шө ШұШ§ ЩҫШұШӘШ§ШЁ ЩҶЪ©ЩҶШҜШҢ Ъ©ШҜЫҢ Ъ©ЩҮ ШЁЩҮ ЪҜШұЩҒШӘЩҶ ШўЩҶ Ш§ШіШӘШ«ЩҶШ§ШЎ Щ…ШӘЪ©ЫҢ Ш§ШіШӘШҢ ЩҶШ§ШҜШұШіШӘ ШұЩҒШӘШ§Шұ Ш®ЩҲШ§ЩҮШҜ Ъ©ШұШҜ.
