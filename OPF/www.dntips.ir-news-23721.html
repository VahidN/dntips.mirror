‫ساخت داشبورد تحلیل احساسات Bluesky با Alpine و هوش مصنوعی Chrome


نویسنده: راویAI
تاریخ: ۱۴۰۴/۱۱/۱۷ ۲۰:۱۸
آدرس: www.dntips.ir
در این مقاله، روش ساخت یک داشبورد تحلیل احساسات برای Bluesky با استفاده از Alpine.js و هوش مصنوعی Chrome توضیح داده شده است. این داشبورد به کاربران اجازه می‌دهد تا با وارد کردن یک عبارت، گزارش مربوط به احساسات عمومی پست‌های حاوی آن عبارت را مشاهده کنند. نویسنده قبلاً این پروژه را با استفاده از Transformers.js و برای Twitter نیز پیاده‌سازی کرده بود. هدف اصلی این پروژه، آزمایش و استفاده از ابزارهای هوش مصنوعی Chrome در یک محیط عملی است. این پروژه از Alpine.js برای مدیریت رابط کاربری و Shoelace (که در حال حاضر با Web Awesome جایگزین شده است) برای ایجاد کامپوننت‌های وب استفاده می‌کند. همچنین، از Prompt API هوش مصنوعی Chrome برای تحلیل احساسات متن استفاده می‌شود. در این مقاله، ساختار HTML داشبورد به طور خلاصه توضیح داده شده است. داشبورد شامل یک هدر برای نمایش وضعیت کار و افزودن کلمات کلیدی، و مجموعه‌ای از کارت‌ها برای نمایش نتایج تحلیل احساسات (احساسات، تعداد پست‌ها، زمان آخرین و اولین پست) است. همچنین یک فرم برای افزودن کلمات کلیدی جدید وجود دارد. یکی از چالش‌های اصلی، تغییر در API جستجوی Bluesky است. endpoint قبلی (unauthenticated) دیگر کار نمی‌کند و endpoint احراز هویت (authenticated) باید استفاده شود. این endpoint حتی بدون احراز هویت نیز قابل استفاده است. نویسنده قصد دارد در آینده، نمایش احساسات (خوب یا بد) را به صورت مستقیم در داده‌ها مدیریت کند تا HTML ساده‌تر شود. این پروژه یک نمونه کاربردی از استفاده از هوش مصنوعی Chrome برای تحلیل احساسات در یک برنامه وب است و نشان می‌دهد که چگونه می‌توان با استفاده از ابزارهای ساده، داشبوردهای قدرتمندی ایجاد کرد.
