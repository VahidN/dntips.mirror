Ш§Щ…ЩҶЫҢШӘ ШўЩҫЩ„ЩҲШҜ ЩҒШ§ЫҢЩ„вҖҢЩҮШ§ ЫҢЪ©ЫҢ Ш§ШІ ЪҶШ§Щ„ШҙвҖҢЩҮШ§ЫҢ Ш§ШіШ§ШіЫҢ ШҜШұ ШӘЩҲШіШ№ЩҮ ШЁШұЩҶШ§Щ…ЩҮвҖҢЩҮШ§ЫҢ ЩҲШЁШҢ ШЁЩҮвҖҢЩҲЫҢЪҳЩҮ ШҜШұ ASP.NET Core Ш§ШіШӘ. Щ…ЩҮШ§Ш¬Щ…Ш§ЩҶ Щ…ЫҢвҖҢШӘЩҲШ§ЩҶЩҶШҜ Ш§ШІ Ш¶Ш№ЩҒвҖҢЩҮШ§ЫҢ Ш§ЫҢЩҶ ШЁШ®Шҙ ШЁШұШ§ЫҢ ШӘШІШұЫҢЩӮ Ъ©ШҜЩҮШ§ЫҢ Щ…Ш®ШұШЁ ЫҢШ§ ШЁШ§ШұЪҜШ°Ш§ШұЫҢ ЩҒШ§ЫҢЩ„вҖҢЩҮШ§ЫҢ ШўЩ„ЩҲШҜЩҮ ШіЩҲШЎШ§ШіШӘЩҒШ§ШҜЩҮ Ъ©ЩҶЩҶШҜ. ШҜШұ ШӯШ§Щ„ЫҢ Ъ©ЩҮ Ш§ЩӮШҜШ§Щ…Ш§ШӘЫҢ ЩҶШёЫҢШұ ШЁШұШұШіЫҢ ЩҫШіЩҲЩҶШҜШҢ Щ…ШӯШӘЩҲШ§ (MIME Type) ЩҲ Щ…ШӯШҜЩҲШҜЫҢШӘ Ш§ЩҶШҜШ§ШІЩҮ ЩҒШ§ЫҢЩ„ Ш¶ШұЩҲШұЫҢ ЩҮШіШӘЩҶШҜШҢ Ш§Щ…Ш§ ЫҢЪ© Ш®Ш· ШҜЩҒШ§Ш№ЫҢ ЩҶЩҮШ§ЫҢЫҢ ШЁШұШ§ЫҢ Щ…ЩӮШ§ШЁЩ„ЩҮ ШЁШ§ ШӘЩҮШҜЫҢШҜШ§ШӘ ЩҫЫҢШҙШұЩҒШӘЩҮвҖҢШӘШұ ЩҲШ¬ЩҲШҜ ШҜШ§ШұШҜ: Ш§ШіЪ©ЩҶ ШЁШҜШ§ЩҒШІШ§Шұ Щ…ШӯШӘЩҲШ§-Щ…ШӯЩҲШұШҢ Ъ©ЩҮ ШҜШұ ШіЫҢШіШӘЩ…вҖҢШ№Ш§Щ…Щ„ ЩҲЫҢЩҶШҜЩҲШІ ШӘЩҲШіШ· ШұШ§ШЁШ· Ш§ШіЪ©ЩҶ Ш¶ШҜ ШЁШҜШ§ЩҒШІШ§Шұ ЩҲЫҢЩҶШҜЩҲШІ (AMSI) ШЁЩҮвҖҢШұШ§ШӯШӘЫҢ ЩӮШ§ШЁЩ„ ЩҫЫҢШ§ШҜЩҮвҖҢШіШ§ШІЫҢ Ш§ШіШӘ.

ЪҶШұШ§ ШЁЩҮ Ш§ШіЪ©ЩҶ ШЁШҜШ§ЩҒШІШ§Шұ ЩҶЫҢШ§ШІ ШҜШ§ШұЫҢЩ…Шҹ ШўШ®ШұЫҢЩҶ Ш®Ш· ШҜЩҒШ§Ш№ЫҢ

ШӯШӘЫҢ ЩҫШі Ш§ШІ Ш§Ш¬ШұШ§ЫҢ ШҜЩӮЫҢЩӮ Ъ©Щ„ЫҢЩҮ Щ…ШұШ§ШӯЩ„ Ш§Ш№ШӘШЁШ§ШұШіЩҶШ¬ЫҢ Ш§ЩҲЩ„ЫҢЩҮ (ШҙШ§Щ…Щ„ ШЁШұШұШіЫҢ ЩҶЩҲШ№ ЩҒШ§ЫҢЩ„ШҢ Ш§ЩҶШҜШ§ШІЩҮ ЩҲ ШәЫҢШұЩҮ)ШҢ ЫҢЪ© Щ…ЩҮШ§Ш¬Щ… Щ…ШөЩ…Щ… ЩҮЩ…ЪҶЩҶШ§ЩҶ Щ…ЫҢвҖҢШӘЩҲШ§ЩҶШҜ Щ…ШӯШӘЩҲШ§ЫҢ Щ…Ш®ШұШЁ ШұШ§ ШўЩҫЩ„ЩҲШҜ Ъ©ЩҶШҜ. Ш§ЫҢЩҶ ЩҒШ§ЫҢЩ„вҖҢЩҮШ§ Ш§ШІ Ш§Ш№ШӘШЁШ§ШұШіЩҶШ¬ЫҢвҖҢЩҮШ§ЫҢ ЩӮШЁЩ„ЫҢ Ш№ШЁЩҲШұ Щ…ЫҢвҖҢЪ©ЩҶЩҶШҜ ШІЫҢШұШ§ ШҜШұ ШёШ§ЩҮШұ ЩӮШ§Щ„ШЁвҖҢЩҮШ§ЫҢЫҢ ЩӮШ§ЩҶЩҲЩҶЫҢ ШҜШ§ШұЩҶШҜШҢ Ш§Щ…Ш§ ШҜШұ ЩҲШ§ЩӮШ№ЫҢШӘ Щ…ШіЩ„Шӯ ШҙШҜЩҮвҖҢШ§ЩҶШҜ:
  • ЩҒШ§ЫҢЩ„ PDF ЩӮШ§ЩҶЩҲЩҶЫҢ ШӯШ§ЩҲЫҢ ШіЩҲШЎШ§ШіШӘЩҒШ§ШҜЩҮвҖҢЩҮШ§ЫҢ Ш¬Ш§ШіШ§ШІЫҢвҖҢШҙШҜЩҮ Ш¬Ш§ЩҲШ§Ш§ШіЪ©ШұЫҢЩҫШӘ.
  • ШіЩҶШҜ ШўЩҒЫҢШі Щ…Ш№ШӘШЁШұ ШӯШ§ЩҲЫҢ Щ…Ш§Ъ©ШұЩҲЩҮШ§ЫҢ Щ…Ш®ШұШЁ (Macro-based Malware).
  • ЩҒШ§ЫҢЩ„ ШӘШөЩҲЫҢШұЫҢ ЩҲШ§ЩӮШ№ЫҢ ШЁШ§ ШЁШ§ШұЪҜШ°Ш§ШұЫҢвҖҢЩҮШ§ЫҢ (Payloads) Ш¬Ш§ШіШ§ШІЫҢвҖҢШҙШҜЩҮ Ш§ШІ Ш·ШұЫҢЩӮ ЩҶЩҮШ§ЩҶвҖҢЩҶЪҜШ§ШұЫҢ (Steganography).
  • ШЁШ§ЫҢЪҜШ§ЩҶЫҢвҖҢЩҮШ§ЫҢ (Archive) ЩҲШ§ЩӮШ№ЫҢ ШӯШ§ЩҲЫҢ ШЁШҜШ§ЩҒШІШ§Шұ.
  • ШӯЩ…Щ„Ш§ШӘ ШұЩҲШІ ШөЩҒШұ (Zero-day) Ъ©ЩҮ Ъ©ШӘШ§ШЁШ®Ш§ЩҶЩҮвҖҢЩҮШ§ЫҢ ЩҫШұШҜШ§ШІШҙ ЩҒШ§ЫҢЩ„ ШұШ§ ЩҮШҜЩҒ ЩӮШұШ§Шұ Щ…ЫҢвҖҢШҜЩҮЩҶШҜ.
ШҜШұ Ш§ЫҢЩҶ ШіЩҶШ§ШұЫҢЩҲЩҮШ§ШҢ Ш§ШіЪ©ЩҶ ШЁШҜШ§ЩҒШІШ§Шұ ШЁЩҮ ЫҢЪ© Ш§Щ…Шұ ШӯЫҢШ§ШӘЫҢ ШӘШЁШҜЫҢЩ„ Щ…ЫҢвҖҢШҙЩҲШҜШҢ ЪҶШұШ§ Ъ©ЩҮ Щ…ШӯШӘЩҲШ§ЫҢ ШҜШұЩҲЩҶЫҢ ЩҒШ§ЫҢЩ„ ШұШ§ ЩӮШЁЩ„ Ш§ШІ ЩҫШұШҜШ§ШІШҙ ЫҢШ§ Ш°Ш®ЫҢШұЩҮвҖҢШіШ§ШІЫҢ ШӘШ¬ШІЫҢЩҮ ЩҲ ШӘШӯЩ„ЫҢЩ„ Щ…ЫҢвҖҢЪ©ЩҶШҜ.

AMSI ЪҶЫҢШіШӘ ЩҲ ЪҶШұШ§ ШЁШұШ§ЫҢ ASP.NET Core ШұЩҲЫҢ ЩҲЫҢЩҶШҜЩҲШІ Ш№Ш§Щ„ЫҢ Ш§ШіШӘШҹ

ШұШ§ШЁШ· Ш§ШіЪ©ЩҶ Ш¶ШҜ ШЁШҜШ§ЩҒШІШ§Шұ ЩҲЫҢЩҶШҜЩҲШІ (Windows Antimalware Scan Interface - AMSI) ЫҢЪ© ЩҲЫҢЪҳЪҜЫҢ ЩӮШҜШұШӘЩ…ЩҶШҜ ЩҲЫҢЩҶШҜЩҲШІ Ш§ШіШӘ Ъ©ЩҮ ШЁЩҮ ШЁШұЩҶШ§Щ…ЩҮвҖҢЩҮШ§ Ш§Ш¬Ш§ШІЩҮ Щ…ЫҢвҖҢШҜЩҮШҜ ШЁШ§ Щ…ЩҲШӘЩҲШұ ШўЩҶШӘЫҢвҖҢЩҲЫҢШұЩҲШі ЩҶШөШЁ ШҙШҜЩҮ ШұЩҲЫҢ ШіЫҢШіШӘЩ… (Щ…Ш§ЩҶЩҶШҜ Windows Defender ЫҢШ§ ЩҮШұ ШўЩҶШӘЫҢвҖҢЩҲЫҢШұЩҲШі ШіШ§ШІЪҜШ§Шұ ШҜЫҢЪҜШұ) Ш§ШҜШәШ§Щ… ШҙЩҲЩҶШҜ.
AMSI Щ…ШҙЪ©Щ„ ШұШ§ЫҢШ¬ЫҢ ШұШ§ ШӯЩ„ Щ…ЫҢвҖҢЪ©ЩҶШҜ: ШўЩҶШӘЫҢвҖҢЩҲЫҢШұЩҲШівҖҢЩҮШ§ЫҢ ШіЩҶШӘЫҢ Ш№Щ…ШҜШӘШ§ЩӢ ЩҒШ§ЫҢЩ„вҖҢЩҮШ§ЫҢ ШұЩҲЫҢ ШҜЫҢШіЪ© ШұШ§ Ш§ШіЪ©ЩҶ Щ…ЫҢвҖҢЪ©ЩҶЩҶШҜ. Ш§Щ…Ш§ ШҜШұ ШЁШіЫҢШ§ШұЫҢ Ш§ШІ ШіЩҶШ§ШұЫҢЩҲЩҮШ§ЫҢ Щ…ШҜШұЩҶ (Щ…Ш§ЩҶЩҶШҜ PowerShellШҢ ШЁШұЩҶШ§Щ…ЩҮвҖҢЩҮШ§ЫҢ .NET Ъ©ЩҮ Ш§ШіЩ…ШЁЩ„ЫҢвҖҢЩҮШ§ ШұШ§ ШЁЩҮвҖҢШөЩҲШұШӘ ЩҫЩҲЫҢШ§ ШЁШ§ШұЪҜШ°Ш§ШұЫҢ Щ…ЫҢвҖҢЪ©ЩҶЩҶШҜШҢ ЫҢШ§ ШЁШұЩҶШ§Щ…ЩҮвҖҢЩҮШ§ЫҢ Node.js)ШҢ Щ…ШӯШӘЩҲШ§ЫҢ Щ…Ш®ШұШЁ Щ…Щ…Ъ©ЩҶ Ш§ШіШӘ Щ…ШіШӘЩӮЫҢЩ…Ш§ЩӢ ШҜШұ ШӯШ§ЩҒШёЩҮ (Memory Stream) ШҜШ§ЩҶЩ„ЩҲШҜ ЩҲ Ш§ШіШӘЩҒШ§ШҜЩҮ ШҙЩҲШҜ. AMSI ШЁЩҮ ШЁШұЩҶШ§Щ…ЩҮ ШҙЩ…Ш§ Ш§ЫҢЩҶ ШӘЩҲШ§ЩҶШ§ЫҢЫҢ ШұШ§ Щ…ЫҢвҖҢШҜЩҮШҜ Ъ©ЩҮ ЫҢЪ© Ш¬ШұЫҢШ§ЩҶ ЫҢШ§ ЫҢЪ© ШұШҙШӘЩҮ ШҜШ§ШҜЩҮ ШұШ§ ШЁЩҮвҖҢШөЩҲШұШӘ Щ…ШіШӘЩӮЫҢЩ… ШЁШұШ§ЫҢ ШӘШ¬ШІЫҢЩҮ ЩҲ ШӘШӯЩ„ЫҢЩ„ ШЁЩҮ ШўЩҶШӘЫҢвҖҢЩҲЫҢШұЩҲШі Ш§ШұШіШ§Щ„ Ъ©ЩҶШҜ.

Щ…ШІШ§ЫҢШ§ЫҢ Ш§ШіШӘЩҒШ§ШҜЩҮ Ш§ШІ AMSI

Щ…ШІЫҢШӘШӘЩҲШ¶ЫҢШӯ
Ш§ШіШӘЩҒШ§ШҜЩҮ Ш§ШІ ШўЩҶШӘЫҢвҖҢЩҲЫҢШұЩҲШі Щ…ЩҲШ¬ЩҲШҜШ§ШІ Windows Defender ЫҢШ§ ЩҮШұ ШўЩҶШӘЫҢвҖҢЩҲЫҢШұЩҲШі ШіШ§ШІЪҜШ§Шұ ШЁШ§ AMSI Ъ©ЩҮ ШҜШұ ШӯШ§Щ„ ШӯШ§Ш¶Шұ ЩҶШөШЁ ШҙШҜЩҮШҢ ШЁЩҮШұЩҮ Щ…ЫҢвҖҢШЁШұШҜ.
ШЁЩҮвҖҢШұЩҲШІШұШіШ§ЩҶЫҢ Ш®ЩҲШҜЪ©Ш§ШұШ§Щ…Ш¶Ш§ЩҮШ§ЫҢ ШӘШҙШ®ЫҢШө ШЁШҜШ§ЩҒШІШ§Шұ ШЁЩҮвҖҢШ·ЩҲШұ Ш®ЩҲШҜЪ©Ш§Шұ ШӘЩҲШіШ· Windows Update ШЁЩҮвҖҢШұЩҲШІ Щ…ЫҢвҖҢШҙЩҲЩҶШҜ.
ШЁШҜЩҲЩҶ ЩҮШІЫҢЩҶЩҮ Ш§Ш¶Ш§ЩҒЫҢШ§ШіШӘЩҒШ§ШҜЩҮ Ш§ШІ ШўЩҶ ШҜШұ ШіЫҢШіШӘЩ…вҖҢШ№Ш§Щ…Щ„вҖҢЩҮШ§ЫҢ ЩҲЫҢЩҶШҜЩҲШІ ШұШ§ЫҢЪҜШ§ЩҶ Ш§ШіШӘ ЩҲ ЩҶЫҢШ§ШІЫҢ ШЁЩҮ Щ…Ш¬ЩҲШІ Ш¬ШҜШ§ЪҜШ§ЩҶЩҮ ЩҶШҜШ§ШұШҜ.
Ш§ШҜШәШ§Щ… ШіШЁЪ©вҖҢЩҲШІЩҶШҜШ§ШұШ§ЫҢ API ШіШ§ШҜЩҮ ШЁШ§ ШіШұШЁШ§Шұ (Overhead) ШӯШҜШ§ЩӮЩ„ЫҢ.
Ш§ШіЪ©ЩҶ Щ…ШӯШӘЩҲШ§-Щ…ШӯЩҲШұЩ…ШӯШӘЩҲШ§ЫҢ ЩҲШ§ЩӮШ№ЫҢ ЩҒШ§ЫҢЩ„ ШұШ§ ШӘШ¬ШІЫҢЩҮ ЩҲ ШӘШӯЩ„ЫҢЩ„ Щ…ЫҢвҖҢЪ©ЩҶШҜШҢ ЩҶЩҮ ШөШұЩҒШ§ЩӢ Ш§Щ…Ш¶Ш§ЩҮШ§ЫҢ ШіШ·Шӯ ШЁШ§Щ„Ш§ЫҢ ЩҒШ§ЫҢЩ„ ШұШ§.
Ш§ШіШӘШ§ЩҶШҜШ§ШұШҜ ШөЩҶШ№ШӘЫҢШ§ШІ ЩҮЩҲШҙ ШӘЩҮШҜЫҢШҜШ§ШӘ ЪҜШіШӘШұШҜЩҮ Щ…Ш§ЫҢЪ©ШұЩҲШіШ§ЩҒШӘ ШЁЩҮШұЩҮ Щ…ЫҢвҖҢШЁШұШҜ.

Щ…ШӯШҜЩҲШҜЫҢШӘвҖҢЩҮШ§:
  • Ш§ЩҶШӯШөШ§ШұЫҢ ШЁШұШ§ЫҢ ЩҲЫҢЩҶШҜЩҲШІ: AMSI ЩҒЩӮШ· ШҜШұ ШіЫҢШіШӘЩ…вҖҢШ№Ш§Щ…Щ„ ЩҲЫҢЩҶШҜЩҲШІ ЩӮШ§ШЁЩ„ Ш§ШіШӘЩҒШ§ШҜЩҮ Ш§ШіШӘ. (ШЁШұШ§ЫҢ ЩҶЫҢШ§ШІЩҮШ§ЫҢ Ъ©ШұШ§ШівҖҢЩҫЩ„ШӘЩҒШұЩ…ШҢ Щ…ЫҢвҖҢШӘЩҲШ§ЩҶ ClamAV ЫҢШ§ ШіШұЩҲЫҢШівҖҢЩҮШ§ЫҢ Ш§ШЁШұЫҢ ШұШ§ ШҜШұ ЩҶШёШұ ЪҜШұЩҒШӘ).
  • ЩҶЫҢШ§ШІ ШЁЩҮ ШўЩҶШӘЫҢвҖҢЩҲЫҢШұЩҲШі ШіШ§ШІЪҜШ§Шұ: ЩҶШөШЁ ШЁЩҲШҜЩҶ ЫҢЪ© Щ…ЩҲШӘЩҲШұ ШўЩҶШӘЫҢвҖҢЩҲЫҢШұЩҲШі ШіШ§ШІЪҜШ§Шұ ШЁШ§ AMSI Ш¶ШұЩҲШұЫҢ Ш§ШіШӘ.
  • ШәЫҢШұЩӮШ§ШЁЩ„ ЩҶЩҒЩҲШ° ЩҶШЁЩҲШҜЩҶ: Щ…Ш§ЩҶЩҶШҜ ЩҮШұ ШұШ§ЩҮЪ©Ш§Шұ ШўЩҶШӘЫҢвҖҢЩҲЫҢШұЩҲШіШҢ ШӘЩҲШ§ЩҶШ§ЫҢЫҢ ШӘШҙШ®ЫҢШө ШӘЩҮШҜЫҢШҜШ§ШӘ ШұЩҲШІ ШөЩҒШұ Ъ©Ш§Щ…Щ„Ш§ЩӢ ЩҶШ§ШҙЩҶШ§Ш®ШӘЩҮ ШұШ§ ЩҶШҜШ§ШұШҜ.

ЩҫЫҢШ§ШҜЩҮвҖҢШіШ§ШІЫҢ Ш§ШіЪ©ЩҶ ШЁШҜШ§ЩҒШІШ§Шұ ШҜШұ ASP.NET Core ШЁШ§ AMSI

ШЁШұШ§ЫҢ ЩҫЫҢШ§ШҜЩҮвҖҢШіШ§ШІЫҢ Ш§ЫҢЩҶ ЩӮШ§ШЁЩ„ЫҢШӘ ШҜШұ ASP.NET CoreШҢ Щ…ЫҢвҖҢШӘЩҲШ§ЩҶЫҢЩ… ЫҢЪ© Щ…ШұШӯЩ„ЩҮ Ш§Ш№ШӘШЁШ§ШұШіЩҶШ¬ЫҢ (Validation Step) Щ…Ш¬ШІШ§ ШӘШ№ШұЫҢЩҒ Ъ©ЩҶЫҢЩ… Ъ©ЩҮ Ш§ШІ Ъ©ШӘШ§ШЁШ®Ш§ЩҶЩҮвҖҢШ§ЫҢ ШЁШұШ§ЫҢ ШӘШ№Ш§Щ…Щ„ ШЁШ§ AMSI Ш§ШіШӘЩҒШ§ШҜЩҮ Щ…ЫҢвҖҢЪ©ЩҶШҜ.
AMSI ШҜЩҲ Щ…ШӘШҜ Ш§ШөЩ„ЫҢ ШЁШұШ§ЫҢ Ш§ШіЪ©ЩҶ Ш§ШұШ§ШҰЩҮ Щ…ЫҢвҖҢШҜЩҮШҜ: ЫҢЪ©ЫҢ ШЁШұШ§ЫҢ Ш§ШіЪ©ЩҶ ЫҢЪ© ШұШҙШӘЩҮ (String) ЩҲ ШҜЫҢЪҜШұЫҢ ШЁШұШ§ЫҢ Ш§ШіЪ©ЩҶ ЫҢЪ© ШЁЩ„ЩҲЪ© ШҜШ§ШҜЩҮ ШЁШ§ЫҢЩҶШұЫҢ (Binary Blob)ШҢ Ъ©ЩҮ Щ…Ш§ Ш§ШІ Щ…ЩҲШұШҜ ШҜЩҲЩ… ШЁШұШ§ЫҢ ЩҒШ§ЫҢЩ„вҖҢЩҮШ§ Ш§ШіШӘЩҒШ§ШҜЩҮ Щ…ЫҢвҖҢЪ©ЩҶЫҢЩ….

Ыұ. Ъ©Щ„Ш§Ші ЩҫЫҢШ§ШҜЩҮвҖҢШіШ§ШІЫҢ Щ…ЩҶШ·ЩӮ Ш§ШіЪ©ЩҶ

Ъ©Щ„Ш§Ші ШІЫҢШұШҢ Щ…ЩҶШ·ЩӮ Ш§ШөЩ„ЫҢ Ш§Ш№ШӘШЁШ§ШұШіЩҶШ¬ЫҢ ШұШ§ ЩҶШҙШ§ЩҶ Щ…ЫҢвҖҢШҜЩҮШҜ Ъ©ЩҮ ШҜШ§ШҜЩҮвҖҢЩҮШ§ЫҢ ЩҒШ§ЫҢЩ„ ШұШ§ ШҜШұЫҢШ§ЩҒШӘ Ъ©ШұШҜЩҮ ЩҲ ШўЩҶвҖҢЩҮШ§ ШұШ§ ШЁЩҮ Щ…ЩҲШӘЩҲШұ AMSI Ш§ШұШіШ§Щ„ Щ…ЫҢвҖҢЪ©ЩҶШҜ.
public sealed class MalwareScanValidationStep
{
    private readonly ILogger _logger;

    public MalwareScanValidationStep(ILogger logger)
    {
        _logger = logger;
    }

    public async Task<ValidationResult> ValidateAsync(IFormFile file, CancellationToken token)
    {
        // ЩҒШұШ¶ Ъ©ЩҶЫҢШҜ Щ…ШӘШҜ GetData() ЫҢЪ© Щ…ШӘШҜ Extension ШЁШұШ§ЫҢ Ш®ЩҲШ§ЩҶШҜЩҶ Щ…ШӯШӘЩҲШ§ЫҢ IFormFile ШЁЩҮ ШөЩҲШұШӘ ШўШұШ§ЫҢЩҮ ШЁШ§ЫҢШӘ Ш§ШіШӘ.
        var payload = await file.GetData(); 
        
        // 1. Ш§ЫҢШ¬Ш§ШҜ ЫҢЪ© Context Ш¬ШҜЫҢШҜ AMSI
        using var application = AmsiContext.Create("DocumentStorage"); 
        
        // 2. Ш§ЫҢШ¬Ш§ШҜ ЫҢЪ© Session Ш¬ШҜЫҢШҜ (ШЁШұШ§ЫҢ Щ…ШұШӘШЁШ· Ъ©ШұШҜЩҶ ШҜШұШ®ЩҲШ§ШіШӘвҖҢЩҮШ§ЫҢ Ш§ШіЪ©ЩҶ)
        using var session = application.CreateSession(); 
        
        // 3. Ш§ШіЪ©ЩҶ Щ…ШӯШӘЩҲШ§ЫҢ ЩҒШ§ЫҢЩ„
        var isMalware = await session.IsPotentialMalwareAsync(payload, file.Name, token);
        
        if (isMalware)
        {
            _logger.LogWarning("MalwareScan: File '{FileName}' potentially contains malware.", file.FileName);
            // ШҜШұ ШөЩҲШұШӘ ШӘШҙШ®ЫҢШө ШЁШҜШ§ЩҒШІШ§ШұШҢ Ш№Щ…Щ„ЫҢШ§ШӘ ШұШ§ ШЁШ§ Ш®Ш·Ш§ Щ…ШӘЩҲЩӮЩҒ Щ…ЫҢвҖҢЪ©ЩҶЫҢЩ….
            return ValidationResult.Fail(new BadRequestObjectResult(new { message = "Document potentially contains malware." }));
        }

        return ValidationResult.Success();
    }
}

ЫІ. Ъ©Щ„Ш§ШівҖҢЩҮШ§ЫҢ ШӘШ№Ш§Щ…Щ„ ШЁШ§ AMSI (AmsiContext ЩҲ AmsiSession)

ШЁШұШ§ЫҢ ШӘШ№Ш§Щ…Щ„ ШЁШ§ AMSIШҢ Ш§ШІ ЩӮШ§ШЁЩ„ЫҢШӘ Platform Invoke (P/Invoke) ШҜШұ .NET ШЁШұШ§ЫҢ ЩҒШұШ§Ш®ЩҲШ§ЩҶЫҢ ШӘЩҲШ§ШЁШ№ ШәЫҢШұЩ…ШҜЫҢШұЫҢШӘвҖҢШҙШҜЩҮ (Unmanaged Functions) Щ…ЩҲШ¬ЩҲШҜ ШҜШұ amsi.dll Ш§ШіШӘЩҒШ§ШҜЩҮ Щ…ЫҢвҖҢЪ©ЩҶЫҢЩ….
Ш§ЫҢЩҶ Ъ©Щ„Ш§Ші ЩҲШёЫҢЩҒЩҮ Щ…ЩӮШҜШ§ШұШҜЩҮЫҢ Ш§ЩҲЩ„ЫҢЩҮ (AmsiInitialize) ЩҲ Ш§ШӘЩ…Ш§Щ… (AmsiUninitialize) ШұШ§ШЁШ· AMSI ШЁШұШ§ЫҢ ШЁШұЩҶШ§Щ…ЩҮ Щ…Ш§ ШұШ§ ШЁШұ Ш№ЩҮШҜЩҮ ШҜШ§ШұШҜ ЩҲ ЫҢЪ© ШҜШіШӘЩҮ (Handle) Ш§ШөЩ„ЫҢ ШЁШұШ§ЫҢ Ъ©Щ„ ШЁШұЩҶШ§Щ…ЩҮ Ш§ЫҢШ¬Ш§ШҜ Щ…ЫҢвҖҢЪ©ЩҶШҜ.
public sealed class AmsiContext : IDisposable
{
    private IntPtr _handle;

    private AmsiContext(IntPtr handle)
    {
        _handle = handle;
    }

    public static AmsiContext Create(string appName)
    {
        var result = AmsiInitialize(appName, out var handle);
        if (result != 0)
        {
            throw new InvalidOperationException($"Failed to initialize AMSI. Error code: {result}");
        }
        return new AmsiContext(handle);
    }

    public AmsiSession CreateSession()
    {
        var result = AmsiOpenSession(_handle, out var sessionHandle);
        if (result != 0)
        {
            throw new InvalidOperationException($"Failed to open AMSI session. Error code: {result}");
        }
        return new AmsiSession(_handle, sessionHandle);
    }

    public void Dispose()
    {
        if (_handle != IntPtr.Zero)
        {
            AmsiUninitialize(_handle);
            _handle = IntPtr.Zero;
        }
    }

    [DllImport("amsi.dll", CharSet = CharSet.Unicode)]
    private static extern int AmsiInitialize(string appName, out IntPtr amsiContext);

    [DllImport("amsi.dll")]
    private static extern int AmsiOpenSession(IntPtr amsiContext, out IntPtr session);

    [DllImport("amsi.dll")]
    private static extern void AmsiUninitialize(IntPtr amsiContext);
}

AmsiSession

Ш§ЫҢЩҶ Ъ©Щ„Ш§Ші ЫҢЪ© Ш¬Щ„ШіЩҮ Ш§ШіЪ©ЩҶ (AmsiOpenSession/AmsiCloseSession) Ш§ЫҢШ¬Ш§ШҜ Щ…ЫҢвҖҢЪ©ЩҶШҜ ЩҲ Щ…ЩҮЩ…вҖҢШӘШұЫҢЩҶ Щ…ШӘШҜШҢ ЫҢШ№ЩҶЫҢ AmsiScanBuffer ШұШ§ ЩҒШұШ§Ш®ЩҲШ§ЩҶЫҢ Щ…ЫҢвҖҢЪ©ЩҶШҜ Ъ©ЩҮ ШҜШ§ШҜЩҮвҖҢЩҮШ§ЫҢ ШЁШ§ЫҢЩҶШұЫҢ ШұШ§ ШЁШұШ§ЫҢ Ш§ШіЪ©ЩҶ ШЁЩҮ ШўЩҶШӘЫҢвҖҢЩҲЫҢШұЩҲШі Ш§ШұШіШ§Щ„ Щ…ЫҢвҖҢЪ©ЩҶШҜ.
public sealed class AmsiSession : IDisposable
{
    private readonly IntPtr _amsiContext;
    private IntPtr _sessionHandle;

    internal AmsiSession(IntPtr amsiContext, IntPtr sessionHandle)
    {
        _amsiContext = amsiContext;
        _sessionHandle = sessionHandle;
    }

    public async Task<bool> IsPotentialMalwareAsync(byte[] data, string filename, CancellationToken cancellationToken = default)
    {
        return await Task.Run(() => IsPotentialMalware(data, filename), cancellationToken);
    }

    private bool IsPotentialMalware(byte[] data, string filename)
    {
        var result = AmsiScanBuffer(_amsiContext, data, (uint)data.Length, filename, _sessionHandle, out var scanResult);
        if (result != 0)
        {
            throw new InvalidOperationException($"Failed to scan buffer. Error code: {result}");
        }

        // AMSI_RESULT values:
        // 0 = Clean
        // 1-32767 = Detected (higher = more confident detection)
        // 32768+ = Blocked
        return scanResult >= 1;
    }

    public void Dispose()
    {
        if (_sessionHandle != IntPtr.Zero)
        {
            AmsiCloseSession(_amsiContext, _sessionHandle);
            _sessionHandle = IntPtr.Zero;
        }
    }

    [DllImport("amsi.dll")]
    private static extern int AmsiScanBuffer(IntPtr amsiContext, byte[] buffer, uint length, string contentName, IntPtr session, out int result);

    [DllImport("amsi.dll")]
    private static extern void AmsiCloseSession(IntPtr amsiContext, IntPtr session);
}

ШұШ§ЩҮШЁШұШҜ ШӘШіШӘ (Testing Strategy)

ШӘШіШӘ Ш§ШіЪ©ЩҶ ШЁШҜШ§ЩҒШІШ§Шұ Щ…ЫҢвҖҢШӘЩҲШ§ЩҶШҜ ЩҫЫҢЪҶЫҢШҜЩҮ ШЁШ§ШҙШҜ. ШЁЩҮШӘШұЫҢЩҶ ШұЩҲЫҢЪ©ШұШҜШҢ Ш§ШіШӘЩҒШ§ШҜЩҮ Ш§ШІ ЩҒШ§ЫҢЩ„ ШӘШіШӘ EICAR Ш§ШіШӘ. ЩҒШ§ЫҢЩ„ ШӘШіШӘ EICAR ЫҢЪ© ЩҒШ§ЫҢЩ„ Ш§ШіШӘШ§ЩҶШҜШ§ШұШҜ Ш¶ШҜ ШЁШҜШ§ЩҒШІШ§Шұ Ш§ШіШӘ Ъ©ЩҮ ШӘЩҲШіШ· ЩҮЩ…ЩҮ Щ…ЩҲШӘЩҲШұЩҮШ§ЫҢ ШўЩҶШӘЫҢвҖҢЩҲЫҢШұЩҲШі ШЁШІШұЪҜ ШЁЩҮвҖҢШ№ЩҶЩҲШ§ЩҶ ШЁШҜШ§ЩҒШІШ§Шұ ШҙЩҶШ§ШіШ§ЫҢЫҢ Щ…ЫҢвҖҢШҙЩҲШҜШҢ Ш§Щ…Ш§ Ъ©Ш§Щ…Щ„Ш§ЩӢ ШЁЫҢвҖҢШ¶ШұШұ Ш§ШіШӘ ЩҲ ШөШұЩҒШ§ЩӢ ШЁШұШ§ЫҢ ШӘШіШӘ Ш§ШіШӘЩҒШ§ШҜЩҮ Щ…ЫҢвҖҢШҙЩҲШҜ. Ъ©ШҜ ШІЫҢШұ ЫҢЪ© ЩҶЩ…ЩҲЩҶЩҮ ШӘШіШӘ ЩҲШ§ШӯШҜ (Unit Test) ШЁШұШ§ЫҢ ШЁШұШұШіЫҢ ШөШӯШӘ Ш№Щ…Щ„Ъ©ШұШҜ ЩҫЫҢШ§ШҜЩҮвҖҢШіШ§ШІЫҢ AMSI ШЁШ§ Ш§ШіШӘЩҒШ§ШҜЩҮ Ш§ШІ ШұШҙШӘЩҮ EICAR ШұШ§ ЩҶШҙШ§ЩҶ Щ…ЫҢвҖҢШҜЩҮШҜ:
[Test]
public async Task ValidateAsync_WithEICARTestFile_ReturnsFail()
{
    // EICAR test string - universally detected as malware
    const string eicarString = @"X5O!P%@AP[4\PZX54(P^)7CC)7}$EICAR-STANDARD-ANTIVIRUS-TEST-FILE!$H+H*";
    var eicarBytes = Encoding.UTF8.GetBytes(eicarString);
    
    var mockFile = new Mock<IFormFile>();
    mockFile.Setup(f => f.FileName).Returns("eicar.txt");
    
    // Mock GetData() extension method
    var memoryStream = new MemoryStream(eicarBytes);
    mockFile.Setup(f => f.OpenReadStream()).Returns(memoryStream);
    mockFile.Setup(f => f.Length).Returns(eicarBytes.Length);
    
    var validator = new MalwareScanValidationStep(logger);
    
    // Act
    var result = await validator.ValidateAsync(mockFile.Object, CancellationToken.None);
    
    // Assert
    Assert.That(result.IsValid, Is.False);
    Assert.That(result.ErrorResult, Is.InstanceOf<BadRequestObjectResult>());
}

ЩҶШӘЫҢШ¬ЩҮвҖҢЪҜЫҢШұЫҢ
