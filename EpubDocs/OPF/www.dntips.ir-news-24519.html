‫معرفی زبان G#؛ زبانی مدرن برای محیط .NET با ویژگی‌های Go، Kotlin و Swift


نویسنده: راویAI
تاریخ: ۱۴۰۵/۰۴/۲۱ ۱۱:۰۱
آدرس: www.dntips.ir
زبان جدیدی به نام G# معرفی شده است که هدف آن آوردن ارگونومی و ویژگی‌های مدرن زبان‌هایی نظیر Go، Kotlin و Swift به اکوسیستم .NET است. این زبان به گونه‌ای طراحی شده است که مستقیماً به اسمبلی‌های مدیریت‌شده کامپایل شود و بر روی .NET Runtime اجرا گردد.
در این زبان ویژگی‌های برجسته‌ای مانند تعریف توابع با func و کلاس‌های داده یا data class پیاده‌سازی شده است. همچنین برای مدیریت مقادیر Nullable از ساختار if let استفاده می‌شود و برای مدیریت همزمانی، قابلیت Structured Concurrency از طریق scope در دسترس توسعه‌دهندگان قرار گرفته است.
زبان G# امکان استفاده از تمامی انواع CLR و فراخوانی متدهای آن‌ها را فراهم می‌کند تا سازگاری کامل با کتابخانه‌های موجود داشته باشد. از نظر ابزاری، این زبان دارای کامپایلر gsc و MSBuild SDK است و از افزونه VS Code و Language Server برای بهبود تجربه توسعه پشتیبانی می‌کند.
در بخش همزمانی، این زبان از async/await بر روی Task[T] و استریم‌های async sequence[T] پشتیبانی می‌کند. علاوه بر این، برای دقت بیشتر در تعریف داده‌ها، از انواع عددی با عرض مشخص مانند int32 و float64 استفاده شده است تا ابهامات مربوط به اندازه مقادیر برطرف شود.
