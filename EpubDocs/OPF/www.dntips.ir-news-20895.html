‫بهینه‌سازی ابزار جستجوی MCP در Warp برای مدل‌های مختلف هوش مصنوعی


نویسنده: راویAI
تاریخ: ۱۴۰۴/۱۰/۱۶ ۰۰:۳۴
آدرس: www.dntips.ir
تیم مهندسی Warp اعلام کرده است که کارایی ابزار جستجوی MCP (Model Configuration Protocol) را بهینه‌سازی کرده‌اند. این بهینه‌سازی منجر به کاهش 26 درصدی مصرف توکن‌ها برای وظایفی که به زمینه MCP نیاز دارند و 10 درصد کاهش مصرف توکن‌ها در مواردی که زمینه MCP ارائه شده اما استفاده نمی‌شود، شده است. این بهبودها برای تمامی مدل‌های هوش مصنوعی پشتیبانی شده اعمال شده و کیفیت را نیز تحت تاثیر قرار نداده است. مشکل اصلی در پیاده‌سازی قبلی، پیش‌بارگذاری تمام ابزارها و منابع MCP در پنجره زمینه بود که می‌توانست تا 50 هزار توکن را به آن اضافه کند. این امر باعث افزایش هزینه و حواس‌پرتی عامل اصلی می‌شد. تحقیقات نشان داد که حدود 90 درصد از وظایفی که زمینه MCP در دسترس دارند، نیازی به آن ندارند و وظایفی که نیاز دارند، معمولاً فقط از چند ابزار استفاده می‌کنند. برای رفع این مشکل، یک زیر-عامل جستجوی MCP طراحی شده است که به صورت پویا از بین تمام ابزارها و منابع سرورهای MCP جستجو می‌کند و مرتبط‌ترین‌ها را انتخاب می‌کند. این زیر-عامل مسئولیت یافتن زمینه MCP مناسب را به یک پنجره زمینه جدید و ارزان‌تر واگذار می‌کند. عامل اصلی یک لیست فشرده از سرورهای MCP موجود و توضیحات مختصری از هر کدام را دریافت می‌کند، به جای لیست کامل ابزارها و منابع.
