‫آنچه تا بدینجا قرار است در ASP.NET Core 10.0 بیاید


نویسنده: مجید شهاب فر
تاریخ: ۱۴۰۳/۱۲/۰۲ ۱۱:۴۵
آدرس: www.dntips.ir
این سند ویژگی‌های کلیدی جدید در ASP.NET Core 10.0 را خلاصه می‌کند.

بلیزر (Blazor)

  • پارامتر QuickGrid RowClass: امکان اعمال کلاس‌های شیوه نامه به ردیف‌های جدول بر اساس داده‌های آیتم ردیف را فراهم می‌کند.
  • اسکریپت بلیزر: به عنوان یک دارایی وب استاتیک با فشرده‌سازی و اثر انگشت خودکار ارائه می‌شود (قبلاً منبع تعبیه شده بود).
  • برجسته‌سازی الگوی مسیر (در ویژوال استودیو): یک کمک بصری برای نمایش بهتر ساختار الگوی مسیر در ویژگی [Route] است.

اوپن API (OpenAPI)

  • پشتیبانی از OpenAPI 3.1: شامل پشتیبانی کامل از پیش نویس طرح JSON 2020-12 است. نسخه پیش‌فرض OpenAPI اکنون 3.1 است (قابل تنظیم). انواع Nullable به شکل متفاوتی نمایش داده می‌شوند (با استفاده از type: [null, ...] ).
  • تغییرات مهم OpenAPI 3.1: نیاز به به‌روزرسانی کتابخانه OpenAPI.NET (نسخه 2.0) دارد که به طور بالقوه بر ترانسفورماتورهای سند/عملیات/طرح تأثیر می‌گذارد. OpenApiAny با JsonNode جایگزین شده است.
  • OpenAPI در YAML: از ارائه اسناد OpenAPI در قالب YAML از طریق پسوند نقطه .yaml یا .yml پشتیبانی می‌کند. تولید YAML در زمان ساخت در نسخه آینده ارائه خواهد شد.
  • شرح پاسخ: ویژگی‌های ProducesAttribute، ProducesResponseTypeAttribute و ProducesDefaultResponseTypeAttribute اکنون پارامتر Description را برای توضیحات پاسخ در OpenAPI می‌پذیرند.

متفرقه (Miscellaneous)

  • بهبود پشتیبانی از تست برنامه‌ها با دستورات سطح بالا: مولد منبع به طور خودکار public partial class Program را ایجاد می‌کند. تحلیلگری برای توصیه حذف اعلان‌های صریح اضافه شده است.
  • RedirectHttpResult.IsLocalUrl: روش کمکی جدید برای تأیید URLها قبل از تغییر مسیر، برای جلوگیری از حملات تغییر مسیر باز. عدم وجود میزبان/مرجع، وجود مسیر مطلق را بررسی می‌کند و از مسیرهای مجازی (~/) پشتیبانی می‌کند.
