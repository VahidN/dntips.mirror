‫مقایسه عملی معماری Clean و Vertical Slice در .NET با استفاده از APIهای واقعی


نویسنده: راویAI
تاریخ: ۱۴۰۴/۱۰/۲۴ ۰۴:۰۶
آدرس: www.dntips.ir
این مقاله به مقایسه عملی معماری Clean (CA) و Vertical Slice (VSA) در توسعه APIهای .NET می‌پردازد. نویسنده دو API ساده و یکسان با استفاده از این دو معماری ایجاد کرده تا تفاوت‌های عملی در سرعت پیاده‌سازی، سهولت تست واحد و افزودن ویژگی‌ها را مشاهده کند. هر دو API با .NET و Minimal APIs نوشته شده‌اند و امکان دریافت داده‌های آب و هوا از Weatherbit از طریق RapidAPI و همچنین مدیریت مکان‌های مورد علاقه در یک پایگاه داده in-memory را فراهم می‌کنند. تمرکز اصلی مقاله بر یافته‌های عملی حاصل از پیاده‌سازی این معماری‌ها است و به جای توضیح مفاهیم اصلی، به مقایسه سرعت پیاده‌سازی و سهولت تست می‌پردازد. یک عملیات حذف (Delete) به عنوان یک ویژگی دنیای واقعی برای مقایسه جامع‌تر استفاده شده است.
