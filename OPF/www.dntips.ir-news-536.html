    Ш§ШҙШӘШұШ§Ъ©вҖҢЩҮШ§ШӘШІШұЫҢЩӮ ЩҲШ§ШЁШіШӘЪҜЫҢвҖҢЩҮШ§ ШҜШұ ЩҫШұЩҲЪҳЩҮвҖҢЩҮШ§ЫҢ Web formsШ§ШҙШӘШұШ§Ъ©вҖҢЩҮШ§ЫҢШ§ШҜЪҜЫҢШұЫҢ Web API ШЁШ§ Ш§ШіШӘЩҒШ§ШҜЩҮ Ш§ШІ WPF,WebFormЩҲ XamarinЩ…Ш·Ш§Щ„ШЁШҙШ§ЫҢШ№Ш§ШӘЫҢ ШҜШұ Щ…ЩҲШұШҜ ЩҶШіШ®ЩҮвҖҢЫҢ ШЁШ№ШҜЫҢ ASP.NET WebformsЩ…Ш·Ш§Щ„ШЁШ¬Ш§ЫҢЪҜШІЫҢЩҶЫҢ Ш§ШіЪ©ШұЫҢЩҫШӘвҖҢЩҮШ§ЫҢ WebResource.axd ШЁШ§ ЩҒШ§ЫҢЩ„вҖҢЩҮШ§ЫҢ Ш§ШіШӘШ§ШӘЫҢЪ© ШҜШұ ASP.NET Web formsЩҶШёШұШ§ШӘ Щ…Ш·Ш§Щ„ШЁШ§ЩҶШӘЩӮШ§Щ„ ШҜШ§ШҰЩ…ЫҢ ШўШҜШұШі ЩҮШ§ (Permanent Ridirect) ШҜШұ ASP.NET WebformЩҶШёШұШ§ШӘ Щ…Ш·Ш§Щ„ШЁШўЩҫЩ„ЩҲШҜ ЩҮЩ…ШІЩ…Ш§ЩҶ ЪҶЩҶШҜЫҢЩҶ ЩҒШ§ЫҢЩ„ ШҜШұ Asp.Net Web FormsЩҶШёШұШ§ШӘ Щ…Ш·Ш§Щ„ШЁШіШ§ШІЪҜШ§Шұ Ъ©ШұШҜЩҶ Щ„ЫҢЩҶЪ©вҖҢЩҮШ§ЫҢ ЩӮШҜЫҢЩ…ЫҢ ЫҢЪ© ШіШ§ЫҢШӘ ШЁШ§ ШіШ§Ш®ШӘШ§Шұ Ш¬ШҜЫҢШҜ ШўЩҶ ШҜШұ ASP.NET MVCЩҶШёШұШ§ШӘ Щ…Ш·Ш§Щ„ШЁШіШ§ШІЪҜШ§Шұ Ъ©ШұШҜЩҶ GridView ШЁШ§ Ш§ЩҒШІЩҲЩҶЩҮвҖҢЩҮШ§ЫҢ jQueryЩҶШёШұШ§ШӘ Щ…Ш·Ш§Щ„ШЁШӘШҙШ®ЫҢШө ЩҶЩҲШ№ ЩҒШ§ЫҢЩ„ ШЁШ§ Ш§ШіШӘЩҒШ§ШҜЩҮ Ш§ШІ Щ…ШӯШӘЩҲШ§ЫҢ ЩҒШ§ЫҢЩ„ЩҶШёШұШ§ШӘ Щ…Ш·Ш§Щ„ШЁЩҶШӯЩҲЩҮ Ш§Ш¬ШЁШ§ШұЫҢ Ъ©ШұШҜЩҶ Ш§ШіШӘЩҒШ§ШҜЩҮ Ш§ШІ WWW ШҜШұ ASP.NET MVCЩҶШёШұШ§ШӘ Щ…Ш·Ш§Щ„ШЁШ§ШіШӘЩҒШ§ШҜЩҮ Ш§ШІ OpenID ШҜШұ ЩҲШЁ ШіШ§ЫҢШӘ Ш¬ЩҮШӘ Ш§ШӯШұШ§ШІ ЩҮЩҲЫҢШӘ Ъ©Ш§ШұШЁШұШ§ЩҶ
