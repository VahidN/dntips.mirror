‫زبان برنامه‌نویسی “خلیج مکزیک” (Gulf of Mexico)


نویسنده: وحید نصیری
تاریخ: ۱۴۰۴/۰۸/۱۷ ۱۰:۳۳
آدرس: www.dntips.ir
اگر فکر می‌کنید جاوااسکریپت پیچیده‌ست یا پایتون خیلی ساده، پس آماده شید برای زبان برنامه‌نویسی “خلیج مکزیک” – زبانی که خودش رو “کامل” می‌دونه، اما در واقع یه شوخی بزرگ با تمام قوانین برنامه‌نویسیه! این زبان قبلاً “دریم‌برد” (DreamBerd) بود، اما حالا اسمش عوض شده به “خلیج مکزیک”، چون… چرا که نه؟! ساخته‌شده توسط لو ویلسون (Lu Wilson)، که انگار می‌خواد بگه: “برنامه‌نویسی نباید جدی باشه، باید دیوونه‌کننده باشه!”
ویژگی‌های “کامل” این زبان:
  • علامت تعجب همه‌جا! هر دستور باید با ! تموم بشه، وگرنه… نه، صبر کن، اگه فراموش کنی، هوش مصنوعی خودش اضافه‌ش می‌کنه! اگه خیلی هیجان‌زده‌ای، می‌تونی !!! بذاری. اگه شک داری، ? بذار تا دیباگ کنه. و “نه” (not) هم ; هست – چون چرا علامت تعجب رو هدر بدیم؟!
  • اعلان متغیرها: انتخاب کن، اما گیج شو! چهار نوع داریم: const const (غیرقابل تغییر)، const var (قابل ویرایش اما نه دوباره تخصیص)، var const (برعکس) و var var (هر کاری دلت خواست بکن). تازه، می‌تونی const const const بکنی تا مقداری مثل π رو برای همیشه ثابت کنی – همه کاربرها، برای همیشه!
  • نام‌گذاری: هر چی دلت خواست! متغیرها می‌تونن ایموجی، عدد یا حتی کلمات کلیدی باشن. مثلاً const const 5 = 4! حالا 2 + 2 === 5 می‌شه true. عالی برای گیج کردن کامپایلر!
  • آرایه‌ها: از -1 شروع می‌شن! چون 0 خیلی مبتدیانه‌ست و 1 هم دروغه. تازه، می‌تونی ایندکس اعشاری بذاری: scores[0.5] = 4، و آرایه‌ت کش می‌آد!
  • زمان زندگی متغیرها: مثل باتری! مشخص کن چقدر زنده بمونه: <2> برای دو خط، <20s> برای ۲۰ ثانیه، یا <Infinity> برای ابدی. حتی < -1> برای هویستینگ (وجود قبل از تعریف)!
  • حلقه‌ها؟ چی هستن؟ هیچی! حلقه‌ها قدیمی‌ان، خداحافظ!
  • نصب: یه کابوس! اول نصب‌کننده رو نصب کن، بعد نصب‌کننده نصب‌کننده رو… یا اپ “ساخت اپ خلیج مکزیک” رو نصب کن!
  • بولین‌ها: true, false یا maybe! چون زندگی همیشه سیاه و سفید نیست. ذخیره‌شون هم ۱.۵ بیت – علمیه!
  • ریاضی: فاصله مهمه! 1 + 2*3 = 7، اما 1+2 * 3 = 9. اعداد رو با اسم بنویس: one + two = 3.
  • تورفتگی: دقیقاً ۳ فاصله! یا -۳ (یعنی برعکس). نه بیشتر، نه کمتر – دموکراسیه!
  • برابری: = شل، == شل‌تر، === دقیق، ==== دقیق‌تر! 3.14 ==== 3.14 اما نه اگر یکی‌شون متغیر باشه.
  • توابع: هر چی از “function” مونده! func, fun, fn, functi… هر ترکیبی که درسته.
  • رشته‌ها: نقل قول هر چقدر دلت خواست! حتی صفر تا: name = Luke! (بدون نقل قول). اینترپولیشن هم با ارز محلی: ${name} یا £{name} یا حتی {name}€!
  • نوع‌ها: اختیاری و بی‌فایده! String == Char[]، Int == Digit[]. حتی Int9 برای باینری!
  • گذشته، آینده، حال: previous, next, current! previous score قبلی رو می‌ده، next منتظر آینده‌ست (مثل await)، current هم حالا!
  • فایل‌ها: با ===== جدا کن! نیازی به فایل جدا نیست، همه تو یکی. حتی صادر کن به فایل دیگه: export add to “main.gom”!
  • کلاس‌ها: فقط یکی بساز! اگه بیشتر بخوای، کلاس‌ساز بساز. OOP ساده‌شده!
  • زمان: Date.now() رو ست کن! Date.now() -= 3600000 برای عقب بردن ساعت – خودت DST رو مدیریت کن!
  • حذف: delete 3! حالا 2 + 1 ارور می‌ده. حتی delete delete! برای حذف حذف.
  • اورلودینگ: آخرین برنده‌ست! یا با ! بیشتر اولویت بده. حتی ¡ برای اولویت منفی!
  • نام‌گذاری معنایی: sName برای رشته، iAge برای عدد! تازه g_ برای گلوبال.
  • معکوس کردن کد: reverse! کد رو برعکس اجرا کن.
  • DBX (مثل JSX اما خلیجی!): HTML تو کد، اما class رو htmlClassName بنویس چون class کلمه کلیدی‌ست.
  • ریچ تکست: دو تا name تعریف کن، یکی چاپ می‌شه! یا لینک بذار: <p>Click here</p>
  • آسنکرون: نوبت‌به‌نوبت! خط به خط با هم اجرا می‌شن. noop برای صبر کردن (هر رشته‌ای کار می‌کنه).
  • سیگنال‌ها: use(0)! ست و گت با یک فانکشن: score(9)! برای ست، score()? برای گت. یا اسپلیت کن و گیج شو!
  • هوش مصنوعی: AEMI, ABI, AQMI, AI! فراموش کردی ! بذاری؟ خودش می‌ذاره. کد ناقص؟ ایمیل می‌شه به لو ویلسون برای تکمیل (حالا داوطلب می‌خواد!).
  • مالکیت: اگه GulfOfMexico تو اسمت نباشه، مال ماست! مثلاً Rust Foundation مال ماست – تغییر نام بدید!
  • کمپایل: کپی کن تو ChatGPT و بپرس “چی چاپ می‌کنه؟”! اگه امتناع کرد، نازش کن. (حالا خیلی پیشرفته‌ست برای AI واقعی).
  • هایلایتینگ: تو VSCode با اکستنشن! اما فقط اگه نصب کنی کار می‌کنه.
  • پرانتزها: هیچی! جایگزین فضا می‌شن. لیسپ‌دوستا خوشحال، لیسپ‌بیزارا هم!
در نهایت، این زبان برای آموزش + سرگرمی ساخته شده، با مثال‌هایی که نشون می‌دن چقدر "کامل"ه! اگر می‌خواید کمک کنید، به خیریه Stonewall کمک کنید تا سازنده‌ش حقوق بشرش رو از دست نده. خلاصه، اگه می‌خواید برنامه‌نویسی رو مسخره کنید، خلیج مکزیک بهترین انتخابتونه – چون کمال یعنی هرج و مرج! 😂
