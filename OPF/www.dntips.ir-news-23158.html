‫خرید Cloudflare از Astro و ادغام Copilot در OpenCode


نویسنده: راویAI
تاریخ: ۱۴۰۴/۱۱/۰۹ ۰۵:۵۸
آدرس: www.dntips.ir
Cloudflare شرکت Astro Technology Company را خریداری کرده است. این اقدام به Cloudflare امکان می‌دهد تا از فناوری‌های Astro برای بهبود پلتفرم وب خود استفاده کند. Astro به توسعه‌دهندگان کمک می‌کند تا وب‌سایت‌ها و برنامه‌های وب را با استفاده از اجزای قابل ترکیب و رندر شده در سمت سرور بسازند. گیت‌هاب (GitHub) همچنین اعلام کرده است که Copilot اکنون از OpenCode پشتیبانی می‌کند. این به توسعه‌دهندگان اجازه می‌دهد تا از اشتراک Copilot خود برای استفاده از OpenCode، یک محیط توسعه یکپارچه برای ساخت برنامه‌های کاربردی مبتنی بر هوش مصنوعی، بهره ببرند. همچنین، به‌روزرسانی‌هایی برای Copilot CLI و SDK نیز ارائه شده است که امکان ساخت Agentهای سفارشی را در برنامه‌ها فراهم می‌کند. تیم GitHub همچنین پروژه Handy را به عنوان پروژه برجسته معرفی کرده است. Handy یک ابزار خط فرمان است که به توسعه‌دهندگان کمک می‌کند تا وظایف مختلف توسعه را خودکار کنند. این پروژه به ویژه برای توسعه‌دهندگان .NET مفید خواهد بود، زیرا می‌تواند برای خودکارسازی وظایف رایج مانند ساخت، آزمایش و استقرار برنامه‌ها استفاده شود. در پایان، Christina Warren، نویسنده این گزارش، Harry Styles' Aperture را به عنوان انتخاب هفته خود معرفی کرد. این نشان‌دهنده تنوع محتوای ارائه شده در The Download است که شامل اخبار فناوری، به‌روزرسانی‌های توسعه و حتی انتخاب‌های شخصی است.
