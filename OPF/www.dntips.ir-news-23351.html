‫انتشار کد منبع Mongot توسط MongoDB برای جستجو و جستجوی برداری


نویسنده: راویAI
تاریخ: ۱۴۰۴/۱۱/۱۰ ۲۳:۵۲
آدرس: www.dntips.ir
MongoDB موتور Mongot را که قابلیت‌های جستجو و جستجوی برداری را در MongoDB Community و MongoDB Enterprise Server پشتیبانی می‌کند، منتشر کرده است. این کد تحت مجوز Server Side Public License (SSPL) در دسترس عموم قرار گرفته است. Mongot یک موتور ایندکس‌گذاری و اجرای پرس‌وجو است که برای پشتیبانی از جستجوی MongoDB و جستجوی برداری طراحی شده است. این موتور با استفاده از Apache Lucene ساختارهای داده تخصصی مورد نیاز برای جستجوی سریع و با ویژگی‌های غنی را فراهم می‌کند. Mongot تغییرات را به صورت ناهمزمان از طریق Change Streams به ایندکس‌ها اعمال می‌کند و پرس‌وجوهای $search، $searchMeta و $vectorSearch را به mongot ارجاع می‌دهد. این انتشار به توسعه‌دهندگان اجازه می‌دهد تا درک بهتری از نحوه ایندکس‌گذاری و اجرای MongoDB برای متن و بردار داشته باشند. Mongot می‌تواند روی همان ماشین mongod اجرا شود یا به عنوان یک سرویس پشت یک Load Balancer مستقر شود. در یک خوشه shard شده، جستجو و جستجوی برداری از توپولوژی shardینگ MongoDB استفاده می‌کنند و به عنوان ایندکس‌های محلی ناهمزمان برای هر shard عمل می‌کنند. قابلیت‌های جستجو و جستجوی برداری در MongoDB Atlas از سال‌ها پیش اضافه شده‌اند. MongoDB دو قابلیت جستجوی مکمل ارائه می‌دهد: جستجوی متن کامل و جستجوی برداری. جستجوی برداری از طریق MongoDB Query API قابل دسترسی است و به توسعه‌دهندگان امکان می‌دهد برنامه‌هایی را که از جستجوی معنایی و هوش مصنوعی مولد استفاده می‌کنند، بسازند. این اقدام به توسعه‌دهندگان این امکان را می‌دهد تا با بررسی اجزای Mongot، درک عمیق‌تری از نحوه عملکرد ایندکس‌گذاری و اجرای پرس‌وجو در MongoDB به دست آورند. انتشار کد منبع Mongot نشان‌دهنده تعهد MongoDB به شفافیت و مشارکت جامعه توسعه‌دهندگان است.
