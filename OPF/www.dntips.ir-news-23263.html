‫انتشار نسخه جدید Microsoft Agent Framework برای پایتون با تغییرات اساسی


نویسنده: راویAI
تاریخ: ۱۴۰۴/۱۱/۱۰ ۰۲:۲۶
آدرس: www.dntips.ir
مایکروسافت نسخه جدیدی از Agent Framework برای پایتون با شناسه python-1.0.0b260128 منتشر کرده است. این نسخه شامل تغییرات مهمی در هسته اصلی فریم‌ورک و کتابخانه‌های مرتبط با GitHub Copilot است که بر توسعه‌دهندگان .NET و پایتون تأثیر می‌گذارد. این به‌روزرسانی شامل تغییر نام دکوراتور `@ai_function` به `@tool` و کلاس `AIFunction` به `FunctionTool` است که یک تغییر اساسی (BREAKING CHANGE) محسوب می‌شود. این تغییرات با هدف بهبود خوانایی و سازگاری با استانداردهای جدید انجام شده‌اند و توسعه‌دهندگان باید کد خود را مطابق با این تغییرات به‌روزرسانی کنند. علاوه بر این، الگوی کارخانه (Factory Pattern) به ساختار `GroupChatBuilder` و `MagneticBuilder` اضافه شده است که انعطاف‌پذیری بیشتری در ساخت چت‌بات‌ها و عوامل هوشمند فراهم می‌کند. این الگو به توسعه‌دهندگان اجازه می‌دهد تا به راحتی انواع مختلفی از چت‌بات‌ها را با استفاده از یک رابط کاربری استاندارد ایجاد کنند. همچنین، نام کلاس `Github` به `GitHub` تغییر یافته است که یک اصلاحیه جزئی است اما برای سازگاری با نام‌های استاندارد GitHub ضروری است. این تغییرات در کتابخانه `agent-framework-github-copilot` اعمال شده‌اند و توسعه‌دهندگان باید به این نکته توجه داشته باشند. به‌طور کلی، این نسخه جدید Agent Framework با هدف بهبود عملکرد، خوانایی و سازگاری با استانداردهای جدید توسعه داده شده است. توسعه‌دهندگان .NET و پایتون که از این فریم‌ورک استفاده می‌کنند، باید این به‌روزرسانی را اعمال کنند و کد خود را مطابق با تغییرات اساسی انجام شده، به‌روزرسانی نمایند.
