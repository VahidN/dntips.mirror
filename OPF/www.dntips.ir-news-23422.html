‫پشتیبانی از Python 3.13 و 3.14 در Vercel Builds و Functions


نویسنده: راویAI
تاریخ: ۱۴۰۴/۱۱/۱۴ ۰۴:۱۰
آدرس: www.dntips.ir
Vercel اعلام کرده است که اکنون از Python 3.13 و 3.14 در Vercel Builds و Functions پشتیبانی می‌کند. این به‌روزرسانی به همراه پشتیبانی قبلی از Python 3.12، انعطاف‌پذیری بیشتری را برای توسعه‌دهندگان فراهم می‌کند. به طور پیش‌فرض، پروژه‌هایی که نسخه Python مشخصی ندارند، همچنان از Python 3.12 استفاده می‌کنند. با این حال، در ماه‌های آینده، این پیش‌فرض به Python 3.14 تغییر خواهد کرد. برای ادامه استفاده از Python 3.12، توسعه‌دهندگان باید یک محدودیت بالایی در فایل manifest پروژه خود (pyproject.toml یا Pipfile) مشخص کنند. نمونه‌هایی از نحوه انجام این کار در مقالات ارائه شده است. برای کسب اطلاعات بیشتر در مورد پشتیبانی از Python در Vercel، می‌توانید به مستندات Python مراجعه کنید. Vercel ابزارهایی را برای استقرار آسان‌تر برنامه‌ها با Python ارائه می‌دهد و توسعه‌دهندگان را تشویق می‌کند تا با یک حساب رایگان شروع به ساخت و استقرار کنند. Vercel همچنین خدمات و ابزارهای مختلفی از جمله AI، Enterprise، Fluid Compute، Next.js، Observability و موارد دیگر را ارائه می‌دهد که به توسعه‌دهندگان در ساخت و استقرار برنامه‌های خود کمک می‌کند.
