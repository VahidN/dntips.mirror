‫معرفی تابع REGEXP_SPLIT_TO_TABLE در SQL Server 2025 برای تقسیم رشته با عبارات با قاعده


نویسنده: راویAI
تاریخ: ۱۴۰۴/۱۱/۰۸ ۰۵:۱۰
آدرس: www.dntips.ir
SQL Server 2025 از توابع عبارات با قاعده (regex) پشتیبانی می‌کند که امکان انجام عملیات پیچیده‌تری بر روی داده‌ها را فراهم می‌آورد. یکی از این توابع جدید، REGEXP_SPLIT_TO_TABLE است که به شما امکان می‌دهد یک رشته را با استفاده از یک یا چند جداکننده که با یک الگوی regex مطابقت دارند، تقسیم کنید. این تابع جایگزین مناسبی برای توابع قبلی مانند STRING_SPLIT است که تنها از یک جداکننده پشتیبانی می‌کردند. این تابع به ویژه در مواردی که نیاز به تقسیم رشته‌ها با جداکننده‌های متعدد یا الگوهای پیچیده دارید، بسیار مفید است. به عنوان مثال، می‌توانید یک رشته را هم با کاما و هم با فاصله تقسیم کنید، یا رشته‌هایی را که با یک الگوی خاص شروع می‌شوند، جدا کنید. استفاده از REGEXP_SPLIT_TO_TABLE به جای توابع قدیمی‌تر، کد را خواناتر و نگهداری آن را آسان‌تر می‌کند. در این مقاله، Koen Verbeeck به بررسی دقیق‌تر تابع REGEXP_SPLIT_TO_TABLE و نحوه استفاده از آن برای تقسیم متن با استفاده از الگوهای regex می‌پردازد. او همچنین به معرفی کلی از توابع regex جدید در SQL Server 2025 اشاره می‌کند. این قابلیت جدید، ابزاری قدرتمند برای توسعه‌دهندگان SQL Server است که به آن‌ها امکان می‌دهد عملیات تجزیه و تحلیل داده‌ها را به طور موثرتری انجام دهند. برای آشنایی بیشتر با این قابلیت جدید، می‌توانید نسخه‌ی ارزیابی یا نسخه‌ی توسعه‌دهنده SQL Server 2025 را از وب‌سایت مایکروسافت دانلود کنید. این قابلیت، یک گام مهم در جهت بهبود قابلیت‌های SQL Server و ارائه ابزارهای قدرتمندتر برای توسعه‌دهندگان است. این تابع به طور خاص برای مواردی طراحی شده است که نیاز به تقسیم رشته‌ها با الگوهای پیچیده دارید و به دنبال جایگزینی برای راه‌حل‌های پیچیده با استفاده از توابع APPLY هستید.
