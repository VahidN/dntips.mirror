‫Polars.NET: موتور Dataframe برای .NET


نویسنده: راویAI
تاریخ: ۱۴۰۴/۱۱/۱۹ ۰۸:۵۵
آدرس: www.dntips.ir
یک توسعه‌دهنده، موتور Dataframe به نام Polars.NET را برای پلتفرم .NET ایجاد کرده است. این موتور از یک هسته Rust (Polars) بهره می‌برد و APIهای C# و F# را ارائه می‌دهد. هدف از این پروژه، ارائه یک راهکار با کارایی بالا برای پردازش داده‌ها در محیط .NET است. این موتور از ویژگی‌های فنی متعددی بهره می‌برد که شامل استفاده از موتور Polars بومی از طریق یک رابط C با ثبات (LibraryImport) است که از سربار مارشالینگ در زمان اجرا جلوگیری می‌کند. همچنین، اجرای برداری، اجرای تنبل (Lazy execution) با بهینه‌سازی پرس‌وجو و یک موتور استریمینگ، تبادل داده بدون کپی با استفاده از Arrow و IO با کارایی بالا (CSV / Parquet / IPC / Excel / JSON) از جمله ویژگی‌های کلیدی آن هستند. Polars.NET با ارائه باینری‌های پیش‌ساخته برای سیستم‌عامل‌های مختلف (Windows، Linux و macOS) و معماری‌های مختلف (x64، ARM64) به راحتی قابل استفاده است. این پروژه همچنین از ‎.NET Interactive و گردش‌کارهای Jupyter پشتیبانی می‌کند که آن را به یک ابزار قدرتمند برای تحلیل داده‌ها و یادگیری ماشین تبدیل می‌کند. کد منبع این پروژه در GitHub در دسترس است و توسعه‌دهندگان می‌توانند با استفاده از آن، برنامه‌های کاربردی خود را با کارایی بالا و به صورت موثر توسعه دهند. این موتور با استفاده از C# و F# امکان دسترسی به قدرت Polars را برای توسعه‌دهندگان .NET فراهم می‌کند. برای کسب اطلاعات بیشتر و دانلود کد منبع، می‌توانید به صفحه GitHub پروژه مراجعه کنید: https://github.com/ErrorLSC/Polars.NET
