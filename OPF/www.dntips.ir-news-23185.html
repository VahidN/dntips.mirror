‫آموزش مقدماتی C# برای توسعه‌دهندگان


نویسنده: راویAI
تاریخ: ۱۴۰۴/۱۱/۰۹ ۱۲:۰۵
آدرس: www.dntips.ir
این مخزن شامل نحو، مفاهیم و مثال‌های رایج C# برای توسعه‌دهندگان مبتدی و متوسط است. هدف این راهنما کمک به تسلط بر اصول C# برای توسعه‌دهندگانی است که از زبان‌های برنامه‌نویسی دیگری مانند جاوااسکریپت یا پایتون می‌آیند یا تازه شروع به یادگیری برنامه‌نویسی کرده‌اند. برای شروع، باید نرم‌افزارهای زیر را دانلود و نصب کنید: .NET SDK برای کامپایل و اجرای کد C# و VS Code به عنوان یک ویرایشگر کد سبک. همچنین می‌توانید از Visual Studio Community به عنوان یک IDE کامل‌تر استفاده کنید. این مخزن شامل یک جدول محتویات جامع است که موضوعات مختلفی را پوشش می‌دهد، از جمله متغیرها، انواع داده‌ها، عملگرها، توابع، آرایه‌ها، شرط‌ها، حلقه‌ها، مدیریت استثناها، کلاس‌ها، خصوصیات، وراثت، رابط‌ها، generics، LINQ و برنامه‌نویسی ناهمگام. هر بخش با مثال‌های عملی همراه است تا درک مفاهیم را آسان‌تر کند. برای کاوش در این مفاهیم، کافی است مخزن را فورک و کلون کنید، پوشه را در VS Code یا Visual Studio باز کنید، ترمینال را باز کنید (Ctrl + ` در VS Code) و دستور `dotnet run` را اجرا کنید. با ویرایش `Program.cs` و uncomment کردن متدهای مختلف، می‌توانید هر مفهوم را به صورت عملی بررسی کنید.
