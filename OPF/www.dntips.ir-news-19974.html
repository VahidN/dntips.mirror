Щ…Ш·Ш§Щ„ШЁ ЫіЫ¶ЫёЫё
ЩҶЩҲЫҢШіЩҶШҜЪҜШ§ЩҶ ЫІЫ·Ы¶
ЪҜШұЩҲЩҮвҖҢЩҮШ§ЫҢ Щ…Ш·Ш§Щ„ШЁ ЫұЫ°ЫұЫ№
ЩҶЩӮШҙЩҮвҖҢЩҮШ§ЫҢ ШұШ§ЩҮ ЫұЫұЫ№
ШҜЩҲШұЩҮвҖҢЩҮШ§ ЫұЫҙ
Ш§ШҙШӘШұШ§Ъ©вҖҢЩҮШ§ ЫұЫ·Ы№Ы°Ы№
 
вҖ«вҖ«ШЁШұШ®ЫҢ Ш§ШІ Ъ©Ш§ШіШӘЫҢвҖҢЩҮШ§ЫҢ Ш№Щ…Щ„ЫҢ GitHub Copilot
→ вҖ«ШўЫҢШ§ ЩҫШҙШӘ ШөЩҒШӯЩҮ Ъ©Щ„ЫҢШҜ Ш®ЩҲШ§ШЁЫҢШҜЩҮвҖҢШ§ЫҢШҜШҹ Ш§ШұШІЫҢШ§ШЁЫҢ Ш§Щ…ЩҶЫҢШӘ Ъ©ШҜЩҮШ§ЫҢ ШӘЩҲЩ„ЫҢШҜ ШҙШҜЩҮ ШӘЩҲШіШ· GitHub Copilot
вҖ«ШӘШЁШҜЫҢЩ„ ШӘШөШ§ЩҲЫҢШұ ШЁЩҮ Ъ©ШҜ ШЁШ§ Ш§ЩҒШІЩҲЩҶЩҮвҖҢЫҢ Vision for Copilot ←
Ш№ЩҶЩҲШ§ЩҶ:

вҖ«ШЁШұШ®ЫҢ Ш§ШІ Ъ©Ш§ШіШӘЫҢвҖҢЩҮШ§ЫҢ Ш№Щ…Щ„ЫҢ GitHub Copilot


ЩҶЩҲЫҢШіЩҶШҜЩҮ: ЩҲШӯЫҢШҜ ЩҶШөЫҢШұЫҢ
ШӘШ§ШұЫҢШ®: ЫұЫҙЫ°Ыі/ЫұЫұ/ЫІЫұ Ы°Ы·:ЫҙЫө
ШўШҜШұШі: www.dntips.ir
Ш§ЫҢЩҶ Щ…ЩӮШ§Щ„ЩҮШҢ ШЁШ§ ЩҲШ¬ЩҲШҜ Ш§Ш°Ш№Ш§ЩҶ ШЁЩҮ Щ…ЩҒЫҢШҜ ШЁЩҲШҜЩҶ GitHub Copilot ШЁШұШ§ЫҢ ШӘЩҲЩ„ЫҢШҜ Ъ©ШҜЩҮШ§ЫҢ boilerplateШҢ ЩҶЩҲШҙШӘЩҶ query/regex ЩҲ ШұШ§ЩҮвҖҢШӯЩ„вҖҢЩҮШ§ЫҢ ШіШұЫҢШ№ШҢ ЪҶЩҶШҜЫҢЩҶ Ъ©Ш§ШіШӘЫҢ ШўЩҶ ШұШ§ ШЁШұШ¬ШіШӘЩҮ Щ…ЫҢвҖҢЪ©ЩҶШҜ. ЩҶЩҲЫҢШіЩҶШҜЩҮ ШЁШұ Ш§ЩҮЩ…ЫҢШӘ ШӘШ¬ШұШЁЩҮ ШӘЩҲШіШ№ЩҮвҖҢШҜЩҮЩҶШҜЩҮ ШҜШұ ШҙЩҶШ§ШіШ§ЫҢЫҢ Ш®Ш·Ш§ЩҮШ§ЫҢ Copilot ШӘШЈЪ©ЫҢШҜ Щ…ЫҢвҖҢЪ©ЩҶШҜ.

Ъ©Ш§ШіШӘЫҢвҖҢЩҮШ§ЫҢ Ъ©Щ„ЫҢШҜЫҢ:

  • ШўЪҜШ§ЩҮЫҢ ЩҶШ§ШҜШұШіШӘ Ш§ШІ API:
    • API endpointЩҮШ§ЫҢ Ш®ЫҢШ§Щ„ЫҢ ШұШ§ ШӘШөЩҲШұ Щ…ЫҢвҖҢЪ©ЩҶШҜ ЩҲ ШўЩҶЩҮШ§ ШұШ§ ШЁШ§ Ш§Ш·Щ…ЫҢЩҶШ§ЩҶ Ш§ШұШ§ШҰЩҮ Щ…ЫҢвҖҢШҜЩҮШҜ.
    • Ш§ШІ endpointЩҮШ§ЫҢ Щ…ЩҲШ¬ЩҲШҜ Ш§ШіШӘЩҒШ§ШҜЩҮ ЩҶЩ…ЫҢвҖҢЪ©ЩҶШҜ ЩҲ ШӘШұШ¬ЫҢШӯ Щ…ЫҢвҖҢШҜЩҮШҜ ЪҶШұШ® ШұШ§ Ш§ШІ ЩҶЩҲ ШЁШіШ§ШІШҜ. Ш§ЫҢЩҶ Щ…ЫҢвҖҢШӘЩҲШ§ЩҶШҜ Щ…ЩҶШ¬Шұ ШЁЩҮ Ъ©ШҜЩҮШ§ЫҢ ЩҫШұ Ш§ШІ Ш§ШҙЪ©Ш§Щ„ ШҙЩҲШҜ Ъ©ЩҮ ШӘЩҲШіШ№ЩҮвҖҢШҜЩҮЩҶШҜЪҜШ§ЩҶ ШұШ§ Щ…Щ„ШІЩ… ШЁЩҮ ШЁШұШұШіЫҢ Щ…ШіШӘЩҶШҜШ§ШӘ API ШҜШұ Ъ©ЩҶШ§Шұ ЩҫЫҢШҙЩҶЩҮШ§ШҜШ§ШӘ Copilot Щ…ЫҢвҖҢЪ©ЩҶШҜ.
  • ШұШ§ЩҮвҖҢШӯЩ„вҖҢЩҮШ§ ШҜШұ ШӘШ§ШұЫҢЪ©ЫҢ:
    • ЩҲШ¬ЩҲШҜ ШЁШұШұШіЫҢвҖҢЩҮШ§ЫҢ ШәЫҢШұШ¶ШұЩҲШұЫҢ (Щ…Ш§ЩҶЩҶШҜ typeof window !== undefined) ШҜШұ Ъ©ШҜ.
    • Ш§ШІ ШіШ§Ш®ШӘШ§ШұЩҮШ§ЫҢ ШҜШ§ШҜЩҮ APIЩҮШ§ШҢ Щ…ЫҢЪ©ШұЩҲШіШұЩҲЫҢШівҖҢЩҮШ§ ЫҢШ§ ШіШұЩҲЫҢШівҖҢЩҮШ§ЫҢ ШҙШ®Шө Ш«Ш§Щ„Ш« ШЁЫҢвҖҢШ§Ш·Щ„Ш§Ш№ Ш§ШіШӘ.
    • ШіЫҢШіШӘЩ…вҖҢЩҮШ§ЫҢ Ш·ШұШ§ШӯЫҢ Щ…ЩҲШ¬ЩҲШҜ ШұШ§ ЩҶШ§ШҜЫҢШҜЩҮ Щ…ЫҢвҖҢЪҜЫҢШұШҜ ЩҲ Ш§ШәЩ„ШЁ ШЁЩҮвҖҢШ¬Ш§ЫҢ Ш§Ш¬ШІШ§ЫҢ Ш®Ш§Шө ЩҫШұЩҲЪҳЩҮШҢ Ш§Ш¬ШІШ§ЫҢ Ш№Щ…ЩҲЩ…ЫҢ ШұШ§ ШӘЩҲЩ„ЫҢШҜ Щ…ЫҢвҖҢЪ©ЩҶШҜ.
  • Ъ©ШҜЩҮШ§ЫҢ Ш¶Ш№ЫҢЩҒ ЫҢШ§ ЩӮШҜЫҢЩ…ЫҢ:
    • Щ…Щ…Ъ©ЩҶ Ш§ШіШӘ Щ…ШӘШҜЩҮШ§ЫҢ Щ…ЩҶШіЩҲШ®вҖҢШҙШҜЩҮ (Щ…Ш§ЩҶЩҶШҜ substr) ШұШ§ ЩҫЫҢШҙЩҶЩҮШ§ШҜ Ъ©ЩҶШҜ.
    • ШҜШұ ЩҫЫҢШҙЩҶЩҮШ§ШҜ ЪҳЩҶШұШ§ШӘЩҲШұЩҮШ§ЫҢ UUID ЫҢШ§ ШіШ§ЫҢШұ Ъ©ШӘШ§ШЁШ®Ш§ЩҶЩҮвҖҢЩҮШ§ЫҢ Щ…ШұШӘШЁШ· ШЁШұШ§ЫҢ Ш§ЫҢШ¬Ш§ШҜ ШұШҙШӘЩҮвҖҢЩҮШ§ЫҢ Щ…ЩҶШӯШөШұ ШЁЩҮвҖҢЩҒШұШҜШҢ ЩҶШ§Ъ©Ш§Щ… Ш§ШіШӘ.
    • Ш§ШІ Щ…ЩӮШ§ШҜЫҢШұ Щ…ЩҶШӯШөШұ ШЁЩҮвҖҢЩҒШұШҜ Щ…ЩҲШ¬ЩҲШҜ ШҜШұ Ъ©ШҜ ЩҫШ§ЫҢЩҮШҢ Ш§ШіШӘЩҒШ§ШҜЩҮ ЩҶЩ…ЫҢвҖҢЪ©ЩҶШҜ.
  • ШӘШіШӘвҖҢЩҮШ§ЫҢ ЩҲШ§ШӯШҜ ЩҶШ§ЩӮШө:
    • ШӘШіШӘвҖҢЩҮШ§ШҢ Ш§ШәЩ„ШЁ ЩҒШ§ЩӮШҜ ШЁШұШұШіЫҢвҖҢЩҮШ§ЫҢ Ш¶ШұЩҲШұЫҢ ЩҮШіШӘЩҶШҜ (ШЁЩҮ Ш№ЩҶЩҲШ§ЩҶ Щ…Ш«Ш§Щ„ШҢ ШЁШұШұШіЫҢ ЩҲШ¬ЩҲШҜ Ш№ЩҶШөШұШҢ ЩӮШЁЩ„ Ш§ШІ ШЁШұШұШіЫҢ ЩҶШ§ЩҫШҜЫҢШҜ ШҙШҜЩҶ ШўЩҶ).
    • Mocks ЩҫЫҢШҙЩҶЩҮШ§ШҜ Щ…ЫҢвҖҢШҙЩҲЩҶШҜ ШЁШҜЩҲЩҶ ЩҫЫҢШҙЩҶЩҮШ§ШҜШ§ШӘ Щ…ШұШЁЩҲШ·ЩҮ ШЁШұШ§ЫҢ ЩҫШ§Ъ© Ъ©ШұШҜЩҶ ШўЩҶЩҮШ§.



Щ…ШҙШ§ЩҮШҜЩҮ Щ…Ш·Щ„ШЁ Ш§ШөЩ„ЫҢ
Щ…Ш·Ш§Щ„ШЁ Щ…ШҙШ§ШЁЩҮ
    Ш§ШҙШӘШұШ§Ъ©вҖҢЩҮШ§ЩҶШӯЩҲЩҮ ШЁШ§ШІШіШ§ШІЫҢ Ъ©ШҜ ШЁШ§ GitHub CopilotШ§ШҙШӘШұШ§Ъ©вҖҢЩҮШ§ШұШ§ЩҮЩҶЩ…Ш§ЫҢ ШҙШұЩҲШ№ ШЁЩҮ Ъ©Ш§Шұ ШЁШ§ GitHub Copilot ШҜШұ 15 ШҜЩӮЫҢЩӮЩҮШ§ШҙШӘШұШ§Ъ©вҖҢЩҮШ§ШўЫҢШ§ ЩҫШҙШӘ ШөЩҒШӯЩҮ Ъ©Щ„ЫҢШҜ Ш®ЩҲШ§ШЁЫҢШҜЩҮвҖҢШ§ЫҢШҜШҹ Ш§ШұШІЫҢШ§ШЁЫҢ Ш§Щ…ЩҶЫҢШӘ Ъ©ШҜЩҮШ§ЫҢ ШӘЩҲЩ„ЫҢШҜ ШҙШҜЩҮ ШӘЩҲШіШ· GitHub CopilotШ§ШҙШӘШұШ§Ъ©вҖҢЩҮШ§Ш§ШіШӘЩҒШ§ШҜЩҮ Ш§ШІ ШҜШіШӘЩҲШұШ§Щ„Ш№Щ…Щ„вҖҢЩҮШ§ЫҢ ШіЩҒШ§ШұШҙЫҢ ЩҲ ШӘЩҶШёЫҢЩ…Ш§ШӘ Visual Studio Code ШЁШұШ§ЫҢ ШЁЩҮШЁЩҲШҜ ШӘЩҲШіШ№ЩҮ ШЁШ§ GitHub CopilotШ§ШҙШӘШұШ§Ъ©вҖҢЩҮШ§ЩҲЫҢЪҳЪҜЫҢ ЩҲЫҢШұШ§ЫҢШҙ ШЁШ№ШҜЫҢ ШҜШұ GitHub Copilot (ЩҫЫҢШҙвҖҢЩҶЩ…Ш§ЫҢШҙ)Ш§ШҙШӘШұШ§Ъ©вҖҢЩҮШ§Щ…ЩҮЩҶШҜШіЫҢ ШҜШұШ®ЩҲШ§ШіШӘвҖҢЩҮШ§ ШЁШ§ GitHub Copilot ШҜШұ Visual Studio CodeШ§ШҙШӘШұШ§Ъ©вҖҢЩҮШ§Ш®Щ„Ш§ШөЩҮ ШӘШ¬ШұШЁЩҮ Ш§ШіШӘЩҒШ§ШҜЩҮ Ш§ШІ GitHub Copilot ШұШ§ЫҢЪҜШ§ЩҶШ§ШҙШӘШұШ§Ъ©вҖҢЩҮШ§ШЁЩҮЫҢЩҶЩҮвҖҢШіШ§ШІЫҢ ШӘШ¬ШұШЁЩҮ Ъ©Ш§Шұ ШЁШ§ GitHub Copilot ШҜШұ VSCodeШ§ШҙШӘШұШ§Ъ©вҖҢЩҮШ§Щ…ЩҮЩҶШҜШіЫҢ ЩҫШұШ§Щ…ЩҫШӘ ШЁШұШ§ЫҢ ШӘЩҲШіШ№ЩҮвҖҢШҜЩҮЩҶШҜЪҜШ§ЩҶШ§ШҙШӘШұШ§Ъ©вҖҢЩҮШ§Ш§Щ…ЩҶЫҢШӘ ЩҫЩ„Ш§ЪҜЫҢЩҶвҖҢЩҮШ§ШҢ Ш№Ш§Щ…Щ„вҖҢЩҮШ§ ЩҲ Ш§ШӘШөШ§Щ„Ш§ШӘ ЪҜШұШ§ЩҒЫҢЪ©ЫҢ ШҜШұ Copilot Щ…Ш§ЫҢЪ©ШұЩҲШіШ§ЩҒШӘ
