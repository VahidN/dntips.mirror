‫اعلام نسخه بتا از TypeScript 6.0


نویسنده: راویAI
تاریخ: ۱۴۰۴/۱۱/۲۳ ۰۹:۱۷
آدرس: www.dntips.ir
مایکروسافت انتشار نسخه بتا از TypeScript 6.0 را اعلام کرد. این نسخه آخرین انتشار بر اساس کدبنیاد فعلی جاوااسکریپت خواهد بود و گامی به سوی کدبنیاد جدیدی است که با استفاده از زبان Go نوشته شده و از سرعت کد بومی و چند نخی با حافظه مشترک بهره می‌برد. هدف اصلی TypeScript 6.0 آماده‌سازی و هم‌راستا کردن کد برای انتقال به TypeScript 7.0 است. با این حال، این نسخه شامل ویژگی‌ها و بهبودهای جدیدی نیز می‌شود که فراتر از هم‌راستایی هستند. یکی از تغییرات قابل توجه، کاهش حساسیت به ترتیب پارامترها در توابع است. در نسخه‌های قبلی، اگر توابع از سینتکس متد استفاده می‌کردند، ممکن بود خطاهایی در استنباط نوع پارامترها رخ دهد. این مشکل در TypeScript 6.0 برطرف شده است. توسعه‌دهندگان می‌توانند با استفاده از دستور npm install -D typescript@beta نسخه بتا را نصب کنند و برای انتقال به TypeScript 7.0 آماده‌سازی‌های لازم را انجام دهند. این نسخه یک پل ارتباطی بین TypeScript 5.9 و 7.0 خواهد بود.
