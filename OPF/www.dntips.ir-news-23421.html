‫توسعه‌دهنده اپلیکیشنی برای ارتباط با افراد تصادفی از سراسر جهان با استفاده از .NET 8 و Flutter


نویسنده: راویAI
تاریخ: ۱۴۰۴/۱۱/۱۴ ۰۲:۰۵
آدرس: www.dntips.ir
یک توسعه‌دهنده در حال ساخت اپلیکیشنی به نام Meridian است که به کاربران امکان می‌دهد با افراد تصادفی از هر کشوری و با هر زبانی ارتباط برقرار کنند. این اپلیکیشن ترکیبی از Omegle، Duolingo و یک گذرنامه دیجیتال است و هدف آن ایجاد فرصت‌هایی برای گفتگو و آشنایی با فرهنگ‌های مختلف است. توسعه‌دهنده از .NET 8 Minimal APIs برای ساخت API بک‌اند و PostgreSQL بر روی Render برای ذخیره‌سازی داده‌ها استفاده کرده است. همچنین از Entity Framework Core برای دسترسی به داده‌ها بهره برده است. در قسمت فرانت‌اند، از HTML، CSS و JavaScript به صورت خالص (بدون فریم‌ورک) استفاده شده و از CSS custom properties برای تم‌پردازی و Intersection Observer برای انیمیشن‌های اسکرول بهره برده است. برای جلوگیری از اسپم، توسعه‌دهنده به جای استفاده از reCAPTCHA، از یک روش "honeypot" استفاده کرده است. این روش شامل یک فیلد پنهان است که ربات‌ها آن را پر می‌کنند، در حالی که کاربران انسانی متوجه آن نمی‌شوند. همچنین، از محدودیت نرخ (rate limiting) برای جلوگیری از ثبت‌نام‌های متعدد توسط یک کاربر استفاده شده است. این پروژه از Render برای میزبانی API، وب‌سایت استاتیک و پایگاه داده استفاده می‌کند و از GitHub برای استقرار خودکار کد بهره می‌برد. توسعه‌دهنده قصد دارد این پروژه را به صورت public به اشتراک بگذارد و از جامعه توسعه‌دهندگان برای بهبود آن کمک بگیرد. این پروژه نشان‌دهنده استفاده از فناوری‌های مدرن .NET و Flutter برای ساخت یک اپلیکیشن نوآورانه و کاربردی است. استفاده از .NET 8 Minimal APIs نشان‌دهنده تمایل به استفاده از APIهای ساده و کارآمد در توسعه بک‌اند است.
