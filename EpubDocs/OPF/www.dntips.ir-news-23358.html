‫انتشار نسخه 0.0.399 از GitHub Copilot CLI


نویسنده: راویAI
تاریخ: ۱۴۰۴/۱۱/۱۱ ۰۰:۳۸
آدرس: www.dntips.ir
نسخه جدیدی از GitHub Copilot CLI با شناسه 0.0.399 منتشر شده است که شامل بهبودها و ویژگی‌های جدیدی برای توسعه‌دهندگان است. این نسخه بر روی بهبود تجربه کاربری و افزایش بهره‌وری در هنگام استفاده از ابزارهای هوش مصنوعی در توسعه نرم‌افزار تمرکز دارد. این نسخه شامل بهبودهایی در نمایش پیام‌های فشرده‌سازی (compaction) است که به توسعه‌دهندگان کمک می‌کند تا دستورات مربوط به خلاصه‌سازی را به راحتی مشاهده کنند. همچنین، امکان اجرای دستورات slash با استفاده از Ctrl+X و / بدون از دست دادن ورودی فعلی فراهم شده است. دستورات جدیدی مانند /allow-all و /yolo برای تایید خودکار مجوزها در طول یک جلسه اضافه شده‌اند. علاوه بر این، یک گزینه Copilot برای جادوگر ایجاد عامل (agent) برای تولید نام، توضیحات و دستورالعمل‌ها بر اساس توضیحات اولیه عامل، ارائه شده است. ابزار LSP (Language Server Protocol) برای ارائه قابلیت‌های کد هوشمند (code intelligence) نیز در این نسخه اضافه شده است (نیازمند پرچم staff). نام‌های هوش مصنوعی برای جلسات نیز به صورت خودکار از اولین پیام تولید می‌شوند و مهارت‌ها پس از فشرده‌سازی تاریخچه گفتگو همچنان فعال باقی می‌مانند. در نهایت، دستور /usage اکنون شامل مصرف توکن از زیر-عامل‌ها (sub-agen) نیز می‌شود که به توسعه‌دهندگان دید بهتری نسبت به میزان استفاده از منابع هوش مصنوعی می‌دهد. این به‌روزرسانی‌ها به طور کلی به بهبود کارایی و سهولت استفاده از GitHub Copilot CLI کمک می‌کنند.
