وبلاگ‌های ایرانی

  • پایگاه داده‌ها و مشکل بسته‌های نرم افزاری در ایران
  • نگاهی به Ubuntu 8.10 نسخه جدید اوبونتو
  • گزارشی از PDC 2008 microsoftpdc.com و یک سری ویدیوی مرتبط با دات نت
  • یافتن کار نیمه وقت و کسب تجربه در زمینه نرم افزار (در ایران)
  • همکاران و مشکل مبادله تجربیات -( نکات پراکنده )
  • اوپن آفیس۳ در هفته‌ی اول سه میلیون بار دانلود شد.

Visual Studio

  • ویژوال استودیو 2010 و دات نت فریم ورک 4، نگارش CTP برای دریافت!
  • و تازه‌های آن (توسط یکی از اعضای اصلی تیم NHibernate)
  • دمویی از ایجاد نمودارهای UML با VS.Net 2010 . (خوشبختانه حجم این مورد در مقایسه با دموهای PDC2008 بسیار کمتر است!)
  • پوستر دات نت فریم ورک 4

امنیت اطلاعات
  • تولید جفت کلیدها و رمزنگاری و رمزگشایی اطلاعات در دات نت (الگوریتم RSA)

ASP. Net
  • نگارش CTP2 پروژهVelocity . (این پروژه قرار است راه حل جامع caching مایکروسافت برای ASP.Net باشد)
  • نتیجه استفاده از Response.Redirect
  • فشرده سازی سشن‌ها بدون تغییر کد

طراحی وب

  • فرم‌های وب خود را با CSS زیباتر کنید
  • 20 جلوه ویژه Ajax که باید از آنها مطلع باشید
  • 13 منوی جالب جاوا اسکریپتی

اس‌کیوال سرور

  • آیا می‌دانید چه زمانی پشتیبانی از نسخه‌های مختلف SQL Server به پایان می‌رسد؟
  • آشنایی با مبحث Log Shipping در SQL Server 2005

به روز رسانی‌ها

  • سرویس پک 3 بتا اس کیوال سرور 2005 هم ارائه شد (نگارش نهایی آن تا پایان سال جاری میلادی ارائه می‌شود.)

ابزارها
  • مجموعه CodeRush یک نگارش express رایگان نیز ارائه داد. (به نظر من نگارش کامل آن بهترین ابزار refactoring برای VS.Net است)
  • افزودنی رایگان CopySourceAsHtml مخصوص VS2008‌ منتشر شد.
  • برنامه رایگان SVN-Monitor منتشر شد.
  • کنترل‌های ترسیم نمودار رایگان از مایکروسافت (مخصوص دات نت فریم ورک 3 و نیم)

سی‌شارپ
  • نحوه صحیح overriding (تحریف) اپراتور == در سی شارپ
  • ویژگی‌های جدید C# 4.0 ، قسمت اول، واژه کلیدی جدیدی به نام dynamic
  • ویژگی‌های جدید C# 4.0 ، قسمت دوم، پارامترهای پیش فرض (یا آرگومانهای اختیاری). (چیزی شبیه به VB !! بدون نیاز به overloading برای پیاده سازی آن)
  • نکته‌ای در سی شارپ: اجزای یک رشته را با هم جمع نزنید! از تابع Format استفاده کنید.
  • آیا می‌دانستید از API مربوط به Java در دات نت هم می‌توانید استفاده کنید؟

دلفی
  • محصور کننده جدید SQLite‌ برای دلفی 2009 (رفع مشکلات عدم سازگاری نگارش قبلی با سیستم یونیکد دلفی 2009)
  • ویدیویی از Delphi Prism . (نگارشی از دلفی که به شکل افزونه‌ای کاملا یکپارچه در VS.Net قابل دسترسی است)

SharePoint
  • سطوح دسترسی کدها در SharePoint . (به شکل زیبایی این مساله را که مشکل اولیه اکثر وب پارت نویس‌ها است توضیح داده است)
  • دمویی از بهترین تجربیات کاری توسعه شیرپوینت

ویندوز
  • PHP بر روی ویندوز سرور 2008 چندین مرتبه سریعتر از Linux اجرا می‌شود.
  • ویندوز embedded 2009 ارائه شد.
  • بررسی اولیه نگارش pre-beta ویندوز 7

متفرقه

  • لوگوی جدید دات نت معرفی شد!
  • بررسی صحت لینک‌های رپیدشیر با یک نگاه!
  • 90 نکته برای اینکه برنامه نویس بهتری شویم.
  • بازی Quake در Silverlight !
  • دسترسی به PageGlimpse API از طریق یک کتابخانه دات نت (جهت تهیه تصویر thumbnail یک سایت)
  • و Hug a developer today !
