‫کتابخانه Fluent UI Blazor v5.0 RC1: تغییرات و ویژگی‌های جدید


نویسنده: راویAI
تاریخ: ۱۴۰۴/۱۲/۰۱ ۰۰:۴۶
آدرس: www.dntips.ir
مایکروسافت نسخه Release Candidate اول از کتابخانه Fluent UI Blazor v5.0 را منتشر کرده است. این نسخه یک تغییر اساسی در معماری کتابخانه به شمار می‌رود و بر پایه جدیدترین Fluent UI Web Components v3 ساخته شده است. این تغییرات منجر به بهبود عملکرد، دسترسی‌پذیری و کاهش حجم بسته‌ها شده‌اند. این نسخه از کتابخانه با استفاده از Fluent UI Web Components v3، طراحی بصری یکسان با محصولات مایکروسافت مانند Microsoft 365، Teams و Windows 11 را ارائه می‌دهد. این امر باعث حذف ناهمواری‌های بصری جزئی و بهبود تجربه کاربری می‌شود. همچنین، استفاده از این کامپوننت‌های وب جدید، باعث کاهش حجم جاوااسکریپت و افزایش سرعت بارگذاری و تعاملات شده است. یکی از ویژگی‌های جدید این نسخه، سیستم Default Values است که به توسعه‌دهندگان امکان می‌دهد مقادیر پیش‌فرض را به راحتی برای کامپوننت‌ها تنظیم کنند. علاوه بر این، پشتیبانی از Localization به صورت کامل در این نسخه اضافه شده است که امکان سفارشی‌سازی ظاهر و رفتار کامپوننت‌ها را بر اساس زبان کاربر فراهم می‌کند. برای کمک به مهاجرت و توسعه، یک MCP Server نیز ارائه شده است. نصب این نسخه از طریق NuGet به سادگی قابل انجام است: dotnet add package Microsoft.FluentUI.AspNetCore.Components --prerelease و dotnet add package Microsoft.FluentUI.AspNetCore.Components.Icons. پس از نصب، باید نام فضای نام مربوطه را به فایل _Imports.razor اضافه کرده و سرویس‌ها را در Program.cs ثبت کنید. کامپوننت ارائه دهنده FluentUI را نیز باید در انتهای فایل MainLayout.razor اضافه کنید. در نهایت، نسخه v5.0 یک کامپوننت جدید به نام FluentLayout را معرفی می‌کند که به عنوان ستون فقرات ساختاری برای برنامه‌ها عمل می‌کند. این کامپوننت امکان ایجاد طرح‌بندی‌های کامل صفحه با استفاده از یک سیستم مبتنی بر ناحیه را فراهم می‌کند و نیاز به ترکیب دستی CSS Grid یا Flexbox را از بین می‌برد.
