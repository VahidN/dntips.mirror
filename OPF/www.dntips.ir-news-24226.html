Ш§ЫҢЩҶ Щ…ЩӮШ§Щ„ЩҮ ЩҶШӯЩҲЩҮ ЩҫЫҢШ§ШҜЩҮвҖҢШіШ§ШІЫҢ ШіШұЩҲШұ Model Context Protocol (MCP) ШЁШ§ Ш§ШіШӘЩҒШ§ШҜЩҮ Ш§ШІ .NET SDK ШұШ§ ШЁЩҮ ЩҶЩ…Ш§ЫҢШҙ Щ…ЫҢвҖҢЪҜШ°Ш§ШұШҜ. Ш§ЫҢЩҶ ШіШұЩҲШұ Ш§ШЁШІШ§ШұЩҮШ§ЫҢ ШЁШұШұШіЫҢ ШіЩ„Ш§Щ…ШӘ ЩҲ ЩҫЫҢШҙвҖҢШЁЫҢЩҶЫҢ ШўШЁвҖҢЩҲЩҮЩҲШ§ ШұШ§ Ш§ШұШ§ШҰЩҮ Щ…ЫҢвҖҢШҜЩҮШҜ ЩҲ ШЁШ§ GitHub Copilot ШҜШұ VS Code ЫҢЪ©ЩҫШ§ШұЪҶЩҮ ШҙШҜЩҮ Ш§ШіШӘ ШӘШ§ ШЁЩҮ ШҜШіШӘЫҢШ§ШұШ§ЩҶ ЩҮЩҲШҙ Щ…ШөЩҶЩҲШ№ЫҢ Ш§Ш¬Ш§ШІЩҮ ШҜЩҮШҜ Ш§ШЁШІШ§ШұЩҮШ§ЫҢ ШіЩҒШ§ШұШҙЫҢ ШҙЩ…Ш§ ШұШ§ ЩҒШұШ§Ш®ЩҲШ§ЩҶЫҢ Ъ©ЩҶЩҶШҜ. ЩҫШұЩҲЪҳЩҮ ЫҢЪ© ЩҫЫҢШ§ШҜЩҮвҖҢШіШ§ШІЫҢ C# Ш§ШІ ШіШұЩҲШұ MCP Ш§ШіШӘ Ъ©ЩҮ Ш§ШЁШІШ§ШұЩҮШ§ЫҢ ЩҫЫҢШҙвҖҢШЁЫҢЩҶЫҢ ШўШЁвҖҢЩҲЩҮЩҲШ§ ЩҲ ШЁШұШұШіЫҢ ШіЩ„Ш§Щ…ШӘ ШұШ§ ШҜШұ Ш§Ш®ШӘЫҢШ§Шұ ЩӮШұШ§Шұ Щ…ЫҢвҖҢШҜЩҮШҜ. Ш§ЫҢЩҶ ШіШұЩҲШұ ШЁШ§ GitHub Copilot ШҜШұ VS Code Ш§ШҜШәШ§Щ… Щ…ЫҢвҖҢШҙЩҲШҜ ЩҲ Ш§Щ…Ъ©Ш§ЩҶ ЩҒШұШ§Ш®ЩҲШ§ЩҶЫҢ Ш§ШЁШІШ§ШұЩҮШ§ЫҢ ШіЩҒШ§ШұШҙЫҢ ШӘЩҲШіШ· ШҜШіШӘЫҢШ§ШұШ§ЩҶ ЩҮЩҲШҙ Щ…ШөЩҶЩҲШ№ЫҢ ШұШ§ ЩҒШұШ§ЩҮЩ… Щ…ЫҢвҖҢЪ©ЩҶШҜ. ЩҫШұЩҲЪҳЩҮ ШҙШ§Щ…Щ„ ШҜЩҲ Ш§ШЁШІШ§Шұ ping (ШЁШұШұШіЫҢ ШіЩ„Ш§Щ…ШӘ ШЁШ§ Ш§Щ…Ъ©Ш§ЩҶ ШЁШ§ШІШӘШ§ШЁ ЩҫЫҢШ§Щ…) ЩҲ get_weather (ШҜШұЫҢШ§ЩҒШӘ ЩҫЫҢШҙвҖҢШЁЫҢЩҶЫҢ ШўШЁвҖҢЩҲЩҮЩҲШ§ ШЁШұШ§ЫҢ ШҙЩҮШұЩҮШ§) Ш§ШіШӘ. ШЁШұШ§ЫҢ ШҙШұЩҲШ№ШҢ ШЁЩҮ SDK ЩҶШіШ®ЩҮ 8.0 ЫҢШ§ ШЁШ§Щ„Ш§ШӘШұ Ш§ШІ .NETШҢ Visual Studio Code ЩҲ Ш§ЩҒШІЩҲЩҶЩҮ GitHub Copilot (ШЁШ§ ЩҫШҙШӘЫҢШЁШ§ЩҶЫҢ Ш§ШІ MCP) ЩҲ Ш§ЩҒШІЩҲЩҶЩҮ C# Dev Kit (ШӘЩҲШөЫҢЩҮ ШҙШҜЩҮ) ЩҶЫҢШ§ШІ ШҜШ§ШұЫҢШҜ. ШіШ§Ш®ШӘШ§Шұ ЩҫШұЩҲЪҳЩҮ ШҙШ§Щ…Щ„ ЩҒШ§ЫҢЩ„вҖҢЩҮШ§ЫҢ McpServerTools.cs (ШӘШ№ШұЫҢЩҒ Ш§ШЁШІШ§ШұЩҮШ§)ШҢ Program.cs (ЩҶЩӮШ·ЩҮ ЩҲШұЩҲШҜ ШіШұЩҲШұ) ЩҲ McpServer.csproj (ЩҒШ§ЫҢЩ„ ЩҫШұЩҲЪҳЩҮ) Ш§ШіШӘ. ЩҮЩ…ЪҶЩҶЫҢЩҶ ЩҒШ§ЫҢЩ„ mcp.json ШЁШұШ§ЫҢ ШӘЩҶШёЫҢЩ…Ш§ШӘ ШіШұЩҲШұ MCP ШҜШұ ЩҫЩҲШҙЩҮ .vscode ЩҲШ¬ЩҲШҜ ШҜШ§ШұШҜ. ШЁШұШ§ЫҢ Ш§Ш¬ШұШ§ЫҢ ШіШұЩҲШұШҢ ШӘШұЩ…ЫҢЩҶШ§Щ„ ШұШ§ ШҜШұ ШұЫҢШҙЩҮ ЩҫШұЩҲЪҳЩҮ ШЁШ§ШІ Ъ©ШұШҜЩҮ ЩҲ ШҜШіШӘЩҲШұ dotnet run --project .\src\McpServer\McpServer\McpServer.csproj ШұШ§ Ш§Ш¬ШұШ§ Ъ©ЩҶЫҢШҜ. ЩҫШі Ш§ШІ ШұШ§ЩҮвҖҢШ§ЩҶШҜШ§ШІЫҢШҢ Щ…ЫҢвҖҢШӘЩҲШ§ЩҶЫҢШҜ ШЁШ§ Ш§ШіШӘЩҒШ§ШҜЩҮ Ш§ШІ Command Palette ШҜШұ VS Code (ШЁШ§ ЩҒШҙШұШҜЩҶ Ctrl + Shift + P ЫҢШ§ Cmd + Shift + P) ШіШұЩҲШұ ШұШ§ Щ„ЫҢШіШӘ Ъ©ШұШҜЩҮ ЩҲ ШЁЩҮ ШўЩҶ Щ…ШӘШөЩ„ ШҙЩҲЫҢШҜ. ЩҮЩ…ЪҶЩҶЫҢЩҶ Щ…ЫҢвҖҢШӘЩҲШ§ЩҶЫҢШҜ ЩҲШ¶Ш№ЫҢШӘ ШіШұЩҲШұ ШұШ§ ШҜШұ ЩҫЩҶЩ„ Ш®ШұЩҲШ¬ЫҢ ШЁШұШұШіЫҢ Ъ©ЩҶЫҢШҜ. ШЁШұШ§ЫҢ ШўШІЩ…Ш§ЫҢШҙ ШіШұЩҲШұ MCPШҢ Щ…ЫҢвҖҢШӘЩҲШ§ЩҶЫҢШҜ Ш§ШІ Ш§ШЁШІШ§ШұЩҮШ§ЫҢ ping ЩҲ get_weather ШҜШұ GitHub Copilot Chat Ш§ШіШӘЩҒШ§ШҜЩҮ Ъ©ЩҶЫҢШҜ. ШЁЩҮ Ш№ЩҶЩҲШ§ЩҶ Щ…Ш«Ш§Щ„ШҢ ШЁШ§ ЩҫШұШіЫҢШҜЩҶ "Ping the server" ШЁШ§ЫҢШҜ ЩҫШ§ШіШ® "вң… MCP server is alive." ШұШ§ ШҜШұЫҢШ§ЩҒШӘ Ъ©ЩҶЫҢШҜ. Ш§ЫҢЩҶ ЩҫШұЩҲЪҳЩҮ ЫҢЪ© ШұШ§ЩҮ ШӯЩ„ Ш№Щ…Щ„ЫҢ ШЁШұШ§ЫҢ ШӘЩҲШіШ№ЩҮвҖҢШҜЩҮЩҶШҜЪҜШ§ЩҶ .NET Ш§ШіШӘ Ъ©ЩҮ Щ…ЫҢвҖҢШ®ЩҲШ§ЩҮЩҶШҜ Ш§ШЁШІШ§ШұЩҮШ§ЫҢ ШіЩҒШ§ШұШҙЫҢ Ш®ЩҲШҜ ШұШ§ ШЁШ§ Ш§ШіШӘЩҒШ§ШҜЩҮ Ш§ШІ ЩҫШұЩҲШӘЪ©Щ„ MCP ШҜШұ ШҜШіШӘШұШі Щ…ШҜЩ„вҖҢЩҮШ§ЫҢ ЩҮЩҲШҙ Щ…ШөЩҶЩҲШ№ЫҢ ЩӮШұШ§Шұ ШҜЩҮЩҶШҜ.

Щ…Ш·Ш§Щ„ШЁ Щ…ШҙШ§ШЁЩҮ
    Ш§ШҙШӘШұШ§Ъ©вҖҢЩҮШ§ШұШ§ЩҮЩҶЩ…Ш§ЫҢ Ъ©Ш§ШұШЁШұШҜЫҢ MCPШ§ШҙШӘШұШ§Ъ©вҖҢЩҮШ§ЫҢЪ©ЩҫШ§ШұЪҶЩҮвҖҢШіШ§ШІЫҢ ШіШұЩҲШұ MCP GitHub ШҜШұ VS Code ШЁШұШ§ЫҢ ШӘЩҲШіШ№ЩҮвҖҢШҜЩҮЩҶШҜЪҜШ§ЩҶ .NETШ§ШҙШӘШұШ§Ъ©вҖҢЩҮШ§ШіШұЩҲШұЩҮШ§ЫҢ MCP Ш§ШІ ШұШ§ЩҮ ШҜЩҲШұ ШЁШ§ Ш§ШіШӘЩҒШ§ШҜЩҮ Ш§ШІ .NET SDK вҖ“ ЫҢЪ©ЩҫШ§ШұЪҶЩҮвҖҢШіШ§ШІЫҢ ШЁШ§ ШҜШ§ШҜЩҮвҖҢЩҮШ§ ЩҲ APIЩҮШ§ЫҢ ШіЩҒШ§ШұШҙЫҢШ§ШҙШӘШұШ§Ъ©вҖҢЩҮШ§Щ…Ш№ШұЩҒЫҢ ЩҫШұЩҲШӘЪ©Щ„ ШІЩ…ЫҢЩҶЩҮ Щ…ШҜЩ„ (Model Context Protocol - MCP) ШҜШұ ШЁШұЩҶШ§Щ…ЩҮвҖҢЩҶЩҲЫҢШіЫҢ ЩҲ ШӘЩҲШіШ№ЩҮ ШЁШ§ NET.Ш§ШҙШӘШұШ§Ъ©вҖҢЩҮШ§ШіШ§Ш®ШӘ Ш§ЩҫЩ„ЫҢЪ©ЫҢШҙЩҶ MCP ШЁШ§ C# вҖ“ ЩҶЩ…ЩҲЩҶЩҮ Ш§ЩҶШӘШ®Ш§ШЁвҖҢЪҜШұ ШұЩҶЪҜШ§ШҙШӘШұШ§Ъ©вҖҢЩҮШ§ШӘШЁШҜЫҢЩ„ Ш§ШЁШІШ§ШұЩҮШ§ЫҢ Ш®Ш· ЩҒШұЩ…Ш§ЩҶ ШҜШ§ШӘвҖҢЩҶШӘ ШЁЩҮ ШіШұЩҲШұ Щ…ШӯЩ„ЫҢ MCP ШЁШұШ§ЫҢ Ш§ШіШӘЩҒШ§ШҜЩҮ ШЁШ§ GitHub Copilot ШҜШұ VS CodeШ§ШҙШӘШұШ§Ъ©вҖҢЩҮШ§ЩҫШұЩҲШӘЪ©Щ„ ШІЩ…ЫҢЩҶЩҮ Щ…ШҜЩ„ ШҜШұ .NET: ШіШ§Ш®ШӘ ЩҲ Ш§ШіШӘЩҒШ§ШҜЩҮ Ш§ШІ Ш§ШЁШІШ§ШұЩҮШ§ЫҢ ЩҮЩҲШҙ Щ…ШөЩҶЩҲШ№ЫҢ ЩҮЩ…ЩҮвҖҢЪ©Ш§ШұЩҮШ§ШҙШӘШұШ§Ъ©вҖҢЩҮШ§MCP ШЁШұШ§ЫҢ .NET: Ш№Ш§Щ…Щ„вҖҢЩҮШ§ЫҢ ЩҮЩҲШҙ Щ…ШөЩҶЩҲШ№ЫҢ Ъ©ЩҮ Щ…ЫҢвҖҢШӘЩҲШ§ЩҶЩҶШҜ Ъ©Ш§ШұЩҮШ§ЫҢ ЩҲШ§ЩӮШ№ЫҢ Ш§ЩҶШ¬Ш§Щ… ШҜЩҮЩҶШҜШ§ШҙШӘШұШ§Ъ©вҖҢЩҮШ§Щ…Ш№ШұЩҒЫҢ Щ…Ш®ШІЩҶ В«MCP for BeginnersВ»Ш§ШҙШӘШұШ§Ъ©вҖҢЩҮШ§ЩҫШұЩҲШӘЪ©Щ„ ШІЩ…ЫҢЩҶЩҮ Щ…ШҜЩ„ (MCP) ЪҶЫҢШіШӘ ЩҲ ЪҶШұШ§ ЩҮЩ…ЩҮ ШҜШұШЁШ§ШұЩҮвҖҢЫҢ ШўЩҶ ШөШӯШЁШӘ Щ…ЫҢвҖҢЪ©ЩҶЩҶШҜШҹ
