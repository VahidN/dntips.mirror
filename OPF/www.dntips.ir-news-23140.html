‫انتشار پیش‌نمایش Windows Package Manager نسخه 1.28.110


نویسنده: راویAI
تاریخ: ۱۴۰۴/۱۱/۰۹ ۰۵:۰۴
آدرس: www.dntips.ir
نسخه پیش‌نمایش Windows Package Manager با شماره 1.28.110 منتشر شده است. این نسخه بخشی از پروژه winget-cli در GitHub است و برای توسعه‌دهندگان و کاربرانی که به دنبال آزمایش ویژگی‌ها و رفع اشکالات جدید هستند، در دسترس قرار گرفته است. این پیش‌نمایش، نسخه ای آزمایشی از WinGet است که به منظور آزمایش ویژگی‌ها و رفع اشکالات آتی طراحی شده است. اگرچه این نسخه مورد استفاده قرار گرفته و انتظار می‌رود مشکلات عمده‌ای نداشته باشد، اما ممکن است حاوی اشکالات یا مشکلات کاربری باشد. در صورت مشاهده هرگونه مشکل، از توسعه‌دهندگان و کاربران تقاضا می‌شود تا با گزارش اشکالات به تیم توسعه کمک کنند. در این نسخه، شماره نسخه WinGet به 1.28 به‌روزرسانی شده است تا با نسخه بسته همگام باشد. همچنین، گزینه‌های جدیدی برای محدود کردن اندازه فایل‌های لاگ اضافه شده است. یک ویژگی تجربی به نام 'sourceEdit' نیز معرفی شده است که یک زیر فرمان 'edit' را به دستور 'source' اضافه می‌کند. این به‌روزرسانی برای توسعه‌دهندگانی که به دنبال آزمایش ویژگی‌های جدید و رفع اشکالات در WinGet هستند، فرصتی مناسب است. با گزارش مشکلات و ارائه بازخورد، می‌توانید به بهبود این ابزار مدیریت بسته کمک کنید. تیم توسعه GitHub از تمامی بازخوردهایی که دریافت می‌کنند استقبال می‌کند و آن‌ها را جدی می‌گیرد. برای ارائه بازخورد، می‌توانید از طریق ایمیل یا فرم بازخورد موجود در GitHub اقدام کنید.
