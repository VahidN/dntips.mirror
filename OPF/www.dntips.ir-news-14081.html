‫راه معقول نصب و مدیریت نرم افزار در ویندوز توسط Chocolatey


نویسنده: کورشا
تاریخ: ۱۳۹۷/۱۰/۲۲ ۱۳:۱۴
آدرس: www.dntips.ir
به راحتی تمام جنبه‌های نرم افزاری ویندوز (نصب، پیکر بندی، ارتقاء و حذف) را مدیریت کنید.
 Chocolatey قابل اطمینان‌ترین راه نصب نرم افزار می‌باشد, زمانی که نرم افزار به صورت Package  شده  است، که به راحتی توسط Chocolatey  قابل نصب بر روی ویندوز است . 

نصب Chocolatey 
پیش نیاز:
  • Windows 7+ / Windows Server 2003
  • PowerShell v2
  • .NET Framework 4
 روش نصب:
1) توسط (cmd) Command prompt
cmd را به صورت Run as administrator اجرا سپس قطعه کد زیر را در آن اجرا کنید: 
@"%SystemRoot%\System32\WindowsPowerShell\v1.0\powershell.exe" -NoProfile -InputFormat None -ExecutionPolicy Bypass -Command "iex ((New-Object System.Net.WebClient).DownloadString('https://chocolatey.org/install.ps1'))" && SET "PATH=%PATH%;%ALLUSERSPROFILE%\chocolatey\bin"
2) توسط PowerShell
PowerShell را به صورت Run as administrator اجرا سپس قطعه کد زیر را در آن اجرا کنید: 
Set-ExecutionPolicy Bypass -Scope Process -Force; iex ((New-Object System.Net.WebClient).DownloadString('https://chocolatey.org/install.ps1'))

بعد از نصب Chocolatey به راحتی می‌توانید نرم افزار‌های بسته بندی شده توسط Chocolatey را توسط خط فرمان نصب کنید.
به طور مثال برای نصب نرم افزار معروف ++ nodepad کافی هست کد زیر را در خط فرمان اجرا کنیم:
choco install notepadplusplus.install

دسترسی به تمام Package‌های ارائه شده توسط Chocolatey
https://chocolatey.org/packages
