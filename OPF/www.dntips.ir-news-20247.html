Щ…ЩӮШҜЩ…ЩҮ
ШҜШұ ШіШ§Щ„вҖҢЩҮШ§ЫҢ Ш§Ш®ЫҢШұШҢ Ш§Щ„ЪҜЩҲЩҮШ§ЫҢ Щ…Ш№Щ…Ш§ШұЫҢ Щ…Ш§ЩҶЩҶШҜ Щ…Ш№Щ…Ш§ШұЫҢ ШӘЩ…ЫҢШІ (Clean Architecture) ЩҲ Ш¬ШҜШ§ШіШ§ШІЫҢ Щ…ШіШҰЩҲЩ„ЫҢШӘвҖҢЩҮШ§ЫҢ ШҜШіШӘЩҲШұ ЩҲ ЩҫШұШівҖҢЩҲШ¬ЩҲ (CQRS) ШҜШұ Ш§Ъ©ЩҲШіЫҢШіШӘЩ… .NET ШЁЩҮ ШҙШҜШӘ Щ…ЩҲШұШҜ ШӘЩҲШ¬ЩҮ ЩӮШұШ§Шұ ЪҜШұЩҒШӘЩҮвҖҢШ§ЩҶШҜ. ШҜШұ Ш§ЫҢЩҶ Щ…ЫҢШ§ЩҶШҢ Ъ©ШӘШ§ШЁШ®Ш§ЩҶЩҮ MediatR ШЁЩҮ Ш№ЩҶЩҲШ§ЩҶ Ш§ШЁШІШ§ШұЫҢ Щ…ШӯШЁЩҲШЁ ШЁШұШ§ЫҢ ЩҫЫҢШ§ШҜЩҮвҖҢШіШ§ШІЫҢ Ш§Щ„ЪҜЩҲЫҢ ЩҲШ§ШіШ·ЩҮ (Mediator Pattern) ЩҲ ЩҫШҙШӘЫҢШЁШ§ЩҶЫҢ Ш§ШІ CQRSШҢ ШҜШұ ШЁШіЫҢШ§ШұЫҢ Ш§ШІ ЩҫШұЩҲЪҳЩҮвҖҢЩҮШ§ ШЁЩҮ Ъ©Ш§Шұ ЪҜШұЩҒШӘЩҮ ШҙШҜЩҮ Ш§ШіШӘ. Ш§ЫҢЩҶ Ъ©ШӘШ§ШЁШ®Ш§ЩҶЩҮ ШЁШ§ ЩҮШҜЩҒ Ъ©Ш§ЩҮШҙ ЩҲШ§ШЁШіШӘЪҜЫҢвҖҢЩҮШ§ЫҢ Щ…ШіШӘЩӮЫҢЩ… ШЁЫҢЩҶ Ш§Ш¬ШІШ§ЫҢ ШЁШұЩҶШ§Щ…ЩҮ ЩҲ Ш§ЫҢШ¬Ш§ШҜ ШіШ§Ш®ШӘШ§ШұЫҢ Щ…ЩҶШіШ¬Щ… ЩҲ ЩӮШ§ШЁЩ„ ШӘШіШӘШҢ Ш·ШұШ§ШӯЫҢ ШҙШҜЩҮ Ш§ШіШӘ. Ш§Щ…Ш§ ШўЫҢШ§ Ш§ШіШӘЩҒШ§ШҜЩҮ Ш§ШІ MediatR ШҜШұ ЩҮШұ ЩҫШұЩҲЪҳЩҮвҖҢШ§ЫҢ Ш¶ШұЩҲШұЫҢ ЫҢШ§ ШӯШӘЫҢ ШЁЩҮЫҢЩҶЩҮ Ш§ШіШӘШҹ ШЁШұШ®ЫҢ Щ…Ш№ШӘЩӮШҜЩҶШҜ Ъ©ЩҮ Ш§ЫҢЩҶ Ш§ШЁШІШ§Шұ ШҜШұ ЩҫШұЩҲЪҳЩҮвҖҢЩҮШ§ЫҢ ШіШ§ШҜЩҮ Щ…ЫҢвҖҢШӘЩҲШ§ЩҶШҜ ЩҫЫҢЪҶЫҢШҜЪҜЫҢ ШәЫҢШұШ¶ШұЩҲШұЫҢ Ш§ЫҢШ¬Ш§ШҜ Ъ©ЩҶШҜШҢ ШҜШұ ШӯШ§Щ„ЫҢ Ъ©ЩҮ ШҜЫҢЪҜШұШ§ЩҶ ШўЩҶ ШұШ§ ШұШ§ЩҮвҖҢШӯЩ„ЫҢ ЩӮШҜШұШӘЩ…ЩҶШҜ ШЁШұШ§ЫҢ Щ…ШҜЫҢШұЫҢШӘ Щ…ЩҶШ·ЩӮ ШЁШұЩҶШ§Щ…ЩҮ Щ…ЫҢвҖҢШҜШ§ЩҶЩҶШҜ. ШҜШұ Ш§ЫҢЩҶ Щ…ЩӮШ§Щ„ЩҮШҢ ШЁШ§ ШӘШӯЩ„ЫҢЩ„ ШҜЫҢШҜЪҜШ§ЩҮвҖҢЩҮШ§ЫҢ Щ…Ш®ШӘЩ„ЩҒШҢ ШЁЩҮ ШЁШұШұШіЫҢ Щ…ШІШ§ЫҢШ§ШҢ Щ…Ш№Ш§ЫҢШЁ ЩҲ ШҙШұШ§ЫҢШ· Ш§ШіШӘЩҒШ§ШҜЩҮ Ш§ШІ MediatR Щ…ЫҢвҖҢЩҫШұШҜШ§ШІЫҢЩ… ШӘШ§ ШЁЩҮ ШӘЩҲШіШ№ЩҮвҖҢШҜЩҮЩҶШҜЪҜШ§ЩҶ .NET Ъ©Щ…Ъ© Ъ©ЩҶЫҢЩ… ШӘШөЩ…ЫҢЩ… ШўЪҜШ§ЩҮШ§ЩҶЩҮвҖҢШӘШұЫҢ ШЁЪҜЫҢШұЩҶШҜ.

MediatR ЪҶЫҢШіШӘ ЩҲ ЪҶЪҜЩҲЩҶЩҮ Ъ©Ш§Шұ Щ…ЫҢвҖҢЪ©ЩҶШҜШҹ
MediatR ЫҢЪ© Ъ©ШӘШ§ШЁШ®Ш§ЩҶЩҮ Щ…ШӘЩҶвҖҢШЁШ§ШІ (Open-Source) Ш§ШіШӘ Ъ©ЩҮ Ш§Щ„ЪҜЩҲЫҢ ЩҲШ§ШіШ·ЩҮ ШұШ§ ШҜШұ ШЁШұЩҶШ§Щ…ЩҮвҖҢЩҮШ§ЫҢ .NET ЩҫЫҢШ§ШҜЩҮвҖҢШіШ§ШІЫҢ Щ…ЫҢвҖҢЪ©ЩҶШҜ. Ш§ЫҢЩҶ Ш§Щ„ЪҜЩҲ ШЁШ§ ЩӮШұШ§Шұ ШҜШ§ШҜЩҶ ЫҢЪ© ЩҲШ§ШіШ·ЩҮ Щ…ШұЪ©ШІЫҢ ШЁЫҢЩҶ ЩҒШұШіШӘЩҶШҜЩҮ ШҜШұШ®ЩҲШ§ШіШӘвҖҢЩҮШ§ ЩҲ ЪҜЫҢШұЩҶШҜЩҮвҖҢЩҮШ§ (HandlerЩҮШ§)ШҢ ЩҲШ§ШЁШіШӘЪҜЫҢ Щ…ШіШӘЩӮЫҢЩ… ШЁЫҢЩҶ Ш§Ш¬ШІШ§ЫҢ ШЁШұЩҶШ§Щ…ЩҮ ШұШ§ ШӯШ°ЩҒ Щ…ЫҢвҖҢЪ©ЩҶШҜ. ШҜШұ Ш№Щ…Щ„ШҢ MediatR ШЁЩҮ ШҙЩ…Ш§ Ш§Ш¬Ш§ШІЩҮ Щ…ЫҢвҖҢШҜЩҮШҜ ШҜШұШ®ЩҲШ§ШіШӘвҖҢЩҮШ§ (Requests) Щ…Ш§ЩҶЩҶШҜ ШҜШіШӘЩҲШұШ§ШӘ (Commands) ЫҢШ§ ЩҫШұШівҖҢЩҲШ¬ЩҲЩҮШ§ (Queries) ШұШ§ ШӘШ№ШұЫҢЩҒ Ъ©ЩҶЫҢШҜ ЩҲ ШіЩҫШі Ш§ЫҢЩҶ ШҜШұШ®ЩҲШ§ШіШӘвҖҢЩҮШ§ ШұШ§ ШЁЩҮ HandlerЩҮШ§ЫҢ Щ…ШұШЁЩҲШ·ЩҮ Ш§ШұШіШ§Щ„ Ъ©ЩҶЫҢШҜ. ШЁЩҮ Ш№ЩҶЩҲШ§ЩҶ Щ…Ш«Ш§Щ„ШҢ ШҜШұ ЫҢЪ© ЩҫШұЩҲЪҳЩҮ ШіШ§ШҜЩҮШҢ Щ…ЫҢвҖҢШӘЩҲШ§ЩҶЫҢШҜ ЫҢЪ© ШҜШіШӘЩҲШұ ШЁШұШ§ЫҢ Ш§ЩҒШІЩҲШҜЩҶ ЫҢЪ© ШҙЫҢШЎ Ш¬ШҜЫҢШҜ ШӘШ№ШұЫҢЩҒ Ъ©ЩҶЫҢШҜ:
public record AddGoatCommand(string Name, string Description) : ICommand<AddGoatResponse>;
public record AddGoatResponse(Guid GoatId);

public class AddGoatHandler : IHandler<AddGoatCommand, AddGoatResponse>
{
    private readonly IRepository _repository;
    public AddGoatHandler(IRepository repository) => _repository = repository;

    public Task<AddGoatResponse> Handle(AddGoatCommand request, CancellationToken cancellationToken)
    {
        var goat = new Goat { Name = request.Name, Description = request.Description };
        _repository.Save(goat);
        return Task.FromResult(new AddGoatResponse(goat.Id));
    }
}
ШҜШұ Ш§ЫҢЩҶШ¬Ш§ШҢ ЩҲШ§ШіШ·ЩҮ ШҜШұШ®ЩҲШ§ШіШӘ ШұШ§ ШЁЩҮ Handler Щ…ЩҶШ§ШіШЁ ЩҮШҜШ§ЫҢШӘ Щ…ЫҢвҖҢЪ©ЩҶШҜШҢ ШЁШҜЩҲЩҶ Ш§ЫҢЩҶЪ©ЩҮ ЩҒШұШіШӘЩҶШҜЩҮ (Щ…Ш«Щ„Ш§ЩӢ ЫҢЪ© Ъ©ЩҶШӘШұЩ„Шұ) ЩҶЫҢШ§ШІЫҢ ШЁЩҮ ШҙЩҶШ§Ш®ШӘ Щ…ШіШӘЩӮЫҢЩ… Handler ШҜШ§ШҙШӘЩҮ ШЁШ§ШҙШҜ. Ш§ЫҢЩҶ ШұЩҲЫҢЪ©ШұШҜ ШЁЩҮ ЩҲЫҢЪҳЩҮ ШҜШұ ЩҫШұЩҲЪҳЩҮвҖҢЩҮШ§ЫҢЫҢ ШЁШ§ Щ…Ш№Щ…Ш§ШұЫҢ ШӘЩ…ЫҢШІШҢ ШЁЩҮ ШҜЩ„ЫҢЩ„ Ш¬ШҜШ§ШіШ§ШІЫҢ Щ„Ш§ЫҢЩҮвҖҢЩҮШ§ ЩҲ Ъ©Ш§ЩҮШҙ ЩҲШ§ШЁШіШӘЪҜЫҢвҖҢЩҮШ§ШҢ Ш¬Ш°Ш§ШЁ Ш§ШіШӘ.

Щ…ШІШ§ЫҢШ§ЫҢ Ш§ШіШӘЩҒШ§ШҜЩҮ Ш§ШІ MediatR
ЫҢЪ©ЫҢ Ш§ШІ ШЁШІШұЪҜвҖҢШӘШұЫҢЩҶ Щ…ШІШ§ЫҢШ§ЫҢ MediatRШҢ Ш¬ШҜШ§ШіШ§ШІЫҢ ЩҲШ§Ш¶Шӯ ШЁЫҢЩҶ Щ„Ш§ЫҢЩҮвҖҢЩҮШ§ЫҢ ШЁШұЩҶШ§Щ…ЩҮ Ш§ШіШӘ. ШЁШ§ Ш§ШіШӘЩҒШ§ШҜЩҮ Ш§ШІ Ш§ЫҢЩҶ Ъ©ШӘШ§ШЁШ®Ш§ЩҶЩҮШҢ Ъ©ЩҶШӘШұЩ„ШұЩҮШ§ ЫҢШ§ ЩҶЩӮШ§Ш· ЩҲШұЩҲШҜЫҢ (Endpoints) ЩҶЫҢШ§ШІЫҢ ШЁЩҮ ШӘШІШұЫҢЩӮ Щ…ШіШӘЩӮЫҢЩ… ШіШұЩҲЫҢШівҖҢЩҮШ§ ЩҶШҜШ§ШұЩҶШҜ ЩҲ ШӘЩҶЩҮШ§ ШЁШ§ ЩҲШ§ШіШ·ЩҮ Ш§ШұШӘШЁШ§Ш· ШЁШұЩӮШұШ§Шұ Щ…ЫҢвҖҢЪ©ЩҶЩҶШҜ. Ш§ЫҢЩҶ Щ…ЩҲШ¶ЩҲШ№ ШӘШіШӘвҖҢЩҫШ°ЫҢШұЫҢ (Testability) ШұШ§ ШЁЩҮШЁЩҲШҜ Щ…ЫҢвҖҢШЁШ®ШҙШҜШҢ ШІЫҢШұШ§ Щ…ЫҢвҖҢШӘЩҲШ§ЩҶЫҢШҜ HandlerЩҮШ§ ШұШ§ ШЁЩҮ ШөЩҲШұШӘ Ш¬ШҜШ§ЪҜШ§ЩҶЩҮ ШўШІЩ…Ш§ЫҢШҙ Ъ©ЩҶЫҢШҜ. ШЁЩҮ Ш№Щ„Ш§ЩҲЩҮШҢ MediatR Ш§Щ…Ъ©Ш§ЩҶ ЩҫЫҢШ§ШҜЩҮвҖҢШіШ§ШІЫҢ ШұЩҒШӘШ§ШұЩҮШ§ЫҢ ШіШұШ§ШіШұЫҢ (Pipeline Behaviors) Щ…Ш§ЩҶЩҶШҜ Ш§Ш№ШӘШЁШ§ШұШіЩҶШ¬ЫҢ (Validation) ЫҢШ§ Щ„Ш§ЪҜвҖҢЪҜШ°Ш§ШұЫҢ (Logging) ШұШ§ ЩҒШұШ§ЩҮЩ… Щ…ЫҢвҖҢЪ©ЩҶШҜ Ъ©ЩҮ ШЁЩҮ ШөЩҲШұШӘ Щ…ШӘЩ…ШұЪ©ШІ Ш§Ш№Щ…Ш§Щ„ Щ…ЫҢвҖҢШҙЩҲЩҶШҜ.
ШЁШұШ§ЫҢ Щ…Ш«Ш§Щ„ШҢ ЩҒШұШ¶ Ъ©ЩҶЫҢШҜ ШЁШ®ЩҲШ§ЩҮЫҢШҜ ЩҮЩҶЪҜШ§Щ… Ш§ЩҒШІЩҲШҜЩҶ ЫҢЪ© ШҙЫҢШЎШҢ ЫҢЪ© ШұШҜЩҫШ§ЫҢ ШӯШіШ§ШЁШұШіЫҢ (Audit Trail) Ш«ШЁШӘ Ъ©ЩҶЫҢШҜ. ШЁШ§ MediatR Щ…ЫҢвҖҢШӘЩҲШ§ЩҶЫҢШҜ Ш§ЫҢЩҶ Ъ©Ш§Шұ ШұШ§ ШЁЩҮ ШіШ§ШҜЪҜЫҢ ШЁШ§ ШӘШ№ШұЫҢЩҒ ЫҢЪ© ШұЩҒШӘШ§Шұ (Behavior) Ш§ЩҶШ¬Ш§Щ… ШҜЩҮЫҢШҜШҢ ШЁШҜЩҲЩҶ Ш§ЫҢЩҶЪ©ЩҮ Щ…ЩҶШ·ЩӮ Ш§ШөЩ„ЫҢ Handler ШӘШәЫҢЫҢШұ Ъ©ЩҶШҜ. Ш§ЫҢЩҶ Ш§ЩҶШ№Ш·Ш§ЩҒвҖҢЩҫШ°ЫҢШұЫҢ ШҜШұ ЩҫШұЩҲЪҳЩҮвҖҢЩҮШ§ЫҢ ЩҫЫҢЪҶЫҢШҜЩҮ Ъ©ЩҮ ЩҶЫҢШ§ШІ ШЁЩҮ Щ…ШҜЫҢШұЫҢШӘ ЪҶЩҶШҜЫҢЩҶ ЩҶЪҜШұШ§ЩҶЫҢ ШіШұШ§ШіШұЫҢ (Cross-Cutting Concerns) ШҜШ§ШұЩҶШҜШҢ ШЁШіЫҢШ§Шұ Ш§ШұШІШҙЩ…ЩҶШҜ Ш§ШіШӘ.

Щ…Ш№Ш§ЫҢШЁ ЩҲ ЪҶШ§Щ„ШҙвҖҢЩҮШ§ЫҢ MediatR
ШЁШ§ ЩҲШ¬ЩҲШҜ Щ…ШІШ§ЫҢШ§ШҢ Ш§ШіШӘЩҒШ§ШҜЩҮ Ш§ШІ MediatR ШЁШҜЩҲЩҶ ЪҶШ§Щ„Шҙ ЩҶЫҢШіШӘ. ЫҢЪ©ЫҢ Ш§ШІ Ш§ЩҶШӘЩӮШ§ШҜШ§ШӘ Ш§ШөЩ„ЫҢШҢ ЩҫЫҢЪҶЫҢШҜЪҜЫҢ Ш§Ш¶Ш§ЩҒЩҮвҖҢШ§ЫҢ Ш§ШіШӘ Ъ©ЩҮ ШЁЩҮ ЩҫШұЩҲЪҳЩҮвҖҢЩҮШ§ЫҢ ШіШ§ШҜЩҮ ЩҲШ§ШұШҜ Щ…ЫҢвҖҢЪ©ЩҶШҜ. ШҜШұ ЩҫШұЩҲЪҳЩҮвҖҢЩҮШ§ЫҢЫҢ Ъ©ЩҮ ШӘШ№ШҜШ§ШҜ ШҜШұШ®ЩҲШ§ШіШӘвҖҢЩҮШ§ Щ…ШӯШҜЩҲШҜ Ш§ШіШӘ (Щ…Ш«Щ„Ш§ЩӢ Ъ©Щ…ШӘШұ Ш§ШІ 10 ЩҶЩӮШ·ЩҮ ЩҲШұЩҲШҜЫҢ)ШҢ ШӘШ№ШұЫҢЩҒ ШҜШұШ®ЩҲШ§ШіШӘвҖҢЩҮШ§ШҢ ЩҫШ§ШіШ®вҖҢЩҮШ§ ЩҲ HandlerЩҮШ§ Щ…ЫҢвҖҢШӘЩҲШ§ЩҶШҜ ШЁЫҢШҙ Ш§ШІ ШӯШҜ ШЁЩҮ ЩҶШёШұ ШЁШұШіШҜ. ШЁЩҮ Ш№ЩҶЩҲШ§ЩҶ Щ…Ш«Ш§Щ„ШҢ ШҜШұ ЫҢЪ© ШЁШұЩҶШ§Щ…ЩҮ ШіШ§ШҜЩҮ Ъ©ЩҮ ЩҒЩӮШ· ЪҶЩҶШҜ Ш№Щ…Щ„ЫҢШ§ШӘ CRUD (Ш§ЫҢШ¬Ш§ШҜШҢ Ш®ЩҲШ§ЩҶШҜЩҶШҢ ШЁЩҮвҖҢШұЩҲШІШұШіШ§ЩҶЫҢШҢ ШӯШ°ЩҒ) Ш§ЩҶШ¬Ш§Щ… Щ…ЫҢвҖҢШҜЩҮШҜШҢ Ш§ШіШӘЩҒШ§ШҜЩҮ Ш§ШІ MediatR Щ…Щ…Ъ©ЩҶ Ш§ШіШӘ ШІЩ…Ш§ЩҶ ШӘЩҲШіШ№ЩҮ ШұШ§ Ш§ЩҒШІШ§ЫҢШҙ ШҜЩҮШҜ ШЁШҜЩҲЩҶ Ш§ЫҢЩҶЪ©ЩҮ Ш§ШұШІШҙ ЩӮШ§ШЁЩ„ ШӘЩҲШ¬ЩҮЫҢ Ш§Ш¶Ш§ЩҒЩҮ Ъ©ЩҶШҜ.
Щ…Ш«Ш§Щ„ ШІЫҢШұ ЩҶШҙШ§ЩҶ Щ…ЫҢвҖҢШҜЩҮШҜ Ъ©ЩҮ ЪҶЪҜЩҲЩҶЩҮ ЫҢЪ© Ш№Щ…Щ„ЫҢШ§ШӘ ШіШ§ШҜЩҮ ШЁШҜЩҲЩҶ MediatR Щ…ЫҢвҖҢШӘЩҲШ§ЩҶШҜ Щ…ШіШӘЩӮЫҢЩ…вҖҢШӘШұ ШЁШ§ШҙШҜ:
public class GoatService
{
    private readonly IRepository _repository;
    public GoatService(IRepository repository) => _repository = repository;

    public Guid AddGoat(string name, string description)
    {
        var goat = new Goat { Name = name, Description = description };
        _repository.Save(goat);
        return goat.Id;
    }
}
ШҜШұ Ш§ЫҢЩҶШ¬Ш§ШҢ ШіШұЩҲЫҢШі Щ…ШіШӘЩӮЫҢЩ…Ш§ЩӢ ШӘЩҲШіШ· Ъ©ЩҶШӘШұЩ„Шұ ЩҒШұШ§Ш®ЩҲШ§ЩҶЫҢ Щ…ЫҢвҖҢШҙЩҲШҜ ЩҲ ЩҶЫҢШ§ШІЫҢ ШЁЩҮ ЩҲШ§ШіШ·ЩҮ ЩҶЫҢШіШӘ. Ш§ЫҢЩҶ ШұЩҲШҙ ШҜШұ ЩҫШұЩҲЪҳЩҮвҖҢЩҮШ§ЫҢ Ъ©ЩҲЪҶЪ©вҖҢШӘШұШҢ Ш®ЩҲШ§ЩҶШ§ЫҢЫҢ (Readability) ЩҲ ШіШ§ШҜЪҜЫҢ ШЁЫҢШҙШӘШұЫҢ ШҜШ§ШұШҜ. Ш§Щ…Ш§ Щ…ЩҶШӘЩӮШҜШ§ЩҶ Щ…ЫҢвҖҢЪҜЩҲЫҢЩҶШҜ Ъ©ЩҮ ШЁШ§ Ш§ЩҒШІШ§ЫҢШҙ Щ…ЩӮЫҢШ§Ші ЩҫШұЩҲЪҳЩҮШҢ Ш§ЫҢЩҶ ШұЩҲЫҢЪ©ШұШҜ Щ…ЫҢвҖҢШӘЩҲШ§ЩҶШҜ ШЁЩҮ ЩҲШ§ШЁШіШӘЪҜЫҢвҖҢЩҮШ§ЫҢ ЩҫЫҢЪҶЫҢШҜЩҮ ЩҲ Ъ©ШҜ ШіШ®ШӘвҖҢШӘШұ ШЁШұШ§ЫҢ ЩҶЪҜЩҮШҜШ§ШұЫҢ Щ…ЩҶШ¬Шұ ШҙЩҲШҜ.

ШўЫҢШ§ MediatR ШЁШұШ§ЫҢ CQRS Ш¶ШұЩҲШұЫҢ Ш§ШіШӘШҹ
ЫҢЪ©ЫҢ Ш§ШІ ШіЩҲШЎШӘЩҒШ§ЩҮЩ…вҖҢЩҮШ§ЫҢ ШұШ§ЫҢШ¬ Ш§ЫҢЩҶ Ш§ШіШӘ Ъ©ЩҮ ШЁШұШ§ЫҢ ЩҫЫҢШ§ШҜЩҮвҖҢШіШ§ШІЫҢ CQRS ШӯШӘЩ…Ш§ЩӢ ШЁШ§ЫҢШҜ Ш§ШІ MediatR Ш§ШіШӘЩҒШ§ШҜЩҮ Ъ©ШұШҜ. ШҜШұ ЩҲШ§ЩӮШ№ШҢ CQRS ЫҢЪ© Ш§Щ„ЪҜЩҲЫҢ Щ…Ш№Щ…Ш§ШұЫҢ Ш§ШіШӘ Ъ©ЩҮ ШЁШұ Ш¬ШҜШ§ШіШ§ШІЫҢ Ш№Щ…Щ„ЫҢШ§ШӘ Ш®ЩҲШ§ЩҶШҜЩҶ (Queries) ЩҲ ЩҶЩҲШҙШӘЩҶ (Commands) ШӘЩ…ШұЪ©ШІ ШҜШ§ШұШҜ ЩҲ Щ…ЫҢвҖҢШӘЩҲШ§ЩҶШҜ ШЁШҜЩҲЩҶ ЩҮЫҢЪҶ Ъ©ШӘШ§ШЁШ®Ш§ЩҶЩҮвҖҢШ§ЫҢ ЩҫЫҢШ§ШҜЩҮвҖҢШіШ§ШІЫҢ ШҙЩҲШҜ. ШЁШұШ§ЫҢ Щ…Ш«Ш§Щ„ШҢ Щ…ЫҢвҖҢШӘЩҲШ§ЩҶЫҢШҜ ШЁЩҮ ШіШ§ШҜЪҜЫҢ ШҜЩҲ ШұШ§ШЁШ· Ш¬ШҜШ§ЪҜШ§ЩҶЩҮ ШЁШұШ§ЫҢ ШҜШіШӘЩҲШұШ§ШӘ ЩҲ ЩҫШұШівҖҢЩҲШ¬ЩҲЩҮШ§ ШӘШ№ШұЫҢЩҒ Ъ©ЩҶЫҢШҜ:
public interface ICommand<T>
{
    Task<T> Execute();
}

public interface IQuery<T>
{
    Task<T> Fetch();
}
ШіЩҫШіШҢ HandlerЩҮШ§ ШұШ§ Щ…ШіШӘЩӮЫҢЩ…Ш§ЩӢ ШҜШұ Ъ©ЩҶШӘШұЩ„ШұЩҮШ§ ШӘШІШұЫҢЩӮ Ъ©ЩҶЫҢШҜ. Ш§ЫҢЩҶ ШұЩҲШҙШҢ Ш§ЩҶШ№Ш·Ш§ЩҒвҖҢЩҫШ°ЫҢШұЫҢ ШЁШ§Щ„Ш§ЫҢЫҢ ШҜШ§ШұШҜ ЩҲ ЩҲШ§ШЁШіШӘЪҜЫҢ ШЁЩҮ Ъ©ШӘШ§ШЁШ®Ш§ЩҶЩҮ Ш®Ш§ШұШ¬ЫҢ ШұШ§ ШӯШ°ЩҒ Щ…ЫҢвҖҢЪ©ЩҶШҜ. ШЁШ§ Ш§ЫҢЩҶ ШӯШ§Щ„ШҢ MediatR ШЁШ§ Ш§ШұШ§ШҰЩҮ ЫҢЪ© ШіШ§Ш®ШӘШ§Шұ ШўЩ…Ш§ШҜЩҮ ЩҲ ЩҫШҙШӘЫҢШЁШ§ЩҶЫҢ Ш§ШІ ШұЩҒШӘШ§ШұЩҮШ§ЫҢ ШіШұШ§ШіШұЫҢШҢ Ш§ЫҢЩҶ ЩҒШұШўЫҢЩҶШҜ ШұШ§ ШіШ§ШҜЩҮвҖҢШӘШұ Щ…ЫҢвҖҢЪ©ЩҶШҜШҢ ШЁЩҮвҖҢЩҲЫҢЪҳЩҮ ШҜШұ ЩҫШұЩҲЪҳЩҮвҖҢЩҮШ§ЫҢЫҢ Ъ©ЩҮ ЩҶЫҢШ§ШІ ШЁЩҮ Щ…ЩӮЫҢШ§ШівҖҢЩҫШ°ЫҢШұЫҢ (Scalability) ШҜШ§ШұЩҶШҜ.

ЪҶЩҮ ШІЩ…Ш§ЩҶЫҢ Ш§ШІ MediatR Ш§ШіШӘЩҒШ§ШҜЩҮ Ъ©ЩҶЫҢЩ…Шҹ
ШӘШөЩ…ЫҢЩ… ШЁЩҮ Ш§ШіШӘЩҒШ§ШҜЩҮ Ш§ШІ MediatR ШЁШ§ЫҢШҜ ШЁШұ Ш§ШіШ§Ші ЩҶЫҢШ§ШІЩҮШ§ЫҢ ЩҫШұЩҲЪҳЩҮ ШЁШ§ШҙШҜШҢ ЩҶЩҮ ШөШұЩҒШ§ЩӢ ШЁЩҮ ШҜЩ„ЫҢЩ„ Щ…ШӯШЁЩҲШЁЫҢШӘ ШўЩҶ. ШҜШұ ЩҫШұЩҲЪҳЩҮвҖҢЩҮШ§ЫҢ Ъ©ЩҲЪҶЪ© ШЁШ§ ШӘШ№ШҜШ§ШҜ Щ…ШӯШҜЩҲШҜЫҢ ЩҶЩӮШ·ЩҮ ЩҲШұЩҲШҜЫҢ (Щ…Ш«Щ„Ш§ЩӢ Ъ©Щ…ШӘШұ Ш§ШІ 20)ШҢ Ш§ШіШӘЩҒШ§ШҜЩҮ Ш§ШІ ШіШұЩҲЫҢШівҖҢЩҮШ§ЫҢ ШіШ§ШҜЩҮ ЫҢШ§ Щ…Ш№Щ…Ш§ШұЫҢ Ш№Щ…ЩҲШҜЫҢ (Vertical Slice Architecture) Щ…ЫҢвҖҢШӘЩҲШ§ЩҶШҜ Ъ©Ш§ЩҒЫҢ ШЁШ§ШҙШҜ. Ш§Щ…Ш§ ШҜШұ ЩҫШұЩҲЪҳЩҮвҖҢЩҮШ§ЫҢ ШЁШІШұЪҜвҖҢШӘШұ Ъ©ЩҮ ШҙШ§Щ…Щ„ ШҜЩҮвҖҢЩҮШ§ ЫҢШ§ ШөШҜЩҮШ§ Ш№Щ…Щ„ЫҢШ§ШӘ ЩҮШіШӘЩҶШҜШҢ MediatR Щ…ЫҢвҖҢШӘЩҲШ§ЩҶШҜ ШЁШ§ Ъ©Ш§ЩҮШҙ ЩҲШ§ШЁШіШӘЪҜЫҢвҖҢЩҮШ§ ЩҲ Ш§ЩҒШІШ§ЫҢШҙ ЩӮШ§ШЁЩ„ЫҢШӘ ЩҶЪҜЩҮШҜШ§ШұЫҢ (Maintainability)ШҢ Ш§ШұШІШҙЩ…ЩҶШҜ ШЁШ§ШҙШҜ.
ЫҢЪ© ЩӮШ§ЩҶЩҲЩҶ Ъ©Щ„ЫҢ ЩҫЫҢШҙЩҶЩҮШ§ШҜЫҢ Ш§ЫҢЩҶ Ш§ШіШӘ: Ш§ЪҜШұ ЩҫШұЩҲЪҳЩҮ ШҙЩ…Ш§ ШЁЫҢШҙ Ш§ШІ 20 ЩҶЩӮШ·ЩҮ ЩҲШұЩҲШҜЫҢ ШҜШ§ШұШҜ ЫҢШ§ ЩҶЫҢШ§ШІ ШЁЩҮ Щ…ШҜЫҢШұЫҢШӘ ЩҫЫҢЪҶЫҢШҜЩҮ ЩҶЪҜШұШ§ЩҶЫҢвҖҢЩҮШ§ЫҢ ШіШұШ§ШіШұЫҢ Щ…Ш§ЩҶЩҶШҜ Щ„Ш§ЪҜвҖҢЪҜШ°Ш§ШұЫҢШҢ Ш§Ш№ШӘШЁШ§ШұШіЩҶШ¬ЫҢ ЫҢШ§ ШұЩҲЫҢШҜШ§ШҜЩҮШ§ЫҢ ШҜШ§Щ…ЩҶЩҮ (Domain Events) ШҜШ§ШұЫҢШҜШҢ MediatR Щ…ЫҢвҖҢШӘЩҲШ§ЩҶШҜ Ш§ЩҶШӘШ®Ш§ШЁ Щ…ЩҶШ§ШіШЁЫҢ ШЁШ§ШҙШҜ. ШҜШұ ШәЫҢШұ Ш§ЫҢЩҶ ШөЩҲШұШӘШҢ ШұЩҲЫҢЪ©ШұШҜЩҮШ§ЫҢ ШіШ§ШҜЩҮвҖҢШӘШұ Щ…Щ…Ъ©ЩҶ Ш§ШіШӘ Ъ©Ш§ШұШўЩ…ШҜШӘШұ ШЁШ§ШҙЩҶШҜ.

Ш¬Ш§ЫҢЪҜШІЫҢЩҶвҖҢЩҮШ§ЫҢ MediatR
Ш§ЪҜШұ ШӘШөЩ…ЫҢЩ… ЪҜШұЩҒШӘЫҢШҜ Ш§ШІ MediatR Ш§ШіШӘЩҒШ§ШҜЩҮ ЩҶЪ©ЩҶЫҢШҜШҢ ЪҜШІЫҢЩҶЩҮвҖҢЩҮШ§ЫҢ ШҜЫҢЪҜШұЫҢ ЩҶЫҢШІ ЩҲШ¬ЩҲШҜ ШҜШ§ШұЩҶШҜ. FastEndpoints ЫҢЪ© Ъ©ШӘШ§ШЁШ®Ш§ЩҶЩҮ Щ…ШӘЩҶвҖҢШЁШ§ШІ Ш§ШіШӘ Ъ©ЩҮ ШұЩҲЫҢЪ©ШұШҜЫҢ Щ…ШҙШ§ШЁЩҮ Minimal APIs Ш§ШұШ§ШҰЩҮ Щ…ЫҢвҖҢШҜЩҮШҜШҢ Ш§Щ…Ш§ ШЁШ§ Ъ©Ш§ШұШ§ЫҢЫҢ ШЁШ§Щ„Ш§ШӘШұ ЩҲ ЩӮШ§ШЁЩ„ЫҢШӘвҖҢЩҮШ§ЫҢ ШЁЫҢШҙШӘШұ Щ…Ш§ЩҶЩҶШҜ Ш§Ш№ШӘШЁШ§ШұШіЩҶШ¬ЫҢ ШҜШ§Ш®Щ„ЫҢ (Built-in Validation). ЩҮЩ…ЪҶЩҶЫҢЩҶШҢ Щ…ЫҢвҖҢШӘЩҲШ§ЩҶЫҢШҜ Ш§Щ„ЪҜЩҲЫҢ ЩҲШ§ШіШ·ЩҮ ШұШ§ Ш®ЩҲШҜШӘШ§ЩҶ ЩҫЫҢШ§ШҜЩҮвҖҢШіШ§ШІЫҢ Ъ©ЩҶЫҢШҜ. Ш§ЫҢЩҶ Ъ©Ш§Шұ ЩҶЩҮ ШӘЩҶЩҮШ§ ШЁЩҮ ШҙЩ…Ш§ Ъ©ЩҶШӘШұЩ„ Ъ©Ш§Щ…Щ„ Щ…ЫҢвҖҢШҜЩҮШҜШҢ ШЁЩ„Ъ©ЩҮ ШҜШұЪ© Ш№Щ…ЫҢЩӮвҖҢШӘШұЫҢ Ш§ШІ Щ…Ш№Щ…Ш§ШұЫҢ ШЁШұЩҶШ§Щ…ЩҮвҖҢШӘШ§ЩҶ ЩҒШұШ§ЩҮЩ… Щ…ЫҢвҖҢЪ©ЩҶШҜ.

ЩҶШӘЫҢШ¬ЩҮвҖҢЪҜЫҢШұЫҢ
MediatR Ш§ШЁШІШ§ШұЫҢ ЩӮШҜШұШӘЩ…ЩҶШҜ ШЁШұШ§ЫҢ ЩҫЫҢШ§ШҜЩҮвҖҢШіШ§ШІЫҢ Ш§Щ„ЪҜЩҲЫҢ ЩҲШ§ШіШ·ЩҮ ЩҲ ЩҫШҙШӘЫҢШЁШ§ЩҶЫҢ Ш§ШІ CQRS ШҜШұ ЩҫШұЩҲЪҳЩҮвҖҢЩҮШ§ЫҢ .NET Ш§ШіШӘШҢ Ш§Щ…Ш§ Ш§ШіШӘЩҒШ§ШҜЩҮ Ш§ШІ ШўЩҶ ШЁШ§ЫҢШҜ ШЁШ§ ШҜЩӮШӘ ЩҲ ШЁШұ Ш§ШіШ§Ші ЩҶЫҢШ§ШІЩҮШ§ЫҢ Ш®Ш§Шө ЩҫШұЩҲЪҳЩҮ Ш§ШұШІЫҢШ§ШЁЫҢ ШҙЩҲШҜ. ШҜШұ ЩҫШұЩҲЪҳЩҮвҖҢЩҮШ§ЫҢ Ъ©ЩҲЪҶЪ© ЩҲ ШіШ§ШҜЩҮШҢ ШұЩҲЫҢЪ©ШұШҜЩҮШ§ЫҢ Щ…ШіШӘЩӮЫҢЩ…вҖҢШӘШұ Щ…ЫҢвҖҢШӘЩҲШ§ЩҶЩҶШҜ ШІЩ…Ш§ЩҶ ШӘЩҲШіШ№ЩҮ ШұШ§ Ъ©Ш§ЩҮШҙ ШҜЩҮЩҶШҜ ЩҲ ЩҫЫҢЪҶЫҢШҜЪҜЫҢ ШәЫҢШұШ¶ШұЩҲШұЫҢ ШұШ§ ШӯШ°ЩҒ Ъ©ЩҶЩҶШҜ. Ш§Щ…Ш§ ШҜШұ ЩҫШұЩҲЪҳЩҮвҖҢЩҮШ§ЫҢ ШЁШІШұЪҜвҖҢШӘШұ ЩҲ ЩҫЫҢЪҶЫҢШҜЩҮШҢ Щ…ШІШ§ЫҢШ§ЫҢ MediatR Щ…Ш§ЩҶЩҶШҜ Ш¬ШҜШ§ШіШ§ШІЫҢШҢ ШӘШіШӘвҖҢЩҫШ°ЫҢШұЫҢ ЩҲ Ш§ЩҶШ№Ш·Ш§ЩҒвҖҢЩҫШ°ЫҢШұЫҢШҢ ШўЩҶ ШұШ§ ШЁЩҮ ЪҜШІЫҢЩҶЩҮвҖҢШ§ЫҢ Ш§ШұШІШҙЩ…ЩҶШҜ ШӘШЁШҜЫҢЩ„ Щ…ЫҢвҖҢЪ©ЩҶШҜ. ШӘЩҲШіШ№ЩҮвҖҢШҜЩҮЩҶШҜЪҜШ§ЩҶ .NET ШЁШ§ЫҢШҜ ШЁШ§ ШҜШұ ЩҶШёШұ ЪҜШұЩҒШӘЩҶ Щ…ЩӮЫҢШ§Ші ЩҫШұЩҲЪҳЩҮШҢ ЩҶЫҢШ§ШІЩҮШ§ЫҢ ШўЫҢЩҶШҜЩҮ ЩҲ ШӘШұШ¬ЫҢШӯШ§ШӘ ШӘЫҢЩ…ШҢ ШЁЫҢЩҶ ШіШ§ШҜЪҜЫҢ ЩҲ ЩӮШ§ШЁЩ„ЫҢШӘвҖҢЩҮШ§ЫҢ ЩҫЫҢШҙШұЩҒШӘЩҮ ШӘШ№Ш§ШҜЩ„ ШЁШұЩӮШұШ§Шұ Ъ©ЩҶЩҶШҜ. ШҜШұ ЩҶЩҮШ§ЫҢШӘШҢ Ш§ЩҶШӘШ®Ш§ШЁ Ш§ШЁШІШ§Шұ Щ…ЩҶШ§ШіШЁШҢ ЩҶЩҮ ШҜЩҶШЁШ§Щ„ Ъ©ШұШҜЩҶ ШұЩҲЩҶШҜЩҮШ§ШҢ Ъ©Щ„ЫҢШҜ Щ…ЩҲЩҒЩӮЫҢШӘ ЫҢЪ© ЩҫШұЩҲЪҳЩҮ Ш§ШіШӘ.


Щ…ШҙШ§ЩҮШҜЩҮ Щ…Ш·Щ„ШЁ Ш§ШөЩ„ЫҢ
Щ…Ш·Ш§Щ„ШЁ Щ…ШҙШ§ШЁЩҮ
    Ш§ШҙШӘШұШ§Ъ©вҖҢЩҮШ§ЩҫЫҢШ§ШҜЩҮ ШіШ§ШІЫҢ Ш§Щ„ЪҜЩҲЫҢ Mediator ШҜШұ ШҜШ§ШӘвҖҢЩҶШӘ: ШЁШұШұШіЫҢ ЩҲ Щ…ЩӮШ§ЫҢШіЩҮ ШҜЩҲ Ш§ШЁШІШ§Шұ ЩӮШҜШұШӘЩ…ЩҶШҜШ§ШҙШӘШұШ§Ъ©вҖҢЩҮШ§ШұШ§ЩҮЩҶЩ…Ш§ЫҢ Ш§ЩҶШӘШ®Ш§ШЁ ШЁЫҢЩҶ MediatR ЩҲ FastEndpoints ШЁШұШ§ЫҢ ЩҫЫҢШ§ШҜЩҮвҖҢШіШ§ШІЫҢ Ш§Щ„ЪҜЩҲЫҢ Ш·ШұШ§ШӯЫҢ CQRSШ§ШҙШӘШұШ§Ъ©вҖҢЩҮШ§ШұЩҲШҙвҖҢЩҮШ§ЫҢ Ш¬Ш§ЫҢЪҜШІЫҢЩҶ Ъ©ШұШҜЩҶ MediatR ШҜШұ ЩҫШұЩҲЪҳЩҮвҖҢЩҮШ§ЫҢ NET.Ш§ШҙШӘШұШ§Ъ©вҖҢЩҮШ§ШЁШ§ШІЩҶЪҜШұЫҢ ШҜШұ Ш§ШіШӘЩҒШ§ШҜЩҮ Ш§ШІ MediatR ШҜШұ ШӘЩҲШіШ№ЩҮ ШЁШұЩҶШ§Щ…ЩҮвҖҢЩҮШ§ЫҢ ASP.NET CoreШ§ШҙШӘШұШ§Ъ©вҖҢЩҮШ§ЩҲЩ„ЩҲШұЫҢЩҶ: Ш¬Ш§ЫҢЪҜШІЫҢЩҶЫҢ ЩӮШҜШұШӘЩ…ЩҶШҜ ШЁШұШ§ЫҢ MediatR ШҜШұ ШӘЩҲШіШ№ЩҮ ЩҶШұЩ…вҖҢШ§ЩҒШІШ§ШұЩҮШ§ЫҢ ШҜШ§ШӘвҖҢЩҶШӘШ§ШҙШӘШұШ§Ъ©вҖҢЩҮШ§Ш¬Ш§ЫҢЪҜШІЫҢЩҶЫҢ MediatR ШЁШ§ System.Threading.Channels Ш§ШҙШӘШұШ§Ъ©вҖҢЩҮШ§ШӘШәЫҢЫҢШұ Щ…ШҜЩ„ ЩҫШұЩҲЪҳЩҮвҖҢЩҮШ§ЫҢ AutoMapper ЩҲ MediatR ШЁЩҮ ШӘШ¬Ш§ШұЫҢЩ…Ш·Ш§Щ„ШЁDomain-Driven Design (DDD)Шӣ ШЁШ®Шҙ ШўШ®Шұ: ЫҢЪ© ЩҫШұЩҲЪҳЩҮЩ…Ш·Ш§Щ„ШЁЩ…ШҜЫҢШұЫҢШӘ Ш§ЫҢЩҲЩҶШӘвҖҢЩҮШ§ ШҜШұ MediatR ЩҶШёШұШ§ШӘ Щ…Ш·Ш§Щ„ШЁШ·ШұШ§ШӯЫҢ ЩҲ ЩҫЫҢШ§ШҜЩҮ ШіШ§ШІЫҢ DomainEvents

ЩҶШёШұШ§ШӘ

  • Ш№Щ„ЫҢ ЫҢЪҜШ§ЩҶЩҮ Щ…ЩӮШҜЩ… ШҜШұ ЫұЫҙЫ°Ыҙ/Ы°Ыұ/ЫұЫ· ЫұЫө:Ы°Ыё
    Ш§Щ„ШЁШӘЩҮ ШЁШ§ЫҢШҜ ШҜШ§ЩҶШіШӘ Ш§ЫҢЩҶ ШЁШІШұЪҜ ШҙШҜЩҶ ЩҫШұЩҲЪҳЩҮ ШұЩҲ ЩҮЩ… ШҙШ§Щ…Щ„ Щ…ЫҢШҙЩҲШҜ ЫҢШ§ ЩҶЩҮ
    ЫҢШ№ЩҶЫҢ Ш§Щ„Ш§ЩҶ ЩҫШұЩҲЪҳЩҮ Ъ©ЩҲЪҶЪ© ЩҮШіШӘ ЩҲЩ„ЫҢ ШЁШ№ШҜШ§ Ш§Щ…Ъ©Ш§ЩҶ ШЁШІШұЪҜ ШҙШҜЩҶ ШўЩҶ ЩҲШ¬ЩҲШҜ ШҜШ§ШұШҜ ЫҢШ§ Ш®ЫҢШұ. ШӘЩҲЫҢ Ш§ЫҢШұШ§ЩҶ ЩҮЩ… ШІЫҢШ§ШҜ Ш§ЫҢЩҶ Ш§ШӘЩҒШ§ЩӮ Щ…ЫҢЩҒШӘЩҮ ЩҲ Ъ©ШіЫҢ ЩҮЩ… ШЁШұЩҶЩ…ЫҢЪҜШұШҜЩҮ ШЁШұШ§ЫҢ Ш§ШөЩ„Ш§Шӯ ЫҢШ§ ШӘШәЫҢЫҢШұШ§ШӘ
    ШҜШұ Ш§Ъ©Ш«Шұ ЩҫШұЩҲЪҳЩҮ ЩҮШ§ Ш§ЫҢЩҶ Ш§ШӘЩҒШ§ЩӮ ШІЫҢШ§ШҜ Щ…ЫҢЩҒШӘШҜ. Щ…ЩҶ Ш®ЩҲШҜЩ… ЫҢЪ© ЩӮШ§Щ„ШЁ ШўЩ…Ш§ШҜЩҮ ШҜШ§ШұЩ… Ъ©ЩҮ ЩҮШұ ЩҫШұЩҲЪҳЩҮ Ш§ЫҢ ШұЩҲ Щ…ШЁШӘЩҶЫҢ ШЁШұ Ш§ЩҲЩҶ Ш¬Щ„ЩҲ Щ…ЫҢШЁШұЩ… ЩҮЩ… ЫҢЪ©ШіШ§ЩҶ ЩҮШіШӘ ЩҲ ЩҮЩ… ШўЩ…Ш§ШҜЩҮ. ЩҒЩӮШ· ШЁЩҮ ЩҶШіШЁШӘ ШўЩҶ ЩҫШұЩҲЪҳЩҮ ШӘШәЫҢЫҢШұШ§ШӘЫҢ Ш§ШЁШӘШҜШ§ЫҢЫҢ ШұЩҲЫҢ ШўЩҶ Щ…ЫҢШҜЩҮЩ….
