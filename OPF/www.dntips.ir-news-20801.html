‫توسعه‌دهندگان Visual Studio با پروژه‌های شبیه SDK، خداحافظی با فایل‌های XML پیچیده!


نویسنده: راویAI
تاریخ: ۱۴۰۴/۱۰/۱۵ ۱۲:۲۶
آدرس: www.dntips.ir
توسعه‌دهنده کالوین، یک SDK جدید به نام CodingWithCalvin.VsixSdk را منتشر کرده است که امکان استفاده از فایل‌های .csproj به سبک SDK را در توسعه افزونه‌های Visual Studio فراهم می‌کند. این SDK با حذف نیاز به XML‌های طولانی و مدیریت دستی فایل‌ها، توسعه را ساده‌تر کرده و امکان استفاده از دستور dotnet build را از خط فرمان فراهم می‌سازد. این SDK شامل ویژگی‌هایی مانند تولید خودکار کلاس VsixInfo از فایل مانیفست، تولید ثابت‌ها برای فایل‌های .vsct، و امکان تعیین نسخه در زمان ساخت است. این ویژگی‌ها فرآیند توسعه و انتشار افزونه‌ها را به طور قابل توجهی بهبود می‌بخشند و به خصوص در محیط‌های CI/CD، مدیریت نسخه‌بندی را آسان‌تر می‌کنند. برای شروع، می‌توانید از قالب dotnet استفاده کنید و با دستورات ساده، یک پروژه جدید ایجاد و آن را بسازید. این SDK با هدف ارائه یک تجربه توسعه مدرن و کارآمد برای توسعه‌دهندگان افزونه Visual Studio طراحی شده است.
