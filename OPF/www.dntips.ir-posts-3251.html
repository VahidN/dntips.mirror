→ вҖ«ШҜШіШӘЩҮ ШЁЩҶШҜЫҢ Ъ©ШұШҜЩҶ Щ„Ш§ЪҜвҖҢЩҮШ§ ШҜШұ Serilog
вҖ«Ш§ЫҢШ¬Ш§ШҜ Ш§ШЁШІШ§ШұЩҮШ§ЫҢ ШіШұШ§ШіШұЫҢ ЩҲЫҢЪҳЩҮ NET Core. ←
Ш№ЩҶЩҲШ§ЩҶ:

вҖ«ШўШІЩ…ЩҲЩҶвҖҢЩҮШ§ЫҢ ЫҢЪ©ЩҫШ§ШұЪҶЪҜЫҢ ШҜШұ ШЁШұЩҶШ§Щ…ЩҮвҖҢЩҮШ§ЫҢ ASP.NET Core


ЩҶЩҲЫҢШіЩҶШҜЩҮ: ЩҲШӯЫҢШҜ ЩҶШөЫҢШұЫҢ
ШӘШ§ШұЫҢШ®: ЫұЫіЫ№Ы№/Ы°Ы·/ЫұЫ¶ ЫұЫ°:ЫұЫө
ШўШҜШұШі: www.dntips.ir
ШӘШ§ Ш§ЫҢЩҶШ¬Ш§ ШҜЩҲ ШұЩҲШҙ ШұШ§ ШЁШұШ§ЫҢ ШўШІЩ…Ш§ЫҢШҙ Ъ©Щ„ЫҢ ЫҢЪ© ШіЫҢШіШӘЩ… Web API ШЁЩҮ ЩҮЩ…ШұШ§ЩҮ ШӘЩ…Ш§Щ… ШІЫҢШұ ШіШ§Ш®ШӘвҖҢЩҮШ§ЫҢ ШўЩҶШҢ ШЁШұШұШіЫҢ Ъ©ШұШҜЫҢЩ…:
- Ш§ШіШӘЩҒШ§ШҜЩҮ Ш§ШІ Postman ШЁШұШ§ЫҢ ШўШІЩ…Ш§ЫҢШҙ ЫҢЪ© ШЁШұЩҶШ§Щ…ЩҮвҖҢЫҢ Web API
- Ш§ШіШӘЩҒШ§ШҜЩҮ Ш§ШІ strest ШЁШұШ§ЫҢ ШўШІЩ…Ш§ЫҢШҙ ЫҢЪ© ШЁШұЩҶШ§Щ…ЩҮвҖҢЫҢ Web API

ШұЩҲШҙ ШіЩҲЩ…ЫҢ ЩҮЩ… ШЁШұШ§ЫҢ Ш§ЩҶШ¬Ш§Щ… Ш§ЫҢЩҶЪ©Ш§Шұ ЩҲШ¬ЩҲШҜ ШҜШ§ШұШҜ Ъ©ЩҮ ШЁЩҮ ШөЩҲШұШӘ ШӘЩҲЪ©Ш§Шұ Ш§ШІ ШІЩ…Ш§ЩҶ Ш§ШұШ§ШҰЩҮвҖҢЫҢ ASP.NET Core 2.1 ШЁЩҮ ЩҮЩ…ШұШ§ЩҮ TestServer ШўШІЩ…Ш§ЫҢШҙЫҢ ШўЩҶ Щ…ЫҢШіШұ ШҙШҜ. Ш§ЫҢЩҶ ШұЩҲШҙ ШҜШұ ЩҶЪҜШ§ШұШҙ 3.1ШҢ ШЁШ§ ШӘШәЫҢЫҢШұ ШұЩҲШҙ ШӘШ№ШұЫҢЩҒ ЩҒШ§ЫҢЩ„ program.csШҢ Ш¬ЩҮШӘ ШіШ§ШІЪҜШ§ШұЫҢ ШўЩҶ ШЁШ§ ШўШІЩ…ЩҲЩҶвҖҢЩҮШ§ЫҢ ЫҢЪ©ЩҫШ§ШұЪҶЪҜЫҢ/ШўШІЩ…Ш§ЫҢШҙ Ъ©Щ„ ШіЫҢШіШӘЩ…ШҢ ШЁЩҮШЁЩҲШҜ ЫҢШ§ЩҒШӘЩҮвҖҢШ§ШіШӘ Ъ©ЩҮ Ш®Щ„Ш§ШөЩҮвҖҢШ§ЫҢ Ш§ШІ ШўЩҶ ШұШ§ ШҜШұ Ш§ЫҢЩҶ Щ…Ш·Щ„ШЁ ШЁШұШұШіЫҢ Щ…ЫҢвҖҢЪ©ЩҶЫҢЩ….


ШўШІЩ…ЩҲЩҶвҖҢЩҮШ§ЫҢ ЫҢЪ©ЩҫШ§ШұЪҶЪҜЫҢ ШҜШұ ASP.NET Core

ШўШІЩ…ЩҲЩҶвҖҢЩҮШ§ЫҢ ЫҢЪ©ЩҫШ§ШұЪҶЪҜЫҢШҢ ШЁШұШ®Щ„Ш§ЩҒ ШўШІЩ…ЩҲЩҶвҖҢЩҮШ§ЫҢ ЩҲШ§ШӯШҜ Ъ©ЩҮ Ш№Щ…ЩҲЩ…Ш§ Ш§ШІ Ш§ШҙЫҢШ§ШЎ ШӘЩӮЩ„ЫҢШҜЫҢ Ш§ШіШӘЩҒШ§ШҜЩҮ Щ…ЫҢвҖҢЪ©ЩҶЩҶШҜШҢ ШҜЩӮЫҢЩӮШ§ ШЁШұ ШұЩҲЫҢ ЩҮЩ…Ш§ЩҶ ШіЫҢШіШӘЩ…ЫҢ Ъ©ЩҮ ЩӮШұШ§Шұ Ш§ШіШӘ ШЁЩҮ Ъ©Ш§ШұШЁШұ ЩҶЩҮШ§ЫҢЫҢ Ш§ШұШ§ШҰЩҮ ШҙЩҲШҜШҢ Ш§Ш¬ШұШ§ Щ…ЫҢвҖҢШҙЩҲЩҶШҜ. ШЁЩҮ ЩҮЩ…ЫҢЩҶ Ш¬ЩҮШӘ ШӘЩҶШёЫҢЩ…Ш§ШӘ Ш§ЩҲЩ„ЫҢЩҮвҖҢЫҢ ШўЩҶвҖҢЩҮШ§ Ъ©Щ…ЫҢ ШЁЫҢШҙШӘШұ Ш§ШіШӘ ЩҲ ЩҮЩ…ЪҶЩҶЫҢЩҶ ШІЩ…Ш§ЩҶ Ш§Ш¬ШұШ§ЫҢ ШўЩҶвҖҢЩҮШ§ ЩҶЫҢШІ ШЁЩҮ Ш№Щ„ШӘ ЩҲШ§ШЁШіШӘЪҜЫҢ ШЁЩҮ ШЁШ§ЩҶЪ© Ш§Ш·Щ„Ш§Ш№Ш§ШӘЫҢ ЩҲШ§ЩӮШ№ЫҢШҢ ЩҒШ§ЫҢЩ„ ШіЫҢШіШӘЩ…ШҢ ШҙШЁЪ©ЩҮ ЩҲ ШәЫҢШұЩҮШҢ ЩҶШіШЁШӘ ШЁЩҮ ШўШІЩ…ЩҲЩҶвҖҢЩҮШ§ЫҢ ЩҲШ§ШӯШҜ ШЁЫҢШҙШӘШұ Ш§ШіШӘ.

ШЁШұШ§ЫҢ Ш§ЫҢШ¬Ш§ШҜ ШўШІЩ…ЩҲЩҶвҖҢЩҮШ§ЫҢ ЫҢЪ©ЩҫШ§ШұЪҶЪҜЫҢ ШҜШұ ШЁШұЩҶШ§Щ…ЩҮвҖҢЩҮШ§ЫҢ ASP.NET CoreШҢ ШӯШҜШ§ЩӮЩ„ ШіЩҮ Щ…ШұШӯЩ„ЩҮ ШЁШ§ЫҢШҜ Ш·ЫҢ ШҙЩҲЩҶШҜ:
Ш§Щ„ЩҒ) Ш§ЫҢШ¬Ш§ШҜ ЫҢЪ© class library Ъ©ЩҮ Ш§ШұШ¬Ш§Ш№ЫҢ ШұШ§ ШЁЩҮ ЩҫШұЩҲЪҳЩҮвҖҢЫҢ Ш§ШөЩ„ЫҢ ШҜШ§ШұШҜ. Ш§ЫҢЩҶ ЩҫШұЩҲЪҳЩҮ ШӯШ§ЩҲЫҢ ШўШІЩ…Ш§ЫҢШҙвҖҢЩҮШ§ЫҢ Щ…Ш§ Ш®ЩҲШ§ЩҮШҜ ШЁЩҲШҜ.
ШЁ) ШұШ§ЩҮ Ш§ЩҶШҜШ§ШІЫҢ ЫҢЪ© ЩҮШ§ШіШӘ ЩҲШЁ ШўШІЩ…Ш§ЫҢШҙЫҢ ШЁШұШ§ЫҢ Ш§ШұШіШ§Щ„ ШҜШұШ®ЩҲШ§ШіШӘвҖҢЩҮШ§ ШЁЩҮ ШўЩҶ ЩҲ ШҜШұЫҢШ§ЩҒШӘ ЩҫШ§ШіШ®вҖҢЩҮШ§ЫҢ ЩҶЩҮШ§ЫҢЫҢ.
Ш¬) Ш§ШіШӘЩҒШ§ШҜЩҮ Ш§ШІ ЫҢЪ© test runner (Ш§ЩҶЩҲШ§Ш№ ЩҲ Ш§ЩӮШіШ§Щ… ЩҒШұЫҢЩ… ЩҲШұЪ©вҖҢЩҮШ§ЫҢ unit testing) ШЁШұШ§ЫҢ Ш§Ш¬ШұШ§ЫҢ ШўШІЩ…Ш§ЫҢШҙвҖҢЩҮШ§


Ш§ЫҢШ¬Ш§ШҜ ЫҢЪ© ЩҫШұЩҲЪҳЩҮвҖҢЫҢ Ъ©ШӘШ§ШЁШ®Ш§ЩҶЩҮ ШЁШұШ§ЫҢ ЩҮШ§ШіШӘ ЩҲ Ш§Ш¬ШұШ§ЫҢ ШўШІЩ…Ш§ЫҢШҙвҖҢЩҮШ§ЫҢ ЫҢЪ©ЩҫШ§ШұЪҶЪҜЫҢ

ЩҒШұШ¶ Ъ©ЩҶЫҢШҜ Щ…ЫҢвҖҢШ®ЩҲШ§ЩҮЫҢЩ… ШЁШұШ§ЫҢ ЩҮЩ…Ш§ЩҶ ЩҫШұЩҲЪҳЩҮвҖҢЫҢ Ш§ЫҢШ¬Ш§ШҜ JWTЩҮШ§ШҢ ШўШІЩ…Ш§ЫҢШҙ ЫҢЪ©ЩҫШ§ШұЪҶЪҜЫҢ ШЁЩҶЩҲЫҢШіЫҢЩ…. ЩҫШі Ш§ШІ Ш§ЫҢШ¬Ш§ШҜ ЫҢЪ© ЩҫШұЩҲЪҳЩҮвҖҢЫҢ Ъ©ШӘШ§ШЁШ®Ш§ЩҶЩҮвҖҢЫҢ Ш¬ШҜЫҢШҜ Ъ©ЩҮ ЩӮШұШ§Шұ Ш§ШіШӘ ЩҮШ§ШіШӘ ШўШІЩ…Ш§ЫҢШҙвҖҢЩҮШ§ЫҢ Щ…Ш§ ШҙЩҲШҜШҢ ЩҶЫҢШ§ШІ Ш§ШіШӘ Щ…ШӯШӘЩҲШ§ЫҢ ЩҒШ§ЫҢЩ„ csproj ШўЩҶвҖҢШұШ§ ШЁЩҮ ШөЩҲШұШӘ ШІЫҢШұ ШӘШәЫҢЫҢШұ ШҜШ§ШҜ:

ШҜШұ Ш§ЫҢЩҶШ¬Ш§ШҢ Ш§ЫҢЩҶ ЩҶЪ©Ш§ШӘ ЩӮШ§ШЁЩ„ Щ…ШҙШ§ЩҮШҜЩҮ ЩҮШіШӘЩҶШҜ:
1) TargetFramework ШўЩҶ ШЁШ§ЫҢШҜ ШЁЩҮ netcoreapp ШӘЩҶШёЫҢЩ… ШҙЩҲШҜ.
2) ШЁШ§ЫҢШҜ Ш§ШұШ¬Ш§Ш№ Щ…ШіШӘЩӮЫҢЩ…ЫҢ ШЁЩҮ Ъ©Щ„ ЩҫШұЩҲЪҳЩҮвҖҢЫҢ ЩҶЩҮШ§ЫҢЫҢ WebApp ШҜШұ ШўЩҶ ЩҲШ¬ЩҲШҜ ШҜШ§ШҙШӘЩҮ ШЁШ§ШҙШҜ. ЪҶЩҲЩҶ ШҜШұ Ш§ШҜШ§Щ…ЩҮ Щ…ЫҢвҖҢШ®ЩҲШ§ЩҮЫҢЩ… ЩҒШ§ЫҢЩ„ Program.cs ШўЩҶвҖҢШұШ§ ШЁШұШ§ЫҢ ШұШ§ЩҮ Ш§ЩҶШҜШ§ШІЫҢ ЫҢЪ© ЩҮШ§ШіШӘ ЩҲШЁ ШўШІЩ…Ш§ЫҢШҙЫҢШҢ ЩҒШұШ§Ш®ЩҲШ§ЩҶЫҢ Ъ©ЩҶЫҢЩ….
3) ШЁШіШӘЩҮвҖҢЫҢ ЩҶЫҢЩҲЪҜШӘЫҢ Ъ©ЩҮ Ъ©Ш§Шұ ШұШ§ЩҮ Ш§ЩҶШҜШ§ШІЫҢ ЩҮШ§ШіШӘ ЩҲШЁ ШўШІЩ…Ш§ЫҢШҙЫҢ ШұШ§ Ш§ЩҶШ¬Ш§Щ… Щ…ЫҢвҖҢШҜЩҮШҜШҢ Microsoft.AspNetCore.Mvc.Testing ЩҶШ§Щ… ШҜШ§ШұШҜ. Ш§ЫҢЩҶ ШЁШіШӘЩҮШҢ Ъ©Ш§Шұ Ъ©ЩҫЫҢ ЩҒШ§ЫҢЩ„вҖҢЩҮШ§ЫҢ ЩҫШұЩҲЪҳЩҮвҖҢЫҢ Ш§ШөЩ„ЫҢ ЩҲ ЩҮЩ…ЪҶЩҶЫҢЩҶ ШӘЩҶШёЫҢЩ… Щ…ШіЫҢШұ ЩҫШұЩҲЪҳЩҮ ШұШ§ ШЁЩҮ Ш§ЫҢЩҶ Щ…ШіЫҢШұ Ш¬ШҜЫҢШҜ ЩҶЫҢШІ Ш§ЩҶШ¬Ш§Щ… Щ…ЫҢвҖҢШҜЩҮШҜ.
4) ШұЩҲШҙ Ш§ЩҒШІЩҲШҜЩҶ ШЁШіШӘЩҮвҖҢЩҮШ§ЫҢ MSTest ШұШ§ Щ…ШҙШ§ЩҮШҜЩҮ Щ…ЫҢвҖҢЪ©ЩҶЫҢШҜ.
5) ЩҮЩ…ЪҶЩҶЫҢЩҶ Ш¬ЩҮШӘ ШіШ§ШҜЩҮвҖҢШӘШұ ШҙШҜЩҶ ШЁШұШұШіЫҢ ЩҶШӘШ§ЫҢШ¬ ШўШІЩ…ЩҲЩҶвҖҢЩҮШ§ЫҢ Ш§ЩҶШ¬Ш§Щ… ШҙШҜЩҮ Щ…ЫҢвҖҢШӘЩҲШ§ЩҶ Ш§ШІ fluentassertions ЩҶЫҢШІ Ш§ШіШӘЩҒШ§ШҜЩҮ Ъ©ШұШҜ.


ШұШ§ЩҮ Ш§ЩҶШҜШ§ШІЫҢ ЩҮШ§ШіШӘ ЩҲШЁ ШўШІЩ…Ш§ЫҢШҙЫҢ Ш¬ЩҮШӘ Ш§ЩҶШ¬Ш§Щ… ШўШІЩ…ЩҲЩҶвҖҢЩҮШ§ЫҢ ЩҲШ§ШӯШҜ

ЩҫШі Ш§ШІ Ш§ЩҶШ¬Ш§Щ… ШӘЩҶШёЫҢЩ…Ш§ШӘ Ш§ШЁШӘШҜШ§ЫҢЫҢ ЩҫШұЩҲЪҳЩҮвҖҢЫҢ ШўШІЩ…ЩҲЩҶ ЫҢЪ©ЩҫШ§ШұЪҶЪҜЫҢШҢ ЩҶЫҢШ§ШІ Ш§ШіШӘ ЫҢЪ© WebApplicationFactory ШіЩҒШ§ШұШҙЫҢ ШұШ§ Ш§ЫҢШ¬Ш§ШҜ Ъ©ШұШҜ:
using ASPNETCore2JwtAuthentication.WebApp;

namespace ASPNETCore2JwtAuthentication.IntegrationTests
{
    public class CustomWebApplicationFactory : WebApplicationFactory<Program>
    {
        protected override IWebHostBuilder CreateWebHostBuilder()
        {
            var builder = base.CreateWebHostBuilder();
            builder.ConfigureLogging(logging =>
            {
                //TODO: ...
            });
            return builder;
        }

        protected override void ConfigureWebHost(IWebHostBuilder builder)
        {
            builder.ConfigureTestServices(services =>
            {
                // Don't run `IHostedService`s when running as a test

            });
        }

}
ШҜШұ Ш§ЫҢЩҶ ШӘШ№ШұЫҢЩҒШҢ Program ШҜШұ <WebApplicationFactory<ProgramШҢ ШҜЩӮЫҢЩӮШ§ ШЁЩҮ ЩҮЩ…Ш§ЩҶ Ъ©Щ„Ш§Ші Program ШЁШұЩҶШ§Щ…ЩҮвҖҢЫҢ Ш§ШөЩ„ЫҢ ЩҲШЁ Ш§ШҙШ§ШұЩҮ Щ…ЫҢвҖҢЪ©ЩҶШҜ. using System;
using System.Threading;
using System.Net.Http;

namespace ASPNETCore2JwtAuthentication.IntegrationTests
{
    public static class TestsHttpClient
    {
        private static readonly Lazy<HttpClient> _serviceProviderBuilder =
                new Lazy<HttpClient>(getHttpClient, LazyThreadSafetyMode.ExecutionAndPublication);

        /// <summary>
        /// A lazy loaded thread-safe singleton
        /// </summary>
        public static HttpClient Instance { get; } = _serviceProviderBuilder.Value;

        private static HttpClient getHttpClient()
        {
            var services = new CustomWebApplicationFactory();
            return services.CreateClient(); //NOTE: This action is very time consuming, so it should be defined as a singleton.
        }
    }
}

using System.Net.Http;
using System.Net.Http.Headers;
using System.Text;
using System.Text.Json;
using System.Threading.Tasks;
using FluentAssertions;
using Microsoft.VisualStudio.TestTools.UnitTesting;

namespace ASPNETCore2JwtAuthentication.IntegrationTests
{
    [TestClass]
    public class JwtTests
    {
        [TestMethod]
        public async Task TestLoginWorks()
        {
            // Arrange
            var client = TestsHttpClient.Instance;

            // Act
            var token = await doLoginAsync(client);

            // Assert
            token.Should().NotBeNull();
            token.AccessToken.Should().NotBeNullOrEmpty();
            token.RefreshToken.Should().NotBeNullOrEmpty();
        }

        [TestMethod]
        public async Task TestCallProtectedApiWorks()
        {
            // Arrange
            var client = TestsHttpClient.Instance;

            // Act
            var token = await doLoginAsync(client);

            // Assert
            token.Should().NotBeNull();
            token.AccessToken.Should().NotBeNullOrEmpty();
            token.RefreshToken.Should().NotBeNullOrEmpty();

            // Act
            const string protectedApiUrl = "/api/MyProtectedApi";
            client.DefaultRequestHeaders.Authorization = new AuthenticationHeaderValue("Bearer", token.AccessToken);
            var response = await client.GetAsync(protectedApiUrl);
            response.EnsureSuccessStatusCode();

            // Assert
            var responseString = await response.Content.ReadAsStringAsync();
            responseString.Should().NotBeNullOrEmpty();
            var options = new JsonSerializerOptions { PropertyNamingPolicy = JsonNamingPolicy.CamelCase };
            var apiResponse = JsonSerializer.Deserialize<MyProtectedApiResponse>(responseString, options);
            apiResponse.Title.Should().NotBeNullOrEmpty();
            apiResponse.Title.Should().Be("Hello from My Protected Controller! [Authorize]");
        }

        private static async Task<Token> doLoginAsync(HttpClient client)
        {
            const string loginUrl = "/api/account/login";
            var user = new { Username = "Vahid", Password = "1234" };
            var response = await client.SendAsync(new HttpRequestMessage(HttpMethod.Post, loginUrl)
            {
                Content = new StringContent(JsonSerializer.Serialize(user), Encoding.UTF8, "application/json")
            });
            response.EnsureSuccessStatusCode();
            var responseString = await response.Content.ReadAsStringAsync();
            responseString.Should().NotBeNullOrEmpty();
            return JsonSerializer.Deserialize<Token>(responseString);
        }
    }
}

ЪҶЩҲЩҶ ШёШ§ЩҮШұ Ш§ЫҢЩҶ ШўШІЩ…ЩҲЩҶвҖҢЩҮШ§ ШЁШ§ ШўШІЩ…ЩҲЩҶвҖҢЩҮШ§ЫҢ ЩҲШ§ШӯШҜ MSTest ЫҢШ§ ЩҮШұ ЩҒШұЫҢЩ… ЩҲШұЪ© Щ…ШҙШ§ШЁЩҮ ШҜЫҢЪҜШұЫҢ ЫҢЪ©ШіШ§ЩҶ Ш§ШіШӘШҢ Щ…ЫҢвҖҢШӘЩҲШ§ЩҶ Ш§ШІ Ш§Щ…Ъ©Ш§ЩҶШ§ШӘ IDEЩҮШ§ ШЁШұШ§ЫҢ Ш§Ш¬ШұШ§ЫҢ ШўЩҶвҖҢЩҮШ§ Ш§ШіШӘЩҒШ§ШҜЩҮ Ъ©ШұШҜ ЩҲ ЫҢШ§ ШӯШӘЫҢ Щ…ЫҢвҖҢШӘЩҲШ§ЩҶ ШҜШіШӘЩҲШұ dotnet test ШұШ§ ЩҶЫҢШІ ШҜШұ ШұЫҢШҙЩҮвҖҢЫҢ Ш§ЫҢЩҶ ЩҫШұЩҲЪҳЩҮвҖҢЫҢ Ш¬ШҜЫҢШҜ ШЁШұШ§ЫҢ Ш§Ш¬ШұШ§ЫҢ ШӘЩ…Ш§Щ… ШўШІЩ…ЩҲЩҶвҖҢЩҮШ§ЫҢ ЩҶЩҲШҙШӘЩҮ ШҙШҜЩҮШҢ Ш§Ш¬ШұШ§ Ъ©ШұШҜ:



Ъ©ШҜЩҮШ§ЫҢ Ъ©Ш§Щ…Щ„ Ш§ЫҢЩҶ Щ…Ш·Щ„ШЁ ШұШ§ ШҜШұ Ш§ЫҢЩҶШ¬Ш§ Щ…ЫҢвҖҢШӘЩҲШ§ЩҶЫҢШҜ Щ…ШҙШ§ЩҮШҜЩҮ Ъ©ЩҶЫҢШҜ.
Щ…Ш·Ш§Щ„ШЁ Щ…ШҙШ§ШЁЩҮ
    Щ…Ш·Ш§Щ„ШЁЩҶЩҲШҙШӘЩҶ ШўШІЩ…ЩҲЩҶвҖҢЩҮШ§ЫҢ ЩҲШ§ШӯШҜ ШЁЩҮ Ъ©Щ…Ъ© Ъ©ШӘШ§ШЁШ®Ш§ЩҶЩҮвҖҢЫҢ Moq - ЩӮШіЩ…ШӘ Ш§ЩҲЩ„ - Щ…Ш№ШұЩҒЫҢЩ…Ш·Ш§Щ„ШЁШ§ШұШіШ§Щ„ Ш®Ш·Ш§ЩҮШ§ЫҢ ШұШ®вҖҢШҜШ§ШҜЩҮвҖҢЫҢ ШҜШұ ШЁШұЩҶШ§Щ…ЩҮвҖҢЩҮШ§ЫҢ ШіЩ…ШӘ Ъ©Щ„Ш§ЫҢЩҶШӘ Blazor WASMШҢ ШЁЩҮ ШӘЩ„ЪҜШұШ§Щ…Щ…Ш·Ш§Щ„ШЁЫҢЪ©ЩҫШ§ШұЪҶЩҮ ШіШ§ШІЫҢ Angular CLI ЩҲ ASP.NET Core ШҜШұ VS 2017Щ…Ш·Ш§Щ„ШЁШ§Щ…ЩҶ ШіШ§ШІЫҢ ШЁШұЩҶШ§Щ…ЩҮвҖҢЩҮШ§ЫҢ ASP.NET Core ШӘЩҲШіШ· IdentityServer 4x - ЩӮШіЩ…ШӘ ЩҮЩҒШӘЩ…- Ш§Щ…ЩҶ ШіШ§ШІЫҢ Web APIЩ…Ш·Ш§Щ„ШЁШ§Щ…Ъ©Ш§ЩҶ ШіШ§Ш®ШӘ ШЁШұЩҶШ§Щ…ЩҮвҖҢЩҮШ§ЫҢ ШҜШіЪ©ШӘШ§Щҫ ЪҶЩҶШҜШіЪ©ЩҲЫҢЫҢ Blazor ШҜШұ ШҜШ§ШӘ ЩҶШӘ 6Щ…Ш·Ш§Щ„ШЁШЁШұШұШіЫҢ ШӘШәЫҢЫҢШұШ§ШӘ Blazor 8x - ЩӮШіЩ…ШӘ ЩҮЩҒШӘЩ… - Ш§Щ…Ъ©Ш§ЩҶ ШӘШ№ШұЫҢЩҒ Ш¬ШІЫҢШұЩҮвҖҢЩҮШ§ЫҢ ШӘШ№Ш§Щ…Щ„ЫҢ Blazor WASMЩ…Ш·Ш§Щ„ШЁBlazor 5x - ЩӮШіЩ…ШӘ 30 - ШЁШұЩҶШ§Щ…ЩҮвҖҢЫҢ Blazor WASM - Ш§ЩҒШІЩҲШҜЩҶ ЩҫШұШҜШ§Ш®ШӘ ШўЩҶЩ„Ш§ЫҢЩҶ ШӘЩҲШіШ· ШҜШұЪҜШ§ЩҮ Щ…Ш¬Ш§ШІЫҢ ЩҫШұШЁШ§ШҜЩ…Ш·Ш§Щ„ШЁШұЩҲШҙ ЫҢЪ©ЫҢ Ъ©ШұШҜЩҶ ЩҫШұЩҲЪҳЩҮвҖҢЩҮШ§ЫҢ React ЩҲ ASP.NET CoreЩҶШёШұШ§ШӘ Щ…Ш·Ш§Щ„ШЁШўШІЩ…ЩҲЩҶвҖҢЩҮШ§ЫҢ ЫҢЪ©ЩҫШ§ШұЪҶЪҜЫҢ ШҜШұ ШЁШұЩҶШ§Щ…ЩҮвҖҢЩҮШ§ЫҢ ASP.NET CoreЩҶШёШұШ§ШӘ Щ…Ш·Ш§Щ„ШЁШ§Ш№ШӘШЁШ§ШұШіЩҶШ¬ЫҢ Щ…ШЁШӘЩҶЫҢ ШЁШұ JWT ШҜШұ ASP.NET Core 2.0 ШЁШҜЩҲЩҶ Ш§ШіШӘЩҒШ§ШҜЩҮ Ш§ШІ ШіЫҢШіШӘЩ… Identity

ЩҶШёШұШ§ШӘ

  • ЩҲШӯЫҢШҜ ЩҶШөЫҢШұЫҢ ШҜШұ ЫұЫҙЫ°Ы°/ЫұЫІ/ЫұЫІ ЫұЫ°:Ыө
    ЫҢЪ© ЩҶЪ©ШӘЩҮвҖҢЫҢ ШӘЪ©Щ…ЫҢЩ„ЫҢ: ШҜШіШӘШұШіЫҢ ШЁЩҮ Ъ©Щ„Ш§Ші Program Ш§ШІ ЩҶЩҲШ№ internal ШҜШ§ШӘ ЩҶШӘ 6 ШҜШұ ЫҢЪ© ЩҫШұЩҲЪҳЩҮвҖҢЫҢ ШҜЫҢЪҜШұ

    ШҜШұ Щ…Ш·Щ„ШЁ ЩҒЩҲЩӮ ЪҶЩҶЫҢЩҶ ЩӮШ·Ш№ЩҮ Ъ©ШҜЫҢ ШұШ§ ШҜШ§ШұЫҢЩ…:
    public class CustomWebApplicationFactory : WebApplicationFactory<Program>
    Ш§ЪҜШұ Ш§ШІ top level programs ШҜШ§ШӘ ЩҶШӘ 6 Ш§ШіШӘЩҒШ§ШҜЩҮ Ъ©ЩҶЫҢЩ…ШҢ Ш§ЫҢЩҶШЁШ§Шұ Ъ©Щ„Ш§Ші Program ШҜЫҢЪҜШұ public ЩҶЫҢШіШӘ ЩҲ internal Ш§ШіШӘ. ШЁЩҮ ЩҮЩ…ЫҢЩҶ Ш¬ЩҮШӘ ЩҶЩ…ЫҢвҖҢШӘЩҲШ§ЩҶ Ш§ШІ Ъ©Щ„Ш§Ші Program ШҜШұ ЫҢЪ© Ш§ШіЩ…ШЁЩ„ЫҢ ШҜЫҢЪҜШұ Ш§ШіШӘЩҒШ§ШҜЩҮ Ъ©ШұШҜШӣ Щ…ЪҜШұ Ш§ЫҢЩҶЪ©ЩҮ ШҜШұ ЩҒШ§ЫҢЩ„ csproj ЩҫШұЩҲЪҳЩҮвҖҢЫҢ Ш§ШөЩ„ЫҢШҢ ШӘЩҶШёЫҢЩ… ШІЫҢШұ ШұШ§ Ш§Ш¶Ш§ЩҒЩҮ Ъ©ЩҶЫҢЩ…. ШҜШұ Ш§ЫҢЩҶШ¬Ш§ Include ШЁЩҮ ЩҶШ§Щ… ЩҫШұЩҲЪҳЩҮвҖҢЫҢ Test (ЩҲ ЩҒШ¶Ш§ЫҢ ЩҶШ§Щ… Ш§ШөЩ„ЫҢ Ш§ШіЩ…ШЁЩ„ЫҢ ШўЩҶ) Ш§ШҙШ§ШұЩҮ Щ…ЫҢвҖҢЪ©ЩҶШҜ. ЩҮЩ…ЪҶЩҶЫҢЩҶ ЪҶЩҲЩҶ Program Ш§ШІ ЩҶЩҲШ№ internal Ш§ШіШӘШҢ Ш§ЫҢЩҶШЁШ§Шұ Ъ©Щ„Ш§Ші ЩҒЪ©ШӘЩҲШұЫҢ ШӘЩҮЫҢЩҮ ШҙШҜЩҮ ЩҶЫҢШІ ШЁШ§ЫҢШҜ internal ШӘШ№ШұЫҢЩҒ ШҙЩҲШҜ:
    <Project Sdk="Microsoft.NET.Sdk">
  <PropertyGroup>
    <TargetFramework>net6.0</TargetFramework>
  </PropertyGroup>

  <ItemGroup>
    <InternalsVisibleTo Include="ProjName.Tests" />
