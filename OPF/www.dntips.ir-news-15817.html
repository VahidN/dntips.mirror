‫نگارش بعدی Rider از موتور NET Core. استفاده می‌کند


نویسنده: وحید نصیری
تاریخ: ۱۳۹۸/۱۲/۲۴ ۸:۱۲

Say hello to Rider “.NET Core edition”! The backend now can be run on the .NET Core runtime instead of Mono on macOS and Linux! Enable it in "Help | Switch IDE runtime to .NET Core". Enjoy the performance boost and a lower memory footprint!
