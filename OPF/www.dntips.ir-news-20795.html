‫توسعه‌دهنده وب سایت «مانیفستو» با بازی‌های ترمینال پنهان در تعطیلات کریسمس ساخت - استفاده از جاوااسکریپت خالص بدون فریم‌ورک


نویسنده: راویAI
تاریخ: ۱۴۰۴/۱۰/۱۵ ۱۲:۰۷
آدرس: www.dntips.ir
یک توسعه‌دهنده وب، در تعطیلات کریسمس، یک وب‌سایت «مانیفستو» برای یک پروژه سخت‌افزاری هوش مصنوعی محلی‌محور ساخت. این وب‌سایت از HTML، CSS و جاوااسکریپت خالص استفاده می‌کند و هیچ مرحله ساخت (build step) ندارد. طراحی وب‌سایت به سبک بروتالیستی با خطوط اسکن، جلوه‌های گلیچ و فونت monospace است. وب‌سایت شامل سه صفحه مقاله طولانی با اجزای بصری سفارشی و همچنین بازی‌های پنهان در یک رابط ترمینال جعلی است. بازی‌های پنهان شامل یک بازی Snake و یک بازی شبیه‌سازی تصرف قلمرو به سبک Volfied است که هر دو با استفاده از Canvas API و با جلوه‌های CRT chunky رندر می‌شوند. توسعه این بازی‌ها با کمک هوش مصنوعی بسیار آسان بوده است. بازی Snake حدود 30 دقیقه و بازی Volfied حدود 2 ساعت طول کشیده است. تکنولوژی‌های مورد استفاده شامل HTML/CSS/JS خالص، CSS custom properties برای تم‌پردازی، Canvas API برای بازی‌ها و Claude برای کمک در کارهای مختلف از جمله SEO، تولید تصاویر OG و تولید سایت‌مپ است. وب‌سایت بر روی یک سرور Nginx ساده مستقر شده است. توسعه‌دهنده که قبلاً CTO و یکی از بنیان‌گذاران یک شرکت داده‌های رمزنگاری بوده است، این پروژه را به عنوان یک تجربه جالب و لذت‌بخش در توسعه وب توصیف می‌کند، زیرا هیچ وابستگی، خطای ساخت یا مشکلات دیگری وجود نداشته است. کد منبع پروژه به طور کامل در GitHub در دسترس است.
