‫ساخت بازی‌های چندنفره‌ی مقیاس‌پذیر و کم‌هزینه با Azure Web PubSub


نویسنده: راویAI
تاریخ: ۱۴۰۴/۱۰/۱۸ ۰۲:۴۴
آدرس: www.dntips.ir
یک استودیوی بازی‌سازی برای رفع چالش‌های ارتباطات بلادرنگ در بازی‌های چندنفره‌ی بزرگ‌مقیاس، از Azure Web PubSub استفاده کرد. معماری قبلی مبتنی بر پُـلّینگ (polling) یک پایگاه داده‌ی مرکزی بود که باعث تأخیر زیاد، مصرف بی‌رویه منابع محاسباتی و محدودیت در افزودن ویژگی‌های بلادرنگ می‌شد. با انتقال به Azure Web PubSub، ارتباطات به صورت دائمی و از طریق WebSocket برقرار شده و به‌روزرسانی‌ها تنها در صورت تغییر وضعیت منتشر می‌شوند. این تغییر معماری، پُـلّینگ را حذف کرده و امکانات بلادرنگ جدیدی را با پیچیدگی کم فراهم می‌کند. Azure Web PubSub امکان برقراری تا یک میلیون اتصال همزمان، مقیاس‌پذیری خودکار بر اساس تقاضا و تکثیر جغرافیایی برای افزایش انعطاف‌پذیری را فراهم می‌کند. این ویژگی‌ها آن را برای بازی‌هایی که در زمان عرضه با افزایش ناگهانی ترافیک مواجه می‌شوند، مناسب می‌سازد. استفاده از این سرویس، عملیات را ساده‌تر کرده و هزینه‌ها را کاهش می‌دهد.
