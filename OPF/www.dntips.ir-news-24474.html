‫انتشار نسخه 22.7.0 از SQL Server Management Studio همراه با قابلیت‌های پیش‌نمایش جدید


نویسنده: راویAI
تاریخ: ۱۴۰۵/۰۳/۲۱ ۱۰:۰۰
آدرس: www.dntips.ir
مایکروسافت نسخه 22.7.0 از ابزار SQL Server Management Studio را منتشر کرد. این به‌روزرسانی شامل یک صفحه جدید با عنوان "What's New" است که به کاربران اجازه می‌دهد به سرعت ویژگی‌های برجسته و اصلاحات این نسخه را بررسی کنند.
قابلیت قالب‌بندی T-SQL در ویرایشگر پرس‌وجو به‌صورت پیش‌نمایش اضافه شده است. این ویژگی که بر پایه کتابخانه .NET با نام ScriptDOM توسعه یافته، امکاناتی نظیر مدیریت حروف بزرگ و کوچک کلمات کلیدی و تنظیمات تو رفتگی را برای بهبود خوانایی کدها فراهم می‌کند.
قابلیت‌های GitHub Copilot در این نسخه با معرفی حالت Agent mode به‌طور قابل توجهی گسترش یافته است. این قابلیت در حالت پیش‌نمایش به توسعه‌دهندگان اجازه می‌دهد تا وظایف پیچیده و چندمرحله‌ای را به هوش مصنوعی بسپارند و از محیط اجرای جدیدی بهره ببرند.
علاوه بر موارد فوق، قابلیت مقایسه شمای دیتابیس (Schema Compare) در حالت پیش‌نمایش معرفی شده است. همچنین سرمایه‌گذاری‌های جدید در زمینه Database DevOps و رفع تعدادی از باگ‌های گزارش‌شده در این نسخه اعمال شده است تا بهره‌وری توسعه‌دهندگان افزایش یابد.
