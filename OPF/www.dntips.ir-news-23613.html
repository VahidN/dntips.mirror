‫افزونه File Change Follower برای VS Code منتشر شد


نویسنده: راویAI
تاریخ: ۱۴۰۴/۱۱/۱۶ ۱۰:۴۱
آدرس: www.dntips.ir
یک افزونه جدید برای Visual Studio Code به نام File Change Follower منتشر شده است که تغییرات فایل‌ها را در زمان واقعی دنبال می‌کند. این افزونه به طور خودکار ویرایشگرها را باز کرده و به خطوط ویرایش شده اسکرول می‌کند، که برای نظارت بر عملکرد ابزارهای کدنویسی مبتنی بر خط فرمان (CLI) بسیار مفید است. این افزونه دارای ویژگی‌هایی مانند دنبال کردن فایل‌ها در زمان واقعی، برجسته‌سازی تغییرات، یکپارچه‌سازی با نوار وضعیت، دسترسی سریع از طریق پالت دستورات، الگوهای قابل تنظیم برای گنجاندن یا حذف فایل‌ها، و به‌روزرسانی‌های debounce برای جلوگیری از بارگذاری بیش از حد ویرایشگر است. این ابزار برای مواردی مانند نظارت بر ابزارهای کدنویسی هوش مصنوعی، برنامه‌نویسی جفتی، نظارت بر فایل‌های تولید شده توسط ابزارهای ساخت، و مشاهده به‌روزرسانی فایل‌های لاگ بسیار کاربردی است. نصب این افزونه از طریق VS Code Marketplace یا از طریق فایل VSIX امکان‌پذیر است. برای استفاده از آن، می‌توانید از طریق نوار وضعیت یا پالت دستورات، حالت دنبال کردن را فعال یا غیرفعال کنید. افزونه همچنین تنظیماتی را برای فعال‌سازی خودکار در هنگام راه‌اندازی، تعیین الگوهای گنجاندن و حذف فایل‌ها، تنظیم فاصله debounce و مدت زمان برجسته‌سازی تغییرات ارائه می‌دهد. نحوه عملکرد افزونه به این صورت است که با استفاده از file watcher داخلی VS Code، تغییرات فایل‌ها را نظارت می‌کند. هنگامی که یک فایل باز ویرایش می‌شود، افزونه محدوده تغییرات را ثبت کرده و پس از debounce کردن، فایل مربوطه را باز کرده و به خطوط تغییر یافته اسکرول می‌کند. خطوط تغییر یافته نیز به طور موقت با استفاده از رنگ‌های diff editor برجسته می‌شوند. این افزونه به VS Code 1.85.0 یا بالاتر نیاز دارد و ممکن است در فایل‌های بسیار بزرگ، تأخیر جزئی در هنگام اسکرول به تغییرات وجود داشته باشد.
