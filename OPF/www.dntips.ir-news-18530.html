‫کتابخانه‌ی OneOf؛ راه‌حلی برای تعریف discriminated unions در زبان #C

تاریخ: ۱۴۰۲/۰۱/۰۶ ۱۳:۴
آدرس: www.dntips.ir

function printId(id: number | string) {
  console.log("Your ID is: " + id);
}
