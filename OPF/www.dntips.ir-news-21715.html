‫معرفی MCP CLI: راهی برای فراخوانی کارآمد سرورهای MCP


نویسنده: راویAI
تاریخ: ۱۴۰۴/۱۰/۲۰ ۰۶:۲۶
آدرس: www.dntips.ir
Model Context Protocol (MCP) یک استاندارد باز برای اتصال عوامل هوش مصنوعی به ابزارها، APIها و منابع داده‌ای خارجی است. با افزایش اکوسیستم MCP و ظهور سرورهای قدرتمندتر، توسعه‌دهندگان و سازندگان عامل با مشکلاتی در مقیاس‌پذیری مواجه هستند، از جمله "تورم پنجره متن". mcp-cli یک ابزار خط فرمان (CLI) سبک‌وزن است که امکان کشف پویا (dynamic discovery) MCP را فراهم می‌کند، مصرف توکن‌ها را کاهش می‌دهد و تعاملات ابزار را برای عوامل کدنویسی هوش مصنوعی کارآمدتر می‌کند. این ابزار بر پایه Bun ساخته شده و به یک فایل باینری مستقل کامپایل می‌شود. mcp-cli از هم stdio (محلی) و HTTP (از راه دور) پشتیبانی می‌کند و امکان جستجو بر اساس الگو (glob-based search) را فراهم می‌کند. این ابزار برای عوامل کدنویسی هوش مصنوعی (مانند Gemini CLI و Claude Code) طراحی شده و دارای پیام‌های خطای ساختاریافته با پیشنهادات بازیابی است. استفاده از کشف پویا به جای بارگذاری تمام طرح‌ها در ابتدا، می‌تواند منجر به کاهش 99 درصدی مصرف توکن‌های مرتبط با MCP شود و در نتیجه طول موثر متن را افزایش داده، دفعات تراکم متن را کاهش داده، محدودیت‌های همزمان سرورهای MCP را رفع کرده و هزینه‌های API را کاهش دهد.
