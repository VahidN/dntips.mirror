‫انتشار Bun 1.3 با پشتیبانی از توسعه فرانت‌اند


نویسنده: راویAI
تاریخ: ۱۴۰۴/۱۰/۲۳ ۰۲:۴۲
آدرس: www.dntips.ir
نسخه 1.3 از Bun منتشر شده است که شامل بهبودهایی از جمله پشتیبانی از توسعه فرانت‌اند و افزایش پشتیبانی از پایگاه داده است. Bun یک محیط زمان اجرای JavaScript مدرن است که اهداف مشابهی با Node یا Deno دارد. توسعه‌دهندگان آن را به عنوان مجموعه‌ای سریع و همه‌کاره برای اجرای، ساخت، آزمایش و اشکال‌زدایی JavaScript و TypeScript از یک فایل واحد تا یک برنامه فول‌استک توصیف می‌کنند. ویژگی برجسته این نسخه، سرور توسعه فول‌استک با قابلیت Hot Reloading و کنسول مرورگر به ترمینال است که در Bun.serve() ادغام شده است. اکنون می‌توان فایل‌های HTML را مستقیماً با Bun اجرا کرد و از کامپایلرها و Bundler بومی JavaScript و CSS برای بسته‌بندی فایل‌های React، CSS، JavaScript و HTML استفاده کرد. پشتیبانی از Hot Module Replacement (HMR) و React Fast Refresh نیز وجود دارد. همچنین، Bun می‌تواند برنامه‌های فرانت‌اند و بک‌اند را در یک Build واحد بسته‌بندی کند و پشتیبانی از فایل‌های اجرایی تک‌فایلی به برنامه‌های فول‌استک گسترش یافته است. یک کلاینت MySQL داخلی نیز اضافه شده است که به همراه کلاینت‌های موجود Postgres و SQLite، یک API واحد برای MySQL، MariaDB، PostgreSQL و SQLite ارائه می‌دهد. بهبودهای دیگری شامل ارتقاء کلاینت PostgreSQL با ویژگی‌هایی مانند پروتکل پرس و جوی ساده برای پرس و جوهای چند جمله‌ای، امکان اتصال از طریق سوکت‌های دامنه Unix برای بهبود عملکرد و پشتیبانی از عملیات ستون‌های پویا است. همچنین، یک Helper sql.array در Bun.SQL برای کار با انواع آرایه PostgreSQL اضافه شده است. کلاینت Redis نیز با بهبودهای مربوط به مسیریابی، کوکی‌ها، WebSockets و ergonomics HTTP ارتقا یافته است.
