‫راه‌اندازی Claude Code با مدل‌های Microsoft Foundry در macOS


نویسنده: راویAI
تاریخ: ۱۴۰۴/۱۱/۲۲ ۰۹:۳۲
آدرس: www.dntips.ir
این مقاله راهنمایی برای راه‌اندازی Claude Code به صورت محلی در macOS با استفاده از مدل‌های مستقر شده در Microsoft Foundry ارائه می‌دهد. هدف از این راهنما، پل زدن شکاف بین محیط توسعه محلی و زیرساخت ابری سازمانی Foundry است. این روش به توسعه‌دهندگان امکان می‌دهد تا از مزایای پایداری، انطباق و مدیریت ساختاریافته Microsoft بهره‌مند شوند، در حالی که همچنان در محیط توسعه محلی خود کار می‌کنند. برای راه‌اندازی، به یک اشتراک Azure با محیط Microsoft Foundry، یک حساب کاربری Anthropic و Claude Code Client روی macOS نیاز دارید. مهم‌ترین بخش، داشتن اشتراک Azure فعال است که در آن محیط Foundry مقداردهی اولیه شده باشد و مدل Claude مورد نظر شما مستقر و فعال باشد. همچنین، باید URL نقطه پایانی و کلیدهای API مربوطه را از این استقرار به دست آورید. نصب Claude Code روی macOS از طریق Homebrew یا Curl توصیه می‌شود. پس از نصب، باید به پورتال Microsoft Foundry بروید، مدل Claude را مستقر کنید و URLهای نقطه پایانی و کلیدها را برای پیکربندی Claude Code جمع‌آوری کنید. سپس، باید متغیرهای محیطی را در ترمینال macOS تنظیم کنید تا درخواست‌ها از طریق Microsoft Foundry هدایت شوند. این کار با افزودن خطوطی به فایل .zshrc انجام می‌شود. در نهایت، این راهنما به شما نشان می‌دهد که چگونه با تنظیم متغیرهای محیطی مناسب، Claude Code را به گونه‌ای پیکربندی کنید که از مدل‌های مستقر شده در Microsoft Foundry استفاده کند و از مزایای زیرساخت ابری Microsoft بهره‌مند شوید. این روش به ویژه برای کسانی که قصد انتقال از یک نمونه اولیه به یک راه حل واقعی را دارند، مفید است.
