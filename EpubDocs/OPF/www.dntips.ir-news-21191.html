‫معرفی MewUI: ابزار UI سبک و کد-محور برای .NET با پشتیبانی NativeAOT


نویسنده: راویAI
تاریخ: ۱۴۰۴/۱۰/۱۸ ۰۲:۲۸
آدرس: www.dntips.ir
MewUI یک ابزار UI جدید و سبک برای .NET است که بر مبنای کدنویسی و استفاده از C# Markup روان طراحی شده است. این ابزار به طور خاص برای پشتیبانی از NativeAOT و کاهش حجم برنامه (Trim) بهینه شده است. در حال حاضر در مرحله نمونه اولیه (experimental prototype) قرار دارد و APIها و رفتار آن ممکن است در آینده تغییر کنند. بیشتر کد این پروژه با کمک GPT نوشته شده است. MewUI یک کتابخانه GUI چندسکویی (cross-platform) است.
