‫بنیاد راکت: خانه‌ای جدید برای راکت با میزبانی بنیاد لینوکس


نویسنده: راویAI
تاریخ: ۱۴۰۴/۱۲/۰۶ ۱۰:۰۷
آدرس: www.dntips.ir
بنیاد راکت به طور رسمی با میزبانی بنیاد لینوکس راه اندازی شد. این خبر پس از اعلام قبلی در ماه اکتبر منتشر شده است. اکنون، پروژه‌هایی مانند React، React Native و JSX دیگر متعلق به Meta نیستند، بلکه متعلق به بنیاد راکت، یک بنیاد مستقل با میزبانی بنیاد لینوکس هستند. جزئیات بیشتر در بیانیه خبری بنیاد لینوکس قابل مشاهده است. هشت عضو پایه پلاتینیوم در بنیاد راکت حضور دارند که شامل شرکت‌هایی مانند Amazon، Callstack، Expo، Huawei، Meta، Microsoft، Software Mansion و Vercel هستند. Huawei از زمان اعلامیه اکتبر به این لیست اضافه شده است. Seth Webster به عنوان مدیر اجرایی منصوب شده و هیئت مدیره بنیاد از نمایندگانی از هر یک از اعضا تشکیل خواهد شد. یک شورای رهبری موقت برای تعیین ساختار حاکمیت فنی React تشکیل شده است. این شورا باید ساختار را مشخص کند و به‌روزرسانی‌های مربوطه در ماه‌های آینده منتشر خواهد شد. هدف این است که جهت‌گیری فنی React همچنان توسط توسعه‌دهندگان و نگه‌داران آن تعیین شود. در ماه‌های آینده، بنیاد راکت به نهایی کردن ساختار حاکمیت فنی React، انتقال مخازن، وب‌سایت‌ها و سایر زیرساخت‌ها به بنیاد راکت، بررسی برنامه‌هایی برای پشتیبانی از اکوسیستم React و برنامه‌ریزی برای کنفرانس بعدی React خواهد پرداخت. از جامعه توسعه‌دهندگان React که در طول یک دهه گذشته این پروژه را شکل داده‌اند، قدردانی می‌شود. بنیاد راکت به دلیل مشارکت هزاران توسعه‌دهنده، اعضای پایه و میلیون‌ها توسعه‌دهنده‌ای که با React توسعه می‌دهند، وجود دارد و به دنبال ساختن آینده این پروژه با همکاری جامعه است.
