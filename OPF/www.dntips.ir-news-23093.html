‫معرفی دستور CLI جدید vercel api برای دسترسی مستقیم به APIهای Vercel


نویسنده: راویAI
تاریخ: ۱۴۰۴/۱۱/۰۹ ۰۲:۱۴
آدرس: www.dntips.ir
Vercel یک دستور CLI جدید به نام api معرفی کرده است که امکان دسترسی مستقیم به مجموعه کامل APIهای Vercel را از طریق ترمینال فراهم می‌کند. این دستور به طور خاص برای تسهیل تعامل عوامل هوش مصنوعی (AI agents) با Vercel طراحی شده است. این ابزار جدید به عوامل هوش مصنوعی مانند Claude Code اجازه می‌دهد تا بدون نیاز به پیکربندی اضافی، مستقیماً با Vercel ارتباط برقرار کنند. دسترسی عامل به Vercel به طور خودکار با مجوزهای کاربر مطابقت دارد، به شرطی که عامل به محیط و CLI Vercel دسترسی داشته باشد. کاربران می‌توانند با استفاده از دستورات `vercel api ls` لیست APIهای موجود را مشاهده کنند، درخواست‌ها را به صورت تعاملی با `vercel api` ایجاد کنند، یا درخواست‌ها را مستقیماً با `vercel api [endpoint] [options]` ارسال کنند. برای شروع، می‌توانید از دستور `npx vercel@latest api --help` استفاده کنید. این ابزار جدید یک گام مهم در جهت تسهیل توسعه و استقرار برنامه‌های مبتنی بر هوش مصنوعی با استفاده از Vercel است. با ارائه دسترسی مستقیم به APIها، Vercel به توسعه‌دهندگان این امکان را می‌دهد تا فرآیندهای خود را خودکار کرده و برنامه‌های خود را با سرعت بیشتری مستقر کنند.
