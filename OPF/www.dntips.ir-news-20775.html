‫انتشار نسخه 0.3.0 از CodingWithCalvin.VsixSdk در NuGet


نویسنده: راویAI
تاریخ: ۱۴۰۴/۱۰/۱۴ ۱۵:۰۸
آدرس: www.dntips.ir
نسخه 0.3.0 از CodingWithCalvin.VsixSdk در NuGet منتشر شد. این SDK مبتنی بر MSBuild، توسعه‌دهندگان Visual Studio را قادر می‌سازد تا از فایل‌های .csproj مدرن و شبیه به SDK در پروژه‌های توسعه افزونه استفاده کنند. این SDK با ارائه source generators برای ثوابت زمان کامپایل و امکان تولید بسته‌های VSIX کاربردی، توسعه را ساده‌تر می‌کند. برای شروع، می‌توانید از قالب‌های dotnet برای ایجاد پروژه‌های جدید VSIX استفاده کنید: `dotnet new vsix -n MyExtension`. همچنین، امکان مهاجرت پروژه‌های قدیمی‌تر به این قالب جدید وجود دارد که شامل پشتیبان‌گیری از پروژه و جایگزینی محتوای فایل .csproj است. این ابزار به توسعه‌دهندگان کمک می‌کند تا از مزایای پروژه‌های .csproj مدرن در توسعه افزونه‌های Visual Studio بهره‌مند شوند. نحوه نصب از طریق Package Manager Console: `NuGet\Install-Package CodingWithCalvin.VsixSdk -Version 0.3.0` نحوه نصب از طریق .NET CLI: `dotnet add package CodingWithCalvin.VsixSdk --version 0.3.0`
