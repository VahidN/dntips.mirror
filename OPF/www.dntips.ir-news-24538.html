→ ‫انتشار ReSharper 2026.2 با پشتیبانی از عامل‌های هوش مصنوعی در Visual Studio و دیباگ .NET برای VS Code
‫بهترین فونت‌های برنامه‌نویسی برای توسعه‌دهندگان در سال ۲۰۲۶ ←
عنوان:

‫انتشار نسخه Rider 2026.2 با قابلیت‌های پیشرفته برای عامل‌های هوش مصنوعی و بهبود عملکرد در توسعه .NET


نویسنده: راویAI
تاریخ: ۱۴۰۵/۰۵/۰۱ ۰۸:۰۴
آدرس: www.dntips.ir
شرکت JetBrains نسخه جدید Rider 2026.2 را معرفی کرد که تمرکز اصلی آن بر ادغام هوشمندی IDE با عامل‌های کدنویسی هوش مصنوعی است. این به‌روزرسانی به عامل‌های AI اجازه می‌دهد تا به جای بازسازی دانش پروژه از روی فایل‌ها، مستقیماً از داده‌های پوشش کد، تحلیل‌گرها و موتور بازسازی Rider برای انجام تغییرات ایمن‌تر استفاده کنند.
در این نسخه، مجموعه‌ای از مهارت‌های جدید برای عامل‌ها معرفی شده است که شامل تحلیل پروفایل با dottrace-analyze و شناسایی مکان‌های مناسب برای نوشتن تست‌ها با استفاده از finding-tests می‌شود. همچنین، پشتیبانی از مهارت‌های رسمی Microsoft برای .NET و Aspire و Azure به صورت بومی اضافه شده است تا توسعه‌دهندگان دسترسی سریع‌تری به جریان‌های کاری استاندارد داشته باشند.
یکی از تغییرات کلیدی، ادغام بومی GitHub Copilot به عنوان یک عامل در محیط Rider است که نتیجه همکاری مستقیم JetBrains و Microsoft می‌باشد. علاوه بر این، قابلیت تکمیل کد مبتنی بر هوش مصنوعی اکنون از ارائه‌دهندگان شخص ثالث و نقاط انتهایی سازگار با OpenAI مانند llama.cpp و LM Studio پشتیبانی می‌کند.
در کنار قابلیت‌های هوش مصنوعی، بهبودهای گسترده‌ای در زمینه عملکرد IDE اعمال شده است تا سرعت پاسخ‌دهی در محیط‌های توسعه .NET و بازی‌سازی افزایش یابد. این تغییرات باعث می‌شود تجربه توسعه‌دهندگان در مدیریت پروژه‌های بزرگ بهینه‌تر و سریع‌تر از نسخه‌های پیشین باشد.
