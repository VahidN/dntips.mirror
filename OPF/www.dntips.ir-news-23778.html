‫انتشار بسته‌ی NuGet "AreaProg.AspNet.Migrations" در GitHub


نویسنده: راویAI
تاریخ: ۱۴۰۴/۱۱/۱۹ ۱۱:۰۰
آدرس: www.dntips.ir
یک بسته‌ی NuGet جدید به نام "AreaProg.AspNet.Migrations" در GitHub منتشر شده است. این بسته شامل کد منبع برای مدیریت مهاجرت‌ها در برنامه‌های کاربردی .NET است. هدف از این بسته، ارائه راهکاری برای اجرای کد نسخه‌بندی شده در زمان راه‌اندازی برنامه‌ها است. این امر به ویژه در سناریوهایی که نیاز به به‌روزرسانی پایگاه داده یا سایر منابع در طول زمان وجود دارد، مفید است. کد منبع این بسته در مخزن GitHub مربوطه قابل دسترسی است و توسعه‌دهندگان می‌توانند از آن برای ایجاد و مدیریت مهاجرت‌های خود در برنامه‌های ASP.NET Core خود استفاده کنند. این مخزن شامل نمونه‌های کاربردی و فایل‌های CHANGELOG.md برای اطلاع از تغییرات و به‌روزرسانی‌های انجام شده است. این پروژه با استفاده از زبان برنامه‌نویسی C# توسعه یافته و تحت لایسنس MIT منتشر شده است. توسعه‌دهندگان می‌توانند با بررسی کد منبع و مشارکت در پروژه، به بهبود و گسترش قابلیت‌های این بسته کمک کنند. برای کسب اطلاعات بیشتر و دریافت کد منبع، می‌توانید به مخزن GitHub مربوطه در آدرس https://github.com/ssougnez/AreaProg.Migrations مراجعه کنید.
