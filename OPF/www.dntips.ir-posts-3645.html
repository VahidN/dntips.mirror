Zync ЫҢЪ© Щ…ШҜЫҢШұ ШЁШіШӘЩҮЩ” ШіШЁЪ© ЩҲ Щ…ШЁШӘЩҶЫҢ ШЁШұ Git ШЁШұШ§ЫҢ SQL Server Ш§ШіШӘ Ъ©ЩҮ Щ…Ш¬Щ…ЩҲШ№ЩҮвҖҢШ§ЫҢ Ш§ШІ Ш§ШіЪ©ШұЫҢЩҫШӘвҖҢЩҮШ§ЫҢ SQL ШұШ§ ШЁЩҮ ШЁШіШӘЩҮвҖҢЩҮШ§ЫҢ ЩӮШ§ШЁЩ„ ЩҶШөШЁ ЩҲ ЩҲШ§ШЁШіШӘЪҜЫҢвҖҢЩ…ШӯЩҲШұ ШӘШЁШҜЫҢЩ„ Щ…ЫҢвҖҢЪ©ЩҶШҜ. ШЁШ§ Zync Щ…ЫҢвҖҢШӘЩҲШ§ЩҶЫҢШҜ Ш§ШЁШІШ§ШұЩҮШ§ЫҢ ЩҫШ§ЫҢЪҜШ§ЩҮвҖҢШҜШ§ШҜЩҮЩ” ЩӮШ§ШЁЩ„вҖҢШ§Ш№ШӘЩ…Ш§ШҜШҢ ШӘЪ©ШұШ§ШұЩҫШ°ЫҢШұ ЩҲ ЩӮШ§ШЁЩ„ Ш§ШҙШӘШұШ§Ъ© ШұШ§ ШӘЩҶЩҮШ§ ШЁШ§ ЪҶЩҶШҜ ШҜШіШӘЩҲШұ ШіШ§ШҜЩҮ ШЁЩҮ ШҜШіШӘ ШўЩҲШұЫҢШҜ.
Ш§ЫҢЩҶШ¬Ш§ Щ…ЫҢШӘЩҲЩҶЫҢШҜ ШЁЫҢШҙШӘШұ ШҜШұ Щ…ЩҲШұШҜ Zync Щ…Ш·Ш§Щ„Ш№ЩҮ Ъ©ЩҶЫҢШҜ : https://www.dntips.ir/post/3635
ШҜШұ Ш§ЫҢЩҶ Щ…ЩӮШ§Щ„ЩҮ ШЁЩҮ ШЁШіШӘЩҮЩ” Geographic Щ…ЫҢвҖҢЩҫШұШҜШ§ШІЫҢЩ… ЩҲ ШӘЩҲШ¶ЫҢШӯ Щ…ЫҢвҖҢШҜЩҮЫҢЩ… Ъ©ЩҮ ЪҶШұШ§ ЩҲ ЪҶЪҜЩҲЩҶЩҮ Ш№Щ…ШҜШ§ЩӢ ШЁШұ Ш§ШіШ§Ші Ш§Щ„ЪҜЩҲЩҮШ§ ЩҲ ЩӮШұШ§ШұШҜШ§ШҜЩҮШ§ЫҢ PostGIS ШҜШұ PostgreSQL Ш·ШұШ§ШӯЫҢ ШҙШҜЩҮ Ш§ШіШӘ ШӘШ§ ШӘЩҲШіШ№ЩҮвҖҢШҜЩҮЩҶШҜЪҜШ§ЩҶ Щ…Ъ©Ш§ЩҶЫҢ ЩҮЩҶЪҜШ§Щ… Ъ©Ш§Шұ ШЁШ§ SQL Server Ш§ШӯШіШ§Ші ШўШҙЩҶШ§ЫҢЫҢ Ъ©ЩҶЩҶШҜ.

ЪҶШұШ§ Ш§Щ„ЪҜЩҲЫҢ PostGISШҹ
PostGIS Ш§ШіШӘШ§ЩҶШҜШ§ШұШҜ ШҙЩҶШ§Ш®ШӘЩҮвҖҢШҙШҜЩҮЩ” ЩҫШұШҜШ§ШІШҙ ШҜШ§ШҜЩҮвҖҢЩҮШ§ЫҢ Щ…Ъ©Ш§ЩҶЫҢ ШҜШұ PostgreSQL Ш§ШіШӘ. Ш·ШұШ§ШӯЫҢ API ШӯШіШ§ШЁвҖҢШҙШҜЩҮШҢ ЩҶШ§Щ…вҖҢЪҜШ°Ш§ШұЫҢ ЫҢЪ©ШҜШіШӘ (ST_*) ЩҲ Щ…Ш¬Щ…ЩҲШ№ЩҮЩ” ШӘЩҲШ§ШЁШ№ ШәЩҶЫҢЩҗ PostGIS ШЁШ§Ш№Ш« ШҙШҜЩҮ ШЁШіЫҢШ§ШұЫҢ Ш§ШІ Щ…ЩҮЩҶШҜШіШ§ЩҶ ШЁШ§ ЩҮЩ…Ш§ЩҶ Ш°ЩҮЩҶЫҢШӘ ШіШұШ§Шә Ъ©Ш§ШұЩҮШ§ЫҢ ЪҳШҰЩҲШ§ШіЩҫЫҢШҙЫҢШ§Щ„ ШЁШұЩҲЩҶШҜ.
ШЁШіШӘЩҮЩ” Geographic ШҜШұ Zync ЩҮЩ…Ш§ЩҶ Щ…ШҜЩ„ Ш°ЩҮЩҶЫҢ ЩҲ ЩӮШұШ§ШұШҜШ§ШҜЩҮШ§ЫҢ ЩҶШ§Щ…вҖҢЪҜШ°Ш§ШұЫҢ ШұШ§ Ш§ШӘШ®Ш§Ш° Ъ©ШұШҜЩҮ ШӘШ§:
  • ЩҶШ§Щ… ЩҲ Щ…Ш№ЩҶШ§ШҙЩҶШ§ШіЫҢ ШӘЩҲШ§ШЁШ№ ШЁШұШ§ЫҢ Ъ©Ш§ШұШЁШұШ§ЩҶ PostGIS ШўШҙЩҶШ§ ШЁШ§ШҙШҜ.
  • Ъ©Ш§ШұЩҮШ§ЫҢ ШұШ§ЫҢШ¬ (ШіШ§Ш®ШӘ ЩҮЩҶШҜШіЩҮШҢ ШұЩҲШ§ШЁШ· Щ…Ъ©Ш§ЩҶЫҢШҢ ШЁШ§ЩҒШұШҢ ШіЩҶШ¬Шҙ ЩҒШ§ШөЩ„ЩҮ) ШёШ§ЩҮШұ ЩҲ Ш§ШӯШіШ§ШіЫҢ Щ…ШҙШ§ШЁЩҮ ШҜШ§ШҙШӘЩҮ ШЁШ§ШҙШҜ.
  • Щ…ЩҮШ§Ш¬ШұШӘ ШЁЫҢЩҶ PostgreSQL/PostGIS ЩҲ SQL Server ШЁШұШ§ЫҢ ШӘЫҢЩ…вҖҢЩҮШ§ЫҢ ЪҶЩҶШҜШіЪ©ЩҲЫҢЫҢ ШўШіШ§ЩҶвҖҢШӘШұ ШҙЩҲШҜ.
ШҜШұ Щ„Ш§ЫҢЩҮЩ” ЩҫЫҢШ§ШҜЩҮвҖҢШіШ§ШІЫҢШҢ Ш§ЫҢЩҶ ШӘЩҲШ§ШЁШ№ ШЁШұ ШЁШіШӘШұ Ш§ЩҶЩҲШ§Ш№ Щ…Ъ©Ш§ЩҶЫҢ ШЁЩҲЩ…ЫҢ SQL Server (geometry ЩҲ geography) ЩҲ TвҖ‘SQL ШіШ§Ш®ШӘЩҮ ШҙШҜЩҮвҖҢШ§ЩҶШҜ ЩҲ ШӘШ§ Ш¬Ш§ЫҢ Щ…Щ…Ъ©ЩҶ ШіШ·Шӯ APIЩҗ ШҙШЁЫҢЩҮ PostGIS Ш§ШұШ§ШҰЩҮ Щ…ЫҢвҖҢШҜЩҮЩҶШҜ.

ЩҶЪ©Ш§ШӘ Ш·ШұШ§ШӯЫҢ
  • ЩҶШ§Щ…вҖҢЪҜШ°Ш§ШұЫҢ ST_: ЩҮЩ…ЩҮЩ” Ъ©Щ…Ъ©вҖҢШӘШ§ШЁШ№вҖҢЩҮШ§ЫҢ Щ…Ъ©Ш§ЩҶЫҢ ШЁШ§ ЩӮШұШ§ШұШҜШ§ШҜ ST_ ЩҲ ЩҫЫҢШҙЩҲЩҶШҜ Щ…ШӯШ§ЩҒШёШӘЫҢ Zz Ш§ШұШ§ШҰЩҮ Щ…ЫҢвҖҢШҙЩҲЩҶШҜШӣ Щ…Ш«Щ„ ZzST_BufferШҢ ZzST_ContainsШҢ ZzST_DWithin.
  • Ш§ЩҲЩ„ЩҲЫҢШӘ ШЁШ§ geometry: ШӘЩҲШ§ШЁШ№ Ш§ШөЩ„ЫҢ ST_* ШұЩҲЫҢ ЩҶЩҲШ№ geometry Ъ©Ш§Шұ Щ…ЫҢвҖҢЪ©ЩҶЩҶШҜШӣ ШҜШұ Ш¬Ш§ЩҮШ§ЫҢЫҢ Ъ©ЩҮ ШҜЩӮШӘ ЪҳШҰЩҲШҜШӘЫҢЪ© Щ„Ш§ШІЩ… Ш§ШіШӘШҢ ШӘЩҲШ§ШЁШ№ Ъ©Щ…Ъ©ЫҢЩҗ Ш¬ШҜШ§ЪҜШ§ЩҶЩҮ ШЁШ§ ЩҲШұЩҲШҜЫҢвҖҢЩҮШ§ЫҢ Ш№ШұШ¶/Ш·ЩҲЩ„ Ш¬ШәШұШ§ЩҒЫҢШ§ЫҢЫҢ (lat/lon) ЩҲ ЩҒШұЩ…ЩҲЩ„ ЩҮШ§ЩҲШұШіШ§ЫҢЩҶ Ш§ШіШӘЩҒШ§ШҜЩҮ Щ…ЫҢвҖҢШҙЩҲЩҶШҜ.
  • ШіШ§ШІЪҜШ§Шұ ШЁШ§ WKT/WKB: ШіШ§ШІЩҶШҜЩҮвҖҢЩҮШ§ ЩҲ ШӘЩҲШ§ШЁШ№ I/OШҢ WKT/WKB ШұШ§ Щ…ЫҢвҖҢЩҫШ°ЫҢШұЩҶШҜ/ШЁШұЩ…ЫҢвҖҢЪҜШұШҜШ§ЩҶЩҶШҜ ШӘШ§ ШӘШіШӘ ЩҲ ШӘШЁШ§ШҜЩ„ ШҜШ§ШҜЩҮ ШіШ§ШҜЩҮ ШЁШ§ШҙШҜ.
  • SRID ЩҫЫҢШҙвҖҢЩҒШұШ¶: Ш§ЪҜШұ SRID ШөШұШ§ШӯШӘШ§ЩӢ ШӘШ№ЫҢЫҢЩҶ ЩҶШҙЩҲШҜШҢ Щ…ЩӮШҜШ§Шұ ЩҫЫҢШҙвҖҢЩҒШұШ¶ 4326 (WGS 84) ШҜШұ ЩҶШёШұ ЪҜШұЩҒШӘЩҮ Щ…ЫҢвҖҢШҙЩҲШҜ.
  • Щ…Ш№ЩҶШ§ШҙЩҶШ§ШіЫҢ ШўШҙЩҶШ§: ЩҶШ§Щ…вҖҢЩҮШ§ ЩҲ ШұЩҒШӘШ§ШұЩҮШ§ ШўЪҜШ§ЩҮШ§ЩҶЩҮ ШЁШ§ Щ…ШұШ¬Ш№ PostGIS ЩҮЩ…ШіЩҲ ШҙШҜЩҮвҖҢШ§ЩҶШҜШӣ ШӘШ§ Ш¬Ш§ЫҢЫҢ Ъ©ЩҮ Ш§Щ…Ъ©Ш§ЩҶШ§ШӘ Щ…Ъ©Ш§ЩҶЫҢ SQL Server Ш§Ш¬Ш§ШІЩҮ Щ…ЫҢвҖҢШҜЩҮШҜ.
ШЁШұШ§ЫҢ Щ…ШұШ¬Ш№ ШіШұЫҢШ№ШҢ ШЁЩҮ: MsSql/Doc/POSTGIS_FUNCTIONALITIES.md Щ…ШұШ§Ш¬Ш№ЩҮ Ъ©ЩҶЫҢШҜ.

ЪҶЩҮ ЪҶЫҢШІЩҮШ§ЫҢЫҢ ШҜШ§Ш®Щ„ ШЁШіШӘЩҮ Ш§ШіШӘШҹ
Ш№Щ„Ш§ЩҲЩҮ ШЁШұ ШӘЩҲШ§ШЁШ№ Ъ©Ш§ШұШЁШұШҜЫҢ Щ…Ш«Щ„ Ш§Ш№ШӘШЁШ§ШұШіЩҶШ¬ЫҢ Щ…Ш®ШӘШөШ§ШӘ ЩҲ ШӘШЁШҜЫҢЩ„ ЩҲШ§ШӯШҜЩҮШ§ШҢ ШЁШіШӘЩҮ ШЁЫҢШҙ Ш§ШІ 45 ШӘШ§ШЁШ№ ШЁЩҮ ШіШЁЪ© PostGIS (ШЁШ§ ЩҫЫҢШҙЩҲЩҶШҜ ZzST_) Ш§ШұШ§ШҰЩҮ Щ…ЫҢвҖҢШҜЩҮШҜШҢ Ш§ШІ Ш¬Щ…Щ„ЩҮ:
  • ШіШ§ШІЩҶШҜЩҮвҖҢЩҮШ§ЫҢ ЩҮЩҶШҜШіЩҮ: ZzST_PointШҢ ZzST_MakePointШҢ ZzST_MakeLineШҢ ZzST_MakePolygonШҢ ZzST_MakeEnvelope
  • ШӘЩҲШ§ШЁШ№ ЩҲШұЩҲШҜЫҢ/Ш®ШұЩҲШ¬ЫҢ: ZzST_AsTextШҢ ZzST_AsBinaryШҢ ZzST_GeomFromTextШҢ ZzST_GeomFromWKB
  • ШҜШіШӘШұШіЫҢвҖҢШҜЩҮЩҶШҜЩҮвҖҢЩҮШ§: ZzST_XШҢ ZzST_YШҢ ZzST_GeometryTypeШҢ ZzST_SRID
  • ШіЩҶШ¬Шҙ ЩҲ ЪҜШІШ§ШұЩҮвҖҢЩҮШ§: ZzST_DistanceШҢ ZzST_LengthШҢ ZzST_AreaШҢ ZzST_PerimeterШҢ ZzST_ContainsШҢ ZzST_WithinШҢ ZzST_IntersectsШҢ ZzST_DWithin
  • ЩҫШұШҜШ§ШІШҙ ЩҮЩҶШҜШіЫҢ: ZzST_BufferШҢ ZzST_CentroidШҢ ZzST_EnvelopeШҢ ZzST_ConvexHullШҢ ZzST_PointOnSurface
  • Ш№Щ…Щ„ЫҢШ§ШӘ Щ…Ш¬Щ…ЩҲШ№ЩҮвҖҢШ§ЫҢ: ZzST_UnionШҢ ZzST_IntersectionШҢ ZzST_DifferenceШҢ ZzST_SymDifference

ЩҮЩ…ЪҶЩҶЫҢЩҶ ЪҶЩҶШҜ Ъ©Щ…Ъ©вҖҢШӘШ§ШЁШ№ Ш№Щ…Щ„вҖҢЪҜШұШ§ЫҢ Щ…ШӯШЁЩҲШЁ ШЁШұШ§ЫҢ BackendЩҮШ§ЫҢ Ш§ЩҫЩ„ЫҢЪ©ЫҢШҙЩҶвҖҢЩҮШ§:
  • ZzIsPointInRadiusШҢ ZzGetPointsInRadiusШҢ ZzGetBoundingBoxШҢ ZzGetNearestPoint
  • Щ…ШЁШҜЩ„вҖҢЩҮШ§ ЩҲ ЩӮШ§Щ„ШЁвҖҢШҜЩҮЩҶШҜЩҮвҖҢЩҮШ§: ZzConvertDMSToDecimalШҢ ZzConvertDecimalToDMSШҢ ZzConvertDistance/Area/Speed
ЩҒЩҮШұШіШӘ Ъ©Ш§Щ…Щ„ ЩҲ Щ…Ш«Ш§Щ„вҖҢЩҮШ§ ШұШ§ ШҜШұ MsSql/Packages/Geographic/README.md ШЁШЁЫҢЩҶЫҢШҜ.

Щ…Ш«Ш§Щ„вҖҢЩҮШ§ЫҢ Ъ©Ш§ШұШЁШұШҜЫҢ
ЩҶЪ©ШӘЩҮ: ШҜШұ Щ…Ш«Ш§Щ„вҖҢЩҮШ§ Ш§ШІ geometry Ш§ШіШӘЩҒШ§ШҜЩҮ ШҙШҜЩҮ Ш§ШіШӘШӣ ШҜШұ ШөЩҲШұШӘ ЩҶЫҢШ§ШІ ШЁЩҮ Щ…ШӯШ§ШіШЁШ§ШӘ ЪҳШҰЩҲШҜШӘЫҢЪ© ШҜЩӮЫҢЩӮШҢ Ш§ШІ geography ШЁЩҮШұЩҮ ШЁЪҜЫҢШұЫҢШҜ. SRID ЩҫЫҢШҙвҖҢЩҒШұШ¶ 4326 Ш§ШіШӘ Щ…ЪҜШұ Ш§ЫҢЩҶЪ©ЩҮ Ш®Щ„Ш§ЩҒ ШўЩҶ ШұШ§ ШӘШ№ЫҢЫҢЩҶ Ъ©ЩҶЫҢШҜ.
  • ШіШ§Ш®ШӘ ЩҶЩӮШ·ЩҮ ЩҲ Ш®Ш· (ШҙШЁЫҢЩҮ PostGIS):
-- POINT(lon lat)
SELECT [dbo].[ZzST_AsText](
 [dbo].[ZzST_MakePoint](51.3890, 35.6892, NULL, NULL)
) AS WKT;

SELECT [dbo].[ZzST_AsText](
 [dbo].[ZzST_MakeLine]('51.3890,35.6892;51.4000,35.7000', 4326)
) AS WKT;
  • ШұЩҲШ§ШЁШ· Щ…Ъ©Ш§ЩҶЫҢ ЩҲ ЪҜШІШ§ШұЩҮвҖҢЩҮШ§:
-- ЩҫШ§Ъ©ШӘ (BBox) Ъ©ЩҮ ЫҢЪ© ЩҶЩӮШ·ЩҮ ШұШ§ ШҜШұШЁШұЪҜЫҢШұШҜ
DECLARE @bbox geometry = [dbo].[ZzST_MakeEnvelope](51.3800, 35.6800, 51.4200, 35.7200, 4326);
DECLARE @pt geometry = [dbo].[ZzST_GeomFromText]('POINT(51.3890 35.6892)', 4326);
SELECT [dbo].[ZzST_Contains](@bbox, @pt) AS ContainsPoint; -- 1 = ШҜШұШіШӘШҢ 0 = ЩҶШ§ШҜШұШіШӘ
  • ЩҒШ§ШөЩ„ЩҮ ЩҲ ЩҮЩ…ШіШ§ЫҢЪҜЫҢ:
-- ШўЫҢШ§ ШҜЩҲ ЩҶЩӮШ·ЩҮ ШҜШұ ШҙШ№Ш§Ш№ 1000 Щ…ШӘШұ Ш§ШІ ЩҮЩ… ЩҮШіШӘЩҶШҜШҹ
DECLARE @a geometry = [dbo].[ZzST_GeomFromText]('POINT(51.3890 35.6892)', 4326);
DECLARE @b geometry = [dbo].[ZzST_GeomFromText]('POINT(51.4000 35.7000)', 4326);
SELECT [dbo].[ZzST_DWithin](@a, @b, 1000.0) AS IsNearby;
  • ШЁШ§ЩҒШұ Ъ©ШұШҜЩҶ:
-- Ш§ЫҢШ¬Ш§ШҜ ШЁШ§ЩҒШұ Ъ©ЩҲЪҶЪ© ЩҫЫҢШұШ§Щ…ЩҲЩҶ ЫҢЪ© ЩҶЩӮШ·ЩҮ (~ЩҲШ§ШӯШҜ ШЁЩҮ SRID ШЁШіШӘЪҜЫҢ ШҜШ§ШұШҜ)
DECLARE @pt geometry = [dbo].[ZzST_GeomFromText]('POINT(51.3890 35.6892)', 4326);
SELECT [dbo].[ZzST_AsText]([dbo].[ZzST_Buffer](@pt, 0.01)) AS Buffered;
  • Ъ©Щ…Ъ©вҖҢШӘШ§ШЁШ№вҖҢЩҮШ§ЫҢ В«ШҜЩҲШіШӘвҖҢШҜШ§Шұ Ш§ЩҫЩ„ЫҢЪ©ЫҢШҙЩҶВ» ШЁШ§ ЩҲШұЩҲШҜЫҢ lat/lon:
-- ШЁШұШұШіЫҢ ЩӮШұШ§ШұЪҜЫҢШұЫҢ ЫҢЪ© ЩҶЩӮШ·ЩҮ (lat/lon) ШҜШұЩҲЩҶ ШҙШ№Ш§Ш№ Щ…ШҙШ®Шө (KM/MILE/M)
SELECT [dbo].[ZzIsPointInRadius](35.6892, 51.3890, 35.7000, 51.4000, 10, 'KM') AS InRadius;

-- ЫҢШ§ЩҒШӘЩҶ ЩҶЩӮШ§Ш· ШҜШұЩҲЩҶ ЫҢЪ© ШҙШ№Ш§Ш№ ЩҫЫҢШұШ§Щ…ЩҲЩҶ Щ…ШұЪ©ШІ (Schema ЩҶЩ…ЩҲЩҶЩҮ ЩҒШұШ¶ЫҢ)
-- SELECT * FROM [dbo].[ZzGetPointsInRadius](
-- @centerLat, @centerLon, @radiusValue, @radiusUnit, @YourPointsTable
-- );

ЩҶЪҜШ§ШҙШӘ ШЁШұШ§ЫҢ Ъ©Ш§ШұШЁШұШ§ЩҶ PostGIS
  • ЩҶШ§Щ…вҖҢЪҜШ°Ш§ШұЫҢ ST_* ШЁШ§ ЩҫЫҢШҙЩҲЩҶШҜ Zz ШӯЩҒШё ШҙШҜЩҮ Ш§ШіШӘ: Щ…Ш«Щ„Ш§ЩӢ ST_Buffer ШҜШұ PostGIS вүҲ ZzST_Buffer ШҜШұ Ш§ЫҢЩҶ ШЁШіШӘЩҮ.
  • ЩҲШұЩҲШҜЫҢвҖҢЩҮШ§ЫҢ WKT/WKB Ш§ШІ Ш·ШұЫҢЩӮ ZzST_GeomFromText ЩҲ ZzST_GeomFromWKB ЩҫШҙШӘЫҢШЁШ§ЩҶЫҢ Щ…ЫҢвҖҢШҙЩҲЩҶШҜ.
  • Щ…ШҜЫҢШұЫҢШӘ SRID ШЁШ§ ZzST_SRID ЩҲ ZzST_SetSRID (ШЁШҜЩҲЩҶ Transform) Ш§ЩҶШ¬Ш§Щ… Щ…ЫҢвҖҢШҙЩҲШҜ.
  • Щ…Ш№Ш§ШҜЩ„вҖҢЩҮШ§ЫҢ SQL Server (ШҜШұЩҲЩҶЫҢ) Щ…Щ…Ъ©ЩҶ Ш§ШіШӘ ЩҶШ§Щ… Щ…ШӘЩҒШ§ЩҲШӘЫҢ ШҜШ§ШҙШӘЩҮ ШЁШ§ШҙЩҶШҜ (Щ…Ш«Щ„ STBufferШҢ STDistance)ШҢ Ш§Щ…Ш§ WrapperЩҮШ§ Ш§ШұЪҜЩҲЩҶЩҲЩ…ЫҢ Щ…ШҙШ§ШЁЩҮ PostGIS Ш§ШұШ§ШҰЩҮ Щ…ЫҢвҖҢШҜЩҮЩҶШҜ.

Ш¬ШҜЩҲЩ„ ЩҶЪҜШ§ШҙШӘ Ъ©ЩҲШӘШ§ЩҮ
PostGIS Zync Geographic ШӘЩҲШ¶ЫҢШӯ/ШӘЩҒШ§ЩҲШӘвҖҢЩҮШ§
ST_BufferZzST_BufferШЁШ§ЩҒШұ ШұЩҲЫҢ geometryШӣ ЩҲШ§ШӯШҜ ШЁЩҮ SRID ЩҲШ§ШЁШіШӘЩҮ Ш§ШіШӘ.
ST_DWithinZzST_DWithinШЁШұШұШіЫҢ ЩҶШІШҜЫҢЪ©ЫҢ ШҜШұ ЩҒШ§ШөЩ„ЩҮЩ” Щ…ШҙШ®Шө (ЩҲШ§ШӯШҜ = ЩҲШ§ШӯШҜ ШіЫҢШіШӘЩ… Щ…Ш®ШӘШөШ§ШӘ). ШЁШұШ§ЫҢ Щ…ШіЫҢШұЩҮШ§ЫҢ Ш·ЩҲЩ„Ш§ЩҶЫҢ geodeticШҢ geography ШұШ§ ШҜШұ ЩҶШёШұ ШЁЪҜЫҢШұЫҢШҜ.
ST_IntersectsZzST_IntersectsШЁШ§ШІЪҜШҙШӘ 0/1Шӣ ШӘЩӮШ§Ш·Ш№ ЩҮШұ ЩҶЩҲШ№ ЩҮЩҶШҜШіЩҮ.
ST_ContainsZzST_ContainsШўЫҢШ§ A ШҙШ§Щ…Щ„ B Ш§ШіШӘ (B Ъ©Ш§Щ…Щ„Ш§ЩӢ ШҜШұЩҲЩҶ A).
ST_WithinZzST_WithinШўЫҢШ§ A ШҜШұЩҲЩҶ B ЩӮШұШ§Шұ ШҜШ§ШұШҜ.
ST_DistanceZzST_DistanceШӯШҜШ§ЩӮЩ„ ЩҒШ§ШөЩ„ЩҮЩ” ШҜЩҲШЁШ№ШҜЫҢШӣ ШЁШұШ§ЫҢ lat/lon ШҜЩӮЫҢЩӮШҢ Ш§ШІ ZzCalculateDistance (Haversine) Ш§ШіШӘЩҒШ§ШҜЩҮ Ъ©ЩҶЫҢШҜ.
ST_AsTextZzST_AsTextШ®ШұЩҲШ¬ЫҢ WKT ШЁШұШ§ЫҢ Щ…ШҙШ§ЩҮШҜЩҮ/ШҜЫҢШЁШ§ЪҜ.
ST_GeomFromTextZzST_GeomFromTextШіШ§ШІЩҶШҜЩҮЩ” ЩҮЩҶШҜШіЩҮ Ш§ШІ WKT ШЁШ§ SRID ЩҲШұЩҲШҜЫҢ.
ЩҶЪ©Ш§ШӘ ЩҲ ШӘЩҲШөЫҢЩҮвҖҢЩҮШ§:
  • ЩҮЩҶЪҜШ§Щ… ШӘШұЪ©ЫҢШЁ ШҜШ§ШҜЩҮ Ш§ШІ Щ…ЩҶШ§ШЁШ№ ЪҜЩҲЩҶШ§ЪҜЩҲЩҶШҢ SRID ШұШ§ ШөШұЫҢШӯ ШӘШ№ЫҢЫҢЩҶ Ъ©ЩҶЫҢШҜ.
  • geography ШҜШұ ШЁШұШ§ШЁШұ geometry: ШЁШұШ§ЫҢ Щ…ШӯШ§ШіШЁШ§ШӘ ЪҳШҰЩҲШҜШӘЫҢЪ© Щ…ШіШ§ЩҒШӘвҖҢЩҮШ§ЫҢ ШЁЩ„ЩҶШҜ Ш§ШІ geography Ш§ШіШӘЩҒШ§ШҜЩҮ Ъ©ЩҶЫҢШҜШӣ ШЁШұШ§ЫҢ Ш№Щ…Щ„ЫҢШ§ШӘ ШөЩҒШӯЩҮвҖҢШ§ЫҢ ЩҲ Ш§Ъ©Ш«Шұ ЪҜШІШ§ШұЩҮвҖҢЩҮШ§ geometry Ъ©ЩҒШ§ЫҢШӘ Щ…ЫҢвҖҢЪ©ЩҶШҜ.
  • ШӘЩҲЩ„ШұШ§ЩҶШі ЩҲ ЩҲШ§ШӯШҜ: ШҙШ№Ш§Ш№ ШЁШ§ЩҒШұ ЩҲ ЩҒШ§ШөЩ„ЩҮвҖҢЩҮШ§ ШЁЩҮ ШіЫҢШіШӘЩ… Щ…Ш®ШӘШөШ§ШӘ ЩҲШ§ШЁШіШӘЩҮвҖҢШ§ЩҶШҜвҖ”SRIDЩҮШ§ЫҢ ЩҮЩ…ШіШ§ЩҶ (Щ…Ш«Щ„ WGS 84 ШЁШұШ§ЫҢ GPS) ШұШ§ ШӘШұШ¬ЫҢШӯ ШҜЩҮЫҢШҜ.

ЩҶШөШЁ
ЩҶШөШЁ Щ…ШіШӘЩӮЫҢЩ… Ш§ШІ Щ…Ш®ШІЩҶ Zync ШЁШ§ ШҜШіШӘЩҲШұ Zync:
-- ЩҶШөШЁ Ъ©Щ„ ШЁШіШӘЩҮЩ” Geographic
EXEC [dbo].[Zync] 'i Geographic';

-- ЫҢШ§ ЩҒЩӮШ· ЩҶШөШЁ ШӘЩҲШ§ШЁШ№ Щ…Ъ©Ш§ЩҶЫҢ
EXEC [dbo].[Zync] 'i Geographic/ZzST_*.sql';
Ш§ЪҜШұ ЩҮЩҶЩҲШІ Zync ШұШ§ ЩҶШөШЁ ЩҶЪ©ШұШҜЩҮвҖҢШ§ЫҢШҜШҢ Ш§ШіЪ©ШұЫҢЩҫШӘ MsSql/Zync.sql ШұШ§ Ш§ШІ Щ…Ш®ШІЩҶ Ш§Ш¬ШұШ§ Ъ©ЩҶЫҢШҜ. ШЁШұШ§ЫҢ Щ…Ш№ШұЩҒЫҢ ШіШұЫҢШ№ ШЁЩҮ MsSql/Doc/ARTICLE_EN.md ЩҲ ШЁШұШ§ЫҢ Ш®ЩҲШҜЪ©Ш§ШұШіШ§ШІЫҢ ШЁЩҮ MsSql/scripts/README.md Щ…ШұШ§Ш¬Ш№ЩҮ Ъ©ЩҶЫҢШҜ.

Щ…ЩҶШ§ШЁШ№

Ш§ЪҜШұ ШЁШ§ PostGIS ШұШ§ШӯШӘ ШЁЩҲШҜЩҮвҖҢШ§ЫҢШҜШҢ Ш§ЫҢЩҶШ¬Ш§ ЩҮЩ… Ш§ШӯШіШ§Ші ШўШҙЩҶШ§ЫҢЫҢ Щ…ЫҢвҖҢЪ©ЩҶЫҢШҜ: ШЁШіШӘЩҮЩ” Geographic ШҜШұ Zync ЫҢЪ© Ш¬Ш№ШЁЩҮвҖҢШ§ШЁШІШ§Шұ Щ…Ъ©Ш§ЩҶЫҢ ШўШҙЩҶШ§ШҢ Щ…ШҜШұЩҶ ЩҲ ProductionвҖ‘Ready ШұШ§ ШЁШұШ§ЫҢ SQL Server Щ…ЫҢвҖҢШўЩҲШұШҜШӣ Щ…ШіШӘЩҶШҜШҢ ШўШІЩ…ЩҲШҜЩҮ ЩҲ ШўЩ…Ш§ШҜЩҮЩ” Ш§ШіШӘЩҒШ§ШҜЩҮ.
Щ…Ш·Ш§Щ„ШЁ Щ…ШҙШ§ШЁЩҮ
    Ш§ШҙШӘШұШ§Ъ©вҖҢЩҮШ§Щ…Ш№ШұЩҒЫҢ Ш§ЩҒШІЩҲЩҶЩҮ Ш¬ШҜЫҢШҜ PostgreSQL ШЁШұШ§ЫҢ VS CodeШ§ШҙШӘШұШ§Ъ©вҖҢЩҮШ§GeoJSON ЩҒШұЩ…ШӘ ШіШ§Ш®ШӘШ§ШұЫҢ Ш§Ш·Щ„Ш§Ш№Ш§ШӘ Ш¬ШәШұШ§ЩҒЫҢШ§ЫҢЫҢШ§ШҙШӘШұШ§Ъ©вҖҢЩҮШ§ШұЩҲШҙ ЫҢЪ©ЩҫШ§ШұЪҶЩҮ ШіШ§ШІЫҢ NET Aspire. ШЁШ§ SQL ServerШЁШ§ШІШ®ЩҲШұШҜЩҮШ§ЫҢ ШҜЩҲШұЩҮШҜШұЫҢШ§ЩҒШӘ ЩӮШ§Щ„ШЁ WpfFramework.vsix ЩҲ ЩҶШӯЩҲЩҮ ЩҶШөШЁ ЩҲ ШұШ§ЩҮ Ш§ЩҶШҜШ§ШІЫҢ ШўЩҶЩ…Ш·Ш§Щ„ШЁШҜШіШӘ Ш§ШІ Щ…ШҜЫҢШұЫҢШӘ ШҜШіШӘЫҢ Ш§ШіЪ©ШұЫҢЩҫШӘвҖҢЩҮШ§ЫҢ SQL ШЁШұШҜШ§ШұЫҢШҜ: Щ…Ш№ШұЩҒЫҢ ZyncШҢ Щ…ШҜЫҢШұ ШЁШіШӘЩҮ ЩҫШ§ЫҢЪҜШ§ЩҮ ШҜШ§ШҜЩҮЩ…Ш·Ш§Щ„ШЁЩҫШұШҜШ§ШІШҙ ШҜШ§ШҜЩҮвҖҢЩҮШ§ЫҢ Ш¬ШәШұШ§ЩҒЫҢШ§ЫҢЫҢ ШЁЩҮ Ъ©Щ…Ъ© SQL Server ЩҲ Entity frameworkЩ…Ш·Ш§Щ„ШЁLocalDB ЪҶЫҢШіШӘШҹЩҶШёШұШ§ШӘ Щ…Ш·Ш§Щ„ШЁШ§ЩҒШІЩҲШҜЩҶ ЫҢЪ© DataType Ш¬ШҜЫҢШҜ ШЁШұШ§ЫҢ ЩҶЪҜЩҮвҖҢШҜШ§ШұЫҢ ШӘШ§ШұЫҢШ® Ш®ЩҲШұШҙЫҢШҜЫҢ - 3ЩҶШёШұШ§ШӘ Щ…Ш·Ш§Щ„ШЁЩ…ЩҶШіЩҲШ® ШҙШҜЩҶ DllImport ШҜШұ ШҜШ§ШӘ ЩҶШӘ 7ЩҶШёШұШ§ШӘ Щ…Ш·Ш§Щ„ШЁШЁШұШұШіЫҢ Ш®Ш·Ш§ЩҮШ§ЫҢ Щ…Щ…Ъ©ЩҶ ШҜШұ ШӯЫҢЩҶ ШұШ§ЩҮ Ш§ЩҶШҜШ§ШІЫҢ Ш§ЩҲЩ„ЫҢЩҮ ШЁШұЩҶШ§Щ…ЩҮвҖҢЩҮШ§ЫҢ ASP.NET Core ШҜШұ IIS
