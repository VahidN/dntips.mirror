Щ…ЩӮШҜЩ…ЩҮ
ЩӮШ§ШЁЩ„ЫҢШӘ Reflection ЫҢЪ©ЫҢ Ш§ШІ Ш§ШЁШІШ§ШұЩҮШ§ЫҢ ЩӮШҜШұШӘЩ…ЩҶШҜ ЩҲ Ш§ЩҶШ№Ш·Ш§ЩҒвҖҢЩҫШ°ЫҢШұ ШЁШұШ§ЫҢ ШҜШіШӘШұШіЫҢ ШЁЩҮ Ш§Ш·Щ„Ш§Ш№Ш§ШӘ Щ…ШӘШ§ШҜЫҢШӘШ§ (Metadata) ШҜШұ ШІЩ…Ш§ЩҶ Ш§Ш¬ШұШ§ (Runtime) Ш§ШіШӘ. Ш§ЫҢЩҶ Ш§Щ…Ъ©Ш§ЩҶ ШЁЩҮ ШӘЩҲШіШ№ЩҮвҖҢШҜЩҮЩҶШҜЪҜШ§ЩҶ Ш§Ш¬Ш§ШІЩҮ Щ…ЫҢвҖҢШҜЩҮШҜ ШӘШ§ ШЁЩҮ ШөЩҲШұШӘ ЩҫЩҲЫҢШ§ (Dynamic) ШЁЩҮ ШіШ§Ш®ШӘШ§Шұ Ъ©ШҜЩҮШ§ШҢ Щ…Ш§ЩҶЩҶШҜ Ъ©Щ„Ш§ШівҖҢЩҮШ§ШҢ Щ…ШӘШҜЩҮШ§ШҢ ЩҲ Ш®ШөЩҲШөЫҢШ§ШӘ (Properties)ШҢ ШҜШіШӘШұШіЫҢ ЩҫЫҢШҜШ§ Ъ©ЩҶЩҶШҜ ЩҲ ШӯШӘЫҢ ШўЩҶвҖҢЩҮШ§ ШұШ§ ШӘШәЫҢЫҢШұ ШҜЩҮЩҶШҜ ЫҢШ§ ЩҒШұШ§Ш®ЩҲШ§ЩҶЫҢ Ъ©ЩҶЩҶШҜ. ШЁШ§ Ш§ЫҢЩҶ ШӯШ§Щ„ШҢ Ш§ШіШӘЩҒШ§ШҜЩҮ ЩҶШ§ШҜШұШіШӘ ЫҢШ§ ШЁЫҢШҙ Ш§ШІ ШӯШҜ Ш§ШІ Reflection Щ…ЫҢвҖҢШӘЩҲШ§ЩҶШҜ ШЁЩҮ Ъ©Ш§ЩҮШҙ Ъ©Ш§ШұШ§ЫҢЫҢ (Performance) ШЁШұЩҶШ§Щ…ЩҮ Щ…ЩҶШ¬Шұ ШҙЩҲШҜШҢ ШІЫҢШұШ§ Ш§ЫҢЩҶ Щ…Ъ©Ш§ЩҶЫҢШІЩ… Щ…Ш№Щ…ЩҲЩ„Ш§ЩӢ ЩҮШІЫҢЩҶЩҮвҖҢЩҮШ§ЫҢ Щ…ШӯШ§ШіШЁШ§ШӘЫҢ ШЁШ§Щ„Ш§ЫҢЫҢ ШҜШ§ШұШҜ.
ЩҮШҜЩҒ Ш§ЫҢЩҶ Щ…ЩӮШ§Щ„ЩҮШҢ ШЁШұШұШіЫҢ ШұЩҲШҙвҖҢЩҮШ§ЫҢ ШЁЩҮЫҢЩҶЩҮвҖҢШіШ§ШІЫҢ Ш§ШіШӘЩҒШ§ШҜЩҮ Ш§ШІ Reflection ШҜШұ ШІШЁШ§ЩҶ #C Ш§ШіШӘ.

Reflection ЪҶЫҢШіШӘ ЩҲ ЪҶШұШ§ ШЁЩҮ ШЁЩҮЫҢЩҶЩҮвҖҢШіШ§ШІЫҢ ЩҶЫҢШ§ШІ ШҜШ§ШұШҜШҹ
Reflection ШҜШұ #C ШЁЩҮ Щ…Ш№ЩҶШ§ЫҢ ШӘЩҲШ§ЩҶШ§ЫҢЫҢ ШЁШ§ШІШӘШ§ШЁ (Reflection) Ш§Ш·Щ„Ш§Ш№Ш§ШӘ Щ…ШұШЁЩҲШ· ШЁЩҮ Ш§ЩҶЩҲШ§Ш№ (Types) ЩҲ Ш§Ш№Ш¶Ш§ЫҢ (Members) ШўЩҶвҖҢЩҮШ§ ШҜШұ ШІЩ…Ш§ЩҶ Ш§Ш¬ШұШ§ Ш§ШіШӘ. Ш§ЫҢЩҶ ЩӮШ§ШЁЩ„ЫҢШӘ Ш§ШІ Ш·ШұЫҢЩӮ ЩҒШ¶Ш§ЫҢ ЩҶШ§Щ… System.Reflection ШҜШұ ШҜШіШӘШұШі ЩӮШұШ§Шұ ШҜШ§ШұШҜ ЩҲ ШЁЩҮ ШЁШұЩҶШ§Щ…ЩҮвҖҢЩҶЩҲЫҢШіШ§ЩҶ Ш§Ш¬Ш§ШІЩҮ Щ…ЫҢвҖҢШҜЩҮШҜ ШӘШ§ ШЁШҜЩҲЩҶ ШҜШ§ЩҶШіШӘЩҶ ШіШ§Ш®ШӘШ§Шұ ШҜЩӮЫҢЩӮ ЫҢЪ© ШҙЫҢШЎ ШҜШұ ШІЩ…Ш§ЩҶ Ъ©Ш§Щ…ЩҫШ§ЫҢЩ„ (Compile-Time)ШҢ ШЁЩҮ ШўЩҶ ШҜШіШӘШұШіЫҢ ЩҫЫҢШҜШ§ Ъ©ЩҶЩҶШҜ. ШЁШұШ§ЫҢ Щ…Ш«Ш§Щ„ШҢ Щ…ЫҢвҖҢШӘЩҲШ§ЩҶ ЩҶШ§Щ… ЫҢЪ© Ъ©Щ„Ш§ШіШҢ Щ„ЫҢШіШӘ Щ…ШӘШҜЩҮШ§ЫҢ ШўЩҶШҢ ЫҢШ§ ШӯШӘЫҢ Щ…ЩӮШ§ШҜЫҢШұ Ш®ШөЩҲШөЫҢШ§ШӘШҙ ШұШ§ ШҜШұ ШІЩ…Ш§ЩҶ Ш§Ш¬ШұШ§ Ш§ШіШӘШ®ШұШ§Ш¬ Ъ©ШұШҜ.
ШЁШ§ Ш§ЫҢЩҶ ШӯШ§Щ„ШҢ Reflection ШЁЩҮ ШҜЩ„ЫҢЩ„ Ш§ШіШӘЩҒШ§ШҜЩҮ Ш§ШІ Ш№Щ…Щ„ЫҢШ§ШӘ ЩҫЩҲЫҢШ§ ЩҲ Ш№ШҜЩ… ШЁЩҮЫҢЩҶЩҮвҖҢШіШ§ШІЫҢ ШӘЩҲШіШ· Ъ©Ш§Щ…ЩҫШ§ЫҢЩ„Шұ (Compiler)ШҢ Щ…Ш№Щ…ЩҲЩ„Ш§ЩӢ Ъ©ЩҶШҜШӘШұ Ш§ШІ Ъ©ШҜЩҮШ§ЫҢ Ш§ЫҢШіШӘШ§ (Static) Ш§ШіШӘ. Ш§ЫҢЩҶ Ъ©ЩҶШҜЫҢ ШЁЩҮ Ш®Ш§Ш·Шұ Ш¬ШіШӘШ¬ЩҲ ШҜШұ Щ…ШӘШ§ШҜЫҢШӘШ§ШҢ Ш§ЫҢШ¬Ш§ШҜ Ш§ШҙЫҢШ§ШЎ Ш¬ШҜЫҢШҜШҢ ЫҢШ§ ЩҒШұШ§Ш®ЩҲШ§ЩҶЫҢ Щ…ШӘШҜЩҮШ§ Ш§ШІ Ш·ШұЫҢЩӮ ЩҲШ§ШіШ·ЩҮвҖҢЩҮШ§ЫҢЫҢ Щ…Ш§ЩҶЩҶШҜ MethodInfo Ш§ШіШӘ. ШЁЩҶШ§ШЁШұШ§ЫҢЩҶШҢ ШЁЩҮЫҢЩҶЩҮвҖҢШіШ§ШІЫҢ Ш§ШіШӘЩҒШ§ШҜЩҮ Ш§ШІ ШўЩҶ ШЁШұШ§ЫҢ ШЁШұЩҶШ§Щ…ЩҮвҖҢЩҮШ§ЫҢЫҢ Ъ©ЩҮ ШЁЩҮ Ш№Щ…Щ„Ъ©ШұШҜ ШЁШ§Щ„Ш§ ЩҶЫҢШ§ШІ ШҜШ§ШұЩҶШҜШҢ Ш¶ШұЩҲШұЫҢ Ш§ШіШӘ.

ЩҶЪ©Ш§ШӘ Ъ©Щ„ЫҢШҜЫҢ ШЁШұШ§ЫҢ ШЁЩҮЫҢЩҶЩҮвҖҢШіШ§ШІЫҢ
ШЁШұШ§ЫҢ Ш§ШіШӘЩҒШ§ШҜЩҮ ШЁЩҮЫҢЩҶЩҮ Ш§ШІ ReflectionШҢ ШЁШ§ЫҢШҜ ШЁЩҮ ЪҶЩҶШҜ Ш§ШөЩ„ Ш§ШіШ§ШіЫҢ ШӘЩҲШ¬ЩҮ Ъ©ШұШҜ:
  1. Ъ©Ш§ЩҮШҙ Ш§ШіШӘЩҒШ§ШҜЩҮ ШәЫҢШұШ¶ШұЩҲШұЫҢ: Reflection ШұШ§ ШӘЩҶЩҮШ§ ШІЩ…Ш§ЩҶЫҢ ШЁЩҮ Ъ©Ш§Шұ ШЁШЁШұЫҢШҜ Ъ©ЩҮ ШұШ§ЩҮвҖҢШӯЩ„вҖҢЩҮШ§ЫҢ Ш§ЫҢШіШӘШ§ ЫҢШ§ Ш¬ЩҶШұЫҢЪ© (Generic) ЩҫШ§ШіШ®ЪҜЩҲ ЩҶШЁШ§ШҙЩҶШҜ.
  2. Ш°Ш®ЫҢШұЩҮвҖҢШіШ§ШІЫҢ ЩҶШӘШ§ЫҢШ¬ (Caching): ЩҶШӘШ§ЫҢШ¬ Щ…ШӘШ§ШҜЫҢШӘШ§ ЫҢШ§ Ш§ШҙЫҢШ§ШЎ ШӘЩҲЩ„ЫҢШҜШҙШҜЩҮ ШӘЩҲШіШ· Reflection ШұШ§ Ш°Ш®ЫҢШұЩҮ Ъ©ЩҶЫҢШҜ ШӘШ§ Ш§ШІ Щ…ШӯШ§ШіШЁШ§ШӘ ШӘЪ©ШұШ§ШұЫҢ Ш¬Щ„ЩҲЪҜЫҢШұЫҢ ШҙЩҲШҜ.
  3. Ш§ШіШӘЩҒШ§ШҜЩҮ Ш§ШІ DelegateЩҮШ§: ШЁЩҮ Ш¬Ш§ЫҢ ЩҒШұШ§Ш®ЩҲШ§ЩҶЫҢ Щ…ШіШӘЩӮЫҢЩ… Щ…ШӘШҜЩҮШ§ ШЁШ§ ReflectionШҢ Ш§ШІ DelegateЩҮШ§ ШЁШұШ§ЫҢ Ш§ЩҒШІШ§ЫҢШҙ ШіШұШ№ШӘ Ш§ШіШӘЩҒШ§ШҜЩҮ Ъ©ЩҶЫҢШҜ.
  4. ШЁЩҮШұЩҮвҖҢЪҜЫҢШұЫҢ Ш§ШІ Expression Trees: ШЁШ§ Ш§ШіШӘЩҒШ§ШҜЩҮ Ш§ШІ ШҜШұШ®ШӘвҖҢЩҮШ§ЫҢ Ш№ШЁШ§ШұШӘ (Expression Trees)ШҢ Ъ©ШҜЩҮШ§ЫҢ ЩҫЩҲЫҢШ§ ШұШ§ ШЁЩҮЫҢЩҶЩҮвҖҢШӘШұ ЩҲ ШіШұЫҢШ№вҖҢШӘШұ Ъ©ЩҶЫҢШҜ.
  5. Ш§ШіШӘЩҒШ§ШҜЩҮ Ш§ШІ Ш§ШЁШІШ§ШұЩҮШ§ЫҢ Щ…ШҜШұЩҶ ШҜШ§ШӘвҖҢЩҶШӘ: ШҜШұ ЩҶШіШ®ЩҮвҖҢЩҮШ§ЫҢ Ш¬ШҜЫҢШҜ ШҜШ§ШӘвҖҢЩҶШӘШҢ Ш§ШЁШІШ§ШұЩҮШ§ЫҢЫҢ Щ…Ш§ЩҶЩҶШҜ Reflection.Emit ЫҢШ§ Source Generators Щ…ЫҢвҖҢШӘЩҲШ§ЩҶЩҶШҜ Ш¬Ш§ЫҢЪҜШІЫҢЩҶ ШЁЩҮШӘШұЫҢ ШЁШ§ШҙЩҶШҜ.

ШҜШұ Ш§ШҜШ§Щ…ЩҮШҢ ЩҮШұ ЫҢЪ© Ш§ШІ Ш§ЫҢЩҶ ЩҶЪ©Ш§ШӘ ШұШ§ ШЁШ§ Ш¬ШІШҰЫҢШ§ШӘ ШЁЫҢШҙШӘШұЫҢ ШЁШұШұШіЫҢ Щ…ЫҢвҖҢЪ©ЩҶЫҢЩ….
1. Ъ©Ш§ЩҮШҙ Ш§ШіШӘЩҒШ§ШҜЩҮ ШәЫҢШұШ¶ШұЩҲШұЫҢ
Ш§ЩҲЩ„ЫҢЩҶ ЩӮШҜЩ… ШҜШұ ШЁЩҮЫҢЩҶЩҮвҖҢШіШ§ШІЫҢШҢ Ш§ШұШІЫҢШ§ШЁЫҢ ЩҶЫҢШ§ШІ ЩҲШ§ЩӮШ№ЫҢ ШЁЩҮ Reflection Ш§ШіШӘ. ШЁШұШ§ЫҢ Щ…Ш«Ш§Щ„ШҢ Ш§ЪҜШұ ЩҮШҜЩҒ ШӘЩҶЩҮШ§ ШҜШіШӘШұШіЫҢ ШЁЩҮ ЫҢЪ© Ш®Ш§ШөЫҢШӘ Щ…ШҙШ®Шө ШЁШ§ШҙШҜШҢ Ш§ШіШӘЩҒШ§ШҜЩҮ Ш§ШІ ЫҢЪ© ШұШ§ШЁШ· (Interface) ЫҢШ§ Щ…ШӘШҜ Ш¬ЩҶШұЫҢЪ© Щ…ЫҢвҖҢШӘЩҲШ§ЩҶШҜ Ш¬Ш§ЫҢЪҜШІЫҢЩҶ ШЁЩҮШӘШұЫҢ ШЁШ§ШҙШҜ. Reflection ШЁШ§ЫҢШҜ ШІЩ…Ш§ЩҶЫҢ Ш§ШіШӘЩҒШ§ШҜЩҮ ШҙЩҲШҜ Ъ©ЩҮ Ш§ЩҶШ№Ш·Ш§ЩҒвҖҢЩҫШ°ЫҢШұЫҢ ЩҫЩҲЫҢШ§ ЩҲШ§ЩӮШ№Ш§ЩӢ Ш¶ШұЩҲШұЫҢ ШЁШ§ШҙШҜШҢ Щ…Ш§ЩҶЩҶШҜ ШЁШ§ШұЪҜШ°Ш§ШұЫҢ Ш§ШіЩ…ШЁЩ„ЫҢвҖҢЩҮШ§ (Assemblies) ШҜШұ ШІЩ…Ш§ЩҶ Ш§Ш¬ШұШ§ ЫҢШ§ Ъ©Ш§Шұ ШЁШ§ ШіШ§Ш®ШӘШ§ШұЩҮШ§ЫҢЫҢ Ъ©ЩҮ ШҜШұ ШІЩ…Ш§ЩҶ Ъ©Ш§Щ…ЩҫШ§ЫҢЩ„ ЩҶШ§ШҙЩҶШ§Ш®ШӘЩҮвҖҢШ§ЩҶШҜ.

2. Ш°Ш®ЫҢШұЩҮвҖҢШіШ§ШІЫҢ/Ъ©Шҙ Ъ©ШұШҜЩҶ ЩҶШӘШ§ЫҢШ¬ (Caching)
ЫҢЪ©ЫҢ Ш§ШІ Щ…ШӨШ«ШұШӘШұЫҢЩҶ ШұЩҲШҙвҖҢЩҮШ§ ШЁШұШ§ЫҢ ШЁЩҮЫҢЩҶЩҮвҖҢШіШ§ШІЫҢШҢ Ш°Ш®ЫҢШұЩҮвҖҢШіШ§ШІЫҢ/Ъ©Шҙ Ъ©ШұШҜЩҶ ЩҶШӘШ§ЫҢШ¬ Reflection Ш§ШіШӘ. ЩҒШұШ¶ Ъ©ЩҶЫҢШҜ ЩҶЫҢШ§ШІ ШҜШ§ШұЫҢШҜ ШЁШ§ШұЩҮШ§ ШЁЩҮ ЫҢЪ© Щ…ШӘШҜ Ш®Ш§Шө Ш§ШІ ЫҢЪ© Ъ©Щ„Ш§Ші ШҜШіШӘШұШіЫҢ ЩҫЫҢШҜШ§ Ъ©ЩҶЫҢШҜ. ШЁЩҮ Ш¬Ш§ЫҢ Ш§ЫҢЩҶЪ©ЩҮ ЩҮШұ ШЁШ§Шұ MethodInfo ШұШ§ ШҜЩҲШЁШ§ШұЩҮ Ш§ШіШӘШ®ШұШ§Ш¬ Ъ©ЩҶЫҢШҜШҢ Щ…ЫҢвҖҢШӘЩҲШ§ЩҶЫҢШҜ ШўЩҶ ШұШ§ ЫҢЪ©вҖҢШЁШ§Шұ Щ…ШӯШ§ШіШЁЩҮ ЩҲ ШҜШұ ЫҢЪ© Щ…ШӘШәЫҢШұ Ш§ШіШӘШ§ШӘЫҢЪ© Ш°Ш®ЫҢШұЩҮ Ъ©ЩҶЫҢШҜ. Щ…Ш«Ш§Щ„ ШІЫҢШұ Ш§ЫҢЩҶ Щ…ЩҒЩҮЩҲЩ… ШұШ§ ЩҶШҙШ§ЩҶ Щ…ЫҢвҖҢШҜЩҮШҜ:
public class Person
{
    public int Age { get; set; }
    public void SetAge(int age) => Age = age;
}

public class ReflectionExample
{
    private static readonly MethodInfo _setAgeMethod = 
        typeof(Person).GetMethod("SetAge", new[] { typeof(int) });

    public void SetAgeWithReflection(Person person, int age)
    {
        _setAgeMethod.Invoke(person, new object[] { age });
    }
}
ШҜШұ Ш§ЫҢЩҶ Ъ©ШҜШҢ MethodInfo ЫҢЪ©вҖҢШЁШ§Шұ ШҜШұ Щ…ШӘШәЫҢШұ Ш§ШіШӘШ§ШӘЫҢЪ©_setAgeMethod Ш°Ш®ЫҢШұЩҮ ШҙШҜЩҮ ЩҲ ШҜШұ ЩҒШұШ§Ш®ЩҲШ§ЩҶЫҢвҖҢЩҮШ§ЫҢ ШЁШ№ШҜЫҢ ШҜЩҲШЁШ§ШұЩҮ Ш§ШіШӘЩҒШ§ШҜЩҮ Щ…ЫҢвҖҢШҙЩҲШҜ. Ш§ЫҢЩҶ ШұЩҲШҙ ЩҮШІЫҢЩҶЩҮ Ш§ЩҲЩ„ЫҢЩҮ Ш¬ШіШӘШ¬ЩҲ ШұШ§ ШӘЩҶЩҮШ§ ЫҢЪ©вҖҢШЁШ§Шұ ЩҫШұШҜШ§Ш®ШӘ Щ…ЫҢвҖҢЪ©ЩҶШҜ ЩҲ Ш№Щ…Щ„Ъ©ШұШҜ ШұШ§ ШЁЩҮШЁЩҲШҜ Щ…ЫҢвҖҢШЁШ®ШҙШҜ.

3. Ш§ШіШӘЩҒШ§ШҜЩҮ Ш§ШІ DelegateЩҮШ§ ШЁШұШ§ЫҢ ЩҒШұШ§Ш®ЩҲШ§ЩҶЫҢ ШіШұЫҢШ№вҖҢШӘШұ
ЩҒШұШ§Ш®ЩҲШ§ЩҶЫҢ Щ…ШӘШҜЩҮШ§ ШЁШ§ Invoke ШҜШұ Reflection Ъ©ЩҶШҜ Ш§ШіШӘШҢ ШІЫҢШұШ§ ШҙШ§Щ…Щ„ ШӘШЁШҜЫҢЩ„ ЩҶЩҲШ№ (Boxing) ШўШұЪҜЩҲЩ…Ш§ЩҶвҖҢЩҮШ§ ЩҲ ШЁШұШұШіЫҢвҖҢЩҮШ§ЫҢ Ш§Щ…ЩҶЫҢШӘЫҢ ШҜШұ ШІЩ…Ш§ЩҶ Ш§Ш¬ШұШ§ Щ…ЫҢвҖҢШҙЩҲШҜ. ЫҢЪ© ШұШ§ЩҮЪ©Ш§Шұ ШЁЩҮШӘШұШҢ ШӘШЁШҜЫҢЩ„ MethodInfo ШЁЩҮ ЫҢЪ© Delegate Ш§ШіШӘ. DelegateЩҮШ§ Ъ©ШҜ ШұШ§ ШЁЩҮ ШөЩҲШұШӘ Щ…ШіШӘЩӮЫҢЩ… Ш§Ш¬ШұШ§ Щ…ЫҢвҖҢЪ©ЩҶЩҶШҜ ЩҲ Ш№Щ…Щ„Ъ©ШұШҜЫҢ ЩҶШІШҜЫҢЪ© ШЁЩҮ ЩҒШұШ§Ш®ЩҲШ§ЩҶЫҢ Ш§ЫҢШіШӘШ§ ШҜШ§ШұЩҶШҜ. Щ…Ш«Ш§Щ„ ШІЫҢШұ Ш§ЫҢЩҶ ШӘЪ©ЩҶЫҢЪ© ШұШ§ ЩҶШҙШ§ЩҶ Щ…ЫҢвҖҢШҜЩҮШҜ:
public class DelegateExample
{
    private static readonly Action<Person, int> _setAgeDelegate = 
        (Action<Person, int>)Delegate.CreateDelegate(
            typeof(Action<Person, int>), 
            typeof(Person).GetMethod("SetAge"));

    public void SetAgeWithDelegate(Person person, int age)
    {
        _setAgeDelegate(person, age);
    }
}
ШҜШұ Ш§ЫҢЩҶШ¬Ш§ШҢ _setAgeDelegate ЫҢЪ©вҖҢШЁШ§Шұ Ш§ЫҢШ¬Ш§ШҜ ШҙШҜЩҮ ЩҲ ШіЩҫШі ШЁЩҮ ШіШұШ№ШӘ ЩҒШұШ§Ш®ЩҲШ§ЩҶЫҢ Щ…ЫҢвҖҢШҙЩҲШҜ. Ш§ЫҢЩҶ ШұЩҲШҙ ШЁЩҮ Щ…ШұШ§ШӘШЁ ШіШұЫҢШ№вҖҢШӘШұ Ш§ШІ Invoke Ш§ШіШӘ ЩҲ ШЁШұШ§ЫҢ ШіЩҶШ§ШұЫҢЩҲЩҮШ§ЫҢЫҢ Ъ©ЩҮ Ш№Щ…Щ„Ъ©ШұШҜ ШӯЫҢШ§ШӘЫҢ Ш§ШіШӘШҢ Ш§ЫҢШҜЩҮвҖҢШўЩ„ Щ…ШӯШіЩҲШЁ Щ…ЫҢвҖҢШҙЩҲШҜ.

4. ШЁЩҮШұЩҮвҖҢЪҜЫҢШұЫҢ Ш§ШІ Expression Trees
ШҜШұШ®ШӘвҖҢЩҮШ§ЫҢ Ш№ШЁШ§ШұШӘ (Expression Trees) ЫҢЪ©ЫҢ ШҜЫҢЪҜШұ Ш§ШІ Ш§ШЁШІШ§ШұЩҮШ§ЫҢ ЩӮШҜШұШӘЩ…ЩҶШҜ ШҜШұ ШҜШ§ШӘвҖҢЩҶШӘ ЩҮШіШӘЩҶШҜ Ъ©ЩҮ Щ…ЫҢвҖҢШӘЩҲШ§ЩҶЩҶШҜ ШЁШұШ§ЫҢ ШЁЩҮЫҢЩҶЩҮвҖҢШіШ§ШІЫҢ Reflection Ш§ШіШӘЩҒШ§ШҜЩҮ ШҙЩҲЩҶШҜ. Ш§ЫҢЩҶ Ш§ШЁШІШ§Шұ ШЁЩҮ ШҙЩ…Ш§ Ш§Ш¬Ш§ШІЩҮ Щ…ЫҢвҖҢШҜЩҮШҜ ШӘШ§ Ш№ШЁШ§ШұШ§ШӘ Ъ©ШҜ ШұШ§ ШЁЩҮ ШөЩҲШұШӘ ЩҫЩҲЫҢШ§ ШЁШіШ§ШІЫҢШҜ ЩҲ ШіЩҫШі ШўЩҶвҖҢЩҮШ§ ШұШ§ ШЁЩҮ Ъ©ШҜ Ъ©Ш§Щ…ЩҫШ§ЫҢЩ„вҖҢШҙШҜЩҮ (Compiled Code) ШӘШЁШҜЫҢЩ„ Ъ©ЩҶЫҢШҜ. Ш§ЫҢЩҶ ШұЩҲШҙ Ш№Щ…Щ„Ъ©ШұШҜЫҢ ЩҶШІШҜЫҢЪ© ШЁЩҮ Ъ©ШҜЩҮШ§ЫҢ Ш§ЫҢШіШӘШ§ Ш§ШұШ§ШҰЩҮ Щ…ЫҢвҖҢШҜЩҮШҜ ЩҲ Ш§ШІ ЩҮШІЫҢЩҶЩҮвҖҢЩҮШ§ЫҢ Reflection Ш®Ш§Щ„Шө Ъ©Ш§ШіШӘЩҮ Щ…ЫҢвҖҢШҙЩҲШҜ. Щ…Ш«Ш§Щ„ ШІЫҢШұ Ш§ШіШӘЩҒШ§ШҜЩҮ Ш§ШІ Expression Trees ШұШ§ ШЁШұШ§ЫҢ ШҜШіШӘШұШіЫҢ ШЁЩҮ ЫҢЪ© Ш®Ш§ШөЫҢШӘ ЩҶШҙШ§ЩҶ Щ…ЫҢвҖҢШҜЩҮШҜ:
using System.Linq.Expressions;

public class ExpressionExample
{
    private static readonly Func<Person, int> _getAgeCompiled;

    static ExpressionExample()
    {
        var parameter = Expression.Parameter(typeof(Person), "p");
        var property = Expression.Property(parameter, "Age");
        var lambda = Expression.Lambda<Func<Person, int>>(property, parameter);
        _getAgeCompiled = lambda.Compile();
    }

    public int GetAgeWithExpression(Person person)
    {
        return _getAgeCompiled(person);
    }
}
ШҜШұ Ш§ЫҢЩҶ Ъ©ШҜШҢ ЫҢЪ© Expression Tree ШЁШұШ§ЫҢ ШҜШіШӘШұШіЫҢ ШЁЩҮ Ш®Ш§ШөЫҢШӘ Age ШіШ§Ш®ШӘЩҮ ШҙШҜЩҮ ЩҲ ШіЩҫШі ШЁЩҮ ЫҢЪ© Delegate Ъ©Ш§Щ…ЩҫШ§ЫҢЩ„ Щ…ЫҢвҖҢШҙЩҲШҜ. ЩҶШӘЫҢШ¬ЩҮШҢ ЫҢЪ© ЩҒШұШ§Ш®ЩҲШ§ЩҶЫҢ ШіШұЫҢШ№ ЩҲ ШЁЩҮЫҢЩҶЩҮ Ш§ШіШӘ Ъ©ЩҮ ЩҶЫҢШ§ШІЫҢ ШЁЩҮ Ш¬ШіШӘШ¬ЩҲЫҢ Щ…Ъ©ШұШұ Щ…ШӘШ§ШҜЫҢШӘШ§ ЩҶШҜШ§ШұШҜ. Ш§ЫҢЩҶ ШұЩҲШҙ ШЁЩҮвҖҢЩҲЫҢЪҳЩҮ ШІЩ…Ш§ЩҶЫҢ Щ…ЩҒЫҢШҜ Ш§ШіШӘ Ъ©ЩҮ ШЁШ®ЩҲШ§ЩҮЫҢШҜ ШҜШіШӘШұШіЫҢ ШЁЩҮ Ш®ШөЩҲШөЫҢШ§ШӘ ЫҢШ§ Щ…ШӘШҜЩҮШ§ ШұШ§ ШЁЩҮ ШөЩҲШұШӘ ЩҫЩҲЫҢШ§ ЩҲ ШЁШ§ Ш№Щ…Щ„Ъ©ШұШҜ ШЁШ§Щ„Ш§ Щ…ШҜЫҢШұЫҢШӘ Ъ©ЩҶЫҢШҜ.

5. Ш§ШіШӘЩҒШ§ШҜЩҮ Ш§ШІ Ш§ШЁШІШ§ШұЩҮШ§ЫҢ Щ…ШҜШұЩҶ ШҜШ§ШӘвҖҢЩҶШӘ
ШҜШұ ЩҶШіШ®ЩҮвҖҢЩҮШ§ЫҢ Ш¬ШҜЫҢШҜШӘШұ ШҜШ§ШӘвҖҢЩҶШӘШҢ Ш§ШЁШІШ§ШұЩҮШ§ЫҢЫҢ Щ…Ш§ЩҶЩҶШҜ Reflection.Emit ЩҲ Source Generators Щ…Ш№ШұЩҒЫҢ ШҙШҜЩҮвҖҢШ§ЩҶШҜ Ъ©ЩҮ Щ…ЫҢвҖҢШӘЩҲШ§ЩҶЩҶШҜ Ш¬Ш§ЫҢЪҜШІЫҢЩҶ Reflection ШҙЩҲЩҶШҜ ЫҢШ§ ШўЩҶ ШұШ§ ШЁЩҮЫҢЩҶЩҮвҖҢШӘШұ Ъ©ЩҶЩҶШҜ. Reflection.Emit ШЁЩҮ ШҙЩ…Ш§ Ш§Ш¬Ш§ШІЩҮ Щ…ЫҢвҖҢШҜЩҮШҜ Ъ©ШҜ IL (Intermediate Language) ШұШ§ ШҜШұ ШІЩ…Ш§ЩҶ Ш§Ш¬ШұШ§ ШӘЩҲЩ„ЫҢШҜ Ъ©ЩҶЫҢШҜШҢ Ъ©ЩҮ Ш№Щ…Щ„Ъ©ШұШҜ ШЁЩҮШӘШұЫҢ ЩҶШіШЁШӘ ШЁЩҮ Reflection Щ…Ш№Щ…ЩҲЩ„ЫҢ ШҜШ§ШұШҜ. Щ…Ш«Ш§Щ„ ШіШ§ШҜЩҮ ШІЫҢШұ Ш§ШіШӘЩҒШ§ШҜЩҮ Ш§ШІ Reflection.Emit ШұШ§ ЩҶШҙШ§ЩҶ Щ…ЫҢвҖҢШҜЩҮШҜ:
using System.Reflection.Emit;

public class EmitExample
{
    private static readonly DynamicMethod _dynamicMethod;

    static EmitExample()
    {
        _dynamicMethod = new DynamicMethod(
            "SetAgeDynamic",
            null,
            new[] { typeof(Person), typeof(int) },
            typeof(EmitExample).Module);

        ILGenerator il = _dynamicMethod.GetILGenerator();
        il.Emit(OpCodes.Ldarg_0); // Load Person instance
        il.Emit(OpCodes.Ldarg_1); // Load age value
        il.Emit(OpCodes.Callvirt, typeof(Person).GetMethod("SetAge"));
        il.Emit(OpCodes.Ret);

        _setAgeDynamic = (Action<Person, int>)_dynamicMethod.CreateDelegate(typeof(Action<Person, int>));
    }

    private static readonly Action<Person, int> _setAgeDynamic;

    public void SetAgeWithEmit(Person person, int age)
    {
        _setAgeDynamic(person, age);
    }
}
Ш§ЫҢЩҶ Ъ©ШҜ ЫҢЪ© Щ…ШӘШҜ ЩҫЩҲЫҢШ§ Ш§ЫҢШ¬Ш§ШҜ Щ…ЫҢвҖҢЪ©ЩҶШҜ Ъ©ЩҮ Щ…ШіШӘЩӮЫҢЩ…Ш§ЩӢ Щ…ШӘШҜ SetAge ШұШ§ ЩҒШұШ§Ш®ЩҲШ§ЩҶЫҢ Щ…ЫҢвҖҢЪ©ЩҶШҜ ЩҲ ЩҶЫҢШ§ШІЫҢ ШЁЩҮ Reflection ШҜШұ ШІЩ…Ш§ЩҶ Ш§Ш¬ШұШ§ ЩҶШҜШ§ШұШҜ. ЩҮЩ…ЪҶЩҶЫҢЩҶШҢ Source Generators ШҜШұ ШҜШ§ШӘвҖҢЩҶШӘ Щ…ЫҢвҖҢШӘЩҲШ§ЩҶЩҶШҜ Ъ©ШҜ Щ…ЩҲШұШҜ ЩҶЫҢШ§ШІ ШұШ§ ШҜШұ ШІЩ…Ш§ЩҶ Ъ©Ш§Щ…ЩҫШ§ЫҢЩ„ ШӘЩҲЩ„ЫҢШҜ Ъ©ЩҶЩҶШҜ ЩҲ ЩҲШ§ШЁШіШӘЪҜЫҢ ШЁЩҮ Reflection ШұШ§ ШЁЩҮ Ш·ЩҲШұ Ъ©Ш§Щ…Щ„ ШӯШ°ЩҒ Ъ©ЩҶЩҶШҜ.

Щ…Ш«Ш§Щ„ Ъ©Ш§ШұШЁШұШҜЫҢ: ШіШ§Ш®ШӘ ЫҢЪ© ЩҒШ№Ш§Щ„вҖҢШіШ§ШІ ЩҫЫҢШҙШұЩҒШӘЩҮ (Activator)
ШЁШұШ§ЫҢ ШҜШұЪ© ШЁЩҮШӘШұ Ш§ЫҢЩҶ ШӘЪ©ЩҶЫҢЪ©вҖҢЩҮШ§ШҢ ЫҢЪ© ЩҶЩ…ЩҲЩҶЩҮ Ш№Щ…Щ„ЫҢ Ш§ШІ ШіШ§Ш®ШӘ Ш§ШҙЫҢШ§ШЎ ШЁШ§ Reflection ШұШ§ ШЁШұШұШіЫҢ Щ…ЫҢвҖҢЪ©ЩҶЫҢЩ…. ЩҒШұШ¶ Ъ©ЩҶЫҢШҜ Щ…ЫҢвҖҢШ®ЩҲШ§ЩҮЫҢЩ… ЫҢЪ© ЩҶЩ…ЩҲЩҶЩҮ Ш§ШІ Ъ©Щ„Ш§Ші Person ШұШ§ ШЁШ§ Ш§ШіШӘЩҒШ§ШҜЩҮ Ш§ШІ ЩҫШ§ШұШ§Щ…ШӘШұЩҮШ§ЫҢ Щ…ШҙШ®Шө Ш§ЫҢШ¬Ш§ШҜ Ъ©ЩҶЫҢЩ…:
public class Person
{
    public int Id { get; set; }
    public string FirstName { get; set; }
    public string LastName { get; set; }

    public Person(int id, string firstName, string lastName)
    {
        Id = id;
        FirstName = firstName;
        LastName = lastName;
    }
}

public class AdvanceActivator
{
    private static readonly Dictionary<Type, Delegate> _constructors = new();

    public static T CreateInstance<T>(params object[] args)
    {
        var type = typeof(T);
        if (!_constructors.TryGetValue(type, out var constructor))
        {
            var ctor = type.GetConstructor(args.Select(a => a.GetType()).ToArray());
            constructor = Delegate.CreateDelegate(typeof(Func<object[], T>), ctor);
            _constructors[type] = constructor;
        }

        return ((Func<object[], T>)constructor)(args);
    }
}

// Ш§ШіШӘЩҒШ§ШҜЩҮ
Person person = AdvanceActivator.CreateInstance<Person>(10, "User", "Name");
Console.WriteLine($"{person.Id}, {person.FirstName}, {person.LastName}");
ШҜШұ Ш§ЫҢЩҶ Щ…Ш«Ш§Щ„ШҢ Ш§ШІ ЫҢЪ© ШҜЫҢЪ©ШҙЩҶШұЫҢ Ш§ШіШӘШ§ШӘЫҢЪ© ШЁШұШ§ЫҢ Ш°Ш®ЫҢШұЩҮвҖҢШіШ§ШІЫҢ DelegateЩҮШ§ЫҢ ШіШ§ШІЩҶШҜЩҮ (Constructor) Ш§ШіШӘЩҒШ§ШҜЩҮ ШҙШҜЩҮ Ш§ШіШӘ. Ш§ЫҢЩҶ ШұЩҲШҙ ШӘШұЪ©ЫҢШЁЫҢ Ш§ШІ Ш°Ш®ЫҢШұЩҮвҖҢШіШ§ШІЫҢ (Ъ©Шҙ Ъ©ШұШҜЩҶ) ЩҲ DelegateЩҮШ§ ШұШ§ ШЁЩҮ Ъ©Ш§Шұ Щ…ЫҢвҖҢШЁШұШҜ ШӘШ§ Ш№Щ…Щ„Ъ©ШұШҜ ШұШ§ ШЁЩҮ ШӯШҜШ§Ъ©Ш«Шұ ШЁШұШіШ§ЩҶШҜ.

ЪҶШ§Щ„ШҙвҖҢЩҮШ§ ЩҲ Щ…ШӯШҜЩҲШҜЫҢШӘвҖҢЩҮШ§
ШЁШ§ ЩҲШ¬ЩҲШҜ Щ…ШІШ§ЫҢШ§ЫҢ ReflectionШҢ ШЁШ§ЫҢШҜ ШЁЩҮ Щ…ШӯШҜЩҲШҜЫҢШӘвҖҢЩҮШ§ЫҢ ШўЩҶ ЩҶЫҢШІ ШӘЩҲШ¬ЩҮ Ъ©ШұШҜ. Ш§ШіШӘЩҒШ§ШҜЩҮ ШЁЫҢШҙ Ш§ШІ ШӯШҜ Ш§ШІ ШўЩҶ Щ…ЫҢвҖҢШӘЩҲШ§ЩҶШҜ Ъ©ШҜ ШұШ§ ЩҫЫҢЪҶЫҢШҜЩҮвҖҢШӘШұ Ъ©ЩҶШҜ ЩҲ Ш®ЩҲШ§ЩҶШ§ЫҢЫҢ (Readability) ШұШ§ Ъ©Ш§ЩҮШҙ ШҜЩҮШҜ. ЩҮЩ…ЪҶЩҶЫҢЩҶШҢ ШҜШұ Щ…ШӯЫҢШ·вҖҢЩҮШ§ЫҢЫҢ Ъ©ЩҮ Ш§Щ…ЩҶЫҢШӘ Ш§ЩҲЩ„ЩҲЫҢШӘ ШҜШ§ШұШҜШҢ Reflection Щ…Щ…Ъ©ЩҶ Ш§ШіШӘ ШЁЩҮ ШҜЩ„ЫҢЩ„ ШҜШіШӘШұШіЫҢ ШЁЩҮ Ш§Ш№Ш¶Ш§ЫҢ Ш®ШөЩҲШөЫҢ (Private Members) ШұЫҢШіЪ©вҖҢЩҮШ§ЫҢЫҢ Ш§ЫҢШ¬Ш§ШҜ Ъ©ЩҶШҜ. ШЁЩҶШ§ШЁШұШ§ЫҢЩҶШҢ ЩҮЩ…ЫҢШҙЩҮ ШЁШ§ЫҢШҜ ШӘШ№Ш§ШҜЩ„ЫҢ ШЁЫҢЩҶ Ш§ЩҶШ№Ш·Ш§ЩҒвҖҢЩҫШ°ЫҢШұЫҢ ЩҲ ШіШ§ШҜЪҜЫҢ ШЁШұЩӮШұШ§Шұ ШҙЩҲШҜ.

ЩҶШӘЫҢШ¬ЩҮвҖҢЪҜЫҢШұЫҢ
Reflection ШҜШұ C# Ш§ШЁШІШ§ШұЫҢ ЩӮШҜШұШӘЩ…ЩҶШҜ Ш§ШіШӘ Ъ©ЩҮ Ш§ЩҶШ№Ш·Ш§ЩҒвҖҢЩҫШ°ЫҢШұЫҢ ШЁШ§Щ„Ш§ЫҢЫҢ ШЁЩҮ ШЁШұЩҶШ§Щ…ЩҮвҖҢЩҶЩҲЫҢШіШ§ЩҶ ШҜШ§ШӘвҖҢЩҶШӘ Ш§ШұШ§ШҰЩҮ Щ…ЫҢвҖҢШҜЩҮШҜШҢ Ш§Щ…Ш§ Ш§ШіШӘЩҒШ§ШҜЩҮ ЩҶШ§ШҜШұШіШӘ Ш§ШІ ШўЩҶ Щ…ЫҢвҖҢШӘЩҲШ§ЩҶШҜ ШЁЩҮ Ъ©Ш§ЩҮШҙ Ш№Щ…Щ„Ъ©ШұШҜ Щ…ЩҶШ¬Шұ ШҙЩҲШҜ. ШЁШ§ ШұШ№Ш§ЫҢШӘ ЩҶЪ©Ш§ШӘЫҢ Щ…Ш§ЩҶЩҶШҜ Ъ©Ш§ЩҮШҙ Ш§ШіШӘЩҒШ§ШҜЩҮ ШәЫҢШұШ¶ШұЩҲШұЫҢШҢ Ш°Ш®ЫҢШұЩҮвҖҢШіШ§ШІЫҢ ЩҶШӘШ§ЫҢШ¬ШҢ ШЁЩҮШұЩҮвҖҢЪҜЫҢШұЫҢ Ш§ШІ DelegateЩҮШ§ШҢ Ш§ШіШӘЩҒШ§ШҜЩҮ Ш§ШІ Expression TreesШҢ ЩҲ Ш§ШЁШІШ§ШұЩҮШ§ЫҢ Щ…ШҜШұЩҶ Щ…Ш§ЩҶЩҶШҜ Reflection.Emit ЫҢШ§ Source GeneratorsШҢ Щ…ЫҢвҖҢШӘЩҲШ§ЩҶ Ъ©Ш§ШұШ§ЫҢЫҢ ШЁШұЩҶШ§Щ…ЩҮ ШұШ§ ШЁЩҮ Ш·ЩҲШұ ЩӮШ§ШЁЩ„вҖҢШӘЩҲШ¬ЩҮЫҢ ШЁЩҮШЁЩҲШҜ ШҜШ§ШҜ. Щ…Ш«Ш§Щ„вҖҢЩҮШ§ЫҢ Ш§ШұШ§ШҰЩҮвҖҢШҙШҜЩҮ ШҜШұ Ш§ЫҢЩҶ Щ…ЩӮШ§Щ„ЩҮ ЩҶШҙШ§ЩҶ Щ…ЫҢвҖҢШҜЩҮЩҶШҜ Ъ©ЩҮ ЪҶЪҜЩҲЩҶЩҮ Щ…ЫҢвҖҢШӘЩҲШ§ЩҶ Ш§ЫҢЩҶ ШӘЪ©ЩҶЫҢЪ©вҖҢЩҮШ§ ШұШ§ ШҜШұ Ш№Щ…Щ„ ЩҫЫҢШ§ШҜЩҮвҖҢШіШ§ШІЫҢ Ъ©ШұШҜ.
