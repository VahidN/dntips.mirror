‫DotnetGeminiSDK: SDK.NET؛ برای ادغام یکپارچه و کارآمد با Google Gemini


نویسنده: وحید نصیری
تاریخ: ۱۴۰۳/۱۲/۲۶ ۱۳:۳۹
آدرس: www.dntips.ir
این SDK (.NET SDK) ابزاری برای توسعه‌دهندگان است تا بتوانند به راحتی از API (.NET API) گوگل جمنای (Google Gemini) در پروژه‌های .NET خود استفاده کنند.

ویژگی‌های پیاده‌سازی شده
  • درخواست متن (Text Prompt): ارسال درخواست‌های متنی به API جمنای.
  • درخواست متن جریانی (Stream Text Prompt): دریافت پاسخ‌های متنی به صورت جریانی و تکه تکه.
  • درخواست متن چندگانه (Multiple Text Prompt): ارسال چندین درخواست متنی به صورت همزمان.
  • درخواست تصویر (Image Prompt): ارسال درخواست‌های تصویری به همراه متن.
  • شمارش توکن‌ها (Counting Tokens): شمارش تعداد توکن‌های استفاده شده در یک درخواست.
  • دریافت مدل (Get Model): دریافت جزئیات مدل خاص جمنای.
  • لیست مدل‌ها (List Models): دریافت لیست تمام مدل‌های جمنای.
  • تعبیه (Embedding): استفاده از تکنیک تعبیه برای پردازش متن.
  • تعبیه دسته‌ای (Batch Embedding): استفاده از تکنیک تعبیه دسته‌ای برای پردازش متن.
