‫کوئری نویسی SQLite به کمک LLMs


نویسنده: وحید نصیری
تاریخ: ۱۴۰۴/۰۲/۳۰ ۱۰:۱۹
آدرس: www.dntips.ir
LiteCLI، یک کلاینت (Client) خط فرمان (Command Line Interface - CLI) برای SQLite، با افزودن قابلیت مدل زبانی بزرگ (Large Language Model - LLM)، ارتقا یافته است تا به کاربران در نوشتن کوئری‌های (Query) SQL کمک کند. این ارتقا، تجربه کاربری را با استفاده از مدل‌های پایه مانند OpenAI (از طریق کلید API)، Ollama یا Llamafile بهبود می‌بخشد.

یکپارچه‌سازی LLM
برای استفاده از این قابلیت، ابتدا باید افزونه (Extension) llm را از طریق دستور \llm در اعلان LiteCLI نصب کرد. پس از نصب، کاربران می‌توانند سوالات خود را به زبان طبیعی در مورد پایگاه داده (Database) بپرسند. سوال به همراه فراداده (Metadata) پایگاه داده به مدل LLM ارسال می‌شود. سپس، LLM کوئری SQL مربوطه را تولید و به اعلان LiteCLI برمی‌گرداند تا اجرا شود.

مثال کاربردی
در مستندات رسمی، مثالی از پرس و جو در پایگاه داده تاریخچه مرورگر کروم (Chrome) ارائه شده است. در این مثال، کاربر با استفاده از زبان طبیعی، پنج آدرس اینترنتی (URL) برتر را درخواست می‌کند و LLM آن را به کوئری SQL تبدیل می‌کند.

نکات کلیدی
  • ابزارهای CLI همچنان با وجود رابط‌های کاربری گرافیکی (Graphical User Interface - GUI) محبوبیت خود را حفظ کرده‌اند.
  • راهکارهای تبدیل متن به SQL (Text-to-SQL) در حال افزایش هستند.
