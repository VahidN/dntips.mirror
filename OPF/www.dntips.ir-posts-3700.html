Щ…Ш·Ш§Щ„ШЁ ЫіЫ·Ы°Ы°
ЩҶЩҲЫҢШіЩҶШҜЪҜШ§ЩҶ ЫІЫ·Ы¶
ЪҜШұЩҲЩҮвҖҢЩҮШ§ЫҢ Щ…Ш·Ш§Щ„ШЁ ЫұЫ°ЫІЫө
ЩҶЩӮШҙЩҮвҖҢЩҮШ§ЫҢ ШұШ§ЩҮ ЫұЫұЫ№
ШҜЩҲШұЩҮвҖҢЩҮШ§ ЫұЫҙ
Ш§ШҙШӘШұШ§Ъ©вҖҢЩҮШ§ ЫұЫ·Ы№ЫұЫ№
→ вҖ«ШӘШ§ШұЫҢШ®ЪҶЩҮ ШӘШӯЩҲЩ„ Щ…Ъ©Ш§ЩҶЫҢШІЩ…вҖҢЩҮШ§ЫҢ ШӘЩҲЩ„ЫҢШҜ Ш§Ш№ШҜШ§ШҜ ШӘШөШ§ШҜЩҒЫҢ ШӘШ§ ШҜШ§ШӘвҖҢЩҶШӘ ЫұЫұ
вҖ«Ш·ШұШ§ШӯЫҢ ШіЫҢШіШӘЩ…Ш§ШӘЫҢЪ© Ш®ШөЫҢШөЩҮвҖҢЩҮШ§ЫҢ ШӘШ§ШұЫҢШ®ЪҶЩҮвҖҢШҜШ§Шұ (Historized Attributes) ШҜШұ EF Core ←
Ш№ЩҶЩҲШ§ЩҶ:

вҖ«ЩҶЪҜШ§ЩҮЫҢ Ш¬Ш§Щ…Ш№ ШЁЩҮ Щ…ШҜЫҢШұЫҢШӘ MIME TypeЩҮШ§ ШҜШұ NET 11.


ЩҶЩҲЫҢШіЩҶШҜЩҮ: ЩҲШӯЫҢШҜ ЩҶШөЫҢШұЫҢ
ШӘШ§ШұЫҢШ®: ЫұЫҙЫ°Ыө/Ы°Ыҙ/Ы°Ыё ЫұЫ°:ЫұЫ°
ШўШҜШұШі: www.dntips.ir
ШҜШұ ШӘЩҲШіШ№ЩҮ ЩҶШұЩ…вҖҢШ§ЩҒШІШ§ШұЩҮШ§ЫҢ ШӘШӯШӘ ЩҲШЁ ЩҲ ШіЫҢШіШӘЩ…вҖҢЩҮШ§ЫҢ Щ…ШҜЫҢШұЫҢШӘ ЩҒШ§ЫҢЩ„ШҢ ШӘШ№Ш§Щ…Щ„ ШЁШ§ ЩҫШіЩҲЩҶШҜ ЩҒШ§ЫҢЩ„вҖҢЩҮШ§ ЩҲ ЩҶЪҜШ§ШҙШӘ ШўЩҶвҖҢЩҮШ§ ШЁЩҮ Ш§ШіШӘШ§ЩҶШҜШ§ШұШҜ MIME Types (ЫҢШ§ ЩҮЩ…Ш§ЩҶ ШҙЩҶШ§ШіЩҮ ШұШіШ§ЩҶЩҮ) ЫҢЪ© ЩҶЫҢШ§ШІ ЩҮЩ…ЫҢШҙЪҜЫҢ Ш§ШіШӘ. ШЁШұШ§ЫҢ Щ…Ш«Ш§Щ„ШҢ ЩҮЩҶЪҜШ§Щ… ШўЩҫЩ„ЩҲШҜ ЩҒШ§ЫҢЩ„ШҢ ШҜШ§ЩҶЩ„ЩҲШҜ Щ…ШіШӘЩҶШҜШ§ШӘ ЫҢШ§ ШӘЩҶШёЫҢЩ… ЩҮШҜШұЩҮШ§ЫҢ ЩҫШ§ШіШ® HTTPШҢ ШіЫҢШіШӘЩ… ШЁШ§ЫҢШҜ ШЁШҜШ§ЩҶШҜ Ъ©ЩҮ ЫҢЪ© ЩҫШіЩҲЩҶШҜ Щ…Ш§ЩҶЩҶШҜ .xlsx ШЁШ§ ЪҶЩҮ ЩҒШұЩ…ШӘЫҢ (Щ…Ш§ЩҶЩҶШҜ application/vnd.openxmlformats-officedocument.spreadsheetml.sheet) ШҙЩҶШ§Ш®ШӘЩҮ Щ…ЫҢвҖҢШҙЩҲШҜ.
ШӘШ§ ЩҫЫҢШҙ Ш§ШІ ШҜШ§ШӘвҖҢЩҶШӘ ЫұЫұШҢ ШұШ§ЩҮЪ©Ш§Шұ ШЁЩҲЩ…ЫҢ ЩҲ ШіШұШұШ§ШіШӘЫҢ ШЁШұШ§ЫҢ ЩҶЪҜШ§ШҙШӘ ШҜЩҲШ·ШұЩҒЩҮ Ш§ЫҢЩҶ Ш№ЩҶШ§ЩҲЫҢЩҶ ЩҲШ¬ЩҲШҜ ЩҶШҜШ§ШҙШӘ. ШӘЩҲШіШ№ЩҮвҖҢШҜЩҮЩҶШҜЪҜШ§ЩҶ Щ…Ш№Щ…ЩҲЩ„Ш§ЩӢ Щ…Ш¬ШЁЩҲШұ ШЁЩҲШҜЩҶШҜ ШЁЩҮ ШіШұШ§Шә ШіШ§Ш®ШӘШ§ШұЩҮШ§ЫҢ ШӯШ¬ЫҢЩ… switch-case ШЁШұЩҲЩҶШҜШҢ ШҜЫҢЪ©ШҙЩҶШұЫҢвҖҢЩҮШ§ЫҢ ШҙШ®ШөЫҢ ЩҶЪҜЩҮШҜШ§ШұЫҢ Ъ©ЩҶЩҶШҜ ЫҢШ§ ШЁЩҮ Ъ©ШӘШ§ШЁШ®Ш§ЩҶЩҮвҖҢЩҮШ§ЫҢ ШҙШ®Шө Ш«Ш§Щ„Ш« (Nuget Packages) Щ…ШӘЪ©ЫҢ ШҙЩҲЩҶШҜ. ШҜШұ Ш§Ъ©ЩҲШіЫҢШіШӘЩ… ШҜШ§ШӘвҖҢЩҶШӘ ЫұЫұШҢ Щ…Ш§ЫҢЪ©ШұЩҲШіШ§ЩҒШӘ ШЁШ§ Щ…Ш№ШұЩҒЫҢ Ъ©Щ„Ш§Ші Ъ©Ш§ШұШЁШұШҜЫҢ MediaTypeMap ШҜШұ ЩҒШ¶Ш§ЫҢ ЩҶШ§Щ… System.Net.MimeШҢ Ш§ЫҢЩҶ ЪҶШ§Щ„Шҙ ШұШ§ ШЁЩҮ ШөЩҲШұШӘ ШұЫҢШҙЩҮвҖҢШ§ЫҢ ЩҲ ШЁЩҮЫҢЩҶЩҮвҖҢШіШ§ШІЫҢвҖҢШҙШҜЩҮ ШӯЩ„ Ъ©ШұШҜЩҮ Ш§ШіШӘ. ШҜШұ Ш§ЫҢЩҶ Щ…ЩӮШ§Щ„ЩҮ ШЁЩҮ ШЁШұШұШіЫҢ ЩӮШ§ШЁЩ„ЫҢШӘвҖҢЩҮШ§ЫҢ Ш¬ШҜЫҢШҜ ШҜШ§ШӘвҖҢЩҶШӘ ЫұЫұ ШҜШұ Щ…ШҜЫҢШұЫҢШӘ Ш§ЫҢЩҶ ШіШ§Ш®ШӘШ§ШұЩҮШ§ ЩҲ ШұЩҲШҙвҖҢЩҮШ§ЫҢ ШЁЩҮШЁЩҲШҜ Ъ©ЫҢЩҒЫҢШӘ Ъ©ШҜ ШҜШұ Ш§ЫҢЩҶ ШӯЩҲШІЩҮ Щ…ЫҢвҖҢЩҫШұШҜШ§ШІЫҢЩ….

Ыұ. ЪҶШ§Щ„Шҙ ШұШҙШӘЩҮвҖҢЩҮШ§ЫҢ ЩҮШ§ШұШҜЪ©ШҜ ШҙШҜЩҮ ЩҲ ШұШ§ЩҮвҖҢШӯЩ„ Ш§ЩҲЩ„: Ш§ШіШӘЩҒШ§ШҜЩҮ Ш§ШІ Ш«Ш§ШЁШӘвҖҢЩҮШ§ (Constants)
ЫҢЪ©ЫҢ Ш§ШІ ШұШ§ЫҢШ¬вҖҢШӘШұЫҢЩҶ Ш®Ш·Ш§ЩҮШ§ ШҜШұ ЩҮЩҶЪҜШ§Щ… ШӘЩҲШіШ№ЩҮ ШіЫҢШіШӘЩ…вҖҢЩҮШ§ЫҢ Ш§ШұШӘШЁШ§Ш·ЫҢШҢ Ш®Ш·Ш§ЫҢ ЩҶЩҲШҙШӘШ§ШұЫҢ (Typo) ШҜШұ ШІЩ…Ш§ЩҶ Ш§ШіШӘЩҒШ§ШҜЩҮ Ш§ШІ ЩҮШҜШұЩҮШ§ЫҢ Щ…ШӘЩҶЫҢ Ш§ШіШӘ. ШЁЩҮ Ъ©ШҜ ШІЫҢШұ ЩҶЪҜШ§ЩҮ Ъ©ЩҶЫҢШҜ:
var client = new HttpClient();
client.DefaultRequestHeaders.Accept.Add(new MediaTypeWithQualityHeaderValue("appliction/json")); 
// Ш®Ш·Ш§ЫҢ Ш§Щ…Щ„Ш§ЫҢЫҢ ШҜШұ Ъ©Щ„Щ…ЩҮ application Ъ©ЩҮ ШЁШ§Ш№Ш« Ш§Ш®ШӘЩ„Ш§Щ„ ШҜШұ Ш§ШұШӘШЁШ§Ш· API Щ…ЫҢвҖҢШҙЩҲШҜ
Ш§ЫҢЩҶ Ш®Ш·Ш§ЫҢ Ъ©ЩҲЪҶЪ© Щ…Щ…Ъ©ЩҶ Ш§ШіШӘ ШіШ§Ш№ШӘвҖҢЩҮШ§ ЩҲЩӮШӘ ШӘЩҲШіШ№ЩҮвҖҢШҜЩҮЩҶШҜЩҮ ШұШ§ ШЁШұШ§ЫҢ Ш№ЫҢШЁвҖҢЫҢШ§ШЁЫҢ (Debug) ШӘЩ„ЩҒ Ъ©ЩҶШҜ. Ш№Щ„Ш§ЩҲЩҮ ШЁШұ Ш§ЫҢЩҶШҢ ШҜШұ ШөЩҲШұШӘ ЩҶЫҢШ§ШІ ШЁЩҮ ШӘШәЫҢЫҢШұ ЫҢЪ© Ш§ШіШӘШ§ЩҶШҜШ§ШұШҜ ШұШіШ§ЩҶЩҮШҢ ШЁШ§ШІЩҶЩҲЫҢШіЫҢ ШұШҙШӘЩҮвҖҢЩҮШ§ ШҜШұ ШіШұШ§ШіШұ ЩҫШұЩҲЪҳЩҮ ШЁШіЫҢШ§Шұ ШІЩ…Ш§ЩҶвҖҢШЁШұ ЩҲ Щ…ШіШӘШ№ШҜ Ш®Ш·Ш§ Ш®ЩҲШ§ЩҮШҜ ШЁЩҲШҜ.
ШұШ§ЩҮвҖҢШӯЩ„ Ш§ШөЩҲЩ„ЫҢ Ш§ЫҢЩҶ Щ…ШҙЪ©Щ„ШҢ Ш§ШіШӘЩҒШ§ШҜЩҮ Ш§ШІ Ш«Ш§ШЁШӘвҖҢЩҮШ§ЫҢ Ш§ШІ ЩҫЫҢШҙ ШӘШ№ШұЫҢЩҒвҖҢШҙШҜЩҮ ШҜШұ Ъ©Щ„Ш§Ші MediaTypeNames Ш§ШіШӘ Ъ©ЩҮ ШҜШұ ЩҒШ¶Ш§ЫҢ ЩҶШ§Щ… System.Net.Mime ЩӮШұШ§Шұ ШҜШ§ШұШҜ. Ш§ЫҢЩҶ Ъ©Щ„Ш§Ші ШіШ§Ш®ШӘШ§ШұЫҢ ШҜШұШ®ШӘвҖҢЩҲШ§ШұЩҮ ЩҲ Ш§ШіШӘШ§ЩҶШҜШ§ШұШҜ ШұШ§ ШЁШұШ§ЫҢ ШҜШіШӘШұШіЫҢ ШЁЩҮ Ш§ЩҶЩҲШ§Ш№ Щ…ШӯШӘЩҲШ§ ЩҒШұШ§ЩҮЩ… Щ…ЫҢвҖҢЪ©ЩҶШҜ:
var client = new HttpClient();
client.DefaultRequestHeaders.Accept.Add(new MediaTypeWithQualityHeaderValue(MediaTypeNames.Application.Json));
Ш§ЫҢЩҶ Ш§Щ„ЪҜЩҲ ШЁШұШ§ЫҢ ШӘЩ…Ш§Щ… ШҜШіШӘЩҮвҖҢШЁЩҶШҜЫҢвҖҢЩҮШ§ЫҢ Ш§ШіШӘШ§ЩҶШҜШ§ШұШҜ Ш§ШІ Ш¬Щ…Щ„ЩҮ ЩҒЩҲЩҶШӘвҖҢЩҮШ§ (Font)ШҢ ШӘШөШ§ЩҲЫҢШұ (Image)ШҢ Щ…ШӘЩҶвҖҢЩҮШ§ (Text) ЩҲ ШЁШ®ШҙвҖҢЩҮШ§ЫҢ ЪҶЩҶШҜЪҜШ§ЩҶЩҮ (Multipart) ЩҶЫҢШІ ШөШ§ШҜЩӮ Ш§ШіШӘ. ШЁЩҮ Ш№ЩҶЩҲШ§ЩҶ Щ…Ш«Ш§Щ„ШҢ ШЁШұШ§ЫҢ ШҜШіШӘШұШіЫҢ ШЁЩҮ ЩҒШұЩ…ШӘ ЩҒШ§ЫҢЩ„вҖҢЩҮШ§ЫҢ CSV Щ…ЫҢвҖҢШӘЩҲШ§ЩҶЫҢШҜ Ш§ШІ Ш«Ш§ШЁШӘ MediaTypeNames.Text.Csv Ш§ШіШӘЩҒШ§ШҜЩҮ Ъ©ЩҶЫҢШҜ.

ЫІ. ЩҶЪҜШ§ШҙШӘ ЩҫШіЩҲЩҶШҜ ЩҒШ§ЫҢЩ„ ШЁЩҮ MIME Type ШЁШ§ MediaTypeMap
Ш§ЪҜШұЪҶЩҮ Ъ©Щ„Ш§Ші MediaTypeNames Щ…ШҙЪ©Щ„ ШұШҙШӘЩҮвҖҢЩҮШ§ЫҢ Ш«Ш§ШЁШӘ ШұШ§ ШӯЩ„ Щ…ЫҢвҖҢЪ©ЩҶШҜШҢ Ш§Щ…Ш§ ШҜШұ ШіЩҶШ§ШұЫҢЩҲЩҮШ§ЫҢ ЩҫЩҲЫҢШ§ (Dynamic) - Ш¬Ш§ЫҢЫҢ Ъ©ЩҮ ЩҫШіЩҲЩҶШҜ ЩҒШ§ЫҢЩ„ ШұШ§ ШҜШұ ШІЩ…Ш§ЩҶ Ш§Ш¬ШұШ§ Ш§ШІ Ъ©Ш§ШұШЁШұ ШҜШұЫҢШ§ЩҒШӘ Щ…ЫҢвҖҢЪ©ЩҶЫҢЩ… ЩҲ Щ…Ш§ЫҢЩ„ЫҢЩ… MIME Type ШўЩҶ ШұШ§ ШЁЫҢШ§ШЁЫҢЩ… - Ъ©Ш§ШұШ§ЫҢЫҢ ЩҶШҜШ§ШұШҜ. ШҜШұ ЪҜШ°ШҙШӘЩҮ ЩҶШ§ЪҶШ§Шұ ШЁЩҮ ЩҶЩҲШҙШӘЩҶ Ъ©ШҜЩҮШ§ЫҢЫҢ ШҙШЁЫҢЩҮ ШЁЩҮ Ш§ЫҢЩҶ ШЁЩҲШҜЫҢЩ…:
string extension = ".pdf";
string mimeType = extension switch
{
    ".pdf" => "application/pdf",
    ".txt" => "text/plain",
    _ => "application/octet-stream"
};
ШҜШұ ШҜШ§ШӘвҖҢЩҶШӘ ЫұЫұШҢ Ш§ЫҢЩҶ ЩҒШұШўЫҢЩҶШҜ ШЁШ§ Щ…ШӘШҜ GetMediaType Ш§ШІ Ъ©Щ„Ш§Ші Ш¬ШҜЫҢШҜ MediaTypeMap ШЁЩҮ ШіШ§ШҜЩҮвҖҢШӘШұЫҢЩҶ ШҙЪ©Щ„ Щ…Щ…Ъ©ЩҶ ШӘШәЫҢЫҢШұ ЫҢШ§ЩҒШӘЩҮ Ш§ШіШӘ. Ш§ЫҢЩҶ Щ…ШӘШҜ ЩҮЩҲШҙЩ…ЩҶШҜ Ш§ШіШӘ ЩҲ Щ…ЫҢвҖҢШӘЩҲШ§ЩҶШҜ ЩҲШұЩҲШҜЫҢвҖҢЩҮШ§ ШұШ§ ШЁШ§ ЩҒШұЩ…ШӘвҖҢЩҮШ§ЫҢ Щ…Ш®ШӘЩ„ЩҒ (ЩҮЩ…ШұШ§ЩҮ ШЁШ§ ЩҶЩӮШ·ЩҮШҢ ШЁШҜЩҲЩҶ ЩҶЩӮШ·ЩҮ ЫҢШ§ ШӯШӘЫҢ ЩҶШ§Щ… Ъ©Ш§Щ…Щ„ ЩҒШ§ЫҢЩ„) ШЁЩҮ ШҜШұШіШӘЫҢ ЩҫШұШҜШ§ШІШҙ Ъ©ЩҶШҜ:
// ЩҮШұ ШіЩҮ ШҜШіШӘЩҲШұ ШІЫҢШұ Ш®ШұЩҲШ¬ЫҢ ЫҢЪ©ШіШ§ЩҶ "text/plain" ШұШ§ ШӘЩҲЩ„ЫҢШҜ Щ…ЫҢвҖҢЪ©ЩҶЩҶШҜ:
var res1 = MediaTypeMap.GetMediaType(".txt");
var res2 = MediaTypeMap.GetMediaType("txt");
var res3 = MediaTypeMap.GetMediaType("document.txt");
Щ…Ш«Ш§Щ„вҖҢЩҮШ§ЫҢ ШЁЫҢШҙШӘШұ Ш¬ЩҮШӘ ШЁШұШұШіЫҢ ШұЩҒШӘШ§Шұ Щ…ШӘШҜ:
Console.WriteLine(MediaTypeMap.GetMediaType(".pdf"));  // Output: application/pdf
Console.WriteLine(MediaTypeMap.GetMediaType(".mp4"));  // Output: video/mp4
Console.WriteLine(MediaTypeMap.GetMediaType(".dsf"));  // Output: null (ЩҫШіЩҲЩҶШҜ ЩҶШ§ШҙЩҶШ§Ш®ШӘЩҮ)
ЩҶЪ©ШӘЩҮ ЩҒЩҶЫҢ ЩҲ Ш§Щ…ЩҶЫҢШӘЫҢ (Security & Fallback):
Ш§ЪҜШұ Щ…ШӘШҜ GetMediaType ЩҶШӘЩҲШ§ЩҶШҜ ЩҫШіЩҲЩҶШҜЫҢ ШұШ§ ШҙЩҶШ§ШіШ§ЫҢЫҢ Ъ©ЩҶШҜШҢ Щ…ЩӮШҜШ§Шұ null ШЁШұЩ…ЫҢвҖҢЪҜШұШҜШ§ЩҶШҜ. ЩҫЫҢШҙЩҶЩҮШ§ШҜ Щ…ЫҢвҖҢШҙЩҲШҜ ШҜШұ ШЁШұЩҶШ§Щ…ЩҮвҖҢЩҮШ§ЫҢ Ш№Щ…Щ„ЫҢШ§ШӘЫҢ ЩҮЩ…ЩҲШ§ШұЩҮ Ш§ШІ ЫҢЪ© Щ…ЩӮШҜШ§Шұ ЩҫЫҢШҙвҖҢЩҒШұШ¶ Ш§Щ…ЩҶ (Щ…Ш§ЩҶЩҶШҜ application/octet-stream ШЁШұШ§ЫҢ ЩҒШ§ЫҢЩ„вҖҢЩҮШ§ЫҢ ШЁШ§ЫҢЩҶШұЫҢ ЩҶШ§ШҙЩҶШ§Ш®ШӘЩҮ) ШЁЩҮ Ш№ЩҶЩҲШ§ЩҶ Fallback Ш§ШіШӘЩҒШ§ШҜЩҮ Ъ©ЩҶЫҢШҜ ШӘШ§ Ш§ШІ ШЁШұЩҲШІ ШұЩҒШӘШ§ШұЩҮШ§ЫҢ ШәЫҢШұЩ…ЩҶШӘШёШұЩҮ Ш¬Щ„ЩҲЪҜЫҢШұЫҢ ШҙЩҲШҜ:
var mediaType = MediaTypeMap.GetMediaType(fileName) ?? "application/octet-stream";

Ыі. Ш№Щ…Щ„ЫҢШ§ШӘ Щ…Ш№Ъ©ЩҲШі: ШҜШұЫҢШ§ЩҒШӘ ЩҫШіЩҲЩҶШҜ Ш§ШІ ШұЩҲЫҢ MIME Type
ЪҜШ§ЩҮ ШіЩҶШ§ШұЫҢЩҲ Ъ©Ш§Щ…Щ„Ш§ЩӢ ШЁШұШ№Ъ©Ші Ш§ШіШӘШӣ ШЁШұШ§ЫҢ Щ…Ш«Ш§Щ„ШҢ ШҙЩ…Ш§ ЫҢЪ© ЩҒШ§ЫҢЩ„ ШұШ§ Ш§ШІ Ш·ШұЫҢЩӮ ЫҢЪ© API ЫҢШ§ ЩҲШЁвҖҢШіШұЩҲЫҢШі ШҜШұЫҢШ§ЩҒШӘ Ъ©ШұШҜЩҮвҖҢШ§ЫҢШҜ ЩҲ ЩҮШҜШұ Content-Type ШЁЩҮ ШҙЩ…Ш§ Щ…ЫҢвҖҢЪҜЩҲЫҢШҜ Ъ©ЩҮ ЩҶЩҲШ№ Щ…ШӯШӘЩҲШ§ application/vnd.openxmlformats-officedocument.spreadsheetml.sheet Ш§ШіШӘШҢ Ш§Щ…Ш§ ЩҒШ§ЫҢЩ„ ШҜШұЫҢШ§ЩҒШӘЫҢ ЩҒШ§ЩӮШҜ ЩҶШ§Щ… ЩҲ ЩҫШіЩҲЩҶШҜ Щ…ШҙШ®Шө Ш§ШіШӘ. ШҜШұ Ш§ЫҢЩҶ ШӯШ§Щ„ШӘ ШҙЩ…Ш§ ШЁШ§ЫҢШҜ ШЁШҜШ§ЩҶЫҢШҜ ЪҶЩҮ ЩҫШіЩҲЩҶШҜЫҢ ШұШ§ ШЁШұШ§ЫҢ Ш°Ш®ЫҢШұЩҮвҖҢШіШ§ШІЫҢ ШөШӯЫҢШӯ ЩҒШ§ЫҢЩ„ ШұЩҲЫҢ ШҜЫҢШіЪ© Ш§ЩҶШӘШ®Ш§ШЁ Ъ©ЩҶЫҢШҜ.
ШҜШ§ШӘвҖҢЩҶШӘ ЫұЫұ Ш§ЫҢЩҶ ЩҶЫҢШ§ШІ ШұШ§ Ш§ШІ Ш·ШұЫҢЩӮ Щ…ШӘШҜ GetExtension ШҜШұ Ъ©Щ„Ш§Ші MediaTypeMap ШЁШұШ·ШұЩҒ ШіШ§Ш®ШӘЩҮ Ш§ШіШӘ:
var extension = MediaTypeMap.GetExtension("application/vnd.openxmlformats-officedocument.spreadsheetml.sheet");
Console.WriteLine(extension); // Output: .xlsx
ШҜШұ ШөЩҲШұШӘЫҢ Ъ©ЩҮ MIME Type Ш§ШұШіШ§Щ„ЫҢ ШҜШұ ЩҫШ§ЫҢЪҜШ§ЩҮ ШҜШ§ШҜЩҮ ШҜШ§Ш®Щ„ЫҢ ШҜШ§ШӘвҖҢЩҶШӘ ЩҲШ¬ЩҲШҜ ЩҶШҜШ§ШҙШӘЩҮ ШЁШ§ШҙШҜ ЫҢШ§ ШіШ§Ш®ШӘШ§Шұ ШәЫҢШұШ§ШіШӘШ§ЩҶШҜШ§ШұШҜЫҢ ШҜШ§ШҙШӘЩҮ ШЁШ§ШҙШҜШҢ Ш§ЫҢЩҶ Щ…ШӘШҜ Щ…ЩӮШҜШ§Шұ null ШұШ§ ШЁШұЩ…ЫҢвҖҢЪҜШұШҜШ§ЩҶШҜ Ъ©ЩҮ ШЁЩҮ ШұШ§ШӯШӘЫҢ ЩӮШ§ШЁЩ„ ШіЩҶШ¬Шҙ ЩҲ Щ…ШҜЫҢШұЫҢШӘ Ш§ШіШӘ.

ЩҶШӘЫҢШ¬ЩҮвҖҢЪҜЫҢШұЫҢ
Ш§Ш¶Ш§ЩҒЩҮ ШҙШҜЩҶ Ъ©Щ„Ш§Ші MediaTypeMap ШЁЩҮ .NET 11 Preview ЩҶЩ…ЩҲЩҶЩҮвҖҢШ§ЫҢ Ш№Ш§Щ„ЫҢ Ш§ШІ ШӘЪ©Ш§Щ…Щ„ Ш§ЫҢЩҶ ЩҒШұЫҢЩ…вҖҢЩҲШұЪ© ШҜШұ ШұШ§ШіШӘШ§ЫҢ ШӯШ°ЩҒ ЩҲШ§ШЁШіШӘЪҜЫҢвҖҢЩҮШ§ЫҢ Ш®Ш§ШұШ¬ЫҢ (Third-party) ЩҲ Ш§ЩҒШІШ§ЫҢШҙ ШЁЩҮШұЩҮвҖҢЩҲШұЫҢ ШӘЩҲШіШ№ЩҮвҖҢШҜЩҮЩҶШҜЪҜШ§ЩҶ Ш§ШіШӘ. ШЁШ§ Ш§ШіШӘЩҒШ§ШҜЩҮ Ш§ШІ Ш§ЫҢЩҶ ЩӮШ§ШЁЩ„ЫҢШӘШҢ ШҜЫҢЪҜШұ ЩҶЫҢШ§ШІЫҢ ШЁЩҮ ЩҶЪҜЩҮШҜШ§ШұЫҢ Щ…ЩҫЫҢЩҶЪҜвҖҢЩҮШ§ЫҢ ШҜШіШӘЫҢ ЫҢШ§ ЩҶШөШЁ ЩҫЪ©ЫҢШ¬вҖҢЩҮШ§ЫҢ ШӯШ¬ЫҢЩ… ШіШ§Ш®ШӘШ§Шұ ШіЫҢШіШӘЩ…вҖҢШ№Ш§Щ…Щ„ ЩҶШҜШ§ШұЫҢШҜ.
ШЁЩҮ Ш№ЩҶЩҲШ§ЩҶ ЫҢЪ© Ш®Щ„Ш§ШөЩҮ Щ…ШҜЫҢШұЫҢШӘЫҢ ЩҲ ЩҒЩҶЫҢШҢ ШӘЩҲШөЫҢЩҮ Щ…ЫҢвҖҢШҙЩҲШҜ:
  • ШЁШұШ§ЫҢ ШұШҙШӘЩҮвҖҢЩҮШ§ЫҢ Ш«Ш§ШЁШӘ ШҜШұ Ъ©ШҜШҢ ШӯШӘЩ…Ш§ЩӢ Ш§ШІ Ш«Ш§ШЁШӘвҖҢЩҮШ§ЫҢ Щ…ЩҲШ¬ЩҲШҜ ШҜШұ Ъ©Щ„Ш§Ші MediaTypeNames Ш§ШіШӘЩҒШ§ШҜЩҮ Ъ©ЩҶЫҢШҜ ШӘШ§ Ш§ШІ Ш®Ш·Ш§ЩҮШ§ЫҢ Ш§Щ…Щ„Ш§ЫҢЫҢ (Typos) Ш¬Щ„ЩҲЪҜЫҢШұЫҢ ШҙЩҲШҜ.
  • ШЁШұШ§ЫҢ ЩҶЪҜШ§ШҙШӘвҖҢЩҮШ§ЫҢ ЩҫЩҲЫҢШ§ ЩҲ ШӘШЁШҜЫҢЩ„ ЩҫШіЩҲЩҶШҜ ШЁЩҮ MIME Type ЫҢШ§ ШЁШұШ№Ъ©ШіШҢ Щ…ШӘШҜЩҮШ§ЫҢ Ъ©Щ„Ш§Ші MediaTypeMap ШұШ§ ШЁЩҮ Ъ©Ш§Шұ ШЁЪҜЫҢШұЫҢШҜ.
  • ЩҮЩ…ЩҲШ§ШұЩҮ Ш®ШұЩҲШ¬ЫҢвҖҢЩҮШ§ЫҢ null ШӯШ§ШөЩ„ Ш§ШІ ЩҒШұЩ…ШӘвҖҢЩҮШ§ЫҢ ЩҶШ§ШҙЩҶШ§Ш®ШӘЩҮ ШұШ§ Щ…ШҜЫҢШұЫҢШӘ (Null-checking) Ъ©ЩҶЫҢШҜ ШӘШ§ ЩҫШ§ЫҢШҜШ§ШұЫҢ ЩҶШұЩ…вҖҢШ§ЩҒШІШ§Шұ ШҙЩ…Ш§ ШӘШ¶Щ…ЫҢЩҶ ШҙЩҲШҜ.
Щ…Ш·Ш§Щ„ШЁ Щ…ШҙШ§ШЁЩҮ
    Щ…Ш·Ш§Щ„ШЁЩ…ШҙЪ©Щ„ IIS6 ЩҲ ШҜШұЫҢШ§ЩҒШӘ ЩҒШ§ЫҢЩ„вҖҢЩҮШ§ЫҢ ШўЩҒЫҢШі 2007Щ…Ш·Ш§Щ„ШЁШҜШіШӘЪ©Ш§ШұЫҢ mime types ШҜШұ IISЩ…Ш·Ш§Щ„ШЁШ§ШіЪ©ЩҶ ШЁШҜШ§ЩҒШІШ§ШұЩҮШ§ ШҜШұ ШӯЫҢЩҶ ШўЩҫЩ„ЩҲШҜ ЩҒШ§ЫҢЩ„вҖҢЩҮШ§ ШҜШұ ASP.NET Core ШЁШ§ Ш§ШіШӘЩҒШ§ШҜЩҮ Ш§ШІ Windows AMSI: ШўШ®ШұЫҢЩҶ Ш®Ш· ШҜЩҒШ§Ш№ЫҢЩ…Ш·Ш§Щ„ШЁШ®ШұЩҲШ¬ЫҢ Excel ШЁШ§ ШӯШ¬Щ… ШЁШ§Щ„Ш§ ШҜШұ ШЁШұЩҶШ§Щ…ЩҮвҖҢЩҮШ§ЫҢ вҖҢASP.NET Core ШЁШ§ Ш§ШіШӘЩҒШ§ШҜЩҮ Ш§ШІ MiniExcelЩ…Ш·Ш§Щ„ШЁШ§ШұШӘЩӮШ§ШЎ ШЁЩҮ ASP.NET Core 1.0 - ЩӮШіЩ…ШӘ 4 - ЩҒШ№Ш§Щ„ ШіШ§ШІЫҢ ЩҫШұШҜШ§ШІШҙ ЩҒШ§ЫҢЩ„вҖҢЩҮШ§ЫҢ Ш§ШіШӘШ§ШӘЫҢЪ©Щ…Ш·Ш§Щ„ШЁШЁЩҮЫҢЩҶЩҮвҖҢШіШ§ШІЫҢ ШӘШ®ШөЫҢШө ШӯШ§ЩҒШёЩҮ ШЁШ§ Щ…ШӘШҜ Ш¬ШҜЫҢШҜ StringBuilder.MoveChunks ШҜШұ NET 11.Щ…Ш·Ш§Щ„ШЁЩ„ЫҢЩҶЩҲЪ©Ші ШЁШұШ§ЫҢ ШӘЩҲШіШ№ЩҮвҖҢШҜЩҮЩҶШҜЪҜШ§ЩҶ ШҜШ§ШӘвҖҢЩҶШӘ: Ш§ШІ ЩҲЫҢЩҶШҜЩҲШІ ШӘШ§ Ъ©ШұШ§Ші-ЩҫЩ„ШӘЩҒШұЩ…Шӣ ШЁШ®Шҙ ШҜЩҲЩ…: Ъ©ШҜЩҶЩҲЫҢШіЫҢ Ъ©ШұШ§Ші-ЩҫЩ„ШӘЩҒШұЩ… ШҜШұ ШҜШ§ШӘвҖҢЩҶШӘЩҶШёШұШ§ШӘ Щ…Ш·Ш§Щ„ШЁЩҒШ№Ш§Щ„ ШіШ§ШІЫҢ ЩӮШіЩ…ШӘ ШўЩҫЩ„ЩҲШҜ ШӘШөЩҲЫҢШұ ЩҲ ЩҒШ§ЫҢЩ„ Kendo UI EditorЩҶШёШұШ§ШӘ Щ…Ш·Ш§Щ„ШЁШҜШіШӘЪ©Ш§ШұЫҢ mime types ШҜШұ IISЩҶШёШұШ§ШӘ Щ…Ш·Ш§Щ„ШЁШӘШҙШ®ЫҢШө ЩҶЩҲШ№ ЩҒШ§ЫҢЩ„ ШЁШ§ Ш§ШіШӘЩҒШ§ШҜЩҮ Ш§ШІ Щ…ШӯШӘЩҲШ§ЫҢ ЩҒШ§ЫҢЩ„
