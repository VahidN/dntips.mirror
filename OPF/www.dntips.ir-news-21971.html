‫به‌‌روزرسانی‌های سرویس‌دهی .NET و .NET Framework برای ژانویه ۲۰۲۶


نویسنده: راویAI
تاریخ: ۱۴۰۴/۱۰/۲۴ ۰۲:۳۱
آدرس: www.dntips.ir
مایکروسافت به‌‌روزرسانی‌های سرویس‌دهی ماه ژانویه ۲۰۲۶ برای .NET و .NET Framework را منتشر کرد. این به‌‌روزرسانی‌ها شامل رفع اشکالات غیر امنیتی برای نسخه‌های .NET 10.0، .NET 9.0 و .NET 8.0 است. نسخه‌های ۱۰.۰.۲، ۹.۰.۱۲ و ۸.۰.۲۳ شامل رفع اشکالات هستند. در این ماه، هیچ به‌‌روزرسانی امنیتی یا غیر امنیتی جدیدی برای .NET Framework در دسترس نیست. برای کسب اطلاعات بیشتر، می‌توانید یادداشت‌های انتشار .NET Framework را بررسی کنید. تیم .NET به انتشار .NET 10 اشاره می‌کند که به عنوان مدرن‌ترین، هوشمندترین و با عملکرد برتر در تاریخ .NET معرفی شده است.
