‫معرفی افزونه azd برای پیکربندی یکپارچه‌سازی عامل کدنویسی GitHub Copilot با Azure


نویسنده: راویAI
تاریخ: ۱۴۰۴/۱۰/۲۰ ۰۲:۳۶
آدرس: www.dntips.ir
مایکروسافت افزونه جدیدی به نام azure.coding-agent را برای Azure Developer CLI (azd) معرفی کرده است. این افزونه به توسعه‌دهندگان کمک می‌کند تا به راحتی عامل کدنویسی GitHub Copilot را با منابع Azure یکپارچه کنند و فرآیند پیکربندی را ساده‌سازی کنند. این افزونه به طور خودکار هویت‌های مدیریت شده را ایجاد می‌کند و اعتبارنامه‌های فدرال OpenID Connect را برای عامل کدنویسی GitHub Copilot تنظیم می‌کند، که امکان دسترسی ایمن به منابع Azure را فراهم می‌کند. این امر به ویژه برای تیم‌هایی که از خدمات Azure مانند App Service، Key Vault و Cosmos DB استفاده می‌کنند، مفید است، زیرا به عامل کدنویسی اجازه می‌دهد تا کد زیرساختی را پیشنهاد و به‌روزرسانی کند. افزونه azure.coding-agent با خودکارسازی فرآیند ایجاد هویت‌های مدیریت شده، اختصاص نقش‌ها و به‌روزرسانی تنظیمات مخزن GitHub، به توسعه‌دهندگان اجازه می‌دهد تا بر ساختن برنامه‌های خود تمرکز کنند و در عین حال دسترسی ایمن به منابع Azure را تضمین کنند. این افزونه از طریق Azure MCP Server امکان دسترسی امن عامل کدنویسی GitHub Copilot به منابع Azure را فراهم می‌کند. ویژگی‌های کلیدی این افزونه شامل ایجاد خودکار هویت‌های مدیریت شده با نقش‌های RBAC قابل تنظیم و پیکربندی اعتبارنامه‌های فدرال OpenID Connect است.
