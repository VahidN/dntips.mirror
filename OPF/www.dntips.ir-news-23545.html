Ш§ЩҶШӘШҙШ§Шұ ЫҢЪ© ШӘЩ… ШЁШұШ§ЫҢ Visual Studio Code Щ…ЫҢвҖҢШӘЩҲШ§ЩҶШҜ ЩҒШұШўЫҢЩҶШҜЫҢ ЪҶШ§Щ„ШҙвҖҢШЁШұШ§ЩҶЪҜЫҢШІ ШЁШ§ШҙШҜШҢ ШІЫҢШұШ§ ЩҶЫҢШ§ШІ ШЁЩҮ Ш§ЩҶШӘШҙШ§Шұ ШҜШұ ШҜЩҲ Щ…Ъ©Ш§ЩҶ Ш§ШөЩ„ЫҢ ЫҢШ№ЩҶЫҢ Visual Studio Marketplace ЩҲ Open VSX ШҜШ§ШұШҜ. Ш№Щ„Ш§ЩҲЩҮ ШЁШұ Ш§ЫҢЩҶШҢ ШЁШұШ§ЫҢ Ш§ШіШӘЩҒШ§ШҜЩҮ ШўШіШ§ЩҶвҖҢШӘШұ ШҜШұ ШІЩ…ЫҢЩҶЩҮвҖҢЩҮШ§ЫҢ ШҜЫҢЪҜШұ Щ…Ш§ЩҶЩҶШҜ ШЁШұШ¬ШіШӘЩҮвҖҢШіШ§ШІЫҢ ЩҶШӯЩҲ Ш§ШІ Ш·ШұЫҢЩӮ ShikiШҢ Щ…Щ…Ъ©ЩҶ Ш§ШіШӘ ШЁШ®ЩҲШ§ЩҮЫҢШҜ ШӘЩ… Ш®ЩҲШҜ ШұШ§ ШҜШұ npm ЩҶЫҢШІ Щ…ЩҶШӘШҙШұ Ъ©ЩҶЫҢШҜ. ШЁШұШ§ЫҢ ШўЩ…Ш§ШҜЩҮвҖҢШіШ§ШІЫҢ ШӘЩ…ШҢ ЩҶШ§Щ… ШўЩҶ ЩҶШЁШ§ЫҢШҜ ШҙШ§Щ…Щ„ Щ…ШӯШҜЩҲШҜЩҮ (scope) ШЁШ§ШҙШҜШҢ ШІЫҢШұШ§ Ш§ЫҢЩҶ Ъ©Ш§Шұ Щ…Ш§ЩҶШ№ Ш§ШІ Ш§ЩҶШӘШҙШ§Шұ ШҜШұ Open VSX Щ…ЫҢвҖҢШҙЩҲШҜ. ЩҮЩ…ЪҶЩҶЫҢЩҶШҢ ШЁШұШ§ЫҢ ЩҶЩ…Ш§ЫҢШҙ ШўЫҢЪ©ЩҲЩҶ ШӘЩ…ШҢ ШЁШ§ЫҢШҜ ЫҢЪ© ЩҒШ§ЫҢЩ„ ШӘШөЩҲЫҢШұ Щ…ШұШЁШ№ ШҙЪ©Щ„ ШЁШ§ Ш§ШЁШ№Ш§ШҜ 128 ЩҫЫҢЪ©ШіЩ„ ШҜШ§ШҙШӘЩҮ ШЁШ§ШҙЫҢШҜ ЩҲ Щ…ШіЫҢШұ ШўЩҶ ШұШ§ ШҜШұ ЩҲЫҢЪҳЪҜЫҢ icon ШҜШұ ЩҒШ§ЫҢЩ„ package.json Щ…ШҙШ®Шө Ъ©ЩҶЫҢШҜ. ШҜШұ ЩҒШ§ЫҢЩ„ package.jsonШҢ ШЁШ§ЫҢШҜ Ъ©Щ„ЫҢШҜ contributes ШұШ§ ШЁШ§ Ш§Ш·Щ„Ш§Ш№Ш§ШӘ Щ…ШұШЁЩҲШ· ШЁЩҮ ШӘЩ… Ш®ЩҲШҜ ШӘШ№ШұЫҢЩҒ Ъ©ЩҶЫҢШҜ. Ш§ЫҢЩҶ Ш§Ш·Щ„Ш§Ш№Ш§ШӘ ШҙШ§Щ…Щ„ ШЁШұЪҶШіШЁ (label)ШҢ ЩҶЩҲШ№ ШұШ§ШЁШ· Ъ©Ш§ШұШЁШұЫҢ (uiTheme) ЩҲ Щ…ШіЫҢШұ ЩҒШ§ЫҢЩ„ ШӘЩ… (path) Щ…ЫҢвҖҢШҙЩҲШҜ. ШЁШұШ§ЫҢ Ш§ЩҒШІШ§ЫҢШҙ ЩӮШ§ШЁЩ„ЫҢШӘ Ш¬ШіШӘШ¬ЩҲЫҢ ШӘЩ…ШҢ Ш§Ш¶Ш§ЩҒЩҮ Ъ©ШұШҜЩҶ Ъ©Щ„Щ…Ш§ШӘ Ъ©Щ„ЫҢШҜЫҢ Щ…ШұШӘШЁШ· ШҜШұ ШЁШ®Шҙ keywords ЩҶЫҢШІ Ш¶ШұЩҲШұЫҢ Ш§ШіШӘ. ШЁШұШ§ЫҢ Ш§ЩҶШӘШҙШ§Шұ ШҜШұ Visual Studio MarketplaceШҢ Щ…ЫҢвҖҢШӘЩҲШ§ЩҶЫҢШҜ Ш§ШІ Ш§ШЁШІШ§Шұ vsce ШЁШ§ Ш§ШіШӘЩҒШ§ШҜЩҮ Ш§ШІ ЫҢЪ© ШӘЩҲЪ©ЩҶ ШҜШіШӘШұШіЫҢ ШҙШ®ШөЫҢ Ш§ШІ ШӯШіШ§ШЁ Azure DevOps Ш®ЩҲШҜ Ш§ШіШӘЩҒШ§ШҜЩҮ Ъ©ЩҶЫҢШҜ. ШҜШұ ШөЩҲШұШӘ ШЁШұЩҲШІ Щ…ШҙЪ©Щ„ ШҜШұ ШӘЩҶШёЫҢЩ… ШӯШіШ§ШЁ Azure DevOpsШҢ Щ…ЫҢвҖҢШӘЩҲШ§ЩҶЫҢШҜ Ш§ШІ ШұЩҲШҙ ШҜШіШӘЫҢ ЩҶЫҢШІ ШЁШұШ§ЫҢ Ш§ЩҶШӘШҙШ§Шұ Ш§ШіШӘЩҒШ§ШҜЩҮ Ъ©ЩҶЫҢШҜ. ШҜШұ ЩҶЩҮШ§ЫҢШӘШҢ ЩҫШі Ш§ШІ Ш§ЫҢШ¬Ш§ШҜ ШӯШіШ§ШЁ Ъ©Ш§ШұШЁШұЫҢ ШҜШұ Visual Studio Marketplace ЩҲ Ш§ЫҢШ¬Ш§ШҜ ЫҢЪ© ШӯШіШ§ШЁ ЩҶШ§ШҙШұШҢ Щ…ЫҢвҖҢШӘЩҲШ§ЩҶЫҢШҜ ШӘЩ… Ш®ЩҲШҜ ШұШ§ ШЁШ§ Ш§ШіШӘЩҒШ§ШҜЩҮ Ш§ШІ ШҜШіШӘЩҲШұ npx vsce publish Щ…ЩҶШӘШҙШұ Ъ©ЩҶЫҢШҜ.
