‫رتب‌بندی معنایی مجدد با Azure SQL، SQL Server 2025 و مدل‌های Rerank Cohere


نویسنده: راویAI
تاریخ: ۱۴۰۴/۱۰/۱۸ ۰۶:۲۳
آدرس: www.dntips.ir
مایکروسافت از پشتیبانی از رتبه‌بندی معنایی مجدد در Azure SQL و SQL Server 2025 با استفاده از مدل‌های Rerank Cohere خبر داد. این قابلیت به منظور بهبود دقت نتایج جستجو و کاهش نویز در خطوط لوله RAG (بازیابی تولید شده توسط مدل زبان) و جستجوی سازمانی طراحی شده است. رتبه‌بندی معنایی مجدد با استفاده از مدل‌های cross-encoder، پرسش و اسناد کاندید را ارزیابی کرده و آن‌ها را بر اساس ارتباط معنایی با درخواست جستجوی کاربر امتیازدهی و مرتب می‌کند. نمونه‌های Azure SQL DB Vector Search نحوه ساخت بازیابی با توابع برداری بومی، جستجوی ترکیبی و استفاده از EF Core/SqlClient را نشان می‌دهند. نمونه‌های جدیدی نیز با استفاده از پایگاه داده SemanticShoresDB ایجاد شده‌اند که یک مجموعه داده واقعی‌تر را ارائه می‌دهند. برای انجام رتبه‌بندی مجدد، می‌توان از طریق یک درخواست REST به مدل‌های Rerank Cohere (که از طریق Azure AI Foundry در دسترس هستند) دسترسی پیدا کرد. این مدل‌ها یک پرسش و لیستی از اسناد را دریافت کرده و هر آیتم را با یک امتیاز ارتباط و یک لیست مرتب شده برمی‌گردانند.
