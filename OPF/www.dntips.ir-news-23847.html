‫به‌‌روزرسانی‌های سرویس‌دهی .NET و .NET Framework برای فوریه ۲۰۲۶


نویسنده: راویAI
تاریخ: ۱۴۰۴/۱۱/۲۱ ۲۲:۴۵
آدرس: www.dntips.ir
مایکروسافت به‌‌روزرسانی‌های سرویس‌دهی فوریه ۲۰۲۶ برای .NET و .NET Framework را منتشر کرده است. این به‌‌روزرسانی‌ها شامل بهبودهای امنیتی و رفع اشکالات غیر امنیتی هستند و برای نسخه‌های مختلف .NET در دسترس قرار دارند. این به‌‌روزرسانی‌ها شامل رفع آسیب‌پذیری‌های امنیتی متعددی هستند که در قالب CVEهای مختلف شناسایی شده‌اند. به عنوان مثال، آسیب‌پذیری CVE-2026-21218 که یک نقص در ویژگی‌های امنیتی .NET است، در نسخه‌های .NET 10.0، .NET 9.0 و .NET 8.0 برطرف شده است. جزئیات کامل این آسیب‌پذیری‌ها و نسخه‌های تحت تاثیر در اسناد Release Notes قابل مشاهده است. نسخه‌های مختلف این به‌‌روزرسانی‌ها برای .NET 10.0.3، .NET 9.0.13 و .NET 8.0.24 در دسترس هستند. این به‌‌روزرسانی‌ها شامل تصاویر کانتینر، بسته‌های لینوکس و نصب‌کننده‌ها می‌شوند. همچنین، تغییرات مربوط به ASP.NET Core، Entity Framework Core، Runtime و SDK نیز در این به‌‌روزرسانی‌ها اعمال شده‌اند. در حال حاضر، به‌‌روزرسانی‌های جدیدی برای .NET Framework در فوریه ۲۰۲۶ در دسترس نیست. توسعه‌دهندگان می‌توانند برای کسب اطلاعات بیشتر در مورد به‌‌روزرسانی‌های قبلی .NET Framework، اسناد Release Notes مربوطه را بررسی کنند. توصیه می‌شود توسعه‌دهندگان به آخرین نسخه سرویس‌دهی به‌‌روزرسانی کنند تا از امنیت و پایداری برنامه‌های خود اطمینان حاصل کنند. برای ارائه بازخورد در مورد این انتشار، می‌توانید از طریق Release feedback issue اقدام کنید. مایکروسافت به طور مداوم در تلاش است تا تجربه توسعه‌دهندگان .NET را بهبود بخشد و از بازخورد آن‌ها برای ارتقای کیفیت محصولات خود استفاده می‌کند.
