مراحل:
  • مرحله 1: مبانی (ماه 1-2)

  • مرحله 2: یکپارچه‌سازی API (ماه 3)
    • یکپارچه‌سازی API چت جی‌پی‌تی با NET.

  • مرحله 3: Azure AI (ماه 4-5)
    • گواهینامه Azure AI-102 (Azure OpenAI، سرویس‌های شناختی، استقرار)
    • مستندات سرویس Azure OpenAI (یکپارچه‌سازی NET.)
  • مرحله 4: موارد استفاده پیشرفته (ماه 6+ )
    • هوش مصنوعی برای رهبران کسب و کار (جذب هوش مصنوعی سازمانی)
    • تنظیم دقیق مدل‌های OpenAI (استودیوی Azure AI)
پروژه‌های عملی:
  • ربات چت NET. (API OpenAI و Azure AI)
  • دستیار جستجوی مبتنی بر هوش مصنوعی (embeddingهای OpenAI)
  • LangChain.NET برای گردش‌های کاری هوش مصنوعی
جدول زمانی:
  • ماه‌های 1-2: مبانی هوش مصنوعی مولد
  • ماه 3: یکپارچه‌سازی API
  • ماه‌های 4-5: Azure AI
  • ماه 6+: هوش مصنوعی سازمانی
نتیجه:
  • تجربه عملی با مدل‌های هوش مصنوعی مولد
  • یکپارچه‌سازی هوش مصنوعی در برنامه‌های NET. و Azure
  • درک جذب هوش مصنوعی سازمانی
  • آماده برای استقرار برنامه‌های هوش مصنوعی با استفاده از Azure AI
