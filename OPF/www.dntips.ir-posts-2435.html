Ш§ШҙШӘШұШ§Ъ©вҖҢЩҮШ§ ЫұЫ·Ы№ЫұЫҙ
→ вҖ«ШӯЩ„ Щ…ШҙЪ©Щ„ ЩҲЫҢЪҳЩҲШ§Щ„ Ш§ШіШӘЩҲШҜЫҢЩҲ ШҜШұ ШіЫҢШіШӘЩ…ЩҮШ§ЫҢЫҢ Ъ©ЩҮ Ш§ШІ ШұШІЩҲЩ„ЩҲШҙЩҶ (DPI) ШЁШ§Щ„Ш§ ЩҲ Щ…Ш§ЩҶЫҢШӘЩҲШұ ЩҮШ§ЫҢ 4K Ш§ШіШӘЩҒШ§ШҜЩҮ Щ…ЫҢ Ъ©ЩҶЩҶШҜ
вҖ«Ш§ШіШӘЩҒШ§ШҜЩҮ Ш§ШІ dll ЩҮШ§ЫҢ ШҜШ§ШӘ ЩҶШӘ ШҜШұ Ш§Щ„Ъ©ШӘШұЩҲЩҶ ←

ЩҶЩҲЫҢШіЩҶШҜЩҮ: ЩҲШӯЫҢШҜ ЩҶШөЫҢШұЫҢ
ШӘШ§ШұЫҢШ®: ЫұЫіЫ№Ыө/Ы°Ыі/ЫіЫ° ЫұЫі:ЫіЫ°

Ш§ЫҢЩҶ Щ„ЫҢЩҶЪ© ШұШ§ ШҜШұЩҶШёШұ ШЁЪҜЫҢШұЫҢШҜ:
 http://en.ЩҮШҙШ§Щ….com/post/build-customizable-language-switcher-tag-helper-with-bootstrap
ШҜШұ ШҜШ§Щ…ЩҶЩҮвҖҢЫҢ ШўЩҶШҢ ШӯШұЩҲЩҒ ЫҢЩҲЩҶЫҢЪ©ШҜ (ЩҒШ§ШұШіЫҢ/Ш№ШұШЁЫҢ) ШЁЪ©Ш§Шұ ШұЩҒШӘЩҮвҖҢШ§ЩҶШҜ. Ш§ЪҜШұ ШөШұЩҒШ§ ШЁШ§ Ш§ШіШӘЩҒШ§ШҜЩҮ Ш§ШІ ЩӮШ·Ш№ЩҮ Ъ©ШҜ ШІЫҢШұ ШЁШ®ЩҲШ§ЩҮЫҢЩ… ЩҲШ¬ЩҲШҜ Ш§ЫҢЩҶ ШўШҜШұШі ШұШ§ ШЁШұШұШіЫҢ Ъ©ЩҶЫҢЩ…:
 WebRequest wr = WebRequest.Create(uri);
using (WebResponse response = wr.GetResponse()) { }
ШЁЩҮ Ш®Ш·Ш§ЫҢ ШІЫҢШұ ШЁШұШ®ЩҲШ§ЩҮЫҢЩ… Ш®ЩҲШұШҜ:
 The remote name could not be resolved: 'en.ЩҮШҙШ§Щ….com'

Ш§Щ„ШЁШӘЩҮ Щ…Щ…Ъ©ЩҶ Ш§ШіШӘ Ъ©ШҜ ЩҒЩҲЩӮ ШұШ§ ШЁШұ ШұЩҲЫҢ ЩҲЫҢЩҶШҜЩҲШІЩҮШ§ЫҢ Ш¬ШҜЫҢШҜ ШЁШҜЩҲЩҶ Щ…ШҙЪ©Щ„ Ш§Ш¬ШұШ§ Ъ©ЩҶЫҢШҜ. Ш№Щ„ШӘ Ш§ЫҢЩҶШ¬Ш§ Ш§ШіШӘ Ъ©ЩҮ Ш§ЪҜШұ Ш§ШІ ЩҲЫҢЩҶШҜЩҲШІЩҮШ§ЫҢ ЩӮШЁЩ„ Ш§ШІ ЩҲЫҢЩҶШҜЩҲШІ ШіШұЩҲШұ 2012 Ш§ШіШӘЩҒШ§ШҜЩҮ Щ…ЫҢвҖҢЪ©ЩҶЫҢШҜШҢ ШҜШ§ШӘ ЩҶШӘ ЩҒШұЫҢЩ… ЩҲШұЪ© Ш§ШІ RFC 3490 Ш§ШіШӘЩҒШ§ШҜЩҮ Щ…ЫҢвҖҢЪ©ЩҶШҜШӣ ШҜШұ ШәЫҢШұШ§ЫҢЩҶШөЩҲШұШӘ Ш§ШІ RFC 5891 (ЩҒЩӮШ· ШЁШұШ§ЫҢ Windows 8, Windows 8.1, Windows 10, or Windows Server 2012)ШҢ ШЁШ§ Ш§ЫҢЩҶ ШӘЩҒШ§ЩҲШӘвҖҢЩҮШ§.

ШұЩҲШҙ ШұЩҒШ№ ШўЩҶ ЩҮЩ… ЩҒШ№Ш§Щ„ ШіШ§ШІЫҢ ЩҫШұШҜШ§ШІШҙ Ш§ЫҢЩҶ ЩҶЩҲШ№ ШҜШ§Щ…ЩҶЩҮвҖҢЩҮШ§ (ШЁШұ ШұЩҲЫҢ ШӘЩ…Ш§Щ… ЩҲЫҢЩҶШҜЩҲШІЩҮШ§) ШҜШұ ЩҒШ§ЫҢЩ„вҖҢЩҮШ§ЫҢ app.config ЩҲ ЫҢШ§ web.config ШЁЩҮ ШөЩҲШұШӘ ШІЫҢШұ Ш§ШіШӘ:
<configuration>
    <uri>
        <idn enabled="All" />
        <iriParsing enabled="true" />
    </uri>
</configuration>
Ъ©Ш§ШұЫҢ Ъ©ЩҮ Ш§ЫҢЩҶ ЩҒШ№Ш§Щ„ ШіШ§ШІЫҢ Ш§ЩҶШ¬Ш§Щ… Щ…ЫҢвҖҢШҜЩҮШҜШҢ ШӘШЁШҜЫҢЩ„ Ш®ЩҲШҜЪ©Ш§Шұ ЩҶШ§Щ… ЫҢЩҲЩҶЫҢЪ©ШҜ ШЁЩҮ В«PunycodeВ» Ш§ШіШӘ:
 var unicode = @"en.ЩҮШҙШ§Щ….com";
 
var mapping = new IdnMapping(); // IDN  = Internationalizing Domain Names
var ascii = mapping.GetAscii(unicode);
Console.WriteLine(ascii);
 
string convertedBackToUnicode = mapping.GetUnicode(ascii);
Console.WriteLine(convertedBackToUnicode);
ШҜШұ Ш§ЫҢЩҶШ¬Ш§ ШЁЩҮ Щ…Ш№Ш§ШҜЩ„ Ш§ШіЪ©ЫҢ ЩҲЫҢЪҳЩҮвҖҢЫҢ ШҜШ§Щ…ЩҶЩҮвҖҢЫҢ ЫҢЩҲЩҶЫҢЪ©ШҜ ЫҢШ§ В«en.xn--mgbz4cf.comВ» Ъ©ЩҮ Punycode ЩҶШ§Щ… ЪҜШұЩҒШӘЩҮ Щ…ЫҢвҖҢШұШіЫҢЩ…. Ш§ЫҢЩҶ ШҜШ§Щ…ЩҶЩҮвҖҢШ§ЫҢ Ш§ШіШӘ Ъ©ЩҮ ШЁШұ ШұЩҲЫҢ ШӘЩ…Ш§Щ… ЩҲЫҢЩҶШҜЩҲШІЩҮШ§ ШЁШҜЩҲЩҶ Щ…ШҙЪ©Щ„ Ъ©Ш§Шұ Щ…ЫҢвҖҢЪ©ЩҶШҜ. Ш§Щ„ШЁШӘЩҮ ЩҮЩ…Ш§ЩҶШ·ЩҲШұ Ъ©ЩҮ Ш№ЩҶЩҲШ§ЩҶ ШҙШҜ ЩҶЫҢШ§ШІЫҢ ШЁЩҮ Ш§ЩҶШ¬Ш§Щ… ШҜШіШӘЫҢ Ш§ЫҢЩҶ ЩҶЩҲШ№ ШӘШЁШҜЫҢЩ„Ш§ШӘ ЩҶЫҢШіШӘ ЩҲ ЩҮЩ…Ш§ЩҶ ЪҶЩҶШҜ ШіШ·Шұ ШӘЩҶШёЫҢЩ…Ш§ШӘ ЩҒШ§ЫҢЩ„ config ШЁШұШ§ЫҢ ЩҒШ№Ш§Щ„ ШіШ§ШІЫҢ Ш®ЩҲШҜЪ©Ш§Шұ Ш§ЫҢЩҶ ЩҶЩҲШ№ ШӘШЁШҜЫҢЩ„Ш§ШӘ Ъ©Ш§ЩҒЫҢ Ш§ШіШӘ.
Щ…Ш·Ш§Щ„ШЁ Щ…ШҙШ§ШЁЩҮ
    Ш§ШҙШӘШұШ§Ъ©вҖҢЩҮШ§ЩҫШҙШӘЫҢШЁШ§ЩҶЫҢ ШЁЩҲЩ…ЫҢ UTF-8 ШҜШұ SQL Server 2019: ЩҶШ¬Ш§ШӘвҖҢШҜЩҮЩҶШҜЩҮ ЫҢШ§ ЩҲШ№ШҜЩҮвҖҢШ§ЫҢ ШӘЩҲШ®Ш§Щ„ЫҢШҹШ§ШҙШӘШұШ§Ъ©вҖҢЩҮШ§10 Щ…ЩӮШ§Щ„ЩҮвҖҢШ§ЫҢ Ъ©ЩҮ ЩҮШұ ШЁШұЩҶШ§Щ…ЩҮ ЩҶЩҲЫҢШіЫҢ ШЁШ§ЫҢШҜ ШЁШ®ЩҲШ§ЩҶШҜЩ…Ш·Ш§Щ„ШЁШҜШ§ШіШӘШ§ЩҶЫҢ Ш§ШІ UnicodeЩ…Ш·Ш§Щ„ШЁHtml EncodingЩ…Ш·Ш§Щ„ШЁreturn File ШҜШұ ASP.NET MVC ЩҲ ЩҶШ§Щ…вҖҢЩҮШ§ЫҢ ЫҢЩҲЩҶЫҢЪ©ШҜЩҶШёШұШ§ШӘ Ш§ШҙШӘШұШ§Ъ©вҖҢЩҮШ§ШӘШЁШҜЫҢЩ„ЪҜШұ Ш§ЫҢШұШ§ЩҶ ШіЫҢШіШӘЩ… ШЁЩҮ ЫҢЩҲЩҶЫҢЪ©ШҜЩҶШёШұШ§ШӘ Ш§ШҙШӘШұШ§Ъ©вҖҢЩҮШ§ШӘШЁШҜЫҢЩ„ЪҜШұ Ш§ЫҢШұШ§ЩҶ ШіЫҢШіШӘЩ… ШЁЩҮ ЫҢЩҲЩҶЫҢЪ©ШҜЩҶШёШұШ§ШӘ Ш§ШҙШӘШұШ§Ъ©вҖҢЩҮШ§NET Framework 4.7. Щ…ЩҶШӘШҙШұ ШҙШҜЩҶШёШұШ§ШӘ Щ…Ш·Ш§Щ„ШЁЩ…Ш№ШұЩҒЫҢ System.Text.Json ШҜШұ NET Core 3.0.ЩҶШёШұШ§ШӘ Щ…Ш·Ш§Щ„ШЁШӘШЁШҜЫҢЩ„ HTML ШЁЩҮ PDF ШЁШ§ Ш§ШіШӘЩҒШ§ШҜЩҮ Ш§ШІ Ъ©ШӘШ§ШЁШ®Ш§ЩҶЩҮвҖҢЫҢ iTextSharp
