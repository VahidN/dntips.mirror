ЩҶЩҲЫҢШіЩҶШҜЩҮ: ЩҲШӯЫҢШҜ ЩҶШөЫҢШұЫҢ
ШӘШ§ШұЫҢШ®: ЫұЫіЫ№Ыё/Ы°Ы¶/Ы°Ыё ЫұЫө:ЫҙЫө
ШўШҜШұШі: www.dntips.ir
ШҜШұ ЩҶЪҜШ§ШұШҙвҖҢЩҮШ§ЫҢ Ш§ЩҲЩ„ЫҢЩҮвҖҢЫҢ ASP.NET CoreШҢ ЩҫШҙШӘЫҢШЁШ§ЩҶЫҢ Ш§ШІ authorizationШҢ ШөШұЩҒШ§ ШӘЩҲШіШ· ЩҲЫҢЪҳЪҜЫҢ [Authorize]ШҢ ЩӮШ§ШЁЩ„ Ш§Ш№Щ…Ш§Щ„ ШЁЩҮ Ш§Ъ©ШҙЩҶ Щ…ШӘШҜ Ш®Ш§ШөЫҢ ШЁЩҲШҜ. ШЁШұШ§ЫҢ ШЁЩҮШЁЩҲШҜ ШӘЩҶШёЫҢЩ… Ш§ЫҢЩҶ ЩӮШ§ШЁЩ„ЫҢШӘШҢ Щ…ЫҢШ§ЩҶвҖҢШ§ЩҒШІШ§Шұ Ш¬ШҜЫҢШҜ Authorization ШЁЩҮ ASP.NET Core 3.0 Ш§Ш¶Ш§ЩҒЩҮ ШҙШҜЩҮвҖҢШ§ШіШӘ ЩҲ ШӘЩҶШёЫҢЩ… ШўЩҶ Ш¬ЩҮШӘ Ъ©Ш§Шұ ШЁШ§ Ш§Щ…Ъ©Ш§ЩҶШ§ШӘ Ш§Щ…ЩҶЫҢШӘЫҢ ШЁШұЩҶШ§Щ…ЩҮШҢ Ш§Щ„ШІШ§Щ…ЫҢ Ш§ШіШӘШӣ ШҜШұ ШәЫҢШұ Ш§ЫҢЩҶШөЩҲШұШӘ ШҜШұ ШӯЫҢЩҶ Щ…ШұЩҲШұ Ш§ЫҢЩҶ ШөЩҒШӯШ§ШӘ ЩҲ ЩӮШіЩ…ШӘвҖҢЩҮШ§ЫҢ Щ…ШӯШ§ЩҒШёШӘ ШҙШҜЩҮШҢ ШЁШұЩҶШ§Щ…ЩҮ ШЁШ§ Ш®Ш·Ш§ЫҢ ШІЫҢШұ Щ…ШӘЩҲЩӮЩҒ Ш®ЩҲШ§ЩҮШҜ ШҙШҜ:
Endpoint xyz contains authorization metadata, but a middleware was not found that supports authorization.
Configure your application startup by adding app.UseAuthorization() inside the call to Configure(..) in the application startup code.


Щ…ШӯЩ„ ШөШӯЫҢШӯ ШӘШ№ШұЫҢЩҒ Щ…ЫҢШ§ЩҶвҖҢШ§ЩҒШІШ§Шұ Authorization ШҜШұ ASP.NET Core 3.0

ШӘЩҲШөЫҢЩҮ ШҙШҜЩҮвҖҢШ§ШіШӘ Щ…ЫҢШ§ЩҶвҖҢШ§ЩҒШІШ§Шұ Ш¬ШҜЫҢШҜ AuthorizationШҢ ШЁШ§ ЩҒШұШ§Ш®ЩҲШ§ЩҶЫҢ Щ…ШӘШҜ UseAuthorizationШҢ ШЁЩ„Ш§ЩҒШ§ШөЩ„ЩҮ ЩҫШі Ш§ШІ ЩҒШұШ§Ш®ЩҲШ§ЩҶЫҢ Щ…ШӘШҜ UseAuthentication Щ…Ш№ШұЩҒЫҢ ШҙЩҲШҜ. ШҜШұ Ш§ЫҢЩҶ ШӯШ§Щ„ШӘ Ш§ЫҢЩҶ Щ…ЫҢШ§ЩҶвҖҢШ§ЩҒШІШ§ШұШҢ ШЁШ§ ЫҢЪ© Policy ЩҫЫҢШҙвҖҢЩҒШұШ¶ ШӘЩҶШёЫҢЩ… Щ…ЫҢвҖҢШҙЩҲШҜ Ъ©ЩҮ ЩӮШ§ШЁЩ„ ШӘШәЫҢЫҢШұ ЩҲ ШЁШ§ШІЩҶЩҲЫҢШіЫҢ Ш§ШіШӘ:
public void Configure(IApplicationBuilder app)
{
    ...

    app.UseRouting();

    app.UseAuthentication();
    app.UseAuthorization();

    app.UseEndpoints(endpoints =>
    {
        endpoints.MapDefaultControllerRoute();
    });
}
ШҜШұ Щ…ЩҲШұШҜ Щ…ЩҒЩҮЩҲЩ… Щ…ШӘШҜ MapDefaultControllerRouteШҢ Щ…ЫҢвҖҢШӘЩҲШ§ЩҶЫҢШҜ ШЁЩҮ ЩҶШёШұШ§ШӘ ШӘЪ©Щ…ЫҢЩ„ЫҢ Щ…Ш·Щ„ШЁ Endpoint routing Щ…ШұШ§Ш¬Ш№ЩҮ Ъ©ЩҶЫҢШҜ.
ШҜШұ ASP.NET Core 3.0ШҢ ЩҫШі Ш§ШІ ШӘЩҶШёЫҢЩ…Ш§ШӘ ЩҒЩҲЩӮ Ш§ШіШӘ Ъ©ЩҮ ЩӮШ·Ш№ЩҮ Ъ©ШҜ ШІЫҢШұ ЩҲ Ш§Ш№Щ…Ш§Щ„ ЩҒЫҢЩ„ШӘШұ AuthorizeШҢ Щ…Ш¬ШҜШҜШ§ Ъ©Ш§Шұ Щ…ЫҢвҖҢЪ©ЩҶШҜ:
public class HomeController : ControllerBase
{
    [Authorize]
    public IActionResult BuyWidgets()
    {
        ...
    }
}


ШұЩҲШҙ ШӘШ№ШұЫҢЩҒ ЩҒЫҢЩ„ШӘШұ Authorize ШЁЩҮ ШөЩҲШұШӘ ШіШұШ§ШіШұЫҢ ШҜШұ ASP.NET Core 3.0

Щ…ЫҢвҖҢШӘЩҲШ§ЩҶ AuthorizeFilter ШұШ§ ШЁЩҮ ШөЩҲШұШӘ ЫҢЪ© ЩҒЫҢЩ„ШӘШұ ШіШұШ§ШіШұЫҢ (global filter in MVC) ШӘШ№ШұЫҢЩҒ Ъ©ШұШҜ ШӘШ§ ШЁЩҮ ШӘЩ…Ш§Щ… Ъ©ЩҶШӘШұЩ„ШұЩҮШ§ ЩҲ Ш§Ъ©ШҙЩҶ Щ…ШӘШҜЩҮШ§ Ш§Ш№Щ…Ш§Щ„ ШҙЩҲШҜ. ЩҮШұЪҶЩҶШҜ Ш§ЫҢЩҶ ШұЩҲШҙ ЩҮЩҶЩҲШІ ЩҮЩ… ШҜШұ ASP.NET Core 3.0 Ъ©Ш§Шұ Щ…ЫҢвҖҢЪ©ЩҶШҜШҢ Ш§Щ…Ш§ ШұЩҲШҙ ШӘЩҲШөЫҢЩҮ ШҙШҜЩҮвҖҢЫҢ Ш¬ШҜЫҢШҜ ШўЩҶ ШЁЩҮ ШөЩҲШұШӘ ШІЫҢШұ Ш§ШіШӘ:
public void Configure(IApplicationBuilder app)
{
    ...

    app.UseRouting();

    app.UseAuthentication();
    app.UseAuthorization();

    app.UseEndpoints(endpoints =>
    {
        endpoints.MapDefaultControllerRoute().RequireAuthorization();
    });
}
ШҜШұ Ш§ЫҢЩҶ ШӯШ§Щ„ШӘ ШЁШ§ Ш§Ш№Щ…Ш§Щ„ RequireAuthorization ШЁЩҮ MapDefaultControllerRouteШҢ ШіШЁШЁ Ш®ЩҲШ§ЩҮЫҢЩ… ШҙШҜ ШӘШ§ Ш§Ш№ШӘШЁШ§ШұШіЩҶШ¬ЫҢ Ъ©Ш§ШұШЁШұ ШЁШұШ§ЫҢ Ш§ШіШӘЩҒШ§ШҜЩҮвҖҢЫҢ Ш§ШІ ШӘЩ…Ш§Щ… ЩӮШіЩ…ШӘвҖҢЩҮШ§ЫҢ ШЁШұЩҶШ§Щ…ЩҮШҢ Ш§Ш¬ШЁШ§ШұЫҢ ШҙЩҲШҜ. ШҜШұ Ш§ЫҢЩҶ ШЁЫҢЩҶ Ш§ЪҜШұ Ъ©ЩҶШӘШұЩ„ШұЫҢ ЩҲ ЫҢШ§ Ш§Ъ©ШҙЩҶ Щ…ШӘШҜЫҢ ЩҶШЁШ§ЫҢШҜ Щ…ШӯШ§ЩҒШёШӘ ШҙЩҲШҜ ЩҲ ШҜШіШӘШұШіЫҢ ШўШІШ§ШҜШ§ЩҶЩҮвҖҢЫҢ ШЁЩҮ ШўвҖҢЩҶвҖҢЩҮШ§ Щ…ШҜЩҶШёШұ Ш§ШіШӘШҢ Щ…ЫҢвҖҢШӘЩҲШ§ЩҶ Ш§ШІ ЩҒЫҢЩ„ШӘШұ AllowAnonymous Ш§ШіШӘЩҒШ§ШҜЩҮ Ъ©ШұШҜ:
[AllowAnonymous]
public class HomeController : ControllerBase
{
    ...
}


ШіЩҒШ§ШұШҙЫҢ ШіШ§ШІЫҢ ШіЫҢШ§ШіШӘвҖҢЩҮШ§ЫҢ Authorization ШҜШұ ASP.NET Core 3.0

Ш§ЪҜШұ ШЁШ®ЩҲШ§ЩҮЫҢЩ… DefaultPolicy Щ…ЫҢШ§ЩҶвҖҢШ§ЩҒШІШ§Шұ Authorization ШұШ§ ШЁШ§ШІЩҶЩҲЫҢШіЫҢ Ъ©ЩҶЫҢЩ…ШҢ Щ…ЫҢвҖҢШӘЩҲШ§ЩҶ Ш§ШІ Щ…ШӘШҜ services.AddAuthorization ШЁЩҮ ШөЩҲШұШӘ ШІЫҢШұ Ш§ШіШӘЩҒШ§ШҜЩҮ Ъ©ШұШҜ Ъ©ЩҮ ШҜШұ ШўЩҶ authentication ЩҲ ШҜШ§ШҙШӘЩҶ ЫҢЪ© Scope Ш®Ш§Шө ШұШ§ Ш§Ш¬ШЁШ§ШұЫҢ Щ…ЫҢвҖҢЪ©ЩҶШҜ:
public void ConfigureServices(IServiceCollection services)
{
    ...
    
    services.AddAuthorization(options =>
    {
        options.DefaultPolicy = new AuthorizationPolicyBuilder()
          .RequireAuthenticatedUser()
          .RequireScope("MyScope")
          .Build();
    });
}


ШӘЩҶШёЫҢЩ… ШіЫҢШ§ШіШӘвҖҢЩҮШ§ЫҢ ШҜШіШӘШұШіЫҢШҢ ШІЩ…Ш§ЩҶЫҢЪ©ЩҮ ЩҮЫҢЪҶ ЩҶЩҲШ№ ШӘЩҶШёЫҢЩ…ЫҢ Ш§ШІ ЩҫЫҢШҙ ШӘШ№ШұЫҢЩҒ ЩҶШҙШҜЩҮвҖҢШ§ШіШӘ

ШЁШ§ ШӘЩҶШёЫҢЩ… FallbackPolicyШҢ Щ…ЫҢвҖҢШӘЩҲШ§ЩҶ ШӘЩ…Ш§Щ… endpointЩҮШ§ЫҢЫҢ ШұШ§ Ъ©ЩҮ ШӘЩҲШіШ· ЩҒЫҢЩ„ШӘШұ [Authorize] ЩҲ ЫҢШ§ ()RequireAuthorization Щ…ШӯШ§ЩҒШёШӘ ЩҶШҙШҜЩҮвҖҢШ§ЩҶШҜШҢ Щ…ШӯШ§ЩҒШёШӘ Ъ©ШұШҜ.
DefaultPolicy ШӘЩҲШіШ· ЩҒЫҢЩ„ШӘШұ [Authorize] ЩҲ ЫҢШ§ ()RequireAuthorization Щ…ЩҲШұШҜ Ш§ШіШӘЩҒШ§ШҜЩҮ ЩӮШұШ§Шұ Щ…ЫҢвҖҢЪҜЫҢШұШҜШӣ Ш§Щ…Ш§ FallbackPolicy ШІЩ…Ш§ЩҶЫҢ ШЁЪ©Ш§Шұ Ш®ЩҲШ§ЩҮШҜ ШұЩҒШӘ Ъ©ЩҮ ЩҮЫҢЪҶ ЩҶЩҲШ№ ШіЫҢШ§ШіШӘ ШҜШіШӘШұШіЫҢ ШӘЩҶШёЫҢЩ… ЩҶШҙШҜЩҮвҖҢШЁШ§ШҙШҜ. ШӯШ§Щ„ШӘ ЩҫЫҢШҙвҖҢЩҒШұШ¶ FallbackPolicyШҢ ЩҫШ°ЫҢШұШҙ ШӘЩ…Ш§Щ… ШҜШұШ®ЩҲШ§ШіШӘвҖҢЩҮШ§ ШЁШҜЩҲЩҶ Ш§Ш№ШӘШЁШ§ШұШіЩҶШ¬ЫҢ Ш§ШіШӘ.
Ъ©Ш§ШұЪ©ШұШҜ Щ…Ш«Ш§Щ„ ШІЫҢШұ ШЁШ§ Щ…Ш«Ш§Щ„вҖҢЩҮШ§ЫҢ ЩӮШЁЩ„ЫҢ Ъ©ЩҮ Ш§ШІ DefaultPolicy Ш§ШіШӘЩҒШ§ШҜЩҮ Щ…ЫҢвҖҢЪ©ШұШҜЩҶШҜШҢ ЫҢЪ©ЫҢ Ш§ШіШӘ ЩҲ ЩҮШҜЩҒ ШўЩҶШҢ Ш§Ш¬ШЁШ§Шұ ШЁЩҮ Ш§Ш№ШӘШЁШ§ШұШіЩҶШ¬ЫҢ Ъ©Ш§ШұШЁШұШ§ЩҶ ШҜШұ ЩҮЩ…ЩҮвҖҢШ¬Ш§ (ШӘЩ…Ш§Щ… endpoints)ШҢ Щ…ЩҶЩҮШ§ЫҢ Щ…ЩҲШ§ШұШҜЫҢ Ш§ШіШӘ Ъ©ЩҮ Ш§ШІ ЩҒЫҢЩ„ШӘШұ AllowAnonymous Ш§ШіШӘЩҒШ§ШҜЩҮ Щ…ЫҢвҖҢШҙЩҲШҜ:
public void ConfigureServices(IServiceCollection services)
{
    services.AddAuthorization(options =>
    {
        options.FallbackPolicy = new AuthorizationPolicyBuilder()
          .RequireAuthenticatedUser()
          .RequireScope("MyScope")
          .Build();
    });
}

public void Configure(IApplicationBuilder app)
{
    app.UseRouting();

    app.UseAuthentication();
    app.UseAuthorization();

    app.UseEndpoints(endpoints =>
    {
        endpoints.MapDefaultControllerRoute();
    });
}

[AllowAnonymous]
public class HomeController : ControllerBase
{
}


Ш§Щ…Ъ©Ш§ЩҶ Ш§Ш№Щ…Ш§Щ„ ШіЩҒШ§ШұШҙЫҢ Щ…ЫҢШ§ЩҶвҖҢШ§ЩҒШІШ§Шұ Authorization ШЁЩҮ EndpointЩҮШ§ЫҢ Щ…Ш¬ШІШ§

ШЁШұШ§ЫҢ Щ…Ш«Ш§Щ„ ШҜШұ Щ…ШЁШӯШ« В«ШЁШұШұШіЫҢ ШіЩ„Ш§Щ…ШӘ ШЁШұЩҶШ§Щ…ЩҮВ»ШҢ ЩҒШұЫҢЩ…вҖҢЩҲШұЪ©ШҢ Ш§Ш№ШӘШЁШ§ШұШіЩҶШ¬ЫҢ ШҜШұШ®ЩҲШ§ШіШӘвҖҢЩҮШ§ЫҢ ШўЩҶвҖҢШұШ§ ЩҫЫҢШҙ ШЁЫҢЩҶЫҢ ЩҶЪ©ШұШҜЩҮвҖҢШ§ШіШӘШӣ Ш§Щ…Ш§ Ш§ЫҢЩҶШЁШ§Шұ Щ…ЫҢвҖҢШӘЩҲШ§ЩҶ Ш§Ш№ШӘШЁШ§ШұШіЩҶШ¬ЫҢ ШұШ§ ШЁЩҮ endpoint ШўЩҶ Ш§Ш№Щ…Ш§Щ„ Ъ©ШұШҜ:
public void Configure(IApplicationBuilder app)
{
    ...

    app.UseRouting();

    app.UseAuthentication();
    app.UseAuthorization();

    app.UseEndpoints(endpoints =>
    {
        endpoints
            .MapHealthChecks("/healthz")
            .RequireAuthorization(new AuthorizeAttribute(){ Roles = "admin", });
    });
}
