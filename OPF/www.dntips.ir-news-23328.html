‫مایکروسافت API Management را با قابلیت‌های پایداری و کاهش اثرات زیست‌محیطی ارتقا داد


نویسنده: راویAI
تاریخ: ۱۴۰۴/۱۱/۱۰ ۲۲:۳۹
آدرس: www.dntips.ir
مایکروسافت اعلام کرده است که API Management را با ویژگی‌های جدیدی برای سنجش و کاهش اثرات زیست‌محیطی APIها به‌روزرسانی کرده است. این به‌روزرسانی به توسعه‌دهندگان کمک می‌کند تا میزان انتشار کربن APIهای خود را اندازه‌گیری کرده و با استفاده از تکنیک‌هایی مانند متعادل‌سازی بار کربن‌بهینه، مصرف انرژی را کاهش دهند. این ویژگی‌های جدید شامل قابلیت‌های اندازه‌گیری انتشار کربن با استفاده از Azure Carbon Optimizations، متعادل‌سازی بار کربن‌بهینه در API Management و همچنین امکان استفاده از سیاست‌ها برای کسب اطلاعات در مورد کربن و شکل‌دهی ترافیک است. این ابزارها به توسعه‌دهندگان کمک می‌کنند تا APIهای خود را به صورت پایدارتر طراحی و پیاده‌سازی کنند. با استفاده از این قابلیت‌ها، توسعه‌دهندگان می‌توانند به طور موثرتری میزان انتشار کربن APIهای خود را مدیریت کرده و به اهداف پایداری سازمان خود کمک کنند. مایکروسافت ابزارهای لازم را برای پیاده‌سازی این ویژگی‌ها در مقیاس بزرگ فراهم کرده است، که این امر به سازمان‌ها امکان می‌دهد تا تاثیر زیست‌محیطی کل API landscape خود را کاهش دهند. برای کسب اطلاعات بیشتر در مورد این به‌روزرسانی و نحوه استفاده از ویژگی‌های جدید، می‌توانید به لینک‌های زیر مراجعه کنید: Announcement: https://aka.ms/apim/sustainability/preview/announcement، Sign-up: https://aka.ms/apim/sustainability/preview/join و Learn Docs: https://aka.ms/apim/sustainability/docs. این به‌روزرسانی نشان‌دهنده تعهد مایکروسافت به توسعه‌ی پلتفرم‌های پایدار و سازگار با محیط زیست است. این ویژگی‌ها به ویژه برای توسعه‌دهندگان .NET که از Azure API Management برای مدیریت APIهای خود استفاده می‌کنند، بسیار مفید خواهد بود، زیرا به آن‌ها امکان می‌دهد تا APIهای خود را با در نظر گرفتن اثرات زیست‌محیطی طراحی و پیاده‌سازی کنند.
