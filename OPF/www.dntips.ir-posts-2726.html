ЪҜШұЩҲЩҮвҖҢЩҮШ§ЫҢ Щ…Ш·Ш§Щ„ШЁ ЫұЫ°ЫІЫҙ
ЩҶЩӮШҙЩҮвҖҢЩҮШ§ЫҢ ШұШ§ЩҮ ЫұЫұЫ№

→ вҖ«ШЁШұШұШіЫҢ Bad code smell ЩҮШ§: ШҜЩ„Ш§Щ„ ЫҢШ§ Middle Man
вҖ«ШӘШЁШҜЫҢЩ„ ЩӮШ§Щ„ШЁвҖҢЩҮШ§ЫҢ ШіЩҒШ§ШұШҙЫҢ ЩҫШұЩҲЪҳЩҮвҖҢЩҮШ§ЫҢ NET Core. ШЁЩҮ ШЁШіШӘЩҮвҖҢЩҮШ§ЫҢ NuGet ←
Ш№ЩҶЩҲШ§ЩҶ:

вҖ«ШұЩҲШҙвҖҢЩҮШ§ЫҢЫҢ ШЁШұШ§ЫҢ Щ…ШҜЫҢШұЫҢШӘ ШЁЩҮШӘШұ Ш№Щ…Щ„ЪҜШұЩҮШ§ЫҢ RxJS ШҜШұ ШЁШұЩҶШ§Щ…ЩҮвҖҢЩҮШ§ЫҢ Angular


ЩҶЩҲЫҢШіЩҶШҜЩҮ: ЩҲШӯЫҢШҜ ЩҶШөЫҢШұЫҢ
ШӘШ§ШұЫҢШ®: ЫұЫіЫ№Ы¶/Ы°Ыө/ЫұЫ№ ЫұЫі:ЫөЫө
ШўШҜШұШі: www.dntips.ir
Ъ©ШӘШ§ШЁШ®Ш§ЩҶЩҮвҖҢЫҢ RxJSШҢ Ш¬ШІЩҲ ЩҫШ§ЫҢЩҮвҖҢШ§ЫҢ Ъ©Ш§Шұ ШЁШ§ ШЁШұЩҶШ§Щ…ЩҮвҖҢЩҮШ§ЫҢ Angular Ш§ШіШӘ ЩҲ ШіШ§ШҜЩҮвҖҢШӘШұЫҢЩҶ ШұЩҲШҙ Ъ©Ш§Шұ ШЁШ§ ШўЩҶШҢ ШӘШ№ШұЫҢЩҒ ЫҢЪ© ШіШ·Шұ Ш°ЫҢЩ„ Ш§ШіШӘ:
 import {Observable} from 'rxjs';
ШЁЩҮ Ш§ЫҢЩҶ ШӘШұШӘЫҢШЁ ШӘЩ…Ш§Щ… Ш№Щ…Щ„ЪҜШұЩҮШ§ЫҢ RxJS Щ…Ш§ЩҶЩҶШҜ map ШҢdo ШҢcatch ЩҲ ШәЫҢШұЩҮ ЩҶЫҢШІ import Ш®ЩҲШ§ЩҮЩҶШҜ ШҙШҜ. Ш§Щ…Ш§ Ш§ЫҢЩҶ ШіШ§ШҜЪҜЫҢ ... ШЁЩҮ ЩӮЫҢЩ…ШӘ Ш§ЩҒШІЩҲШҜЩҮ ШҙШҜЩҶ ЫҢЪ© ШЁШіШӘЩҮвҖҢЫҢ 586 KB (ШәЫҢШұЩҒШҙШұШҜЩҮ) ШЁЩҮ ЩҒШ§ЫҢЩ„вҖҢЩҮШ§ЫҢ ЩҶЩҮШ§ЫҢЫҢШҢ ШӘЩ…Ш§Щ… Ш®ЩҲШ§ЩҮШҜ ШҙШҜ.


ШұЩҲШҙвҖҢЩҮШ§ЫҢ Щ…Ш®ШӘЩ„ЩҒ import ЩҲЫҢЪҳЪҜЫҢвҖҢЩҮШ§ЫҢ Ъ©ШӘШ§ШЁШ®Ш§ЩҶЩҮвҖҢЫҢ RxJS

Ш§Щ„ЩҒ) import ЩҮЩ…ЩҮ ЪҶЫҢШІ ШЁЩҮ ШөЩҲШұШӘ ЫҢЪ©Ш¬Ш§ 
 import Rx from "rxjs/Rx";
ЫҢЪ© Щ…Ш«Ш§Щ„
 Rx.Observable.of(1, 2, 3).map(i => i.toString());
Ш§ЫҢЩҶ ШұЩҲШҙ ШЁШіЫҢШ§Шұ ШіШ§ШҜЩҮвҖҢШ§ШіШӘ. Ш§Щ…Ш§ ШіШЁШЁ import Ъ©Ш§Щ…Щ„ ЫҢЪ© Ъ©ШӘШ§ШЁШ®Ш§ЩҶЩҮвҖҢЫҢ 586 KB ЩҶЫҢШІ Щ…ЫҢвҖҢШҙЩҲШҜ.

ШЁ) ШӘЩҶЩҮШ§ import ЩҲЫҢЪҳЪҜЫҢвҖҢЩҮШ§ЫҢ Щ…ЩҲШұШҜ ЩҶЫҢШ§ШІ 
import { Observable } from "rxjs/Observable";
import "rxjs/add/observable/of";
import "rxjs/add/operator/map";
ШЁЩҮ Ш§ЫҢЩҶ ШұЩҲШҙ В«patching the Observable prototypeВ» ЩҶЫҢШІ ЪҜЩҒШӘЩҮ Щ…ЫҢвҖҢШҙЩҲШҜ.
ЫҢЪ© Щ…Ш«Ш§Щ„
 Observable.of(1, 2, 3).map(i => i.toString());
Щ…ШІЫҢШӘ Ш§ЫҢЩҶ ШұЩҲШҙШҢ Ъ©Ш§ЩҮШҙ ЩӮШ§ШЁЩ„ Щ…Щ„Ш§ШӯШёЩҮвҖҢЫҢ ШӯШ¬Щ… ЩҶЩҮШ§ЫҢЫҢ ШЁШұЩҶШ§Щ…ЩҮ Ш§ШіШӘ. ЩҒЩӮШ· ШЁШ§ЫҢШҜ ШҜЩӮШӘ ШҜШ§ШҙШӘ Ъ©ЩҮ Щ…ШӘШҜЩҮШ§ЫҢ Щ…ЩҲШұШҜ ЩҶЫҢШ§ШІШҢ ШӯШӘЩ…Ш§ import ШҙШҜЩҮ ШЁШ§ШҙЩҶШҜ.

Ш¬) ЩҒШұШ§Ш®ЩҲШ§ЩҶЫҢ Щ…ШіШӘЩӮЫҢЩ… Щ…ШӘШҜЩҮШ§ЫҢ RxJS ЩҶЩҮ Ш§ШІ Ш·ШұЫҢЩӮ Observable 
import { Observable } from "rxjs/Observable";
import { of } from "rxjs/observable/of";
import { map } from "rxjs/operator/map";
ЫҢЪ© Щ…Ш«Ш§Щ„
 const source = of(1, 2, 3);
const mapped = map.call(source, i => i.toString());
ШҜШұ Ш§ЫҢЩҶ ШӯШ§Щ„ШӘ Ъ©ШҜ ШЁЫҢШҙШӘШұЫҢ ШЁШ§ЫҢШҜ ЩҶЩҲШҙШӘЩҮ ШҙЩҲШҜ ЩҲ Щ…ШІЫҢШӘ Ш§Ш·Щ„Ш§Ш№Ш§ШӘ ЩҶЩҲШ№ Щ…ШӘШҜ ШұШ§ ЩҶЫҢШІ Ш§ШІ ШҜШіШӘ Щ…ЫҢвҖҢШҜЩҮЫҢЩ…. Ш§ШІ Ш§ЫҢЩҶ Ш¬ЩҮШӘ Ъ©ЩҮ Ш§ЫҢЩҶ Щ…ШӘШҜЩҮШ§ ШІЩ…Ш§ЩҶЫҢЪ©ЩҮ ШЁШ§ Ш§ЫҢЩҶ ШұЩҲШҙ import Щ…ЫҢвҖҢШҙЩҲЩҶШҜШҢ any ШұШ§ ШЁШ§ШІЪҜШҙШӘ Щ…ЫҢвҖҢШҜЩҮЩҶШҜ (Function.prototype.call Ш®ШұЩҲШ¬ЫҢ Ш§ШІ ЩҶЩҲШ№ any ШҜШ§ШұШҜ).


Щ…ЩӮЫҢШҜ Ъ©ШұШҜЩҶ ШЁШұЩҶШ§Щ…ЩҮ ШЁЩҮ Ш№ШҜЩ… Ш§ШіШӘЩҒШ§ШҜЩҮ Ш§ШІ ШӯШ§Щ„ШӘ В«Ш§Щ„ЩҒВ» ЩҲ Ш§Ш¬ШЁШ§Шұ ШЁЩҮ Ш§ШіШӘЩҒШ§ШҜЩҮ Ш§ШІ ШӯШ§Щ„ШӘ В«ШЁВ»

Ш§ЪҜШұ ШЁЩҮ ШұЫҢШҙЩҮвҖҢЫҢ ЩҫЩҲШҙЩҮвҖҢЫҢ ЩҫШұЩҲЪҳЩҮвҖҢЩҮШ§ЫҢ Щ…ШЁШӘЩҶЫҢ ШЁШұ Angular CLI ШҜЩӮШӘ Ъ©ЩҶЫҢШҜШҢ ЩҒШ§ЫҢЩ„ tslint.json ЩҶЫҢШІ ШҜШұ ШўЩҶвҖҢЩҮШ§ ЩӮШ§ШЁЩ„ Щ…ШҙШ§ЩҮШҜЩҮ Ш§ШіШӘ ЩҲ Ш§ЪҜШұ Ш§ЩҒШІЩҲЩҶЩҮвҖҢЫҢ VSCode ШўЩҶвҖҢШұШ§ ЩҶЫҢШІ ЩҶШөШЁ Ъ©ШұШҜЩҮ ШЁШ§ШҙЫҢШҜШҢ ШҜШұ ШӯЫҢЩҶ Ъ©Ш§Шұ ШЁШ§ VSCodeШҢ Ш®Ш·Ш§ЩҮШ§ЫҢ Щ…ШұШӘШЁШ· ШұШ§ ШҜШұЩҲЩҶ Ш§ШҜЫҢШӘЩҲШұ Щ…ШҙШ§ЩҮШҜЩҮ Ш®ЩҲШ§ЩҮЫҢШҜ Ъ©ШұШҜ. TSLintШҢ ЩӮШ§ШЁЩ„ЫҢШӘ ШӘЩҲШіШ№ЩҮ ШҜШ§ШҙШӘЩҮ ЩҲ ЫҢЪ© ЩҶЩ…ЩҲЩҶЩҮвҖҢЫҢ Ш§ШІ Ш§ЫҢЩҶвҖҢЩҮШ§ШҢ ШЁШіШӘЩҮвҖҢЫҢ TSLint rules for RxJS Ш§ШіШӘ. ШЁШұШ§ЫҢ ЩҶШөШЁ ШўЩҶ Ш§ШЁШӘШҜШ§ ШҜШіШӘЩҲШұ Ш°ЫҢЩ„ ШұШ§ ШөШ§ШҜШұ Ъ©ЩҶЫҢШҜ:
 npm install rxjs-tslint-rules --save-dev
ШіЩҫШі ЩҒШ§ЫҢЩ„ tslint.json ШұШ§ ЪҜШҙЩҲШҜЩҮ ЩҲ ШӘШәЫҢЫҢШұШ§ШӘ Ш°ЫҢЩ„ ШұШ§ ШЁЩҮ ШўЩҶ Ш§Ш№Щ…Ш§Щ„ ЩҶЩ…Ш§ШҰЫҢШҜ:
{
  "rulesDirectory": [
    "node_modules/codelyzer"
  ],
  "extends": [
    "rxjs-tslint-rules"
  ],  
  "rules": {
    "rxjs-add": { "severity": "error" },
    "rxjs-no-patched": { "severity": "error" },
    "rxjs-no-unused-add": { "severity": "error" },
    "rxjs-no-wholesale": { "severity": "error" },
    "rxjs-no-subject-unsubscribe": { "severity": "error" },
rxjs-no-wholesale ШўЩҶ ШіШЁШЁ Щ…ЩҶШ№ ШҜШұШ¬ ШӯШ§Щ„ШӘ В«Ш§Щ„ЩҒВ» Щ…ЫҢвҖҢШҙЩҲШҜ ЩҲ ШіШ§ЫҢШұ ШӯШ§Щ„Ш§ШӘ ШұЩҲШҙ В«ШЁВ» ШұШ§ ШӘШҙЩҲЫҢЩӮ Щ…ЫҢвҖҢЪ©ЩҶЩҶШҜ.



Щ…ШҜЫҢШұЫҢШӘ ШЁЩҮШӘШұ ШӯШ§Щ„ШӘ В«ШЁВ» ЫҢШ§ В«ШӘЩҶЩҮШ§ import ЩҲЫҢЪҳЪҜЫҢвҖҢЩҮШ§ЫҢ Щ…ЩҲШұШҜ ЩҶЫҢШ§ШІВ»

ШІЩ…Ш§ЩҶЫҢЪ©ЩҮ Ш§ШІ ШұЩҲШҙ В«ШЁВ» Ш§ШіШӘЩҒШ§ШҜЩҮ Щ…ЫҢвҖҢЪ©ЩҶЫҢЩ…ШҢ Ъ©Щ„Ш§ШівҖҢЩҮШ§ЫҢ ШіШұЩҲЫҢШі ШЁШұЩҶШ§Щ…ЩҮ ЩҫШұ Ш®ЩҲШ§ЩҮЩҶШҜ ШҙШҜ Ш§ШІ Ъ©ШҜЩҮШ§ЫҢ ШӘЪ©ШұШ§ШұЫҢ Ш°ЫҢЩ„:
import 'rxjs/add/operator/map';
import 'rxjs/add/operator/do';
import 'rxjs/add/operator/catch';
ШҜШұ Ш§ЫҢЩҶШ¬Ш§ ЩҮШұ Щ…ШӘШҜЫҢ ШұШ§ Ъ©ЩҮ ЩҶЫҢШ§ШІ Ш§ШіШӘ ШЁШ§ЫҢШҜ ЫҢЪ©ШЁШ§Шұ import Ъ©ШұШҜ ЩҲ Ш§ЫҢЩҶвҖҢЪ©Ш§Шұ ШұШ§ ШЁШ§ЫҢШҜ ШЁЩҮ Ш§ШІШ§ЫҢ ШӘЪ©вҖҢШӘЪ© ШіШұЩҲЫҢШівҖҢЩҮШ§ЫҢ ШЁШұЩҶШ§Щ…ЩҮ ЩҶЫҢШІ ШӘЪ©ШұШ§Шұ ЩҶЩ…ЩҲШҜ.
ШЁШұШ§ЫҢ Щ…ШҜЫҢШұЫҢШӘ ШЁЩҮШӘШұ Ш§ЫҢЩҶЪ©Ш§ШұШҢ ЩҒШ§ЫҢЩ„ Ш¬ШҜЫҢШҜЫҢ ШұШ§ ШЁЩҮ ЩҶШ§Щ… src\app\shared\rxjs-operators.ts Ш§ЫҢШ¬Ш§ШҜ Щ…ЫҢвҖҢЪ©ЩҶЫҢЩ…Шӣ ШЁШ§ Щ…ШӯШӘЩҲШ§ЫҢ Ш°ЫҢЩ„:

ШҜШұ Ш§ЫҢЩҶШ¬Ш§ ШӘЩ…Ш§Щ… importЩҮШ§ЫҢ Щ…ЩҲШұШҜ ЩҶЫҢШ§ШІ ШЁШұЩҶШ§Щ…ЩҮ ШЁЩҮ ШөЩҲШұШӘ ЫҢЪ©Ш¬Ш§ ШҜШұШ¬ Щ…ЫҢвҖҢШҙЩҲЩҶШҜ.
ШіЩҫШі Ъ©Ш§ЩҒЫҢ Ш§ШіШӘ ШЁЩҮ ЩҒШ§ЫҢЩ„ src\app\app.module.ts Щ…ШұШ§Ш¬Ш№ЩҮ Ъ©ШұШҜЩҮ ЩҲ Ш§ЫҢЩҶ ЩҒШ§ЫҢЩ„ ШұШ§ import Ъ©ЩҶЫҢЩ…:

//...

ШЁШ§ Ш§ЫҢЩҶЪ©Ш§Шұ ШіШЁШЁ Ш®ЩҲШ§ЩҮЫҢЩ… ШҙШҜ ШӘШ§ ШҜЫҢЪҜШұ ШҜШұ ШіШұЩҲЫҢШівҖҢЩҮШ§ЫҢ ШЁШұЩҶШ§Щ…ЩҮ (ЩҲ ШӘЩ…Ш§Щ… ЩӮШіЩ…ШӘвҖҢЩҮШ§ЫҢ ШўЩҶ) ЩҶЫҢШ§ШІЫҢ ШЁЩҮ ШӘШ№ШұЫҢЩҒвҖҢЩҮШ§ЫҢ ШӘЪ©ШұШ§ШұЫҢ RxJS ЩҲШ¬ЩҲШҜ ЩҶШҜШ§ШҙШӘЩҮ ШЁШ§ШҙШҜ ЩҲ ШӘЩҶЩҮШ§ ШӘШ№ШұЫҢЩҒЫҢ Ъ©ЩҮ ШҜШұ ШўЩҶШ¬Ш§ ЩҶЫҢШ§ШІ Ш§ШіШӘШҢ ЫҢЪ© Щ…ЩҲШұШҜ Ш°ЫҢЩ„ Ш§ШіШӘ:
 import { Observable } from "rxjs/Observable";
Ш№Щ„ШӘ Ш§ЫҢЩҶШ¬Ш§ Ш§ШіШӘШҢ ЩҮЩ…Ш§ЩҶШ·ЩҲШұ Ъ©ЩҮ Ш№ЩҶЩҲШ§ЩҶ ШҙШҜШҢ ШЁЩҮ Ш§ЫҢЩҶ ШұЩҲШҙ В«patching the Observable prototypeВ» ЩҶЫҢШІ ЪҜЩҒШӘЩҮ Щ…ЫҢвҖҢШҙЩҲШҜ. ШЁЩҮ Ш§ЫҢЩҶ Щ…Ш№ЩҶШ§ Ъ©ЩҮ Ш§ЫҢЩҶ Ш№Щ…Щ„ЪҜШұЩҮШ§ ЩҲ ЫҢШ§ Ш§ЩҒШІЩҲЩҶЩҮвҖҢЩҮШ§ШҢ Ш¬ШІШҰЫҢ Ш§ШІ Ъ©Щ„Ш§Ші Observable Щ…ЫҢвҖҢШҙЩҲЩҶШҜ ЩҲ ШҜШұ ШӘЩ…Ш§Щ… ЩӮШіЩ…ШӘвҖҢЩҮШ§ЫҢ ШЁШұЩҶШ§Щ…ЩҮШҢ ЩҫШі Ш§ШІ import ШўЩҶвҖҢЩҮШ§ ЩӮШ§ШЁЩ„ ШҜШіШӘШұШіЫҢ Ш®ЩҲШ§ЩҮЩҶШҜ ШЁЩҲШҜ.
