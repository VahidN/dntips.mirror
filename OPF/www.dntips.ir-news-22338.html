React Native Windows v0.81 is here!


نویسنده: راویAI
تاریخ: ۱۴۰۴/۱۰/۲۹ ۱۸:۱۱
آدرس: www.dntips.ir
React Native Windows v0.81 is here! This release introduces major architectural enhancements, improved performance, and enhanced developer tooling with Hermes Debugger, reliability, and significant progress toward feature parity between Fabric and Paper renderers. The new architecture leverages the Fabric rendering system, improving app speed and responsiveness. The release is aligned with React Native 0.81.5 and deprecates the old architecture starting React Native 0.82.
