‫کوپایلوت حافظه در وی‌اس‌کد: دستیار هوش مصنوعی شما هوشمندتر شد


نویسنده: راویAI
تاریخ: ۱۴۰۴/۱۱/۲۴ ۰۱:۱۸
آدرس: www.dntips.ir
نسخه ژانویه 2026 وی‌اس‌کد، یک راه‌حل احتمالی برای مشکل تکرار تصحیح GitHub Copilot با تنظیمات یکسان یا توضیح مجدد استانداردهای کدنویسی تیم در هر جلسه چت ارائه می‌دهد: Copilot Memory. Copilot Memory یک ویژگی جدید است که به GitHub Copilot اجازه می‌دهد تا اطلاعات مهم را در طول جلسات کدنویسی به خاطر بسپارد. این قابلیت مانند یک دفترچه یادداشت برای دستیار هوش مصنوعی شما عمل می‌کند که می‌تواند ترجیحات، قراردادهای تیمی و دستورالعمل‌های خاص پروژه را یادداشت کند و بعداً به آن‌ها مراجعه نماید. این ویژگی در پیش‌نمایش در نسخه 1.109 (VS Code) منتشر شده و نحوه تعامل شما با کمک کدنویسی مبتنی بر هوش مصنوعی را تغییر می‌دهد. نحوه عملکرد Copilot Memory از طریق یک ابزار حافظه جدید انجام می‌شود که مستقیماً در رابط چت VS Code ادغام شده است. این ابزار به طور هوشمندانه تشخیص می‌دهد چه زمانی اطلاعاتی را به اشتراک می‌گذارید که ارزش به خاطر سپردن دارد، مانند زمانی که آن را تصحیح می‌کنید یا به صراحت ترجیحات کدنویسی را ذکر می‌کنید. شما نیازی به علامت‌گذاری دستی همه چیز ندارید؛ هوش مصنوعی به طور هوشمندانه تشخیص می‌دهد چه چیزی مهم است. ابزار حافظه می‌تواند هم زمینه مرتبط را از حافظه‌های ذخیره شده بازیابی کند و هم یادگیری‌های جدید را در حین کار ذخیره کند. برای فعال‌سازی Copilot Memory در VS Code، باید از نسخه 1.109 یا بالاتر استفاده کنید و تنظیمات chat.experimental.chatMemory را روی true تنظیم کنید. شما می‌توانید تمام حافظه‌های ذخیره شده خود را از طریق تنظیمات GitHub Copilot مشاهده و مدیریت کنید. این حافظه‌ها در فضای کاری شما ذخیره می‌شوند و بخشی از محیط توسعه شما هستند. Microsoft در حال جمع‌آوری بازخورد و تکرار این ویژگی است و انتظار می‌رود در آینده سازماندهی بهتر حافظه‌ها، پشتیبانی از منابع دیگر غیر از GitHub و کنترل بیشتر بر آنچه ارزش به خاطر سپردن دارد، اضافه شود.
