NSA advises organizations to consider making a strategic shift from programming languages that provide little or no inherent memory protection, such as C/C++, to a memory safe language when possible. Some examples of memory safe languages are C#, Go, Java, Rubyв„ў, and SwiftВ®.

    Ш§ШҙШӘШұШ§Ъ©вҖҢЩҮШ§ШЁШ§ШІЩҶЩҲЫҢШіЫҢ ШІЫҢШұШіШ§Ш®ШӘвҖҢЩҮШ§ЫҢ Android ШЁШ§ Rust ЩҲ Ъ©Ш§ЩҮШҙ 52 ШҜШұШөШҜЫҢ Щ…ШҙЪ©Щ„Ш§ШӘ Ш§Щ…ЩҶЫҢШӘЫҢ ШўЩҶШ§ШҙШӘШұШ§Ъ©вҖҢЩҮШ§Ш§ЫҢЩҶШЁШ§Шұ ШҜШұШ®ЩҲШ§ШіШӘ FBI ШЁШұШ§ЫҢ Ш®Ш§ШӘЩ…ЩҮвҖҢЫҢ Ш§ШіШӘЩҒШ§ШҜЩҮ Ш§ШІ C ЩҲ ++CШ§ШҙШӘШұШ§Ъ©вҖҢЩҮШ§30 ШіШ§Щ„ ШЁШ§ ++CШ§ШҙШӘШұШ§Ъ©вҖҢЩҮШ§ШіШ§ШІЩ…Ш§ЩҶ ЩҫШұЩҲЪҳЩҮвҖҢЩҮШ§ЫҢ ЩҫЪҳЩҲЩҮШҙЫҢ ЩҫЫҢШҙШұЩҒШӘЩҮЩ” ШҜЩҒШ§Ш№ЫҢ (DARPA): ШӘЩ…Ш§Щ… Ъ©ШҜЩҮШ§ЫҢ C ШЁШ§ЫҢШҜ ШЁЩҮ Rust ШӘШЁШҜЫҢЩ„ ШҙЩҲЩҶШҜ!Ш§ШҙШӘШұШ§Ъ©вҖҢЩҮШ§ШӘЩҲШөЫҢЩҮвҖҢЫҢ Щ…ШҙШ§ЩҲШұШ§ЩҶ Ш§Щ…ЩҶЫҢШӘЫҢ: Ш§ШІ Rust ЩҲ ЫҢШ§ #C Ш§ШіШӘЩҒШ§ШҜЩҮ Ъ©ЩҶЫҢШҜ ЩҲ ++C ШұШ§ ШұЩҮШ§ Ъ©ЩҶЫҢШҜ!Ш§ШҙШӘШұШ§Ъ©вҖҢЩҮШ§Ъ©ШӘШ§ШЁ ШІШЁШ§ЩҶ ШЁШұЩҶШ§Щ…ЩҮвҖҢЩҶЩҲЫҢШіЫҢ ++CШҢ ЩҶЪҜШ§ШұШҙ ЪҶЩҮШ§ШұЩ…Ш§ШҙШӘШұШ§Ъ©вҖҢЩҮШ§ЩҫЩҲШіШӘШұ Щ…ШұШ¬Ш№ ШіШұЫҢШ№ #C ЩҲ SwiftШ§ШҙШӘШұШ§Ъ©вҖҢЩҮШ§ЩҫЫҢШҙЩҶЩҮШ§ШҜ Ш·ШұШ§ШӯЫҢ ++C Ш§Щ…ЩҶШ§ШҙШӘШұШ§Ъ©вҖҢЩҮШ§ШҜЩҲШұЩҮ #C ШЁШұШ§ЫҢ ШӘШ§ШІЩҮвҖҢЪ©Ш§ШұЩҮШ§ЩҶШёШұШ§ШӘ Ш§ШҙШӘШұШ§Ъ©вҖҢЩҮШ§ШӘЩҲШөЫҢЩҮвҖҢЫҢ Щ…ШҙШ§ЩҲШұШ§ЩҶ Ш§Щ…ЩҶЫҢШӘЫҢ: Ш§ШІ Rust ЩҲ ЫҢШ§ #C Ш§ШіШӘЩҒШ§ШҜЩҮ Ъ©ЩҶЫҢШҜ ЩҲ ++C ШұШ§ ШұЩҮШ§ Ъ©ЩҶЫҢШҜ!
