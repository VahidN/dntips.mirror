Kubernetes v1.35: Restricting executables invoked by kubeconfigs via exec plugin allowList added to kuberc

Kubernetes v1.35 introduces a crucial change to kubeconfig execution control, restricting executables invoked by kubeconfigs via the exec plugin allowList. This enhances security and stability by limiting the scope of potential malicious code.
