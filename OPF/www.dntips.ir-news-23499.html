‫دسترسی به پوشه‌های شناخته‌شده ویندوز در C# با استفاده از SHGetKnownFolderPath


نویسنده: راویAI
تاریخ: ۱۴۰۴/۱۱/۱۴ ۱۸:۳۷
آدرس: www.dntips.ir
این مقاله روش دسترسی به پوشه‌های شناخته‌شده ویندوز مانند Documents، Pictures و Music در C# را توضیح می‌دهد. روش Environment.GetFolderPath معمولاً برای این کار استفاده می‌شود، اما این روش محدود به پوشه‌های تعریف‌شده در enumeration Environment.SpecialFolder است. برای دسترسی به پوشه‌های دیگر مانند Desktop یا Downloads، می‌توان از متد SHGetKnownFolderPath از Windows API استفاده کرد. برای این منظور، می‌توان از Microsoft.Windows.CsWin32 برای تولید bindings C# برای این APIها استفاده نمود. کد نمونه‌ای در مقاله نشان داده شده است که نحوه استفاده از این متد را با استفاده از GUID مربوط به پوشه Desktop نشان می‌دهد. در این مقاله همچنین به معرفی NuGet package Meziantou.Framework.FullPath اشاره شده است که امکان دسترسی آسان‌تر به پوشه‌های شناخته‌شده را فراهم می‌کند. این package با استفاده از متد FullPath.GetKnownFolderPath(KnownFolder.Downloads) به راحتی مسیر پوشه Downloads را بدست می‌آورد. در نهایت، این مقاله راهنمایی مفیدی برای توسعه‌دهندگان .NET است که نیاز به دسترسی به پوشه‌های شناخته‌شده ویندوز در برنامه‌های C# خود دارند و به دنبال راهکارهای جایگزین برای محدودیت‌های Environment.GetFolderPath هستند. استفاده از SHGetKnownFolderPath یا NuGet package Meziantou.Framework.FullPath می‌تواند این فرآیند را ساده‌تر کند.
