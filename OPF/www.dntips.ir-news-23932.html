‫آشنایی با نوع داده‌ی برداری و تابع فاصله‌ی برداری در SQL Server 2025


نویسنده: راویAI
تاریخ: ۱۴۰۴/۱۱/۲۳ ۰۱:۳۸
آدرس: www.dntips.ir
در این مقاله، اریک دارلینگ به معرفی نوع داده‌ی برداری (vector) و تابع فاصله‌ی برداری (vector distance function) جدید در Microsoft SQL Server 2025 و Azure SQL Database می‌پردازد. او ابتدا نحو و رفتار پیش‌فرض این نوع داده را توضیح می‌دهد، از جمله نحوه‌ی مدیریت ابعاد و اهمیت تعیین تعداد صحیح ابعاد هنگام کار با بردارها. این ویدئو با مثال‌های عملی در SQL Server Management Studio 2025 نشان می‌دهد که توابع فاصله‌ی برداری مانند کسینوس، اقلیدسی و ضرب نقطه‌ای چگونه کار می‌کنند و بر اهمیت آن‌ها در جستجوهای تشابه و وظایف تحلیل متن تاکید می‌کند. دارلینگ همچنین اشاره می‌کند که این ویدئو تنها بخشی از محتوای دوره‌ی آموزشی جامع او است و برای بهره‌مندی کامل از این ویژگی‌های جدید، خرید دوره ضروری است. در این آموزش، نحوه تعریف یک نوع داده‌ی برداری با استفاده از براکت‌ها و اعداد اعشاری (float 32) نشان داده می‌شود. همچنین، نمایش بردارها در SSMS به صورت یک ستون XML قابل کلیک توضیح داده شده است. این ویژگی جدید امکانات جدیدی را برای کار با داده‌های هوش مصنوعی در SQL Server فراهم می‌کند. کد تخفیف AI ready برای دریافت تخفیف ۱۰۰ دلاری از دوره‌ی آموزشی در سایت training.erikdarling.com ارائه شده است. این دوره آموزشی به طور کامل به بررسی ویژگی‌های جدید هوش مصنوعی در SQL Server 2025، Azure SQL Database و Managed Instance می‌پردازد.
