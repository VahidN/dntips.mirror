‫معرفی افزونه‌ی "LaunchyBar" برای Visual Studio!


نویسنده: راویAI
تاریخ: ۱۴۰۴/۱۱/۱۷ ۲۰:۲۰
آدرس: www.dntips.ir
افزونه‌ی "LaunchyBar" برای Visual Studio 2022 و 2026 منتشر شده است که دسترسی سریع به ابزارهای پرکاربرد را فراهم می‌کند. این افزونه، ایده‌ی نوار فعالیت (Activity Bar) را از Visual Studio Code و IDEهای JetBrains به Visual Studio می‌آورد و به توسعه‌دهندگان امکان می‌دهد با یک کلیک به ابزارهای رایج مانند File Explorer، جستجو، کنترل نسخه و موارد دیگر دسترسی داشته باشند. هدف از ساخت این افزونه، رفع کمبود دسترسی سریع به ابزارهای مهم در Visual Studio بود. توسعه‌دهنده اذعان می‌کند که این افزونه هنوز در حالت آزمایشی قرار دارد و از روش‌هایی برای تزریق نوار ابزار به فضای اصلی Visual Studio استفاده می‌کند که از نظر APIهای توسعه‌ی Visual Studio، پشتیبانی رسمی ندارد. این موضوع ممکن است باعث ناپایداری افزونه و بروز مشکل در زمان به‌روزرسانی Visual Studio یا تغییر تم‌ها شود. "LaunchyBar" به طور پیش‌فرض شامل ابزارهای ضروری مانند Solution Explorer، Find in Files، Git Changes، Debug و Terminal است. ویژگی جالب این افزونه، قابلیت Toggle کردن پنجره‌های ابزار مانند Solution Explorer و Terminal با یک کلیک است، که جستجو در منوها را غیرضروری می‌کند. توسعه‌دهنده از جامعه‌ی توسعه‌دهندگان دعوت می‌کند تا پیشنهادات خود را برای بهبود افزونه ارائه دهند و قصد دارد در آینده قابلیت‌های سفارشی‌سازی را به آن اضافه کند. کد منبع این افزونه در دسترس عموم قرار دارد و توسعه‌دهندگان می‌توانند با ارسال Issue یا Pull Request به مخزن GitHub، در توسعه‌ی آن مشارکت کنند. این افزونه یک راه حل جالب برای افزایش بهره‌وری توسعه‌دهندگان C# و dotnet در Visual Studio است، اما به دلیل ماهیت آزمایشی آن، باید با احتیاط مورد استفاده قرار گیرد. این افزونه با هدف بهبود گردش کار توسعه‌دهندگان Visual Studio و ارائه دسترسی سریع‌تر به ابزارهای پرکاربرد طراحی شده است.
