‫اجرای سریع فایل‌های سی‌شارپ در وی‌اس‌کد با بهره‌گیری از دات‌نت ۱۰


نویسنده: وحید نصیری
تاریخ: ۱۴۰۴/۰۳/۱۹ ۰۸:۴۹
آدرس: www.dntips.ir
این مطلب به قابلیت جدیدی در پیش‌نمایش (preview) دات‌نت ۱۰ اشاره می‌کند که امکان اجرای مستقیم فایل‌های منفرد سی‌شارپ (.cs) را از محیط وی‌اس‌کد (VS Code) فراهم می‌آورد. هدف اصلی این قابلیت، تسهیل و تسریع فرآیند توسعه برای برنامه‌نویسان است.

ویژگی اصلی: اجرای فایل‌ها باdotnet run
به لطف ویژگی جدید dotnet run در دات‌نت ۱۰ (نسخه پیش‌نمایش)، توسعه‌دهندگان قادر خواهند بود تا فایل‌های سی‌شارپ مجزا را مستقیماً و بدون نیاز به پروژه کامل، از وی‌اس‌کد اجرا کنند. این قابلیت، فرآیند آزمایش و اجرای قطعه کدها یا فایل‌های کوچک را به شکل قابل توجهی ساده می‌کند.

پیکربندی وی‌اس‌کد باlaunch.json
برای فعال‌سازی قابلیت "اجرا با یک کلیک" در وی‌اس‌کد، یک تنظیم ساده در فایل launch.json مورد نیاز است. این پیکربندی به وی‌اس‌کد آموزش می‌دهد که چگونه فایل فعال سی‌شارپ را با استفاده از دستور dotnet run اجرا کند. تنظیمات پیشنهادی به شرح زیر است:
{
  "version": "0.2.0",
  "configurations": [
    {
      "name": ".NET: Launch Active File",
      "type": "coreclr",
      "request": "launch",
      "program": "dotnet",
      "args": ["run", "${file}"],
      "cwd": "${workspaceFolder}",
      "stopAtEntry": false,
      "console": "internalConsole"
    }
  ]
}
در این پیکربندی، name نامی برای پیکربندی را مشخص می‌کند، type نوع دیباگر (debugger) را coreclr تعیین می‌کند، request حالت اجرا را launch قرار می‌دهد و program به اجرای dotnet اشاره دارد. آرگومان‌های (args) run و ${file} به دات‌نت دستور می‌دهند که فایل فعال فعلی را اجرا کند. cwd پوشه کاری فعلی را به عنوان ریشه پروژه (workspaceFolder) تعیین می‌کند و console خروجی را در کنسول داخلی وی‌اس‌کد نمایش می‌دهد.

نحوه استفاده و اجرای سریع
پس از ذخیره کردن فایل launch.json با پیکربندی فوق، توسعه‌دهندگان می‌توانند به سادگی با باز کردن یک فایل سی‌شارپ در وی‌اس‌کد و فشردن کلید F5، فایل فعال را به صورت فوری اجرا کنند. این امر به ویژه برای نمونه‌سازی سریع، تست کدهای مستقل، یا تمرین با قطعات کد سی‌شارپ بسیار مفید است.
