‫شبیه‌سازی تابع `contrast-color()` با ویژگی‌های CSS دیگر


نویسنده: راویAI
تاریخ: ۱۴۰۴/۱۱/۲۳ ۰۹:۱۹
آدرس: www.dntips.ir
مقاله‌ای در مورد شبیه‌سازی تابع contrast-color() در CSS با استفاده از ویژگی‌های دیگر ارائه شده است. این تابع که در حال حاضر فقط توسط سافاری و فایرفاکس پشتیبانی می‌شود، برای محاسبه رنگ پیش‌زمینه مناسب با توجه به رنگ پس‌زمینه استفاده می‌شود. نویسنده در این مقاله روشی را برای پیاده‌سازی این قابلیت به صورت سازگار با مرورگرهای مختلف ارائه می‌دهد. نویسنده با بررسی فرمول‌های WCAG 2.2 برای محاسبه کنتراست رنگ، به یک راه حل مبتنی بر محاسبات ریاضی و استفاده از رنگ‌های RGB دست یافته است. این راه حل شامل محاسبه درخشندگی (luminance) رنگ‌ها و استفاده از تابع round() برای تعیین رنگ مناسب (سفید یا سیاه) با توجه به کنتراست است. در نهایت، از نحو رنگی نسبی (rgb(from)) برای اعمال رنگ محاسبه شده استفاده می‌شود. با وجود اینکه این پیاده‌سازی پیچیده و دشوار خواندنی است، اما یک راه حل موقت برای استفاده از قابلیت contrast-color() در مرورگرهایی که از آن پشتیبانی نمی‌کنند، ارائه می‌دهد. نویسنده همچنین به معرفی رویکرد جدیدی با استفاده از فضاهای رنگی (color spaces) و مقدار L* (درخشندگی) در فضای رنگ CIELAB اشاره می‌کند که می‌تواند جایگزین بهتری برای روش فعلی باشد. در نهایت، نویسنده تاکید می‌کند که این پیاده‌سازی به دلیل پیچیدگی و دشواری نگهداری، یک راه حل ایده‌آل نیست و در آینده با معرفی فرمول‌های جدیدتر (مانند APCA) ممکن است نیاز به بازنگری داشته باشد. با این حال، این مقاله یک بررسی جالب از روش‌های جایگزین برای پیاده‌سازی قابلیت contrast-color() در CSS ارائه می‌دهد.
