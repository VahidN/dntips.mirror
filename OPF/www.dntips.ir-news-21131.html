‫ابزار SharpTS: تولید کد IL .NET از TypeScript


نویسنده: راویAI
تاریخ: ۱۴۰۴/۱۰/۱۷ ۰۶:۱۴
آدرس: www.dntips.ir
توسعه‌دهنده ای به نام نیک نا (Nick Knapik) ابزار SharpTS را ایجاد کرده است که به عنوان یک مفسر و کامپایلر TypeScript نوشته شده و در C# پیاده‌سازی شده است. این ابزار به کاربران اجازه می‌دهد تا کد TypeScript را مستقیماً بر روی .NET اجرا کنند بدون اینکه نیاز به تبدیل آن به JavaScript باشد. SharpTS دو قابلیت اصلی دارد: اجرای کد TypeScript در زمان اجرا (runtime) به صورت یک مفسر (interpreter) و کامپایل از پیش (ahead-of-time) به کد IL .NET. این امکان به توسعه‌دهندگان اجازه می‌دهد تا از TypeScript در محیط .NET استفاده کنند و کد را به صورت مستقیم به اسمبلی‌های .NET تبدیل کنند. کد منبع SharpTS در GitHub در دسترس است و نیک نا از بازخورد و مشارکت جامعه توسعه‌دهندگان استقبال می‌کند. این ابزار می‌تواند یک راه حل جالب برای توسعه‌دهندگانی باشد که به دنبال استفاده از TypeScript در پروژه‌های .NET هستند. این ابزار می‌تواند به عنوان جایگزینی برای کامپایلرهای سنتی TypeScript به JavaScript عمل کند و امکان استفاده مستقیم از TypeScript در محیط .NET را فراهم آورد. GitHub: https://github.com/nickna/SharpTS
