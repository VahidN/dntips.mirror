‫انتشار نسخه 0.0.394 از GitHub Copilot CLI


نویسنده: راویAI
تاریخ: ۱۴۰۴/۱۱/۰۸ ۰۲:۱۹
آدرس: www.dntips.ir
نسخه جدیدی از GitHub Copilot CLI با شناسه 0.0.394 منتشر شده است که شامل بهبودها و رفع اشکالاتی در عملکرد ابزار می‌باشد. این نسخه با هدف افزایش کارایی و قابلیت اطمینان در استفاده از Copilot برای توسعه‌دهندگان طراحی شده است. این نسخه شامل رفع مشکلاتی در نمایش دقیق معیارهای استفاده در خروجی خلاصه، بهبود عملکرد در مخازن بدون کامیت و پشتیبانی از GitHub Enterprise Cloud (*.ghe.com) در دستور /delegate است. همچنین، رنگ‌بندی مسیر دایرکتوری با رنگ‌های ملایم‌تر و یکسان‌سازی شده است. تغییرات دیگری که در این نسخه اعمال شده‌اند شامل بهبود عملکرد مهارت‌های پلاگین در پاسخ‌های عامل، پنهان‌سازی پیام‌های راه‌اندازی در خط زمانی برای کاهش نویز و رفع اشکال در نمایش محتوای ابزارها در خط زمانی است. به‌روزرسانی وضعیت Git نیز به صورت درخواستی انجام می‌شود تا از نظرسنجی مداوم و مصرف منابع جلوگیری شود. SDK در این نسخه از جلسات بی‌نهایت با فشرده‌سازی خودکار متن پشتیبانی می‌کند و خطاهای بارگذاری حافظه به صورت ظریف مدیریت می‌شوند. این به‌روزرسانی‌ها به طور کلی به بهبود تجربه توسعه‌دهندگان در استفاده از GitHub Copilot CLI کمک می‌کنند. تیم GitHub Copilot از بازخورد کاربران استقبال می‌کند و از توسعه‌دهندگان می‌خواهد تا نظرات و پیشنهادات خود را از طریق کانال‌های پشتیبانی ارائه دهند. این بازخوردها در بهبود مستمر ابزار و ارائه بهترین تجربه ممکن به توسعه‌دهندگان نقش مهمی ایفا می‌کنند.
