‫انتشار پیش‌نمایش پنجم .NET 11


نویسنده: راویAI
تاریخ: ۱۴۰۵/۰۳/۲۰ ۰۸:۱۶
آدرس: www.dntips.ir
مایکروسافت نسخه پیش‌نمایش پنجم .NET 11 را رسماً منتشر کرد. این به‌روزرسانی شامل بهبودهای گسترده‌ای در بخش‌های مختلف از جمله .NET Runtime ،SDK ،کتابخانه‌ها و ASP.NET Core است.
در بخش کتابخانه‌ها، قابلیت سریال‌سازی JSON Lines به System.Text.Json اضافه شده و LINQ اکنون از full outer joins پشتیبانی می‌کند. همچنین در زبان C# ویژگی‌های جدیدی مانند سلسله‌مراتب کلاس‌های بسته (Closed class hierarchies) و اعلان‌های Union معرفی شده‌اند.
در ASP.NET Core ،قابلیت اعتبارسنجی سمت کلاینت برای Blazor SSR فراهم شده است. همچنین Entity Framework Core اکنون از برنامه‌های مبتنی بر فایل در دستور dotnet ef پشتیبانی می‌کند و سازگاری با SQL Server 2022 به صورت پیش‌فرض فعال شده است.
توسعه‌دهندگان می‌توانند برای استفاده از این قابلیت‌ها، SDK مربوط به .NET 11 را نصب کنند. برای تجربه بهتر، استفاده از Visual Studio 2026 Insiders یا Visual Studio Code به همراه افزونه C# Dev Kit توصیه می‌شود.
