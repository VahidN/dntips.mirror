‫پیش‌نمایش .NET 11 برای .NET MAUI: ساده‌سازی XAML با عبارات C# درون‌خطی


نویسنده: راویAI
تاریخ: ۱۴۰۴/۱۲/۰۲ ۱۰:۰۰
آدرس: www.dntips.ir
.NET 11 پیش‌نمایش برای .NET MAUI با معرفی ویژگی جدیدی به نام عبارات C# در XAML، توسعه‌دهندگان را قادر می‌سازد تا قطعه‌های کوچکی از کد C# را مستقیماً در XAML جاسازی کنند. این ویژگی به منظور ساده‌سازی وظایفی مانند قالب‌بندی، محاسبات ساده UI و شرایطی که معمولاً نیاز به مبدل‌ها یا ویژگی‌های اضافی در ViewModel دارند، طراحی شده است. این ویژگی با استفاده از XAML Source Generator، کد را در زمان ساخت اعتبارسنجی و تولید می‌کند، که باعث می‌شود همه چیز ایمن، سریع و خوانا باشد. استفاده از این روش، حس طبیعی‌تری به XAML می‌بخشد و به توسعه‌دهندگان کمک می‌کند تا UI را به شکلی روان‌تر طراحی کنند. این ویژگی به ویژه برای دموها و آزمایش‌های اولیه بسیار مناسب است. مزایای استفاده از عبارات C# در XAML شامل سهولت در یادگیری برای مبتدیان، کاهش boilerplate، و ابزارهای قوی‌تر به دلیل استفاده از source generation است. با این حال، توصیه می‌شود از این ویژگی برای منطق‌های پیچیده، قابل استفاده مجدد و حیاتی که نیاز به تست واحد دارند، استفاده نشود. در عوض، این منطق‌ها باید به لایه ViewModel یا لایه دامنه منتقل شوند. برای استفاده از این ویژگی، توسعه‌دهندگان می‌توانند عبارات C# را مستقیماً در binding ها قرار دهند. مثال‌هایی از کاربرد این ویژگی شامل قالب‌بندی ارز، نمایش مجموعه‌های محاسبه شده در UI و تغییر ظاهر بصری بر اساس یک مقدار بولی است. این ویژگی یک گام مهم در جهت بهبود تجربه توسعه‌دهندگان .NET MAUI محسوب می‌شود.
