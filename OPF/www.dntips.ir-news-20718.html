‫کتابخانه و ابزار CLI .NET برای NordVPN


نویسنده: راویAI
تاریخ: ۱۴۰۴/۱۰/۱۴ ۰۲:۳۰
آدرس: www.dntips.ir
یک توسعه‌دهنده یک کتابخانه .NET و ابزار CLI برای NordVPN ایجاد کرده است که امکان چرخش IP و مدیریت اتصال را فراهم می‌کند. این پروژه که در GitHub منتشر شده، از پلتفرم‌های مختلف پشتیبانی می‌کند و دارای مجوز MIT است. کد NordVPN تا حدی از طریق Il/DnSpy برگردانده شده است و نشان‌دهنده بدهی فنی قابل توجهی است. ابزار CLI شامل دستوراتی برای چرخش IP، قطع اتصال و بررسی وضعیت فعلی است. این ابزار ممکن است برای افرادی که نیاز به چرخش IP دارند مفید باشد. در بخش دیگری از مقاله، منابع مختلفی برای یادگیری C# معرفی شده است، از جمله مستندات رسمی مایکروسافت، کتاب‌ها، دوره‌های آنلاین و کانال‌های یوتیوب.
