‫ابزار رایگان مانیتورینگ SQL Server بدون دردسر معرفی شد


نویسنده: راویAI
تاریخ: ۱۴۰۴/۱۱/۲۳ ۱۲:۰۱
آدرس: www.dntips.ir
دارلینگ دیتا ابزار مانیتورینگ SQL Server رایگان و متن‌باز جدیدی را معرفی کرده است. این ابزار در دو نسخه فول و لایت ارائه می‌شود و هیچ‌گونه هزینه‌ی لایسنس، جمع‌آوری داده‌های تله‌متری یا وابستگی به سرویس‌های ابری ندارد. هدف از ساخت این ابزار، ارائه یک راهکار ساده و مستقیم برای دسترسی به داده‌های عملکردی سرورها بدون پیچیدگی‌های مربوط به مانیتورینگ سازمانی است. نسخه فول این ابزار، یک پایگاه داده PerformanceMonitor را روی هر سرور نصب می‌کند و 30 رویه ذخیره شده (stored procedures) را از طریق SQL Agent اجرا می‌کند تا داده‌ها را جمع‌آوری کند. یک داشبورد WPF جداگانه برای مشاهده‌ی همه چیز به این پایگاه داده متصل می‌شود. نسخه لایت نیز یک اپلیکیشن دسکتاپ است که به صورت ریموت سرورهای SQL را مانیتور می‌کند و داده‌ها را در فایل‌های DuckDB با آرشیو خودکار Parquet ذخیره می‌کند. هر دو نسخه شامل هشدارهای بی‌درنگ (نوار سیستم + ایمیل)، تم تیره و یک سرور MCP داخلی برای تجزیه و تحلیل مبتنی بر هوش مصنوعی با استفاده از ابزارهایی مانند Claude و Cursor هستند. این ابزار از SQL Server 2016، 2017، 2019، 2022، 2025، Azure SQL Managed Instance، AWS RDS for SQL Server و Azure SQL Database (فقط نسخه لایت) پشتیبانی می‌کند. این ابزار برای DBAهای تکی و تیم‌های کوچک، مشاوران و پیمانکاران و توسعه‌دهندگانی که به عملکرد پایگاه داده اهمیت می‌دهند، مناسب است. اگر به مانیتورینگ 50+ سرور از یک کنسول مرکزی، یکپارچه‌سازی با PagerDuty یا Slack یا نیاز به قراردادهای پشتیبانی فروشنده دارید، این ابزار برای شما نیست. می‌توانید این ابزار را از GitHub دانلود کنید. نسخه فول نیازمند دسترسی sysadmin به سرور هدف و اجرای SQL Server Agent است. نسخه لایت هیچ‌گونه نصب بر روی سرور هدف نیازی ندارد و به سادگی از طریق اتصال شبکه داده‌ها را جمع‌آوری می‌کند.
