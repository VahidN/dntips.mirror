‫پشتیبانی داخلی NET 9. از OpenAPI برای مستندسازی Web API شما


نویسنده: وحید نصیری
تاریخ: ۱۴۰۳/۱۲/۲۲ ۰۸:۵۰
آدرس: www.dntips.ir
معرفی Microsoft.AspNetCore.OpenApi:
NET 9. بجای Swagger از Microsoft.AspNetCore.OpenApi برای تولید مشخصات OpenAPI استفاده می‌کند.

راه‌اندازی و استفاده اولیه:
  • برای پروژه‌های موجود، بسته Microsoft.AspNetCore.OpenApi را نصب کنید.
  • builder.Services.AddOpenApi و app.MapOpenApi را به Program.cs اضافه کنید.
  • به مشخصات OpenAPI تولید شده در مسیر openapi/v1.json/ دسترسی پیدا کنید.

استفاده از Swagger UI:
  • Swagger UI با مشخصات OpenAPI تولید شده سازگار است.
  • بسته Swashbuckle.AspNetCore.SwaggerUi را نصب کنید.
  • Swagger UI را در Program.cs طوری پیکربندی کنید که به openapi/v1.json/ اشاره کند.

if (app.Environment.IsDevelopment())
{
    app.MapOpenApi();
    app.UseSwaggerUI(options =>
    {
        options.SwaggerEndpoint("/openapi/v1.json", "v1");
    });
}
