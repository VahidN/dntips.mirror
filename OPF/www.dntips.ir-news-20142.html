‫دوره آموزشی «توسعه تست واحد با استفاده از ابزارهای GitHub Copilot»


نویسنده: وحید نصیری
تاریخ: ۱۴۰۳/۱۲/۱۸ ۱۴:۱۵
آدرس: www.dntips.ir
این ماژول آموزشی به شما نحوه استفاده از ابزارهای GitHub Copilot و GitHub Copilot Chat را در محیط Visual Studio Code برای ایجاد تست های واحد (Unit Tests) مؤثر در پروژه های #C آموزش می دهد. تست واحد، به شما کمک می کند تا تکه های کوچک کد را به صورت جداگانه آزمایش کنید و از صحت عملکرد آن ها مطمئن شوید.

اهداف یادگیری:
  • ایجاد تست های واحد با استفاده از افزونه های GitHub Copilot و GitHub Copilot Chat.
  • توسعه تست های واحد برای شرایط خاص و موارد حاشیه ای (Edge Cases).
  • استفاده از Visual Studio Code، .NET SDK و افزونه C# Dev Kit برای ایجاد و تأیید پروژه های تست.

پیش نیازها:
  • اشتراک فعال GitHub Copilot (برای حساب شخصی یا سازمانی).
  • تجربه کار با افزونه های GitHub Copilot و GitHub Copilot Chat.
  • حداقل یک سال تجربه توسعه نرم افزار.
  • تجربه توسعه برنامه های C# در Visual Studio Code.
  • آگاهی از شرایط دوره آزمایشی رایگان GitHub Copilot.

محتوای ماژول:
  • مقدمه ای بر ابزارها و محیط تست واحد.
  • تمرین های عملی:
    • ایجاد تست های واحد با استفاده از GitHub Copilot Chat.
    • ایجاد تست های واحد برای شرایط خاص.
    • انجام چالش "ایجاد تست های واحد".
  • بررسی راه حل تمرین ها.
  • آزمون دانش (Knowledge Check).
  • خلاصه.

نکات کلیدی:
  • این ماژول در سطح متوسط برای توسعه دهندگان طراحی شده است.
  • این ماژول بخشی از مسیرهای یادگیری "تسریع توسعه برنامه با استفاده از GitHub Copilot" و "مبانی GitHub Copilot - درک برنامه نویس همکار هوش مصنوعی" است.
  • برای انجام تمرین ها، به Visual Studio Code و افزونه C# Dev Kit نیاز دارید.
