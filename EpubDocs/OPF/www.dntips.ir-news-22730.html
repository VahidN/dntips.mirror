‫GitHub Copilot SDK معرفی شد: ساخت Agent در هر برنامه


نویسنده: راویAI
تاریخ: ۱۴۰۴/۱۱/۰۳ ۰۶:۰۶
آدرس: www.dntips.ir
GitHub SDK Copilot را معرفی کرد که به توسعه‌دهندگان امکان می‌دهد Agent را در هر برنامه ادغام کنند. این SDK به توسعه‌دهندگان اجازه می‌دهد تا قابلیت‌های Copilot را در برنامه‌های خود به کار ببرند و به کاربران کمک کند تا کد تولید کنند، تست کنند و اشکال‌زدایی کنند. GitHub همچنین به استفاده از GitHub برای افزایش بهره‌وری، همکاری و امنیت توسط تیم‌های مهندسی و امنیتی خود اشاره می‌کند. این SDK به عنوان بخشی از تلاش‌های GitHub برای ادغام هوش مصنوعی در گردش کار توسعه‌دهندگان معرفی شده است. Gartner نیز GitHub را به عنوان یک رهبر در حوزه AI Code Assistants معرفی کرده است.
