‫اعلام دسترسی عمومی (GA) به SQL Server Management Studio (SSMS) 21


نویسنده: وحید نصیری
تاریخ: ۱۴۰۴/۰۲/۳۰ ۲۲:۰۳
آدرس: www.dntips.ir
مایکروسافت به تازگی انتشار عمومی (General Availability - GA) نسخه ۲۱ نرم‌افزار SQL Server Management Studio (SSMS) را اعلام کرده است. این نسخه که از پایه بازسازی شده، بهبودهای عملکردی قابل توجه و قابلیت‌های جدیدی را ارائه می‌دهد و تجربه کاربری توسعه‌دهندگان و مدیران پایگاه داده را متحول خواهد کرد.

ویژگی‌های برجسته SSMS 21
SSMS 21 بر پایه محیط توسعه یکپارچه (Integrated Development Environment - IDE) ویژوال استودیو (Visual Studio) ساخته شده و از معماری ۶۴ بیتی (64-bit) پشتیبانی می‌کند. این نسخه، به عنوان ویرایش پیشنهادی برای پیش‌نمایش SQL Server 2025 (SQL Server 2025 Preview) معرفی شده و همچنین شامل قابلیت کوپایلوت (Copilot) در SSMS (در مرحله پیش‌نمایش) و پشتیبانی اولیه از تم تیره (Dark Theme) است.
از جمله بهبودها و قابلیت‌های کلیدی می‌توان به موارد زیر اشاره کرد:
  • نصب و به‌روزرسانی: فرآیند نصب SSMS 21 به طور چشمگیری تغییر کرده و اکنون از طریق ابزار Visual Studio Installer مدیریت می‌شود. این تغییر امکان نصب با حجم کمتر و سفارشی‌سازی نصب را از طریق "بارهای کاری" (Workloads) و "مولفه‌های مجزا" (Individual Components) فراهم می‌کند. به‌روزرسانی‌های خودکار نیز از طریق همین ابزار انجام خواهد شد.
  • یکپارچه‌سازی گیت (Git Integration): این نسخه پشتیبانی از گیت (Git)، سیستم کنترل نسخه توزیع‌شده، را از طریق بار کاری Code tools ارائه می‌دهد که به توسعه‌دهندگان امکان مدیریت بهتر کدها و همکاری تیمی را می‌دهد.
  • گفتگوی اتصال مدرن (Modern Connection Dialog): یک تجربه جدید و مدرن برای اتصال به پایگاه داده (در مرحله پیش‌نمایش) ارائه شده که رابط کاربری بهبود یافته‌ای دارد.
  • گزینه‌های شخصی‌سازی (Customization Options): قابلیت‌های جدیدی برای شخصی‌سازی محیط SSMS اضافه شده است، از جمله تب‌های عمودی (vertical tabs)، رنگ‌آمیزی تب‌ها بر اساس پروژه یا نوع فایل (tab coloring by project or file type)، تنظیم حداقل و حداکثر عرض تب (minimum and maximum tab width)، و تغییر رنگ فونت و پس‌زمینه سلول‌های حاوی مقادیر NULL در گرید نتایج (font and background color for results grid cells containing NULL values).
  • ارزیابی Always Encrypted (Always Encrypted Assessment): اکنون این قابلیت از طریق ویزارد Always Encrypted (Always Encrypted Wizard) در دسترس است. ویژگی Always Encrypted در SQL Server به شما اجازه می‌دهد تا داده‌های حساس را در پایگاه داده رمزگذاری کنید، به گونه‌ای که کلیدهای رمزگذاری هرگز در دسترس موتور پایگاه داده قرار نگیرند. این امر امنیت داده‌ها را در برابر دسترسی‌های غیرمجاز افزایش می‌دهد.
  • کمک در مهاجرت (Migration Assistance): امکانات جدیدی برای کمک به مهاجرت از طریق بار کاری "هیبرید و مهاجرت" (Hybrid and Migration Workload) در نظر گرفته شده است. این بار کاری ابزارهایی را برای تسهیل فرآیند انتقال پایگاه داده‌ها به محیط‌های ابری یا ترکیبی (Hybrid Cloud) ارائه می‌دهد.

ملاحظات مهم
  • نصب همزمان (Side-by-Side Installation): SSMS 21 GA می‌تواند در کنار نسخه‌های قبلی SSMS (مانند ۲۰.x) نصب شود و نسخه پیش‌نمایش SSMS 21 را جایگزین نخواهد کرد. مایکروسافت توصیه می‌کند که به‌روزرسانی‌های آینده SSMS 21 از طریق کانال "Release" نصب شوند.
  • SQL Server Integration Services (SSIS) و برنامه‌های نگهداری (Maintenance Plans): در این نسخه، پشتیبانی از SQL Server Integration Services (SSIS) که یک پلتفرم برای ساخت راه‌حل‌های یکپارچه‌سازی داده با عملکرد بالا، شامل بسته‌های استخراج، تبدیل و بارگذاری (ETL) است، و همچنین Maintenance Plans که شامل وظایف خودکار برای بهینه‌سازی، پشتیبان‌گیری منظم و بررسی سازگاری پایگاه داده‌ها هستند، به طور موقت متوقف شده است. این تصمیم به دلیل نیاز به زمان بیشتر برای تضمین کیفیت و پایداری این قابلیت‌ها اتخاذ شده است. توسعه‌دهندگان مایکروسافت اکیداً توصیه می‌کنند در صورت وابستگی به SSIS، از SSMS 20 استفاده کنند که قابلیت نصب همزمان با SSMS 21 GA را دارد. تیم مایکروسافت فعالانه در تلاش است تا این ویژگی‌ها را در نسخه‌های آتی بازگرداند.
  • سیاست پشتیبانی (Support Policy): با انتشار SSMS 21، نسخه‌های قبلی SSMS دیگر پشتیبانی نخواهند شد. مایکروسافت قویاً توصیه می‌کند برای دسترسی به جدیدترین ویژگی‌ها، به‌روزرسانی‌های امنیتی و رفع اشکالات حیاتی، به SSMS 21 ارتقا یابید.
