‫مایکروسافت SQL Server 2025 منتشر شد!


نویسنده: وحید نصیری
تاریخ: ۱۴۰۴/۰۸/۲۸ ۰۹:۵۶
آدرس: www.dntips.ir
در تاریخ ۱۹ نوامبر ۲۰۲۵، مایکروسافت به‌طور رسمی SQL Server 2025 را منتشر کرد. این نسخه با تغییرات مهمی همراه است که مهم‌ترین آن‌ها برای کاربران نسخه Standard است:

تغییرات کلیدی در نسخه‌ها:
  • Standard Edition: حالا تا ۳۲ هسته CPU و ۲۵۶ گیگابایت RAM را پشتیبانی می‌کند (قبلاً ۲۴ هسته و ۱۲۸ گیگابایت بود). همچنین برای اولین بار Resource Governor در نسخه Standard در دسترس قرار گرفته است.
  • Express Edition: ظرفیت هر دیتابیس به ۵۰ گیگابایت افزایش یافت (هنوز محدود به ۱ هسته).
  • Web Edition: کاملاً حذف شده است.
  • Standard Developer Edition جدید: نسخه‌ای رایگان برای توسعه‌دهندگان که دقیقاً محدودیت‌های نسخه Standard تولیدی را شبیه‌سازی می‌کند.

ویژگی‌های مهم جدید:
  • پشتیبانی از JSON data type جدید و تابع جستجوی JSON_CONTAINS با قابلیت ایندکس‌گذاری
  • پشتیبانی از Vector data type و جستجوی برداری (برای پروژه‌های هوش مصنوعی)
  • فراخوانی مستقیم APIهای REST خارجی (مثل ChatGPT) با sp_invoke_external_rest_endpoint
  • پشتیبانی داخلی از Regular Expression (بدون نیاز به ابزارهای ثالث)
  • ویژگی Optimized Locking + Accelerated Database Recovery برای جلوگیری از Lock Escalation حتی در تراکنش‌های بسیار بزرگ
  • بهبودهای متعدد در Columnstore Indexes، Intelligent Query Processing و Availability Groups
  • الگوریتم جدید فشرده‌سازی بک‌آپ (سریع‌تر اما حجم مشابه یا کمی بیشتر)

تغییرات ناسازگار با نگارش‌های قبلی (Breaking Changes):
  • استفاده از پروتکل جدید TDS 8.0 با TLS 1.3 که ممکن است برخی Linked Serverها و Replicationها را مختل کند (راه‌حل‌های رسمی ارائه شده است).

قیمت‌گذاری:
برخلاف شایعات، قیمت‌ها تغییری نکرده است:
  • Enterprise: همچنان ۷٬۵۶۲ دلار به ازای هر هسته
  • Standard: همچنان ۱٬۹۷۳ دلار به ازای هر هسته

توصیه‌ها:
  • اگر اپلیکیشن خودتان را می‌سازید: هرچه زودتر SQL Server 2025 را در محیط توسعه نصب کنید، اما ابتدا compatibility level را روی نسخه قبلی نگه دارید.
  • اگر از نرم‌افزارهای ثالث استفاده می‌کنید: تا زمانی که vendor رسماً پشتیبانی از 2025 را اعلام نکند، صبر کنید.

در کل، SQL Server 2025 یکی از جذاب‌ترین آپدیت‌های سال‌های اخیر برای کاربران نسخه Standard به شمار می‌رود و مایکروسافت با افزایش چشمگیر سقف سخت‌افزاری این نسخه، فشار رقابتی دیتابیس‌های دیگر را به خوبی پاسخ داده است.
