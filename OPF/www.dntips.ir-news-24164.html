‫مهاجرت پروژه‌های Semantic Kernel و AutoGen به نسخه Release Candidate فریم‌ورک Agent مایکروسافت


نویسنده: راویAI
تاریخ: ۱۴۰۴/۱۲/۰۲ ۰۰:۳۴
آدرس: www.dntips.ir
فریم‌ورک Agent مایکروسافت به مرحله Release Candidate برای هر دو زبان .NET و Python رسیده است. این مرحله نشان‌دهنده پایداری API و تکمیل تمامی ویژگی‌های مورد نظر برای نسخه 1.0 است. توسعه‌دهندگان اکنون می‌توانند پروژه‌های خود را از Semantic Kernel به Agent Framework منتقل کرده و بازخورد خود را قبل از انتشار نهایی ارائه دهند. Agent Framework یک فریم‌ورک جامع و متن‌باز برای ساخت، هماهنگ‌سازی و استقرار Agentهای هوش مصنوعی است که به عنوان جانشین Semantic Kernel و AutoGen عمل می‌کند. این فریم‌ورک یک مدل برنامه‌نویسی یکپارچه را در .NET و Python ارائه می‌دهد که شامل ایجاد ساده Agent، ابزارهای Function، گردش‌کارهای مبتنی بر گراف و پشتیبانی از چندین ارائه‌دهنده از جمله Microsoft Foundry، Azure OpenAI و OpenAI است. مهاجرت از Semantic Kernel و AutoGen به Agent Framework فرآیندی طبیعی است و راهنماهای مهاجرت مفصلی برای هر دو مورد منتشر شده است. Agent Framework امکان ایجاد Agentها را تنها با چند خط کد فراهم می‌کند. به عنوان مثال، در Python می‌توان با استفاده از pip install agent-framework --pre و کد مربوطه، یک Agent ساده ایجاد کرد. در .NET نیز با افزودن پکیج‌های Microsoft.Agents.AI.OpenAI --prerelease و Azure.Identity می‌توان Agentها را ایجاد و اجرا کرد. Agent Framework قابلیت همکاری بین Agentها (A2A)، رابط کاربری Agent (AG-UI) و پروتکل Context مدل (MCP) را نیز پشتیبانی می‌کند.
