ШЁШҜЫҢЩҮЫҢ Ш§ШіШӘ Щ…ШӘШҜ getdateШҢ ЫҢЪ© Щ…ШӘШҜ ШЁЩҲЩ…ЫҢ ШіЩ…ШӘ SQL Server Ш§ШіШӘ ЩҲ Ш§ЫҢЩҶ ШұЩҲШҙ Ш®Ш§Шө ШӘШ№ЫҢЫҢЩҶ Щ…ЩӮШҜШ§Шұ ЩҫЫҢШҙ ЩҒШұШ¶ ЩҒЫҢЩ„ШҜЩҮШ§ШҢ ЩҒЩӮШ· ШЁШ§ SQL Server Ъ©Ш§Шұ Щ…ЫҢвҖҢЪ©ЩҶШҜ. ЩҮЩ…ЪҶЩҶЫҢЩҶ Ш§ЫҢЩҶ getdateШҢ ШЁЩҮ Щ…Ш№ЩҶШ§ЫҢ ШҜШұЫҢШ§ЩҒШӘ ШӘШ§ШұЫҢШ® ЩҲ ШіШ§Ш№ШӘ ШіШұЩҲШұЫҢ Ш§ШіШӘ Ъ©ЩҮ SQL Server ШЁШұ ШұЩҲЫҢ ШўЩҶ ЩҶШөШЁ ШҙШҜЩҮвҖҢШ§ШіШӘ ЩҲ ЩҶЩҮ ШіШұЩҲШұЫҢ Ъ©ЩҮ ШЁШұЩҶШ§Щ…ЩҮвҖҢЫҢ ЩҲШЁ ШҙЩ…Ш§ ШҜШұ ШўЩҶ ЩӮШұШ§Шұ ШҜШ§ШұШҜ ЩҲ ШЁШұЩҶШ§Щ…ЩҮ Ъ©ЩҲЪҶЪ©ШӘШұЫҢЩҶ ШҜШ®Ш§Щ„ШӘЫҢ ШұШ§ ШҜШұ Щ…ЩӮШҜШ§Шұ ШҜЩҮЫҢ Ш§ЫҢЩҶ Щ…ЩӮШ§ШҜЫҢШұ ЩҶШ®ЩҲШ§ЩҮШҜ ШҜШ§ШҙШӘ.
ШҜШұ ЩӮШіЩ…ШӘвҖҢЩҮШ§ЫҢ ШЁШ№ШҜЫҢ Ъ©ЩҮ Щ…ШЁШ§ШӯШ« Tracking ШұШ§ ШЁШұШұШіЫҢ Ш®ЩҲШ§ЩҮЫҢЩ… Ъ©ШұШҜШҢ ШұЩҲШҙ ШҜЫҢЪҜШұЫҢ ШұШ§ ШЁШұШ§ЫҢ Ш·ШұШ§ШӯЫҢ Ъ©Щ„Ш§ШівҖҢЩҮШ§ЫҢ ЩҫШ§ЫҢЩҮ ЩҲ Щ…ЩӮШҜШ§Шұ ШҜЩҮЫҢ Ш®ЩҲШ§Шө ЩҲЫҢЪҳЩҮвҖҢЫҢ ШўЩҶвҖҢЩҮШ§ Щ…Ш·ШұШӯ Щ…ЫҢвҖҢЪ©ЩҶЫҢЩ… Ъ©ЩҮ Щ…ШіШӘЩӮЩ„ Ш§ШіШӘ Ш§ШІ ЩҶЩҲШ№ ШЁШ§ЩҶЪ© Ш§Ш·Щ„Ш§Ш№Ш§ШӘЫҢ Щ…ЩҲШұШҜ Ш§ШіШӘЩҒШ§ШҜЩҮ.


ШЁШұШұШіЫҢ ШӘЩҶШёЫҢЩ…Ш§ШӘ ШұШ§ШЁШ·ЩҮвҖҢЫҢ  Table per Hierarchy ЫҢШ§ TPH

ШұШ§ШЁШ·ЩҮвҖҢЫҢ TPH ЫҢШ§ ШӘШҙЪ©ЫҢЩ„ ЫҢЪ© Ш¬ШҜЩҲЩ„ ШЁШ§ЩҶЪ© Ш§Ш·Щ„Ш§Ш№Ш§ШӘЫҢШҢ ШЁЩҮ Ш§ШІШ§ЫҢ ШӘЩ…Ш§Щ… Ъ©Щ„Ш§ШівҖҢЩҮШ§ЫҢ ШҜШ®ЫҢЩ„ ШҜШұ ШіЩ„ШіЩҮ Щ…ШұШ§ШӘШЁ Ш§ШұШ« ШЁШұЫҢ ШӘШ№ШұЫҢЩҒ ШҙШҜЩҮШҢ ШЁШіЫҢШ§Шұ ШҙШЁЫҢЩҮ Ш§ШіШӘ ШЁЩҮ ШӯШ§Щ„ШӘ BaseEntity ЩҒЩҲЩӮ Ъ©ЩҮ ШҜШұ ШўЩҶ ЩҶЫҢШІ Ш§ШұШ« ШЁШұЫҢ ШӘШ№ШұЫҢЩҒ ШҙШҜЩҮШҢ ШҜШұ ЩҶЩҮШ§ЫҢШӘ Щ…ЩҶШ¬Шұ ШЁЩҮ ШӘШҙЪ©ЫҢЩ„ ЫҢЪ© Ш¬ШҜЩҲЩ„ШҢ ШҜШұ ШіЩ…ШӘ ШЁШ§ЩҶЪ© Ш§Ш·Щ„Ш§Ш№Ш§ШӘЫҢ Щ…ЫҢвҖҢЪҜШұШҜШҜ. ШЁШ§ Ш§ЫҢЩҶ ШӘЩҒШ§ЩҲШӘ Ъ©ЩҮ ШҜШұ ШӯШ§Щ„ШӘ TPHШҢ ЩҒЫҢЩ„ШҜ Ш¬ШҜЫҢШҜЫҢ ЩҶЫҢШІ ШЁЩҮ ЩҶШ§Щ… DiscriminatorШҢ ШЁЩҮ ШӘШ№ШұЫҢЩҒ ЩҶЩҮШ§ЫҢЫҢ Ш¬ШҜЩҲЩ„ Ш§ЫҢШ¬Ш§ШҜ ШҙШҜЩҮШҢ Ш§Ш¶Ш§ЩҒЩҮ Щ…ЫҢвҖҢШҙЩҲШҜ. Ш§ШІ ЩҒЫҢЩ„ШҜ Discriminator Ш¬ЩҮШӘ ШҜШұШ¬ ЩҶШ§Щ… Ъ©Щ„Ш§ШівҖҢЩҮШ§ЫҢ Щ…ШӘЩҶШ§ШёШұ ШЁШ§ ЩҮШұ ШұЪ©ЩҲШұШҜШҢ Ш§ШіШӘЩҒШ§ШҜЩҮ ШҙШҜЩҮ Ш§ШіШӘ. ШЁЩҮ Ш§ЫҢЩҶ ШӘШұШӘЫҢШЁ EF ШҜШұ ШӯЫҢЩҶ Ъ©Ш§Шұ ШЁШ§ Ш§ШҙЫҢШ§ШЎШҢ ШҜЩӮЫҢЩӮШ§ Щ…ЫҢвҖҢШҜШ§ЩҶШҜ Ъ©ЩҮ ЪҶЪҜЩҲЩҶЩҮ ШЁШ§ЫҢШҜ Ш®ЩҲШ§Шө Щ…ШӘЩҶШ§ШёШұ ШЁШ§ Ъ©Щ„Ш§ШівҖҢЩҮШ§ЫҢ Щ…Ш®ШӘЩ„ЩҒ ШұШ§ Щ…ЩӮШҜШ§Шұ ШҜЩҮЫҢ Ъ©ЩҶШҜ ЩҲ ЩҶЩҲШ№ ШұШҜЫҢЩҒ ШҜШұШ¬ ШҙШҜЩҮвҖҢЫҢ ШҜШұ ШЁШ§ЩҶЪ© Ш§Ш·Щ„Ш§Ш№Ш§ШӘЫҢ ЪҶЫҢШіШӘШҹ
ШЁШ§ЫҢШҜ ШҜЩӮШӘ ШҜШ§ШҙШӘ Ъ©ЩҮ ШӘЩҶШёЫҢЩ…Ш§ШӘ TPHШҢ ШҙЫҢЩҲЩҮ ШЁШұШ®ЩҲШұШҜ ЩҫЫҢШҙ ЩҒШұШ¶ EF Core ШЁШ§ Ш§ШұШ« ШЁШұЫҢ Ъ©Щ„Ш§ШівҖҢЩҮШ§ Ш§ШіШӘ ЩҲ ЩҶЫҢШ§ШІ ШЁЩҮ ЩҮЫҢЪҶЪҜЩҲЩҶЩҮ ШӘЩҶШёЫҢЩ… Ш§Ш¶Ш§ЩҒЩҮвҖҢШӘШұЫҢ ШұШ§ ЩҶШҜШ§ШұШҜ. Ш§Щ…Ш§ Ш§ЪҜШұ Ш№Щ„Ш§ЩӮЩ…ЩҶШҜ ШЁЩҲШҜЫҢШҜ ШӘШ§ ЩҶШ§Щ… ЩҒЫҢЩ„ШҜ Ш®ЩҲШҜЪ©Ш§Шұ Discriminator ЩҲ Щ…ЩӮШ§ШҜЫҢШұЫҢ ШұШ§ Ъ©ЩҮ ШҜШұ ШўЩҶ ШҜШұШ¬ Щ…ЫҢвҖҢШҙЩҲЩҶШҜШҢ ШіЩҒШ§ШұШҙЫҢ ШіШ§ШІЫҢ Ъ©ЩҶЫҢШҜШҢ ШұЩҲШҙ Ъ©Ш§Шұ ШөШұЩҒШ§ ШӘЩҲШіШ· Fluent API Щ…ЫҢШіШұ Ш§ШіШӘ ЩҲ ШЁЩҮ ШөЩҲШұШӘ ШІЫҢШұ Щ…ЫҢвҖҢШЁШ§ШҙШҜ:
public class Blog
{
   public int BlogId { get; set; }
   public string Url { get; set; }
}

public class RssBlog : Blog
{
   public string RssUrl { get; set; }
}

class MyContext : DbContext
{
   public DbSet<Blog> Blogs { get; set; }
   protected override void OnModelCreating(ModelBuilder modelBuilder)
   {
      modelBuilder.Entity<Blog>()
       .HasDiscriminator<string>("blog_type")
       .HasValue<Blog>("blog_base")
       .HasValue<RssBlog>("blog_rss");
   }
}
ШҜШұ Ш§ЫҢЩҶШ¬Ш§ ЩҶШ§Щ… ЩҒЫҢЩ„ШҜ DiscriminatorШҢ ШЁЩҮ blog_typeШҢ Щ…ЩӮШҜШ§Шұ ЩҶЩҲШ№ Щ…ШӘЩҶШ§ШёШұ ШЁШ§ Ъ©Щ„Ш§Ші BlogШҢ ШЁЩҮ blog_base ЩҲ Щ…ЩӮШҜШ§Шұ ЩҶЩҲШ№ Щ…ШӘЩҶШ§ШёШұ ШЁШ§ Ъ©Щ„Ш§Ші RssBlogШҢ ШЁЩҮ blog_rss ШӘЩҶШёЫҢЩ… ШҙШҜЩҮвҖҢШ§ШіШӘ.
Ш§ЪҜШұ Ш§ЫҢЩҶ ШӘЩҶШёЫҢЩ…Ш§ШӘ ШіЩҒШ§ШұШҙЫҢ ШөЩҲШұШӘ ЩҶЪҜЫҢШұЩҶШҜШҢ Ш§ШІ ЩҶШ§Щ…вҖҢЩҮШ§ЫҢ ЩҫЫҢШҙ ЩҒШұШ¶ ЩҶЩҲШ№вҖҢЩҮШ§ ШЁШұШ§ЫҢ Щ…ЩӮШҜШ§Шұ ШҜЩҮЫҢ ШіШӘЩҲЩҶ DiscriminatorШҢ Щ…Ш§ЩҶЩҶШҜ ШӘШөЩҲЫҢШұ Ш°ЫҢЩ„ Ш§ШіШӘЩҒШ§ШҜЩҮ Ш®ЩҲШ§ЩҮШҜ ШҙШҜ:


ШЁШұШ§ЫҢ Ъ©ЩҲШҰШұЫҢ ЩҶЩҲШҙШӘЩҶ ШҜШұ Ш§ЫҢЩҶ ШӯШ§Щ„ШӘ Щ…ЫҢвҖҢШӘЩҲШ§ЩҶ Ш§ШІ Щ…ШӘШҜ Ш§Щ„ШӯШ§ЩӮЫҢ OfType Ш¬ЩҮШӘ ЩҒЫҢЩ„ШӘШұ Ъ©ШұШҜЩҶ Ш§Ш·Щ„Ш§Ш№Ш§ШӘ ШЁШұ Ш§ШіШ§Ші Ъ©Щ„Ш§ШіЫҢ Ш®Ш§ШөШҢ Ъ©Щ…Ъ© ЪҜШұЩҒШӘ:
