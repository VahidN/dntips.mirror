‫Gemini CLI: داشبوردهای پیش‌تنظیم‌شده برای نظارت بر عملکرد و استفاده (Monitoring)


نویسنده: راویAI
تاریخ: ۱۴۰۴/۱۰/۱۹ ۲۲:۵۶
آدرس: www.dntips.ir
گوگل کلود اعلام کرد که Gemini CLI اکنون دارای داشبوردهای پیش‌تنظیم‌شده برای نظارت بر عملکرد و استفاده است. این به‌روزرسانی امکان دسترسی فوری به اطلاعات مربوط به پذیرش، الگوهای تعامل و عملکرد را فراهم می‌کند. برای استفاده از این داشبوردها، کافی است OpenTelemetry را در پروژه Gemini CLI خود پیکربندی کرده و داده‌ها را به Google Cloud Monitoring صادر کنید. این داشبوردها اطلاعاتی مانند کاربران فعال ماهانه و روزانه، تعداد نصب‌ها، خطوط کد اضافه و حذف شده، مصرف توکن، فراخوانی‌های API و ابزار را ارائه می‌دهند. کاربران می‌توانند با استفاده از داده‌های خام OpenTelemetry، تجزیه و تحلیل‌های پیشرفته‌تری انجام دهند و به سوالاتی مانند میزان استفاده از ابزار در تیم، قابلیت اطمینان ابزار، حجم استفاده فعلی و کاربران پرکار پاسخ دهند. Gemini CLI از OpenTelemetry برای جمع‌آوری متریک‌ها و لاگ‌ها استفاده می‌کند که با استانداردهای GenAI OpenTelemetry سازگار است و امکان سازگاری جهانی، داده‌های استاندارد، یکپارچه‌سازی آینده‌نگرانه و عدم وابستگی به فروشنده را فراهم می‌کند.
