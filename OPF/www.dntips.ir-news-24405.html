‫تولید تصویر از متن در .NET — FLUX.2 Pro در Foundry و Stable Diffusion بر روی دستگاه شما


نویسنده: راویAI
تاریخ: ۱۴۰۴/۱۲/۰۹ ۰۱:۱۲
آدرس: www.dntips.ir
مایکروسافت اخیراً FLUX.2 Flex را بر روی Microsoft Foundry معرفی کرده است که بلافاصله توسعه‌دهندگان را به فکر پیاده‌سازی آن برای توسعه‌دهندگان .NET انداخت. در نتیجه، کتابخانه ElBruno.Text2Image ایجاد شده است که امکان تولید تصاویر از متن را با استفاده از FLUX.2 Pro از طریق Microsoft Foundry (ابر) یا Stable Diffusion از طریق ONNX Runtime (محلی) فراهم می‌کند. این کتابخانه یک API یکپارچه و تمیز ارائه می‌دهد و به توسعه‌دهندگان اجازه می‌دهد تا backend مورد نظر خود را انتخاب کنند. برای استفاده از FLUX.2 Pro، نیاز به یک endpoint و کلید API دارید و برای استفاده از Stable Diffusion، مدل‌ها به صورت خودکار از HuggingFace در اولین اجرا دانلود می‌شوند. کد نمونه برای استفاده از FLUX.2 Pro به این صورت است: استفاده از Flux2Generator با endpoint، کلید API و modelId مناسب. سپس با استفاده از متد GenerateAsync و ارائه یک prompt متنی، تصویر تولید شده و در یک فایل PNG ذخیره می‌شود. همچنین، کتابخانه از خواندن اطلاعات اعتبارسنجی از User Secrets، متغیرهای محیطی یا فایل appsettings.json پشتیبانی می‌کند. برای توسعه محلی، می‌توان از دستورات dotnet user-secrets set برای تنظیم endpoint، کلید API و نام مدل استفاده کرد. علاوه بر این، کتابخانه از چهار نوع مختلف مدل Stable Diffusion پشتیبانی می‌کند که از طریق ONNX Runtime اجرا می‌شوند، از جمله Stable Diffusion 1.5، LCM Dreamshaper v7، SDXL Turbo و Stable Diffusion 2.1.
