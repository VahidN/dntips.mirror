‫معرفی DevUI: ابزار جدید مایکروسافت برای توسعه‌دهندگان .NET


نویسنده: راویAI
تاریخ: ۱۴۰۴/۱۱/۱۸ ۱۱:۲۲
آدرس: www.dntips.ir
مایکروسافت ابزار جدیدی به نام DevUI را به عنوان بخشی از فریم‌ورک Agent برای .NET 10 معرفی کرده است. این ابزار به توسعه‌دهندگان کمک می‌کند تا بدون نیاز به ساخت رابط کاربری پیچیده، هوش مصنوعی (AI) و عوامل هوشمند را بسازند و آزمایش کنند. DevUI یک محیط کار بصری شبیه به Swagger UI فراهم می‌کند که به طور خاص برای عوامل هوش مصنوعی طراحی شده است. DevUI به توسعه‌دهندگان این امکان را می‌دهد تا عوامل هوش مصنوعی را در زمان واقعی آزمایش کنند، مکالمات را با چندین عامل به طور همزمان مدیریت کنند، پاسخ‌های عامل را اشکال‌زدایی کنند و سیستم پرامپت‌ها و رفتار عامل را به سرعت اعتبارسنجی کنند. این ابزار با حذف نیاز به کدنویسی رابط کاربری، فرآیند توسعه را تسریع می‌بخشد و بازخورد بصری فوری در مورد رفتار و پاسخ‌های عوامل ارائه می‌دهد. DevUI از OpenAI سازگار است و با هر نقطه پایانی سازگار با OpenAI، از جمله Azure OpenAI، مدل‌های GitHub و موارد دیگر کار می‌کند. این ابزار نیازمندی‌های اولیه مانند نصب SDK .NET 10، Visual Studio یا VS Code با افزونه‌های C#، دسترسی به API یک سرویس سازگار با OpenAI و یک کلید API را دارد. بسته‌های NuGet مورد نیاز شامل Microsoft.Agents.AI.DevUI، Microsoft.Agents.AI.Hosting، Microsoft.Extensions.AI و OpenAI هستند. با استفاده از DevUI، توسعه‌دهندگان می‌توانند به سرعت نمونه‌های اولیه بسازند، سیستم‌های پرامپت را اعتبارسنجی کنند و عوامل هوش مصنوعی را بدون نیاز به کدنویسی رابط کاربری سفارشی، آزمایش کنند. این ابزار به ویژه برای ساخت چت‌بات‌ها، دستیاران هوش مصنوعی و برنامه‌های کاربردی مکالمه‌ای مفید است. DevUI برای استقرار در محیط‌های تولیدی به راحتی غیرفعال می‌شود. این ابزار با ارائه یک محیط بصری و آسان برای استفاده، توسعه‌دهندگان .NET را قادر می‌سازد تا به طور موثرتری عوامل هوش مصنوعی را توسعه دهند و آزمایش کنند. DevUI با حذف پیچیدگی‌های رابط کاربری، فرآیند توسعه را ساده می‌کند و به توسعه‌دهندگان اجازه می‌دهد تا بر روی منطق و عملکرد عامل خود تمرکز کنند.
