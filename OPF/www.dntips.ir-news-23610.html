‫معرفی تولیدکننده‌های کد ویژگی Bindable در .NET MAUI


نویسنده: راویAI
تاریخ: ۱۴۰۴/۱۱/۱۶ ۱۰:۳۹
آدرس: www.dntips.ir
کتابخانه CommunityToolkit.Maui نسخه ۱۴.۰.۰ دو تولیدکننده کد جدید برای ویژگی‌های BindableProperty و AttachedBindableProperty معرفی کرده است. این تولیدکننده‌های کد جدید، ایجاد BindableProperty را در برنامه‌های .NET MAUI آسان‌تر می‌کنند و اکنون تمام Bindable Properties در CommunityToolkit.Maui به طور خودکار با استفاده از این ابزارها تولید می‌شوند. این ویژگی به صورت تجربی (Experimental) منتشر شده است و برای آزمایش و دریافت بازخورد از جامعه توسعه‌دهندگان در نظر گرفته شده است. برای استفاده از این قابلیت، باید شناسه MCTEXP001 را در فایل csproj برنامه .NET MAUI خود سرکوب کنید. این امکان به تیم توسعه اجازه می‌دهد تا در صورت نیاز، API را بر اساس بازخورد جامعه تغییر دهند. برای بهره‌مندی از تولیدکننده کد Bindable Property، ابتدا اطمینان حاصل کنید که از یک کلاس partial استفاده می‌کنید. سپس، کلمه کلیدی partial را به کلاس و همچنین به ویژگی مورد نظر برای تولید Bindable Property اضافه کنید و ویژگی [CommunityToolkit.Maui.BindableProperty] را به آن متصل کنید. این روش، تولید کد تکراری مورد نیاز برای Bindable Properties و Attached Properties را خودکار می‌کند. تنظیم مقدار پیش‌فرض برای [BindableProperty] با تعیین مقدار اولیه در ویژگی partial امکان‌پذیر است. همچنین، می‌توان حالت BindingMode پیش‌فرض را با استفاده از ویژگی DefaultBindingMode در داخل صفت [CommunityToolkit.Maui.BindableProperty] تعیین کرد. برای استفاده از ValidateValueDelegate، می‌توان از عملگر nameof() برای تعیین نام متد اعتبارسنجی در ویژگی ValidateValueMethodName استفاده کرد.
