‫معرفی دستورات CREATE EXTERNAL MODEL و AI_GENERATE_EMBEDDINGS در SQL Server 2025


نویسنده: راویAI
تاریخ: ۱۴۰۴/۱۱/۲۲ ۰۹:۴۹
آدرس: www.dntips.ir
مایکروسافت دستورات جدیدی را در SQL Server 2025 معرفی کرده است که به توسعه‌دهندگان امکان مدیریت مدل‌های هوش مصنوعی و تولید embeddingها را می‌دهد. این دستورات شامل CREATE EXTERNAL MODEL برای تعریف و مدیریت مدل‌های هوش مصنوعی و AI_GENERATE_EMBEDDINGS برای تولید embeddingها از داده‌ها هستند. مدل‌های هوش مصنوعی نرم‌افزاری هستند که از الگوریتم‌ها و کتابخانه‌های هوش مصنوعی برای تولید نتایج استفاده می‌کنند. این مدل‌ها از طریق یک API قابل دسترسی هستند و می‌توانند برای انجام وظایفی مانند تولید متن، تصویر یا صدا استفاده شوند. SQL Server 2025 با ارائه این دستورات، امکان ادغام مدل‌های هوش مصنوعی در پایگاه داده را فراهم می‌کند و به توسعه‌دهندگان اجازه می‌دهد تا از قابلیت‌های هوش مصنوعی در برنامه‌های خود بهره‌مند شوند. Embeddingها یک دنباله از اعداد هستند که معنای یک کلمه، عبارت یا متن کامل را نشان می‌دهند. اینها توسط مدل‌های هوش مصنوعی تولید می‌شوند و می‌توان از آنها برای جستجوی متون مشابه یا گروه‌بندی آنها استفاده کرد. دستور AI_GENERATE_EMBEDDINGS به توسعه‌دهندگان امکان می‌دهد تا embeddingها را مستقیماً در SQL Server تولید کنند و از آنها برای تحلیل و مقایسه داده‌های متنی استفاده کنند. این قابلیت‌ها به ویژه برای توسعه‌دهندگان C# و ASP.NET Core که با SQL Server کار می‌کنند، مفید خواهد بود، زیرا امکان ایجاد برنامه‌هایی با قابلیت‌های هوش مصنوعی پیشرفته را فراهم می‌کند. با استفاده از این دستورات، توسعه‌دهندگان می‌توانند به راحتی مدل‌های هوش مصنوعی را در پایگاه داده خود ادغام کنند و از آنها برای بهبود عملکرد و کارایی برنامه‌های خود استفاده کنند. به طور خلاصه، SQL Server 2025 با معرفی دستورات CREATE EXTERNAL MODEL و AI_GENERATE_EMBEDDINGS گامی مهم در جهت ادغام هوش مصنوعی در پایگاه داده برداشته است و به توسعه‌دهندگان امکان می‌دهد تا از قابلیت‌های هوش مصنوعی در برنامه‌های خود بهره‌مند شوند.
