ЩҶЩҲЫҢШіЩҶШҜЩҮ: ШЁЩҮШұЩҲШІ ШЁШ®ШӘЫҢШ§ШұЫҢ

ШўШҜШұШі: www.dntips.ir

ШҜШұ ШіШ§Щ„ ЫұЫ№Ы·Ы¶ ШҙШ®ШөЫҢ ШЁЩҮ Thomas McCabe Щ…ШӘШұЫҢЪ©ЫҢ ШұШ§ ШЁШұШ§ЫҢ Ш§ЩҶШҜШ§ШІЩҮвҖҢЪҜЫҢШұЫҢ ЩҫЫҢЪҶЫҢШҜЪҜЫҢ ШҜШұ ЩҶШұЩ…вҖҢШ§ЩҒШІШ§Шұ ШЁЩҮ ЩҶШ§Щ… Cyclomatic complexity Щ…Ш№ШұЩҒЫҢ Ъ©ШұШҜ. Ш§ЩҲ ЩҶЫҢШІ ШӘШ№ШҜШ§ШҜ Щ…ШіЫҢШұЩҮШ§ЫҢ Щ…ШіШӘЩӮЩ„ЫҢ Ъ©ЩҮ Ъ©Щ„ ЫҢЪ© Щ…Ш§ЪҳЩҲЩ„ ЫҢШ§ Щ…ШӘШҜ ШұШ§ ЩҫЩҲШҙШҙ ШЁШҜЩҮШҜ ШЁЩҮ Ш№ЩҶЩҲШ§ЩҶ ЩҫЫҢЪҶЫҢШҜЪҜЫҢ ШўЩҶ ШҜШұ ЩҶШёШұ ЪҜШұЩҒШӘ.

Щ…Ш·Ш§Щ„ШЁ Щ…ШҙШ§ШЁЩҮ
    Ш§ШҙШӘШұШ§Ъ©вҖҢЩҮШ§Ш§ШіШӘЩҒШ§ШҜЩҮ Ш§ШІ Щ…Ш№ЫҢШ§ШұЩҮШ§ЫҢ Ъ©ШҜ ШҜШұ .NET ШЁШұШ§ЫҢ ШЁШ§ШІШіШ§ШІЫҢ Ъ©ЩҶШӘШұЩ„ШұЩҮШ§ЫҢ API ЩҫЫҢЪҶЫҢШҜЩҮШ§ШҙШӘШұШ§Ъ©вҖҢЩҮШ§Ш§ШіШӘЩҒШ§ШҜЩҮ Ш§ШІ Щ…Ш№ЫҢШ§ШұЩҮШ§ЫҢ Visual Studio ШЁШұШ§ЫҢ ШЁШ§ШІШіШ§ШІЫҢ ЩҮЩҲШҙЩ…ЩҶШҜШ§ЩҶЩҮ Ъ©ШҜ ШҜШұ .NETШ§ШҙШӘШұШ§Ъ©вҖҢЩҮШ§ШҜШұЫҢШ§ЩҒШӘ Code Metrics Powertool ШЁШұШ§ЫҢ Visual Studio 2013Ш§ШҙШӘШұШ§Ъ©вҖҢЩҮШ§Щ…ЩӮШ§ШЁЩ„ЩҮ ШЁШ§ ШЁШҜЩҮЫҢ ЩҒЩҶЫҢ ШЁШ§ Ш§ШіШӘЩҒШ§ШҜЩҮ Ш§ШІ Ш§ШЁШІШ§ШұЩҮШ§ЫҢ ШўЩҶШ§Щ„ЫҢШІ Щ„Ш§ЪҜ ЪҜЫҢШӘ ШЁШұШ§ЫҢ ШӘЩҲШіШ№ЩҮвҖҢШҜЩҮЩҶШҜЪҜШ§ЩҶ ШҜШ§ШӘвҖҢЩҶШӘШ§ШҙШӘШұШ§Ъ©вҖҢЩҮШ§ШЁШұШұШіЫҢ Code Metrics ШҜШұ ЩҲЫҢЪҳЩҲШ§Щ„ Ш§ШіШӘЩҲШҜЫҢЩҲЩ…Ш·Ш§Щ„ШЁЩ…Ш№ЫҢШ§ШұЩҮШ§ЫҢ Ъ©ШҜ (Code Metrics) ЩҲ ЩҫЫҢЪ©ШұШЁЩҶШҜЫҢЩ…Ш·Ш§Щ„ШЁDomain-Driven Design (DDD)Шӣ ШЁШ®Шҙ ШҜЩҲЩ…: ШІШЁШ§ЩҶЫҢ Щ…ШҙШӘШұЪ© ШЁШұШ§ЫҢ ШӘЫҢЩ…вҖҢЩҮШ§Щ…Ш·Ш§Щ„ШЁШўШҙЩҶШ§ЫҢЫҢ ШЁШ§ Refactoring - ЩӮШіЩ…ШӘ 9ЩҶШёШұШ§ШӘ Щ…Ш·Ш§Щ„ШЁ#Defensive Code in C - ЩӮШіЩ…ШӘ ШҜЩҲЩ…ЩҶШёШұШ§ШӘ Щ…Ш·Ш§Щ„ШЁШўШҙЩҶШ§ЫҢЫҢ ШЁШ§ Refactoring - ЩӮШіЩ…ШӘ 9
