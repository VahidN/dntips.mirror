ШҜШұ ШҜЩҶЫҢШ§ЫҢ ЩҫЩҲЫҢШ§ЫҢ ШӘЩҲШіШ№ЩҮ ЩҶШұЩ…вҖҢШ§ЩҒШІШ§ШұШҢ Щ…ШіШӘЩҶШҜШіШ§ШІЫҢ ЩҒЩҶЫҢ ШЁЩҮ Ш№ЩҶЩҲШ§ЩҶ ШіЩҶЪҜ ШЁЩҶШ§ЫҢ Ш§ЩҶШӘЩӮШ§Щ„ ШҜШ§ЩҶШҙШҢ ШӘШіЩҮЫҢЩ„ ЩҮЩ…Ъ©Ш§ШұЫҢ ЩҲ ШӘШ¶Щ…ЫҢЩҶ ЩҶЪҜЩҮШҜШ§ШұЫҢ ШөШӯЫҢШӯ ЩҫШұЩҲЪҳЩҮвҖҢЩҮШ§ ШҙЩҶШ§Ш®ШӘЩҮ Щ…ЫҢвҖҢШҙЩҲШҜ. ШЁШ§ ШёЩҮЩҲШұ ЩҮЩҲШҙ Щ…ШөЩҶЩҲШ№ЫҢШҢ ШұЩҲШҙвҖҢЩҮШ§ЫҢ ЩҶЩҲЫҢЩҶЫҢ ШЁШұШ§ЫҢ Ш§ШұШӘЩӮШ§ЫҢ Ъ©Ш§ШұШ§ЫҢЫҢ ЩҲ Ъ©ЫҢЩҒЫҢШӘ Щ…ШіШӘЩҶШҜШіШ§ШІЫҢ ЩҫШҜЫҢШҜШ§Шұ ШҙШҜЩҮвҖҢШ§ЩҶШҜ. ЩҮШіШӘЩҮ Ш§ШөЩ„ЫҢ ШӘШ№Ш§Щ…Щ„ Щ…ШӨШ«Шұ ШЁШ§ ЩҮЩҲШҙ Щ…ШөЩҶЩҲШ№ЫҢ ШҜШұ Ш§ЫҢЩҶ ШІЩ…ЫҢЩҶЩҮШҢ Ш§ШіШӘЩҒШ§ШҜЩҮ ЩҮЩҲШҙЩ…ЩҶШҜШ§ЩҶЩҮ Ш§ШІ ШҜШіШӘЩҲШұШ§ШӘ ЫҢШ§ ЩҮЩ…Ш§ЩҶ ЩҫШұШ§Щ…ЩҫШӘвҖҢЩҮШ§ (prompts) Ш§ШіШӘ. Ш§ЫҢЩҶ Щ…ЩӮШ§Щ„ЩҮ ШЁЩҮ ШЁШұШұШіЫҢ ЩҶЩӮШҙ Щ…ШӯЩҲШұЫҢ ЩҫШұШ§Щ…ЩҫШӘвҖҢЩҮШ§ ЩҲ ЩҶШӯЩҲЩҮ Ш§ШіШӘЩҒШ§ШҜЩҮ Ш§ШІ ШўЩҶвҖҢЩҮШ§ ШЁШұШ§ЫҢ ШҜШіШӘЫҢШ§ШЁЫҢ ШЁЩҮ Щ…ШіШӘЩҶШҜШ§ШӘ ЩҒЩҶЫҢ Ш§ШұШІШҙЩ…ЩҶШҜ ЩҲ Ъ©Ш§ШұШўЩ…ШҜШҢ Щ…ЫҢвҖҢЩҫШұШҜШ§ШІШҜ.

ЩҶЩӮШҙ Ъ©Щ„ЫҢШҜЫҢ ЩҫШұШ§Щ…ЩҫШӘвҖҢЩҮШ§ ШҜШұ ШӘШ№Ш§Щ…Щ„ ШЁШ§ ЩҮЩҲШҙ Щ…ШөЩҶЩҲШ№ЫҢ ШЁШұШ§ЫҢ Щ…ШіШӘЩҶШҜШіШ§ШІЫҢ
ЩҮЩҲШҙ Щ…ШөЩҶЩҲШ№ЫҢ ШЁЩҮ Ш®ЩҲШҜЫҢ Ш®ЩҲШҜ ЩӮШ§ШҜШұ ШЁЩҮ ШӘЩҲЩ„ЫҢШҜ Щ…ШіШӘЩҶШҜШ§ШӘ ЩҮШҜЩҒЩ…ЩҶШҜ ЩҶЫҢШіШӘ. Ш§ЫҢЩҶ ЩҫШұШ§Щ…ЩҫШӘвҖҢЩҮШ§ ЩҮШіШӘЩҶШҜ Ъ©ЩҮ Ш¬ЩҮШӘ ЩҲ ЩҮШҜЩҒ ШұШ§ ШЁШұШ§ЫҢ ШіЫҢШіШӘЩ…вҖҢЩҮШ§ЫҢ ЩҮЩҲШҙЩ…ЩҶШҜ ШӘШ№ЫҢЫҢЩҶ Щ…ЫҢвҖҢЪ©ЩҶЩҶШҜ. Ъ©ЫҢЩҒЫҢШӘ ЩҲ ШҜЩӮШӘ ЩҫШұШ§Щ…ЩҫШӘвҖҢЩҮШ§ШҢ ШЁЩҮ Ш·ЩҲШұ Щ…ШіШӘЩӮЫҢЩ… ШЁШұ Ъ©ЫҢЩҒЫҢШӘ ЩҲ Ш§ШұШӘШЁШ§Ш· ЩҫШ§ШіШ®вҖҢЩҮШ§ЫҢ ШӘЩҲЩ„ЫҢШҜ ШҙШҜЩҮ ШӘЩҲШіШ· ЩҮЩҲШҙ Щ…ШөЩҶЩҲШ№ЫҢ ШӘШЈШ«ЫҢШұ Щ…ЫҢвҖҢЪҜШ°Ш§ШұШҜ. ЫҢЪ© ЩҫШұШ§Щ…ЩҫШӘ Ш®ЩҲШҙвҖҢШіШ§Ш®ШӘ ЩҲ ШҜЩӮЫҢЩӮШҢ ЩҮЩҲШҙ Щ…ШөЩҶЩҲШ№ЫҢ ШұШ§ ЩӮШ§ШҜШұ Щ…ЫҢвҖҢШіШ§ШІШҜ ШӘШ§ Ш§Ш·Щ„Ш§Ш№Ш§ШӘ Щ…ЩҲШұШҜ ЩҶЫҢШ§ШІ ШұШ§ ШҜШұ ЩӮШ§Щ„ШЁ Щ…Ш·Щ„ЩҲШЁ ЩҲ ШЁШ§ Ш¬ШІШҰЫҢШ§ШӘ Ъ©Ш§ЩҒЫҢ Ш§ШұШ§ШҰЩҮ ШҜЩҮШҜ. ШҜШұ Щ…ЩӮШ§ШЁЩ„ШҢ ЩҫШұШ§Щ…ЩҫШӘвҖҢЩҮШ§ЫҢ Щ…ШЁЩҮЩ… ЫҢШ§ ЩҶШ§ЩӮШө Щ…ЫҢвҖҢШӘЩҲШ§ЩҶЩҶШҜ Щ…ЩҶШ¬Шұ ШЁЩҮ ШӘЩҲЩ„ЫҢШҜ Щ…ШіШӘЩҶШҜШ§ШӘЫҢ ШәЫҢШұЩ…ЩҒЫҢШҜ ЫҢШ§ ШӯШӘЫҢ ЪҜЩ…ШұШ§ЩҮвҖҢЪ©ЩҶЩҶШҜЩҮ ШҙЩҲЩҶШҜ. ШЁЩҶШ§ШЁШұШ§ЫҢЩҶШҢ ШҜШұЪ© ЩҲ ШЁЩҮвҖҢЪ©Ш§ШұЪҜЫҢШұЫҢ ШөШӯЫҢШӯ ЩҒЩҶЩҲЩҶ Ш·ШұШ§ШӯЫҢ ЩҫШұШ§Щ…ЩҫШӘвҖҢЩҮШ§ЫҢ Щ…ШӨШ«ШұШҢ ШЁШұШ§ЫҢ ЩҮШұ ШӘЩҲШіШ№ЩҮвҖҢШҜЩҮЩҶШҜЩҮ ЫҢШ§ ЩҶЩҲЫҢШіЩҶШҜЩҮ ЩҒЩҶЫҢ Ъ©ЩҮ Ш§ШІ ЩҮЩҲШҙ Щ…ШөЩҶЩҲШ№ЫҢ ШҜШұ ЩҒШұШ§ЫҢЩҶШҜ Щ…ШіШӘЩҶШҜШіШ§ШІЫҢ Ш®ЩҲШҜ Ш§ШіШӘЩҒШ§ШҜЩҮ Щ…ЫҢвҖҢЪ©ЩҶШҜШҢ Ш§Щ…ШұЫҢ Ш¶ШұЩҲШұЫҢ Ш§ШіШӘ.

ШЁШұШұШіЫҢ ЩҲ ШҜШіШӘЩҮвҖҢШЁЩҶШҜЫҢ ЩҫШұШ§Щ…ЩҫШӘвҖҢЩҮШ§ЫҢ Ъ©Щ„ЫҢШҜЫҢ ШҜШұ Щ…ШіШӘЩҶШҜШіШ§ШІЫҢ

ЩҫШұШ§Щ…ЩҫШӘвҖҢЩҮШ§ЫҢ Щ…ШұШЁЩҲШ· ШЁЩҮ ШӘЩҲЩ„ЫҢШҜ ЩҫЫҢШҙвҖҢЩҶЩҲЫҢШі Ш§ЩҲЩ„ЫҢЩҮ:
"Щ…ЩҶ ЩҶЫҢШ§ШІ ШЁЩҮ ЩҶЩҲШҙШӘЩҶ ЫҢЪ© ЩҒШ§ЫҢЩ„ README ШЁШұШ§ЫҢ ЫҢЪ© ЩҫШұЩҲЪҳЩҮ Ш¬ШҜЫҢШҜ ШҜШ§ШұЩ…. [ШҙШұШӯ ЩҫШұЩҲЪҳЩҮ]ШҢ Ъ©ЩҮ Ш§ШІ [ЩҒЩҶШ§ЩҲШұЫҢвҖҢЩҮШ§ЫҢ Щ…ЩҲШұШҜ Ш§ШіШӘЩҒШ§ШҜЩҮ] ШЁЩҮШұЩҮ Щ…ЫҢвҖҢШЁШұШҜ. Ш§ЫҢЩҶ ЩҫШұЩҲЪҳЩҮ ШҜШ§ШұШ§ЫҢ ЩҲЫҢЪҳЪҜЫҢвҖҢЩҮШ§ЫҢ Ъ©Щ„ЫҢШҜЫҢ ШІЫҢШұ Ш§ШіШӘ: [Щ„ЫҢШіШӘ ЩҲЫҢЪҳЪҜЫҢвҖҢЩҮШ§]. ЫҢЪ© ЩҒШ§ЫҢЩ„ README Ш¬Ш§Щ…Ш№ ШӘЩҲЩ„ЫҢШҜ Ъ©ЩҶЫҢШҜ Ъ©ЩҮ ШҙШ§Щ…Щ„ ШЁШ®ШҙвҖҢЩҮШ§ЫҢ ШІЫҢШұ ШЁШ§ШҙШҜ: [ШіШ§Ш®ШӘШ§Шұ ШЁШ®ШҙвҖҢЩҮШ§ЫҢ Щ…ЩҲШұШҜ ЩҶШёШұ]."
  • ШӘЩҲШ¶ЫҢШӯ: Ш§ЫҢЩҶ ЩҫШұШ§Щ…ЩҫШӘШҢ ШЁШ§ Ш§ШұШ§ШҰЩҮ ШІЩ…ЫҢЩҶЩҮ Ъ©Ш§ЩҒЫҢ ШҜШұ Щ…ЩҲШұШҜ ЩҫШұЩҲЪҳЩҮШҢ ЩҒЩҶШ§ЩҲШұЫҢвҖҢЩҮШ§ЫҢ Щ…ЩҲШұШҜ Ш§ШіШӘЩҒШ§ШҜЩҮ ЩҲ ЩҲЫҢЪҳЪҜЫҢвҖҢЩҮШ§ЫҢ Ъ©Щ„ЫҢШҜЫҢШҢ Ш§ШІ ЩҮЩҲШҙ Щ…ШөЩҶЩҲШ№ЫҢ Щ…ЫҢвҖҢШ®ЩҲШ§ЩҮШҜ ШӘШ§ ЫҢЪ© ЩҒШ§ЫҢЩ„ README ШЁШ§ ШіШ§Ш®ШӘШ§Шұ Щ…ШҙШ®Шө ШӘЩҲЩ„ЫҢШҜ Ъ©ЩҶШҜ.
"ШҜШұ Ш§ЫҢЩҶШ¬Ш§ Ш§ЩҒЪ©Ш§Шұ Ш§ЩҲЩ„ЫҢЩҮ Щ…ЩҶ ШЁШұШ§ЫҢ ЫҢЪ© Щ…ЩӮШ§Щ„ЩҮ ШўЩҲШұШҜЩҮ ШҙШҜЩҮ Ш§ШіШӘ. ШўЩҶвҖҢЩҮШ§ ШұШ§ ШҜШұ ЫҢЪ© ШіШ§Ш®ШӘШ§Шұ ЩҲШ§Ш¶Шӯ ШЁШ§ ШЁШ®ШҙвҖҢЩҮШ§ЫҢ Щ…ШҙШ®Шө ШіШ§ШІЩ…Ш§ЩҶШҜЩҮЫҢ Ъ©ЩҶЫҢШҜ."
  • ШӘЩҲШ¶ЫҢШӯ: Ш§ЫҢЩҶ ЩҫШұШ§Щ…ЩҫШӘ ШЁШұШ§ЫҢ ШіШ§ШІЩ…Ш§ЩҶШҜЩҮЫҢ Ш§ЫҢШҜЩҮвҖҢЩҮШ§ЫҢ ЩҫШұШ§Ъ©ЩҶШҜЩҮ ЩҲ Ш§ЫҢШ¬Ш§ШҜ ЫҢЪ© ШіШ§Ш®ШӘШ§Шұ Щ…ЩҶШіШ¬Щ… ШЁШұШ§ЫҢ Щ…ШіШӘЩҶШҜШ§ШӘ Щ…ЩҲШұШҜ Ш§ШіШӘЩҒШ§ШҜЩҮ ЩӮШұШ§Шұ Щ…ЫҢвҖҢЪҜЫҢШұШҜ.

ЩҫШұШ§Щ…ЩҫШӘвҖҢЩҮШ§ЫҢ Щ…ШұШЁЩҲШ· ШЁЩҮ ШЁЩҮШЁЩҲШҜ Щ…ШіШӘЩҶШҜШ§ШӘ Щ…ЩҲШ¬ЩҲШҜ:
"ЩҒШұШ§ЫҢЩҶШҜ ШіШ·Шӯ ШЁШ§Щ„Ш§ЫҢ Щ…Ш§ ШЁЩҮ Ш§ЫҢЩҶ ШөЩҲШұШӘ Ш§ШіШӘ: [ШҙШұШӯ ЩҒШұШ§ЫҢЩҶШҜ]. ЫҢЪ© ШұШ§ЩҮЩҶЩ…Ш§ЫҢ ЪҜШ§Щ…вҖҢШЁЩҮвҖҢЪҜШ§Щ… ШҜЩӮЫҢЩӮ ШЁШұШ§ЫҢ ЩҲШұЩҲШҜ ЫҢЪ© ШӘЩҲШіШ№ЩҮвҖҢШҜЩҮЩҶШҜЩҮ Ш¬ШҜЫҢШҜ ШЁЩҮ ШӘЫҢЩ… ШӘЩҲЩ„ЫҢШҜ Ъ©ЩҶЫҢШҜ ЫҢШ§ ШҜШіШӘЩҲШұШ§Щ„Ш№Щ…Щ„вҖҢЩҮШ§ЫҢ Щ…ЩҲШ¬ЩҲШҜ Щ…Ш§ ШұШ§ ШЁШ§ШІШЁЫҢЩҶЫҢ Ъ©ЩҶЫҢШҜ."
  • ШӘЩҲШ¶ЫҢШӯ: Ш§ЫҢЩҶ ЩҫШұШ§Щ…ЩҫШӘ ЩҮЩ… ШЁШұШ§ЫҢ ШӘЩҲЩ„ЫҢШҜ Щ…ШіШӘЩҶШҜШ§ШӘ Ш¬ШҜЫҢШҜ ЩҲ ЩҮЩ… ШЁШұШ§ЫҢ ШЁШ§ШІШЁЫҢЩҶЫҢ ЩҲ ШЁЩҮШЁЩҲШҜ Щ…ШіШӘЩҶШҜШ§ШӘ Щ…ЩҲШ¬ЩҲШҜ Ъ©Ш§ШұШЁШұШҜ ШҜШ§ШұШҜ.

ЩҫШұШ§Щ…ЩҫШӘвҖҢЩҮШ§ЫҢ Щ…ШұШЁЩҲШ· ШЁЩҮ Ш®Щ„Ш§ШөЩҮвҖҢШіШ§ШІЫҢ:
"Щ…ШіШӘЩҶШҜШ§ШӘ API ШІЫҢШұ ШұШ§ ШҜШұ Ыі-Ыҙ Ш¬Щ…Щ„ЩҮ Ш®Щ„Ш§ШөЩҮ Ъ©ЩҶЫҢШҜ ЩҲ ЩҶЩӮШ§Ш· ЩҫШ§ЫҢШ§ЩҶЫҢ Ъ©Щ„ЫҢШҜЫҢ ЩҲ Ш¬ШІШҰЫҢШ§ШӘ Ш§ШӯШұШ§ШІ ЩҮЩҲЫҢШӘ ШұШ§ ШЁШұШ¬ШіШӘЩҮ Ъ©ЩҶЫҢШҜ: [Щ…ШӘЩҶ Щ…ШіШӘЩҶШҜШ§ШӘ]."
  • ШӘЩҲШ¶ЫҢШӯ: Ш§ЫҢЩҶ ЩҫШұШ§Щ…ЩҫШӘ ШЁШұШ§ЫҢ Ш§ШіШӘШ®ШұШ§Ш¬ Ш§Ш·Щ„Ш§Ш№Ш§ШӘ Ъ©Щ„ЫҢШҜЫҢ Ш§ШІ Щ…ШіШӘЩҶШҜШ§ШӘ Ш·ЩҲЩ„Ш§ЩҶЫҢ ЩҲ Ш§ШұШ§ШҰЩҮ ЫҢЪ© Ш®Щ„Ш§ШөЩҮ Щ…Ш®ШӘШөШұ ЩҲ Щ…ЩҒЫҢШҜ Ш§ШіШӘЩҒШ§ШҜЩҮ Щ…ЫҢвҖҢШҙЩҲШҜ.

ЩҫШұШ§Щ…ЩҫШӘвҖҢЩҮШ§ЫҢ Щ…ШұШЁЩҲШ· ШЁЩҮ ШӘШӯЩ„ЫҢЩ„ ЩҲ ШЁЩҮШЁЩҲШҜ ЩҲШ¶ЩҲШӯ:
"Ш§ЫҢЩҶ Щ…ШіШӘЩҶШҜШ§ШӘ ШұШ§ Ш§ШІ ЩҶШёШұ ЩҲШ¶ЩҲШӯ ЩҲ Ш®ЩҲШ§ЩҶШ§ЫҢЫҢ ШӘШ¬ШІЫҢЩҮ ЩҲ ШӘШӯЩ„ЫҢЩ„ Ъ©ЩҶЫҢШҜ. ЩҮШұ ШЁШ®Шҙ Щ…ШӘШұШ§Ъ©Щ… ЫҢШ§ ЪҜЫҢШ¬вҖҢЪ©ЩҶЩҶШҜЩҮ ШұШ§ ШЁШұШ¬ШіШӘЩҮ Ъ©ШұШҜЩҮ ЩҲ ЩҫЫҢШҙЩҶЩҮШ§ШҜЩҮШ§ЫҢЫҢ ШЁШұШ§ЫҢ Щ…Ш®ШӘШөШұШӘШұ ЩҲ ЩӮШ§ШЁЩ„ ЩҒЩҮЩ…вҖҢШӘШұ Ъ©ШұШҜЩҶ ШўЩҶ Ш§ШұШ§ШҰЩҮ ШҜЩҮЫҢШҜ."
  • ШӘЩҲШ¶ЫҢШӯ: Ш§ЫҢЩҶ ЩҫШұШ§Щ…ЩҫШӘ ШЁЩҮ Щ…ЩҶШёЩҲШұ ШЁЩҮШЁЩҲШҜ Ъ©ЫҢЩҒЫҢШӘ ЩҶЪҜШ§ШұШҙ ЩҲ Ш§ЩҒШІШ§ЫҢШҙ ШҜШұЪ©вҖҢЩҫШ°ЫҢШұЫҢ Щ…ШіШӘЩҶШҜШ§ШӘ ШЁЩҮ Ъ©Ш§Шұ Щ…ЫҢвҖҢШұЩҲШҜ.
"Щ…ШіШӘЩҶШҜШ§ШӘ API ШІЫҢШұ ШұШ§ ШЁШұШұШіЫҢ Ъ©ЩҶЫҢШҜ ЩҲ Ш§Ш·Щ…ЫҢЩҶШ§ЩҶ ШӯШ§ШөЩ„ Ъ©ЩҶЫҢШҜ Ъ©ЩҮ Ш§ШөШ·Щ„Ш§ШӯШ§ШӘ ШҜШұ ШіШұШ§ШіШұ ШўЩҶ ЫҢЪ©ШіШ§ЩҶ Ш§ШіШӘ."
  • ШӘЩҲШ¶ЫҢШӯ: Ш§ЫҢЩҶ ЩҫШұШ§Щ…ЩҫШӘ ШЁШұ ШӯЩҒШё ЫҢЪ©ЩҫШ§ШұЪҶЪҜЫҢ ЩҲ Ш§ЩҶШіШ¬Ш§Щ… ШҜШұ Ш§ШіШӘЩҒШ§ШҜЩҮ Ш§ШІ Ш§ШөШ·Щ„Ш§ШӯШ§ШӘ ЩҒЩҶЫҢ ШҜШұ ШіШұШ§ШіШұ Щ…ШіШӘЩҶШҜШ§ШӘ ШӘЩ…ШұЪ©ШІ ШҜШ§ШұШҜ.

ЩҫШұШ§Щ…ЩҫШӘвҖҢЩҮШ§ЫҢ Щ…ШұШЁЩҲШ· ШЁЩҮ ШӘШЁШҜЫҢЩ„ ЪҜЩҒШӘЪҜЩҲЩҮШ§ ШЁЩҮ Щ…ШіШӘЩҶШҜШ§ШӘ:
"Ш®Щ„Ш§ШөЩҮвҖҢШ§ЫҢ Ш§ШІ Ш§ЫҢЩҶ Щ…ШӘЩҶ Ш¬Щ„ШіЩҮ ШұШ§ ШЁЩҮ ЩҮЩ…ШұШ§ЩҮ Щ…ЩҲШ§ШұШҜ Ш§ЩӮШҜШ§Щ… Ъ©Щ„ЫҢШҜЫҢ ЩҲ ШЁЫҢЩҶШҙвҖҢЩҮШ§ЫҢ ЩҒЩҶЫҢ ШЁШұШ§ЫҢ ШӘЫҢЩ… Ш§ШұШ§ШҰЩҮ ШҜЩҮЫҢШҜ."
  • ШӘЩҲШ¶ЫҢШӯ: Ш§ЫҢЩҶ ЩҫШұШ§Щ…ЩҫШӘ ШЁШұШ§ЫҢ ШӘШЁШҜЫҢЩ„ Ш®ШұЩҲШ¬ЫҢ Ш¬Щ„ШіШ§ШӘ ЩҲ ШЁШӯШ«вҖҢЩҮШ§ ШЁЩҮ Щ…ШіШӘЩҶШҜШ§ШӘ ЩӮШ§ШЁЩ„ Ш§ШіШӘЩҒШ§ШҜЩҮ ШЁЩҮ Ъ©Ш§Шұ Щ…ЫҢвҖҢШұЩҲШҜ.
"ШҜШұ Ш§ЫҢЩҶШ¬Ш§ ЫҢЪ© Щ…ШҙЪ©Щ„ЫҢ Ъ©ЩҮ Ш§Щ…ШұЩҲШІ ШӯЩ„ Ъ©ШұШҜЩ… ШўЩҲШұШҜЩҮ ШҙШҜЩҮ Ш§ШіШӘ: [ШҙШұШӯ Щ…ШҙЪ©Щ„ ЩҲ ШұШ§ЩҮвҖҢШӯЩ„]. Щ…ШұШ§ШӯЩ„ Ш№ЫҢШЁвҖҢЫҢШ§ШЁЫҢ ШұШ§ ШЁЩҮ ЪҜЩҲЩҶЩҮвҖҢШ§ЫҢ ШӘЩҶШёЫҢЩ… Ъ©ЩҶЫҢШҜ Ъ©ЩҮ ШҜЫҢЪҜШұШ§ЩҶ ШЁШӘЩҲШ§ЩҶЩҶШҜ ШҜЩҒШ№ЩҮ ШЁШ№ШҜ ШўЩҶ ШұШ§ ШіШұЫҢШ№вҖҢШӘШұ ШӯЩ„ Ъ©ЩҶЩҶШҜ."
  • ШӘЩҲШ¶ЫҢШӯ: Ш§ЫҢЩҶ ЩҫШұШ§Щ…ЩҫШӘ ШЁШұШ§ЫҢ Ш«ШЁШӘ ШҜШ§ЩҶШҙ ШӯШ§ШөЩ„ Ш§ШІ ШӯЩ„ Щ…ШҙЪ©Щ„Ш§ШӘ ЩҒЩҶЫҢ ЩҲ ШӘШіЩҮЫҢЩ„ ЩҒШұШ§ЫҢЩҶШҜ Ш№ЫҢШЁвҖҢЫҢШ§ШЁЫҢ ШҜШұ ШўЫҢЩҶШҜЩҮ Ш§ШіШӘЩҒШ§ШҜЩҮ Щ…ЫҢвҖҢШҙЩҲШҜ.

ЩҫШұШ§Щ…ЩҫШӘвҖҢЩҮШ§ЫҢ Щ…ШұШЁЩҲШ· ШЁЩҮ ШЁЩҮвҖҢШұЩҲШІШұШіШ§ЩҶЫҢ Щ…ШіШӘЩҶШҜШ§ШӘ:
"Ш§ЫҢЩҶ ШіЩҶШҜ ШұШ§ ШЁШұ Ш§ШіШ§Ші ШўШ®ШұЫҢЩҶ ШӘШәЫҢЫҢШұШ§ШӘ Ъ©ШҜ Ш§ШІ ЩҶШёШұ Ш§Ш·Щ„Ш§Ш№Ш§ШӘ Щ…ЩҶШіЩҲШ® ШӘШ¬ШІЫҢЩҮ ЩҲ ШӘШӯЩ„ЫҢЩ„ Ъ©ЩҶЫҢШҜ: [Щ…ШӘЩҶ ШіЩҶШҜ]."
  • ШӘЩҲШ¶ЫҢШӯ: Ш§ЫҢЩҶ ЩҫШұШ§Щ…ЩҫШӘ ШЁЩҮ Щ…ЩҶШёЩҲШұ ШҙЩҶШ§ШіШ§ЫҢЫҢ ЩҲ ШӯШ°ЩҒ Ш§Ш·Щ„Ш§Ш№Ш§ШӘ ЩӮШҜЫҢЩ…ЫҢ ЩҲ ЩҶШ§Щ…Ш№ШӘШЁШұ Ш§ШІ Щ…ШіШӘЩҶШҜШ§ШӘ ШЁЩҮ Ъ©Ш§Шұ Щ…ЫҢвҖҢШұЩҲШҜ.

ЩҫШұШ§Щ…ЩҫШӘвҖҢЩҮШ§ЫҢ Щ…ШұШЁЩҲШ· ШЁЩҮ ЩҶЩӮШҙвҖҢШўЩҒШұЫҢЩҶЫҢ (Rubber Ducking):
Ш§ШІ ЪҶШӘвҖҢШЁШ§ШӘ Ш®ЩҲШҜ ШЁШ®ЩҲШ§ЩҮЫҢШҜ ЩҶЩӮШҙ ЫҢЪ© ЩҮЩ…Ъ©Ш§Шұ Ъ©ЩҶШ¬Ъ©Ш§ЩҲ ШұШ§ ШЁШ§ШІЫҢ Ъ©ЩҶШҜ ЩҲ ШҙЪ©Ш§ЩҒвҖҢЩҮШ§ЫҢ Ш§ШӯШӘЩ…Ш§Щ„ЫҢ ШҜШұ Щ…ШіШӘЩҶШҜШ§ШӘ ШҙЩ…Ш§ ШұШ§ ЪҜЩҲШҙШІШҜ Ъ©ШұШҜЩҮ ЩҲ ШӘЩҲШ¶ЫҢШӯШ§ШӘ ШЁЩҮШӘШұЫҢ ШұШ§ ШҜШұШ®ЩҲШ§ШіШӘ Ъ©ЩҶШҜ.
  • ШӘЩҲШ¶ЫҢШӯ: Ш§ЫҢЩҶ ЩҶЩҲШ№ ЩҫШұШ§Щ…ЩҫШӘ ШЁШұШ§ЫҢ ШҜШұЫҢШ§ЩҒШӘ ШЁШ§ШІШ®ЩҲШұШҜ ЩҲ ШҙЩҶШ§ШіШ§ЫҢЫҢ ЩҶЩӮШ§Ш· Ш¶Ш№ЩҒ Ш§ШӯШӘЩ…Ш§Щ„ЫҢ ШҜШұ Щ…ШіШӘЩҶШҜШ§ШӘ Ш§ШІ Ш·ШұЫҢЩӮ ШҙШЁЫҢЩҮвҖҢШіШ§ШІЫҢ ШӘШ№Ш§Щ…Щ„ ШЁШ§ ЫҢЪ© ЩҮЩ…Ъ©Ш§Шұ Ш§ШіШӘЩҒШ§ШҜЩҮ Щ…ЫҢвҖҢШҙЩҲШҜ.
Ш§ШІ ЩҮЩҲШҙ Щ…ШөЩҶЩҲШ№ЫҢ ШЁШ®ЩҲШ§ЩҮЫҢШҜ ШҜЫҢШҜЪҜШ§ЩҮ ЫҢЪ© ШӘЩҲШіШ№ЩҮвҖҢШҜЩҮЩҶШҜЩҮ Ш¬ШҜЫҢШҜ ШұШ§ Ш§ШӘШ®Ш§Ш° Ъ©ШұШҜЩҮ ЩҲ ЩҒШұШ§ЫҢЩҶШҜЩҮШ§ ЩҲ ЩҒШ§ЫҢЩ„вҖҢЩҮШ§ЫҢ README ШҙЩ…Ш§ ШұШ§ ШЁШұШұШіЫҢ Ъ©ЩҶШҜ ЩҲ ЩҮШұЪҜЩҲЩҶЩҮ Щ…ШұШӯЩ„ЩҮ Щ…ШЁЩҮЩ… ЫҢШ§ Ш¬ШІШҰЫҢШ§ШӘ Ш§ШІ ШҜШіШӘ ШұЩҒШӘЩҮвҖҢШ§ЫҢ Ъ©ЩҮ Щ…Щ…Ъ©ЩҶ Ш§ШіШӘ ШЁШұШ§ЫҢ ЩҒШұШҜЫҢ ЩҶШ§ШўШҙЩҶШ§ ШЁШ§ ЩҫШұЩҲЪҳЩҮ Щ…ШҙЪ©Щ„вҖҢШіШ§ШІ ШЁШ§ШҙШҜ ШұШ§ Ш№Щ„Ш§Щ…ШӘвҖҢЪҜШ°Ш§ШұЫҢ Ъ©ЩҶШҜ.
  • ШӘЩҲШ¶ЫҢШӯ: Ш§ЫҢЩҶ ЩҫШұШ§Щ…ЩҫШӘ ШЁЩҮ Щ…ЩҶШёЩҲШұ Ш§Ш·Щ…ЫҢЩҶШ§ЩҶ Ш§ШІ ЩҲШ¶ЩҲШӯ ЩҲ Ъ©Ш§Щ…Щ„ ШЁЩҲШҜЩҶ Щ…ШіШӘЩҶШҜШ§ШӘ ШЁШұШ§ЫҢ Ш§ЩҒШұШ§ШҜ ШӘШ§ШІЩҮ ЩҲШ§ШұШҜ ШЁЩҮ ЩҫШұЩҲЪҳЩҮ Ш·ШұШ§ШӯЫҢ ШҙШҜЩҮ Ш§ШіШӘ.

Щ…Щ„Ш§ШӯШёШ§ШӘ ШӘЪ©Щ…ЫҢЩ„ЫҢ ШҜШұ Ш§ШіШӘЩҒШ§ШҜЩҮ Ш§ШІ ЩҫШұШ§Щ…ЩҫШӘвҖҢЩҮШ§
Ш№Щ„Ш§ЩҲЩҮ ШЁШұ Ш§ШұШ§ШҰЩҮ ЩҫШұШ§Щ…ЩҫШӘвҖҢЩҮШ§ЫҢ Щ…ШҙШ®ШөШҢ Щ…ЩӮШ§Щ„ЩҮ Ш§ШөЩ„ЫҢ ШЁШұ Ш§ЩҮЩ…ЫҢШӘ Ш§ШұШ§ШҰЩҮ ШІЩ…ЫҢЩҶЩҮ Ъ©Ш§ЩҒЫҢ (Context)ШҢ ШӘШ№ЫҢЫҢЩҶ ЩҒШұЩ…ШӘ Ш®ШұЩҲШ¬ЫҢ Щ…Ш·Щ„ЩҲШЁ ЩҲ Ш§Ш№Щ…Ш§Щ„ Щ…ШӯШҜЩҲШҜЫҢШӘвҖҢЩҮШ§ЫҢ Щ„Ш§ШІЩ… ШҜШұ ЩҫШұШ§Щ…ЩҫШӘвҖҢЩҮШ§ ШӘШЈЪ©ЫҢШҜ Щ…ЫҢвҖҢЪ©ЩҶШҜ. Ш§ЫҢЩҶ Ш№ЩҶШ§ШөШұ ШЁЩҮ ЩҮЩҲШҙ Щ…ШөЩҶЩҲШ№ЫҢ Ъ©Щ…Ъ© Щ…ЫҢвҖҢЪ©ЩҶЩҶШҜ ШӘШ§ ШҜШұЪ© ШҜЩӮЫҢЩӮвҖҢШӘШұЫҢ Ш§ШІ ЩҶЫҢШ§ШІЩҮШ§ЫҢ Ъ©Ш§ШұШЁШұ ШҜШ§ШҙШӘЩҮ ШЁШ§ШҙШҜ ЩҲ ЩҫШ§ШіШ®вҖҢЩҮШ§ЫҢ Щ…ШұШӘШЁШ·вҖҢШӘШұ ЩҲ Щ…ЩҒЫҢШҜШӘШұЫҢ Ш§ШұШ§ШҰЩҮ ШҜЩҮШҜ. ЩҮЩ…ЪҶЩҶЫҢЩҶШҢ ШұЩҲЫҢЪ©ШұШҜ ШўШІЩ…ЩҲЩҶ ЩҲ Ш®Ш·Ш§ ШҜШұ Ш·ШұШ§ШӯЫҢ ЩҫШұШ§Щ…ЩҫШӘвҖҢЩҮШ§ Щ…ЫҢвҖҢШӘЩҲШ§ЩҶШҜ ШЁШіЫҢШ§Шұ Щ…ШӨШ«Шұ ШЁШ§ШҙШҜ. ШЁШ§ ШўШІЩ…Ш§ЫҢШҙ ЩҫШұШ§Щ…ЩҫШӘвҖҢЩҮШ§ЫҢ Щ…Ш®ШӘЩ„ЩҒ ЩҲ Ш§ШұШІЫҢШ§ШЁЫҢ ЩҶШӘШ§ЫҢШ¬ШҢ Щ…ЫҢвҖҢШӘЩҲШ§ЩҶ ШЁЩҮ ШӘШҜШұЫҢШ¬ ШЁЩҮ ЩҫШұШ§Щ…ЩҫШӘвҖҢЩҮШ§ЫҢ ШЁЩҮЫҢЩҶЩҮ ШЁШұШ§ЫҢ ЩҲШёШ§ЫҢЩҒ Щ…ШіШӘЩҶШҜШіШ§ШІЫҢ Ш®Ш§Шө ШҜШіШӘ ЫҢШ§ЩҒШӘ.
