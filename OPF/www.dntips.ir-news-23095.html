Ш§ЫҢЩҶ Щ…ЩӮШ§Щ„ЩҮ ШЁЩҮ ШЁШұШұШіЫҢ Ш§ШіШӘШұШ§ШӘЪҳЫҢвҖҢЩҮШ§ЫҢ ЩҫЫҢШҙШұЩҒШӘЩҮ Ъ©Шҙ ШЁШ§ Ш§ШіШӘЩҒШ§ШҜЩҮ Ш§ШІ .NET ЩҲ Redis Щ…ЫҢвҖҢЩҫШұШҜШ§ШІШҜ. ШӘЩ…ШұЪ©ШІ Ш§ШөЩ„ЫҢ ШЁШұ ШұЩҲЫҢ ЩҮЩ…ЪҜШ§Щ…вҖҢШіШ§ШІЫҢ Ъ©Шҙ L1 (ШӯШ§ЩҒШёЩҮ ШҜШ§Ш®Щ„ЫҢ) ЩҲ Ъ©Шҙ L2 (Redis) ШҜШұ ЫҢЪ© ШЁШұЩҶШ§Щ…ЩҮ ШӘЩҲШІЫҢШ№вҖҢШҙШҜЩҮ .NET Ш§ШіШӘ. Щ…ШҙЪ©Щ„ Ш§ШөЩ„ЫҢ ШҜШұ Ш§ЫҢЩҶ ШіЩҶШ§ШұЫҢЩҲШҢ Ш§ШұШ§ШҰЩҮ ЩҶШӘШ§ЫҢШ¬ ЩӮШҜЫҢЩ…ЫҢ Ш§ШІ ШӯШ§ЩҒШёЩҮ ШҜШ§Ш®Щ„ЫҢ ЩҮШұ ШіШұЩҲШұ ШҜШұ ШӯШ§Щ„ЫҢ Ъ©ЩҮ ШіШұЩҲШұЩҮШ§ЫҢ ШҜЫҢЪҜШұ ШҜШ§ШҜЩҮвҖҢЩҮШ§ ШұШ§ ШЁЩҮвҖҢШұЩҲШІШұШіШ§ЩҶЫҢ Щ…ЫҢвҖҢЪ©ЩҶЩҶШҜШҢ Ш§ШіШӘ. ШұШ§ЩҮ ШӯЩ„ ЩҫЫҢШҙЩҶЩҮШ§ШҜЫҢШҢ Ш§ШіШӘЩҒШ§ШҜЩҮ Ш§ШІ Redis Pub/Sub ШЁЩҮ Ш№ЩҶЩҲШ§ЩҶ ЫҢЪ© ШЁЪ©вҖҢШ§ЩҶШҜ ШЁШұШ§ЫҢ ЩҶШ§Щ…Ш№ШӘШЁШұ Ъ©ШұШҜЩҶ Ъ©Шҙ L1 ШҜШұ ШіШұЩҲШұЩҮШ§ЫҢ Щ…Ш®ШӘЩ„ЩҒ Ш§ШіШӘ. ШҜШұ Ш§ЫҢЩҶ ШұЩҲШҙШҢ ЩҮШұ ШІЩ…Ш§ЩҶ Ъ©ЩҮ ЫҢЪ© ШіШұЩҲШұ ШҜШ§ШҜЩҮвҖҢЩҮШ§ ШұШ§ ШЁЩҮвҖҢШұЩҲШІШұШіШ§ЩҶЫҢ Щ…ЫҢвҖҢЪ©ЩҶШҜШҢ ЫҢЪ© ЩҫЫҢШ§Щ… ЩҶШ§Щ…Ш№ШӘШЁШұШіШ§ШІЫҢ Щ…ЩҶШӘШҙШұ Щ…ЫҢвҖҢШҙЩҲШҜ ЩҲ ЩҮШұ Ъ©ЩҫЫҢ (replica) Ъ©Шҙ Щ…ШұШЁЩҲШ·ЩҮ ШұШ§ ШЁЩҮ ШөЩҲШұШӘ Щ…ШӯЩ„ЫҢ ЩҫШ§Ъ© Щ…ЫҢвҖҢЪ©ЩҶШҜ. Ш§ЫҢЩҶ ШұЩҲЫҢЪ©ШұШҜ ШЁЩҮ Ш·ЩҲШұ Щ…ЩҲШ«Шұ Щ…ШҙЪ©Щ„ Ш§ШұШ§ШҰЩҮ ЩҶШӘШ§ЫҢШ¬ ЩӮШҜЫҢЩ…ЫҢ ШҜШұ ЫҢЪ© Щ…ШӯЫҢШ· ШӘЩҲШІЫҢШ№вҖҢШҙШҜЩҮ ШұШ§ ШӯЩ„ Щ…ЫҢвҖҢЪ©ЩҶШҜ. Щ…ЩӮШ§Щ„ЩҮ ШЁЩҮ ШЁШұШұШіЫҢ ЩҶШӯЩҲЩҮ Ш№Щ…Щ„Ъ©ШұШҜ Ъ©Шҙ ШҜЩҲ ШіШ·ШӯЫҢ (L1 + L2) ШҜШұ ЫҢЪ© ШіЫҢШіШӘЩ… ШӘЩҲШІЫҢШ№вҖҢШҙШҜЩҮ Щ…ЫҢвҖҢЩҫШұШҜШ§ШІШҜ ЩҲ ШӘЩҲШ¶ЫҢШӯ Щ…ЫҢвҖҢШҜЩҮШҜ Ъ©ЩҮ ЪҶШұШ§ TTLЩҮШ§ (ШІЩ…Ш§ЩҶ Ш§ЩҶЩӮШ¶Ш§) Щ…ЫҢвҖҢШӘЩҲШ§ЩҶЩҶШҜ Щ…ШҙЪ©Щ„ ШұШ§ ЩҫЩҶЩҮШ§ЩҶ Ъ©ЩҶЩҶШҜ Ш§Щ…Ш§ ШўЩҶ ШұШ§ ШӯЩ„ ЩҶЩ…ЫҢвҖҢЪ©ЩҶЩҶШҜ. ЩҮЩ…ЪҶЩҶЫҢЩҶШҢ ЩҶШӯЩҲЩҮ ЩҫЫҢШ§ШҜЩҮвҖҢШіШ§ШІЫҢ ЫҢЪ© ШЁЪ©вҖҢШ§ЩҶШҜ Redis (Pub/Sub) ШЁШұШ§ЫҢ ЩҶШ§Щ…Ш№ШӘШЁШұШіШ§ШІЫҢ ШҜШұ .NET ШЁШ§ Ш§ШіШӘЩҒШ§ШҜЩҮ Ш§ШІ ЫҢЪ© ШіШұЩҲЫҢШі ЩҫШівҖҢШІЩ…ЫҢЩҶЩҮ ЩҲ StackExchange.Redis ШұШ§ ШўЩ…ЩҲШІШҙ Щ…ЫҢвҖҢШҜЩҮШҜ. ШҜШұ ЩҶЩҮШ§ЫҢШӘШҢ Ш§ЫҢЩҶ Щ…ЩӮШ§Щ„ЩҮ ШЁЩҮ ШҙЩ…Ш§ ЩҶШҙШ§ЩҶ Щ…ЫҢвҖҢШҜЩҮШҜ Ъ©ЩҮ ЪҶЪҜЩҲЩҶЩҮ ШЁШ§ Ш§ШіШӘЩҒШ§ШҜЩҮ Ш§ШІ Щ„Ш§ЪҜвҖҢЩҮШ§ ЩҲ ШұШҜЫҢШ§ШЁЫҢвҖҢЩҮШ§ШҢ Ш§ШіШӘШұШ§ШӘЪҳЫҢ Ъ©Шҙ Ш®ЩҲШҜ ШұШ§ ШҜШұ ШӯШ§Щ„ЫҢ Ъ©ЩҮ ЪҶЩҶШҜЫҢЩҶ Ъ©ЩҫЫҢ API ШұШ§ Ш§Ш¬ШұШ§ Щ…ЫҢвҖҢЪ©ЩҶЫҢШҜШҢ Ш§Ш№ШӘШЁШ§ШұШіЩҶШ¬ЫҢ Ъ©ЩҶЫҢШҜ. Ш§ЫҢЩҶ ШұЩҲШҙ ШЁЩҮ ШӘЩҲШіШ№ЩҮвҖҢШҜЩҮЩҶШҜЪҜШ§ЩҶ .NET Ъ©Щ…Ъ© Щ…ЫҢвҖҢЪ©ЩҶШҜ ШӘШ§ ШіЫҢШіШӘЩ…вҖҢЩҮШ§ЫҢ Ъ©Шҙ ШӘЩҲШІЫҢШ№вҖҢШҙШҜЩҮ Ш®ЩҲШҜ ШұШ§ ШЁЩҮ Ш·ЩҲШұ Щ…ЩҲШ«ШұШӘШұЫҢ Щ…ШҜЫҢШұЫҢШӘ ЩҲ ШЁЩҮЫҢЩҶЩҮвҖҢШіШ§ШІЫҢ Ъ©ЩҶЩҶШҜ. Ш§ЫҢЩҶ ШўЩ…ЩҲШІШҙ ШҙШ§Щ…Щ„ Щ…ШЁШ§ШӯШ«ЫҢ Щ…Ш§ЩҶЩҶШҜ ЩҶШӯЩҲЩҮ Ъ©Ш§Шұ Ъ©Шҙ ШҜЩҲ ШіШ·ШӯЫҢШҢ Щ…ШӯШҜЩҲШҜЫҢШӘвҖҢЩҮШ§ЫҢ TTLЩҮШ§ШҢ ЩҫЫҢШ§ШҜЩҮвҖҢШіШ§ШІЫҢ ШЁЪ©вҖҢШ§ЩҶШҜ Redis Pub/Sub ЩҲ Ш§Ш№ШӘШЁШ§ШұШіЩҶШ¬ЫҢ ШЁШ§ Ш§ШіШӘЩҒШ§ШҜЩҮ Ш§ШІ Щ„Ш§ЪҜвҖҢЩҮШ§ ЩҲ ШұШҜЫҢШ§ШЁЫҢвҖҢЩҮШ§ Ш§ШіШӘ.
