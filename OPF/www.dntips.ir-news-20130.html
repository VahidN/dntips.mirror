→ вҖ«Let's Encrypt ЪҜЩҲШ§ЩҮЫҢЩҶШ§Щ…ЩҮвҖҢЩҮШ§ЫҢ 6 ШұЩҲШІЩҮ Ш§ШұШ§ШҰЩҮ Щ…ЫҢвҖҢШҜЩҮШҜ!
вҖ«Ш§ШіШӘЩҒШ§ШҜЩҮ Ш§ШІ ШұЩ…ШІЪҜШ°Ш§ШұЫҢвҖҢЩҮШ§ЫҢ Щ…ШӘЩҶЫҢ ЩӮШҜЫҢЩ…ЫҢ ШҜШұ ШҜШ§ШӘвҖҢЩҶШӘвҖҢЪ©ЩҲШұ: Ш®Щ„Ш§ШөЩҮ Ш¬Ш§Щ…Ш№ ←

ШӘШ§ШұЫҢШ®: ЫұЫҙЫ°Ыі/ЫұЫІ/ЫұЫ¶ Ы°Ыё:ЫұЫ¶

Ш§ЫҢЩҶ Щ…ЩӮШ§Щ„ЩҮ ШЁЩҮ ШЁШұШұШіЫҢ Щ…ШұШӘШЁвҖҢШіШ§ШІЫҢ Ш№ШҜШҜЫҢ (Numeric Sorting) ЫҢШ§ Щ…ШұШӘШЁвҖҢШіШ§ШІЫҢ Ш·ШЁЫҢШ№ЫҢ (Natural Sorting) ШҜШұ ШҜШ§ШӘвҖҢЩҶШӘ Щ…ЫҢвҖҢЩҫШұШҜШ§ШІШҜ. Щ…ШұШӘШЁвҖҢШіШ§ШІЫҢ Ш№ШҜШҜЫҢШҢ ШұШҙШӘЩҮвҖҢЩҮШ§ЫҢЫҢ ШұШ§ Ъ©ЩҮ ШҙШ§Щ…Щ„ Ш§Ш№ШҜШ§ШҜ ЩҮШіШӘЩҶШҜШҢ ШЁЩҮ ШӘШұШӘЫҢШЁЫҢ Щ…ЩҶШ·ЩӮЫҢ ЩҲ ЩӮШ§ШЁЩ„ ЩҒЩҮЩ… ШЁШұШ§ЫҢ Ш§ЩҶШіШ§ЩҶ Щ…ШұШӘШЁ Щ…ЫҢвҖҢЪ©ЩҶШҜ. ШЁЩҮ Ш№ЩҶЩҲШ§ЩҶ Щ…Ш«Ш§Щ„ШҢ ШҜШұ Щ…ШұШӘШЁвҖҢШіШ§ШІЫҢ ЩҫЫҢШҙвҖҢЩҒШұШ¶ШҢ ШұШҙШӘЩҮ "v10" ЩӮШЁЩ„ Ш§ШІ "v3" ЩӮШұШ§Шұ Щ…ЫҢвҖҢЪҜЫҢШұШҜШҢ ШІЫҢШұШ§ Щ…ЩӮШ§ЫҢШіЩҮ Ъ©Ш§ШұШ§Ъ©ШӘШұ ШЁЩҮ Ъ©Ш§ШұШ§Ъ©ШӘШұ Ш§ЩҶШ¬Ш§Щ… Щ…ЫҢвҖҢШҙЩҲШҜ. Ш§Щ…Ш§ ШҜШұ Щ…ШұШӘШЁвҖҢШіШ§ШІЫҢ Ш№ШҜШҜЫҢШҢ "v3" ЩӮШЁЩ„ Ш§ШІ "v10" ЩӮШұШ§Шұ Щ…ЫҢвҖҢЪҜЫҢШұШҜШҢ ШІЫҢШұШ§ Ш№ШҜШҜ 3 Ш§ШІ 10 Ъ©ЩҲЪҶЪ©ШӘШұ Ш§ШіШӘ. ШҜШұ Ш§ШҜШ§Щ…ЩҮШҢ ШіЩҮ ШұЩҲШҙ ШЁШұШ§ЫҢ ЩҫЫҢШ§ШҜЩҮвҖҢШіШ§ШІЫҢ Щ…ШұШӘШЁвҖҢШіШ§ШІЫҢ Ш№ШҜШҜЫҢ ШҜШұ ШҜШ§ШӘвҖҢЩҶШӘ Щ…Ш№ШұЩҒЫҢ Щ…ЫҢвҖҢШҙЩҲЩҶШҜ:

1. Ш§ШіШӘЩҒШ§ШҜЩҮ Ш§ШІ ЩҫШұЪҶЩ… CompareOptions.NumericOrdering ШҜШұ ШҜШ§ШӘвҖҢЩҶШӘ 10

Ш§ЪҜШұ Ш§ШІ ШҜШ§ШӘвҖҢЩҶШӘ 10 (ЩҫШі Ш§ШІ Ш§ЩҲШ§ШіШ· ЩҶЩҲШ§Щ…ШЁШұ 2025) Ш§ШіШӘЩҒШ§ШҜЩҮ Щ…ЫҢвҖҢЪ©ЩҶЫҢШҜШҢ Щ…ЫҢвҖҢШӘЩҲШ§ЩҶЫҢШҜ Ш§ШІ ЩҫШұЪҶЩ… Ш¬ШҜЫҢШҜ CompareOptions.NumericOrdering Ъ©ЩҮ ШҜШұ Ш§ЫҢЩҶ ЩҶШіШ®ЩҮ Щ…Ш№ШұЩҒЫҢ ШҙШҜЩҮ Ш§ШіШӘШҢ ШЁЩҮШұЩҮ ШЁШЁШұЫҢШҜ. Ш§ЫҢЩҶ ЩҫШұЪҶЩ… ШЁЩҮ ШҙЩ…Ш§ Ш§Щ…Ъ©Ш§ЩҶ Щ…ЫҢвҖҢШҜЩҮШҜ ШӘШ§ ЫҢЪ© Щ…ЩӮШ§ЫҢШіЩҮвҖҢЪҜШұ ШұШҙШӘЩҮ ШіЩҒШ§ШұШҙЫҢ (StringComparer) Ш§ЫҢШ¬Ш§ШҜ Ъ©ЩҶЫҢШҜ Ъ©ЩҮ Щ…ШұШӘШЁвҖҢШіШ§ШІЫҢ Ш№ШҜШҜЫҢ ШұШ§ Ш§ЩҶШ¬Ш§Щ… ШҜЩҮШҜ.
var comparer = StringComparer.Create(CultureInfo.InvariantCulture, CompareOptions.NumericOrdering);

string[] tags = ["v1.0.11", "v1.0.2", "v1.0.9"];
Array.Sort(tags, comparer);

Console.WriteLine(string.Join(", ", tags)); // prints v1.0.2, v1.0.9, v1.0.11
ЩҶЪ©ШӘЩҮ Щ…ЩҮЩ… Ш§ЫҢЩҶ Ш§ШіШӘ Ъ©ЩҮ ЫҢЪ© ЩҶЩ…ЩҲЩҶЩҮ ШӘЪ©ЫҢЩҶ Ш№Щ…ЩҲЩ…ЫҢ (Public Singleton) Ш§ШІ StringComparer ШЁШұШ§ЫҢ Щ…ШұШӘШЁвҖҢШіШ§ШІЫҢ Ш№ШҜШҜЫҢ ЩҲШ¬ЩҲШҜ ЩҶШҜШ§ШұШҜ. ШҜЩ„ЫҢЩ„ Ш§ЫҢЩҶ Ш§Щ…ШұШҢ Ш§Щ…Ъ©Ш§ЩҶ ШіЩҒШ§ШұШҙЫҢвҖҢШіШ§ШІЫҢ ШЁЫҢШҙШӘШұ ЩҶШӯЩҲЩҮ ШЁШұШ®ЩҲШұШҜ ШЁШ§ ШіШ§ЫҢШұ Ъ©Ш§ШұШ§Ъ©ШӘШұЩҮШ§ (Щ…Ш§ЩҶЩҶШҜ ШӯШіШ§ШіЫҢШӘ ШЁЩҮ ШӯШұЩҲЩҒ ШЁШІШұЪҜ ЩҲ Ъ©ЩҲЪҶЪ©ШҢ ЩҶШҙШ§ЩҶЩҮвҖҢЩҮШ§ЫҢ ЩҶЪҜШ§ШұШҙЫҢ ЩҲ ЩӮЩҲШ§ЩҶЫҢЩҶ Ш®Ш§Шө ЩҒШұЩҮЩҶЪҜЫҢ) Ш§ШіШӘ. ЩҮЩ…ЪҶЩҶЫҢЩҶШҢ ЩҫШұЪҶЩ… CompareOptions.NumericOrdering ШЁШ§ ЩҫШұЪҶЩ…вҖҢЩҮШ§ЫҢ CompareOptions.Ordinal ЩҲ CompareOptions.OrdinalIgnoreCase ШіШ§ШІЪҜШ§Шұ ЩҶЫҢШіШӘ.

2. Ш§ШіШӘЩҒШ§ШҜЩҮ Ш§ШІ ШЁШіШӘЩҮ NaturalSort.Extension

Ш§ЪҜШұ Ш§ШІ ЩҶШіШ®ЩҮвҖҢЩҮШ§ЫҢ ЩӮШҜЫҢЩ…ЫҢвҖҢШӘШұ ШҜШ§ШӘвҖҢЩҶШӘ Ш§ШіШӘЩҒШ§ШҜЩҮ Щ…ЫҢвҖҢЪ©ЩҶЫҢШҜ ЫҢШ§ ШҜШ§ШӘвҖҢЩҶШӘ 10 ЩҮЩҶЩҲШІ ШҜШұ ШҜШіШӘШұШі ЩҶЫҢШіШӘШҢ Щ…ЫҢвҖҢШӘЩҲШ§ЩҶЫҢШҜ Ш§ШІ ШЁШіШӘЩҮ NaturalSort.Extension Ш§ШіШӘЩҒШ§ШҜЩҮ Ъ©ЩҶЫҢШҜ. Ш§ЫҢЩҶ ШЁШіШӘЩҮШҢ ЫҢЪ© Щ…ЩӮШ§ЫҢШіЩҮвҖҢЪҜШұ Ш·ШЁЫҢШ№ЫҢ (NaturalSortComparer) Ш§ШұШ§ШҰЩҮ Щ…ЫҢвҖҢШҜЩҮШҜ Ъ©ЩҮ Щ…ЩӮШ§ЫҢШіЩҮвҖҢЪҜШұ ШҜЫҢЪҜШұЫҢ ШұШ§ ШҜШұ ШЁШұ Щ…ЫҢвҖҢЪҜЫҢШұШҜ ЩҲ ШЁЩҮ ШҙЩ…Ш§ Ш§Щ…Ъ©Ш§ЩҶ Щ…ЫҢвҖҢШҜЩҮШҜ ШӘШ§ Щ…ШұШӘШЁвҖҢШіШ§ШІЫҢ Ш№ШҜШҜЫҢ ШұШ§ ШЁШ§ ШіШ§ЫҢШұ ЪҜШІЫҢЩҶЩҮвҖҢЩҮШ§ЫҢ Щ…ЩӮШ§ЫҢШіЩҮ (Ш§ШІ Ш¬Щ…Щ„ЩҮ ШӘШұШӘЫҢШЁЫҢ) ШӘШұЪ©ЫҢШЁ Ъ©ЩҶЫҢШҜ.
var comparer = new NaturalSortComparer(StringComparer.OrdinalIgnoreCase);

string[] tags = ["v1.0.11", "v1.0.2", "v1.0.9"];
Array.Sort(tags, comparer);

Console.WriteLine(string.Join(", ", tags)); // also prints v1.0.2, v1.0.9, v1.0.11

3. Ш§ШіШӘЩҒШ§ШҜЩҮ Ш§ШІ ШұШ§ШЁШ· ШЁШұЩҶШ§Щ…ЩҮвҖҢЩҶЩҲЫҢШіЫҢ Ъ©Ш§ШұШЁШұШҜЫҢ ЩҲЫҢЩҶШҜЩҲШІ (Windows API)

Ш§ЪҜШұ ШЁШұЩҶШ§Щ…ЩҮ ШҙЩ…Ш§ ЩҒЩӮШ· ШЁШұШ§ЫҢ ЩҲЫҢЩҶШҜЩҲШІ Ш·ШұШ§ШӯЫҢ ШҙШҜЩҮ Ш§ШіШӘ ЩҲ Щ…ЫҢвҖҢШ®ЩҲШ§ЩҮЫҢШҜ ШұЩҒШӘШ§Шұ Щ…ШұШӘШЁвҖҢШіШ§ШІЫҢ ЩҲЫҢЩҶШҜЩҲШІ Ш§Ъ©ШіЩҫЩ„ЩҲШұШұ ШұШ§ ШҙШЁЫҢЩҮвҖҢШіШ§ШІЫҢ Ъ©ЩҶЫҢШҜШҢ Щ…ЫҢвҖҢШӘЩҲШ§ЩҶЫҢШҜ Ш§ШІ ШӘШ§ШЁШ№ StrCmpLogicalW Ш§ШІ Ш·ШұЫҢЩӮ ЩҒШұШ§Ш®ЩҲШ§ЩҶЫҢ ЩҫЩ„ШӘЩҒШұЩ… (P/Invoke) Ш§ШіШӘЩҒШ§ШҜЩҮ Ъ©ЩҶЫҢШҜ. Ш§ЫҢЩҶ ШӘШ§ШЁШ№ШҢ Щ…ЩӮШ§ЫҢШіЩҮ ШұШ§ ШЁЩҮ ШөЩҲШұШӘ ШәЫҢШұ ШӯШіШ§Ші ШЁЩҮ ШӯШұЩҲЩҒ ШЁШІШұЪҜ ЩҲ Ъ©ЩҲЪҶЪ© Ш§ЩҶШ¬Ш§Щ… Щ…ЫҢвҖҢШҜЩҮШҜ ЩҲ ШіШ·Шӯ ШіЩҒШ§ШұШҙЫҢвҖҢШіШ§ШІЫҢ Ъ©Щ…ШӘШұЫҢ ЩҶШіШЁШӘ ШЁЩҮ ШҜЩҲ ШұЩҲШҙ ЩӮШЁЩ„ЫҢ ШҜШ§ШұШҜ.
internal sealed class WindowsNumericStringComparer : IComparer<string?>
{
    public int Compare(string? x, string? y)
    {
        if (x == y) return 0;
        if (x == null) return -1;
        if (y == null) return 1;

        return NativeMethods.StrCmpLogical(x, y);
    }

    private static class NativeMethods
    {
        [DllImport("shlwapi.dll", EntryPoint = "StrCmpLogicalW", ExactSpelling = true, CharSet = CharSet.Unicode)]
        [DefaultDllImportSearchPaths(DllImportSearchPath.System32)]
        public static extern int StrCmpLogical(string psz1, string psz2);
    }
}

var comparer = new WindowsNumericStringComparer();

string[] tags = ["v1.0.11", "v1.0.2", "v1.0.9"];
Array.Sort(tags, comparer);

Console.WriteLine(string.Join(", ", tags));
ШЁШұШ§ЫҢ Ш§ШіШӘЩҒШ§ШҜЩҮ Ш§ШІ Ш§ЫҢЩҶ ШұЩҲШҙШҢ Щ…ЫҢвҖҢШӘЩҲШ§ЩҶЫҢШҜ Ш§ШІ ШЁШіШӘЩҮ Microsoft.Windows.CsWin32 ШЁШұШ§ЫҢ ШӘЩҲЩ„ЫҢШҜ Ш®ЩҲШҜЪ©Ш§Шұ Ш§Щ…Ш¶Ш§ЫҢ ЩҒШұШ§Ш®ЩҲШ§ЩҶЫҢ ЩҫЩ„ШӘЩҒШұЩ… ЫҢШ§ Ш§ШІ LibraryImportAttribute Ъ©ЩҮ Щ…ШЁШӘЩҶЫҢ ШЁШұ ШӘЩҲЩ„ЫҢШҜЪ©ЩҶЩҶШҜЩҮвҖҢЩҮШ§ЫҢ Щ…ЩҶШЁШ№ Ш§ШіШӘШҢ Ш§ШіШӘЩҒШ§ШҜЩҮ Ъ©ЩҶЫҢШҜ.

    Ш§ШҙШӘШұШ§Ъ©вҖҢЩҮШ§ЩҫЫҢШҙ ЩҶЩ…Ш§ЫҢШҙ 1.0 ШҜШ§ШӘ ЩҶШӘ 10: Щ…ШұШӘШЁ ШіШ§ШІЫҢ Ш№ШҜШҜЫҢ ШЁШұШ§ЫҢ Щ…ЩӮШ§ЫҢШіЩҮ ШұШҙШӘЩҮвҖҢЩҮШ§Ш§ШҙШӘШұШ§Ъ©вҖҢЩҮШ§В«Щ…ЩӮШ§ЫҢШіЩҮ Щ…ЩҶШ·ЩӮЫҢ ШұШҙШӘЩҮ ШЁШұШ§ЫҢ Щ…ШұШӘШЁвҖҢШіШ§ШІЫҢ ШұШҙШӘЩҮвҖҢЩҮШ§ЫҢ Ш№ШҜШҜЫҢВ» ШҜШұ ШҜШ§ШӘвҖҢЩҶШӘ 10Ш§ШҙШӘШұШ§Ъ©вҖҢЩҮШ§Ш§ШІ StringComparison.InvariantCulture Ш§ШіШӘЩҒШ§ШҜЩҮ ЩҶЪ©ЩҶЫҢШҜ!Ш§ШҙШӘШұШ§Ъ©вҖҢЩҮШ§ Ш¬ШіШӘШ¬ЩҲЫҢ ШЁШұШҜШ§ШұЫҢ (Vector Search) ЪҶЫҢШіШӘШҹЩ…ШіЫҢШұШұШ§ЩҮвҖҢЩҮШ§ШЁШұШұШіЫҢ ЩҲЫҢЪҳЪҜЫҢвҖҢвҖҢЩҮШ§ЫҢ Ш¬ШҜЫҢШҜ C# 14 ЩҲ NET 10.Щ…Ш·Ш§Щ„ШЁЩ…Ш№ШұЩҒЫҢ Collection Expression Arguments ШҜШұ C# 15.0Щ…Ш·Ш§Щ„ШЁЩҫЫҢШҜШ§ Ъ©ШұШҜЩҶ ШўЫҢШӘЩ…вҖҢЩҮШ§ЫҢ ШӘЪ©ШұШ§ШұЫҢ ШҜШұ ЫҢЪ© Щ„ЫҢШіШӘ ШЁЩҮ Ъ©Щ…Ъ© LINQЩ…Ш·Ш§Щ„ШЁЩ…ШұШӘШЁвҖҢШіШ§ШІЫҢШҢ ЩҒЫҢЩ„ШӘШұ Ъ©ШұШҜЩҶ ЩҲ ШөЩҒШӯЩҮвҖҢШЁЩҶШҜЫҢ Ш§Ш·Щ„Ш§Ш№Ш§ШӘ ШҜШұ ASP.NET CoreЩ…Ш·Ш§Щ„ШЁШЁШұШұШіЫҢ Щ…ЩҒШ§ЩҮЫҢЩ… Covariant ЩҲ Contravariant ШҜШұ ШІШЁШ§ЩҶ ШіЫҢвҖҢШҙШ§ШұЩҫЩҶШёШұШ§ШӘ Щ…Ш·Ш§Щ„ШЁШ·ШұШ§ШӯЫҢ Ш§ЩҒШІЩҲЩҶЩҮ ЩҫШ°ЫҢШұ ШЁШ§ ASP.NET MVC 4.x/5.x - ЩӮШіЩ…ШӘ ШіЩҲЩ…
