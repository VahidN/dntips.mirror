ШЁШ§ ШӘЩҶШёЫҢЩ…Ш§ШӘ ШҜЩ„Ш®ЩҲШ§ЩҮ

ШЁШ§ ШҜШіШӘШұШіЫҢ ШЁЩҮ HttpContext Щ…ЫҢвҖҢШӘЩҲШ§ЩҶ Ш№ЩҶЩҲШ§ЩҶ Щ…ШіШӘЩҶШҜШ§ШӘ ШұШ§ ШЁЩҮ ШөЩҲШұШӘ ШҜШ§ЫҢЩҶШ§Щ…ЫҢЪ© ШЁШұ Ш§ШіШ§Ші ЩҶЩӮШҙ Ъ©Ш§ШұШЁШұ ЫҢШ§ ЩҶШ§Щ… Ш§ЩҲ ШӘШәЫҢЫҢШұ ШҜШ§ШҜ.

Ыҙ. ЩҫЫҢЪ©ШұШЁЩҶШҜЫҢ ШёШ§ЩҮШұ ЩҲ ШұЩҒШӘШ§Шұ Scalar
app.MapScalarApiReference(options =>
{
    options.WithTitle("E-Commerce API")
           .WithClassicLayout()             
           .ForceDarkMode()                
           .HideSearch()                    
           .ShowOperationId()              
           .ExpandAllTags()                 
           .SortTagsAlphabetically()        
           .SortOperationsByMethod()      
           .PreserveSchemaPropertyOrder()  
           .WithProxy("https://api-gateway.company.com")
           .AddServer("https://api.company.com", "Production")
           .AddServer("https://staging-api.company.com", "Staging");
});
Ш§ЫҢЩҶ ШЁШ®Шҙ ШӘЩ…Ш§Щ… ЩҲЫҢЪҳЪҜЫҢвҖҢЩҮШ§ЫҢ ШёШ§ЩҮШұЫҢ ЩҲ ШұЩҒШӘШ§Шұ Scalar ШұШ§ ШЁЩҮ ШөЩҲШұШӘ Fluent API ШӘЩҶШёЫҢЩ… Щ…ЫҢвҖҢЪ©ЩҶШҜ:
  • WithTitle: ШӘШ№ЫҢЫҢЩҶ Ш№ЩҶЩҲШ§ЩҶ Щ…ШіШӘЩҶШҜШ§ШӘ Ъ©ЩҮ ШҜШұ ШЁШ§Щ„Ш§ЫҢ ШөЩҒШӯЩҮ ЩҲ Title Щ…ШұЩҲШұЪҜШұ ЩҶЩ…Ш§ЫҢШҙ ШҜШ§ШҜЩҮ Щ…ЫҢвҖҢШҙЩҲШҜ.
  • WithClassicLayout: ЩҒШ№Ш§Щ„вҖҢЪ©ШұШҜЩҶ ШёШ§ЩҮШұ Ъ©Щ„Ш§ШіЫҢЪ© Scalar Ъ©ЩҮ ШіШ§Ш®ШӘШ§ШұЫҢ Щ…ШҙШ§ШЁЩҮ Swagger UI ШҜШ§ШұШҜ ЩҲ Щ…ЩҶЩҲЫҢ ШіЩ…ШӘ ЪҶЩҫ ШұШ§ ЩҶЩ…Ш§ЫҢШҙ Щ…ЫҢвҖҢШҜЩҮШҜ.
  • ForceDarkMode: Ш§Ш¬ШЁШ§Шұ UI ШЁЩҮ ЩҶЩ…Ш§ЫҢШҙ ШҜШұ ШӯШ§Щ„ШӘ Dark Mode ШЁШҜЩҲЩҶ ШӘЩҲШ¬ЩҮ ШЁЩҮ ШӘЩҶШёЫҢЩ…Ш§ШӘ ШіЫҢШіШӘЩ… ЫҢШ§ Щ…ШұЩҲШұЪҜШұ.
  • HideSearch: Щ…Ш®ЩҒЫҢ Ъ©ШұШҜЩҶ ЩӮШ§ШЁЩ„ЫҢШӘ Ш¬ШіШӘШ¬ЩҲ ШҜШұ UI Щ…ШіШӘЩҶШҜШ§ШӘ ШЁШұШ§ЫҢ Ъ©Ш§ЩҮШҙ ШҙЩ„ЩҲШәЫҢ ЩҲ ШіШ§ШҜЩҮвҖҢШӘШұ ШҙШҜЩҶ ЩҶЩ…Ш§ЫҢШҙ.
  • ShowOperationId: ЩҶЩ…Ш§ЫҢШҙ Щ…ЩӮШҜШ§Шұ operationId ШЁШұШ§ЫҢ ЩҮШұ endpoint Ъ©ЩҮ ШЁШұШ§ЫҢ Code Generation ЩҲ Debugging ШЁШіЫҢШ§Шұ Ъ©Ш§ШұШЁШұШҜЫҢ Ш§ШіШӘ.
  • ExpandAllTags: ШЁШ§ШІ ШЁЩҲШҜЩҶ ШӘЩ…Ш§Щ… ЪҜШұЩҲЩҮвҖҢЩҮШ§ (Tags) ШЁЩҮ ШөЩҲШұШӘ ЩҫЫҢШҙвҖҢЩҒШұШ¶ ЩҮЩҶЪҜШ§Щ… Щ„ЩҲШҜ ШөЩҒШӯЩҮ ШӘШ§ ЩҮЩ…ЩҮ EndpointЩҮШ§ ШЁШҜЩҲЩҶ Ъ©Щ„ЫҢЪ©вҖҢЪ©ШұШҜЩҶ ЩӮШ§ШЁЩ„ Щ…ШҙШ§ЩҮШҜЩҮ ШЁШ§ШҙЩҶШҜ.
  • SortTagsAlphabetically: Щ…ШұШӘШЁвҖҢШіШ§ШІЫҢ ШҜШіШӘЩҮвҖҢШЁЩҶШҜЫҢвҖҢЩҮШ§ЫҢ API ШЁШұ Ш§ШіШ§Ші ШӯШұЩҲЩҒ Ш§Щ„ЩҒШЁШ§ Ш¬ЩҮШӘ Ш§ЩҒШІШ§ЫҢШҙ ЩҶШёЩ… ЩҲ ШҜШіШӘШұШіЫҢ ШЁЩҮШӘШұ.
  • SortOperationsByMethod: Щ…ШұШӘШЁвҖҢШіШ§ШІЫҢ Ш№Щ…Щ„ЫҢШ§ШӘ ШҜШ§Ш®Щ„ ЩҮШұ ШҜШіШӘЩҮвҖҢШЁЩҶШҜЫҢ ШЁШұ Ш§ШіШ§Ші HTTP Method Щ…Ш§ЩҶЩҶШҜ: GETШҢ POSTШҢ PUTШҢ DELETE ЩҲ ШәЫҢШұЩҮ.
  • PreserveSchemaPropertyOrder: ШӯЩҒШё ШӘШұШӘЫҢШЁ ЩҒЫҢЩ„ШҜЩҮШ§ЫҢ Schema ШҜЩӮЫҢЩӮШ§ЩӢ Щ…Ш·Ш§ШЁЩӮ ШӘШұШӘЫҢШЁ ШӘШ№ШұЫҢЩҒвҖҢШҙШҜЩҮ ШҜШұ Ъ©Щ„Ш§ШівҖҢЩҮШ§ЫҢ C# ЩҲ Ш¬Щ„ЩҲЪҜЫҢШұЫҢ Ш§ШІ ШӘШұШӘЫҢШЁвҖҢШҜЩҮЫҢ Ш®ЩҲШҜЪ©Ш§Шұ.
  • WithProxy: Ш§ШұШіШ§Щ„ ШҜШұШ®ЩҲШ§ШіШӘвҖҢЩҮШ§ЫҢ ШӘШіШӘЫҢ UI Ш§ШІ Ш·ШұЫҢЩӮ API Gateway ШЁШұШ§ЫҢ ШӯЩ„ Щ…ШҙЪ©Щ„ CORS ЩҲ ЫҢЪ©ЩҫШ§ШұЪҶЪҜЫҢ Ш§ШұШӘШЁШ§Ш·Ш§ШӘ.
  • AddServer: Ш§ЩҒШІЩҲШҜЩҶ Щ…ШӯЫҢШ· Production ШЁЩҮ Щ„ЫҢШіШӘ ШіШұЩҲШұЩҮШ§ ШӘШ§ Ъ©Ш§ШұШЁШұ ШЁШӘЩҲШ§ЩҶШҜ ШҜШұШ®ЩҲШ§ШіШӘ ШұШ§ Щ…ШіШӘЩӮЫҢЩ…Ш§ЩӢ ШЁЩҮ Ш§ЫҢЩҶ Щ…ШӯЫҢШ· Ш§ШұШіШ§Щ„ Ъ©ЩҶШҜ ЫҢШ§ Ш§ЩҒШІЩҲШҜЩҶ Щ…ШӯЫҢШ· Staging ШЁШұШ§ЫҢ ШӘШіШӘШҢ QA ЩҲ ШЁШұШұШіЫҢ ЩӮШЁЩ„ Ш§ШІ Ш§ЩҶШӘШҙШ§Шұ ЩҶЩҮШ§ЫҢЫҢ.
