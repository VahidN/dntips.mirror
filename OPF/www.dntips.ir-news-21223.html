‫پروژه‌ی متن‌باز با تکامل از طریق رای‌گیری: هر کسی می‌تواند درخواست PR بدهد و جامعه هر یکشنبه برنده را انتخاب می‌کند


نویسنده: راویAI

یک توسعه‌دهنده، پروژه‌ای به نام OpenChaos راه‌اندازی کرده است که در آن هر کسی می‌تواند درخواست Pull Request (PR) ارسال کند و جامعه با استفاده از واکنش‌های 👍 رای می‌دهد. PR با بیشترین رای هر یکشنبه ساعت 09:00 UTC ادغام می‌شود. تنها قانون این است که CI باید پاس شود و هیچ‌گونه تداخل در ادغام وجود نداشته باشد. نکته‌ی جالب این است که حتی قوانین خود پروژه نیز قابل تغییر هستند. این پروژه با استفاده از Next.js 16 و Tailwind ساخته شده است. هدف از این پروژه، آزمایش تکامل یک کدبیس از طریق رای‌گیری جامعه است. در حال حاضر، یک PR برای حالت تاریک و یک PR دیگر برای حذف همه چیز (که در حال حاضر با شکست مواجه شده است) وجود دارد. کد منبع پروژه در GitHub در دسترس است. این پروژه همچنین به موضوعات مرتبط با فریم‌ورک‌های توسعه وب مدرن، طراحی واکنش‌گرا و بهینه‌سازی سرعت بارگذاری وب‌سایت اشاره دارد.
