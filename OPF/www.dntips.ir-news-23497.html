‫الگوهای مفید برای ساخت ابزارهای HTML با استفاده از LLMها


نویسنده: راویAI
تاریخ: ۱۴۰۴/۱۱/۱۴ ۱۸:۲۲
آدرس: www.dntips.ir
در این مقاله، سایمون ویلیسون مجموعه‌ای از الگوهای مفید را برای ساخت ابزارهای HTML با استفاده از مدل‌های زبانی بزرگ (LLM) ارائه می‌دهد. این ابزارها که او نام "ابزارهای HTML" را برای آن‌ها انتخاب کرده، ترکیبی از HTML، جاوااسکریپت و CSS در یک فایل واحد هستند و برای ارائه قابلیت‌های مفید طراحی شده‌اند. ویلیسون در دو سال گذشته بیش از 150 ابزار از این نوع را با کمک LLMها ساخته است. این ابزارها شامل نمونه‌هایی مانند svg-render که کد SVG را به JPEG یا PNG قابل دانلود تبدیل می‌کند، pypi-changelog که امکان تولید و کپی کردن diff بین نسخه‌های مختلف بسته‌های PyPI را فراهم می‌کند و bluesky-thread که نمای درختی از یک رشته بحث در Bluesky ارائه می‌دهد، هستند. این ابزارها به طور منظم مورد استفاده قرار می‌گیرند و مجموعه کاملی از آن‌ها در وب‌سایت tools.simonwillison.net قابل مشاهده است. ویلیسون بر اهمیت استفاده از یک فایل واحد برای کدها، اجتناب از React (به دلیل نیاز به فرآیند ساخت)، بارگیری وابستگی‌ها از CDNها، و حفظ کدها در اندازه کوچک (چند صد خط) تاکید می‌کند. او همچنین پیشنهاد می‌کند که از Artifacts یا Canvas در ChatGPT، Claude یا Gemini برای نمونه‌سازی اولیه استفاده شود و همیشه عبارت "No React" را در دستورات خود قرار دهید تا از استفاده از React جلوگیری شود. در نهایت، ویلیسون توصیه می‌کند که کدها را با استفاده از Pyodide (برای اجرای کد پایتون در مرورگر) یا WebAssembly گسترش دهید و همیشه prompt و transcript مربوط به هر ابزار را ثبت کنید. این الگوها به توسعه‌دهندگان کمک می‌کنند تا به سرعت و به آسانی ابزارهای HTML کاربردی و قابل استفاده بسازند.
