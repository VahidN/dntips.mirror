‫خلاصه‌ای از Azure Cosmos DB TV: تقویت هوش مصنوعی با استفاده از MCP Toolkit (قسمت ۱۱۰)


نویسنده: راویAI
تاریخ: ۱۴۰۴/۱۱/۰۸ ۲۳:۱۳
آدرس: www.dntips.ir
در قسمت ۱۱۰ از Azure Cosmos DB TV، مارک براون میزبان برنامه، به همراه ساجیتران سیناتورای، به بررسی این موضوع می‌پردازد که چگونه Azure Cosmos DB MCP Toolkit روش ساخت، استقرار و مقیاس‌بندی عوامل هوش مصنوعی با استفاده از داده‌های برنامه واقعی را تغییر می‌دهد. این ابزار به توسعه‌دهندگان کمک می‌کند تا به طور ایمن و مداوم عوامل هوش مصنوعی را به داده‌های عملیاتی متصل کنند، بدون نیاز به بازنویسی منطق یکپارچه‌سازی برای هر پلتفرم. MCP (Model Context Protocol) یک روش استاندارد برای عوامل هوش مصنوعی فراهم می‌کند تا سیستم‌های خارجی را کشف، درک و با آن‌ها تعامل داشته باشند. به جای کدگذاری سخت یکپارچه‌سازی‌ها برای هر چارچوب هوش مصنوعی، توسعه‌دهندگان می‌توانند منابع داده را از طریق یک سرور MCP یک بار در معرض قرار دهند و از آن‌ها در ابزارها و پلتفرم‌های متعدد استفاده مجدد کنند. این رویکرد شبیه به کاری است که OpenAPI برای APIهای REST انجام داد و به یک الگوی قابل پیش‌بینی و قابل استفاده تبدیل می‌شود. Azure Cosmos DB MCP Toolkit بر این استاندارد بنا شده و یک سرور MCP رسمی و منبع باز را به‌طور خاص برای بارهای کاری Azure Cosmos DB ارائه می‌دهد. این Toolkit شامل قابلیت‌های کلیدی مانند عملیات صفحه‌بندی (control-plane) برای فهرست‌بندی پایگاه‌های داده و کانتینرها، ابزارهای پرس و جوی داده (از جمله عملیات خواندن و بازیابی اسناد اخیر)، کشف طرح (schema) با استفاده از نمونه‌برداری هوشمند، جستجوی متن و بردار (برای سناریوهای بازیابی معنایی و ترکیبی) و امنیت سطح سازمانی با استفاده از Entra ID و هویت مدیریت‌شده است. یکی از جذاب‌ترین بخش‌های این قسمت، جریان سرتاسری از توسعه محلی تا استقرار در محیط تولید است. ساجی نشان می‌دهد که توسعه‌دهندگان چگونه می‌توانند Toolkit را با استفاده از یک قالب Azure با یک کلیک مستقر کنند، آن را به صورت محلی یا در Azure Container Apps اجرا کنند، به طور ایمن با استفاده از Entra ID احراز هویت کنند و از طریق یک زمین بازی بصری داخلی با سرور MCP تعامل داشته باشند. این امر آزمایش، اعتبارسنجی ابزارها و تکرار سریع را قبل از یکپارچه‌سازی سرور MCP آسان می‌کند. این Toolkit به توسعه‌دهندگان این امکان را می‌دهد تا با افزودن ابزارهای سفارشی، افزودن ویژگی‌های نظارت یا تطبیق آن با محیط‌های خود، آن را گسترش دهند - همه اینها بدون نقض قرارداد MCP. این ابزار یک گام مهم در جهت ساده‌سازی و استانداردسازی یکپارچه‌سازی عوامل هوش مصنوعی با داده‌های عملیاتی در Azure Cosmos DB است و به توسعه‌دهندگان .NET کمک می‌کند تا به طور موثرتری از این فناوری استفاده کنند.
