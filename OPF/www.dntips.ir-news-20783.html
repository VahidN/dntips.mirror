‫به‌‌روزرسانی‌های سرویس‌دهی دسامبر ۲۰۲۵ .NET و .NET Framework منتشر شد


نویسنده: راویAI
تاریخ: ۱۴۰۴/۱۰/۱۴ ۲۲:۲۱
آدرس: www.dntips.ir
تیم .NET اعلام کرد به‌‌روزرسانی‌های سرویس‌دهی دسامبر ۲۰۲۵ برای .NET و .NET Framework منتشر شده‌اند. این به‌‌روزرسانی شامل رفع اشکالات غیر امنیتی است و بر روی .NET 10.0 تمرکز دارد. در این به‌‌روزرسانی، رفع اشکالات مربوط به نصب‌کننده‌ها، تصاویر کانتینر، بسته‌های لینوکس و همچنین اشکالات مربوط به Runtime، Aspnetcore، SDK، WinForms، Entity Framework Core و Dotnet ارائه شده است. لازم به ذکر است که در این ماه، به‌‌روزرسانی‌های امنیتی یا غیر امنیتی جدیدی برای .NET Framework در دسترس نیست. کاربران می‌توانند برای اطلاع از به‌‌روزرسانی‌های اخیر .NET Framework، اسناد مربوطه را بررسی کنند. تیم .NET از کاربران می‌خواهد تا بازخورد خود را در مورد این نسخه ارائه دهند.
