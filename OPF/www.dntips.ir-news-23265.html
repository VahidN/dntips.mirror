‫به‌‌روزرسانی Visual Studio Code CMake Tools 1.22: نشانه‌گذاری هدف و خروجی CTest بهبودیافته


نویسنده: راویAI
تاریخ: ۱۴۰۴/۱۱/۱۰ ۰۲:۴۳
آدرس: www.dntips.ir
نسخه 1.22 از افزونه CMake Tools برای Visual Studio Code منتشر شده است که مجموعه‌ای از ویژگی‌های جدید را ارائه می‌دهد. این به‌روزرسانی شامل به‌روزرسانی‌های نمای کلی پروژه برای فیلتر کردن و نشانه‌گذاری اهداف CMake در پروژه‌های بزرگ CMake و پشتیبانی گسترده‌تر از CTest برای سفارشی‌سازی خروجی است. این نسخه شامل مشارکت‌هایی از جامعه منبع باز ما است. در این به‌روزرسانی، امکان افزودن نشانه‌گذاری و فیلتر کردن نمای کلی پروژه فراهم شده است که مدیریت اهداف CMake را در پروژه‌های بزرگ آسان‌تر می‌کند. با استفاده از فیلترها، می‌توانید به سرعت اهداف خاص را در پروژه‌های پیچیده پیدا کنید. همچنین، CMake Tools اکنون از نشانه‌گذاری اهداف پرکاربرد پشتیبانی می‌کند تا آنها در یک بخش جداگانه در نوار کناری CMake نمایش داده شوند. این امر دسترسی سریع به اهدافی که اغلب ساخت، اشکال‌زدایی یا اجرا می‌کنید را بدون نیاز به جستجوی مکرر در سلسله‌مراتب کامل پروژه فراهم می‌کند. برای نشانه‌گذاری یک هدف، کافی است در نمای کلی پروژه به آن هدف مورد نظر بروید و گزینه "Toggle Bookmark" را انتخاب کنید. علاوه بر این، این نسخه تجربه CTest را با افزودن پشتیبانی از الگوهای خرابی قابل تنظیم بهبود می‌بخشد. این امکان به توسعه‌دهندگان کمک می‌کند تا خروجی CTest را به طور موثرتری مدیریت و اشکال‌زدایی کنند.
