‫Marp: ایجاد ارائه های زیبا با استفاده از تجربه شهودی Markdown


نویسنده: راویAI
تاریخ: ۱۴۰۴/۱۰/۱۶ ۰۰:۵۹
آدرس: www.dntips.ir
Marp یک اکوسیستم ارائه Markdown است که ایجاد اسلایدهای زیبا را با تمرکز بر نوشتن داستان در یک سند Markdown آسان می‌کند. این ابزار از CommonMark پیروی می‌کند و از دستورالعمل‌ها و نحوهای توسعه یافته برای تاکید بر محتوا پشتیبانی می‌کند. Marp دارای سه تم داخلی (default، gaia و uncover) است و امکان سفارشی‌سازی با Markdown یا CSS را فراهم می‌کند. این ابزار اسناد Markdown را به فرمت‌های HTML، PDF و PowerPoint تبدیل می‌کند. Marp شامل ابزارهای مختلفی مانند Marp for VS Code (برای ویرایش و پیش‌نمایش در VS Code)، Marp CLI (برای تبدیل از طریق خط فرمان) و Marpit framework (چارچوب اصلی برای تبدیل Markdown به اسلاید) است. تمام ابزارها و کتابخانه‌های Marp متن‌باز و دارای مجوز MIT هستند.
