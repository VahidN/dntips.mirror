‫معرفی قابلیت‌های SQL Server 2025 در رویداد مایکروسافت بیلد ۲۰۲۵


نویسنده: وحید نصیری
تاریخ: ۱۴۰۴/۰۳/۰۳ ۰۸:۲۰
آدرس: www.dntips.ir
SQL Server 2025 قابلیت‌های مهمی را با هدف بازتعریف تجربه توسعه‌دهنده پایگاه داده معرفی می‌کند.

ویژگی‌ها و قابلیت‌های جدید:

هوش مصنوعی و مدیریت داده:
  • جستجوی برداری با مدل‌های هوش مصنوعی.
  • پشتیبانی پیشرفته برای استفاده از مدل‌ها در SQL Server.

بهبودهای T-SQL:
  • نوع جدید JSON و قابلیت ایندکس‌گذاری آن.
  • پشتیبانی از عبارات منظم (RegEx).
  • سایر بهبودهای T-SQL.

جریان داده و API ها:
  • جریان‌سازی رویداد تغییر (CES).
  • پشتیبانی از REST API به صورت توکار.

انعطاف‌پذیری در استقرار: گزینه‌های استقرار بر روی ویندوز، لینوکس، کانتینرها، کوبرنتس یا Azure SQL VM.

بهره‌وری توسعه‌دهنده: دستیاری Copilot از طریق SQL Server Management Studio (SSMS) و GitHub.
ویژگی‌های Microsoft Fabric Mirroring.
