Щ„Ш§ЫҢЩҮ API Ш¬Ш§ЫҢЫҢ Ш§ШіШӘ Ъ©ЩҮ ШҜШұШ®ЩҲШ§ШіШӘвҖҢЩҮШ§ЫҢ HTTP ШЁШ§ ШЁШұЩҶШ§Щ…ЩҮ ШҙЩ…Ш§ Щ…Щ„Ш§ЩӮШ§ШӘ Щ…ЫҢвҖҢЪ©ЩҶЩҶШҜ ЩҲ ЩҲШёЫҢЩҒЩҮ ШўЩҶ ШӘШұШ¬Щ…ЩҮ ШЁЫҢЩҶ HTTP ЩҲ ШҜШіШӘЩҲШұШ§ШӘ/ЩҫШұШіШҙвҖҢЩҮШ§ Ш§ШіШӘ. Ш§ЪҜШұ Ъ©ЩҶШӘШұЩ„ШұЩҮШ§ЫҢ ШҙЩ…Ш§ ШӯШ§ЩҲЫҢ Щ…ЩҶШ·ЩӮ ШӘШ¬Ш§ШұЫҢ ШЁШ§ШҙЩҶШҜШҢ ШЁЩҮ Ш§ШӯШӘЩ…Ш§Щ„ ШІЫҢШ§ШҜ Щ…ШҙЪ©Щ„ЫҢ ШҜШұ Щ„Ш§ЫҢЩҮвҖҢЩҮШ§ЫҢ ШЁШ§Щ„Ш§ШӘШұ ЩҲШ¬ЩҲШҜ ШҜШ§ШұШҜ. ЩҲШёШ§ЫҢЩҒ Ш§ШөЩ„ЫҢ Щ„Ш§ЫҢЩҮ API ШҙШ§Щ…Щ„ ШҜШұЫҢШ§ЩҒШӘ ШҜШұШ®ЩҲШ§ШіШӘвҖҢЩҮШ§ЫҢ HTTPШҢ ШӘШ¬ШІЫҢЩҮ JSONШҢ Ш§Ш№ШӘШЁШ§ШұШіЩҶШ¬ЫҢ Щ…ШҜЩ„вҖҢЩҮШ§ШҢ ШӘШұШ¬Щ…ЩҮ ШЁЩҮ ШҜШіШӘЩҲШұШ§ШӘ/ЩҫШұШіШҙвҖҢЩҮШ§ (ШЁШ§ Ш§ШіШӘЩҒШ§ШҜЩҮ Ш§ШІ MediatR) ЩҲ ШЁШұЪҜШұШҜШ§ЩҶШҜЩҶ ЩҫШ§ШіШ®вҖҢЩҮШ§ЫҢ HTTP (ШҙШ§Щ…Щ„ Ъ©ШҜЩҮШ§ЫҢ ЩҲШ¶Ш№ЫҢШӘ ЩҲ ШЁШҜЩҶЩҮ JSON) Ш§ШіШӘ. Ш§ЫҢЩҶ Щ„Ш§ЫҢЩҮ ЩҶШЁШ§ЫҢШҜ ШҙШ§Щ…Щ„ Щ…ЩҶШ·ЩӮ ШӘШ¬Ш§ШұЫҢШҢ ШҜШіШӘШұШіЫҢ ШЁЩҮ ШҜШ§ШҜЩҮвҖҢЩҮШ§ ЫҢШ§ ШӘШөЩ…ЫҢЩ…Ш§ШӘ ЩҫЫҢЪҶЫҢШҜЩҮ Щ…ШұШЁЩҲШ· ШЁЩҮ Щ…Ш¬ЩҲШІ ШЁШ§ШҙШҜ. ШҜШұ Ш§ЫҢЩҶ Щ…ЩӮШ§Щ„ЩҮШҢ ШӘЩҶШёЫҢЩ… ЩҫШұЩҲЪҳЩҮ ШЁШ§ Ш§ШіШӘЩҒШ§ШҜЩҮ Ш§ШІ Program.cs ЩҲ ШіШұЩҲЫҢШівҖҢЩҮШ§ЫҢ Щ…ЩҲШұШҜ ЩҶЫҢШ§ШІ Щ…Ш§ЩҶЩҶШҜ FluentValidationШҢ MediatR ЩҲ Serilog ЩҶШҙШ§ЩҶ ШҜШ§ШҜЩҮ ШҙШҜЩҮ Ш§ШіШӘ. ЩҮЩ…ЪҶЩҶЫҢЩҶШҢ ЩҶШӯЩҲЩҮ Ш§ЩҒШІЩҲШҜЩҶ Ъ©ЩҶШӘШұЩ„ШұЩҮШ§ЫҢ APIШҢ Ъ©Ш§ЩҲШҙЪҜШұ API ЩҲ Swagger ШЁШұШ§ЫҢ Щ…ШіШӘЩҶШҜШіШ§ШІЫҢ API ШӘЩҲШ¶ЫҢШӯ ШҜШ§ШҜЩҮ ШҙШҜЩҮ Ш§ШіШӘ. ШҜШұ ЩҶЩҮШ§ЫҢШӘШҢ Щ…ЩӮШ§Щ„ЩҮ ШЁШұ Ш§ЩҮЩ…ЫҢШӘ ШӘШ№ШұЫҢЩҒ ЩӮШұШ§ШұШҜШ§ШҜЩҮШ§ЫҢ ШҜШұШ®ЩҲШ§ШіШӘ/ЩҫШ§ШіШ® API-specific ШЁЩҮ Ш¬Ш§ЫҢ Ш§ШіШӘЩҒШ§ШҜЩҮ Щ…ШіШӘЩӮЫҢЩ… Ш§ШІ Щ…ЩҲШ¬ЩҲШҜЫҢШӘвҖҢЩҮШ§ЫҢ ШҜШ§Щ…ЩҶЩҮ ЫҢШ§ DTOЩҮШ§ЫҢ ШЁШұЩҶШ§Щ…ЩҮ ШӘШ§Ъ©ЫҢШҜ Щ…ЫҢвҖҢЪ©ЩҶШҜ. ЩҶЩ…ЩҲЩҶЩҮвҖҢШ§ЫҢ Ш§ШІ ЩӮШұШ§ШұШҜШ§ШҜ ШҜШұШ®ЩҲШ§ШіШӘ CreatePromptRequest ШЁШұШ§ЫҢ Ш§ЫҢШ¬Ш§ШҜ ЫҢЪ© Ш§Ш№Щ„Ш§ЩҶ Ш§ШұШ§ШҰЩҮ ШҙШҜЩҮ Ш§ШіШӘ Ъ©ЩҮ ЩҶШҙШ§ЩҶвҖҢШҜЩҮЩҶШҜЩҮ Ш¬ШҜШ§ШіШ§ШІЫҢ Щ„Ш§ЫҢЩҮвҖҢЩҮШ§ ЩҲ Ъ©Ш§ЩҮШҙ ЩҲШ§ШЁШіШӘЪҜЫҢвҖҢЩҮШ§ Ш§ШіШӘ.
