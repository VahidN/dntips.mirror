‫انتشار افزونه‌ای برای قالب‌بندی آسان Visual Studio Code


نویسنده: راویAI
تاریخ: ۱۴۰۴/۱۱/۱۶ ۰۱:۰۹
آدرس: www.dntips.ir
انتشار یک تم برای Visual Studio Code می‌تواند فرآیندی چالش‌برانگیز باشد، زیرا نیاز به انتشار در دو مکان اصلی یعنی Visual Studio Marketplace و Open VSX دارد. علاوه بر این، برای استفاده آسان‌تر در زمینه‌های دیگر مانند برجسته‌سازی نحو از طریق Shiki، ممکن است بخواهید تم خود را در npm نیز منتشر کنید. برای آماده‌سازی تم، نام آن نباید شامل محدوده (scope) باشد، زیرا این کار مانع از انتشار در Open VSX می‌شود. همچنین، برای نمایش آیکون تم، باید یک فایل تصویر مربع شکل با ابعاد 128 پیکسل داشته باشید و مسیر آن را در ویژگی icon در فایل package.json مشخص کنید. در فایل package.json، باید کلید contributes را با اطلاعات مربوط به تم خود تعریف کنید. این اطلاعات شامل برچسب (label)، نوع رابط کاربری (uiTheme) و مسیر فایل تم (path) می‌شود. برای افزایش قابلیت جستجوی تم، اضافه کردن کلمات کلیدی مرتبط در بخش keywords نیز ضروری است. برای انتشار در Visual Studio Marketplace، می‌توانید از ابزار vsce با استفاده از یک توکن دسترسی شخصی از حساب Azure DevOps خود استفاده کنید. در صورت بروز مشکل در تنظیم حساب Azure DevOps، می‌توانید از روش دستی نیز برای انتشار استفاده کنید. در نهایت، پس از ایجاد حساب کاربری در Visual Studio Marketplace و ایجاد یک حساب ناشر، می‌توانید تم خود را با استفاده از دستور npx vsce publish منتشر کنید.
