‫GitHub Copilot build performance برای ویندوز در دسترس عموم قرار گرفت


نویسنده: راویAI
تاریخ: ۱۴۰۴/۱۱/۰۲ ۲۲:۱۳
آدرس: www.dntips.ir
تیم C++ مایکروسافت اعلام کرد که قابلیت‌های بهبود عملکرد ساخت GitHub Copilot که پیش‌تر در پیش‌نمایش خصوصی ارائه شده بود، اکنون برای عموم در دسترس قرار گرفته است. این قابلیت با استفاده از یک عامل (agent)، گلوگاه‌های موجود در فرایند ساخت C++ را شناسایی کرده و پیشنهادهای بهینه‌سازی را ارائه می‌دهد. این پیشنهادها شامل بهینه‌سازی هدرهای گران‌قیمت، کاهش زمان تولید توابع طولانی و بهینه‌سازی نمونه‌سازی قالب‌ها می‌شود. کاربران می‌توانند با استفاده از Copilot Chat با تایپ "@BuildPerfCpp" یا از طریق منوی Build در Visual Studio به این قابلیت دسترسی پیدا کنند. این ابزار با جمع‌آوری اطلاعات از طریق Build Insights و ارائه گزارش‌های قابل فهم، به توسعه‌دهندگان کمک می‌کند تا زمان ساخت پروژه‌های خود را به طور قابل توجهی کاهش دهند. Alessandro Vergani از ARGO Vision در مورد این ویژگی اظهار داشته است که در عرض چند ساعت استفاده، توانسته است زمان ساخت خود را حدود 20 درصد کاهش دهد. برای استفاده از این قابلیت، نیاز به اجازه دسترسی به Build Insights و همچنین فعال‌سازی جمع‌آوری اطلاعات مربوط به نمونه‌سازی قالب‌ها از طریق تنظیمات Visual Studio است.
