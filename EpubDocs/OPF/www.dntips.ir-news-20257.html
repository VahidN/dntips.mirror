Щ…ЩӮШҜЩ…ЩҮ
ЫҢЪ©ЫҢ Ш§ШІ ЪҶШ§Щ„ШҙвҖҢЩҮШ§ЫҢ ШұШ§ЫҢШ¬ ШҜШұ Щ…ШіЫҢШұ ШӘЩҲШіШ№ЩҮвҖҢЫҢ ШЁШұЩҶШ§Щ…ЩҮвҖҢЩҮШ§ЫҢ BlazorШҢ Щ…ШҜЫҢШұЫҢШӘ ШұЩҶШҜШұ (Rendering) Ъ©Ш§Щ…ЩҫЩҲЩҶЩҶШӘвҖҢЩҮШ§ Ш§ШіШӘ. ЪҜШ§ЩҮЫҢ Ш§ЩҲЩӮШ§ШӘШҢ ЫҢЪ© Ъ©Ш§Щ…ЩҫЩҲЩҶЩҶШӘ ШЁЫҢШҙ Ш§ШІ ШӯШҜ Ш§ЩҶШӘШёШ§Шұ ШұЩҶШҜШұ Щ…ЫҢвҖҢШҙЩҲШҜ Ъ©ЩҮ Щ…ЫҢвҖҢШӘЩҲШ§ЩҶШҜ Ш№Щ…Щ„Ъ©ШұШҜ ШЁШұЩҶШ§Щ…ЩҮ ШұШ§ ШӘШӯШӘ ШӘШЈШ«ЫҢШұ ЩӮШұШ§Шұ ШҜЩҮШҜ ЩҲ ШӘШ¬ШұШЁЩҮ Ъ©Ш§ШұШЁШұЫҢ ШұШ§ ШӘШ¶Ш№ЫҢЩҒ Ъ©ЩҶШҜ. Ш§ЫҢЩҶ Щ…ЩӮШ§Щ„ЩҮ ШЁШ§ ЩҮШҜЩҒ ШўЩ…ЩҲШІШҙ ШЁЩҮЫҢЩҶЩҮвҖҢШіШ§ШІЫҢ ШұЩҶШҜШұ ШҜШұ Blazor ЩҶЩҲШҙШӘЩҮ ШҙШҜЩҮ Ш§ШіШӘ ЩҲ ШЁШұ Ш§ШіШ§Ші ШӘШ¬ШұШЁЩҮвҖҢШ§ЫҢ ЩҲШ§ЩӮШ№ЫҢ Ш§ШІ ШӘШӯЩ„ЫҢЩ„ ЩҲ ШұЩҒШ№ Щ…ШҙЪ©Щ„ ШұЩҶШҜШұ ШәЫҢШұШ¶ШұЩҲШұЫҢ ЫҢЪ© Ъ©Ш§Щ…ЩҫЩҲЩҶЩҶШӘШҢ ШЁЩҮ ШҙЩ…Ш§ Ъ©Щ…Ъ© Щ…ЫҢвҖҢЪ©ЩҶШҜ ШӘШ§ ШҜШұЪ© ШЁЩҮШӘШұЫҢ Ш§ШІ ШұЩҒШӘШ§Шұ Blazor ШҜШ§ШҙШӘЩҮ ШЁШ§ШҙЫҢШҜ ЩҲ ШұШ§ЩҮЪ©Ш§ШұЩҮШ§ЫҢЫҢ Ш№Щ…Щ„ЫҢ ШЁШұШ§ЫҢ ШЁЩҮШЁЩҲШҜ ШўЩҶ ЩҫЫҢШ§ШҜЩҮвҖҢШіШ§ШІЫҢ Ъ©ЩҶЫҢШҜ. ШҜШұ Ш§ЫҢЩҶ Щ…ШіЫҢШұШҢ Щ…ЩҒШ§ЩҮЫҢЩ… Ъ©Щ„ЫҢШҜЫҢ Щ…Ш§ЩҶЩҶШҜ ЪҶШұШ®ЩҮ ШӯЫҢШ§ШӘ Ъ©Ш§Щ…ЩҫЩҲЩҶЩҶШӘ (Component Lifecycle)ШҢ Щ…ШӘШҜ ShouldRender ЩҲ Ш§ШіШӘЩҒШ§ШҜЩҮ Ш§ШІ Ш§ШЁШІШ§ШұЩҮШ§ЫҢ ШҜЫҢШЁШ§ЪҜ (Debugging Tools) ШЁШұШұШіЫҢ Щ…ЫҢвҖҢШҙЩҲЩҶШҜ.

ШҜШұЪ© Щ…ШҙЪ©Щ„ ШұЩҶШҜШұ ШЁЫҢШҙвҖҢШ§ШІШӯШҜ
ШҜШұ BlazorШҢ ШұЩҶШҜШұ ЫҢЪ© Ъ©Ш§Щ…ЩҫЩҲЩҶЩҶШӘ ШІЩ…Ш§ЩҶЫҢ ШұШ® Щ…ЫҢвҖҢШҜЩҮШҜ Ъ©ЩҮ ЩҒШұЫҢЩ…вҖҢЩҲШұЪ© ШӘШҙШ®ЫҢШө ШҜЩҮШҜ ЩҲШ¶Ш№ЫҢШӘ (State) ШўЩҶ ШӘШәЫҢЫҢШұ Ъ©ШұШҜЩҮ Ш§ШіШӘ. Ш§ЫҢЩҶ ШӘШәЫҢЫҢШұ Щ…ЫҢвҖҢШӘЩҲШ§ЩҶШҜ ЩҶШ§ШҙЫҢ Ш§ШІ ШЁЩҮвҖҢШұЩҲШІШұШіШ§ЩҶЫҢ ЩҫШ§ШұШ§Щ…ШӘШұЩҮШ§ШҢ Ш§Ш¬ШұШ§ЫҢ ШұЩҲЫҢШҜШ§ШҜЩҮШ§ ЫҢШ§ ЩҒШұШ§Ш®ЩҲШ§ЩҶЫҢ ШҜШіШӘЫҢ Щ…ШӘШҜ StateHasChanged ШЁШ§ШҙШҜ. Ш§Щ…Ш§ ЪҜШ§ЩҮЫҢ Ш§ЩҲЩӮШ§ШӘШҢ ШұЩҶШҜШұ ШЁЫҢШҙ Ш§ШІ ШӯШҜ Ш§ШӘЩҒШ§ЩӮ Щ…ЫҢвҖҢШ§ЩҒШӘШҜШҢ ШӯШӘЫҢ ЩҲЩӮШӘЫҢ ШӘШәЫҢЫҢШұ ЩҲШ§ЩӮШ№ЫҢ ШҜШұ ЩҲШ¶Ш№ЫҢШӘ ЩҲШ¬ЩҲШҜ ЩҶШҜШ§ШұШҜ. ШЁЩҮ Ш№ЩҶЩҲШ§ЩҶ Щ…Ш«Ш§Щ„ШҢ ЩҒШұШ¶ Ъ©ЩҶЫҢШҜ ЫҢЪ© Ъ©Ш§Щ…ЩҫЩҲЩҶЩҶШӘ ШіШ§ШҜЩҮ ШҜШ§ШұЫҢШҜ Ъ©ЩҮ ЫҢЪ© Щ„ЫҢШіШӘ Ъ©ШҙЩҲЫҢЫҢ (Dropdown) ШұШ§ ЩҶЩ…Ш§ЫҢШҙ Щ…ЫҢвҖҢШҜЩҮШҜ:
@inject NavigationManager NavigationManager

<select @onchange="OnValueChanged">
    <option value="">Ш§ЩҶШӘШ®Ш§ШЁ Ъ©ЩҶЫҢШҜ</option>
    @foreach (var item in Items)
    {
        <option value="@item">@item</option>
    }
</select>

@code {
    [Parameter] public List<string> Items { get; set; } = new();
    [Parameter] public EventCallback<string> ValueChanged { get; set; }

    private async Task OnValueChanged(ChangeEventArgs e)
    {
        string value = e.Value?.ToString() ?? "";
        await ValueChanged.InvokeAsync(value);
        NavigationManager.NavigateTo("/new-page");
    }
}
ШҜШұ Ш§ЫҢЩҶ Ъ©ШҜШҢ ЩҲЩӮШӘЫҢ Ъ©Ш§ШұШЁШұ ЪҜШІЫҢЩҶЩҮвҖҢШ§ЫҢ ШұШ§ Ш§ЩҶШӘШ®Ш§ШЁ Щ…ЫҢвҖҢЪ©ЩҶШҜШҢ Щ…ШӘШҜ OnValueChanged Ш§Ш¬ШұШ§ Щ…ЫҢвҖҢШҙЩҲШҜ ЩҲ ЩҫШі Ш§ШІ ШўЩҶШҢ ШЁЩҮ ШөЩҒШӯЩҮ Ш¬ШҜЫҢШҜЫҢ ЩҮШҜШ§ЫҢШӘ Щ…ЫҢвҖҢШҙЩҲШҜ. Ш§Щ…Ш§ Ш§ЪҜШұ Ш§ШІ Ш§ШЁШІШ§ШұЫҢ Щ…Ш§ЩҶЩҶШҜ Console.WriteLine ШҜШұ Щ…ШӘШҜЩҮШ§ЫҢ ЪҶШұШ®ЩҮ ШӯЫҢШ§ШӘ Ш§ШіШӘЩҒШ§ШҜЩҮ Ъ©ЩҶЫҢШҜШҢ Щ…Щ…Ъ©ЩҶ Ш§ШіШӘ Щ…ШӘЩҲШ¬ЩҮ ШҙЩҲЫҢШҜ Ъ©ЩҮ Ш§ЫҢЩҶ Ъ©Ш§Щ…ЩҫЩҲЩҶЩҶШӘ ЪҶЩҶШҜЫҢЩҶ ШЁШ§Шұ ШұЩҶШҜШұ Щ…ЫҢвҖҢШҙЩҲШҜвҖ”ЪҜШ§ЩҮЫҢ ШӯШӘЫҢ ШӘШ§ 20 ШЁШ§Шұ! Ш§ЫҢЩҶ ШұЩҒШӘШ§Шұ ШәЫҢШұЩ…ЩҶШӘШёШұЩҮ Щ…ЫҢвҖҢШӘЩҲШ§ЩҶШҜ ШҜЩ„Ш§ЫҢЩ„ Щ…Ш®ШӘЩ„ЩҒЫҢ ШҜШ§ШҙШӘЩҮ ШЁШ§ШҙШҜ Ъ©ЩҮ ШҜШұ Ш§ШҜШ§Щ…ЩҮ ШЁШұШұШіЫҢ Щ…ЫҢвҖҢЪ©ЩҶЫҢЩ….

ШҜЩ„Ш§ЫҢЩ„ ШұЩҶШҜШұ ШәЫҢШұШ¶ШұЩҲШұЫҢ
ЪҶШұШ®ЩҮ ШӯЫҢШ§ШӘ Blazor ЩҲ ШұЩҲЫҢШҜШ§ШҜЩҮШ§:
Blazor ШЁЩҮвҖҢШ·ЩҲШұ ЩҫЫҢШҙвҖҢЩҒШұШ¶ ЩҫШі Ш§ШІ Ш§Ш¬ШұШ§ЫҢ ЩҮШұ ШұЩҲЫҢШҜШ§ШҜ (Event) Щ…Ш§ЩҶЩҶШҜ @onchange ШҢ Щ…ШӘШҜ StateHasChanged ШұШ§ ЩҒШұШ§Ш®ЩҲШ§ЩҶЫҢ Щ…ЫҢвҖҢЪ©ЩҶШҜ ШӘШ§ Щ…Ш·Щ…ШҰЩҶ ШҙЩҲШҜ ШұШ§ШЁШ· Ъ©Ш§ШұШЁШұЫҢ ШЁЩҮвҖҢШұЩҲШІ Ш§ШіШӘ. Ш§ЫҢЩҶ ШұЩҒШӘШ§Шұ ШҜШұ Ъ©Щ„Ш§Ші ЩҫШ§ЫҢЩҮ ComponentBase ШӘШ№ШұЫҢЩҒ ШҙШҜЩҮ Ш§ШіШӘ. ШҜШұ Щ…Ш«Ш§Щ„ ШЁШ§Щ„Ш§ШҢ ЩҫШі Ш§ШІ Ш§Ш¬ШұШ§ЫҢ OnValueChangedШҢ Blazor ЫҢЪ© ШұЩҶШҜШұ Ш§Ш¶Ш§ЩҒЫҢ ШұШ§ ШўШәШ§ШІ Щ…ЫҢвҖҢЪ©ЩҶШҜШҢ ШӯШӘЫҢ Ш§ЪҜШұ ЩҮШҜШ§ЫҢШӘ (Navigation) ШЁЩҮ ШөЩҒШӯЩҮ Ш¬ШҜЫҢШҜ ШЁЩ„Ш§ЩҒШ§ШөЩ„ЩҮ Ш§ЩҶШ¬Ш§Щ… ШҙЩҲШҜ.

ЩҫШұШұЩҶШҜШұ (Prerendering) ШҜШұ Blazor Server:
Ш§ЪҜШұ Ш§ШІ Blazor Server ШЁШ§ ШӯШ§Щ„ШӘ InteractiveServer ЫҢШ§ ServerPrerendered Ш§ШіШӘЩҒШ§ШҜЩҮ Щ…ЫҢвҖҢЪ©ЩҶЫҢШҜШҢ Ъ©Ш§Щ…ЩҫЩҲЩҶЩҶШӘвҖҢЩҮШ§ Ш§ШЁШӘШҜШ§ ШұЩҲЫҢ ШіШұЩҲШұ ШЁЩҮвҖҢШөЩҲШұШӘ Ш§ШіШӘШ§ШӘЫҢЪ© ШұЩҶШҜШұ Щ…ЫҢвҖҢШҙЩҲЩҶШҜ ЩҲ ШіЩҫШі ШЁШ§ ШЁШұЩӮШұШ§ШұЫҢ Ш§ШӘШөШ§Щ„ SignalRШҢ ШҜЩҲШЁШ§ШұЩҮ ШұЩҶШҜШұ Щ…ЫҢвҖҢШҙЩҲЩҶШҜ. Ш§ЫҢЩҶ ШҜЩҲШЁШ§Шұ ШұЩҶШҜШұ ШҙШҜЩҶ ЪҜШ§ЩҮЫҢ Ш§ЩҲЩӮШ§ШӘ ШЁШ§ ШӘШәЫҢЫҢШұШ§ШӘ ЩҲШ¶Ш№ЫҢШӘ ШӘШұЪ©ЫҢШЁ ШҙШҜЩҮ ЩҲ ШӘШ№ШҜШ§ШҜ ШұЩҶШҜШұЩҮШ§ ШұШ§ Ш§ЩҒШІШ§ЫҢШҙ Щ…ЫҢвҖҢШҜЩҮШҜ.

ШЁЩҮвҖҢШұЩҲШІШұШіШ§ЩҶЫҢ ЩҫШ§ШұШ§Щ…ШӘШұЩҮШ§:
ЩҲЩӮШӘЫҢ ЩҫШ§ШұШ§Щ…ШӘШұЩҮШ§ЫҢ ЫҢЪ© Ъ©Ш§Щ…ЩҫЩҲЩҶЩҶШӘ (Щ…Ш§ЩҶЩҶШҜ Items ШҜШұ Щ…Ш«Ш§Щ„) ШӘШәЫҢЫҢШұ Щ…ЫҢвҖҢЪ©ЩҶЩҶШҜШҢ Щ…ШӘШҜ OnParametersSet Ш§Ш¬ШұШ§ Щ…ЫҢвҖҢШҙЩҲШҜ ЩҲ ШұЩҶШҜШұ Ш¬ШҜЫҢШҜЫҢ ШўШәШ§ШІ Щ…ЫҢвҖҢШҙЩҲШҜ. Ш§ЪҜШұ Ш§ЫҢЩҶ ЩҫШ§ШұШ§Щ…ШӘШұЩҮШ§ ШЁЩҮвҖҢШөЩҲШұШӘ Щ…ШұШ¬Ш№ (Reference Type) ШЁШ§ШҙЩҶШҜ ЩҲ Blazor ЩҶШӘЩҲШ§ЩҶШҜ ШӘШәЫҢЫҢШұШ§ШӘ ЩҲШ§ЩӮШ№ЫҢ ШұШ§ ШӘШҙШ®ЫҢШө ШҜЩҮШҜШҢ ШұЩҶШҜШұ ШәЫҢШұШ¶ШұЩҲШұЫҢ ШұШ® Щ…ЫҢвҖҢШҜЩҮШҜ.

ШұЩҒШӘШ§Шұ ЩҶШ§ШҜШұШіШӘ ШҜШұ Щ…ШӘШҜЩҮШ§ЫҢ async:
ШҜШұ Щ…ШӘШҜЩҮШ§ЫҢ ЩҶШ§ЩҮЩ…ЪҜШ§Щ… (Asynchronous) Щ…Ш§ЩҶЩҶШҜ OnValueChangedШҢ Ш§ЪҜШұ await ШЁЩҮвҖҢШҜШұШіШӘЫҢ Щ…ШҜЫҢШұЫҢШӘ ЩҶШҙЩҲШҜШҢ Blazor Щ…Щ…Ъ©ЩҶ Ш§ШіШӘ ШҜШұ ЩҶЩӮШ§Ш· Щ…Ш®ШӘЩ„ЩҒ Ш§Ш¬ШұШ§ЫҢ Task ШұЩҶШҜШұ ШұШ§ ШўШәШ§ШІ Ъ©ЩҶШҜ. Ш§ЫҢЩҶ ШұЩҒШӘШ§Шұ ШЁЩҮ ШҜЩ„ЫҢЩ„ ШӘШәЫҢЫҢШұ ЩҲШ¶Ш№ЫҢШӘ TaskStatus ШҜШұ ЪҶШұШ®ЩҮ Ш§Ш¬ШұШ§ЫҢ ШӘШіЪ© Ш§ШіШӘ.

Ш§ШЁШІШ§ШұЩҮШ§ЫҢ ШӘШҙШ®ЫҢШө Щ…ШҙЪ©Щ„
ШЁШұШ§ЫҢ ШҙЩҶШ§ШіШ§ЫҢЫҢ ШҜЩ„Ш§ЫҢЩ„ ШұЩҶШҜШұ ШЁЫҢШҙвҖҢШ§ШІШӯШҜШҢ Щ…ЫҢвҖҢШӘЩҲШ§ЩҶЫҢШҜ Ш§ШІ ШӘЪ©ЩҶЫҢЪ©вҖҢЩҮШ§ЫҢ ШіШ§ШҜЩҮвҖҢШ§ЫҢ Ш§ШіШӘЩҒШ§ШҜЩҮ Ъ©ЩҶЫҢШҜ:
  • Щ„Ш§ЪҜвҖҢЪҜШ°Ш§ШұЫҢ ШҜШұ ЪҶШұШ®ЩҮ ШӯЫҢШ§ШӘ: ШЁШ§ Ш§ЩҒШІЩҲШҜЩҶ Console.WriteLine ШЁЩҮ Щ…ШӘШҜЩҮШ§ЫҢ OnInitializedШҢ OnParametersSet ЩҲ ShouldRenderШҢ Щ…ЫҢвҖҢШӘЩҲШ§ЩҶЫҢШҜ ШӘШ№ШҜШ§ШҜ ЩҲ ШІЩ…Ш§ЩҶ ШұЩҶШҜШұЩҮШ§ ШұШ§ ШұШҜЫҢШ§ШЁЫҢ Ъ©ЩҶЫҢШҜ. Щ…Ш«Щ„Ш§ЩӢ:
protected override void OnInitialized()
{
    Console.WriteLine("OnInitialized called");
}

protected override void OnParametersSet()
{
    Console.WriteLine("OnParametersSet called");
}

protected override bool ShouldRender()
{
    Console.WriteLine("ShouldRender called");
    return true;
}
  • Ш§ШЁШІШ§ШұЩҮШ§ЫҢ ШӘЩҲШіШ№ЩҮвҖҢШҜЩҮЩҶШҜЩҮ Щ…ШұЩҲШұЪҜШұ: ШҜШұ Blazor WebAssemblyШҢ Ш§ШІ ШӘШЁ Network ЫҢШ§ Performance ШҜШұ Щ…ШұЩҲШұЪҜШұ Ш§ШіШӘЩҒШ§ШҜЩҮ Ъ©ЩҶЫҢШҜ ШӘШ§ ШҜШұШ®ЩҲШ§ШіШӘвҖҢЩҮШ§ ЩҲ ШІЩ…Ш§ЩҶвҖҢШЁЩҶШҜЫҢ ШұЩҶШҜШұЩҮШ§ ШұШ§ ШЁШұШұШіЫҢ Ъ©ЩҶЫҢШҜ.
  • ШҜЫҢШЁШ§ЪҜШұ Visual Studio: ЩҶЩӮШ§Ш· ШҙЪ©ШіШӘ (Breakpoints) ШұШ§ ШҜШұ Щ…ШӘШҜЩҮШ§ЫҢ Ъ©Щ„ЫҢШҜЫҢ ЩӮШұШ§Шұ ШҜЩҮЫҢШҜ ШӘШ§ Ш¬ШұЫҢШ§ЩҶ Ш§Ш¬ШұШ§ ШұШ§ ШҜЩӮЫҢЩӮвҖҢШӘШұ ШӘШӯЩ„ЫҢЩ„ Ъ©ЩҶЫҢШҜ.

ШұШ§ЩҮЪ©Ш§ШұЩҮШ§ЫҢ ШЁЩҮЫҢЩҶЩҮвҖҢШіШ§ШІЫҢ ШұЩҶШҜШұ

Ш§ШіШӘЩҒШ§ШҜЩҮ Ш§ШІ ShouldRender:
Щ…ШӘШҜ ShouldRender ШЁЩҮ ШҙЩ…Ш§ Ш§Ш¬Ш§ШІЩҮ Щ…ЫҢвҖҢШҜЩҮШҜ Ъ©ЩҶШӘШұЩ„ Ъ©ЩҶЫҢШҜ Ъ©ЩҮ ШўЫҢШ§ Ъ©Ш§Щ…ЩҫЩҲЩҶЩҶШӘ ШЁШ§ЫҢШҜ ШұЩҶШҜШұ ШҙЩҲШҜ ЫҢШ§ Ш®ЫҢШұ. ШЁШ§ ШЁШ§ШІЪҜШұШҜШ§ЩҶШҜЩҶ false ШҜШұ ШҙШұШ§ЫҢШ· Ш®Ш§ШөШҢ Щ…ЫҢвҖҢШӘЩҲШ§ЩҶЫҢШҜ ШұЩҶШҜШұ ШәЫҢШұШ¶ШұЩҲШұЫҢ ШұШ§ Щ…ШӘЩҲЩӮЩҒ Ъ©ЩҶЫҢШҜ. ШЁШұШ§ЫҢ Щ…Ш«Ш§Щ„:
private bool shouldRender = true;

protected override void OnParametersSet()
{
    Console.WriteLine("OnParametersSet called");
    shouldRender = true; // ЩҒЩӮШ· ЩҲЩӮШӘЫҢ ЩҫШ§ШұШ§Щ…ШӘШұЩҮШ§ ШӘШәЫҢЫҢШұ ЩҲШ§ЩӮШ№ЫҢ ШҜШ§ШұЩҶШҜ
}

protected override bool ShouldRender()
{
    Console.WriteLine("ShouldRender called");
    if (!shouldRender)
    {
        return false;
    }
    shouldRender = false; // ЩҫШі Ш§ШІ Ш§ЩҲЩ„ЫҢЩҶ ШұЩҶШҜШұШҢ ШӘШ§ ШӘШәЫҢЫҢШұ ШЁШ№ШҜЫҢ ШұЩҶШҜШұ ЩҶШҙЩҲШҜ
    return true;
}
ШҜШұ Ш§ЫҢЩҶШ¬Ш§ШҢ ЩҫШі Ш§ШІ Ш§ЩҲЩ„ЫҢЩҶ ШұЩҶШҜШұШҢ ШӘШ§ ШІЩ…Ш§ЩҶЫҢ Ъ©ЩҮ ШӘШәЫҢЫҢШұ Ш¬ШҜЫҢШҜЫҢ ШұШ® ЩҶШҜЩҮШҜШҢ ШұЩҶШҜШұ Щ…ШӘЩҲЩӮЩҒ Щ…ЫҢвҖҢШҙЩҲШҜ.

Щ…ШҜЫҢШұЫҢШӘ ШұЩҲЫҢШҜШ§ШҜЩҮШ§ ШЁШ§ ШҜЩӮШӘ:
Ш§ЪҜШұ ШұЩҲЫҢШҜШ§ШҜЫҢ Щ…Ш§ЩҶЩҶШҜ ЩҮШҜШ§ЫҢШӘ (Navigation) ШЁШ§Ш№Ш« Ш®ШұЩҲШ¬ Ш§ШІ Ъ©Ш§Щ…ЩҫЩҲЩҶЩҶШӘ Щ…ЫҢвҖҢШҙЩҲШҜШҢ ЩҶЫҢШ§ШІЫҢ ШЁЩҮ ШұЩҶШҜШұ Ш§Ш¶Ш§ЩҒЫҢ ЩҶЫҢШіШӘ. Щ…ЫҢвҖҢШӘЩҲШ§ЩҶЫҢШҜ Ш§ШІ ЫҢЪ© ЩҫШұЪҶЩ… (Flag) Ш§ШіШӘЩҒШ§ШҜЩҮ Ъ©ЩҶЫҢШҜ:
private bool isNavigating = false;

private async Task OnValueChanged(ChangeEventArgs e)
{
    string value = e.Value?.ToString() ?? "";
    await ValueChanged.InvokeAsync(value);
    isNavigating = true;
    NavigationManager.NavigateTo("/new-page");
}

protected override bool ShouldRender()
{
    return !isNavigating;
}

ШЁЩҮЫҢЩҶЩҮвҖҢШіШ§ШІЫҢ Щ…ШӘШҜЩҮШ§ЫҢ async:
ШҜШұ Щ…ШӘШҜЩҮШ§ЫҢ ЩҶШ§ЩҮЩ…ЪҜШ§Щ…ШҢ Ш§ШІ StateHasChanged ШЁЩҮвҖҢШөЩҲШұШӘ ШҜШіШӘЫҢ Ш§ШіШӘЩҒШ§ШҜЩҮ Ъ©ЩҶЫҢШҜ ШӘШ§ ШұЩҶШҜШұ ЩҒЩӮШ· ШҜШұ ЩҶЩӮШ§Ш· Щ„Ш§ШІЩ… Ш§ЩҶШ¬Ш§Щ… ШҙЩҲШҜ:
private async Task LoadDataAsync()
{
    await Task.Delay(1000); // ШҙШЁЫҢЩҮвҖҢШіШ§ШІЫҢ ШҜШұЫҢШ§ЩҒШӘ ШҜШ§ШҜЩҮ
    progress = 50;
    StateHasChanged(); // ШұЩҶШҜШұ ШҜШіШӘЫҢ ШҜШұ Щ…ЫҢШ§ЩҶЩҮ Ъ©Ш§Шұ
    await Task.Delay(1000);
    progress = 100;
}

Ъ©ЩҶШӘШұЩ„ ЩҫШұШұЩҶШҜШұ:
ШҜШұ Blazor ServerШҢ Ш§ЪҜШұ ЩҫШұШұЩҶШҜШұ ШЁШ§Ш№Ш« ШұЩҶШҜШұ Ш§Ш¶Ш§ЩҒЫҢ Щ…ЫҢвҖҢШҙЩҲШҜШҢ Щ…ЫҢвҖҢШӘЩҲШ§ЩҶЫҢШҜ ШўЩҶ ШұШ§ ШәЫҢШұЩҒШ№Ш§Щ„ Ъ©ЩҶЫҢШҜ. ШҜШұ ЩҒШ§ЫҢЩ„ Program.cs ЫҢШ§ ШҜШұ ШіШ·Шӯ Ъ©Ш§Щ…ЩҫЩҲЩҶЩҶШӘ:
@rendermode InteractiveServer(false) // ШәЫҢШұЩҒШ№Ш§Щ„ Ъ©ШұШҜЩҶ ЩҫШұШұЩҶШҜШұ

Ш§ШіШӘЩҒШ§ШҜЩҮ Ш§ШІ ЩҶЩҲШ№вҖҢЩҮШ§ЫҢ ШҙЩҶШ§Ш®ШӘЩҮвҖҢШҙШҜЩҮ (Known Types):
Blazor ШЁШұШ§ЫҢ ЩҶЩҲШ№вҖҢЩҮШ§ЫҢ ШіШ§ШҜЩҮ (Primitive Types) Щ…Ш§ЩҶЩҶШҜ int ЫҢШ§ stringШҢ ШӘШәЫҢЫҢШұШ§ШӘ ШұШ§ ШЁЩҮвҖҢШ®ЩҲШЁЫҢ ШӘШҙШ®ЫҢШө Щ…ЫҢвҖҢШҜЩҮШҜ ЩҲ ШұЩҶШҜШұ ШәЫҢШұШ¶ШұЩҲШұЫҢ ШұШ§ Ъ©Ш§ЩҮШҙ Щ…ЫҢвҖҢШҜЩҮШҜ. ШҜШұ Щ…ЩӮШ§ШЁЩ„ШҢ ШЁШұШ§ЫҢ ЩҶЩҲШ№вҖҢЩҮШ§ЫҢ Щ…ШұШ¬Ш№ (Reference Types)ШҢ ШЁЩҮШӘШұ Ш§ШіШӘ ШӘШәЫҢЫҢШұШ§ШӘ ШұШ§ ШөШұШ§ШӯШӘШ§ЩӢ ШЁШұШұШіЫҢ Ъ©ЩҶЫҢШҜ.

ЩҶШӘЫҢШ¬ЩҮвҖҢЪҜЫҢШұЫҢ
Щ…ШҜЫҢШұЫҢШӘ ШұЩҶШҜШұ ШҜШұ Blazor Щ…ЩҮШ§ШұШӘЫҢ Ш§ШіШӘ Ъ©ЩҮ ШЁШ§ ШҜШұЪ© Ш№Щ…ЫҢЩӮвҖҢШӘШұ ЪҶШұШ®ЩҮ ШӯЫҢШ§ШӘ ЩҲ ШұЩҒШӘШ§Шұ ЩҒШұЫҢЩ…вҖҢЩҲШұЪ© ШЁЩҮ ШҜШіШӘ Щ…ЫҢвҖҢШўЫҢШҜ. ШұЩҶШҜШұ ШЁЫҢШҙвҖҢШ§ШІШӯШҜ ЩҶЩҮвҖҢШӘЩҶЩҮШ§ Щ…ЩҶШ§ШЁШ№ ШұШ§ ЩҮШҜШұ Щ…ЫҢвҖҢШҜЩҮШҜШҢ ШЁЩ„Ъ©ЩҮ Щ…ЫҢвҖҢШӘЩҲШ§ЩҶШҜ Ш№Щ…Щ„Ъ©ШұШҜ ШЁШұЩҶШ§Щ…ЩҮ ШұШ§ ШӘШӯШӘ ШӘШЈШ«ЫҢШұ ЩӮШұШ§Шұ ШҜЩҮШҜ. ШЁШ§ Ш§ШіШӘЩҒШ§ШҜЩҮ Ш§ШІ Ш§ШЁШІШ§ШұЩҮШ§ЫҢ ШӘШҙШ®ЫҢШө Щ…Ш§ЩҶЩҶШҜ Щ„Ш§ЪҜвҖҢЪҜШ°Ш§ШұЫҢ ЩҲ ШҜЫҢШЁШ§ЪҜШұШҢ ЩҲ ЩҫЫҢШ§ШҜЩҮвҖҢШіШ§ШІЫҢ ШұШ§ЩҮЪ©Ш§ШұЩҮШ§ЫҢЫҢ Щ…Ш§ЩҶЩҶШҜ ShouldRenderШҢ Ъ©ЩҶШӘШұЩ„ ШұЩҲЫҢШҜШ§ШҜЩҮШ§ ЩҲ ШЁЩҮЫҢЩҶЩҮвҖҢШіШ§ШІЫҢ Щ…ШӘШҜЩҮШ§ЫҢ asyncШҢ Щ…ЫҢвҖҢШӘЩҲШ§ЩҶЫҢШҜ Ш§ЫҢЩҶ Щ…ШҙЪ©Щ„ ШұШ§ ШЁЩҮвҖҢШ·ЩҲШұ Щ…ШӨШ«ШұЫҢ ШӯЩ„ Ъ©ЩҶЫҢШҜ.
