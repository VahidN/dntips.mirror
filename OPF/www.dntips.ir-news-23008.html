‫استفاده از AI Gateway با Clawdbot: دسترسی به مدل‌های هوش مصنوعی متعدد از طریق یک نقطه پایانی


نویسنده: راویAI
تاریخ: ۱۴۰۴/۱۱/۰۸ ۰۴:۲۹
آدرس: www.dntips.ir
Clawdbot یک دستیار هوش مصنوعی شخصی است که توسط Claude با حافظه پایدار پشتیبانی می‌شود. این ابزار قابلیت مرور وب، اجرای دستورات شل و مدیریت فایل‌ها را در هر سیستم عاملی دارد. Vercel AI Gateway به شما امکان می‌دهد تا با استفاده از Clawdbot به صدها مدل از چندین ارائه‌دهنده از طریق یک نقطه پایانی واحد دسترسی پیدا کنید. AI Gateway دسترسی یکپارچه API را در سراسر مدل‌ها بدون نیاز به مدیریت کلیدهای API جداگانه فراهم می‌کند. برای استفاده از Clawdbot با Vercel AI Gateway، ابتدا یک کلید API در داشبورد AI Gateway ایجاد کنید. سپس Clawdbot را با استفاده از دستورات curl یا npm نصب کنید. پس از نصب، با اجرای دستور `clawdbot onboard --install-daemon` و انتخاب Vercel AI Gateway به عنوان ارائه‌دهنده، می‌توانید از بین صدها مدل موجود انتخاب کنید. این ادغام به توسعه‌دهندگان کمک می‌کند تا به راحتی مدل‌های هوش مصنوعی مختلف را در برنامه‌های خود ادغام کنند و از مزایای هر کدام بهره‌مند شوند. با استفاده از یک نقطه پایانی واحد، مدیریت و به‌روزرسانی مدل‌ها بسیار ساده‌تر خواهد شد. برای کسب اطلاعات بیشتر در مورد Clawdbot و سایر یکپارچگی‌ها، به مستندات AI Gateway مراجعه کنید. Vercel به توسعه‌دهندگان امکان می‌دهد تا با یک حساب رایگان شروع به ساخت برنامه‌های خود کنند و در صورت نیاز به Pro یا Enterprise، با یک متخصص مشورت نمایندند.
