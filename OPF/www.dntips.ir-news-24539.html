‫انتشار ReSharper 2026.2 با پشتیبانی از عامل‌های هوش مصنوعی در Visual Studio و دیباگ .NET برای VS Code


نویسنده: راویAI
تاریخ: ۱۴۰۵/۰۵/۰۱ ۰۸:۰۵
آدرس: www.dntips.ir
شرکت JetBrains نسخه 2026.2 ابزار ReSharper را منتشر کرد که تمرکز اصلی آن بر ایجاد یک اکوسیستم باز برای عامل‌های هوش مصنوعی در محیط Visual Studio است. این نسخه از پروتکل Agent Client Protocol یا ACP استفاده می‌کند تا توسعه‌دهندگان بتوانند بدون وابستگی به یک ارائه‌دهنده خاص، از مدل‌ها و عامل‌های مختلف هوش مصنوعی متناسب با نیاز خود استفاده کنند.
در این به‌روزرسانی، عامل هوش مصنوعی Junie در حالت پیش‌نمایش معرفی شده است که به صورت LLM-agnostic عمل می‌کند و به کاربر اجازه می‌دهد مدل مورد نظر خود را انتخاب کند. این ابزار قادر است کدهای پیچیده را بنویسد، عملیات بازسازی کد یا refactoring را به صورت خودکار انجام دهد و حتی دستورات ترمینال و گردش کارهای Git را مستقیماً از طریق محیط چت مدیریت کند.
علاوه بر قابلیت‌های هوش مصنوعی، موتور دیباگ ReSharper اکنون به ویرایشگرهای سازگار با VS Code گسترش یافته است. همچنین تحلیل‌های هسته زبان C# و قابلیت‌های بازسازی کد بهبود یافته و پشتیبانی از حالت Out-of-Process برای افزایش پایداری و کارایی گسترده‌تر شده است.
