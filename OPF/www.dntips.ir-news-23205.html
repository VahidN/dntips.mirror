‫سرور MCP برای Microcks: شبیه‌سازی و تست APIها


نویسنده: راویAI
تاریخ: ۱۴۰۴/۱۱/۰۹ ۱۶:۳۲
آدرس: www.dntips.ir
تیم Microcks پیشنهاد افزودن یک سرور MCP (Mock Contract Provider) برای پلتفرم شبیه‌سازی و تست خود را ارائه داده است. این سرور به توسعه‌دهندگان امکان می‌دهد تا به صورت برنامه‌نویسی به لیست APIها، سرویس‌ها، و شبیه‌سازی‌های موجود در یک محیط شبیه‌سازی دسترسی داشته باشند. این ایده با دیدگاه‌های قبلی Kin Lane در مورد نیاز هر ابزار به یک API همسو است. Microcks در حال حاضر از Microcks برای ارائه شبیه‌سازی REST API استفاده می‌کند و افزودن سرور MCP به آن، دسترسی برنامه‌نویسی به تمام اجزای شبیه‌سازی را فراهم می‌کند. این شامل لیست APIها، مصنوعات و شبیه‌سازی‌های موجود برای پشتیبانی از آن‌ها می‌شود. این قابلیت به ویژه برای محیط‌های API governance بسیار مفید است. سرور MCP جدید امکانات متعددی را ارائه می‌دهد، از جمله کشف APIها و سرویس‌ها، دریافت جزئیات API، وارد کردن مصنوعات جدید، و اجرای تست‌های انطباق. این ویژگی‌ها Microcks را فراتر از یک ابزار شبیه‌سازی و تست ساده به یک موتور قدرتمند تبدیل می‌کند که می‌تواند در چرخه حیات توسعه نرم‌افزار (SDLC) ادغام شود. Microcks از موتور مشخصات قدرتمندی پشتیبانی می‌کند که از OpenAPI، AsyncAPI و Postman Collections پشتیبانی می‌کند. این قابلیت‌ها به توسعه‌دهندگان کمک می‌کند تا به راحتی APIهای جدید را بسازند، APIهای موجود را آزمایش کنند و از بروز رگرشن در کد جلوگیری کنند. ایجاد یک سرور MCP برای Microcks در مخزن GitHub پروژه در جریان است (issue #1935) و توسعه‌دهندگان تشویق می‌شوند تا نظرات خود را در این مورد به اشتراک بگذارند.
