‫توسعه اپلیکیشن ویندوزی برای مسدودسازی ردیاب‌ها و رمزنگاری ترافیک به صورت خودکار


نویسنده: راویAI
تاریخ: ۱۴۰۴/۱۰/۱۶ ۰۸:۳۴
آدرس: www.dntips.ir
توسعه‌دهندگان خسته از تنظیمات پیچیده VPN، افزونه‌های مرورگر و تنظیمات DNS برای حفظ حریم خصوصی، اپلیکیشن HSIP را برای ویندوز توسعه داده‌اند. این اپلیکیشن به صورت خودکار ردیاب‌ها (مانند Google Analytics و DoubleClick) را مسدود کرده و ترافیک را با الگوریتم ChaCha20-Poly1305 رمزنگاری می‌کند. HSIP به صورت ساکت در پس‌زمینه اجرا می‌شود و یک آیکون در نوار سیستم نمایش می‌دهد (سبز = محافظت شده، قرمز = آفلاین). نصب و استفاده از آن بسیار ساده است و نیازی به تنظیمات اضافی یا تغییرات DNS ندارد. HSIP از سه کامپوننت اصلی تشکیل شده است: hsip-gateway.exe (پراکسی HTTP/HTTPS برای مسدودسازی ردیاب‌ها)، hsip-cli.exe (دمون پس‌زمینه با API وضعیت) و hsip-tray.exe (آیکون نوار سیستم). این اپلیکیشن با استفاده از زبان Rust توسعه یافته و از Ed25519 برای هویت، X25519 برای تبادل کلید و ChaCha20-Poly1305 برای رمزنگاری استفاده می‌کند. کد منبع آن در GitHub در دسترس است. استفاده شخصی از آن رایگان است، اما استفاده تجاری نیازمند دریافت لایسنس می‌باشد.
