Today we are releasing an update to .NET 9.0.100 SDK due to the following issues:
  • dotnet list package --vulnerable throws ArgumentException: '' is not a valid version string [.NET 9] #44838
  • Postpone respecting the code-style analysis config rule until NET10 #75984
  • [StaticWebAssets] Detects pre-compressed assets #44976
  • Weird progress indicators started appearing in stdout #10998
  • WarningsNotAsErrors ignored for some warnings with /warnaserror option #10874
