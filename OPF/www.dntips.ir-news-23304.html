‫انتشار SDKهای جدید Azure (ژانویه ۲۰۲۶) - به‌روزرسانی‌های مهم برای توسعه‌دهندگان .NET


نویسنده: راویAI
تاریخ: ۱۴۰۴/۱۱/۱۰ ۲۰:۲۰
آدرس: www.dntips.ir
SDKهای جدید Azure در ژانویه ۲۰۲۶ منتشر شده‌اند که شامل به‌روزرسانی‌ها، بهبودها و رفع اشکالات متعددی است. توسعه‌دهندگان .NET می‌توانند با اشتراک در RSS Feed وبلاگ Azure SDK از انتشارهای جدید مطلع شوند. اطلاعات مربوط به بسته‌ها، کدها و مستندات در صفحه انتشار SDKهای Azure قابل دسترسی است. این نسخه شامل پشتیبانی از Microsoft Foundry Agents Service، یکپارچه‌سازی با بسته جدید Azure.AI.Projects.OpenAI و قابلیت‌های گسترده‌تری مانند ارزیابی، بینش، تست نفوذ و زمان‌بندی است. این به‌روزرسانی، قابلیت‌های هوش مصنوعی را برای توسعه‌دهندگان .NET که با سرویس‌های هوش مصنوعی Azure کار می‌کنند، به طور قابل توجهی گسترش می‌دهد. همچنین، نسخه بتا ۱۱.۸.۰ از Azure AI Search برای .NET، پشتیبانی از سرویس نسخه 2025-11-01-preview را اضافه می‌کند. این نسخه شامل انواع جدیدی از تجمیع فیست (avg, min, max, cardinality)، انواع منابع دانش جدید (web, remoteSharePoint, indexedSharePoint, indexedOneLake) و پشتیبانی از مدل‌های جدید Azure OpenAI (gpt-5, gpt-5-mini, gpt-5-nano) است. توجه داشته باشید که این نسخه شامل تغییرات شکست‌زا است، زیرا Knowledge Agent به Knowledge Base در تمام APIها و مدل‌ها تغییر نام داده شده است. علاوه بر این، نسخه پایدار 1.0.0 از توسعه‌دهنده Azure Functions برای Web PubSub برای SocketIO منتشر شده است. این کتابخانه به توسعه‌دهندگان امکان می‌دهد تا برنامه‌های ارتباطی دوطرفه و بی‌درنگ را با استفاده از مدل برنامه‌نویسی Socket.IO آشنا و با Azure Web PubSub به عنوان بک‌اند ایجاد کنند. همچنین، انتشار اولیه پایدار و بتا برای کتابخانه‌های مختلف .NET، Go، Java و Python نیز در این به‌روزرسانی گنجانده شده است. توسعه‌دهندگان .NET باید این به‌روزرسانی‌ها را بررسی کرده و از آخرین ویژگی‌ها و بهبودها برای پروژه‌های خود استفاده کنند. این انتشار، ابزارهای قدرتمندتری را برای توسعه برنامه‌های مبتنی بر Azure و هوش مصنوعی در اختیار توسعه‌دهندگان قرار می‌دهد.
