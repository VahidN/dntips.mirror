ШӘШ§ШұЫҢШ®: ЫұЫіЫ№Ы·/ЫұЫ°/ЫІЫі ЫұЫі:ЫҙЫө
ШўШҜШұШі: www.dntips.ir
ЩҒШұШ¶ Ъ©ЩҶЫҢШҜ ШҜШұ ШЁШұЩҶШ§Щ…ЩҮвҖҢЫҢ Ш®ЩҲШҜШҢ ЫҢЪ© ШіШұЩҲЫҢШі Ш¬ЩҶШұЫҢЪ© ШұШ§ Ш·ШұШ§ШӯЫҢ Ъ©ШұШҜЩҮвҖҢШ§ЫҢШҜ. ШЁШұШ§ЫҢ Щ…Ш«Ш§Щ„ Ш®ЩҲШҜ ASP.NET Core ШЁЩҮ ЩҮЩ…ШұШ§ЩҮ ШіШұЩҲЫҢШі Ш¬ЩҶШұЫҢЪ© <ILogger<T Ш§ШіШӘ ЩҲ Ш§ЪҜШұ ШЁШұШ§ЫҢ ЩҶЩ…ЩҲЩҶЩҮ ШЁШ®ЩҲШ§ЩҮЫҢЩ… ШўЩҶвҖҢШұШ§ ШҜШұ ШіШ§ШІЩҶШҜЩҮвҖҢЫҢ Ъ©ЩҶШӘШұЩ„ШұЫҢ Щ…Ш§ЩҶЩҶШҜ ValuesController ШӘШІШұЫҢЩӮ Ъ©ЩҶЫҢЩ…ШҢ ЩҶШӯЩҲЩҮвҖҢЫҢ ШӘШ№ШұЫҢЩҒ ШўЩҶ ШЁЩҮ ШөЩҲШұШӘ <ILogger<ValuesController Ш®ЩҲШ§ЩҮШҜ ШЁЩҲШҜ. ЩҮШұ ЪҶЩҶШҜ ШӘЩҶШёЫҢЩ…Ш§ШӘ Ш§ЫҢЩҶ ШіШұЩҲЫҢШі ЩҫЫҢШҙШӘШұ Ш§ЩҶШ¬Ш§Щ… ШҙШҜЩҮвҖҢШ§ШіШӘШҢ Ш§Щ…Ш§ Ш§ЪҜШұ ШЁШ®ЩҲШ§ЩҮЫҢЩ… ШўЩҶвҖҢШұШ§ ШЁЩҮ ЩҮЩ…ЫҢЩҶ ЩҶШӯЩҲ <ILogger<T ШЁЩҮ Щ…ШӘШҜЩҮШ§ЫҢЫҢ Щ…Ш§ЩҶЩҶШҜ services.AddScoped Щ…Ш№ШұЩҒЫҢ Ъ©ЩҶЫҢЩ…ШҢ Ъ©Ш§Шұ ЩҶЩ…ЫҢвҖҢЪ©ЩҶШҜШӣ ЩҶЩ…ЩҲЩҶЩҮвҖҢЫҢ ШҜЫҢЪҜШұЫҢ Ш§ШІ Ш§ЫҢЩҶ ШҜШіШӘ Generic RepositoryЩҮШ§ ЩҮШіШӘЩҶШҜ:
 // does not work: services.AddScoped<IGenericRepository<T>,EFRepository<T>>();


ЩҶШӯЩҲЩҮвҖҢЫҢ Щ…Ш№ШұЩҒЫҢ ШіШұЩҲЫҢШівҖҢЩҮШ§ЫҢ Ш¬ЩҶШұЫҢЪ© ЩҶШ§Щ…ШӯШҜЩҲШҜ (Open Generics ЩҲ ЫҢШ§ Unbound Generics) ШЁЩҮ ШіЫҢШіШӘЩ… ШӘШІШұЫҢЩӮ ЩҲШ§ШЁШіШӘЪҜЫҢвҖҢЩҮШ§

Ш§ЪҜШұ ШЁШ®ЩҲШ§ЩҮЫҢЩ… ЫҢЪ© ШіШұЩҲЫҢШі Ш¬ЩҶШұЫҢЪ© ШұШ§ ШЁЩҮ ШіЫҢШіШӘЩ… ШӘШІШұЫҢЩӮ ЩҲШ§ШЁШіШӘЪҜЫҢвҖҢЩҮШ§ЫҢ ШЁШұЩҶШ§Щ…ЩҮвҖҢЩҮШ§ЫҢ NET Core. ШЁЩҮ ЩҶШӯЩҲ Щ…ШӘШҜШ§ЩҲЩ„ЫҢ Щ…Ш№ШұЩҒЫҢ Ъ©ЩҶЫҢЩ…ШҢ ЩҶЫҢШ§ШІ Ш§ШіШӘ ШЁЩҮ Ш§ШІШ§ЫҢ ШӘЪ© ШӘЪ© TЩҮШ§ЫҢ Щ…ЫҢШіШұ ЩҲ ШӘШ№ШұЫҢЩҒ ШҙШҜЩҮвҖҢЫҢ ШҜШұ ШЁШұЩҶШ§Щ…ЩҮШҢ Ш§ЫҢЩҶЪ©Ш§Шұ ШөЩҲШұШӘ ЪҜЫҢШұШҜ:
public void ConfigureServices(IServiceCollection services)
{
    services.AddScoped<IStore<User>, SqlStore<User>>();
    services.AddScoped<IStore<Invoice>, SqlStore<Invoice>>();
    services.AddScoped<IStore<Payment>, SqlStore<Payment>>();
    // ...
}
ЩҲ ШҜШұ Ш§ЫҢЩҶШ¬Ш§ ШЁЩҮ Ш§ШІШ§ЫҢ ЩҮШұ T ЫҢШ§ Щ…ЩҲШ¬ЩҲШҜЫҢШӘ Ш¬ШҜЫҢШҜЫҢ ШҜШұ ШЁШұЩҶШ§Щ…ЩҮШҢ ЩҶЫҢШ§ШІ Ш§ШіШӘ ЫҢЪ© ШіШ·Шұ ШҜЫҢЪҜШұ ШұШ§ ЩҶЫҢШІ ШӘШ№ШұЫҢЩҒ Ъ©ШұШҜ. Ш®ЩҲШҙШЁШ®ШӘШ§ЩҶЩҮ ШҜШұ Ш§ЫҢЩҶ ШіЫҢШіШӘЩ…ШҢ Ш§Щ…Ъ©Ш§ЩҶ ШӘШ№ШұЫҢЩҒ Ш¬ЩҶШұЫҢЪ©вҖҢЩҮШ§ЫҢ ЩҶШ§Щ…ШӯШҜЩҲШҜ ЩҲ ШЁШ§ШІ ЩҶЫҢШІ ЩҲШ¬ЩҲШҜ ШҜШ§ШұШҜ:
public void ConfigureServices(IServiceCollection services)
{
    services.AddScoped(typeof(IStore<>), typeof(SqlStore<>));
}
ШЁЩҮ Ш§ЫҢЩҶ ШӘШұШӘЫҢШЁ ШӘЩ…Ш§Щ… ШіШ·ШұЩҮШ§ЫҢЫҢ Ъ©ЩҮ ЩҫЫҢШҙШӘШұ ШӘШ№ШұЫҢЩҒ Ъ©ШұШҜЫҢЩ…ШҢ ШӘШЁШҜЫҢЩ„ ШЁЩҮ ЫҢЪ© ШіШ·Шұ ЩҒЩҲЩӮ Щ…ЫҢвҖҢШҙЩҲЩҶШҜ. ШҜШұ Ш§ЫҢЩҶШ¬Ш§ Ш§ШІ Overload ШәЫҢШұШ¬ЩҶШұЫҢЪ© Щ…ШӘШҜ AddScoped Ш§ШіШӘЩҒШ§ШҜЩҮ ШҙШҜЩҮШӣ ШЁЩҮ ЩҮЩ…ЫҢЩҶ Ш¬ЩҮШӘ Ш§ШІ ЩҲШ§ЪҳЩҮвҖҢЫҢ Ъ©Щ„ЫҢШҜЫҢ typeof ШЁШұШ§ЫҢ Щ…Ш№ШұЩҒЫҢ ЩҶЩҲШ№вҖҢЩҮШ§ЫҢ Ш¬ЩҶШұЫҢЪ© ШЁШ§ШІ Ъ©Щ…Ъ© ЪҜШұЩҒШӘЩҮ ШҙШҜЩҮвҖҢШ§ШіШӘ. Ш°Ъ©Шұ <> ЩҶЫҢШІ ШЁЩҮ Щ…Ш№ЩҶШ§ЫҢ ШӘЩҒШіЫҢШұ ЩҲ ЩҲЩҮЩ„ЩҮ ШіШ§ШІЫҢ ЩҮШұ ЩҶЩҲШ№ ШҜШұШ®ЩҲШ§ШіШӘЫҢ ШұШіЫҢШҜЩҮШҢ ШҜШұ ШІЩ…Ш§ЩҶ Ш§Ш¬ШұШ§ Щ…ЫҢвҖҢШЁШ§ШҙШҜ.


Щ…ШӯШҜЩҲШҜЫҢШӘ Ъ©Ш§Шұ Ъ©ШұШҜЩҶ ШЁШ§ Ш¬ЩҶШұЫҢЪ©вҖҢЩҮШ§ЫҢ ЩҶШ§Щ…ШӯШҜЩҲШҜ ШҜШұ ШіЫҢШіШӘЩ… ШӘШІШұЫҢЩӮ ЩҲШ§ШЁШіШӘЪҜЫҢвҖҢЩҮШ§

ШЁШ§ ШӘШ№ШұЫҢЩҒ ШӘЪ© ШіШ·Шұ ЩҒЩҲЩӮШҢ ЩҮШұ ЪҶЩҶШҜ ШЁШұЩҶШ§Щ…ЩҮ ШЁШҜЩҲЩҶ Щ…ШҙЪ©Щ„ Ъ©Ш§Щ…ЩҫШ§ЫҢЩ„ Щ…ЫҢвҖҢШҙЩҲШҜШҢ Ш§Щ…Ш§ Ш§ЪҜШұ ШҜШұ ШІЩ…Ш§ЩҶ Ш§Ш¬ШұШ§ЫҢ ШЁШұЩҶШ§Щ…ЩҮШҢ <IStore<T Ш§ЫҢ ШұШ§ ШҜШұШ®ЩҲШ§ШіШӘ Ъ©ЩҶЫҢШҜ Ъ©ЩҮ Щ…ЫҢШіШұ ЩҶШЁШ§ШҙШҜ (ШҜШұ Ш®ЩҲШ§ШіШӘ ЩҮШұ ЩҶЩҲШ№ЫҢ ШҜШұ ШІЩ…Ш§ЩҶ Ш§Ш¬ШұШ§ ШЁШ§ Ш¬ЩҶШұЫҢЪ©вҖҢЩҮШ§ЫҢ ШЁШ§ШІ Щ…Ш№ШұЩҒЫҢ ШҙШҜЩҮШҢ Щ…ЫҢШіШұ Ш§ШіШӘ)ШҢ ЫҢЪ© Ш§ШіШӘШ«ЩҶШ§ЫҢ ШІЩ…Ш§ЩҶ Ш§Ш¬ШұШ§ ШұШ§ ШҜШұЫҢШ§ЩҒШӘ Щ…ЫҢвҖҢЪ©ЩҶЫҢШҜШӣ ШЁШұШ§ЫҢ Щ…Ш«Ш§Щ„ Ш§ЪҜШұ ЩҶЩҲШ№ T ШЁЩҮ Ъ©Щ„Ш§ШівҖҢЩҮШ§ Щ…ШӯШҜЩҲШҜ ШҙШҜЩҮ ШЁШ§ШҙШҜ ЩҲ ШҜШұ ЩӮШіЩ…ШӘЫҢ Ш§ШІ ШЁШұЩҶШ§Щ…ЩҮШҢ <IStore<int ШҜШұШ®ЩҲШ§ШіШӘ ШҙЩҲШҜ. ЩҮШұЪҶЩҶШҜ Ш§ЫҢЩҶ Щ…ЩҲШ§ШұШҜ ШЁШ§ ЫҢЪ©ШЁШ§Шұ ШўШІЩ…Ш§ЫҢШҙ ШЁШұЩҶШ§Щ…ЩҮШҢ ЩӮШ§ШЁЩ„ ЫҢШ§ЩҒШӘ ШҙШҜЩҶ ЩҲ ШұЩҒШ№ Щ…ЫҢвҖҢШЁШ§ШҙЩҶШҜ.


Ъ©ШӘШ§ШЁШ®Ш§ЩҶЩҮвҖҢЫҢ Ъ©Щ…Ъ©ЫҢ Scrutor ЩҶЫҢШІ Ш§ШІ Ш¬ЩҶШұЫҢЪ©вҖҢЩҮШ§ЫҢ ШЁШ§ШІ ЩҫШҙШӘЫҢШЁШ§ЩҶЫҢ Щ…ЫҢвҖҢЪ©ЩҶШҜ

ШҜШұ ЩӮШіЩ…ШӘ ЩӮШЁЩ„ ЩҶШӯЩҲЩҮвҖҢЫҢ Ш§ШіЪ©ЩҶ Ш§ШіЩ…ШЁЩ„ЫҢвҖҢЩҮШ§ЫҢ ШЁШұЩҶШ§Щ…ЩҮ ШұШ§ ШӘЩҲШіШ· Ъ©ШӘШ§ШЁШ®Ш§ЩҶЩҮвҖҢЫҢ Ъ©Щ…Ъ©ЫҢ Scrutor ШЁШұШұШіЫҢ Ъ©ШұШҜЫҢЩ…. Ш§ЫҢЩҶ Ъ©ШӘШ§ШЁШ®Ш§ЩҶЩҮ Ш§Щ…Ъ©Ш§ЩҶ ЫҢШ§ЩҒШӘЩҶ ЩҲ ЩҒЫҢЩ„ШӘШұ Ъ©Щ„Ш§ШівҖҢЩҮШ§ ЩҲ Щ…Ш№ШұЩҒЫҢ ШўЩҶвҖҢЩҮШ§ ШұШ§ ШЁЩҮ ШіЫҢШіШӘЩ… ШӘШІШұЫҢЩӮ ЩҲШ§ШЁШіШӘЪҜЫҢвҖҢЩҮШ§ШҢ ШЁШұ Ш§ШіШ§Ші ЩҲЫҢЪҳЪҜЫҢ Ш¬ЩҶШұЫҢЪ©вҖҢЩҮШ§ЫҢ ШЁШ§ШІ ЩҶЫҢШІ ШҜШ§ШұШ§ Ш§ШіШӘ:
services.Scan(scan => scan 
  .FromAssemblyOf<CombinedService>()  
    .AddClasses(x=> x.AssignableTo(typeof(IOpenGeneric<>))) // Can close generic types 
    .AsMatchingInterface())


ШіШ§ШҜЩҮ ШіШ§ШІЫҢ В«Щ…Ш«Ш§Щ„ 2: ЩҲЩҮЩ„ЩҮ ШіШ§ШІЫҢ ШҜШұ ШөЩҲШұШӘ ЩҶЫҢШ§ШІ ЩҲШ§ШЁШіШӘЪҜЫҢвҖҢЩҮШ§ЫҢ ЫҢЪ© ШіШұЩҲЫҢШі ШЁЩҮ Ъ©Щ…Ъ© Lazy loadingВ» ЩӮШіЩ…ШӘ ШҙШҙЩ… ШЁШ§ Ш¬ЩҶШұЫҢЪ©вҖҢЩҮШ§ЫҢ ЩҶШ§Щ…ШӯШҜЩҲШҜ

ШҜШұ ЩӮШіЩ…ШӘ ШҙШҙЩ… ЩҶШӯЩҲЩҮвҖҢЫҢ ШӘШ№ШұЫҢЩҒ ЩҫЫҢШҙЩҶЫҢШ§ШІЩҮШ§ЫҢ ЩҲЩҮЩ„ЩҮ ШіШ§ШІЫҢ ШЁЩҮ ШӘШ§Ш®ЫҢШұ Ш§ЩҒШӘШ§ШҜЩҮ ШұШ§ ШЁШ§ Ш§ШіШӘЩҒШ§ШҜЩҮ Ш§ШІ Ъ©Щ„Ш§Ші Lazy ШЁШұШұШіЫҢ Ъ©ШұШҜЫҢЩ…: 
services.AddTransient<IOrderHandler, OrderHandlerLazy>();
services.AddTransient<IAccounting, Accounting>()
            .AddTransient(serviceProvider => new Lazy<IAccounting>(() => serviceProvider.GetRequiredService<IAccounting>()));
services.AddTransient<ISales, Sales>()
           .AddTransient(serviceProvider => new Lazy<ISales>(() => serviceProvider.GetRequiredService<ISales>()));
- ШҜШұ Ш§ЫҢЩҶШ¬Ш§ ШҜШұ Ш§ШЁШӘШҜШ§ ШӘЩ…Ш§Щ… ШіШұЩҲЫҢШівҖҢЩҮШ§ (ШӯШӘЫҢ ШўЩҶвҖҢЩҮШ§ЫҢЫҢ Ъ©ЩҮ ЩӮШұШ§Шұ Ш§ШіШӘ ШЁЩҮ ШөЩҲШұШӘ Lazy Ш§ШіШӘЩҒШ§ШҜЩҮ ШҙЩҲЩҶШҜ) ЫҢЪ©ШЁШ§Шұ ШЁЩҮ ШөЩҲШұШӘ Щ…ШӘШҜШ§ЩҲЩ„ЫҢ Щ…Ш№ШұЩҒЫҢ Щ…ЫҢвҖҢШҙЩҲЩҶШҜ.
- ШіЩҫШі ШіШұЩҲЫҢШівҖҢЩҮШ§ЫҢЫҢ Ъ©ЩҮ ЩӮШұШ§Шұ Ш§ШіШӘ ШЁЩҮ ШөЩҲШұШӘ Lazy ЩҶЫҢШІ ЩҲШ§Ъ©ШҙЫҢ ШҙЩҲЩҶШҜШҢ ШЁШ§Шұ ШҜЫҢЪҜШұ ШӘЩҲШіШ· ШұЩҲШҙ factory registration ШЁШ§ ЩҲЩҮЩ„ЩҮ ШіШ§ШІЫҢ new Lazy Ш§ШІ ЩҶЩҲШ№ ШіШұЩҲЫҢШі Щ…ШҜЩҶШёШұ ЩҲ ЩҒШұШ§ЩҮЩ… ШўЩҲШұШҜЩҶ ЩҫЫҢШ§ШҜЩҮ ШіШ§ШІЫҢ ШўЩҶ ШЁШ§ Ш§ШіШӘЩҒШ§ШҜЩҮ Ш§ШІ serviceProvider.GetRequiredServiceШҢ Щ…Ш¬ШҜШҜШ§ Щ…Ш№ШұЩҒЫҢ Ш®ЩҲШ§ЩҮЩҶШҜ ШҙШҜ.

Ш§ЪҜШұ ШЁЩҮ ШҙШұШ· ШҜЩҲЩ… ШҜЩӮШӘ Ъ©ЩҶЫҢШҜШҢ  <new Lazy<IAccounting ЩҲ <new Lazy<ISalesШҢ ШҜЩӮЫҢЩӮШ§ Щ…Ш§ЩҶЩҶШҜ ЩҮЩ…Ш§ЩҶ ШіШұЩҲЫҢШівҖҢЩҮШ§ЫҢ Ш¬ЩҶШұЫҢЪ© <IStore<User ЩҲ <IStore<Invoice ШӘШ№ШұЫҢЩҒ ШҙШҜЩҮвҖҢШ§ЩҶШҜ. ЫҢШ№ЩҶЫҢ ЩҶЫҢШ§ШІ Ш§ШіШӘ ШЁЩҮ Ш§ШІШ§ЫҢ ЩҮШұ T Щ…Щ…Ъ©ЩҶ ШҜШұ ШЁШұЩҶШ§Щ…ЩҮШҢ ЫҢЪ©ШЁШ§Шұ <new Lazy<T ШұШ§ ЩҶЫҢШІ ШЁЩҮ ШіЫҢШіШӘЩ… ШӘШІШұЫҢЩӮ ЩҲШ§ШЁШіШӘЪҜЫҢвҖҢЩҮШ§ Щ…Ш№ШұЩҒЫҢ Ъ©ШұШҜ. ШЁЩҶШ§ШЁШұШ§ЫҢЩҶ ШӘЩ…Ш§Щ… Ш§ЫҢЩҶ ШӘШ№ШұЫҢЩҒвҖҢЩҮШ§ЫҢ Ш§Ш¶Ш§ЩҒЫҢ ШұШ§ Щ…ЫҢвҖҢШӘЩҲШ§ЩҶ ШЁШ§ ЫҢЪ© ШіШ·Шұ Ш¬ЩҶШұЫҢЪ© ЩҶШ§Щ…ШӯШҜЩҲШҜ ШІЫҢШұ Ш¬Ш§ЫҢЪҜШІЫҢЩҶ ЩҲ Ш®Щ„Ш§ШөЩҮ Ъ©ШұШҜ:
services.AddTransient(typeof(Lazy<>), typeof(LazyFactory<>));
ЩҲ ЩҶЪ©ШӘЩҮвҖҢЫҢ Ш¬Ш§Щ„ШЁ ШўЩҶШҢ ЩҶШӯЩҲЩҮвҖҢЫҢ ШӘШ№ШұЫҢЩҒ ЩӮШіЩ…ШӘ factory Щ…ШӘШҜЩҮШ§ЫҢЫҢ Щ…Ш§ЩҶЩҶШҜ AddTransient ШҜШұ Ш§ЫҢЩҶШ¬Ш§ Ш§ШіШӘ Ъ©ЩҮ ШЁЩҮ ШөЩҲШұШӘ ШІЫҢШұ ЩӮШ§ШЁЩ„ ЩҫЫҢШ§ШҜЩҮ ШіШ§ШІЫҢ Ш§ШіШӘ:
public class LazyFactory<T> : Lazy<T> where T : class
{
    public LazyFactory(IServiceProvider provider)
        : base(() => provider.GetRequiredService<T>())
    {
    }
}
Щ…Ш·Ш§Щ„ШЁ Щ…ШҙШ§ШЁЩҮ
    Щ…Ш·Ш§Щ„ШЁЩ…ЩҮШ§ШұШӘвҖҢЩҮШ§ЫҢ ШӘШІШұЫҢЩӮ ЩҲШ§ШЁШіШӘЪҜЫҢвҖҢЩҮШ§ ШҜШұ ШЁШұЩҶШ§Щ…ЩҮвҖҢЩҮШ§ЫҢ NET Core. - ЩӮШіЩ…ШӘ ШҙШҙЩ… - ШҜШ®Ш§Щ„ШӘ ШҜШұ Щ…ШұШ§ШӯЩ„ ЩҲЩҮЩ„ЩҮ ШіШ§ШІЫҢ Ш§ШҙЫҢШ§ШЎ ШӘЩҲШіШ· IoC ContainerЩ…Ш·Ш§Щ„ШЁЩ…ЩҮШ§ШұШӘвҖҢЩҮШ§ЫҢ ШӘШІШұЫҢЩӮ ЩҲШ§ШЁШіШӘЪҜЫҢвҖҢЩҮШ§ ШҜШұ ШЁШұЩҶШ§Щ…ЩҮвҖҢЩҮШ§ЫҢ NET Core. - ЩӮШіЩ…ШӘ ЩҮШҙШӘЩ… - ШіШ§ШҜЩҮ ШіШ§ШІЫҢ Щ…Ш№ШұЩҒЫҢ ШіШұЩҲЫҢШівҖҢЩҮШ§ ШӘЩҲШіШ· ScrutorЩ…Ш·Ш§Щ„ШЁЩ…ЩҮШ§ШұШӘвҖҢЩҮШ§ЫҢ ШӘШІШұЫҢЩӮ ЩҲШ§ШЁШіШӘЪҜЫҢвҖҢЩҮШ§ ШҜШұ ШЁШұЩҶШ§Щ…ЩҮвҖҢЩҮШ§ЫҢ NET Core. - ЩӮШіЩ…ШӘ ЩҮЩҒШӘЩ… - Ъ©Ш§Шұ ШЁШ§ ШіШұЩҲЫҢШівҖҢЩҮШ§ЫҢ Щ…ШӘЩҒШ§ЩҲШӘЫҢ ШЁШ§ ЫҢЪ© Ш§Щ…Ш¶Ш§ШЎЩ…Ш·Ш§Щ„ШЁЩ…ЩҮШ§ШұШӘвҖҢЩҮШ§ЫҢ ШӘШІШұЫҢЩӮ ЩҲШ§ШЁШіШӘЪҜЫҢвҖҢЩҮШ§ ШҜШұ ШЁШұЩҶШ§Щ…ЩҮвҖҢЩҮШ§ЫҢ NET Core. - ЩӮШіЩ…ШӘ ЫҢШ§ШІШҜЩҮЩ… - ЩҫЫҢШ§ШҜЩҮ ШіШ§ШІЫҢ ЩҫЩҲЫҢШ§ЫҢ DecoratorЩҮШ§ ШӘЩҲШіШ· Castle.CoreЩ…Ш·Ш§Щ„ШЁЩ…ЩҮШ§ШұШӘвҖҢЩҮШ§ЫҢ ШӘШІШұЫҢЩӮ ЩҲШ§ШЁШіШӘЪҜЫҢвҖҢЩҮШ§ ШҜШұ ШЁШұЩҶШ§Щ…ЩҮвҖҢЩҮШ§ЫҢ NET Core. - ЩӮШіЩ…ШӘ ШҜЩҲЩ… - Ш§Щ„ЪҜЩҲЫҢ Service LocatorЩ…Ш·Ш§Щ„ШЁШӘШІШұЫҢЩӮ ЩҲШ§ШЁШіШӘЪҜЫҢвҖҢЩҮШ§ ШҜШұ ASP.NET Core - ШЁШ®Шҙ 6 - Implementation FactoryЩ…Ш·Ш§Щ„ШЁЩ…ЩӮШҜЩ…ЩҮвҖҢШ§ЫҢ ШЁШұ ШӘШІШұЫҢЩӮ ЩҲШ§ШЁШіШӘЪҜЫҢвҖҢЩҮШ§ ШҜШұASP.NET CoreЩ…Ш·Ш§Щ„ШЁ ШҜЩҲШұЩҮвҖҢЩҮШ§Lazy loading ШҜШұ ШӘШІШұЫҢЩӮ ЩҲШ§ШЁШіШӘЪҜЫҢвҖҢЩҮШ§ ШЁЩҮ Ъ©Щ…Ъ© StructureMapЩҶШёШұШ§ШӘ Щ…Ш·Ш§Щ„ШЁЩ…ЩҮШ§ШұШӘвҖҢЩҮШ§ЫҢ ШӘШІШұЫҢЩӮ ЩҲШ§ШЁШіШӘЪҜЫҢвҖҢЩҮШ§ ШҜШұ ШЁШұЩҶШ§Щ…ЩҮвҖҢЩҮШ§ЫҢ NET Core. - ЩӮШіЩ…ШӘ ШҙШҙЩ… - ШҜШ®Ш§Щ„ШӘ ШҜШұ Щ…ШұШ§ШӯЩ„ ЩҲЩҮЩ„ЩҮ ШіШ§ШІЫҢ Ш§ШҙЫҢШ§ШЎ ШӘЩҲШіШ· IoC ContainerЩҶШёШұШ§ШӘ Щ…Ш·Ш§Щ„ШЁШ§ШұШӘЩӮШ§ШЎ ШЁЩҮ ASP.NET Core 1.0 - ЩӮШіЩ…ШӘ 6 - ШіШұЩҲЫҢШівҖҢЩҮШ§ ЩҲ ШӘШІШұЫҢЩӮ ЩҲШ§ШЁШіШӘЪҜЫҢвҖҢЩҮШ§

ЩҶШёШұШ§ШӘ

  • ЩҲШӯЫҢШҜ ЩҶШөЫҢШұЫҢ ШҜШұ ЫұЫіЫ№Ыё/Ы°Ыө/ЫІЫ° ЫұЫІ:Ыұ
    ЫҢЪ© ЩҶЪ©ШӘЩҮвҖҢЫҢ ШӘЪ©Щ…ЫҢЩ„ЫҢ
    ШҜШұ ASP.NET Core ЩҲ Ъ©ЩҶШӘШұЩ„ШұЩҮШ§ЫҢ ШўЩҶ Щ…ЫҢвҖҢШӘЩҲШ§ЩҶ  В«Щ…Ш«Ш§Щ„ 2: ЩҲЩҮЩ„ЩҮ ШіШ§ШІЫҢ ШҜШұ ШөЩҲШұШӘ ЩҶЫҢШ§ШІ ЩҲШ§ШЁШіШӘЪҜЫҢвҖҢЩҮШ§ЫҢ ЫҢЪ© ШіШұЩҲЫҢШі ШЁЩҮ Ъ©Щ…Ъ© Lazy loadingВ» ШұШ§ ШЁЩҮ ШөЩҲШұШӘ ШІЫҢШұ ЩҶЫҢШІ ЩҫЫҢШ§ШҜЩҮ ШіШ§ШІЫҢ Ъ©ШұШҜ:
    public async Task<IActionResult> Index(
            [FromServices] MusicStoreContext dbContext,
            [FromServices] IMemoryCache cache)
    ШҜШұ Ш§ЫҢЩҶШ¬Ш§ ЩҲЫҢЪҳЪҜЫҢ FromServicesШҢ ШӘЩҶЩҮШ§ ШіШұЩҲЫҢШівҖҢЩҮШ§ЫҢ  Щ…ЩҲШұШҜ ЩҶЫҢШ§ШІ Ш§ЫҢЩҶ Ш§Ъ©ШҙЩҶ Щ…ШӘШҜ ШұШ§ ШҜШұ Ш§Ш®ШӘЫҢШ§Шұ ШўЩҶ ЩӮШұШ§Шұ Щ…ЫҢвҖҢШҜЩҮШҜ. ШЁЩҮ Ш§ЫҢЩҶ ШӘШұШӘЫҢШЁ ЩҶЫҢШ§ШІЫҢ ШЁЩҮ ШӘШІШұЫҢЩӮ ШўЩҶвҖҢЩҮШ§ ШЁЩҮ ШіШ§ШІЩҶШҜЩҮвҖҢЫҢ Ъ©ЩҶШӘШұЩ„Шұ ЩҶШ®ЩҲШ§ЩҮШҜ ШЁЩҲШҜ. ЩҮЩ…ЫҢЩҶ Щ…ШіШ§Щ„ЩҮШҢ ШЁЩҮ ШӘШ§Ш®ЫҢШұ Ш§ЩҒШӘШ§ШҜЩҶ ЩҲЩҮЩ„ЩҮ ШіШ§ШІЫҢ Ш§ЫҢЩҶ ШіШұЩҲЫҢШівҖҢЩҮШ§ ШұШ§ ШӘШ§ ШІЩ…Ш§ЩҶ Ш§ШіШӘЩҒШ§ШҜЩҮвҖҢЫҢ Ш§ШІ ШўЩҶвҖҢЩҮШ§ ШҜШұ Ш§ЫҢЩҶ Ш§Ъ©ШҙЩҶ Щ…ШӘШҜШҢ ШіШЁШЁ Ш®ЩҲШ§ЩҮШҜ ШҙШҜ.
