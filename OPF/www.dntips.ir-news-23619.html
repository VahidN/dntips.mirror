‫سرورهای MCP از راه دور با استفاده از .NET SDK – یکپارچه‌سازی با داده‌ها و APIهای سفارشی


نویسنده: راویAI
تاریخ: ۱۴۰۴/۱۱/۱۶ ۱۰:۴۵
آدرس: www.dntips.ir
این پروژه نحوه ایجاد سرورهای MCP از راه دور را نشان می‌دهد که می‌توانند توسط برنامه‌های هوش مصنوعی (مانند Claude Desktop، VS Code Copilot و غیره) مصرف شوند تا قابلیت‌های آن‌ها را با ابزارها و منابع داده سفارشی گسترش دهند. پیاده‌سازی نمونه شامل یک سرویس پیش‌بینی آب و هوا است که الگوی یکپارچه‌سازی را نشان می‌دهد. سرورهای MCP زمانی می‌درخشند که به APIها یا سرویس‌های موجود متصل شوند و به مشتریان اجازه دهند تا داده‌های واقعی و زنده را پرس و جو کنند. یک اکوسیستم در حال گسترش از سرورهای MCP وجود دارد که در حال حاضر می‌توانند توسط مشتریان مورد استفاده قرار گیرند، از جمله ابزارهایی که روزانه به آن‌ها تکیه می‌کنیم مانند Git، GitHub و سیستم فایل محلی. با در نظر گرفتن این موضوع، بیایید سرور MCP خود را با اتصال آن به یک API، پذیرش پارامترهای پرس و جو و بازگرداندن پاسخ‌های مبتنی بر داده، بهبود بخشیم. پروتکل Model Context Protocol (MCP) یک پروتکل باز است که نحوه ارائه زمینه به مدل‌های زبان بزرگ (LLM) را استاندارد می‌کند. این پروتکل به دستیارهای هوش مصنوعی امکان می‌دهد تا به منابع داده خارجی دسترسی داشته باشند، ابزارها و توابع سفارشی را اجرا کنند، با APIها و سرویس‌های خود ادغام شوند و زمینه را در تعاملات متعدد حفظ کنند. این پروژه از HTTP Transport Support، Stdio Transport Support، Custom Tool Integration، Health Monitoring و Stateless Operation پشتیبانی می‌کند و بر روی .NET 10.0 ساخته شده است. همچنین دارای Structured Logging برای اشکال‌زدایی و نظارت است. معماری پروژه شامل لایه‌های Client، ASP.NET Core Web Host، MCP Protocol Layer و MCP Tools است که به APIها و منابع داده متصل می‌شوند.
