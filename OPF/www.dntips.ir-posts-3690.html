→ вҖ«Ъ©Ш§Щ„ШЁШҜШҙЪ©Ш§ЩҒЫҢ ШӘШәЫҢЫҢШұШ§ШӘ ШЁШІШұЪҜ ШҜШұ SQLitePCLRaw 3.0
вҖ«Ш§ЩҒШІШ§ЫҢШҙ Ъ©Ш§ШұШ§ЫҢЫҢ ЩҲ ЩҫШ§ШіШ®вҖҢЪҜЩҲЫҢЫҢ ШЁШұЩҶШ§Щ…ЩҮвҖҢЩҮШ§ЫҢ Blazor WebAssembly ШЁШ§ Ш§ШіШӘЩҒШ§ШҜЩҮ Ш§ШІ Blazor Web Worker ШҜШұ ШҜШ§ШӘвҖҢЩҶШӘ 11 ←
Ш№ЩҶЩҲШ§ЩҶ:

вҖ«Ш§ШұШӘЩӮШ§ЫҢ ШіШ·Шӯ ШіШ§ШІЪҜШ§ШұЫҢ ЩҫЫҢШҙвҖҢЩҒШұШ¶ ШЁЩҮ SQL Server 2022 ШҜШұ EF Core 11


ЩҶЩҲЫҢШіЩҶШҜЩҮ: ЩҲШӯЫҢШҜ ЩҶШөЫҢШұЫҢ
ШӘШ§ШұЫҢШ®: ЫұЫҙЫ°Ыө/Ы°Ыҙ/Ы°Ыұ Ы°Ы№:ЫІЫө
ШўШҜШұШі: www.dntips.ir
ШҜШұ ЩҶШіШ®ЩҮ Ш¬ШҜЫҢШҜ EF Core 11ШҢ Ш§ШұШ§ШҰЩҮвҖҢШҜЩҮЩҶШҜЩҮ ЩҫШ§ЫҢЪҜШ§ЩҮ ШҜШ§ШҜЩҮ SQL Server (SQL Server Provider) ШЁЩҮ ШөЩҲШұШӘ ЩҫЫҢШҙвҖҢЩҒШұШ¶ Ш§ШІ Compatibility Level 160 (Щ…ШұШЁЩҲШ· ШЁЩҮ SQL Server 2022) Ш§ШіШӘЩҒШ§ШҜЩҮ Щ…ЫҢвҖҢЪ©ЩҶШҜ. Ш§ЫҢЩҶ ШӘШәЫҢЫҢШұ ШЁЩҮ Ш§ЫҢЩҶ Щ…Ш№ЩҶШ§ШіШӘ Ъ©ЩҮ EF Core ШҜШұ ШІЩ…Ш§ЩҶ ШӘЩҲЩ„ЫҢШҜ Ъ©ЩҲШҰШұЫҢвҖҢЩҮШ§ЫҢ SQLШҢ ЩҒШұШ¶ ШұШ§ ШЁШұ Ш§ЫҢЩҶ Щ…ЫҢвҖҢЪҜШ°Ш§ШұШҜ Ъ©ЩҮ ЩҫШ§ЫҢЪҜШ§ЩҮ ШҜШ§ШҜЩҮ ШҙЩ…Ш§ Ш§ШІ Ш¬ШҜЫҢШҜШӘШұЫҢЩҶ ЩҲЫҢЪҳЪҜЫҢвҖҢЩҮШ§ ЩҲ ШіЫҢЩҶШӘЪ©ШівҖҢЩҮШ§ЫҢ SQL Server 2022 ЩҫШҙШӘЫҢШЁШ§ЩҶЫҢ Щ…ЫҢвҖҢЪ©ЩҶШҜ. ШҜШұ ЩҶШӘЫҢШ¬ЩҮШҢ Ъ©ЩҲШҰШұЫҢвҖҢЩҮШ§ЫҢ ШЁЩҮЫҢЩҶЩҮвҖҢШӘШұ ЩҲ Щ…ШҜШұЩҶвҖҢШӘШұЫҢ ШЁШҜЩҲЩҶ ЩҶЫҢШ§ШІ ШЁЩҮ ШӘЩҶШёЫҢЩ…Ш§ШӘ Ш§Ш¶Ш§ЩҒЫҢ ШӘЩҲЩ„ЫҢШҜ Щ…ЫҢвҖҢШҙЩҲЩҶШҜ. Ш§Щ…Ш§ Ш§ЪҜШұ ЩҫШ§ЫҢЪҜШ§ЩҮ ШҜШ§ШҜЩҮ ШҙЩ…Ш§ ЩӮШҜЫҢЩ…ЫҢвҖҢШӘШұ Ш§ШІ ЩҶШіШ®ЩҮ 2022 ШЁШ§ШҙШҜШҢ Ш§ЫҢЩҶ ШӘШәЫҢЫҢШұ Щ…ЫҢвҖҢШӘЩҲШ§ЩҶШҜ ШЁШ§Ш№Ш« ШЁШұЩҲШІ Ш®Ш·Ш§ ШҜШұ Ш§Ш¬ШұШ§ЫҢ ШЁШұЩҶШ§Щ…ЩҮ ШҙЩҲШҜ.

ЩҶШӯЩҲЩҮ ШЁШ§ШІЪҜШұШҜШ§ЩҶШҜЩҶ ШӘЩҶШёЫҢЩ…Ш§ШӘ ШЁШұШ§ЫҢ ЩҶШіШ®ЩҮвҖҢЩҮШ§ЫҢ ЩӮШҜЫҢЩ…ЫҢвҖҢШӘШұ
Ш§ЪҜШұ ЩҫШұЩҲЪҳЩҮ ШҙЩ…Ш§ Ш§ШІ ЩҶШіШ®ЩҮвҖҢЩҮШ§ЫҢ ЩӮШҜЫҢЩ…ЫҢвҖҢШӘШұ SQL Server (Щ…Ш§ЩҶЩҶШҜ 2019ШҢ 2017 ЫҢШ§ ЩӮШҜЫҢЩ…ЫҢвҖҢШӘШұ) Ш§ШіШӘЩҒШ§ШҜЩҮ Щ…ЫҢвҖҢЪ©ЩҶШҜШҢ ШЁШ§ЫҢШҜ ШЁЩҮ ШөЩҲШұШӘ ШөШұЫҢШӯ ШіШ·Шӯ ШіШ§ШІЪҜШ§ШұЫҢ ШұШ§ ШҜШұ DbContext ШӘЩҶШёЫҢЩ… Ъ©ЩҶЫҢШҜ. ШЁЩҮ Ш№ЩҶЩҲШ§ЩҶ Щ…Ш«Ш§Щ„ШҢ ШЁШұШ§ЫҢ ШіШ§ШІЪҜШ§ШұЫҢ ШЁШ§ SQL Server 2019 (ШіШ·Шӯ 150):
optionsBuilder.UseSqlServer(
    connectionString,
    sqlServerOptions => sqlServerOptions.UseCompatibilityLevel(150)); // SQL Server 2019

Ш®Ш·Ш§ЩҮШ§ЫҢ Ш§ШӯШӘЩ…Ш§Щ„ЫҢ ШҜШұ ШөЩҲШұШӘ Ш№ШҜЩ… ШӘЩҶШёЫҢЩ… ЩҲ Ш§ШіШӘЩҒШ§ШҜЩҮ Ш§ШІ ЩҶШіШ®ЩҮвҖҢЩҮШ§ЫҢ ЩӮШҜЫҢЩ…ЫҢвҖҢШӘШұ
Ш§ЪҜШұ ШЁШұЩҶШ§Щ…ЩҮ Ш®ЩҲШҜ ШұШ§ ШЁЩҮ EF Core 11 Ш§ШұШӘЩӮШ§ ШҜЩҮЫҢШҜ Ш§Щ…Ш§ ЩҫШ§ЫҢЪҜШ§ЩҮ ШҜШ§ШҜЩҮ ШҙЩ…Ш§ SQL Server 2019 ЫҢШ§ ЩҫШ§ЫҢЫҢЩҶвҖҢШӘШұ ШЁШ§ШҙШҜ ЩҲ Ъ©ШҜ ШЁШ§Щ„Ш§ ШұШ§ Ш§Ш¶Ш§ЩҒЩҮ ЩҶЪ©ЩҶЫҢШҜШҢ ШҜШұ ШІЩ…Ш§ЩҶ Ш§Ш¬ШұШ§ЫҢ Ъ©ЩҲШҰШұЫҢвҖҢЩҮШ§ЫҢЫҢ Ъ©ЩҮ Ш§ШІ ЩӮШ§ШЁЩ„ЫҢШӘвҖҢЩҮШ§ЫҢ Ш¬ШҜЫҢШҜ Ш§ШіШӘЩҒШ§ШҜЩҮ Щ…ЫҢвҖҢЪ©ЩҶЩҶШҜ ШЁШ§ Ш®Ш·Ш§ЩҮШ§ЫҢ ШұШіЩ…ЫҢ ШіЩ…ШӘ SQL Server Щ…ЩҲШ§Ш¬ЩҮ Ш®ЩҲШ§ЩҮЫҢШҜ ШҙШҜ. Щ…ЩҮЩ…вҖҢШӘШұЫҢЩҶ Ш§ЫҢЩҶ Ш®Ш·Ш§ЩҮШ§ Ш№ШЁШ§ШұШӘЩҶШҜ Ш§ШІ:
- Ш®Ш·Ш§ЫҢ ШіЫҢЩҶШӘЪ©Ші ШЁШұШ§ЫҢ ШӘЩҲШ§ШЁШ№ Ш¬ШҜЫҢШҜ (Щ…Ш§ЩҶЩҶШҜ GREATEST ЩҲ LEAST):
EF Core 11 ШЁШұШ§ЫҢ ШӘШұШ¬Щ…ЩҮ ШЁШұШ®ЫҢ Щ…ШӘШҜЩҮШ§ЫҢ LINQ (Щ…Ш§ЩҶЩҶШҜ Math.Max ЫҢШ§ Math.Min) Ш§ШІ ШӘЩҲШ§ШЁШ№ Щ…ШҜШұЩҶ GREATEST ЩҲ LEAST Ш§ШіШӘЩҒШ§ШҜЩҮ Щ…ЫҢвҖҢЪ©ЩҶШҜ Ъ©ЩҮ ШҜШұ SQL Server 2022 Щ…Ш№ШұЩҒЫҢ ШҙШҜЩҮвҖҢШ§ЩҶШҜ. ШҜШұ ЩҶШіШ®ЩҮвҖҢЩҮШ§ЫҢ ЩӮШҜЫҢЩ…ЫҢвҖҢШӘШұ ШЁШ§ Ш§ЫҢЩҶ Ш®Ш·Ш§ Щ…ЩҲШ§Ш¬ЩҮ Щ…ЫҢвҖҢШҙЩҲЫҢШҜ: SqlException: 'GREATEST' is not a recognized built-in function name. ЫҢШ§ 'LEAST' is not a recognized built-in function name.

- Ш®Ш·Ш§ ШҜШұ ЩҫШұШҜШ§ШІШҙ JSON (ШӘШ§ШЁШ№ JSON_ARRAY):
ШҜШұ EF Core 11 ШӘШұШ¬Щ…ЩҮ Ъ©Ш§ШұЩҮШ§ЫҢ Щ…ШұШӘШЁШ· ШЁШ§ ШіШӘЩҲЩҶвҖҢЩҮШ§ЫҢ JSON ШЁЩҮШЁЩҲШҜ ЫҢШ§ЩҒШӘЩҮ ЩҲ Ш§ШІ ШӘЩҲШ§ШЁШ№ЫҢ Щ…Ш«Щ„ JSON_ARRAY Ш§ШіШӘЩҒШ§ШҜЩҮ Щ…ЫҢвҖҢШҙЩҲШҜ. ЩҶШіШ®ЩҮвҖҢЩҮШ§ЫҢ ЩӮШҜЫҢЩ…ЫҢвҖҢШӘШұ SQL Server Ш§ЫҢЩҶ ШӘШ§ШЁШ№ ШұШ§ ЩҶЩ…ЫҢвҖҢШҙЩҶШ§ШіЩҶШҜ: SqlException: 'JSON_ARRAY' is not a recognized built-in function name.

- Ш®Ш·Ш§ЫҢ Ш§ШіШӘЩҒШ§ШҜЩҮ Ш§ШІ ШіЫҢЩҶШӘЪ©Ші Ш¬ШҜЫҢШҜ IS DISTINCT FROM:
ШЁШұШ§ЫҢ Щ…ЩӮШ§ЫҢШіЩҮ Щ…ЩӮШ§ШҜЫҢШұ ШЁШ§ Ш§ШӯШӘШіШ§ШЁ ЩҲШ¶Ш№ЫҢШӘ NULL ШЁЩҲШҜЩҶШҢ EF Core 11 Щ…Щ…Ъ©ЩҶ Ш§ШіШӘ Ш§ШІ ШіЫҢЩҶШӘЪ©Ші Щ…ШҜШұЩҶ IS DISTINCT FROM ЫҢШ§ IS NOT DISTINCT FROM Ш§ШіШӘЩҒШ§ШҜЩҮ Ъ©ЩҶШҜ Ъ©ЩҮ ШҜШұ ЩҶШіШ®ЩҮвҖҢЩҮШ§ЫҢ ЩҫЫҢШҙ Ш§ШІ 2022 ЩҲШ¬ЩҲШҜ ЩҶШҜШ§ШұШҜ: SqlException: Incorrect syntax near 'DISTINCT'.

ЩҶЪ©Ш§ШӘ Ш§Ш¶Ш§ЩҒЩҮвҖҢШӘШұ ЩҲ ШЁШіЫҢШ§Шұ Щ…ЩҮЩ… Ъ©ЩҮ ШЁШ§ЫҢШҜ ШЁШҜШ§ЩҶЫҢШҜ
- ШӘШЈШ«ЫҢШұ Щ…ШіШӘЩӮЫҢЩ… ШЁШұ Ш№Щ…Щ„Ъ©ШұШҜ (Performance): ШӯШӘЫҢ Ш§ЪҜШұ Ш®Ш·Ш§ЫҢ ШіЫҢЩҶШӘЪ©Ші ШҜШұЫҢШ§ЩҒШӘ ЩҶЪ©ЩҶЫҢШҜШҢ ШӘЩҶШёЫҢЩ… ШҜШұШіШӘ Compatibility Level ШЁЩҮ ШЁШ§ШІЩҶЩҲЫҢШіЫҢ ШЁЩҮЫҢЩҶЩҮ Ъ©ЩҲШҰШұЫҢвҖҢЩҮШ§ (Query Rewrite) Ъ©Щ…Ъ© Щ…ЫҢвҖҢЪ©ЩҶШҜ. SQL Server 2022 ШҜШ§ШұШ§ЫҢ ЫҢЪ© ЩҲЫҢЪҳЪҜЫҢ ШЁЩҮ ЩҶШ§Щ… Parameter Sensitive Plan Optimization Ш§ШіШӘ Ъ©ЩҮ EF Core 11 ШЁШ§ ЩҒШұШ¶ ЩҒШ№Ш§Щ„ ШЁЩҲШҜЩҶ ШўЩҶШҢ Ъ©ЩҲШҰШұЫҢвҖҢЩҮШ§ ШұШ§ Ш§ШұШіШ§Щ„ Щ…ЫҢвҖҢЪ©ЩҶШҜ.

- ШЁШұШұШіЫҢ ШҜЫҢШӘШ§ШЁЫҢШівҖҢЩҮШ§ЫҢ Щ…ЫҢШІШЁШ§ЩҶЫҢ ШҙШҜЩҮ ШҜШұ Cloud:
  • Ш§ЪҜШұ Ш§ШІ Azure SQL Database ЫҢШ§ ШіШұЩҲЫҢШівҖҢЩҮШ§ЫҢ Ш§ШЁШұЫҢ ШҜШ§Ш®Щ„ЫҢ Ш§ШіШӘЩҒШ§ШҜЩҮ Щ…ЫҢвҖҢЪ©ЩҶЫҢШҜШҢ ШӯШӘЩ…Ш§ЩӢ ШЁШұШұШіЫҢ Ъ©ЩҶЫҢШҜ Ъ©ЩҮ ШіШ·Шӯ ШіШ§ШІЪҜШ§ШұЫҢ ШҜЫҢШӘШ§ШЁЫҢШі ШұЩҲЫҢ ЪҶЩҮ Ш№ШҜШҜЫҢ ШӘЩҶШёЫҢЩ… ШҙШҜЩҮ Ш§ШіШӘ. ЪҜШ§ЩҮЫҢ Ш§ЩҲЩӮШ§ШӘ Ш®ЩҲШҜЩҗ ШҜЫҢШӘШ§ШЁЫҢШі Ш§ШұШӘЩӮШ§ ЫҢШ§ЩҒШӘЩҮ Ш§Щ…Ш§ Compatibility Level ШўЩҶ ШҜШұ ШӘЩҶШёЫҢЩ…Ш§ШӘ ШҜЫҢШӘШ§ШЁЫҢШі ЩҮЩ…ЪҶЩҶШ§ЩҶ ШұЩҲЫҢ ЩҶШіШ®ЩҮвҖҢЩҮШ§ЫҢ ЩӮШҜЫҢЩ…ЫҢ (Щ…Ш«Щ„ 140 ЫҢШ§ 150) ШЁШ§ЩӮЫҢ Щ…Ш§ЩҶШҜЩҮ Ш§ШіШӘ.

- Ш§ШіШӘШұШ§ШӘЪҳЫҢ Щ…ЩҮШ§Ш¬ШұШӘ (Migration):
  • ШЁЩҮШӘШұЫҢЩҶ ШұЩҲШҙ Ш§ЫҢЩҶ Ш§ШіШӘ Ъ©ЩҮ ШҜШұ ШІЩ…Ш§ЩҶ Ш§ШұШӘЩӮШ§ ШЁЩҮ .NET 11 / EF Core 11ШҢ Ш§ШЁШӘШҜШ§ Ъ©ШҜ Ш®ЩҲШҜ ШұШ§ ШЁШ§ UseCompatibilityLevel ШұЩҲЫҢ ЩҶШіШ®ЩҮ ЩҒШ№Щ„ЫҢ ШҜЫҢШӘШ§ШЁЫҢШі Ш®ЩҲШҜ ЩӮЩҒЩ„ Ъ©ЩҶЫҢШҜ. ШіЩҫШі ЩҫШі Ш§ШІ Ш§ШұШӘЩӮШ§ЫҢ ШіШұЩҲШұ ЩҫШ§ЫҢЪҜШ§ЩҮ ШҜШ§ШҜЩҮ ШЁЩҮ ЩҶШіШ®ЩҮ 2022ШҢ Ш§ЫҢЩҶ Ш®Ш· Ъ©ШҜ ШұШ§ ШӯШ°ЩҒ Ъ©ЩҶЫҢШҜ ШӘШ§ Ш§ШІ Щ…ШІШ§ЫҢШ§ЫҢ ШіШұШ№ШӘ ЩҲ ШіЫҢЩҶШӘЪ©Ші Ш¬ШҜЫҢШҜ ШЁЩҮШұЩҮвҖҢЩ…ЩҶШҜ ШҙЩҲЫҢШҜ.
Щ…Ш·Ш§Щ„ШЁ Щ…ШҙШ§ШЁЩҮ
    Ш§ШҙШӘШұШ§Ъ©вҖҢЩҮШ§ШұШ§ЩҮЩҶЩ…Ш§ЫҢ ЪҜШ§Щ… ШЁЩҮ ЪҜШ§Щ… Ш§ШұШӘЩӮШ§ ШЁЩҮ SQL Server 2025Ш§ШҙШӘШұШ§Ъ©вҖҢЩҮШ§Ш§Ш¶Ш§ЩҒЩҮ ШҙШҜЩҶ IS DISTINCT FROM ШЁЩҮ SQL Server 2022Ш§ШҙШӘШұШ§Ъ©вҖҢЩҮШ§ШЁШұШұШіЫҢ ШұЩҲШҙ ЩҶШөШЁ SQL Server 2022Ш§ШҙШӘШұШ§Ъ©вҖҢЩҮШ§Ш§ЩҒШІШ§ЫҢШҙ ЫұЫёЫ° ШҜШұШөШҜЫҢ ШіШұШ№ШӘ Ш®ЩҲШ§ЩҶШҜЩҶ asynchronous ШҜШ§ШҜЩҮвҖҢЩҮШ§ЫҢ ШЁШІШұЪҜ ШЁШ§ Microsoft.Data.SqlClientШ§ШҙШӘШұШ§Ъ©вҖҢЩҮШ§ЩҒШ№Ш§Щ„ ШЁЩҲШҜЩҶ Query Store ШЁЩҮ ШөЩҲШұШӘ ЩҫЫҢШҙвҖҢЩҒШұШ¶ ШҜШұ SQL Server 2022Ш§ШҙШӘШұШ§Ъ©вҖҢЩҮШ§SQL Server 2022 Release Candidate 0 Щ…ЩҶШӘШҙШұ ШҙШҜЩ…ШіЫҢШұШұШ§ЩҮвҖҢЩҮШ§ШЁШұШұШіЫҢ ЩҲЫҢЪҳЪҜЫҢвҖҢвҖҢЩҮШ§ЫҢ Ш¬ШҜЫҢШҜ C# 15 ЩҲ NET 11.Щ…ШіЫҢШұШұШ§ЩҮвҖҢЩҮШ§SQL ServerЩ…Ш·Ш§Щ„ШЁЩҫШҙШӘЫҢШЁШ§ЩҶЫҢ Ш§ШІ ЩҶЩҲШ№ ШҜШ§ШҜЩҮ JSON ШҜШұ EF Core 10: ШӘШӯЩҲЩ„ЫҢ ШҜШұ Щ…ШҜЫҢШұЫҢШӘ ШҜШ§ШҜЩҮвҖҢЩҮШ§ЫҢ ШіШ§Ш®ШӘвҖҢЫҢШ§ЩҒШӘЩҮЩҶШёШұШ§ШӘ Щ…Ш·Ш§Щ„ШЁШӘЩҶШёЫҢЩ… ШҜШұШ¬ЩҮ ШіШ§ШІЪҜШ§ШұЫҢ ЫҢЪ© ШҜЫҢШӘШ§ШЁЫҢШі Ш§Ші Ъ©ЫҢЩҲШ§Щ„ ШіШұЩҲШұ
