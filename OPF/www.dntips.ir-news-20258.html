Щ…ЩӮШҜЩ…ЩҮ
ШЁЩҮЫҢЩҶЩҮвҖҢШіШ§ШІЫҢ Ш№Щ…Щ„Ъ©ШұШҜ (Performance Optimization) ЫҢЪ©ЫҢ Ш§ШІ Щ…ЩҮЩ…вҖҢШӘШұЫҢЩҶ ШҜШәШҜШәЩҮвҖҢЩҮШ§ЫҢ ШЁШұЩҶШ§Щ…ЩҮвҖҢЩҶЩҲЫҢШіШ§ЩҶШҢ ШЁЩҮвҖҢЩҲЫҢЪҳЩҮ ШҜШұ ЩҫШұЩҲЪҳЩҮвҖҢЩҮШ§ЫҢЫҢ Ш§ШіШӘ Ъ©ЩҮ ШЁШ§ ШӯШ¬Щ… ШЁШ§Щ„Ш§ЫҢ ШҜШ§ШҜЩҮ ЫҢШ§ ЩҶЫҢШ§ШІ ШЁЩҮ Ш§Ш¬ШұШ§ЫҢ ШіШұЫҢШ№ Щ…ЩҲШ§Ш¬ЩҮ ЩҮШіШӘЩҶШҜ. ШІШЁШ§ЩҶ ШЁШұЩҶШ§Щ…ЩҮвҖҢЩҶЩҲЫҢШіЫҢ #CШҢ Ш§Щ…Ъ©Ш§ЩҶШ§ШӘ Щ…ШӘШ№ШҜШҜЫҢ ШЁШұШ§ЫҢ Щ…ШҜЫҢШұЫҢШӘ ШӯШ§ЩҒШёЩҮ ЩҲ Ш§ЩҒШІШ§ЫҢШҙ Ъ©Ш§ШұШ§ЫҢЫҢ Ш§ШұШ§ШҰЩҮ Щ…ЫҢвҖҢШҜЩҮШҜ. ЫҢЪ©ЫҢ Ш§ШІ Ш§ЫҢЩҶ Ш§Щ…Ъ©Ш§ЩҶШ§ШӘ ЩҫЫҢШҙШұЩҒШӘЩҮШҢ ШіШ§Ш®ШӘШ§ШұЩҮШ§ЫҢ Щ…ШұШ¬Ш№ ЫҢШ§ ЩҮЩ…Ш§ЩҶ "ref structs" ЩҮШіШӘЩҶШҜ Ъ©ЩҮ Ш§ШІ ЩҶШіШ®ЩҮ 7.2 ШЁЩҮ Ш§ЫҢЩҶ ШІШЁШ§ЩҶ Ш§Ш¶Ш§ЩҒЩҮ ШҙШҜЩҶШҜ. Ш§ЫҢЩҶ ШіШ§Ш®ШӘШ§ШұЩҮШ§ ШЁЩҮ ШӘЩҲШіШ№ЩҮвҖҢШҜЩҮЩҶШҜЪҜШ§ЩҶ Ш§Ш¬Ш§ШІЩҮ Щ…ЫҢвҖҢШҜЩҮЩҶШҜ ШӘШ§ ШЁШ§ Ъ©ЩҶШӘШұЩ„ ШҜЩӮЫҢЩӮвҖҢШӘШұ ШЁШұ ШӯШ§ЩҒШёЩҮ ЩҲ Ъ©Ш§ЩҮШҙ ЩҒШҙШ§Шұ ШЁШұ Ш¬Щ…Ш№вҖҢШўЩҲШұЫҢ ШІШЁШ§Щ„ЩҮ (Garbage Collection)ШҢ Ш№Щ…Щ„Ъ©ШұШҜ ШЁШұЩҶШ§Щ…ЩҮвҖҢЩҮШ§ЫҢ Ш®ЩҲШҜ ШұШ§ ШЁЩҮШЁЩҲШҜ ШЁШЁШ®ШҙЩҶШҜ.
ЩҮШҜЩҒ Ш§ШІ Ш§ЫҢЩҶ Щ…ЩӮШ§Щ„ЩҮШҢ Ш§ШұШ§ШҰЩҮ ЫҢЪ© ШўЩ…ЩҲШІШҙ Ш¬Ш§Щ…Ш№ ЩҲ Ъ©Ш§ШұШЁШұШҜЫҢ ШҜШұШЁШ§ШұЩҮ ref structs Ш§ШіШӘ. ШҜШұ Ш§ЫҢЩҶ Щ…Ш·Щ„ШЁШҢ Ш§ШЁШӘШҜШ§ ШЁЩҮ Щ…ЩҒЩҮЩҲЩ… ЩҲ ЩҲЫҢЪҳЪҜЫҢвҖҢЩҮШ§ЫҢ Ш§ЫҢЩҶ ШіШ§Ш®ШӘШ§ШұЩҮШ§ ЩҫШұШҜШ§Ш®ШӘЩҮ Щ…ЫҢвҖҢШҙЩҲШҜШҢ ШіЩҫШі ШЁШ§ Щ…Ш«Ш§Щ„вҖҢЩҮШ§ЫҢ Ш№Щ…Щ„ЫҢ ЩҶШҙШ§ЩҶ Ш®ЩҲШ§ЩҮЫҢЩ… ШҜШ§ШҜ Ъ©ЩҮ ЪҶЪҜЩҲЩҶЩҮ Щ…ЫҢвҖҢШӘЩҲШ§ЩҶ Ш§ШІ ШўЩҶвҖҢЩҮШ§ ШҜШұ ШіЩҶШ§ШұЫҢЩҲЩҮШ§ЫҢ ЩҲШ§ЩӮШ№ЫҢ Ш§ШіШӘЩҒШ§ШҜЩҮ Ъ©ШұШҜ.

Ref Structs ЪҶЫҢШіШӘ ЩҲ ЪҶШұШ§ Щ…ЩҮЩ… Ш§ШіШӘШҹ
ШіШ§Ш®ШӘШ§ШұЩҮШ§ЫҢ Щ…ШұШ¬Ш№ ЫҢШ§ ref structs ЩҶЩҲШ№ЫҢ Ш§ШІ ШіШ§Ш®ШӘШ§ШұЩҮШ§ (Structs) ШҜШұ #C ЩҮШіШӘЩҶШҜ Ъ©ЩҮ ШЁЩҮвҖҢШөЩҲШұШӘ Ш®Ш§Шө ШЁШұШ§ЫҢ Ъ©Ш§ШұШ§ЫҢЫҢ ШЁШ§Щ„Ш§ Ш·ШұШ§ШӯЫҢ ШҙШҜЩҮвҖҢШ§ЩҶШҜ. ШЁШұШ®Щ„Ш§ЩҒ ШіШ§Ш®ШӘШ§ШұЩҮШ§ЫҢ Щ…Ш№Щ…ЩҲЩ„ЫҢ Ъ©ЩҮ Щ…ЫҢвҖҢШӘЩҲШ§ЩҶЩҶШҜ ШҜШұ Heap ШӘШ®ШөЫҢШө ЫҢШ§ШЁЩҶШҜШҢ ref structs ШӘЩҶЩҮШ§ ШҜШұ ЩҫШҙШӘЩҮ (Stack) Ш°Ш®ЫҢШұЩҮ Щ…ЫҢвҖҢШҙЩҲЩҶШҜ. Ш§ЫҢЩҶ ЩҲЫҢЪҳЪҜЫҢ ШЁШ§Ш№Ш« Щ…ЫҢвҖҢШҙЩҲШҜ Ъ©ЩҮ Ш§ШІ Ш№Щ…Щ„ЫҢШ§ШӘ ШӘШЁШҜЫҢЩ„ ЩҶЩҲШ№ (Boxing) Ш¬Щ„ЩҲЪҜЫҢШұЫҢ ШҙШҜЩҮ ЩҲ ЩҒШҙШ§Шұ ШЁШұ Ш¬Щ…Ш№вҖҢШўЩҲШұЫҢ ШІШЁШ§Щ„ЩҮ Ъ©Ш§ЩҮШҙ ЫҢШ§ШЁШҜ. ШЁЩҮ Ш№ШЁШ§ШұШӘ ШҜЫҢЪҜШұШҢ ref structs ШЁШұШ§ЫҢ ШіЩҶШ§ШұЫҢЩҲЩҮШ§ЫҢЫҢ Щ…ЩҶШ§ШіШЁ ЩҮШіШӘЩҶШҜ Ъ©ЩҮ ЩҶЫҢШ§ШІ ШЁЩҮ Щ…ШҜЫҢШұЫҢШӘ Щ…ШіШӘЩӮЫҢЩ… ШӯШ§ЩҒШёЩҮ ЩҲ Ш§Ш¬ШұШ§ЫҢ ШіШұЫҢШ№ ШҜШ§ШұЩҶШҜШҢ Щ…Ш§ЩҶЩҶШҜ ЩҫШұШҜШ§ШІШҙ ШҜШ§ШҜЩҮвҖҢЩҮШ§ЫҢ ШЁШІШұЪҜ ЫҢШ§ ШЁШұЩҶШ§Щ…ЩҮвҖҢЩҶЩҲЫҢШіЫҢ ШіШ·Шӯ ЩҫШ§ЫҢЫҢЩҶ.
ЫҢЪ©ЫҢ Ш§ШІ Щ…ШӯШҜЩҲШҜЫҢШӘвҖҢЩҮШ§ЫҢ Ъ©Щ„ЫҢШҜЫҢ ref structs Ш§ЫҢЩҶ Ш§ШіШӘ Ъ©ЩҮ ЩҶЩ…ЫҢвҖҢШӘЩҲШ§ЩҶЩҶШҜ ШЁЩҮвҖҢШ№ЩҶЩҲШ§ЩҶ Ш§Ш№Ш¶Ш§ЫҢ ЫҢЪ© Ъ©Щ„Ш§Ші ЫҢШ§ ШҜШұ Щ…Ш¬Щ…ЩҲШ№ЩҮвҖҢЩҮШ§ (Collections) Щ…Ш§ЩҶЩҶШҜ ШўШұШ§ЫҢЩҮвҖҢЩҮШ§ ЫҢШ§ Щ„ЫҢШіШӘвҖҢЩҮШ§ Ш§ШіШӘЩҒШ§ШҜЩҮ ШҙЩҲЩҶШҜ. Ш§ЫҢЩҶ Щ…ШӯШҜЩҲШҜЫҢШӘ ШЁЩҮ ШҜЩ„ЫҢЩ„ Щ…Ш§ЩҮЫҢШӘ stack-only ШўЩҶвҖҢЩҮШ§ШіШӘ Ъ©ЩҮ Ш§ШІ Ш§ЩҶШӘЩӮШ§Щ„ ЩҶШ§Ш®ЩҲШ§ШіШӘЩҮ ШЁЩҮ heap Ш¬Щ„ЩҲЪҜЫҢШұЫҢ Щ…ЫҢвҖҢЪ©ЩҶШҜ. ШЁШ§ Ш§ЫҢЩҶ ШӯШ§Щ„ШҢ ЩҮЩ…ЫҢЩҶ Щ…ШӯШҜЩҲШҜЫҢШӘвҖҢЩҮШ§ШҢ ШўЩҶвҖҢЩҮШ§ ШұШ§ ШЁЩҮ Ш§ШЁШІШ§ШұЫҢ Ш§ЫҢШҜЩҮвҖҢШўЩ„ ШЁШұШ§ЫҢ ШЁЩҮЫҢЩҶЩҮвҖҢШіШ§ШІЫҢ ШӘШЁШҜЫҢЩ„ Ъ©ШұШҜЩҮ Ш§ШіШӘ.

ЩҲЫҢЪҳЪҜЫҢвҖҢЩҮШ§ ЩҲ Щ…ШӯШҜЩҲШҜЫҢШӘвҖҢЩҮШ§
ШЁШұШ§ЫҢ ШҜШұЪ© ШЁЩҮШӘШұ ref structsШҢ ШЁЫҢШ§ЫҢЫҢШҜ ШЁЩҮ ЪҶЩҶШҜ ЩҲЫҢЪҳЪҜЫҢ ЩҲ Щ…ШӯШҜЩҲШҜЫҢШӘ Ш§ШөЩ„ЫҢ ШўЩҶвҖҢЩҮШ§ ЩҶЪҜШ§ЩҮ Ъ©ЩҶЫҢЩ…:
  1. Щ…ШӯШҜЩҲШҜЫҢШӘ ШӘШ®ШөЫҢШө ШӯШ§ЩҒШёЩҮ: ref structs ШӘЩҶЩҮШ§ ШҜШұ ЩҫШҙШӘЩҮ ШӘШ®ШөЫҢШө Щ…ЫҢвҖҢЫҢШ§ШЁЩҶШҜ ЩҲ ЩҶЩ…ЫҢвҖҢШӘЩҲШ§ЩҶЩҶШҜ ШЁЩҮ heap Щ…ЩҶШӘЩӮЩ„ ШҙЩҲЩҶШҜ. Ш§ЫҢЩҶ ЩҲЫҢЪҳЪҜЫҢ Ш§ШІ ШЁШіШӘЩҮвҖҢШЁЩҶШҜЫҢ Ш¬Щ„ЩҲЪҜЫҢШұЫҢ Щ…ЫҢвҖҢЪ©ЩҶШҜ ЩҲ Ъ©Ш§ШұШ§ЫҢЫҢ ШұШ§ Ш§ЩҒШІШ§ЫҢШҙ Щ…ЫҢвҖҢШҜЩҮШҜ.
  2. Ш№ШҜЩ… Ш§Щ…Ъ©Ш§ЩҶ ЩҫЫҢШ§ШҜЩҮвҖҢШіШ§ШІЫҢ ШұШ§ШЁШ·вҖҢЩҮШ§ (Interfaces): ШЁЩҮ ШҜЩ„ЫҢЩ„ Щ…ШӯШҜЩҲШҜЫҢШӘвҖҢЩҮШ§ЫҢ ШӯШ§ЩҒШёЩҮШҢ ref structs ЩҶЩ…ЫҢвҖҢШӘЩҲШ§ЩҶЩҶШҜ ШұШ§ШЁШ·вҖҢЩҮШ§ ШұШ§ ЩҫЫҢШ§ШҜЩҮвҖҢШіШ§ШІЫҢ Ъ©ЩҶЩҶШҜШҢ ШІЫҢШұШ§ Ш§ЫҢЩҶ Ъ©Ш§Шұ Щ…Щ…Ъ©ЩҶ Ш§ШіШӘ ШЁЩҮ ШЁШіШӘЩҮвҖҢШЁЩҶШҜЫҢ Щ…ЩҶШ¬Шұ ШҙЩҲШҜ.
  3. ЩҫШҙШӘЫҢШЁШ§ЩҶЫҢ Ш§ШІ Ш§ШұШ¬Ш§Ш№Ш§ШӘ (References): ref structs Щ…ЫҢвҖҢШӘЩҲШ§ЩҶЩҶШҜ ШҙШ§Щ…Щ„ ЩҒЫҢЩ„ШҜЩҮШ§ЫҢ ref ШЁШ§ШҙЩҶШҜ Ъ©ЩҮ ШЁЩҮ ШӘЩҲШіШ№ЩҮвҖҢШҜЩҮЩҶШҜЪҜШ§ЩҶ Ш§Ш¬Ш§ШІЩҮ Щ…ЫҢвҖҢШҜЩҮШҜ ШЁЩҮвҖҢШөЩҲШұШӘ Щ…ШіШӘЩӮЫҢЩ… ШЁШ§ ШӯШ§ЩҒШёЩҮ Ъ©Ш§Шұ Ъ©ЩҶЩҶШҜ.
  4. Ъ©Ш§ШұШЁШұШҜ ШҜШұ Щ…ШӘШҜЩҮШ§ЫҢ Щ…ШӯЩ„ЫҢ: Ш§ЫҢЩҶ ШіШ§Ш®ШӘШ§ШұЩҮШ§ Ш§ШәЩ„ШЁ ШҜШұ Щ…ШӘШҜЩҮШ§ЫҢ Щ…ШӯЩ„ЫҢ ЫҢШ§ ШіЩҶШ§ШұЫҢЩҲЩҮШ§ЫҢ Ъ©ЩҲШӘШ§ЩҮвҖҢЩ…ШҜШӘ Ш§ШіШӘЩҒШ§ШҜЩҮ Щ…ЫҢвҖҢШҙЩҲЩҶШҜШҢ Ш¬Ш§ЫҢЫҢ Ъ©ЩҮ Ш№Щ…Шұ Щ…ЩҒЫҢШҜ ШҜШ§ШҜЩҮвҖҢЩҮШ§ Щ…ШӯШҜЩҲШҜ Ш§ШіШӘ.

Щ…Ш«Ш§Щ„ ШіШ§ШҜЩҮ: Щ…ЩӮШ§ЫҢШіЩҮ Struct Щ…Ш№Щ…ЩҲЩ„ЫҢ ЩҲ Ref Struct
ШЁШұШ§ЫҢ ШҙШұЩҲШ№ШҢ ШЁЫҢШ§ЫҢЫҢШҜ ЫҢЪ© Щ…Ш«Ш§Щ„ ШіШ§ШҜЩҮ ШұШ§ ШЁШұШұШіЫҢ Ъ©ЩҶЫҢЩ… Ъ©ЩҮ ШӘЩҒШ§ЩҲШӘ ШЁЫҢЩҶ ЫҢЪ© ШіШ§Ш®ШӘШ§Шұ Щ…Ш№Щ…ЩҲЩ„ЫҢ ЩҲ ЫҢЪ© ref struct ШұШ§ ЩҶШҙШ§ЩҶ Щ…ЫҢвҖҢШҜЩҮШҜ. ЩҒШұШ¶ Ъ©ЩҶЫҢШҜ Щ…ЫҢвҖҢШ®ЩҲШ§ЩҮЫҢЩ… ЫҢЪ© ШіШ§Ш®ШӘШ§Шұ ШЁШұШ§ЫҢ Ш°Ш®ЫҢШұЩҮ Щ…Ш®ШӘШөШ§ШӘ ЩҶЩӮШ·ЩҮ ШҜШұ ЩҶШёШұ ШЁЪҜЫҢШұЫҢЩ…:
public struct Point
{
    public int X;
    public int Y;

    public Point(int x, int y)
    {
        X = x;
        Y = y;
    }
}

public ref struct RefPoint
{
    public ref int X;
    public ref int Y;

    public RefPoint(ref int x, ref int y)
    {
        X = ref x;
        Y = ref y;
    }
}
ШҜШұ Ш§ЫҢЩҶШ¬Ш§ШҢ Point ЫҢЪ© ШіШ§Ш®ШӘШ§Шұ Щ…Ш№Щ…ЩҲЩ„ЫҢ Ш§ШіШӘ Ъ©ЩҮ Щ…ЩӮШ§ШҜЫҢШұ ШұШ§ Ъ©ЩҫЫҢ Щ…ЫҢвҖҢЪ©ЩҶШҜШҢ ШҜШұ ШӯШ§Щ„ЫҢ Ъ©ЩҮ RefPoint Ш§ШІ Ш§ШұШ¬Ш§Ш№Ш§ШӘ Ш§ШіШӘЩҒШ§ШҜЩҮ Щ…ЫҢвҖҢЪ©ЩҶШҜ ЩҲ ШЁЩҮ ШҜШ§ШҜЩҮвҖҢЩҮШ§ЫҢ Ш§ШөЩ„ЫҢ Ш§ШҙШ§ШұЩҮ ШҜШ§ШұШҜ. ШӯШ§Щ„Ш§ ШЁЫҢШ§ЫҢЫҢШҜ ЩҶШӯЩҲЩҮ Ш§ШіШӘЩҒШ§ШҜЩҮ Ш§ШІ ШўЩҶвҖҢЩҮШ§ ШұШ§ ШЁШЁЫҢЩҶЫҢЩ…:
static void Main()
{
    int x = 10, y = 20;

    // Ш§ШіШӘЩҒШ§ШҜЩҮ Ш§ШІ Struct Щ…Ш№Щ…ЩҲЩ„ЫҢ
    Point p1 = new Point(x, y);
    p1.X = 15; // Ш§ЫҢЩҶ ШӘШәЫҢЫҢШұ ЩҒЩӮШ· ШҜШұ Ъ©ЩҫЫҢ Ш§Ш№Щ…Ш§Щ„ Щ…ЫҢвҖҢШҙЩҲШҜ
    Console.WriteLine($"x: {x}, p1.X: {p1.X}"); // Ш®ШұЩҲШ¬ЫҢ: x: 10, p1.X: 15

    // Ш§ШіШӘЩҒШ§ШҜЩҮ Ш§ШІ Ref Struct
    RefPoint p2 = new RefPoint(ref x, ref y);
    p2.X = 25; // Ш§ЫҢЩҶ ШӘШәЫҢЫҢШұ Щ…ШіШӘЩӮЫҢЩ…Ш§ЩӢ ШұЩҲЫҢ x Ш§Ш№Щ…Ш§Щ„ Щ…ЫҢвҖҢШҙЩҲШҜ
    Console.WriteLine($"x: {x}, p2.X: {p2.X}"); // Ш®ШұЩҲШ¬ЫҢ: x: 25, p2.X: 25
}
ШҜШұ Ш§ЫҢЩҶ Щ…Ш«Ш§Щ„ШҢ RefPoint ШЁЩҮвҖҢШөЩҲШұШӘ Щ…ШіШӘЩӮЫҢЩ… ШЁЩҮ Щ…ШӘШәЫҢШұЩҮШ§ЫҢ Ш§ШөЩ„ЫҢ Ш§ШұШ¬Ш§Ш№ Щ…ЫҢвҖҢШҜЩҮШҜ ЩҲ ШӘШәЫҢЫҢШұ ШўЩҶвҖҢЩҮШ§ ШұШ§ ШҜШұ Щ…ЩҶШЁШ№ Щ…ЩҶШ№Ъ©Ші Щ…ЫҢвҖҢЪ©ЩҶШҜШҢ ШҜШұ ШӯШ§Щ„ЫҢ Ъ©ЩҮ Point ЫҢЪ© Ъ©ЩҫЫҢ Щ…ШіШӘЩӮЩ„ Ш§ЫҢШ¬Ш§ШҜ Щ…ЫҢвҖҢЪ©ЩҶШҜ.

ШіЩҶШ§ШұЫҢЩҲЫҢ Ъ©Ш§ШұШЁШұШҜЫҢ: ШЁЩҮЫҢЩҶЩҮвҖҢШіШ§ШІЫҢ ШӯЩ„ЩӮЩҮвҖҢЩҮШ§ ШЁШ§Span<T>
ЫҢЪ©ЫҢ Ш§ШІ Ъ©Ш§ШұШЁШұШҜЩҮШ§ЫҢ ШұШ§ЫҢШ¬ ref structs ШҜШұ Ъ©Ш§Шұ ШЁШ§ Span<T> Ш§ШіШӘШҢ Ъ©ЩҮ Ш®ЩҲШҜ ЫҢЪ© ref struct Ш§ШіШӘ ЩҲ ШЁШұШ§ЫҢ ШҜШіШӘШұШіЫҢ Щ…ШіШӘЩӮЫҢЩ… ШЁЩҮ ШӯШ§ЩҒШёЩҮ Ш§ШіШӘЩҒШ§ШҜЩҮ Щ…ЫҢвҖҢШҙЩҲШҜ. ЩҒШұШ¶ Ъ©ЩҶЫҢШҜ Щ…ЫҢвҖҢШ®ЩҲШ§ЩҮЫҢЩ… Щ…Ш¬Щ…ЩҲШ№ Ш№ЩҶШ§ШөШұ ЫҢЪ© ШўШұШ§ЫҢЩҮ ШұШ§ Щ…ШӯШ§ШіШЁЩҮ Ъ©ЩҶЫҢЩ…:
static int SumArray(int[] numbers)
{
    int sum = 0;
    foreach (var num in numbers)
    {
        sum += num;
    }
    return sum;
}

static int SumSpan(Span<int> numbers)
{
    int sum = 0;
    for (int i = 0; i < numbers.Length; i++)
    {
        sum += numbers[i];
    }
    return sum;
}
ШҜШұ Ш§ЫҢЩҶШ¬Ш§ШҢ SumSpan Ш§ШІ Span<int> Ш§ШіШӘЩҒШ§ШҜЩҮ Щ…ЫҢвҖҢЪ©ЩҶШҜ Ъ©ЩҮ ШЁЩҮвҖҢШөЩҲШұШӘ Щ…ШіШӘЩӮЫҢЩ… ШЁЩҮ ШӯШ§ЩҒШёЩҮ ШўШұШ§ЫҢЩҮ ШҜШіШӘШұШіЫҢ ШҜШ§ШұШҜ ЩҲ ЩҶЫҢШ§ШІЫҢ ШЁЩҮ Ъ©ЩҫЫҢ Ъ©ШұШҜЩҶ ШҜШ§ШҜЩҮвҖҢЩҮШ§ ЩҶШҜШ§ШұШҜ. ШЁШұШ§ЫҢ Щ…ЩӮШ§ЫҢШіЩҮ Ш№Щ…Щ„Ъ©ШұШҜШҢ Щ…ЫҢвҖҢШӘЩҲШ§ЩҶЫҢЩ… ЫҢЪ© ШӘШіШӘ ШіШ§ШҜЩҮ ШЁЩҶЩҲЫҢШіЫҢЩ…:
static void Main()
{
    int[] numbers = Enumerable.Range(0, 1000000).ToArray();

    var watch = Stopwatch.StartNew();
    int sum1 = SumArray(numbers);
    watch.Stop();
    Console.WriteLine($"SumArray: {sum1}, Time: {watch.ElapsedMilliseconds} ms");

    watch = Stopwatch.StartNew();
    int sum2 = SumSpan(numbers.AsSpan());
    watch.Stop();
    Console.WriteLine($"SumSpan: {sum2}, Time: {watch.ElapsedMilliseconds} ms");
}
ЩҶШӘШ§ЫҢШ¬ Щ…Ш№Щ…ЩҲЩ„Ш§ЩӢ ЩҶШҙШ§ЩҶ Щ…ЫҢвҖҢШҜЩҮЩҶШҜ Ъ©ЩҮ SumSpan ШЁЩҮ ШҜЩ„ЫҢЩ„ ШҜШіШӘШұШіЫҢ Щ…ШіШӘЩӮЫҢЩ… ЩҲ Ш№ШҜЩ… ЩҶЫҢШ§ШІ ШЁЩҮ Ъ©ЩҫЫҢШҢ ШіШұЫҢШ№вҖҢШӘШұ Ш№Щ…Щ„ Щ…ЫҢвҖҢЪ©ЩҶШҜШҢ ШЁЩҮвҖҢЩҲЫҢЪҳЩҮ ШҜШұ ШўШұШ§ЫҢЩҮвҖҢЩҮШ§ЫҢ ШЁШІШұЪҜ.

ЩҶЪ©Ш§ШӘ ЩҫЫҢШҙШұЩҒШӘЩҮ: ШӘШұЪ©ЫҢШЁ ШЁШ§ Ref Returns ЩҲ In Parameters
ref structs Щ…ЫҢвҖҢШӘЩҲШ§ЩҶЩҶШҜ ШЁШ§ ЩӮШ§ШЁЩ„ЫҢШӘвҖҢЩҮШ§ЫҢ ШҜЫҢЪҜШұЫҢ Щ…Ш§ЩҶЩҶШҜ ШЁШ§ШІЪҜШҙШӘ Ш§ШұШ¬Ш§Ш№ (Ref Returns) ЩҲ ЩҫШ§ШұШ§Щ…ШӘШұЩҮШ§ЫҢ ЩҲШұЩҲШҜЫҢ (In Parameters) ШӘШұЪ©ЫҢШЁ ШҙЩҲЩҶШҜ ШӘШ§ Ъ©Ш§ШұШ§ЫҢЫҢ ШЁЫҢШҙШӘШұЫҢ Ш§ШұШ§ШҰЩҮ ШҜЩҮЩҶШҜ. ШЁЩҮ Щ…Ш«Ш§Щ„ ШІЫҢШұ ШӘЩҲШ¬ЩҮ Ъ©ЩҶЫҢШҜ:
public ref struct NumberManipulator
{
    private Span<int> _data;

    public NumberManipulator(Span<int> data)
    {
        _data = data;
    }

    public ref int GetElement(int index) => ref _data[index];

    public void DoubleElement(in int index)
    {
        _data[index] *= 2;
    }
}

static void Main()
{
    int[] numbers = [1, 2, 3, 4];
    var manipulator = new NumberManipulator(numbers.AsSpan());

    ref int element = ref manipulator.GetElement(1);
    element = 10; // ШӘШәЫҢЫҢШұ Щ…ШіШӘЩӮЫҢЩ… ШҜШұ ШўШұШ§ЫҢЩҮ
    manipulator.DoubleElement(2); // Ш¶ШұШЁШҜШұ 2 Ъ©ШұШҜЩҶ Ш№ЩҶШөШұ ШіЩҲЩ…

    Console.WriteLine(string.Join(", ", numbers)); // Ш®ШұЩҲШ¬ЫҢ: 1, 10, 6, 4
}
ШҜШұ Ш§ЫҢЩҶ Щ…Ш«Ш§Щ„ШҢ GetElement ЫҢЪ© Ш§ШұШ¬Ш§Ш№ ШЁЩҮ Ш№ЩҶШөШұ ШўШұШ§ЫҢЩҮ ШЁШұЩ…ЫҢвҖҢЪҜШұШҜШ§ЩҶШҜ ЩҲ DoubleElement ШЁШ§ Ш§ШіШӘЩҒШ§ШҜЩҮ Ш§ШІ ЩҫШ§ШұШ§Щ…ШӘШұ in Ш§ШІ Ъ©ЩҫЫҢ ШәЫҢШұШ¶ШұЩҲШұЫҢ Ш¬Щ„ЩҲЪҜЫҢШұЫҢ Щ…ЫҢвҖҢЪ©ЩҶШҜ.

ШЁЩҮШӘШұЫҢЩҶ ШұЩҲШҙвҖҢЩҮШ§ ЩҲ ШӘЩҲШөЫҢЩҮвҖҢЩҮШ§
ШЁШұШ§ЫҢ Ш§ШіШӘЩҒШ§ШҜЩҮ ШЁЩҮЫҢЩҶЩҮ Ш§ШІ ref structsШҢ ЪҶЩҶШҜ ЩҶЪ©ШӘЩҮ Ъ©Щ„ЫҢШҜЫҢ ШұШ§ ШҜШұ ЩҶШёШұ ШЁЪҜЫҢШұЫҢШҜ:
  • Ш§ШіШӘЩҒШ§ШҜЩҮ ШҜШұ ШіЩҶШ§ШұЫҢЩҲЩҮШ§ЫҢ Ъ©ЩҲШӘШ§ЩҮвҖҢЩ…ШҜШӘ: Ш§ШІ ref structs ШҜШұ Щ…ШӘШҜЩҮШ§ЫҢ Щ…ШӯЩ„ЫҢ ЫҢШ§ ШЁЩ„ЩҲЪ©вҖҢЩҮШ§ЫҢ Ъ©ШҜ Ъ©ЩҲШӘШ§ЩҮ Ш§ШіШӘЩҒШ§ШҜЩҮ Ъ©ЩҶЫҢШҜ ШӘШ§ Ш§ШІ ЩҫЫҢЪҶЫҢШҜЪҜЫҢ Щ…ШҜЫҢШұЫҢШӘ ШӯШ§ЩҒШёЩҮ Ш¬Щ„ЩҲЪҜЫҢШұЫҢ ШҙЩҲШҜ.
  • Ш§Ш¬ШӘЩҶШ§ШЁ Ш§ШІ Ш°Ш®ЫҢШұЩҮвҖҢШіШ§ШІЫҢ Ш·ЩҲЩ„Ш§ЩҶЫҢвҖҢЩ…ШҜШӘ: Ш§ЫҢЩҶ ШіШ§Ш®ШӘШ§ШұЩҮШ§ ШЁШұШ§ЫҢ ШҜШ§ШҜЩҮвҖҢЩҮШ§ЫҢ Щ…ЩҲЩӮШӘЫҢ Щ…ЩҶШ§ШіШЁ ЩҮШіШӘЩҶШҜ ЩҲ ЩҶШЁШ§ЫҢШҜ ШҜШұ ЩҒЫҢЩ„ШҜЩҮШ§ЫҢ Ъ©Щ„Ш§Ші Ш°Ш®ЫҢШұЩҮ ШҙЩҲЩҶШҜ.
  • ШӘШұЪ©ЫҢШЁ ШЁШ§ Ш§ШЁШІШ§ШұЩҮШ§ЫҢ Щ…ШҜШұЩҶ: Ш§ШІ ЩӮШ§ШЁЩ„ЫҢШӘвҖҢЩҮШ§ЫҢЫҢ Щ…Ш§ЩҶЩҶШҜ Span<T> ЩҲ Memory<T> ШҜШұ Ъ©ЩҶШ§Шұ ref structs ШЁЩҮШұЩҮ ШЁШЁШұЫҢШҜ ШӘШ§ ШӯШҜШ§Ъ©Ш«Шұ Ъ©Ш§ШұШ§ЫҢЫҢ ШұШ§ ШЁЩҮ ШҜШіШӘ ШўЩҲШұЫҢШҜ.

ЩҶШӘЫҢШ¬ЩҮвҖҢЪҜЫҢШұЫҢ
ШіШ§Ш®ШӘШ§ШұЩҮШ§ЫҢ Щ…ШұШ¬Ш№ (ref structs) ШҜШұ #C Ш§ШЁШІШ§ШұЫҢ ЩӮШҜШұШӘЩ…ЩҶШҜ ШЁШұШ§ЫҢ ШӘЩҲШіШ№ЩҮвҖҢШҜЩҮЩҶШҜЪҜШ§ЩҶЫҢ ЩҮШіШӘЩҶШҜ Ъ©ЩҮ ШЁЩҮ ШҜЩҶШЁШ§Щ„ ШЁЩҮЫҢЩҶЩҮвҖҢШіШ§ШІЫҢ Ш№Щ…Щ„Ъ©ШұШҜ ШЁШұЩҶШ§Щ…ЩҮвҖҢЩҮШ§ЫҢ Ш®ЩҲШҜ ШҜШұ ЪҶШ§ШұЪҶЩҲШЁ ШҜШ§ШӘвҖҢЩҶШӘ Щ…ЫҢвҖҢШЁШ§ШҙЩҶШҜ. Ш§ЫҢЩҶ ШіШ§Ш®ШӘШ§ШұЩҮШ§ ШЁШ§ Щ…ШӯШҜЩҲШҜ Ъ©ШұШҜЩҶ ШӘШ®ШөЫҢШө ШӯШ§ЩҒШёЩҮ ШЁЩҮ ЩҫШҙШӘЩҮ ЩҲ Ъ©Ш§ЩҮШҙ ЩҒШҙШ§Шұ ШЁШұ Ш¬Щ…Ш№вҖҢШўЩҲШұЫҢ ШІШЁШ§Щ„ЩҮШҢ ШЁЩҮ ШЁШұЩҶШ§Щ…ЩҮвҖҢЩҶЩҲЫҢШіШ§ЩҶ Ш§Ш¬Ш§ШІЩҮ Щ…ЫҢвҖҢШҜЩҮЩҶШҜ ШӘШ§ Ъ©ЩҶШӘШұЩ„ ШЁЫҢШҙШӘШұЫҢ ШЁШұ Ш§Ш¬ШұШ§ЫҢ Ъ©ШҜ ШҜШ§ШҙШӘЩҮ ШЁШ§ШҙЩҶШҜ. Ш§ШІ Щ…Ш«Ш§Щ„вҖҢЩҮШ§ЫҢ ШіШ§ШҜЩҮ Щ…Ш§ЩҶЩҶШҜ Щ…ШҜЫҢШұЫҢШӘ Щ…Ш®ШӘШөШ§ШӘ ШӘШ§ ШіЩҶШ§ШұЫҢЩҲЩҮШ§ЫҢ ЩҫЫҢЪҶЫҢШҜЩҮвҖҢШӘШұ Щ…Ш§ЩҶЩҶШҜ ЩҫШұШҜШ§ШІШҙ ШўШұШ§ЫҢЩҮвҖҢЩҮШ§ ШЁШ§ Span<T>ШҢ ref structs ЩҶШҙШ§ЩҶ ШҜШ§ШҜЩҮвҖҢШ§ЩҶШҜ Ъ©ЩҮ Щ…ЫҢвҖҢШӘЩҲШ§ЩҶЩҶШҜ Ъ©Ш§ШұШ§ЫҢЫҢ ШұШ§ ШЁЩҮвҖҢШ·ЩҲШұ ЩӮШ§ШЁЩ„вҖҢШӘЩҲШ¬ЩҮЫҢ Ш§ЩҒШІШ§ЫҢШҙ ШҜЩҮЩҶШҜ.


Щ…ШҙШ§ЩҮШҜЩҮ Щ…Ш·Щ„ШЁ Ш§ШөЩ„ЫҢ

    Ш§ШҙШӘШұШ§Ъ©вҖҢЩҮШ§Щ…Ш№ШұЩҒЫҢ ZLinq: Ъ©ШӘШ§ШЁШ®Ш§ЩҶЩҮ LINQ ШЁШ§ ШӘШ®ШөЫҢШө ШӯШ§ЩҒШёЩҮ ШөЩҒШұ ШЁШұШ§ЫҢ ШҜШ§ШӘвҖҢЩҶШӘШ§ШҙШӘШұШ§Ъ©вҖҢЩҮШ§ШўШұШ§ЫҢЩҮвҖҢЩҮШ§ЫҢ ШҜШұЩҲЩҶвҖҢШ®Ш·ЫҢ (Inline Arrays) ШҜШұ NET 9.Ш§ШҙШӘШұШ§Ъ©вҖҢЩҮШ§Щ…ШҜЫҢШұЫҢШӘ ШЁЩҮЫҢЩҶЩҮ ШӯЩ„ЩӮЩҮвҖҢЩҮШ§ЫҢ ШӘЪ©ШұШ§Шұ ШҜШұ NET.: ШұШ§ЩҮЩҶЩ…Ш§ЫҢ Ш¬Ш§Щ…Ш№ Ъ©Ш§Шұ ШЁШ§ Ш§ЩҶЩҲШ§Ш№ Щ…ЩӮШҜШ§ШұЫҢШ§ШҙШӘШұШ§Ъ©вҖҢЩҮШ§ШЁЩҮШЁЩҲШҜЩҮШ§ЫҢ NET 9.
