ШўШҜШұШі: www.dntips.ir
API Ъ©Ш§Шұ ШЁШ§ Ш§Щ…Ъ©Ш§ЩҶШ§ШӘ Reflection ЩҶЫҢШІ ШҜШұ NET Core. ЩҶШіШЁШӘ ШЁЩҮ ЩҶЪҜШ§ШұШҙ Ъ©Ш§Щ…Щ„ ШҜШ§ШӘ ЩҶШӘШҢ ШҜШ§ШұШ§ЫҢ ШӘШәЫҢЫҢШұШ§ШӘЫҢ Ш§ШіШӘ Ъ©ЩҮ ШҜШұ Ш§ШҜШ§Щ…ЩҮ Щ…ЩҮЩ…вҖҢШӘШұЫҢЩҶвҖҢЩҮШ§ЫҢ ШўЩҶвҖҢЩҮШ§ ШұШ§ ШЁШұШұШіЫҢ Ш®ЩҲШ§ЩҮЫҢЩ… Ъ©ШұШҜ.


ЩҫЫҢШҙЩҶЫҢШ§ШІЩҮШ§ЫҢ Ъ©Ш§Шұ ШЁШ§ Reflection ШҜШұ NET Core.

Ш§ШЁШӘШҜШ§ ЩҶЫҢШ§ШІ Ш§ШіШӘ Ш§ШіЩ…ШЁЩ„ЫҢ System.Reflection ШЁЩҮ ЩӮШіЩ…ШӘ ЩҲШ§ШЁШіШӘЪҜЫҢвҖҢЩҮШ§ЫҢ ЩҒШ§ЫҢЩ„ project.json Ш§Ш¶Ш§ЩҒЩҮ ШҙЩҲШҜ:
"dependencies": {
    "System.Reflection": "4.3.0"
},
Ш§Щ„ШЁШӘЩҮ Ш§ЪҜШұ ЩҲШ§ШЁШіШӘЪҜЫҢ ШҜЫҢЪҜШұЫҢ ШҜШұ ЩҒШ§ЫҢЩ„ project.jsonШҢ Ш§ЫҢЩҶ ЩҲШ§ШЁШіШӘЪҜЫҢ ШұШ§ ЩҫЫҢШҙШӘШұ Щ…ЩҲШұШҜ Ш§ШіШӘЩҒШ§ШҜЩҮ ЩӮШұШ§Шұ ШҜШ§ШҜЩҮ ШЁШ§ШҙШҜШҢ ЩҶЫҢШ§ШІЫҢ ШЁЩҮ Ш°Ъ©Шұ Щ…Ш¬ШҜШҜ ШўЩҶ ЩҶЫҢШіШӘ. ШЁШұШ§ЫҢ Ш§ЫҢЩҶ Щ…ЩҶШёЩҲШұ Щ…ЫҢвҖҢШӘЩҲШ§ЩҶ ЩҒШ§ЫҢЩ„ project.lock.json ШұШ§ Ш¬ЩҮШӘ ЫҢШ§ЩҒШӘЩҶ Щ…ШҜШ®Щ„ System.Reflection ШЁШұШұШіЫҢ Ъ©ШұШҜ. Ш§ЪҜШұ Ш§ЫҢЩҶ Щ…ШҜШ®Щ„ ШҜШұ ЩҒШ§ЫҢЩ„ lock Щ…ЩҲШ¬ЩҲШҜ ШЁЩҲШҜШҢ ШӘЩҲШіШ· ЩҫШұЩҲЪҳЩҮ ЩӮШ§ШЁЩ„ ШҜШіШӘШұШіЫҢ Ш§ШіШӘ ЩҲ ЩҶЫҢШ§ШІЫҢ ШЁЩҮ Ш°Ъ©Шұ ШөШұЫҢШӯ ШўЩҶ ЩҶШ®ЩҲШ§ЩҮШҜ ШЁЩҲШҜ.
ЩҫШі Ш§ШІ Ш§ЫҢЩҶ Щ…ЩҲШұШҜШҢ Ш°Ъ©Шұ ЩҒШ¶Ш§ЫҢ ЩҶШ§Щ… System.Reflection ШұШ§ ШҜШұ Ш§ШЁШӘШҜШ§ЫҢ Ъ©ШҜЩҮШ§ ЩҶЫҢШІ ЩҒШұШ§Щ…ЩҲШҙ ЩҶЪ©ЩҶЫҢШҜ. ЩҮЩ…ЫҢЩҶ Щ…ЩҲШұШҜ ШЁШіЫҢШ§ШұЫҢ Ш§ШІ Ш®Ш·Ш§ЩҮШ§ЫҢ Ш§ЩҶШӘЩӮШ§Щ„ Ъ©ШҜЩҮШ§ ШұШ§ ШЁШҜЩҲЩҶ ШӘШәЫҢЫҢШұЫҢ ШЁШұШ·ШұЩҒ Щ…ЫҢвҖҢЪ©ЩҶШҜ:
 using System.Reflection;


ЩҶЩҲШҙШӘЩҶ Ъ©ШҜЩҮШ§ЫҢ ШҙШұШ·ЫҢШҢ ШЁШұШ§ЫҢ Ъ©Ш§Щ…ЩҫШ§ЫҢЩ„ ЫҢЪ© Ъ©ШҜШҢ Щ…Ш®ШөЩҲШө ЩҒШұЫҢЩ… ЩҲШұЪ©вҖҢЩҮШ§ЫҢ Щ…Ш®ШӘЩ„ЩҒ

ШҜШұ Ъ©ШҜЩҮШ§ЫҢ ШіЫҢ ШҙШ§ШұЩҫ Щ…ЫҢвҖҢШӘЩҲШ§ЩҶ ШӘЩҲШіШ· if# ЩҲ else#ШҢ Ъ©Ш§Щ…ЩҫШ§ЫҢЩ„Шұ ШұШ§ Ш¬ЩҮШӘ Ш§ШіШӘЩҒШ§ШҜЩҮ ЫҢШ§ Ш№ШҜЩ… Ш§ШіШӘЩҒШ§ШҜЩҮвҖҢЫҢ Ъ©ШҜЩҮШ§ЫҢ ЩҶЩҲШҙШӘЩҮ ШҙШҜЩҮШҢ ШұШ§ЩҮЩҶЩ…Ш§ЫҢЫҢ Ъ©ШұШҜ. ШЁШұШ§ЫҢ Ш§ЫҢЩҶ Щ…ЩҶШёЩҲШұ ШҜШұ NET Core.ШҢ ШЁЩҮ ЩҒШ§ЫҢЩ„ project.json Щ…ШұШ§Ш¬Ш№ЩҮ Ъ©ШұШҜЩҮ ЩҲ Ш«Ш§ШЁШӘ ШҜЩ„Ш®ЩҲШ§ЩҮЫҢ ШұШ§ ШЁЩҮ ЩҶШ§Щ… COREFX ШЁЩҮ ЩӮШіЩ…ШӘ buildOptions Ш§Ш¶Ш§ЩҒЩҮ Щ…ЫҢвҖҢЪ©ЩҶЫҢЩ….
"buildOptions": {
   "define": [ "COREFX" ]
},
 ЩҫШі Ш§ШІ ШӘШ№ШұЫҢЩҒ Ш«Ш§ШЁШӘ COREFXШҢ Ш§Ъ©ЩҶЩҲЩҶ ЩҶШӯЩҲЩҮвҖҢЫҢ Ш§ШіШӘЩҒШ§ШҜЩҮвҖҢЫҢ Ш§ШІ ШўЩҶ ШЁЩҮ ШөЩҲШұШӘ Ш°ЫҢЩ„ Ш®ЩҲШ§ЩҮШҜ ШЁЩҲШҜ:
 #if COREFX
 // corefx codes
#else
 // other frameworks
#endif
Ш§ШІ Ш§ЫҢЩҶ ШұЩҲШҙ Щ…ЫҢвҖҢШӘЩҲШ§ЩҶ ШЁШұШ§ЫҢ Ъ©Ш§Щ…ЩҫШ§ЫҢЩ„ ЫҢЪ© ЩҒШ§ЫҢЩ„ Ъ©ШҜ csШҢ Ш¬ЩҮШӘ ЩҒШұЫҢЩ… ЩҲШұЪ©вҖҢЩҮШ§ЫҢ Щ…Ш®ШӘЩ„ЩҒ Ш§ШіШӘЩҒШ§ШҜЩҮ Ъ©ШұШҜ. ШҜШұ ШӯШ§Щ„ШӘ ЩҒЩҲЩӮШҢ Ъ©Ш§Щ…ЩҫШ§ЫҢЩ„Шұ ЩҒЩӮШ· ЩӮШіЩ…ШӘ if# ШұШ§ ЩҫШұШҜШ§ШІШҙ Ъ©ШұШҜЩҮ ЩҲ Ш§ШІ ЩӮШіЩ…ШӘ else# ШЁЩҮ Ш·ЩҲШұ Ъ©Ш§Щ…Щ„ ШөШұЩҒЩҶШёШұ Ш®ЩҲШ§ЩҮШҜ Ъ©ШұШҜ.


Ш§Ш¶Ш§ЩҒЩҮ ШҙШҜЩҶ Щ…ШӘШҜЩҮШ§ЫҢ GetMethodInfo ЩҲ GetTypeInfo

ШЁШіЫҢШ§ШұЫҢ Ш§ШІ Ш§Щ…Ъ©Ш§ЩҶШ§ШӘ System.Type ЩҶЪҜШ§ШұШҙ Ъ©Ш§Щ…Щ„ ШҜШ§ШӘ ЩҶШӘ ШЁЩҮ TypeInfo ШҜШұ NET Core. Щ…ЩҶШӘЩӮЩ„ ШҙШҜЩҮвҖҢШ§ЩҶШҜ. ШЁШұШ§ЫҢ Щ…Ш«Ш§Щ„ ШЁШ¬Ш§ЫҢ Ъ©ШҜ ЩҫЫҢШҙЫҢЩҶ
 var members = obj.GetType().GetMembers();
Ш§ЫҢЩҶШЁШ§Шұ ШЁШ§ЫҢШҜ ЩҶЩҲШҙШӘ
 var members = obj.GetType().GetTypeInfo().GetMembers();
ЩҲ ЫҢШ§ ЩҶЩ…ЩҲЩҶЩҮвҖҢЫҢ if/else ШҜШ§Шұ ШўЩҶ ШЁЩҮ ШөЩҲШұШӘ Ш°ЫҢЩ„ Ш®ЩҲШ§ЩҮШҜ ШЁЩҲШҜ:
 #if COREFX
 toType.GetTypeInfo().IsGenericType
#else
 toType.IsGenericType
#endif
ШҜШұ Ш§ЫҢЩҶШ¬Ш§ ЩӮШіЩ…ШӘ if Ш¬ЩҮШӘ NET Core. ЩҶЩҲШҙШӘЩҮ ШҙШҜЩҮвҖҢШ§ШіШӘ ЩҲ ЩӮШіЩ…ШӘ else ШЁШұШ§ЫҢ ЩҶЪҜШ§ШұШҙ Ъ©Ш§Щ…Щ„ ШҜШ§ШӘ ЩҶШӘ ЩӮШ§ШЁЩ„ Ш§ШіШӘЩҒШ§ШҜЩҮ Ш®ЩҲШ§ЩҮШҜ ШЁЩҲШҜ.


ЫҢШ§ЩҒШӘЩҶ Ш§ШіЩ…ШЁЩ„ЫҢ Ш¬Ш§ШұЫҢ

ШҜШұ NET Core. Щ…ШӘШҜ GetExecutingAssembly ШӯШ°ЩҒ ШҙШҜЩҮвҖҢШ§ШіШӘ ЩҲ ШЁШ§ЫҢШҜ Ш§ШЁШӘШҜШ§ ЩҶЩҲШ№ Ъ©Щ„Ш§Ші Ш¬Ш§ШұЫҢ ШұШ§ ЫҢШ§ЩҒШӘ ЩҲ ШіЩҫШі ШӘЩҲШіШ· GetTypeInfo ШЁЩҮ Assembly ШўЩҶ ШҜШіШӘШұШіЫҢ ЫҢШ§ЩҒШӘ:
 #if COREFX
 return typeof(MyClassName).GetTypeInfo().Assembly;
#else
 return System.Reflection.Assembly.GetExecutingAssembly();
#endif


Ш®Щ„Ш§ШөЩҮвҖҢЫҢ ШЁШӯШ«

ШЁШұШ§ЫҢ ШӘШЁШҜЫҢЩ„ Ъ©ШҜЩҮШ§ЫҢ Щ…ШЁШӘЩҶЫҢ ШЁШұ Reflection Щ…ЩҲШ¬ЩҲШҜШҢ Ш§ШЁШӘШҜШ§ Ш§ШІ ЩҲШ¬ЩҲШҜ ЩҲШ§ШЁШіШӘЪҜЫҢ System.Reflection Ш§Ш·Щ…ЫҢЩҶШ§ЩҶ ШӯШ§ШөЩ„ Ъ©ШұШҜЩҮШҢ ЩҒШ¶Ш§ЫҢ ЩҶШ§Щ… ШўЩҶвҖҢвҖҢШұШ§ Ш§Щ„ШӯШ§ЩӮ ЩҲ ШҜШұ Щ…ЩҲШ§ШұШҜ Ш¬ШІШҰЫҢ ЩҶЫҢШ§ШІ Ш§ШіШӘ Ш§ШІ Щ…ШӘШҜ GetTypeInfo ШЁШұШ§ЫҢ ШҜШіШӘШұШіЫҢ ШЁЩҮ Ш®ЩҲШ§Шө API ЩӮШЁЩ„ЫҢ Ш§ШіШӘЩҒШ§ШҜЩҮ Ъ©ШұШҜ.
Щ…Ш·Ш§Щ„ШЁ Щ…ШҙШ§ШЁЩҮ
    Ш§ШҙШӘШұШ§Ъ©вҖҢЩҮШ§ШӘШәЫҢЫҢШұШ§ШӘ Щ…ЩҮЩҶШҜШіЫҢ NET Core.Щ…Ш·Ш§Щ„ШЁШ§ШұШӘЩӮШ§ШЎ ШЁЩҮ ASP.NET Core 1.0 - ЩӮШіЩ…ШӘ 1 - NET Core. ЪҶЫҢШіШӘШҹЩ…Ш·Ш§Щ„ШЁШіШ§Ш®ШӘ ШЁШіШӘЩҮвҖҢЩҮШ§ЫҢ ЩҶЫҢЩҲЪҜШӘ Щ…Ш®ШөЩҲШө NET Core.Щ…Ш·Ш§Щ„ШЁШ§ШұШӘЩӮШ§ШЎ ШЁЩҮ ASP.NET Core 1.0 - ЩӮШіЩ…ШӘ 2 - ШЁШұШұШіЫҢ ШіШ§Ш®ШӘШ§Шұ Ш¬ШҜЫҢШҜ SolutionЩ…Ш·Ш§Щ„ШЁSpan ШҜШұ C# 7.2Щ…Ш·Ш§Щ„ШЁШ§ШіШӘЩҒШ§ШҜЩҮ Ш§ШІ EF7 ШЁШ§ ЩҫШ§ЫҢЪҜШ§ЩҮ ШҜШ§ШҜЩҮ SQLite ШӘШӯШӘ NET Core. ШЁЩҮ Ъ©Щ…Ъ© Visual Studio CodeЩ…Ш·Ш§Щ„ШЁШЁЩҮШЁЩҲШҜ Ъ©Ш§ШұШўЫҢЫҢ Reflection ШҜШұ ШҜШ§ШӘ ЩҶШӘ 7ЩҶШёШұШ§ШӘ Щ…Ш·Ш§Щ„ШЁШіШ§Ш®ШӘ ШЁШіШӘЩҮвҖҢЩҮШ§ЫҢ ЩҶЫҢЩҲЪҜШӘ Щ…Ш®ШөЩҲШө NET Core.ЩҶШёШұШ§ШӘ Щ…Ш·Ш§Щ„ШЁШЁШұШұШіЫҢ ШӘШәЫҢЫҢШұШ§ШӘ Reflection ШҜШұ NET Core.ЩҶШёШұШ§ШӘ Щ…Ш·Ш§Щ„ШЁШЁШұШұШіЫҢ ШӘШәЫҢЫҢШұШ§ШӘ Reflection ШҜШұ NET Core.

ЩҶШёШұШ§ШӘ

  • Ш®Щ„ЫҢЩ„ЫҢ ШҜШұ ЫұЫіЫ№Ыө/ЫұЫ°/Ы°Ыҙ ЫұЫё:ЫІЫІ
    ШЁШ§ ЩҒШұШ¶ Ш§ЫҢЩҶЪ©ЩҮ ЩӮШөШҜ ЩҶЩҲШҙШӘЩҶ Ъ©ШҜЫҢ ШұШ§ ШҜШ§ШұЫҢЩ… Ъ©ЩҮ ЩҮЩ… ШЁШұ ШұЩҲЫҢ dot net ЩҲ ЩҮЩ… dot net core Ъ©Ш§Шұ Ъ©ЩҶШҜ ЩҲ GetTypeInfo ШҜШұ ЩҮШұ ШҜЩҲ Щ…ЩҲШ¬ЩҲШҜ Ш§ШіШӘШҢ Ш§ШіШӘЩҒШ§ШҜЩҮвҖҢЫҢ ШөШұЩҒ Ш§ШІ ШўЩҶ ШЁЩҮШӘШұ Ш§ШІ Ш§ШіШӘЩҒШ§ШҜЩҮ Ш§ШІ Ъ©Ш§Щ…ЩҫШ§ЫҢЩ„Шұ ШҜШ§ЫҢШұЪ©ШӘЫҢЩҲ ЩҶЫҢШіШӘШҹ Ш®ШұЩҲШ¬ЫҢ TypeInfo ШЁЩҮ ШөЩҲШұШӘ Implicit ЩӮШ§ШЁЩ„ ШӘШЁШҜЫҢЩ„ ШҙШҜЩҶ ШЁЩҮ Type ЩҮШіШӘ ЩҲ Ш§ЫҢЩҶ ШұЩҲШҙ Щ…ШҙЪ©Щ„ ШіШ§ШІЪҜШ§ШұЫҢ ШЁШ§ Ъ©ШҜЩҮШ§ЫҢ Щ…ЩҲШ¬ЩҲШҜ ШұШ§ ЩҶЫҢШІ ЩҶШ®ЩҲШ§ЩҮШҜ ШҜШ§ШҙШӘ.
