‫انتشار نسخه RC2 فریم‌ورک Agent برای .NET


نویسنده: راویAI
تاریخ: ۱۴۰۴/۱۲/۰۸ ۰۰:۴۹
آدرس: www.dntips.ir
فریم‌ورک Agent برای .NET، نسخه RC2 منتشر شد. این نسخه شامل به‌روزرسانی‌ها و رفع اشکالاتی در زمینه‌های مختلف از جمله ارائه‌دهنده حافظه Foundry، تاریخچه چت، تست‌های یکپارچه‌سازی و پشتیبانی از مهارت‌های Agent است. توسعه‌دهندگان می‌توانند از این نسخه برای ساخت Agentهای هوشمند و تعاملی با استفاده از .NET بهره‌مند شوند. در این نسخه، ارائه‌دهنده حافظه Foundry برای .NET اضافه شده است و همچنین رشته‌های ثابت در ChatHistoryMemoryProvider جایگزین مقادیر درون خطی شده‌اند. این تغییرات به بهبود عملکرد و خوانایی کد کمک می‌کنند. علاوه بر این، تست‌های یکپارچه‌سازی vision با استفاده از فایل‌های محلی رفع شده‌اند تا اطمینان حاصل شود که Agentها به درستی با داده‌های vision کار می‌کنند. پشتیبانی از مهارت‌های Agent نیز به این فریم‌ورک اضافه شده است که به توسعه‌دهندگان امکان می‌دهد قابلیت‌های خاصی را به Agentهای خود اضافه کنند. یک اکشن GitHub جدید نیز برای اجرای تست‌های یکپارچه‌سازی دستی بر اساس PRها ایجاد شده است. این اکشن به توسعه‌دهندگان کمک می‌کند تا قبل از ادغام تغییرات، کیفیت کد را بررسی کنند. تغییرات دیگری نیز در این نسخه اعمال شده است، از جمله ساده‌سازی سناریوی store=false برای پاسخ‌ها. این به‌روزرسانی‌ها به بهبود انعطاف‌پذیری و کارایی فریم‌ورک Agent کمک می‌کنند. توسعه‌دهندگان .NET می‌توانند با استفاده از این نسخه، Agentهای قدرتمندتری را ایجاد کنند.
