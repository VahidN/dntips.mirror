ЫІ- ЩҮШұЪҶЩҶШҜ Ш§Щ…ШұЩҲШІЩҮ Ш§Ъ©Ш«Шұ Щ…ШұЩҲШұЪҜШұвҖҢЩҮШ§ pwa ШұШ§ ЩҫШҙШӘЫҢШЁШ§ЩҶЫҢ Щ…ЫҢЪ©ЩҶЩҶШҜШҢ ЩҲЩ„ЫҢ ШҜШұ ШЁШ№Ш¶ЫҢ Ш§ШІ Щ…ШұЩҲШұЪҜШұвҖҢЩҮШ§ ЩҲ Щ…ШұЩҲШұЪҜШұвҖҢЩҮШ§ЫҢ ШЁШ§ ЩҲШұЪҳЩҶ ЩҫШ§ЫҢЫҢЩҶШҢ pwa ЩҫШҙШӘЫҢШЁШ§ЩҶЫҢ ЩҶЩ…ЫҢШҙЩҲШҜ.
3- ЩҶЩ…ЫҢШӘЩҲШ§ЩҶ ЫҢЪ© ШЁШұЩҶШ§Щ…ЩҮвҖҢЫҢ Щ…ШЁШӘЩҶЫҢ ШЁШұ os ШұШ§ ЩҶЩҲШҙШӘ ЩҲ Щ…ШӯШҜЩҲШҜ ШЁЩҮ Щ…ШұЩҲШұЪҜШұ Щ…ЫҢШЁШ§ШҙШҜ


ЪҶЩҶШҜ ЩҶЪ©ШӘЩҮ ШҜШұШЁШ§ШұЩҮ pwa

1- ШЁШұШ§ЫҢ pwa  Щ„ШІЩҲЩ…ЫҢ ЩҶШҜШ§ШұШҜ ШӯШӘЩ…Ш§ Ш§ШІ ЩҒШұЫҢЩ… ЩҲШұЪ©вҖҢЩҮШ§ЫҢ spa Ш§ШіШӘЩҒШ§ШҜЩҮ Ъ©ЩҶЫҢШҜ. ШҙЩ…Ш§ Ш§ШІ ЩҮШұ ЩҒШұЫҢЩ… ЩҲШұЪ© Client Side Ш§ЫҢ Щ…ЫҢвҖҢШӘЩҲШ§ЩҶЫҢШҜ Ш§ШіШӘЩҒШ§ШҜЩҮ Ъ©ЩҶЫҢШҜ.
ЫІ- ЪҶЩҲЩҶ ШөЩҒШӯШ§ШӘ ШҙЩ…Ш§ ШҜШұ ЩҫЩ„ШӘЩҒШұЩ…вҖҢЩҮШ§ЫҢ Щ…Ш®ШӘЩ„ЩҒ ЩҲ ШЁШ§ ШөЩҒШӯЩҮ ЩҶЩ…Ш§ЫҢШҙвҖҢЩҮШ§ЫҢ Щ…Ш®ШӘЩ„ЩҒЫҢ Ш§Ш¬ШұШ§ Щ…ЫҢвҖҢШҙЩҲШҜШҢ ШЁШ§ЫҢШҜ ШөЩҒШӯШ§ШӘ ШЁЩҮ ШөЩҲШұШӘ Ъ©Ш§Щ…Щ„Ш§ responsive ШҙШҜЩҮ Ш·ШұШ§ШӯЫҢ ШҙЩҲЩҶШҜ.
Ыі- ШЁШ§ЫҢШҜ Ш§ШІ ЩҫШұЩҲШӘЪ©Щ„ https Ш§ШіШӘЩҒШ§ШҜЩҮ Ъ©ЩҶЫҢШҜ.

Щ…Ш§ ШҜШұ Ш§ЫҢЩҶ Щ…ЩӮШ§Щ„ЩҮ Ш§ШІ ЩҒШұЫҢЩ… ЩҲШұЪ© angular  Ш§ШіШӘЩҒШ§ШҜЩҮ  Ш®ЩҲШ§ЩҮЫҢЩ… Ъ©ШұШҜ.

ЩӮШЁЩ„ Ш§ШІ ШҙШұЩҲШ№ШҢ ШЁШ§ ШҙЫҢЩҲЩҮ Ъ©Ш§Шұ pwa ШўШҙЩҶШ§ Ш®ЩҲШ§ЩҮЫҢЩ… ШҙШҜ. ЫҢЪ©ЫҢ Ш§ШІ ЩӮШіЩ…ШӘвҖҢЩҮШ§ЫҢ Щ…ЩҮЩ…  Service Worker ШҢpwa Щ…ЫҢШЁШ§ШҙШҜ Ъ©ЩҮ Ш§ШІ Ш¬Щ…Щ„ЩҮ Ъ©Шҙ Ъ©ШұШҜЩҶШҢ notification ЩҒШұШіШӘШ§ШҜЩҶ ЩҲ Ш§Ш¬ШұШ§ЫҢ ЩҫШұШҜШ§ШІШҙвҖҢЩҮШ§ ШҜШұ ЩҫШі ШІЩ…ЫҢЩҶЩҮ ШұШ§ ШЁШұ Ш№ЩҮШҜЩҮ ШҜШ§ШұШҜ.


ШЁШ§ ШӘЩҲШ¬ЩҮ ШЁЩҮ ШҙЪ©Щ„ ШЁШ§Щ„Ш§ШҢ Service Worker Щ…Ш§ЩҶЩҶШҜ ЫҢЪ© Щ„Ш§ЫҢЩҮвҖҢЫҢ ЩҶШұЩ… Ш§ЩҒШІШ§ШұЫҢ Щ…Ш§ШЁЫҢЩҶ Ъ©Щ„Ш§ЫҢЩҶШӘ ЩҲ ШіШұЩҲШұ ЩӮШұШ§Шұ ШҜШ§ШұШҜ Ъ©ЩҮ ШҜШұШ®ЩҲШ§ШіШӘвҖҢЩҮШ§ЫҢ ШҜШ§ШҜЩҮ ШҙШҜЩҮ Ш§ШІ Ъ©Щ„Ш§ЫҢЩҶШӘ ШұШ§ ШҜШұ ШөЩҲШұШӘ Ш§ШӘШөШ§Щ„ ШЁЩҮ Ш§ЫҢЩҶШӘШұЩҶШӘШҢ Ш§ШұШіШ§Щ„ Ъ©ШұШҜЩҮ ЩҲ Щ…Ш¬ШҜШҜШ§ response ШұШ§ ШЁШұЪҜШҙШӘ Щ…ЫҢШҜЩҮШҜвҖҢ. Ш§ЪҜШұ ШЁЩҮ ЩҮШұ ШҜЩ„ЫҢЩ„ЫҢ Ш§ЫҢЩҶШӘШұЩҶШӘ ЩӮШ·Ш№ ШЁШ§ШҙШҜШҢ ШҜШұШ®ЩҲШ§ШіШӘ ШЁЩҮ ШөЩҲШұШӘ ШўЩҒЩ„Ш§ЫҢЩҶ ШЁЩҮ Ъ©Шҙ Щ…ШұЩҲШұЪҜШұ Ъ©ЩҮ ШӘЩҲШіШ· proxy ШіШ§Ш®ШӘЩҮ ШҙШҜЩҮ Ш§ШіШӘШҢ ЩҒШұШіШӘШ§ШҜЩҮ Щ…ЫҢШҙЩҲШҜ ЩҲ response ШұШ§ ШЁШұЪҜШҙШӘ Щ…ЫҢШҜЩҮШҜ.


ЪҶЩҶШҜ ЩҶЪ©ШӘЩҮ  ШҜШұ ШұШ§ШЁШ·ЩҮ ШЁШ§ Service Worker

- ЩҶШЁШ§ЫҢШҜ ШЁШұШ§ЫҢ  ЩҶЪҜЩҮШҜШ§ШұЫҢ ШҜШ§ШҜЩҮ global Ш§ШІ Service Worker Ш§ШіШӘЩҒШ§ШҜ Ъ©ШұШҜ. ШЁШұШ§ЫҢ Ш§ШіШӘЩҒШ§ШҜЩҮ Ш§ШІ ШҜШ§ШҜЩҮвҖҢЩҮШ§ЫҢ Global Щ…ЫҢШӘЩҲШ§ЩҶ Ш§ШІ Local Storage ЫҢШ§ IndexedDB Ш§ШіШӘЩҒШ§ШҜЩҮ Ъ©ШұШҜ.
- service worker  ШЁЩҮ dom ШҜШіШӘШұШіЫҢ ЩҶШҜШ§ШұЩҶШҜ.


 ШіЩҶШ§ШұЫҢЩҲ

 ЩӮШЁЩ„ Ш§ШІ ШҙШұЩҲШ№ШҢ ШіЩҶШ§ШұЫҢЩҲЫҢ ЩҫШұЩҲЪҳЩҮ ШұШ§ ШӘШҙШұЫҢШӯ Ш®ЩҲШ§ЩҮЫҢЩ… Ъ©ШұШҜ. ШұЪ©ЩҶ Ш§ШөЩ„ЫҢ ЫҢЪ© ШЁШұЩҶШ§Щ…ЩҮвҖҢЫҢ ЩҲШЁШҢ UI ШўЩҶ Щ…ЫҢШЁШ§ШҙШҜ ЩҲ ЩӮШөШҜ ШҜШ§ШұЫҢЩ… Ъ©Ш§ШұШЁШұШ§ЩҶ ШұШ§ Щ…ШӘЩҲШ¬ЩҮ Ъ©Ш§Шұ ШЁШ§ ЫҢЪ© ШөЩҒШӯЩҮвҖҢЫҢ ЩҲШЁЫҢ ЩҶЪ©ЩҶЫҢЩ…. ЩӮШ§Щ„ШЁЫҢ Ъ©ЩҮ Щ…Ш§ ШҜШұ Ш§ЫҢЩҶ Щ…Ш«Ш§Щ„ ШҜШұ ЩҶШёШұ ЪҜШұЩҒШӘЫҢЩ… Ш®ЫҢЩ„ЫҢ ШҙШЁЫҢЩҮ ШЁЩҮ ЫҢЪ© Ш§ЩҫЩ„ЫҢЪ©ЫҢШҙЩҶ ЩҫЩ„ШӘЩҒШұЩ… Ш§ЩҶШҜШұЩҲЫҢШҜЫҢ Щ…ЫҢШЁШ§ШҙШҜ. ЫҢЪ© Ъ©Ш§ШұШЁШұ ШЁШ§ Ъ©ШҙЫҢШҜЩҶ Щ…ЩҶЩҲЫҢ Ъ©ШҙЩҲЫҢЫҢ Щ…ЫҢШӘЩҲШ§ЩҶШҜ ЪҜШІЫҢЩҶЩҮвҖҢЩҮШ§ЫҢ Ш®ЩҲШҜ ШұШ§ Ш§ЩҶШӘШ®Ш§ШЁ ЩҶЩ…Ш§ЫҢЩҶШҜ. Ш§ЩҲЩ„ЫҢЩҶ ЪҜШІЫҢЩҶЩҮвҖҢШ§ЫҢ Ъ©ЩҮ ЩӮШөШҜ ЩҫЫҢШ§ШҜЩҮ ШіШ§ШІЫҢ ШўЩҶ ШұШ§ ШҜШ§ШұЫҢЩ…ШҢ Ш«ШЁШӘ Ъ©Ш§ШұШЁШұШ§ЩҶ Щ…ЫҢШЁШ§ШҙШҜ. ШЁШ№ШҜ Ш§ШІ Ш«ШЁШӘ Ъ©Ш§ШұШЁШұШ§ЩҶ ШҜШұ ЫҢЪ© Component Ш¬ШҜШ§ШҢ Ъ©Ш§ШұШЁШұШ§ЩҶ ШұШ§ ШҜШұ ЫҢЪ© Ш¬ШҜЩҲЩ„ ЩҶЩ…Ш§ЫҢШҙ Ш®ЩҲШ§ЩҮЫҢЩ… ШҜШ§ШҜ.


 ЩӮШЁЩ„ Ш§ШІ ШҙШұЩҲШ№ Щ„Ш§ШІЩ… Ш§ШіШӘ ШЁЩҮ ЪҶЩҶШҜ ЩҶЪ©ШӘЩҮ ШІЫҢШұ ШӘЩҲШ¬ЩҮ Ъ©ШұШҜ

1-  ШіШұЩҲЫҢШі crud ШұШ§ ШЁЩҮ ШөЩҲШұШӘ Ъ©Ш§Щ…Щ„ ШҜШұ ЩҫШұЩҲЪҳЩҮ ЩӮШұШ§Шұ Ш®ЩҲШ§ЩҮЫҢЩ… ШҜШ§ШҜШҢ ЩҲЩ„ЫҢ ЪҶЩҲЩҶ Ш§ШІ ШӯЩҲШөЩ„ЩҮвҖҢЫҢ Щ…ЩӮШ§Щ„ЩҮ Ш®Ш§ШұШ¬ Ш§ШіШӘШҢ ЩҒЩӮШ· Ш«ШЁШӘ Ъ©Ш§ШұШЁШұШ§ЩҶ ЩҲ ЩҶЩ…Ш§ЫҢШҙ Ъ©Ш§ШұШЁШұШ§ЩҶ ШұШ§ ЩҫЫҢШ§ШҜЩҮ ШіШ§ШІЫҢ Ш®ЩҲШ§ЩҮЫҢЩ… Ъ©ШұШҜ.
2 - ЩҮШұ Ш§ЩҫЩ„ЫҢЪ©ЫҢШҙЩҶЫҢ ЩӮШ·Ш№Ш§ ЩҶЫҢШ§ШІ ШЁЩҮ ЫҢЪ© ЩҲШЁ ШіШұЩҲЫҢШі ШҜШ§ШұШҜ Ъ©ЩҮ ЩҲШ§ШіШ· ШЁЫҢЩҶ ШҜЫҢШӘШ§ШЁЫҢШі ЩҲ Ш§ЩҫЩ„ЫҢЪ©ЫҢШҙЩҶ ШЁШ§ШҙШҜ. ЩҲЩ„ЫҢ Щ…Ш§ ЪҶЩҲЩҶ ШҜШұ Ш§ЫҢЩҶ Щ…Ш«Ш§Щ„ ШЁЩҮ Ш№ЩҶЩҲШ§ЩҶ front end Ъ©Ш§Шұ ЩӮШөШҜ Ш·ШұШ§ШӯЫҢ  Ш§ЩҫЩ„ЫҢЪ©ЫҢШҙЩҶ ШұШ§ ШҜШ§ШұЫҢЩ…ШҢ ШЁШұШ§ЫҢ ШӯШ°ЩҒ back end Ш§ШІ firebase Ш§ШіШӘЩҒШ§ШҜЩҮ Ш®ЩҲШ§ЩҮЫҢЩ… Ъ©ШұШҜ ЩҲ Щ…ШіШӘЩӮЫҢЩ…Ш§ Ш§ШІ Ш§ЩҶЪҜЩҲЩ„Ш§Шұ ШЁЩҮ ШҜЫҢШӘШ§ШЁЫҢШі firebase Ъ©ЩҲШҰШұЫҢ Ш®ЩҲШ§ЩҮЫҢЩ… ШІШҜ.


 ШҙШұЩҲШ№ ШЁЩҮ Ъ©Ш§Шұ

ЩҫЫҢШҙ ЩҶЫҢШ§ШІвҖҢЩҮШ§ЫҢ ЫҢЪ© ЩҫШұЩҲЪҳЩҮвҖҢЫҢ Ш§ЩҶЪҜЫҢЩҲЩ„Ш§ШұЫҢ ШұШ§ ШЁШұ ШұЩҲЫҢ ШіЫҢШіШӘЩ… Ш®ЩҲШҜ ЩҒШұШ§ЩҮЩ… Ъ©ЩҶЫҢШҜ. Щ…Ш§ ШҜШұ Ш§ЫҢЩҶ Щ…Ш«Ш§Щ„ Ш§ШІ ЫҢЪ© template ШўЩ…Ш§ШҜЩҮ Ш§ЩҶЪҜЫҢЩҲЩ„Ш§ШұЫҢ Ш§ШіШӘЩҒШ§ШҜЩҮ Ш®ЩҲШ§ЩҮЫҢЩ… Ъ©ШұШҜ. ЩҫШі ШЁШұШ§ЫҢ Ш§ЫҢЩҶЪ©ЩҮ ШЁШ§ Ш¬ШІШҰЫҢШ§ШӘ ЩҲ Ш·ШұШ§ШӯЫҢ ui ШҜШұЪҜЫҢШұ ЩҶШҙЩҲЫҢЩ…ШҢ Ш§ШІ Щ„ЫҢЩҶЪ© github ЩҫШұЩҲЪҳЩҮ ШұШ§ ШҜШұЫҢШ§ЩҒШӘ Ъ©ЩҶЫҢШҜ.
ШіЩҫШі ЩҲШ§ШұШҜ root ЩҫШұЩҲЪҳЩҮ ШҙЩҲЫҢШҜ ЩҲ ШЁШ§ ШҜШіШӘЩҲШұ ШІЫҢШұ ЩҫЪ©ЫҢШ¬вҖҢЩҮШ§ЫҢ ЩҫШұЩҲЪҳЩҮ ШұШ§ ЩҶШөШЁ Ъ©ЩҶЫҢШҜ:
 npm install
ЩҫШі Ш§ШІ ЩҶШөШЁ ЩҫЪ©ЫҢШ¬вҖҢЩҮШ§ШҢ ШЁШ§ ШҜШіШӘЩҲШұ ng serveШҢ ЩҫШұЩҲЪҳЩҮ ШұШ§ Ш§Ш¬ШұШ§ Ъ©ЩҶЫҢШҜ.

ЩҫШі Ш§ШІ Ш§Ш¬ШұШ§ ШЁШ§ЫҢШҜ Ш®ШұЩҲШ¬ЫҢ ШЁШ§Щ„Ш§ ШұШ§ ШҜШ§ШҙШӘЩҮ ШЁШ§ШҙЫҢШҜ. Щ…ЫҢвҖҢШ®ЩҲШ§ЩҮЫҢЩ… ЫҢЪ©ЫҢ Ш§ШІ ЩӮШ§ШЁЩ„ЫҢШӘвҖҢЩҮШ§ЫҢ pwa ШұШ§ ШЁШұШұШіЫҢ Ъ©ЩҶЫҢЩ…. ШЁШұ ШұЩҲЫҢ ШөЩҒШӯЩҮ Ъ©Щ„ЫҢЪ© ШұШ§ШіШӘ Ъ©ШұШҜЩҮ ЩҲ ЩҲШ§ШұШҜ inspect ШҙЩҲЫҢШҜ. ЩҲШ§ШұШҜ ШӘШЁ Application  ШҙЩҲЫҢШҜ ЩҲ Service Worker ШұШ§ Ш§ЩҶШӘШ®Ш§ШЁ Ъ©ЩҶЫҢШҜ.
 


ЩӮШЁЩ„ Ш§ШІ Ш§ЫҢЩҶЪ©ЩҮ Ъ©Ш§ШұЫҢ ШұШ§ Ш§ЩҶШ¬Ш§Щ… ШҜЩҮЫҢШҜШҢ ЪҶЩҶШҜ ШЁШ§Шұ ШөЩҒШӯЩҮ ШұШ§ refresh Ъ©ЩҶЫҢШҜ! ШөЩҒШӯЩҮ ШЁШҜЩҲЩҶ ЩҮЫҢЪҶ ШӘШәЫҢЫҢШұЫҢ refresh Щ…ЫҢШҙЩҲШҜ. Ш§ЫҢЩҶШЁШ§Шұ ЪҜШІЫҢЩҶЩҮвҖҢЫҢ offline ШұШ§ ЩҒШ№Ш§Щ„ Ъ©ЩҶЫҢШҜ ЩҲ Щ…Ш¬ШҜШҜШ§ ШөЩҒШӯЩҮ ШұШ§ refresh Ъ©ЩҶЫҢШҜ.
Ш§ЫҢЩҶ ШЁШ§Шұ ШөЩҒШӯЩҮ No internet ШЁЩҮ Щ…Ш№ЩҶЫҢ ЩӮШ·Ш№ ШЁЩҲШҜЩҶ Ш§ЫҢЩҶШӘШұЩҶШӘ ЩҶЩ…Ш§ЫҢШҙ ШҜШ§ШҜЩҮ Щ…ЫҢШҙЩҲШҜ ЩҲ ШҙЩ…Ш§ ШҜЫҢЪҜШұ ЩҶЩ…ЫҢвҖҢШӘЩҲШ§ЩҶЫҢШҜ ШЁШұ ШұЩҲЫҢ Ш§ЩҫЩ„ЫҢЪ©ЫҢШҙЩҶ Ш®ЩҲШҜ ЩҒШ№Ш§Щ„ЫҢШӘЫҢ ШҜШ§ШҙШӘЩҮ ШЁШ§ШҙЫҢШҜ!

 Manifest.json : Ш§ЪҜШұ Щ…ШӯШӘЩҲЫҢШ§ШӘ ЩҒШ§ЫҢЩ„ ШұШ§ Щ…ШҙШ§ЩҮШҜЩҮ Ъ©ШұШҜЩҮ ШЁШ§ШҙЫҢШҜШҢ ШҙШ§Щ…Щ„ ШӘЩҶШёЫҢЩ…Ш§ШӘ ЩҒЩҶЫҢ ЩҲШЁ Ш§ЩҫЩ„ЫҢЪ©ЫҢШҙЩҶ Щ…ЫҢвҖҢШЁШ§ШҙШҜШӣ Ш§ШІ Ш¬Щ…Щ„ЩҮ Home Screen Icon ЩҲ ЩҶШ§Щ… ЩҲШЁ Ш§ЩҫЩ„ЫҢЪ©ЫҢШҙЩҶ ЩҲ ШіШ§ЫҢШұ ШӘЩҶШёЫҢЩ…Ш§ШӘ ШҜЫҢЪҜШұ.
 Nsgw-config.json : Ш§ЫҢЩҶ ЩҒШ§ЫҢЩ„ ЩҶШіШЁШӘ ШЁЩҮ ЩҒШ§ЫҢЩ„ manifest ЩҒЩҶЫҢвҖҢШӘШұ Щ…ЫҢШЁШ§ШҙШҜ ЩҲ ШЁЫҢШҙШӘШұ ШЁЩҮ Ъ©Ш§ЩҶЩҒЫҢЪҜ Щ…ШҜ ШўЩҒЩ„Ш§ЫҢЩҶ ЩҲ Ъ©Шҙ Ъ©ШұШҜЩҶ Щ…ШұШӘШЁШ· Щ…ЫҢШҙЩҲШҜ. ШҜШұ Ш§ШҜШ§Щ…ЩҮ ШЁШ§ Ш§ЫҢЩҶ ЩҒШ§ЫҢЩ„ ШЁЫҢШҙШӘШұ Ъ©Ш§Шұ ШҜШ§ШұЫҢЩ….

ШЁШ№ШҜ Ш§ШІ ЩҶШөШЁ pwa  ШЁШұ ШұЩҲЫҢ ЩҫШұЩҲЪҳЩҮШҢ ЩҮЩ…ЩҮ ШӘЩҶШёЫҢЩ…Ш§ШӘ ШЁЩҮ ШөЩҲШұШӘ Ш®ЩҲШҜЪ©Ш§Шұ Ш§ЩҶШ¬Ш§Щ… Щ…ЫҢвҖҢШҙЩҲШҜ. Ш§Щ„ШЁШӘЩҮ Щ…ЫҢвҖҢШӘЩҲШ§ЩҶЫҢШҜ ШӘЩҶШёЫҢЩ…Ш§ШӘ Щ…ШұШЁЩҲШ· ШЁЩҮ Ъ©Шҙ Ъ©ШұШҜЩҶ ШҜШ§ШҜЩҮвҖҢЩҮШ§ ШұШ§ ШЁЩҮ ШөЩҲШұШӘ ЩҫЫҢШҙШұЩҒШӘЩҮвҖҢШӘШұ Ъ©Ш§ЩҶЩҒЫҢЪҜ Ъ©ЩҶЫҢШҜ.


  Ш§Ш¬ШұШ§ Ъ©ШұШҜЩҶ ЩҲШЁ Ш§ЩҫЩ„ЫҢЪ©ЫҢШҙЩҶ

 ШЁШұШ§ЫҢ Ш§Ш¬ШұШ§ Ъ©ШұШҜЩҶ ЩҲ ЩҶЩ…Ш§ЫҢШҙ Ш®ШұЩҲШ¬ЫҢ Ш§ШІ ЩҲШЁ Ш§ЩҫЩ„ЫҢЪ©ЫҢШҙЩҶШҢ Ш§ШЁШӘШҜШ§ ШЁШ§ЫҢШҜ Ш§ШІ ЩҫШұЩҲЪҳЩҮ build ЪҜШұЩҒШӘ. ШЁШ§ Ш§ШіШӘЩҒШ§ШҜЩҮ Ш§ШІ ШҜШіШӘЩҲШұ ШІЫҢШұ Ш§ШІ ЩҫШұЩҲЪҳЩҮ Ш®ЩҲШҜ build ШЁЪҜЫҢШұЫҢШҜ:
 ng build -- prod
ШіЩҫШі  ШЁШ§ ШҜШіШӘЩҲШұ ШІЫҢШұ ЩҲШ§ШұШҜ ЩҫЩҲШҙЩҮвҖҢШ§ЫҢ Ъ©ЩҮ ЩҒШ§ЫҢЩ„вҖҢЩҮШ§ЫҢ Build  ШҙШҜЩҮ ШҜШұ ШўЩҶ ЩӮШұШ§Шұ ШҜШ§ШұЩҶШҜШҢ ШҙЩҲЫҢШҜ:
 cd dist/Web Application  Pwa
ШЁШ№ШҜ ШЁШ§ ШҜШіШӘЩҲШұ ШІЫҢШұ ШЁШұЩҶШ§Щ…ЩҮ ШұШ§ Ш§Ш¬ШұШ§ Ъ©ЩҶЫҢШҜ:
 Http-server -o
Ш§ЪҜШұ ЪҶЩҶШ§ЩҶЪҶЩҮ ШЁШ§ ШҜШіШӘЩҲШұ ШЁШ§Щ„Ш§ ШЁЩҮ Ш®Ш·Ш§ ШЁШұШ®ЩҲШұШҜЫҢШҜШҢ ШЁШ§ ШҜШіШӘЩҲШұ ШІЫҢШұ ЩҫЪ©ЫҢШ¬ npm ШўЩҶ ШұШ§ ЩҶШөШЁ Ъ©ЩҶЫҢШҜ:
 npm i http-server
Ш§ЪҜШұ ШӘШ§ ШЁЩҮ Ш§ЫҢЩҶШ¬Ш§ЫҢ Ъ©Ш§Шұ ШҜШұШіШӘ ЩҫЫҢШҙ ШұЩҒШӘЩҮ ШЁШ§ШҙЫҢШҜШҢ ЩҫШұЩҲЪҳЩҮ ШұШ§ ШЁШҜЩҲЩҶ ЩҮЫҢЪҶ ШӘШәЫҢЫҢШұЫҢ Щ…ШҙШ§ЩҮШҜЩҮ Ш®ЩҲШ§ЩҮЫҢЩ… Ъ©ШұШҜ. inspect ЪҜШұЩҒШӘЩҮ ЩҲ ЩҲШ§ШұШҜ ШӘШЁ Application ШҙЩҲЫҢШҜ. Ш§ЫҢЩҶШЁШ§Шұ ЪҜШІЫҢЩҶЩҮвҖҢЫҢ offline ШұШ§ ЩҒШ№Ш§Щ„ Ъ©ЩҶЫҢШҜ ЩҲ Щ…Ш¬ШҜШҜШ§ ШөЩҒШӯЩҮ ШұШ§ refresh Ъ©ЩҶЫҢШҜ! Ш§ЫҢЩҶШЁШ§Шұ ШЁШұЩҶШ§Щ…ЩҮ ШЁШҜЩҲЩҶ ЩҮЫҢЪҶ Щ…ШҙЪ©Щ„ЫҢ Ъ©Ш§Шұ Щ…ЫҢЪ©ЩҶШҜ.  Ш§ЫҢЩҶ ЩҒШ§ЫҢЩ„ Щ…ШұШЁЩҲШ· ШЁЩҮ Ъ©Шҙ Ъ©ШұШҜЩҶ ШҜШ§ШҜЩҮвҖҢЩҮШ§ Щ…ЫҢШЁШ§ШҙЩҶШҜ ЩҲ Щ…ЫҢШӘЩҲШ§ЩҶШҜ ШҙШ§Щ…Щ„ ЪҶЩҶШҜ object ШІЫҢШұ ШЁШ§ШҙШҜ. Щ…ЩҮЩ…ШӘШұЫҢЩҶ ШўЩҶЩҮШ§ ШұШ§ ШЁШұШұШіЫҢ Ш®ЩҲШ§ЩҮЫҢЩ… Ъ©ШұШҜ:
  Ыұ- assetGroups : Ъ©Шҙ Ъ©ШұШҜЩҶ Ш§Ш·Щ„Ш§Ш№Ш§ШӘ Щ…ШұШЁЩҲШ· ШЁЩҮ Ш§ЩҫЩ„ЫҢЪ©ЫҢШҙЩҶ
  ЫІ- Index : Ъ©Шҙ Ъ©ШұШҜЩҶ  ЩҒШ§ЫҢЩ„ Щ…ШұШЁЩҲШ· ШЁЩҮ index.html
  Ыі-  assetGroups : Ъ©Шҙ Ъ©ШұШҜЩҶ ЩҒШ§ЫҢЩ„вҖҢЩҮШ§ЫҢ Щ…ШұШЁЩҲШ· ШЁЩҮ assetШҢ ШҙШ§Щ…Щ„ ЩҒШ§ЫҢЩ„вҖҢЩҮШ§ЫҢ jsШҢ css  ЩҲ  ШәЫҢШұЩҮ
  Ыҙ- dataGroups : Ш§ЫҢЩҶ object Щ…ШұШЁЩҲШ· ШЁЩҮ ЩҲЩӮШӘЫҢ Ш§ШіШӘ Ъ©ЩҮ ШЁШұЩҶШ§Щ…ЩҮ ШҜШұ ШӯШ§Щ„ Ш§Ш¬ШұШ§ Ш§ШіШӘ. Щ…ЫҢШӘЩҲШ§ЩҶ ШҜШ§ШҜЩҮвҖҢЩҮШ§ЫҢ ШҜШұ ШӯШ§Щ„ Ш§Ш¬ШұШ§ЫҢ Ш§ЩҫЩ„ЫҢЪ©ЫҢШҙЩҶ ШұШ§ Ъ©Шҙ Ъ©ШұШҜ. ШҜШ§ШҜЩҮвҖҢЫҢ ШҜШұ ШӯШ§Щ„ Ш§Ш¬ШұШ§ Щ…ЫҢШӘЩҲШ§ЩҶШҜ ШҙШ§Щ…Щ„ ЩҒШұШ§Ш®ЩҲШ§ЩҶЫҢ apiЩҮШ§ ШЁШ§ШҙШҜ. ШЁШұШ§ЫҢ Щ…Ш«Ш§Щ„ ЩҒШұШ¶ Ъ©ЩҶЫҢШҜ ШҙЩ…Ш§ ШҜШұ ШӯШ§Щ„ШӘ Ъ©Ш§Шұ Ъ©ШұШҜЩҶ online ШЁШ§ Ш§ЩҫЩ„ЫҢЪ©ЫҢШҙЩҶШҢ Щ„ЫҢШіШӘЫҢ Ш§ШІ Ш§ШіШ§Щ…ЫҢ Ъ©Ш§ШұШЁШұШ§ЩҶ ШұШ§ Ш§ШІ api ЪҜШұЩҒШӘЩҮ ЩҲ ЩҶЩ…Ш§ЫҢШҙ Щ…ЫҢШҜЩҮЫҢШҜ. ЩҲЩӮШӘЫҢ ШҜЩҒШ№ЩҮвҖҢЫҢ ШЁШ№ШҜ ШҜШұ ШӯШ§Щ„ШӘ offline  Ш§ЩҫЩ„ЫҢЪ©ЫҢШҙЩҶ ШұШ§ ШЁШ§ШІ Ъ©ЩҶЫҢЩ…ШҢ Ш§ЪҜШұ api ШұШ§ Ъ©Шҙ Ъ©ШұШҜЩҮ ШЁШ§ШҙЫҢЩ…ШҢ Ш§ЩҫЩ„ЫҢЪ©ЫҢШҙЩҶ ШҜШ§ШҜЩҮвҖҢЩҮШ§ ШұШ§ Ш§ШІ Ъ©Шҙ ЩҒШұШ§Ш®ЩҲШ§ЩҶЫҢ Щ…ЫҢЪ©ЩҶШҜ. Ш§ЫҢЩҶ Ш№Щ…Щ„ ШҜШұШЁШ§ШұЩҮ post Ъ©ШұШҜЩҶ ШҜШ§ШҜЩҮвҖҢЩҮШ§ ЩҮЩ… ШөШҜЩӮ Щ…ЫҢЪ©ЩҶШҜ.

Ш®ЩҲШҜ dataGroups ШҙШ§Щ…Щ„ ЪҶЩҶШҜ object ШІЫҢШұ Щ…ЫҢШЁШ§ШҙШҜ:
  Ыұ- name :  ЫҢЪ© ЩҶШ§Щ… Ш§ЩҶШӘШ®Ш§ШЁЫҢ ШЁШұШ§ЫҢ Groups  Щ…ЫҢШЁШ§ШҙШҜ.
 ЫІ- urls  : ШҙШ§Щ…Щ„ ШўШұШ§ЫҢЩҮвҖҢШ§ЫҢ Ш§ШІ ШўШҜШұШівҖҢЩҮШ§ Щ…ЫҢШЁШ§ШҙШҜ. Щ…ЫҢШӘЩҲШ§ЩҶ ШўШҜШұШі ЫҢЪ© ШҜЩҲЩ…ЫҢЩҶ ШұШ§ ЩҮЩ…ШұШ§ЩҮ ШЁШ§ Ъ©Щ„ apiЩҮШ§ ШЁЩҮ ШөЩҲШұШӘ ШІЫҢШұ Ъ©Шҙ Ъ©ШұШҜ:
"https://api/**"
Ыі- cacheConfig  : Ъ©ЩҮ ШҙШ§Щ…Щ„
  Ыұ- maxSize  : ШӯШҜШ§Ъ©Ш«Шұ ШӘШ№ШҜШ§ШҜ Ъ©ШҙвҖҢЩҮШ§ЫҢ Щ…ШұШЁЩҲШ· ШЁЩҮ Groups .
  ЫІ- maxAge  : ШӯШҜШ§Ъ©Ш«Шұ  lifetime Щ…ШұШЁЩҲШ· ШЁЩҮ Ъ©Шҙ.
  Ыі- strategy  : Ъ©ЩҮ Щ…ЫҢШӘЩҲШ§ЩҶШҜ ЫҢЪ©ЫҢ Ш§ШІ Щ…ЩӮШ§ШҜЫҢШұ freshness ШЁЩҮ Щ…Ш№ЩҶЫҢ Network-First ЫҢШ§ performance  ШЁЩҮ Щ…Ш№ЩҶЫҢ Cache-First ШЁШ§ШҙШҜ.
