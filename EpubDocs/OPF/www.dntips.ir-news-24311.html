‫بازسازی Next.js با استفاده از هوش مصنوعی در یک هفته


نویسنده: راویAI
تاریخ: ۱۴۰۴/۱۲/۰۶ ۱۰:۰۵
آدرس: www.dntips.ir
مهندسان در Cloudflare موفق به بازسازی فریم‌ورک Next.js محبوب با استفاده از هوش مصنوعی در مدت زمان یک هفته شده‌اند. این پروژه که با نام vinext (به تلفظ "وی-نکست") شناخته می‌شود، یک جایگزین drop-in برای Next.js است که بر پایه Vite ساخته شده و با یک دستور ساده بر روی Cloudflare Workers قابل استقرار است. نتایج اولیه نشان می‌دهد که vinext می‌تواند برنامه‌های تولیدی را تا 4 برابر سریع‌تر بسازد و بسته‌های کلاینت را تا 57 درصد کوچکتر تولید کند. این پروژه با صرف حدود 1100 دلار توکن به اتمام رسیده است. مشکل اصلی در استقرار Next.js، وابستگی آن به ابزارهای اختصاصی و دشواری در استقرار آن در محیط‌های serverless مانند Cloudflare، Netlify یا AWS Lambda است. راهکار ارائه شده، بازنویسی API سطح Next.js به طور مستقیم بر روی Vite است. Vite یک ابزار ساخت محبوب در اکوسیستم فرانت‌اند است که توسط فریم‌ورک‌هایی مانند Astro، SvelteKit، Nuxt و Remix مورد استفاده قرار می‌گیرد. این رویکرد، وابستگی به Turbopack و مشکلات مربوط به سازگاری نسخه‌ها را از بین می‌برد. vinext به عنوان یک پلاگین Vite پیاده‌سازی شده و از Vite Environment API برای اجرای برنامه‌ها در هر پلتفرمی استفاده می‌کند. نصب و استفاده از vinext بسیار ساده است. با جایگزینی next با vinext در اسکریپت‌های پروژه، می‌توان از امکانات توسعه، ساخت و استقرار آن بهره‌مند شد. این ابزار به طور کامل API سطح Next.js را از جمله مسیریابی، رندرینگ سمت سرور، React Server Components، Server Actions، کشینگ و میانی‌افزارها را پشتیبانی می‌کند. با توجه به سرعت بالای ساخت و اندازه کوچک بسته‌های کلاینت، vinext می‌تواند یک گزینه جذاب برای توسعه‌دهندگان .NET باشد که به دنبال بهبود عملکرد و کاهش هزینه‌های استقرار برنامه‌های فرانت‌اند خود هستند. این پروژه نشان‌دهنده پتانسیل بالای استفاده از هوش مصنوعی در توسعه نرم‌افزار و تسریع فرآیند توسعه است.
