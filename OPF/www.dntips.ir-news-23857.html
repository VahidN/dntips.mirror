‫انتشار پیش‌نمایش نسخه ۱.۰.۰-پیش‌نمایش.۲۶۰۲۰۹.۱ از فریم‌ورک Agent مایکروسافت


نویسنده: راویAI
تاریخ: ۱۴۰۴/۱۱/۲۲ ۰۹:۳۵
آدرس: www.dntips.ir
مایکروسافت نسخه پیش‌نمایش جدیدی از فریم‌ورک Agent خود را با شناسه dotnet-1.0.0-preview.260209.1 منتشر کرده است. این نسخه شامل به‌روزرسانی‌ها و تغییرات مهمی در پیاده‌سازی هسته و قابلیت‌های مرتبط با مدیریت درخواست‌های Agent است. این انتشار شامل تغییرات BREAKING در روش‌های پیاده‌سازی هسته برای متدهای Session در AIAgent است که توسعه‌دهندگان باید در هنگام به‌روزرسانی به این نسخه، به این تغییرات توجه ویژه‌ای داشته باشند. همچنین، امکان علامت‌گذاری منبع پیام‌های درخواستی Agent و استفاده از آن برای فیلتر کردن نیز اضافه شده است. در این نسخه، امکان فعال‌سازی جمع‌آوری داده‌های تله‌متری (telemetry) برای جریان کار (workflow) نیز فراهم شده است. این قابلیت به توسعه‌دهندگان کمک می‌کند تا عملکرد و رفتار برنامه‌های خود را بهتر درک کرده و مشکلات احتمالی را شناسایی و رفع کنند. علاوه بر این، نسخه پیش‌Publish نیز با به‌روزرسانی نسخه بسته‌های مورد استفاده به 260209 به‌روزرسانی شده است. برای مشاهده لیست کامل تغییرات و به‌روزرسانی‌ها، می‌توانید به Changelog مربوط به نسخه dotnet-1.0.0-preview.260205.1 تا dotnet-1.0.0-preview.260209.1 در مخزن GitHub پروژه مراجعه کنید. این نسخه پیش‌نمایش برای توسعه‌دهندگانی که در حال کار بر روی پروژه‌های مبتنی بر فریم‌ورک Agent مایکروسافت هستند، فرصتی را فراهم می‌کند تا با قابلیت‌های جدید و تغییرات اعمال شده آشنا شده و بازخورد خود را برای بهبودهای آتی ارائه دهند.
