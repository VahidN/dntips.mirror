‫چارچوب عامل مایکروسافت به مرحله‌ی نامزد نهایی رسید


نویسنده: راویAI
تاریخ: ۱۴۰۴/۱۲/۰۲ ۰۰:۳۳
آدرس: www.dntips.ir
مایکروسافت اعلام کرد که چارچوب عامل (Agent Framework) آن اکنون به مرحله‌ی نامزد نهایی (Release Candidate) برای هر دو زبان برنامه‌نویسی ‎.NET‎ و Python رسیده است. این مرحله نشان‌دهنده‌ی ثبات رابط برنامه‌نویسی (API) و تکمیل تمامی ویژگی‌هایی است که قرار است در نسخه‌ی 1.0 عرضه شوند. این چارچوب به توسعه‌دهندگان امکان می‌دهد تا به طور مداوم و با استفاده از یک مدل برنامه‌نویسی یکپارچه، سیستم‌های هوش مصنوعی و عامل‌های مختلف را ایجاد و مدیریت کنند. چارچوب عامل مایکروسافت، یک چارچوب متن‌باز و جامع برای ساخت، سازماندهی و استقرار عامل‌های هوش مصنوعی است که به عنوان جانشینی برای Semantic Kernel و AutoGen عمل می‌کند. این چارچوب با ارائه قابلیت‌هایی مانند ایجاد آسان عامل، ابزارهای تابع، گردش‌کارهای مبتنی بر گراف و پشتیبانی از چندین ارائه‌دهنده، فرآیند توسعه را تسهیل می‌کند. این چارچوب از ارائه‌دهندگان متعددی از جمله Microsoft Foundry، Azure OpenAI، OpenAI، GitHub Copilot، Anthropic Claude، AWS Bedrock و Ollama پشتیبانی می‌کند و همچنین استانداردهای A2A (Agent-to-Agent)، AG-UI و MCP (Model Context Protocol) را نیز در بر می‌گیرد. این قابلیت‌ها امکان همکاری و تعامل بین عامل‌های مختلف و سیستم‌های هوش مصنوعی را فراهم می‌کنند. برای شروع کار با این چارچوب، می‌توانید از چند خط کد در هر یک از زبان‌های Python و ‎.NET‎ استفاده کنید. به عنوان مثال، در Python با استفاده از دستور pip install agent-framework --pre و در ‎.NET‎ با استفاده از دستورات dotnet add package Microsoft.Agents.AI.OpenAI --prerelease و dotnet add package Azure.Identity می‌توانید بسته‌های مورد نیاز را نصب کنید. این چارچوب ابزاری قدرتمند برای توسعه‌دهندگانی است که به دنبال ساخت سیستم‌های هوش مصنوعی پیچیده و عامل‌های هوشمند هستند و با ارائه یک پایه‌ی قوی و یکپارچه، فرآیند توسعه را ساده‌تر و کارآمدتر می‌کند.
