ЩҮЩ…Ш§ЩҶШ·ЩҲШұ Ъ©ЩҮ Щ…ЫҢвҖҢШҜШ§ЩҶЫҢШҜ ШҜШұ ЪҶЩҶШҜ ШіШ§Щ„ Ш§Ш®ЫҢШұ Ш§ШіШӘЩҒШ§ШҜЩҮ Ш§ШІ ЩҒШұЩ…ШӘ json ШЁШұШ§ЫҢ ЩҒШ§ЫҢЩ„вҖҢЩҮШ§ЫҢ Ъ©Ш§ЩҶЩҒЫҢЪҜ ШЁШіЫҢШ§Шұ ШұШ§ЫҢШ¬ ШҙШҜЩҮ Ш§ШіШӘ. ШҜШұ Ш§ЫҢЩҶ Щ…ЩҲШұШҜ ЫҢЪ© ШӘЩҲЫҢЫҢШӘ Ш¬Ш§Щ„ШЁ ЩҮЩ…ЩҮ ШұШ§ ШЁЩҮ ЪҶШ§Щ„Шҙ Ъ©ШҙЫҢШҜ: В«Ш®ЩҲШ§ЩҮШҙ Щ…ЫҢвҖҢЪ©ЩҶЩ… Ш§ШІ json ШЁШұШ§ЫҢ Ъ©Ш§ЩҶЩҒЫҢЪҜ ЩҒШ§ЫҢЩ„вҖҢЩҮШ§ Ш§ШіШӘЩҒШ§ШҜЩҮ ЩҶЪ©ЩҶЫҢШҜШҢ ШӘЩҲ json ЩҶЩ…ЫҢвҖҢШҙЩҮ Ъ©Ш§Щ…ЩҶШӘ ЩҶЩҲШҙШӘ ЩҲ ШЁШҜЩҲЩҶ Ъ©Ш§Щ…ЩҶШӘ Щ…ШҜЫҢШұЫҢШӘ Ъ©Ш§ЩҶЩҒЫҢЪҜвҖҢЩҮШ§ Ш®ЫҢЩ„ЫҢ ШіШ®ШӘЩҮВ». Ш§ЫҢЩҶ ЩҶЪ©ШӘЩҮ ШЁШұШ§ЫҢ Щ…ЩҶ Ш®ЫҢЩ„ЫҢ Ш¬Ш§Щ„ШЁ ШЁЩҲШҜ. ШЁЩҮ ЩҶШёШұ Щ…ЫҢвҖҢШұШіШҜ ЩҒШұЩ…ШӘ json ШЁШұШ§ЫҢ ЩҒШ§ЫҢЩ„вҖҢЩҮШ§Щ„ Ъ©Ш§ЩҶЩҒЫҢЪҜ ЪҶШ§Щ„ШҙвҖҢЩҮШ§ЫҢЫҢ ШҜШ§ШұШҜ.

Ш¬Ш§Щ„ШЁЩҮ Ъ©ЩҮ Ъ©ШӘШ§ШЁШ®Ш§ЩҶЩҮвҖҢЩҮШ§ЫҢЫҢ ЩҮЩ… ШЁШұШ§ЫҢ Ш§ЫҢЩҶ Ш§ШіШӘШ§ЩҶШҜШ§ШұШҜ ЩҶЩҲШҙШӘЩҮ ШҙШҜЩҮ Ъ©ЩҮ Ш§ШіШӘЩҒШ§ШҜЩҮ Ш§ШІ Ш§ЩҲЩҶ ШұЩҲ ШҜШұ ШЁЫҢШҙШӘШұ ШІШЁШ§ЩҶвҖҢЩҮШ§ЫҢ Щ…ШӯШЁЩҲШЁ Щ…Ш§ЩҶЩҶШҜ Java, C#, JavaScript, Mono, Ruby, Python, Noder.js, PHP, Go ЩҲ ШІШЁШ§ЩҶЩҮШ§ЫҢ ШҜЫҢЪҜШұ ШіШ§ШҜЩҮ Щ…ЫҢвҖҢЪ©ЩҶШҜ.
