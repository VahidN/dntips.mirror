‫برنامه کنفرانس NET. با تمرکز بر مدرن‌سازی


نویسنده: وحید نصیری
تاریخ: ۱۴۰۴/۰۱/۱۲ ۰۷:۵۵
آدرس: www.dntips.ir
سه‌شنبه، 22 آوریل:
  • مدرن‌سازی .NET: برنامه‌های کاربردی آماده برای آینده در عصر هوش مصنوعی (اسکات هانسلمن): مروری بر مدرن‌سازی برنامه‌های .NET، شامل ابزارها، مزایا و تجربیات توسعه‌دهندگان.
  • بررسی عمیق ابزارهای مبتنی بر هوش مصنوعی برای ارتقاء برنامه‌های .NET (آناند گائوراو، مک‌کنا بارلو): معرفی ابزارهای مبتنی بر هوش مصنوعی برای تسهیل مدرن‌سازی برنامه‌های .NET، شامل تحلیل کد و بازسازی.
  • استفاده از الگوهای برنامه وب قابل اعتماد برای تغییر پلتفرم برنامه‌های وب (پابلو لوپز): راهنمایی در مورد استفاده از الگوهای برنامه وب سازمانی برای انتقال برنامه‌های وب به فضای ابری با حداقل تغییرات کد.
  • مدرن‌سازی برنامه‌های داده‌محور (بدون نیاز به هوش مصنوعی) (جری نیکسون): استراتژی‌هایی برای مدرن‌سازی برنامه‌های داده‌محور، با تمرکز بر ساده‌سازی تعامل با پایگاه داده و کاهش بدهی فنی.

چهارشنبه، 23 آوریل:
  • رفع موانع مدرن‌سازی .NET با ابزارهای ارزیابی برنامه و کد مبتنی بر هوش مصنوعی (مایکل ین-چی هو): استفاده از ابزار ارزیابی مایکروسافت و GitHub Copilot Chat برای تسریع مهاجرت به فضای ابری و تغییرات کد.
  • .NET Aspire برای توسعه ابری (مدی مونتاکیلا): ساده‌سازی توسعه برنامه مدرن با .NET Aspire، پوشش میکروسرویس‌ها، تاب‌آوری و مهاجرت به فضای ابری بدون قفل شدن به فروشنده.
  • معرفی هوش مصنوعی به برنامه شما (جردن متسون): افزودن ویژگی‌های هوش مصنوعی به برنامه‌های ارتقا یافته .NET با استفاده از مدل‌های زبانی بزرگ و کتابخانه Microsoft.Extensions.AI.
  • مدرن‌سازی دسکتاپ شما: از WinForms به Blazor، Azure و هوش مصنوعی (سانتیاگو آرانگو): مدرن‌سازی برنامه‌های دسکتاپ با انتقال از WinForms به Blazor، استقرار در Azure و ادغام هوش مصنوعی.
