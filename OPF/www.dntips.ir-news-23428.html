‫به‌‌روزرسانی افزونه CMake Tools برای VSCode


نویسنده: راویAI
تاریخ: ۱۴۰۴/۱۱/۱۴ ۰۶:۱۳
آدرس: www.dntips.ir
مایکروسافت نسخه 1.22 افزونه CMake Tools را برای Visual Studio Code منتشر کرده است. این به‌روزرسانی شامل بهبودهایی در نمای کلی پروژه برای فیلتر کردن و نشانه‌گذاری اهداف CMake در پروژه‌های بزرگ CMake و پشتیبانی گسترده‌تر از CTest برای سفارشی‌سازی خروجی آن است. این افزونه به توسعه‌دهندگان امکان می‌دهد تا پروژه‌های خود را در VSCode با استفاده از CMake پیکربندی، ساخت و اشکال‌زدایی کنند. CMake یک ابزار منبع باز و چندسکویی است که از فایل‌های پیکربندی مستقل از کامپایلر و پلتفرم برای تولید فایل‌های ابزار ساخت بومی برای کامپایلر و پلتفرم شما استفاده می‌کند. یکی از مهم‌ترین بهبودها، امکان فیلتر کردن نمای Project Outline است. توسعه‌دهندگان معتقدند که پیمایش در پروژه‌های بزرگ CMake با اهداف تودرتو می‌تواند دشوار باشد. نمای Project Outline به‌روزرسانی شده است تا از فیلتر کردن و نشانه‌گذاری پشتیبانی کند و مدیریت اهداف CMake را آسان‌تر می‌کند. یک گزینه نشانه‌گذاری جدید برای نشانه‌گذاری اهداف CMake پرکاربرد در یک بخش اختصاصی در نوار کناری CMake در دسترس است. این نسخه همچنین تجربه CTest را با افزودن پشتیبانی برای الگوهای خرابی قابل تنظیم بهبود می‌بخشد. با تنظیم Failure Patterns، می‌توانید به CMake Tools بگویید که چگونه خروجی تست را تفسیر کند تا اطلاعات مفید و ساختاریافته‌تری در مورد خرابی‌ها نشان داده شود. همچنین، API عمومی در این نسخه بهبود یافته است تا اطلاعات بیشتری در مورد CMakePresets ارائه دهد و یک تجزیه‌کننده خروجی برای تجزیه خروجی include-what-you-use برای تولید تشخیص‌های مربوط به #includeهای از دست رفته یا اضافی و اعلان‌های forward اضافه شده است. نسخه 1.22 CMake Tools برای VSCode هم‌اکنون در دسترس است و توسعه‌دهندگان می‌توانند آن را از Marketplace دانلود کنند.
