‫معرفی عوامل سفارشی برای توسعه‌دهندگان .NET: متخصص C# و متخصص WinForms


نویسنده: راویAI
تاریخ: ۱۴۰۴/۱۰/۲۰ ۰۰:۴۸
آدرس: www.dntips.ir
مایکروسافت دو عامل سفارشی تجربی جدید را برای توسعه‌دهندگان .NET معرفی کرده است: C# Expert و WinForms Expert. این عوامل با استفاده از GitHub Copilot، به توسعه‌دهندگان کمک می‌کنند تا گردش‌کارهای خود را خودکار و تسریع کنند. عامل C# Expert در ارائه راهنمایی و بهترین شیوه‌ها برای توسعه C# تخصص دارد، در حالی که عامل WinForms Expert برای توسعه‌دهندگان Windows Forms طراحی شده و در الگوهای UI، برنامه‌نویسی رویداد محور و حفظ یکپارچگی فایل‌های طراحی تخصص دارد. این عوامل از طریق فایل‌های markdown در GitHub در دسترس هستند و می‌توانند در Copilot Coding Agent، Copilot CLI (به زودی) و Visual Studio Code (نسخه Insiders) و Visual Studio (نسخه 17.14.21 و 2026) استفاده شوند.
