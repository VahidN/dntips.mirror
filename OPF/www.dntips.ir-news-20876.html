‫نصب و استفاده از Microsoft .NET 10 با Raspberry Pi


نویسنده: راویAI
تاریخ: ۱۴۰۴/۱۰/۱۶ ۰۰:۱۸
آدرس: www.dntips.ir
این مقاله به بررسی نحوه نصب و استفاده از Microsoft .NET 10 با Raspberry Pi و شروع به کار با C# و .NET در پروژه‌های IoT می‌پردازد. .NET 10 یک چارچوب توسعه چندسکویی از مایکروسافت است که در کنفرانس Dot NET Conf 2025 منتشر شد. برای این کار به یک Raspberry Pi، یک LED، یک مقاومت 220 اهم، یک دکمه و کابل‌های DuPont نیاز دارید. همچنین، استفاده از یک SSH Client مانند PUTTY و یک ویرایشگر کد مانند Visual Studio Code توصیه می‌شود. نصب .NET 10 در Raspberry Pi معمولاً شامل مراحل دستی زیادی است، اما یک اسکریپت نصب یک خطی برای ساده‌سازی فرآیند ارائه شده است. برای نصب، کافیست دستور `wget -O - https://raw.githubusercontent.com/pjgpetecodes/dotnet10pi/main/install.sh | sudo bash` را در Pi خود اجرا کنید. پس از نصب، برای اعمال تغییرات، نیاز به راه‌اندازی مجدد Raspberry Pi دارید. در حال حاضر، Pi 5 به طور کامل توسط کتابخانه‌های IoT .NET پشتیبانی نمی‌شود، اما یک درخواست pull برای به‌روزرسانی کتابخانه‌ها در حال انجام است. این مقاله برای کاربرانی که از Pi 4 یا نسخه‌های پایین‌تر استفاده می‌کنند مناسب است. در نهایت، نحوه کنترل GPIO با استفاده از .NET 10 نیز توضیح داده شده است.
