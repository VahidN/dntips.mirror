‫معرفی مهارت‌های عامل در GitHub Copilot


نویسنده: راویAI
تاریخ: ۱۴۰۴/۱۱/۲۲ ۱۲:۳۲
آدرس: www.dntips.ir
به گزارش مایکروسافت، از ژانویه 2026، مهارت‌های عامل در نسخه پایدار Visual Studio Code در دسترس قرار گرفته‌اند. این مهارت‌ها ابزارهای جداگانه‌ای هستند که به هوش مصنوعی کمک می‌کنند تا وظایف خاصی را انجام دهد، درست مانند ابزارهای مختلف یک جعبه ابزار. هدف از این مقاله، بررسی ماهیت مهارت‌های عامل، دلایل استفاده از آن‌ها و نمایش نحوه استفاده از آن‌ها با GitHub Copilot در Visual Studio Code است. مهارت‌های عامل در اکتبر 2025 توسط Anthropic منتشر و در دسامبر 2025 به عنوان یک استاندارد باز منتشر شدند. در اصل، این مهارت‌ها فایل‌های Markdown حاوی دستورالعمل‌هایی هستند که به هوش مصنوعی می‌گویند چگونه کاری را انجام دهد. تفاوت اصلی با فایل‌های دستورالعمل‌های قبلی این است که مهارت‌ها بر روی یک وظیفه خاص تمرکز دارند، به جای اینکه تمام دستورالعمل‌ها در یک مکان قرار گیرند. استفاده از مهارت‌های عامل مزایایی نسبت به روش‌های قبلی ارائه می‌دهد. آن‌ها سازماندهی بهتری را فراهم می‌کنند، زیرا به شما امکان می‌دهند دستورالعمل‌های خود را به صورت ماژولار تقسیم کنید. همچنین، با کاهش حجم متن در هر فایل، از مصرف بیش از حد حافظه هوش مصنوعی جلوگیری می‌کنند و استفاده از اسکریپت‌ها برای نتایج سازگارتر را تشویق می‌کنند. به عنوان مثال، می‌توانید یک مهارت برای تست واحد ایجاد کنید و آن را در پوشه .github/skills/unit-testing قرار دهید. برای استفاده از مهارت‌های عامل در Visual Studio Code، ابتدا باید آن‌ها را در تنظیمات VS Code فعال کنید. این کار با جستجوی "Agent Skills" یا "chat.useAgentSkills" در تنظیمات و فعال کردن گزینه "Chat: Use Agent Skills" انجام می‌شود. توجه داشته باشید که اگر نام تنظیمات "Claude Skills" باشد، باید VS Code خود را به روز کنید. پس از فعال‌سازی، می‌توانید مهارت‌های عامل را در پروژه‌های خود استفاده کنید و از مزایای آن‌ها بهره‌مند شوید.
