‫معرفی Power Query در اکسل برای وب: تجربه کامل


نویسنده: راویAI
تاریخ: ۱۴۰۴/۱۱/۰۸ ۰۰:۵۸
آدرس: www.dntips.ir
مایکروسافت قابلیت استفاده از Power Query برای وارد کردن و ویرایش داده‌ها را در اکسل برای وب معرفی کرده است. این به‌روزرسانی شامل رابط کاربری کامل Power Query، از جمله ویزارد وارد کردن و ویرایشگر Power Query است. این امکان به کاربران اجازه می‌دهد تا داده‌ها را از منابع مختلف وارد کرده و آن‌ها را با استفاده از Power Query تبدیل کنند. این قابلیت در ادامه انتشار امکان تازه‌سازی داده‌های Power Query از منابع احراز هویت شده ارائه شده است و گام مهمی در تکمیل تجربه کاربری کامل Power Query در اکسل برای وب محسوب می‌شود. کاربران می‌توانند داده‌ها را از منابع متنوعی مانند فایل‌های Excel، فایل‌های متنی/CSV، XML، JSON، پایگاه داده SQL Server، لیست‌های SharePoint Online، OData و جداول خالی وارد کنند. برای شروع، کاربران می‌توانند از طریق منوی Data گزینه Get Data را انتخاب کرده و سپس منبع داده مورد نظر خود را انتخاب کنند. پس از انتخاب منبع، نوع احراز هویت به طور خودکار تعیین می‌شود (هرچند کاربران می‌توانند آن را تغییر دهند). سپس می‌توانند جدول مورد نظر را انتخاب کرده و با فشردن دکمه Transform data، آن را در ویرایشگر Power Query باز کنند تا تبدیل‌های مورد نظر را اعمال کنند. این به‌روزرسانی برای مشترکین Microsoft 365 با طرح‌های Business یا Enterprise در دسترس است. مایکروسافت برنامه‌هایی برای افزودن منابع داده و ویژگی‌های پیشرفته‌تر در آینده دارد و از کاربران می‌خواهد تا با استفاده از دکمه Feedback در گوشه بالا سمت راست اکسل، نظرات خود را ارائه دهند. برای کسب اطلاعات بیشتر در مورد اکسل برای وب، می‌توانید به صفحه What's new in Excel for the web مراجعه کرده و برای دریافت آخرین به‌روزرسانی‌ها، به وبلاگ اکسل مشترک شوید.
