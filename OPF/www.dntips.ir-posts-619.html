Ш§Щ…Ш§ Ш§ЪҜШұ ЩҶШ®ЩҲШ§ЩҮЫҢЩ… Ш§ШІ MSTest Ш§ШіШӘЩҒШ§ШҜЩҮ Ъ©ЩҶЫҢЩ… ЩҲ ЩҮЩ…ЪҶЩҶЫҢЩҶ Ш§ЩҒШІЩҲЩҶЩҮвҖҢЩҮШ§ ШұШ§ ЩҮЩ… ШЁШ®ЩҲШ§ЩҮЫҢЩ… ШӯШ°ЩҒ Ъ©ЩҶЫҢЩ… (Щ…Ш«Щ„Ш§ Ш§ШІ ЩҶШіШ®ЩҮвҖҢЫҢ ШұШ§ЫҢЪҜШ§ЩҶ express Ш§ШіШӘЩҒШ§ШҜЩҮ Ъ©ЩҶЫҢЩ…)ШҢ ЪҶШ·ЩҲШұШҹ
ШЁШұШ§ЫҢ ШӯЩ„ Ш§ЫҢЩҶ Щ…ШҙЪ©Щ„ ЪҶЩҶШҜЫҢЩҶ ШұЩҲШҙ ЩҲШ¬ЩҲШҜ ШҜШ§ШұШҜ. ЫҢШ§ Щ…ЫҢвҖҢШҙЩҲШҜ Ш§ШІ test runner Ш§ЫҢЩҶвҖҢЩҮШ§ Ш§ШіШӘЩҒШ§ШҜЩҮ Ъ©ШұШҜ Ъ©ЩҮ Ш§ШөЩ„Ш§ ЩҶЫҢШ§ШІЫҢ ШЁЩҮ IDE ЩҶШҜШ§ШұЩҶШҜ ЩҲ Щ…ШіШӘЩӮЩ„ Ш§ШіШӘШӣ ЫҢШ§ Щ…ЫҢвҖҢШӘЩҲШ§ЩҶ ШЁЩҮ ШөЩҲШұШӘ ШІЫҢШұ ЩҮЩ… Ш№Щ…Щ„ Ъ©ШұШҜ:
ШЁЩҮ Ш®ЩҲШ§Шө ЩҫШұЩҲЪҳЩҮ ШҜШұ VS.NET Щ…ШұШ§Ш¬Ш№ЩҮ Ъ©ЩҶЫҢШҜ. ШЁШұЪҜЩҮвҖҢЫҢ Build events ШұШ§ ШЁШ§ШІ Ъ©ЩҶЫҢШҜ. ШҜШұ Ш§ЫҢЩҶШ¬Ш§ Щ…ЫҢвҖҢШ®ЩҲШ§ЩҮЫҢЩ… post-build event ШұШ§ Щ…ЩӮШҜШ§Шұ ШҜЩҮЫҢ Ъ©ЩҶЫҢЩ…. ШЁЩҮ Ш§ЫҢЩҶ Щ…Ш№ЩҶШ§ Ъ©ЩҮ ЩҫШі Ш§ШІ ЩҮШұ build Щ…ЩҲЩҒЩӮШҢ Щ„Ш·ЩҒШ§ Ш§ЫҢЩҶ ШҜШіШӘЩҲШұШ§ШӘ Ш®Ш· ЩҒШұЩ…Ш§ЩҶ ШұШ§ Ш§Ш¬ШұШ§ Ъ©ЩҶ.
NUnit ШЁЩҮ ЩҮЩ…ШұШ§ЩҮ test runner Ш®Ш· ЩҒШұЩ…Ш§ЩҶ ЩҮЩ… Ш§ШұШ§ШҰЩҮ Щ…ЫҢвҖҢШҙЩҲШҜ ЩҲ ЩҶШ§Щ… ШўЩҶ nunit-console.exe Ш§ШіШӘ. Ш§ЪҜШұ ШЁЩҮ Щ…ШӯЩ„ ЩҶШөШЁ ШўЩҶ Щ…ШұШ§Ш¬Ш№ЩҮ Ъ©ЩҶЫҢШҜШҢ Ш№Щ…ЩҲЩ…Ш§ ШҜШұ ШўШҜШұШі C:\Program Files\NUnit xyz\bin\nunit-console.exe ЩӮШұШ§Шұ ШҜШ§ШұШҜ. ШЁШұШ§ЫҢ Ш§ШіШӘЩҒШ§ШҜЩҮ Ш§ШІ ШўЩҶ ШӘЩҶЩҮШ§ Ъ©Ш§ЩҒЫҢ Ш§ШіШӘ ШӘЩҶШёЫҢЩ… ШІЫҢШұ ШөЩҲШұШӘ ЪҜЫҢШұШҜ:

c:\path\nunit-console.exe /nologo $(TargetPath)

TargetPath ШЁЩҮ ШөЩҲШұШӘ Ш®ЩҲШҜЪ©Ш§Шұ ШЁШ§ ЩҶШ§Щ… Ш§ШіЩ…ШЁЩ„ЫҢ Ш¬Ш§ШұЫҢ ЩҫШұЩҲЪҳЩҮ ШҜШұ ШІЩ…Ш§ЩҶ Ш§Ш¬ШұШ§ Ш¬Ш§ЫҢЪҜШІЫҢЩҶ Щ…ЫҢвҖҢШҙЩҲШҜ.
Ш§Ъ©ЩҶЩҲЩҶ ЩҫШі Ш§ШІ ЩҮШұ BuildШҢ ШЁЩҮ ШөЩҲШұШӘ Ш®ЩҲШҜЪ©Ш§Шұ nunit-console.exe Ш§Ш¬ШұШ§ ШҙШҜЩҮШҢ Ш§ШіЩ…ШЁЩ„ЫҢ ШЁШұЩҶШ§Щ…ЩҮ Ъ©ЩҮ ШӯШ§ЩҲЫҢ ШўШІЩ…ЩҲЩҶвҖҢЩҮШ§ЫҢ ЩҲШ§ШӯШҜ Ш§ШіШӘ ШЁЩҮ ШўЩҶ Ш§ШұШіШ§Щ„ ЪҜШұШҜЫҢШҜЩҮ ЩҲ ШіЩҫШі Ш®ШұЩҲШ¬ЫҢ Ъ©Ш§Шұ ШҜШұ output window ЩҶЩ…Ш§ЫҢШҙ ШҜШ§ШҜЩҮ Щ…ЫҢвҖҢШҙЩҲШҜ. Ш§ЪҜШұ Ш®Ш·Ш§ЫҢЫҢ ЩҮЩ… ШұШ® ШҜШ§ШҜЩҮ ШЁШ§ШҙШҜ ШҜШұ ЩӮШіЩ…ШӘ errors ЩӮШ§ШЁЩ„ Щ…ШҙШ§ЩҮШҜЩҮ Ш®ЩҲШ§ЩҮШҜ ШЁЩҲШҜ.
ШҜШұ Ш§ЫҢЩҶШ¬Ш§ ШӯШӘЫҢ ШЁШ¬Ш§ЫҢ ШЁШұЩҶШ§Щ…ЩҮ Ъ©ЩҶШіЩҲЩ„ ЫҢШ§ШҜЩҮ ШҙШҜЩҮ Щ…ЫҢвҖҢШӘЩҲШ§ЩҶ Ш§ШІ ШЁШұЩҶШ§Щ…ЩҮ nunit.exe ЩҮЩ… Ш§ШіШӘЩҒШ§ШҜЩҮ Ъ©ШұШҜ. ШҜШұ Ш§ЫҢЩҶ ШӯШ§Щ„ШӘ GUI Ш§ШөЩ„ЫҢ ЩҫШі Ш§ШІ ЩҮШұ Build ЩҶЩ…Ш§ЫҢШҙ ШҜШ§ШҜЩҮ Щ…ЫҢвҖҢШҙЩҲШҜ:

c:\path\nunit.exe $(TargetPath)


ЪҶЩҶШҜ ЩҶЪ©ШӘЩҮ:
1- ШЁШұЩҶШ§Щ…ЩҮ nunit-console.exe ЪҶЩҲЩҶ ШҜШұ ШӯШ§Щ„ ШӯШ§Ш¶Шұ ШЁШұШ§ЫҢ ШҜШ§ШӘ ЩҶШӘ 2 Ъ©Ш§Щ…ЩҫШ§ЫҢЩ„ ШҙШҜЩҮ Ш§Щ…Ъ©Ш§ЩҶ ШЁШ§ШұЪҜШ°Ш§ШұЫҢ dll ЩҮШ§ЫҢ ШҜШ§ШӘ ЩҶШӘ 4 ШұШ§ ЩҶШҜШ§ШұШҜ. ШЁЩҮ ЩҮЩ…ЫҢЩҶ Щ…ЩҶШёЩҲШұ ЩҒШ§ЫҢЩ„ nunit-console.exe.config ШұШ§ ШЁШ§ШІ Ъ©ШұШҜЩҮ ЩҲ ШӘЩҶШёЫҢЩ…Ш§ШӘ ШІЫҢШұ ШұШ§ ШЁЩҮ ШўЩҶ Ш§Ш№Щ…Ш§Щ„ Ъ©ЩҶЫҢШҜ:

<startup>  
      <supportedRuntime version="v4.0.30319" />  
</startup>

ЩҲ ЩҮЩ…ЪҶЩҶЫҢЩҶ:

<runtime>

2- Ш®ШұЩҲШ¬ЫҢ ЩҶШӘШ§ЫҢШ¬ Ш§Ш¬ШұШ§ЫҢ ШўШІЩ…ЩҲЩҶвҖҢЩҮШ§ ШұШ§ ШЁЩҮ ШөЩҲШұШӘ XML ЩҮЩ… Щ…ЫҢвҖҢШӘЩҲШ§ЩҶ Ш°Ш®ЫҢШұЩҮ Ъ©ШұШҜ. Щ…Ш«Щ„Ш§:
