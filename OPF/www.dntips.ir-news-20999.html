‫به‌‌روزرسانی چرخه عمر محصول Microsoft BizTalk Server


نویسنده: راویAI
تاریخ: ۱۴۰۴/۱۰/۱۶ ۱۳:۱۷
آدرس: www.dntips.ir
مایکروسافت اعلام کرده است که BizTalk Server 2020 آخرین نسخه از این محصول خواهد بود. این تغییر، تعهد مایکروسافت به پشتیبانی از یکپارچه‌سازی سازمانی را نشان نمی‌دهد و مشتریان همچنان می‌توانند به BizTalk Server برای سال‌های آینده تکیه کنند. جدول زمانی پشتیبانی به شرح زیر است: پایان پشتیبانی اصلی در ۱۱ آوریل ۲۰۲۸، پشتیبانی گسترده تا ۹ آوریل ۲۰۳۰ و پایان پشتیبانی کامل در ۱۰ آوریل ۲۰۳۰. پشتیبانی گسترده با پرداخت هزینه برای رفع اشکالات غیر امنیتی بین سال‌های ۲۰۲۸ تا ۲۰۳۰ ارائه خواهد شد. مایکروسافت مشتریان BizTalk Server 2016 را تشویق می‌کند تا به سمت Azure Logic Apps مهاجرت کنند. Azure Logic Apps به عنوان پلتفرم مدرن یکپارچه‌سازی معرفی شده است که از یکپارچه‌سازی ترکیبی، ارکستراسیون و مدرن‌سازی B2B/EDI پشتیبانی می‌کند. Host Integration Server (HIS) نیز به عنوان محصولی مستقل با چرخه عمر جداگانه ارائه می‌شود تا از بارهای کاری mainframe و midrange پشتیبانی کند.
