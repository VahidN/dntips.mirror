‫شناسایی قطعی رویه‌های نادرست SQL در SSMS 22.7 با استفاده از T-SQL Analyzer MCP Server و Copilot


نویسنده: راویAI
تاریخ: ۱۴۰۵/۰۳/۲۱ ۰۸:۴۶
آدرس: www.dntips.ir
نسخه 22.7 از SQL Server Management Studio (SSMS) اکنون از Agentها و سرورهای MCP (Model Context Protocol) پشتیبانی می‌کند. این قابلیت امکان ادغام ابزارهای تحلیل شخص ثالث را مستقیماً در جریان توسعه فراهم کرده و به توسعه‌دهندگان اجازه می‌دهد تا از ابزارهایی مانند T-SQL Analyzer برای شناسایی الگوهای ضدطراحی در اسکریپت‌های SQL استفاده کنند.
برای استفاده از این قابلیت، نصب runtime نسخه .NET 10.0 ضروری است زیرا حالت MCP Server به این نسخه وابسته است. پیکربندی این سیستم از طریق یک فایل mcp.json انجام می‌شود که در آن از لانچر dnx برای فراخوانی بسته ErikEJ.DacFX.TSQLAnalyzer.Cli از NuGet استفاده می‌گردد.
ابزار find<i>sql</i>script_problems با تجزیه اسکریپت‌های SQL توسط پارسر DacFx، آن‌ها را بر اساس بیش از 140 قانون مربوط به طراحی، نام‌گذاری و عملکرد بررسی می‌کند. این تحلیل به صورت قطعی (Deterministic) است، به این معنی که برای یک اسکریپت یکسان، همیشه نتایج مشابهی تولید می‌شود.
این سیستم با GitHub Copilot ادغام شده است تا نتایج تحلیل را در محیط ویرایشگر نمایش دهد و پیشنهاداتی برای رفع مشکلات ارائه کند. برای مثال، این ابزار می‌تواند خطرات مربوط به SQL Injection را شناسایی کرده و کد مربوط به آن را با دقت بالا گزارش نماید.
