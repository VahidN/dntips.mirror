Щ…ЩӮШҜЩ…ЩҮ
ЩҮЩҲШҙ Щ…ШөЩҶЩҲШ№ЫҢ (AI) ШҜШұ ШіШ§Щ„вҖҢЩҮШ§ЫҢ Ш§Ш®ЫҢШұ Ш§ШІ ЫҢЪ© Щ…ЩҒЩҮЩҲЩ… ШӘШӯЩӮЫҢЩӮШ§ШӘЫҢ ШЁЩҮ ЫҢЪ© Ш§ШЁШІШ§Шұ Ш№Щ…Щ„ЫҢ ШҜШұ ШӘЩҲШіШ№ЩҮ ЩҶШұЩ…вҖҢШ§ЩҒШІШ§Шұ ШӘШЁШҜЫҢЩ„ ШҙШҜЩҮ Ш§ШіШӘ. ШҜШұ Щ…ЫҢШ§ЩҶ ЩҫЫҢШҙШұЩҒШӘвҖҢЩҮШ§ЫҢ Ш§ЫҢЩҶ ШӯЩҲШІЩҮШҢ ЩҮЩҲШҙ Щ…ШөЩҶЩҲШ№ЫҢ Ш№Ш§Щ…Щ„вҖҢЩ…ШӯЩҲШұ (Agentic AI) ШЁЩҮвҖҢШ№ЩҶЩҲШ§ЩҶ ЫҢЪ© ЩҫШ§ШұШ§ШҜШ§ЫҢЩ… ЩҶЩҲШёЩҮЩҲШұШҢ ШӘЩҲШ¬ЩҮ ШӘЩҲШіШ№ЩҮвҖҢШҜЩҮЩҶШҜЪҜШ§ЩҶ ШұШ§ ШЁЩҮ Ш®ЩҲШҜ Ш¬Щ„ШЁ Ъ©ШұШҜЩҮ Ш§ШіШӘ. ШЁШұШ®Щ„Ш§ЩҒ Щ…ШҜЩ„вҖҢЩҮШ§ЫҢ ШіЩҶШӘЫҢ ЩҮЩҲШҙ Щ…ШөЩҶЩҲШ№ЫҢ Ъ©ЩҮ ШЁЩҮвҖҢШөЩҲШұШӘ ЩҲШ§Ъ©ЩҶШҙЫҢ (Reactive) Ш№Щ…Щ„ Щ…ЫҢвҖҢЪ©ЩҶЩҶШҜШҢ Ш№Ш§Щ…Щ„вҖҢЩҮШ§ЫҢ ЩҮЩҲШҙ Щ…ШөЩҶЩҲШ№ЫҢ (AI Agents) ЩӮШ§ШҜШұ ШЁЩҮ ШҜШұЪ© Щ…ШӯЫҢШ·ШҢ ШӘШөЩ…ЫҢЩ…вҖҢЪҜЫҢШұЫҢ Ш®ЩҲШҜЪ©Ш§Шұ ЩҲ Ш§Ш¬ШұШ§ЫҢ ЩҲШёШ§ЫҢЩҒ ЩҫЫҢЪҶЫҢШҜЩҮ ШЁШ§ ШӯЩҒШё ШІЩ…ЫҢЩҶЩҮ (Context) ЩҮШіШӘЩҶШҜ. Ш§ЫҢЩҶ ЩӮШ§ШЁЩ„ЫҢШӘвҖҢЩҮШ§ШҢ ЩҫШӘШ§ЩҶШіЫҢЩ„ ШӘШәЫҢЫҢШұ ЩҶШӯЩҲЩҮ Ш·ШұШ§ШӯЫҢ ЩҲ ЩҫЫҢШ§ШҜЩҮвҖҢШіШ§ШІЫҢ ШЁШұЩҶШ§Щ…ЩҮвҖҢЩҮШ§ШҢ ШЁЩҮвҖҢЩҲЫҢЪҳЩҮ ШҜШұ Ш§Ъ©ЩҲШіЫҢШіШӘЩ… NET. Щ…Ш§ЫҢЪ©ШұЩҲШіШ§ЩҒШӘ ШұШ§ ШҜШ§ШұЩҶШҜ.

ШӘШ№ШұЫҢЩҒ ЩҲ ШӘЩ…Ш§ЫҢШІ Ш№Ш§Щ…Щ„вҖҢЩҮШ§ЫҢ ЩҮЩҲШҙ Щ…ШөЩҶЩҲШ№ЫҢ
Ш№Ш§Щ…Щ„вҖҢЩҮШ§ЫҢ ЩҮЩҲШҙ Щ…ШөЩҶЩҲШ№ЫҢШҢ ШЁШұЩҶШ§Щ…ЩҮвҖҢЩҮШ§ЫҢЫҢ ЩҮШіШӘЩҶШҜ Ъ©ЩҮ ШЁЩҮвҖҢШөЩҲШұШӘ Ш®ЩҲШҜЪ©Ш§Шұ ЩҲШёШ§ЫҢЩҒ ЫҢШ§ ШӘШөЩ…ЫҢЩ…вҖҢЪҜЫҢШұЫҢвҖҢЩҮШ§ ШұШ§ ШЁШұ Ш§ШіШ§Ші ШҜШ§ШҜЩҮвҖҢЩҮШ§ЫҢ ЩҲШұЩҲШҜЫҢ Ш§ЩҶШ¬Ш§Щ… Щ…ЫҢвҖҢШҜЩҮЩҶШҜ. ШЁШұШ®Щ„Ш§ЩҒ Щ…ШҜЩ„вҖҢЩҮШ§ЫҢ ШіЩҶШӘЫҢ ШІШЁШ§ЩҶ ШЁШІШұЪҜ (Large Language Models ЫҢШ§ LLMs) Щ…Ш§ЩҶЩҶШҜ ChatGPT Ъ©ЩҮ ШЁЩҮвҖҢШөЩҲШұШӘ ЩҲШ§Ъ©ЩҶШҙЫҢ ЩҲ ШЁШҜЩҲЩҶ ШӯЩҒШё ШІЩ…ЫҢЩҶЩҮ ШЁЩ„ЩҶШҜЩ…ШҜШӘ Ш№Щ…Щ„ Щ…ЫҢвҖҢЪ©ЩҶЩҶШҜШҢ Ш№Ш§Щ…Щ„вҖҢЩҮШ§ЫҢ ЩҮЩҲШҙ Щ…ШөЩҶЩҲШ№ЫҢ ШҜШ§ШұШ§ЫҢ ЩҲЫҢЪҳЪҜЫҢвҖҢЩҮШ§ЫҢ ШІЫҢШұ ЩҮШіШӘЩҶШҜ:
  1. Ш®ЩҲШҜЩ…Ш®ШӘШ§ШұЫҢ (Autonomy): ЩӮШ§ШҜШұ ШЁЩҮ ШӘШөЩ…ЫҢЩ…вҖҢЪҜЫҢШұЫҢ ЩҲ Ш§Ш¬ШұШ§ЫҢ Ш§ЩӮШҜШ§Щ…Ш§ШӘ ШЁШҜЩҲЩҶ ЩҶЫҢШ§ШІ ШЁЩҮ ШҜШ®Ш§Щ„ШӘ Щ…ШҜШ§ЩҲЩ… Ш§ЩҶШіШ§ЩҶ ЩҮШіШӘЩҶШҜ.
  2. ЩҮШҜЩҒвҖҢЩ…ШӯЩҲШұЫҢ (Goal-Oriented): ШЁШұШ§ЫҢ ШҜШіШӘЫҢШ§ШЁЫҢ ШЁЩҮ Ш§ЩҮШҜШ§ЩҒ Щ…ШҙШ®Шө Ш·ШұШ§ШӯЫҢ ШҙШҜЩҮвҖҢШ§ЩҶШҜ ЩҲ Щ…ЫҢвҖҢШӘЩҲШ§ЩҶЩҶШҜ Щ…ШіЫҢШұ Ш®ЩҲШҜ ШұШ§ ШЁШұ Ш§ШіШ§Ші ЩҶШӘШ§ЫҢШ¬ ШӘЩҶШёЫҢЩ… Ъ©ЩҶЩҶШҜ.
  3. ШӘШ№Ш§Щ…Щ„ ШЁШ§ Щ…ШӯЫҢШ· (Environment Interaction): ШӘЩҲШ§ЩҶШ§ЫҢЫҢ ШҜШұЪ© ЩҲ ШӘШ№Ш§Щ…Щ„ ШЁШ§ Щ…ШӯЫҢШ·вҖҢЩҮШ§ЫҢ ШҜЫҢШ¬ЫҢШӘШ§Щ„ Щ…Ш§ЩҶЩҶШҜ ШұШ§ШЁШ·вҖҢЩҮШ§ЫҢ ЩҲШЁ ЫҢШ§ ЩҫШ§ЫҢЪҜШ§ЩҮвҖҢЩҮШ§ЫҢ ШҜШ§ШҜЩҮ ШұШ§ ШҜШ§ШұЩҶШҜ.
ШЁЩҮвҖҢШ№ЩҶЩҲШ§ЩҶ Щ…Ш«Ш§Щ„ШҢ ЫҢЪ© Щ…ШҜЩ„ ШіЩҶШӘЫҢ ЩҮЩҲШҙ Щ…ШөЩҶЩҲШ№ЫҢ Щ…Щ…Ъ©ЩҶ Ш§ШіШӘ ШҜШұ ЩҫШ§ШіШ® ШЁЩҮ ШҜШұШ®ЩҲШ§ШіШӘ В«ЫҢЪ© ЩҶЩ…ЩҲЩҶЩҮ Ъ©ШҜ C# ШЁШұШ§ЫҢ Ш§ШӘШөШ§Щ„ ШЁЩҮ ЩҫШ§ЫҢЪҜШ§ЩҮ ШҜШ§ШҜЩҮ SQL Server ШЁЩҶЩҲЫҢШіВ»ШҢ Ъ©ШҜЫҢ Щ…Ш§ЩҶЩҶШҜ ШІЫҢШұ ШӘЩҲЩ„ЫҢШҜ Ъ©ЩҶШҜ:
using System.Data.SqlClient;

string connectionString = "Server=localhost;Database=SampleDB;Trusted_Connection=True;";
using (SqlConnection connection = new SqlConnection(connectionString))
{
    connection.Open();
    SqlCommand command = new SqlCommand("SELECT * FROM Users", connection);
    SqlDataReader reader = command.ExecuteReader();
    while (reader.Read())
    {
        Console.WriteLine(reader["Username"]);
    }
}
Ш§Щ…Ш§ ЫҢЪ© Ш№Ш§Щ…Щ„ ЩҮЩҲШҙ Щ…ШөЩҶЩҲШ№ЫҢ Щ…ЫҢвҖҢШӘЩҲШ§ЩҶШҜ ЩҒШұШ§ШӘШұ ШұЩҲШҜШӣ Щ…Ш«Щ„Ш§ЩӢШҢ ШЁШ§ ШҜШұЫҢШ§ЩҒШӘ ШҜШұШ®ЩҲШ§ШіШӘ В«ЫҢЪ© Ш§ШӘШөШ§Щ„ ШЁЩҮ ЩҫШ§ЫҢЪҜШ§ЩҮ ШҜШ§ШҜЩҮ SQL Server ШұШ§ Щ…ШҜЫҢШұЫҢШӘ Ъ©ЩҶВ»ШҢ ЩҶЩҮвҖҢШӘЩҶЩҮШ§ Ъ©ШҜ ШұШ§ ШӘЩҲЩ„ЫҢШҜ Ъ©ЩҶШҜШҢ ШЁЩ„Ъ©ЩҮ ШЁЩҮвҖҢШөЩҲШұШӘ Ш®ЩҲШҜЪ©Ш§Шұ ЫҢЪ© Щ…Ш®ШІЩҶ GitHub Ш§ЫҢШ¬Ш§ШҜ Ъ©ЩҶШҜШҢ Ъ©ШҜ ШұШ§ ШҜШұ ШўЩҶ ШўЩҫЩ„ЩҲШҜ Ъ©ЩҶШҜ ЩҲ ШӯШӘЫҢ ЫҢЪ© Pull Request ШЁШұШ§ЫҢ ШЁШ§ШІШЁЫҢЩҶЫҢ ШўЩ…Ш§ШҜЩҮ Ъ©ЩҶШҜ. Ш§ЫҢЩҶ ШӘЩҒШ§ЩҲШӘШҢ ЩҶШҙШ§ЩҶвҖҢШҜЩҮЩҶШҜЩҮ ЩӮШ§ШЁЩ„ЫҢШӘвҖҢЩҮШ§ЫҢ Ш№Ш§Щ…Щ„вҖҢЩ…ШӯЩҲШұ ШҜШұ Ш§Ш¬ШұШ§ЫҢ ЩҲШёШ§ЫҢЩҒ ЪҶЩҶШҜЩ…ШұШӯЩ„ЩҮвҖҢШ§ЫҢ Ш§ШіШӘ.

Щ…Ш№Щ…Ш§ШұЫҢ Ш№Ш§Щ…Щ„вҖҢЩҮШ§ЫҢ ЩҮЩҲШҙ Щ…ШөЩҶЩҲШ№ЫҢ
Щ…Ш№Щ…Ш§ШұЫҢ Ш№Ш§Щ…Щ„вҖҢЩҮШ§ЫҢ ЩҮЩҲШҙ Щ…ШөЩҶЩҲШ№ЫҢ Щ…Ш№Щ…ЩҲЩ„Ш§ЩӢ Ш§ШІ ШіЩҮ Щ„Ш§ЫҢЩҮ Ш§ШөЩ„ЫҢ ШӘШҙЪ©ЫҢЩ„ ШҙШҜЩҮ Ш§ШіШӘ:
  1. Щ„Ш§ЫҢЩҮ Щ…ЫҢШІШЁШ§ЩҶЫҢ ЩҲ ШіШұЩҲЫҢШівҖҢШҜЩҮЫҢ Ш№Ш§Щ…Щ„ (Agent Hosting/Serving): Ш§ЫҢЩҶ Щ„Ш§ЫҢЩҮ ШҙШ§Щ…Щ„ ШІЫҢШұШіШ§Ш®ШӘвҖҢЩҮШ§ЫҢ Ш§ШЁШұЫҢ ЫҢШ§ Щ…ШӯЩ„ЫҢ Ш§ШіШӘ Ъ©ЩҮ Ш№Ш§Щ…Щ„вҖҢЩҮШ§ ШұШ§ Ш§Ш¬ШұШ§ Щ…ЫҢвҖҢЪ©ЩҶЩҶШҜ. ШҜШұ Ш§Ъ©ЩҲШіЫҢШіШӘЩ… NET.ШҢ Ш§ЫҢЩҶ Щ…ЫҢвҖҢШӘЩҲШ§ЩҶШҜ ШҙШ§Щ…Щ„ Azure Functions ЫҢШ§ ШіШұЩҲЫҢШівҖҢЩҮШ§ЫҢ Kubernetes ШЁШ§ШҙШҜ.
  2. ЪҶШ§ШұЪҶЩҲШЁвҖҢЩҮШ§ЫҢ Ш№Ш§Щ…Щ„ (Agent Frameworks): Ш§ШЁШІШ§ШұЩҮШ§ЫҢЫҢ Щ…Ш§ЩҶЩҶШҜ LangChain ЫҢШ§ CrewAI Ъ©ЩҮ Ш§Щ…Ъ©Ш§ЩҶ Ш§ЫҢШ¬Ш§ШҜ ЩҲ Щ…ШҜЫҢШұЫҢШӘ ШӘЫҢЩ…вҖҢЩҮШ§ЫҢ Ш№Ш§Щ…Щ„ ШұШ§ ЩҒШұШ§ЩҮЩ… Щ…ЫҢвҖҢЪ©ЩҶЩҶШҜ. Ш§ЫҢЩҶ ЪҶШ§ШұЪҶЩҲШЁвҖҢЩҮШ§ШҢ ЩҫШұЩҲШӘЪ©Щ„вҖҢЩҮШ§ЫҢ Щ…Ш°Ш§Ъ©ШұЩҮШҢ Ш§Ш¬Щ…Ш§Ш№вҖҢШіШ§ШІЫҢ ЩҲ ШӯЩ„ ШӘШ№Ш§ШұШ¶ ШұШ§ ЩҫЫҢШ§ШҜЩҮвҖҢШіШ§ШІЫҢ Щ…ЫҢвҖҢЪ©ЩҶЩҶШҜ.
  3. Щ…ШҜЩ„вҖҢЩҮШ§ЫҢ ШІШЁШ§ЩҶ ЩҲ Ш°Ш®ЫҢШұЩҮвҖҢШіШ§ШІЫҢ (LLM Models & Storage): Щ…ШҜЩ„вҖҢЩҮШ§ЫҢ ШІШЁШ§ЩҶ ШЁШІШұЪҜ Щ…Ш§ЩҶЩҶШҜ Claude ЫҢШ§ GPT-4 ШЁЩҮвҖҢШ№ЩҶЩҲШ§ЩҶ Щ…ШәШІ Ш№Ш§Щ…Щ„ Ш№Щ…Щ„ Щ…ЫҢвҖҢЪ©ЩҶЩҶШҜ ЩҲ ШЁШ§ ЩҫШ§ЫҢЪҜШ§ЩҮвҖҢЩҮШ§ЫҢ ШҜШ§ЩҶШҙ (Knowledge Bases) ЫҢЪ©ЩҫШ§ШұЪҶЩҮ Щ…ЫҢвҖҢШҙЩҲЩҶШҜ.
ШЁШұШ§ЫҢ ШӘЩҲШіШ№ЩҮвҖҢШҜЩҮЩҶШҜЪҜШ§ЩҶ NET.ШҢ Ш§ШіШӘЩҒШ§ШҜЩҮ Ш§ШІ Azure AI Services ШЁШұШ§ЫҢ ЫҢЪ©ЩҫШ§ШұЪҶЩҮвҖҢШіШ§ШІЫҢ Щ…ШҜЩ„вҖҢЩҮШ§ЫҢ ШІШЁШ§ЩҶ ЩҲ Ш§ЫҢШ¬Ш§ШҜ Ш№Ш§Щ…Щ„вҖҢЩҮШ§ЫҢ ЩҮЩҲШҙЩ…ЩҶШҜШҢ ЪҜШІЫҢЩҶЩҮвҖҢШ§ЫҢ ЩӮШҜШұШӘЩ…ЩҶШҜ Ш§ШіШӘ. ШЁЩҮвҖҢШ№ЩҶЩҲШ§ЩҶ Щ…Ш«Ш§Щ„ШҢ Щ…ЫҢвҖҢШӘЩҲШ§ЩҶ Ш§ШІ Azure Cognitive Services ШЁШұШ§ЫҢ Ш§ЩҒШІЩҲШҜЩҶ ЩӮШ§ШЁЩ„ЫҢШӘвҖҢЩҮШ§ЫҢ ЩҫШұШҜШ§ШІШҙ ШІШЁШ§ЩҶ Ш·ШЁЫҢШ№ЫҢ (NLP) ШЁЩҮ ЫҢЪ© Ш№Ш§Щ…Щ„ Ш§ШіШӘЩҒШ§ШҜЩҮ Ъ©ШұШҜ.

ЩҫШұЩҲШӘЪ©Щ„вҖҢЩҮШ§ЫҢ Ш§ШұШӘШЁШ§Ш·ЫҢ ЩҲ Ш§ШіШӘШ§ЩҶШҜШ§ШұШҜШіШ§ШІЫҢ
ЫҢЪ©ЫҢ Ш§ШІ ЪҶШ§Щ„ШҙвҖҢЩҮШ§ЫҢ Ш§ШөЩ„ЫҢ ШҜШұ ШӘЩҲШіШ№ЩҮ Ш№Ш§Щ…Щ„вҖҢЩҮШ§ЫҢ ЩҮЩҲШҙ Щ…ШөЩҶЩҲШ№ЫҢШҢ ЩҶШЁЩҲШҜ Ш§ШіШӘШ§ЩҶШҜШ§ШұШҜЩҮШ§ЫҢ Ш§ШұШӘШЁШ§Ш·ЫҢ ЫҢЪ©ЩҫШ§ШұЪҶЩҮ Ш§ШіШӘ. ЩҫШұЩҲШӘЪ©Щ„ ШІЩ…ЫҢЩҶЩҮ Щ…ШҜЩ„ (Model Context Protocol ЫҢШ§ MCP)ШҢ Щ…Ш№ШұЩҒЫҢвҖҢШҙШҜЩҮ ШӘЩҲШіШ· AnthropicШҢ ЫҢЪ© Ш§ШіШӘШ§ЩҶШҜШ§ШұШҜ ШЁШ§ШІ ШЁШұШ§ЫҢ Ш§ШӘШөШ§Щ„ Щ…ШҜЩ„вҖҢЩҮШ§ЫҢ ЩҮЩҲШҙ Щ…ШөЩҶЩҲШ№ЫҢ ШЁЩҮ Ш§ШЁШІШ§ШұЩҮШ§ ЩҲ Щ…ЩҶШ§ШЁШ№ ШҜШ§ШҜЩҮ Ш®Ш§ШұШ¬ЫҢ Ш§ШіШӘ. MCP ШЁЩҮвҖҢШ№ЩҶЩҲШ§ЩҶ В«USB-C ШЁШұШ§ЫҢ ЫҢЪ©ЩҫШ§ШұЪҶЩҮвҖҢШіШ§ШІЫҢ ЩҮЩҲШҙ Щ…ШөЩҶЩҲШ№ЫҢВ» ШҙЩҶШ§Ш®ШӘЩҮ Щ…ЫҢвҖҢШҙЩҲШҜ ЩҲ Ш§Щ…Ъ©Ш§ЩҶ ШҜШіШӘШұШіЫҢ Щ…ШіШӘЩӮЫҢЩ… Ш№Ш§Щ…Щ„вҖҢЩҮШ§ ШЁЩҮ ЩҫЩ„ШӘЩҒШұЩ…вҖҢЩҮШ§ЫҢЫҢ Щ…Ш§ЩҶЩҶШҜ GitHub ЫҢШ§ ЩҫШ§ЫҢЪҜШ§ЩҮвҖҢЩҮШ§ЫҢ ШҜШ§ШҜЩҮ SQL Server ШұШ§ ЩҒШұШ§ЩҮЩ… Щ…ЫҢвҖҢЪ©ЩҶШҜ.
ШҜШұ ЫҢЪ© ЩҫШұЩҲЪҳЩҮ NET.ШҢ Щ…ЫҢвҖҢШӘЩҲШ§ЩҶ Ш§ШІ MCP ШЁШұШ§ЫҢ Ш§ШӘШөШ§Щ„ ЫҢЪ© Ш№Ш§Щ…Щ„ ШЁЩҮ ЫҢЪ© API RESTful Ъ©ЩҮ ШЁШ§ ASP.NET Core ЩҫЫҢШ§ШҜЩҮвҖҢШіШ§ШІЫҢ ШҙШҜЩҮШҢ Ш§ШіШӘЩҒШ§ШҜЩҮ Ъ©ШұШҜ. Ш§ЫҢЩҶ API Щ…ЫҢвҖҢШӘЩҲШ§ЩҶШҜ ШЁЩҮ ЫҢЪ© Ш№Ш§Щ…Щ„ ЩҮЩҲШҙ Щ…ШөЩҶЩҲШ№ЫҢ Ш§Ш¬Ш§ШІЩҮ ШҜЩҮШҜ ШӘШ§ ШҜШ§ШҜЩҮвҖҢЩҮШ§ЫҢ Ъ©Ш§ШұШЁШұШ§ЩҶ ШұШ§ Ш§ШІ ЩҫШ§ЫҢЪҜШ§ЩҮ ШҜШ§ШҜЩҮ Ш§ШіШӘШ®ШұШ§Ш¬ Ъ©ШұШҜЩҮ ЩҲ ШЁШұШ§ЫҢ ЩҲШёШ§ЫҢЩҒ ШЁШ№ШҜЫҢ Щ…Ш§ЩҶЩҶШҜ ШӘШӯЩ„ЫҢЩ„ ЫҢШ§ ЪҜШІШ§ШұШҙвҖҢШҜЩҮЫҢ Ш§ШіШӘЩҒШ§ШҜЩҮ Ъ©ЩҶШҜ.

Ш№Ш§Щ…Щ„вҖҢЩҮШ§ЫҢ Щ…ШЁШӘЩҶЫҢ ШЁШұ Щ…ШұЩҲШұЪҜШұ
ЫҢЪ©ЫҢ Ш§ШІ ЩҶЩҲШўЩҲШұЫҢвҖҢЩҮШ§ЫҢ ШЁШұШ¬ШіШӘЩҮ ШҜШұ Ш§Ъ©ЩҲШіЫҢШіШӘЩ… Ш№Ш§Щ…Щ„вҖҢЩҮШ§ЫҢ ЩҮЩҲШҙ Щ…ШөЩҶЩҲШ№ЫҢШҢ ШёЩҮЩҲШұ Ш№Ш§Щ…Щ„вҖҢЩҮШ§ЫҢ Щ…ШЁШӘЩҶЫҢ ШЁШұ Щ…ШұЩҲШұЪҜШұ (Browser-Based Agents) Ш§ШіШӘ. Ш§ЫҢЩҶ Ш№Ш§Щ…Щ„вҖҢЩҮШ§ Щ…ЫҢвҖҢШӘЩҲШ§ЩҶЩҶШҜ Щ…Ш§ЩҶЩҶШҜ Ш§ЩҶШіШ§ЩҶ ШЁШ§ ШұШ§ШЁШ·вҖҢЩҮШ§ЫҢ ЩҲШЁ ШӘШ№Ш§Щ…Щ„ Ъ©ЩҶЩҶШҜШҢ ШөЩҒШӯШ§ШӘ ШұШ§ ЩҫЫҢЩ…Ш§ЫҢШҙ Ъ©ЩҶЩҶШҜШҢ ЩҒШұЩ…вҖҢЩҮШ§ ШұШ§ ЩҫШұ Ъ©ЩҶЩҶШҜ ЩҲ Ш§Ш·Щ„Ш§Ш№Ш§ШӘ ШұШ§ Ш§ШіШӘШ®ШұШ§Ш¬ Ъ©ЩҶЩҶШҜ. Ш§ЫҢЩҶ ЩӮШ§ШЁЩ„ЫҢШӘШҢ ШЁШұШ§ЫҢ Ш§ШӘЩҲЩ…Ш§ШіЫҢЩҲЩҶ ЩҲШЁ ШҜШұ ШіЩҶШ§ШұЫҢЩҲЩҮШ§ЫҢЫҢ Ъ©ЩҮ API ШҜШұ ШҜШіШӘШұШі ЩҶЫҢШіШӘШҢ ШЁШіЫҢШ§Шұ Ш§ШұШІШҙЩ…ЩҶШҜ Ш§ШіШӘ.
ШЁШұШ§ЫҢ ШӘЩҲШіШ№ЩҮвҖҢШҜЩҮЩҶШҜЪҜШ§ЩҶ NET.ШҢ Ш§ЫҢЩҶ Ш№Ш§Щ…Щ„вҖҢЩҮШ§ Щ…ЫҢвҖҢШӘЩҲШ§ЩҶЩҶШҜ ШЁШ§ Ш§ШЁШІШ§ШұЩҮШ§ЫҢЫҢ Щ…Ш§ЩҶЩҶШҜ Selenium ЫҢШ§ Playwright ЫҢЪ©ЩҫШ§ШұЪҶЩҮ ШҙЩҲЩҶШҜ ШӘШ§ ЩҲШёШ§ЫҢЩҒ ЩҫЫҢЪҶЫҢШҜЩҮвҖҢШ§ЫҢ Щ…Ш§ЩҶЩҶШҜ ЩҫШұ Ъ©ШұШҜЩҶ ЩҒШұЩ…вҖҢЩҮШ§ЫҢ ЩҲШЁ ЫҢШ§ Ш§ШіШӘШ®ШұШ§Ш¬ ШҜШ§ШҜЩҮ Ш§ШІ ЩҲШЁвҖҢШіШ§ЫҢШӘвҖҢЩҮШ§ ШұШ§ Ш§ЩҶШ¬Ш§Щ… ШҜЩҮЩҶШҜ. ШЁЩҮвҖҢШ№ЩҶЩҲШ§ЩҶ Щ…Ш«Ш§Щ„ШҢ Ъ©ШҜ ШІЫҢШұ ШЁШ§ Ш§ШіШӘЩҒШ§ШҜЩҮ Ш§ШІ Playwright ЫҢЪ© ЩҒШұЩ… ЩҲШЁ ШұШ§ ШЁЩҮвҖҢШөЩҲШұШӘ Ш®ЩҲШҜЪ©Ш§Шұ ЩҫШұ Щ…ЫҢвҖҢЪ©ЩҶШҜ:
using Microsoft.Playwright;

async Task FillFormAsync()
{
    using var playwright = await Playwright.CreateAsync();
    await using var browser = await playwright.Chromium.LaunchAsync();
    var page = await browser.NewPageAsync();
    await page.GotoAsync("https://example.com/form");
    await page.FillAsync("input[name='username']", "testuser");
    await page.FillAsync("input[name='password']", "password123");
    await page.ClickAsync("button[type='submit']");
}
Ш§ЫҢЩҶ Ъ©ШҜ Щ…ЫҢвҖҢШӘЩҲШ§ЩҶШҜ ШЁЩҮвҖҢШ№ЩҶЩҲШ§ЩҶ ШЁШ®ШҙЫҢ Ш§ШІ ЫҢЪ© Ш№Ш§Щ…Щ„ ЩҮЩҲШҙ Щ…ШөЩҶЩҲШ№ЫҢ Ш№Щ…Щ„ Ъ©ЩҶШҜ Ъ©ЩҮ ЩҲШёШ§ЫҢЩҒ ЩҲШЁ ШұШ§ ШЁЩҮвҖҢШөЩҲШұШӘ Ш®ЩҲШҜЪ©Ш§Шұ Ш§ЩҶШ¬Ш§Щ… Щ…ЫҢвҖҢШҜЩҮШҜ.

ЪҶШ§Щ„ШҙвҖҢЩҮШ§ЫҢ ЩҒЩҶЫҢ ЩҲ Щ…Щ„Ш§ШӯШёШ§ШӘ
ШӘЩҲШіШ№ЩҮ Ш№Ш§Щ…Щ„вҖҢЩҮШ§ЫҢ ЩҮЩҲШҙ Щ…ШөЩҶЩҲШ№ЫҢ ШЁШ§ ЪҶШ§Щ„ШҙвҖҢЩҮШ§ЫҢЫҢ ЩҮЩ…ШұШ§ЩҮ Ш§ШіШӘ Ъ©ЩҮ ШӘЩҲШіШ№ЩҮвҖҢШҜЩҮЩҶШҜЪҜШ§ЩҶ NET. ШЁШ§ЫҢШҜ ШЁЩҮ ШўЩҶвҖҢЩҮШ§ ШӘЩҲШ¬ЩҮ Ъ©ЩҶЩҶШҜ:
  1. ШҜШұЪ© ШЁШөШұЫҢ (Visual Understanding): Ш№Ш§Щ…Щ„вҖҢЩҮШ§ЫҢ Щ…ШЁШӘЩҶЫҢ ШЁШұ Щ…ШұЩҲШұЪҜШұ ЩҶЫҢШ§ШІ ШЁЩҮ ШӘЩҲШ§ЩҶШ§ЫҢЫҢ ШӘЩҒШіЫҢШұ Ш·ШұШӯвҖҢЩҮШ§ЫҢ ШЁШөШұЫҢ ЩҫЫҢЪҶЫҢШҜЩҮ ШҜШ§ШұЩҶШҜШҢ Ъ©ЩҮ ЩҶЫҢШ§ШІЩ…ЩҶШҜ ШӘШұЪ©ЫҢШЁ ШҜЫҢШҜ Ъ©Ш§Щ…ЩҫЫҢЩҲШӘШұЫҢ (Computer Vision) ЩҲ ШҜШұЪ© Щ…Ш№ЩҶШ§ЫҢЫҢ Ш§ШіШӘ.
  2. Щ…ШҜЫҢШұЫҢШӘ ШІЩ…ЫҢЩҶЩҮ (Context Management): ШӯЩҒШё ШІЩ…ЫҢЩҶЩҮ ШҜШұ ЩҲШёШ§ЫҢЩҒ Ш·ЩҲЩ„Ш§ЩҶЫҢвҖҢЩ…ШҜШӘШҢ ШЁЩҮвҖҢЩҲЫҢЪҳЩҮ ШҜШұ ЩҫШұЩҲЪҳЩҮвҖҢЩҮШ§ЫҢ ШЁШІШұЪҜ NET.ШҢ ЩҶЫҢШ§ШІЩ…ЩҶШҜ Щ…Ш№Щ…Ш§ШұЫҢвҖҢЩҮШ§ЫҢ Щ…ЩӮЫҢШ§ШівҖҢЩҫШ°ЫҢШұ Ш§ШіШӘ.
  3. Ш§Щ…ЩҶЫҢШӘ ЩҲ Ш§Ш®Щ„Ш§ЩӮ: Ш№Ш§Щ…Щ„вҖҢЩҮШ§ ШЁШ§ЫҢШҜ Ш§ШІ ЩҶШёШұ Ш§Щ…ЩҶЫҢШӘЫҢ Щ…ШӯШ§ЩҒШёШӘ ШҙЩҲЩҶШҜ ЩҲ Ш§ШІ Ш§ШіШӘЩҒШ§ШҜЩҮ ЩҶШ§ШҜШұШіШӘ Ш§ШІ ШҜШ§ШҜЩҮвҖҢЩҮШ§ Ш¬Щ„ЩҲЪҜЫҢШұЫҢ Ъ©ЩҶЩҶШҜ. ШҜШұ NET.ШҢ Ш§ШіШӘЩҒШ§ШҜЩҮ Ш§ШІ Azure Active Directory ШЁШұШ§ЫҢ Ш§ШӯШұШ§ШІ ЩҮЩҲЫҢШӘ ЩҲ Щ…ШҜЫҢШұЫҢШӘ ШҜШіШӘШұШіЫҢ ШӘЩҲШөЫҢЩҮ Щ…ЫҢвҖҢШҙЩҲШҜ.

Ъ©Ш§ШұШЁШұШҜЩҮШ§ ШҜШұ Ш§Ъ©ЩҲШіЫҢШіШӘЩ… NET.
ШҜШұ ЪҶШ§ШұЪҶЩҲШЁ NET.ШҢ Ш№Ш§Щ…Щ„вҖҢЩҮШ§ЫҢ ЩҮЩҲШҙ Щ…ШөЩҶЩҲШ№ЫҢ Щ…ЫҢвҖҢШӘЩҲШ§ЩҶЩҶШҜ Ъ©Ш§ШұШЁШұШҜЩҮШ§ЫҢ Щ…ШӘШ№ШҜШҜЫҢ ШҜШ§ШҙШӘЩҮ ШЁШ§ШҙЩҶШҜШҢ Ш§ШІ Ш¬Щ…Щ„ЩҮ:
  • Ш§ШӘЩҲЩ…Ш§ШіЫҢЩҲЩҶ ЩҲШёШ§ЫҢЩҒ ШӘЪ©ШұШ§ШұЫҢ: Щ…Ш§ЩҶЩҶШҜ Щ…ШҜЫҢШұЫҢШӘ Щ…Ш®Ш§ШІЩҶ Ъ©ШҜ ЫҢШ§ Ш§ШіШӘШ®ШұШ§Ш¬ ШҜШ§ШҜЩҮ.
  • ШӘШӯЩӮЫҢЩӮ Ш№Щ…ЫҢЩӮ (Deep Research): Ш¬ШіШӘвҖҢЩҲШ¬ЩҲЫҢ Ш®ЩҲШҜЪ©Ш§Шұ ЩҲШЁ ЩҲ ШӘЩҲЩ„ЫҢШҜ ЪҜШІШ§ШұШҙвҖҢЩҮШ§ЫҢ Ш¬Ш§Щ…Ш№.
  • ШҜШіШӘЫҢШ§ШұЩҮШ§ЫҢ ШӘЩҲШіШ№ЩҮвҖҢШҜЩҮЩҶШҜЩҮ: Ъ©Щ…Ъ© ШЁЩҮ ШҜЫҢШЁШ§ЪҜШҢ ШЁШ§ШІШЁЫҢЩҶЫҢ Ъ©ШҜШҢ ЩҲ ЩҫЫҢШҙЩҶЩҮШ§ШҜ Щ…Ш№Щ…Ш§ШұЫҢ.

ЩҶШӘЫҢШ¬ЩҮвҖҢЪҜЫҢШұЫҢ
Ш№Ш§Щ…Щ„вҖҢЩҮШ§ЫҢ ЩҮЩҲШҙ Щ…ШөЩҶЩҲШ№ЫҢШҢ ШўЫҢЩҶШҜЩҮ ШӘЩҲШіШ№ЩҮ ЩҶШұЩ…вҖҢШ§ЩҒШІШ§Шұ ШұШ§ ШЁШ§ШІШӘШ№ШұЫҢЩҒ Щ…ЫҢвҖҢЪ©ЩҶЩҶШҜ. ШЁШұШ§ЫҢ ШӘЩҲШіШ№ЩҮвҖҢШҜЩҮЩҶШҜЪҜШ§ЩҶ NET.ШҢ Ш§ЫҢЩҶ ЩҒЩҶШ§ЩҲШұЫҢ ЩҒШұШөШӘвҖҢЩҮШ§ЫҢ ШЁЫҢвҖҢШіШ§ШЁЩӮЩҮвҖҢШ§ЫҢ ШЁШұШ§ЫҢ Ш§ШӘЩҲЩ…Ш§ШіЫҢЩҲЩҶШҢ ШЁЩҮШЁЩҲШҜ ШЁЩҮШұЩҮвҖҢЩҲШұЫҢ ЩҲ Ш®Щ„ЩӮ ШӘШ¬ШұШЁЩҮвҖҢЩҮШ§ЫҢ Ъ©Ш§ШұШЁШұЫҢ ЩҶЩҲШўЩҲШұШ§ЩҶЩҮ ЩҒШұШ§ЩҮЩ… Щ…ЫҢвҖҢЪ©ЩҶШҜ. ШЁШ§ Ш§ШіШӘЩҒШ§ШҜЩҮ Ш§ШІ Ш§ШЁШІШ§ШұЩҮШ§ЫҢЫҢ Щ…Ш§ЩҶЩҶШҜ Azure AI ServicesШҢ ЪҶШ§ШұЪҶЩҲШЁвҖҢЩҮШ§ЫҢ Ш№Ш§Щ…Щ„вҖҢЩ…ШӯЩҲШұ ЩҲ Ш§ШіШӘШ§ЩҶШҜШ§ШұШҜЩҮШ§ЫҢЫҢ Щ…Ш§ЩҶЩҶШҜ MCPШҢ ШӘЩҲШіШ№ЩҮвҖҢШҜЩҮЩҶШҜЪҜШ§ЩҶ Щ…ЫҢвҖҢШӘЩҲШ§ЩҶЩҶШҜ Ш№Ш§Щ…Щ„вҖҢЩҮШ§ЫҢЫҢ ШЁШіШ§ШІЩҶШҜ Ъ©ЩҮ ЩҲШёШ§ЫҢЩҒ ЩҫЫҢЪҶЫҢШҜЩҮ ШұШ§ ШЁЩҮвҖҢШөЩҲШұШӘ Ш®ЩҲШҜЪ©Ш§Шұ Ш§ЩҶШ¬Ш§Щ… ШҜЩҮЩҶШҜ. ШЁШ§ Ш§ЫҢЩҶ ШӯШ§Щ„ШҢ Щ…ЩҲЩҒЩӮЫҢШӘ ШҜШұ Ш§ЫҢЩҶ ШӯЩҲШІЩҮ ЩҶЫҢШ§ШІЩ…ЩҶШҜ ШҜШұЪ© Ш№Щ…ЫҢЩӮ Щ…Ш№Щ…Ш§ШұЫҢвҖҢЩҮШ§ЫҢ Щ…ЩӮЫҢШ§ШівҖҢЩҫШ°ЫҢШұШҢ ЩҫШұЩҲШӘЪ©Щ„вҖҢЩҮШ§ЫҢ Ш§ШұШӘШЁШ§Ш·ЫҢ ЩҲ Щ…Щ„Ш§ШӯШёШ§ШӘ Ш§Щ…ЩҶЫҢШӘЫҢ Ш§ШіШӘ.


Щ…ШҙШ§ЩҮШҜЩҮ Щ…Ш·Щ„ШЁ Ш§ШөЩ„ЫҢ
Щ…Ш·Ш§Щ„ШЁ Щ…ШҙШ§ШЁЩҮ
    Ш§ШҙШӘШұШ§Ъ©вҖҢЩҮШ§ШіШ§Ш®ШӘ Ш§ЩҲЩ„ЫҢЩҶ Ш№Ш§Щ…Щ„ ЩҮЩҲШҙ Щ…ШөЩҶЩҲШ№ЫҢ ШҜШұ C# ШЁШ§ Ш§ШіШӘЩҒШ§ШҜЩҮ Ш§ШІ Microsoft Agent FrameworkШ§ШҙШӘШұШ§Ъ©вҖҢЩҮШ§Щ…Ш№ШұЩҒЫҢ ЪҶШ§ШұЪҶЩҲШЁ Ш№Ш§Щ…Щ„ Щ…Ш§ЫҢЪ©ШұЩҲШіШ§ЩҒШӘ (ЩҫЫҢШҙвҖҢЩҶЩ…Ш§ЫҢШҙ): ШіШ§Ш®ШӘ Ш№Ш§Щ…Щ„вҖҢЩҮШ§ЫҢ ЩҮЩҲШҙ Щ…ШөЩҶЩҲШ№ЫҢ ШҜШұ .NETШ§ШҙШӘШұШ§Ъ©вҖҢЩҮШ§ЩҫШұЩҲШӘЪ©Щ„ Щ…ШӘЩҶ Щ…ШҜЩ„ (MCP): ШұЩҲЫҢЪ©ШұШҜЫҢ ЩҶЩҲЫҢЩҶ ШҜШұ ШӘШ№Ш§Щ…Щ„ ЩҮЩҲШҙ Щ…ШөЩҶЩҲШ№ЫҢ ШЁШ§ ШҜЩҶЫҢШ§ЫҢ Ш®Ш§ШұШ¬ ШҜШұ NET.Ш§ШҙШӘШұШ§Ъ©вҖҢЩҮШ§Ш№Ш§Щ…Щ„вҖҢЩҮШ§ЫҢ ЩҮЩҲШҙ Щ…ШөЩҶЩҲШ№ЫҢ Ш®ЩҲШҜЪ©Ш§Шұ ЩҫЫҢШҙШұЩҒШӘЩҮ: ШӯШ§ЩҒШёЩҮШҢ ЩҮЩ…Ъ©Ш§ШұЫҢ ЩҲ ШӘЪ©Ш§Щ…Щ„ ШҜШұ Ъ©ЩҶЩҒШұШ§ЩҶШі NDC ManchesterШ§ШҙШӘШұШ§Ъ©вҖҢЩҮШ§ЩҫЫҢШ§ШҜЩҮвҖҢШіШ§ШІЫҢ Ш§Щ„ЪҜЩҲЫҢШӘ Щ…ЩҮШ§ШұШӘвҖҢЩҮШ§ЫҢ ЩҮЩҲШҙ Щ…ШөЩҶЩҲШ№ЫҢ ШҜШұ .NET: ШўЩҲШұШҜЩҶ Ш§Щ„ЪҜЩҲЫҢ Ш№Ш§Щ…Щ„ ШўЩҶШӘШұЩҲЩҫЫҢЪ© ШЁЩҮ Ш§Ъ©ЩҲШіЫҢШіШӘЩ… Щ…Ш§ЫҢЪ©ШұЩҲШіШ§ЩҒШӘШ§ШҙШӘШұШ§Ъ©вҖҢЩҮШ§Щ…Ш№ШұЩҒЫҢ Ъ©ЫҢШӘ ШҙШұЩҲШ№вҖҢЪ©ЩҶЩҶШҜЩҮвҖҢЫҢ Evals ШЁШұШ§ЫҢ ШӘШ№Ш§Щ…Щ„ Ш№Ш§Щ…Щ„ (Agent Interop)Ш§ШҙШӘШұШ§Ъ©вҖҢЩҮШ§GitHub Copilot SDK ШҜШұ ЩҫЫҢШҙвҖҢЩҶЩ…Ш§ЫҢШҙ ЩҒЩҶЫҢ ЩӮШұШ§Шұ ЪҜШұЩҒШӘШ§ШҙШӘШұШ§Ъ©вҖҢЩҮШ§ЪҶШ§ШұЪҶЩҲШЁ Ш№Ш§Щ…Щ„ Щ…Ш§ЫҢЪ©ШұЩҲШіШ§ЩҒШӘ ШЁЩҮ Щ…ШұШӯЩ„ЩҮвҖҢЫҢ ЩҶШ§Щ…ШІШҜ ЩҶЩҮШ§ЫҢЫҢ ШұШіЫҢШҜШ§ШҙШӘШұШ§Ъ©вҖҢЩҮШ§MCP ШЁШұШ§ЫҢ .NET: Ш№Ш§Щ…Щ„вҖҢЩҮШ§ЫҢ ЩҮЩҲШҙ Щ…ШөЩҶЩҲШ№ЫҢ Ъ©ЩҮ Щ…ЫҢвҖҢШӘЩҲШ§ЩҶЩҶШҜ Ъ©Ш§ШұЩҮШ§ЫҢ ЩҲШ§ЩӮШ№ЫҢ Ш§ЩҶШ¬Ш§Щ… ШҜЩҮЩҶШҜШ§ШҙШӘШұШ§Ъ©вҖҢЩҮШ§ЪҶШ§ШұЪҶЩҲШЁ Ш№Ш§Щ…Щ„ Щ…Ш§ЫҢЪ©ШұЩҲШіШ§ЩҒШӘ: Ш§ЩҒШІЩҲШҜЩҶ RAG ШЁЩҮ Ш№Ш§Щ…Щ„ ЩҮЩҲШҙ Щ…ШөЩҶЩҲШ№ЫҢ ШЁШ§ Ш§ШіШӘЩҒШ§ШҜЩҮ Ш§ШІ TextSearchProvider ЩҲ ЩҒШұЩҲШҙЪҜШ§ЩҮ ШЁШұШҜШ§ШұЫҢ ШҜШұ ШӯШ§ЩҒШёЩҮ
