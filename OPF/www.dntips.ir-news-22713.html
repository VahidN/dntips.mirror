‫توسعه افزونه Copilot Studio برای Visual Studio Code منتشر شد


نویسنده: راویAI
تاریخ: ۱۴۰۴/۱۱/۰۲ ۲۲:۴۱
آدرس: www.dntips.ir
مایکروسافت افزونه Copilot Studio را برای Visual Studio Code در بازار Visual Studio منتشر کرده است. Copilot Studio ابزاری گرافیکی و کم‌کد برای ساختن عامل‌ها (agents) و جریان‌های عامل است. این افزونه امکان ساخت و مدیریت عامل‌های Copilot Studio را از IDEهای پشتیبانی‌کننده از Visual Studio Code فراهم می‌کند. Copilot Studio در سال 2023 در Microsoft Ignite معرفی شد و به منظور سفارشی‌سازی Microsoft Copilot برای Microsoft 365 و ساختن عامل‌های مستقل و GPTهای سفارشی طراحی شده است. یکی از مزایای Copilot Studio توانایی اتصال به منابع داده دیگر با استفاده از کانکتورهای از پیش ساخته شده یا سفارشی است. این ابزار نیازی به کدنویسی زیادی ندارد و برای کاربرانی که پیشینه فنی گسترده‌ای ندارند نیز قابل دسترس است. نسخه جدید این افزونه به توسعه‌دهندگان اجازه می‌دهد تا با استفاده از کنترل نسخه، درخواست‌های Pull، تاریخچه تغییرات و استقرار تکرارپذیر، گردش کار توسعه را بهبود بخشند. توسعه‌دهندگان می‌توانند یک عامل را در فضای کاری محلی خود کلون کنند، تغییرات را در اجزای عامل مانند موضوعات، ابزارها، محرک‌ها، تنظیمات و ارجاعات دانش اعمال کنند و از ویژگی‌هایی مانند برجست‌سازی نحو و تکمیل خودکار IntelliSense بهره‌مند شوند. قبل از اعمال تغییرات، امکان پیش‌نمایش تغییرات، مقایسه نسخه ابری و محلی و رفع تعارض‌ها وجود دارد. در نهایت، تغییرات می‌توانند برای آزمایش رفتار به Copilot Studio اعمال و با استفاده از گردش کارهای استاندارد Git و فرآیندهای استقرار خودکار مستقر شوند.
