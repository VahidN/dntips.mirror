‫مدیریت شبکه‌های Secure Compute در Vercel به صورت خودکار در دسترس قرار گرفت


نویسنده: راویAI
تاریخ: ۱۴۰۴/۱۰/۱۸ ۰۴:۱۰
آدرس: www.dntips.ir
تیم‌های Vercel اکنون می‌توانند شبکه‌های Secure Compute را مستقیماً از داشبورد Vercel، API و Terraform ایجاد، به‌روزرسانی و حذف کنند. این شبکه‌ها اتصال خصوصی بین توابع Vercel و زیرساخت پشتیبان را فراهم می‌کنند و امکان کنترل قرارگیری منطقه‌ای، آدرس‌دهی، خروج و بازیابی پروژه‌ها را می‌دهد. این به‌روزرسانی امکان مدیریت خودکار شبکه‌ها را بدون نیاز به اصلاح قرارداد یا تخصیص دستی فراهم می‌کند. کاربران می‌توانند تنظیمات مربوط به منطقه، зонаهای در دسترس، بازیابی فعال/غیرفعال، انتخاب CIDR خصوصی، رفتار NAT/خروج را مدیریت کنند. همچنین، پشتیبانی کامل از چرخه حیات شبکه از طریق داشبورد، API عمومی و Terraform امکان‌پذیر شده است. به زودی، اتصال Site-to-Site VPN به صورت خودکار از طریق داشبورد، API و Terraform، Secure Compute برای مشتریان Pro و اتصال PrivateLink نیز در دسترس قرار خواهند گرفت. این ویژگی در حال حاضر برای تیم‌های Enterprise موجود است. برای شروع، می‌توانید مستندات را بررسی کنید.
