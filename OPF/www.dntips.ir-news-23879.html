‫محاسبات GPU با کارایی بالا در مرورگر با C# و WebGPU


نویسنده: راویAI
تاریخ: ۱۴۰۴/۱۱/۲۲ ۰۹:۵۱
آدرس: www.dntips.ir
کتابخانه SpawnDev.ILGPU امکان نوشتن کرنل‌های C# را برای محاسبات GPU در مرورگر فراهم می‌کند. این کتابخانه با استفاده از Blazor WASM، توسعه‌دهندگان را قادر می‌سازد تا از قابلیت‌های موازی‌سازی گسترده بهره‌مند شوند و عملکرد برنامه‌های تحت وب را به طور قابل توجهی افزایش دهند. SpawnDev.ILGPU چهار بک‌اند اجرایی مختلف ارائه می‌دهد که هر کدام برای مقاصد خاصی طراحی شده‌اند: بک‌اند WebGPU (سریع‌ترین)، بک‌اند Wasm، بک‌اند Workers و بک‌اند CPU. بک‌اند WebGPU با تبدیل کرنل‌های C# به WGSL، محاسبات GPU را به صورت سخت‌افزاری تسریع می‌کند. بک‌اند Wasm از چندین Web Worker و تبدیل C# به WebAssembly برای پردازش موازی با سرعت بالا استفاده می‌کند. برای نمایش قابلیت‌های این کتابخانه، یک کاوشگر فرکتال (Fractal Explorer) ساخته شده است که مجموعه Mandelbrot را با استفاده از بک‌اند WebGPU رندر می‌کند. این امر امکان دستیابی به سطوح زوم بی‌نظیر را فراهم می‌کند که با حلقه‌های استاندارد Blazor قابل دستیابی نیستند. این پروژه همچنین در زمینه‌هایی مانند تبدیل ویدئویی دو بعدی به سه بعدی و ایجاد گروه‌های محاسباتی توزیع‌شده با استفاده از WebRTC کاربرد دارد. این پروژه منبع باز است و توسعه‌دهندگان ‎.NET می‌توانند با بررسی کد و مشارکت در توسعه آن، از این فناوری جدید در پروژه‌های خود استفاده کنند. SpawnDev.ILGPU با استفاده از SpawnDev.BlazorJS برای برقراری ارتباط با جاوااسکریپت، مدیریت حافظه و ارسال کرنل‌ها را به طور خودکار انجام می‌دهد. برای کسب اطلاعات بیشتر و مشارکت در این پروژه، می‌توانید به مخزن GitHub و بسته‌ی NuGet مراجعه کنید. این کتابخانه یک راه حل قدرتمند برای توسعه‌دهندگان ‎.NET است که به دنبال بهره‌گیری از قدرت محاسبات GPU در برنامه‌های تحت وب هستند.
