‫ساخت ابزار ZingIt با استفاده از GitHub Copilot SDK: اشاره، کلیک، رفع اشکال با هوش مصنوعی


نویسنده: راویAI
تاریخ: ۱۴۰۴/۱۱/۱۶ ۱۰:۴۸
آدرس: www.dntips.ir
توسعه‌دهنده‌ای به نام Dan Wahlin ابزاری به نام ZingIt ساخته است که به توسعه‌دهندگان اجازه می‌دهد تا با اشاره به عناصر یک صفحه وب، یادداشت‌گذاری کرده و به طور خودکار از هوش مصنوعی برای رفع اشکالات استفاده کنند. این ابزار فرآیند توصیف باگ‌های رابط کاربری را حذف می‌کند و نیاز به اسکرین‌شات‌ها را کاهش می‌دهد. ZingIt با استفاده از GitHub Copilot SDK ساخته شده است که فرآیند یکپارچه‌سازی را بسیار ساده‌تر کرده است. این ابزار به توسعه‌دهندگان امکان می‌دهد تا با فشار دادن کلید Z، وارد حالت علامت‌گذاری شوند، عناصر مورد نظر را انتخاب کنند، دستورالعمل‌های خود را تایپ کنند و با کلیک بر روی آیکون جرقه (✨)، تغییرات را اعمال کنند. ZingIt از چندین عامل کدنویسی هوش مصنوعی پشتیبانی می‌کند، از جمله Claude Code، GitHub Copilot CLI و OpenAI Codex. این امکان به توسعه‌دهندگان اجازه می‌دهد تا از ابزاری که با اشتراک هوش مصنوعی مورد نظرشان سازگار است، استفاده کنند. توسعه‌دهنده همچنین اشاره کرده که امکان گسترش این ابزار برای پشتیبانی از عوامل هوش مصنوعی بیشتر در آینده وجود دارد. GitHub Copilot SDK نقش مهمی در ساخت ZingIt ایفا کرده است. قبل از SDK، توصیف دقیق عناصر رابط کاربری برای هوش مصنوعی زمان‌بر و دشوار بود. SDK این فرآیند را ساده کرده و امکان ایجاد یک ابزار کارآمد و کاربرپسند را فراهم آورده است.
