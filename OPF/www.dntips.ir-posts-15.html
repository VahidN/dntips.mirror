وبلاگ‌ها و سایت‌های ایرانی

  • آشنایی با گروه‌های ویندوز
  • زبان برنامه‌نویسی فارسی‌نت
  • گزارشات و دفاتر مرسوم حسابداری
  • رفع مشکل طراحی وب برای IE
  • مجله امنیتی اسنوپ
  • آفیس آنلاین مایکروسافت ، به‌زودی خواهد آمد

Visual Studio

  • آشنایی با Historical Debugger در Visual Studio 2010

امنیت

  • مایکروسافت: عمده نقایص امنیتی ویندوز از برنامه‌های جانبی آن ناشی می‌شود
  • رمزنگاری کوانتمی و شبکه‌ای رمزنگاری شده با این روش

ASP. Net

  • معرفی 7 افزودنی jQuery برای استفاده در ASP.Net
  • دموهایی از jQuery و ASP.Net Ajax
  • نحوه پویانمایی محتوا با jQuery
  • استفاده از اکتیو دایرکتوری برای اعتبار سنجی کاربران

طراحی وب

  • IE بیشتر از 31 تگ style را پردازش نمی‌کند!
  • ایجاد گوشه‌های گرد با CSS
  • یک پلاگین tooltip برای jQuery

اس‌کیوال سرور

  • لیستی از ویژگی‌های جدید SQL Server 2008
  • دیباگ کردن رویه‌های ذخیره شده با کمک VS.Net
  • فشرده سازی داده‌ها در SQL server 2008
  • رمزنگاری رویه‌های ذخیره شده و تاثیر آن بر Execution Plan
  • SQL server 2008 و تبدیل داده‌های بایناری به یک رشته هگز
  • توابع توکار SQL Server 2005

به روز رسانی‌ها

  • رفع مشکل نمایش خطای بی‌مورد کامپایلر در نگارش SP1 ویژوال استودیو 2008

ابزارها

  • نگارش جدید برنامه RSS Bandit . (برنامه‌ای سورس باز نوشته شده با سی شارپ)
  • نگارش جدید Application Pool Recycle Utility
  • اسمبلی‌های دات نت را به actionscript تبدیل کنید!
  • نگارش جدید StyleCop for ReSharper
  • Visual Round Trip Analyzer (استفاده از NetMon API)

سی‌شارپ

  • نحوه بدست آوردن آدرس IP کامپیوتر جاری
  • آینده‌ی سی شارپ
  • Interactive GUI Shell از توسعه دهندگان تیم Mono
  • تازه‌های سی شارپ 4 ، واژه کلیدی dynamic ، قسمت‌های یک و دو و سه

عمومی دات نت

  • به نظر پروندهLINQ To SQL بسته شد! (چند روز بعد تکذیب شد!)
  • آشنایی با کلاس CommaDelimitedStringCollection در دات نت فریم ورک
  • مونو و دات نت گزارشی از PDC2008

CPP

  • تازه‌های نگارش‌های مختلف GCC

دلفی

  • CompilerPlugin برای دلفی 2009. (توسط آن می‌توان پروژه‌های دلفی 2007 را با دلفی 2009 نیز کامپایل کرد)
  • نوامبر 2008 ، 25 امین سالگرد انتشار Turbo Pascal v1.0 است!
  • نسخه جدید CnPack منتشر شد. (با پشتیبانی دلفی 5 تا 2009)

ویندوز

  • نگارش R2 ویندوز سرور 2008 تنها 64 بیتی خواهد بود.
  • ویندوز 7 و جستجو در منابع SharePoint
  • آنالیز crash dumps ویندوز . (آیا می‌دانید صفحات آبی ویندوز را چگونه باید تفسیر کرد؟)

Office

  • آشنایی با یک سری از اصطلاحات outlook 2007 برای برنامه نویس‌ها. (اگر قصد داشته باشید یک add-in را برای outlook 2007 با استفاده از امکانات VSTO توسعه دهید، آشنایی با این اصطلاحات بسیار ضروری خواهد بود)

متفرقه

  • 10 عامل کشنده یک پروژه نرم افزاری
  • لیستی از 22 افزودنی فایرفاکس برای داشتن مرورگری حرفه‌ای
  • لیستی از دموهای PDC2008
  • باغ وحشی از متدولوژی‌های توسعه نرم افزاری!
  • ابزارهایی برای مدیریت پروژه و همکاری‌های جمعی
  • 6 نکته برای شروع یک پروژه سورس باز
  • تعیین اعتبار یک regular expression با استفاده از silverlight
  • تصمیم گیری مشکل، PHP فریم ورک‌های زیادی دارد.
  • و پشت رایانه بنشینید تا جوان بمانید!
