Щ…Ш·Ш§Щ„ШЁ Щ…ШҙШ§ШЁЩҮ
    Ш§ШҙШӘШұШ§Ъ©вҖҢЩҮШ§Щ…ЩӮШ§ЫҢШіЩҮ ЩҶШ®вҖҢЩҮШ§ЫҢ Щ…Ш¬Ш§ШІЫҢ (Virtual Threads) ШҜШұ Ш¬Ш§ЩҲШ§ ЩҲ async/await ШҜШұ ШіЫҢвҖҢШҙШ§ШұЩҫ: ШұЩҲЫҢЪ©ШұШҜЩҮШ§ЫҢ Щ…ШҜШұЩҶ ШЁЩҮ ШЁШұЩҶШ§Щ…ЩҮвҖҢЩҶЩҲЫҢШіЫҢ ЩҶШ§ЩҮЩ…ЪҜШ§Щ…Ш§ШҙШӘШұШ§Ъ©вҖҢЩҮШ§Ъ©Ш§ЩҮШҙ Щ…ШөШұЩҒ ШӯШ§ЩҒШёЩҮ HttpClient ШҜШұ ШҜШ§ШӘвҖҢЩҶШӘ 10Ш§ШҙШӘШұШ§Ъ©вҖҢЩҮШ§Ш§ШіШӘЩҒШ§ШҜЩҮ Ш§ШІ ES7 async/await ШҜШұ AureliaШЁШ§ШІШ®ЩҲШұШҜЩҮШ§ЫҢ ШҜЩҲШұЩҮШўШҙЩҶШ§ЫҢЫҢ ШЁШ§ Щ…ШҜЩ„ ШЁШұЩҶШ§Щ…ЩҮ ЩҶЩҲЫҢШіЫҢ TAPШЁШ§ШІШ®ЩҲШұШҜЩҮШ§ЫҢ ШҜЩҲШұЩҮШ§ШіШӘЩҒШ§ШҜЩҮ Ш§ШІ async ЩҲ await ШҜШұ ШЁШұЩҶШ§Щ…ЩҮвҖҢЩҮШ§ЫҢ ASP.NET Web forms 4.5Щ…Ш·Ш§Щ„ШЁRuntime Async ШҜШұ ШҜШ§ШӘвҖҢЩҶШӘ ЫұЫұ: ШӘШӯЩҲЩ„ЫҢ ШЁЩҶЫҢШ§ШҜЫҢЩҶ ШҜШұ ШЁШұЩҶШ§Щ…ЩҮвҖҢЩҶЩҲЫҢШіЫҢ ШәЫҢШұЩҮЩ…ЪҜШ§Щ… (Async)Щ…Ш·Ш§Щ„ШЁЩҫШұШҜШ§ШІШҙвҖҢЩҮШ§ЫҢ Async ШҜШұ Entity framework 6Щ…Ш·Ш§Щ„ШЁ ШҜЩҲШұЩҮвҖҢЩҮШ§Ш§ШіШӘЩҒШ§ШҜЩҮ Ш§ШІ async ЩҲ await ШҜШұ ШЁШұЩҶШ§Щ…ЩҮвҖҢЩҮШ§ЫҢ ASP.NET Web forms 4.5ЩҶШёШұШ§ШӘ Щ…Ш·Ш§Щ„ШЁШұЩҲШҙвҖҢЩҮШ§ЫҢ Щ…Ш®ШӘЩ„ЩҒ Ш§ЩҶШ¬Ш§Щ… ЪҶЩҶШҜ Ъ©Ш§Шұ ШЁЩҮ ШөЩҲШұШӘ ЩҮЩ…ШІЩ…Ш§ЩҶ ШҜШұ C# .NET - ЩӮШіЩ…ШӘ Ш§ЩҲЩ„ЩҶШёШұШ§ШӘ Щ…Ш·Ш§Щ„ШЁШ§ШіШӘЩҒШ§ШҜЩҮ Ш§ШІ EF ШҜШұ Ш§ЩҫЩ„ЫҢЪ©ЫҢШҙЩҶ ЩҮШ§ЫҢ N-Tier : ЩӮШіЩ…ШӘ ЪҶЩҮШ§ШұЩ…ЩҶШёШұШ§ШӘ Щ…Ш·Ш§Щ„ШЁШұЩҲШҙ Ш§ШіШӘЩҒШ§ШҜЩҮвҖҢЫҢ ШөШӯЫҢШӯ Ш§ШІ HttpClient ШҜШұ ШЁШұЩҶШ§Щ…ЩҮвҖҢЩҮШ§ЫҢ ШҜШ§ШӘ ЩҶШӘ
