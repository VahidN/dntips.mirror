ШЁЩҮвҖҢШ№ЩҶЩҲШ§ЩҶ ЫҢЪ© ШӘЩҲШіШ№ЩҮвҖҢШҜЩҮЩҶШҜЩҮШҢ ШЁШ§ Щ„ЩҲЪ©Ш§Щ„вҖҢЩҮШ§ШіШӘ (localhost) ШўШҙЩҶШ§ЫҢЫҢ ШҜШ§ШұЫҢШҜШӣ Ш§ЫҢЩҶ ЩҲЫҢЪҳЪҜЫҢ Ш§ШіШ§ШіЫҢвҖҢШ§ШіШӘ Ъ©ЩҮ ШЁЩҮ ШҙЩ…Ш§ Ш§Ш¬Ш§ШІЩҮ Щ…ЫҢвҖҢШҜЩҮШҜ ШЁШұЩҶШ§Щ…ЩҮвҖҢЩҮШ§ ЩҲ ЩҲШЁвҖҢШіШ§ЫҢШӘвҖҢЩҮШ§ ШұШ§ ШұЩҲЫҢ ШҜШіШӘЪҜШ§ЩҮ Щ…ШӯЩ„ЫҢ Ш®ЩҲШҜ ШӘШіШӘ ЩҲ ШҜЫҢШЁШ§ЪҜ Ъ©ЩҶЫҢШҜ. Ш§Щ…Ш§ Ш§Ш®ЫҢШұШ§ЩӢШҢ ШЁЩҮвҖҢШұЩҲШІШұШіШ§ЩҶЫҢвҖҢЩҮШ§ЫҢ Щ…Ш§ЩҮ Ш§Ъ©ШӘШЁШұ ЩҲЫҢЩҶШҜЩҲШІ ЫұЫұ (Ш§ШІ Ш¬Щ…Щ„ЩҮ KB5066835) ЩҲ ШӯШӘЫҢ ЩҫЫҢШҙвҖҢЩҶЩ…Ш§ЫҢШҙ ШіЩҫШӘШ§Щ…ШЁШұ (KB5065789) ШЁШ§Ш№Ш« Ш§ЫҢШ¬Ш§ШҜ Щ…ШҙЪ©Щ„ЫҢ Ш¬ШҜЫҢ ШҙШҜЩҮвҖҢШ§ЩҶШҜ Ъ©ЩҮ Ш§ШӘШөШ§Щ„Ш§ШӘ HTTP/2 ШЁЩҮ ШўШҜШұШі IP Щ…ШӯЩ„ЫҢ ЫұЫІЫ·.Ы°.Ы°.Ыұ ШұШ§ Щ…Ш®ШӘЩ„ Щ…ЫҢвҖҢЪ©ЩҶШҜ. Ш§ЫҢЩҶ Щ…ШіШҰЩ„ЩҮ ЩҶЩҮвҖҢШӘЩҶЩҮШ§ ШӘЩҲШіШ№ЩҮвҖҢШҜЩҮЩҶШҜЪҜШ§ЩҶ ШұШ§ ШӘШӯШӘ ШӘШЈШ«ЫҢШұ ЩӮШұШ§Шұ Щ…ЫҢвҖҢШҜЩҮШҜШҢ ШЁЩ„Ъ©ЩҮ Ш§ЩҫЩ„ЫҢЪ©ЫҢШҙЩҶвҖҢЩҮШ§ЫҢЫҢ Ъ©ЩҮ ШЁЩҮ ШіШұЩҲЫҢШівҖҢЩҮШ§ЫҢ Щ…ШӯЩ„ЫҢ ЩҲШ§ШЁШіШӘЩҮ ЩҮШіШӘЩҶШҜ ЩҶЫҢШІ ШЁШ§ Ш®Ш·Ш§ Щ…ЩҲШ§Ш¬ЩҮ Щ…ЫҢвҖҢШҙЩҲЩҶШҜ. ШҜШұ Ш§ЫҢЩҶ Щ…ЩӮШ§Щ„ЩҮШҢ ШЁЩҮ ШЁШұШұШіЫҢ ШҜЩӮЫҢЩӮ Ш§ЫҢЩҶ Щ…ШҙЪ©Щ„ШҢ Ш№Щ„Щ„ ШўЩҶ ЩҲ ШұШ§ЩҮвҖҢШӯЩ„вҖҢЩҮШ§ЫҢ Щ…ЩҲШ¬ЩҲШҜ Щ…ЫҢвҖҢЩҫШұШҜШ§ШІЫҢЩ… ШӘШ§ ШЁШӘЩҲШ§ЩҶЫҢШҜ ШіШұЫҢШ№вҖҢШӘШұ ШЁЩҮ Ъ©Ш§ШұШӘШ§ЩҶ ШЁШұЪҜШұШҜЫҢШҜ.

ШӘЩҲШ¶ЫҢШӯ Щ…ШҙЪ©Щ„ ЩҲ Ш№Щ„Ш§ШҰЩ… ШўЩҶ
Щ„ЩҲЪ©Ш§Щ„вҖҢЩҮШ§ШіШӘ Ш§ШіШ§ШіШ§ЩӢ ШЁЩҮ Ъ©Ш§Щ…ЩҫЫҢЩҲШӘШұ ЫҢШ§ ШҜШіШӘЪҜШ§ЩҮЫҢ Ш§ШҙШ§ШұЩҮ ШҜШ§ШұШҜ Ъ©ЩҮ ШҜШұ ШӯШ§Щ„ Ш§ШіШӘЩҒШ§ШҜЩҮ Ш§ШІ ШўЩҶ ЩҮШіШӘЫҢШҜ ЩҲ Ш§ШІ Ш·ШұЫҢЩӮ ШўШҜШұШі IP ЩҲЫҢЪҳЩҮ ЫұЫІЫ·.Ы°.Ы°.Ыұ ЩӮШ§ШЁЩ„ ШҜШіШӘШұШіЫҢ Ш§ШіШӘ. Ш§ЫҢЩҶ ШўШҜШұШі ШЁШұШ§ЫҢ ШӯЩ„ЩӮЩҮ ШЁШұЪҜШҙШӘЫҢ (loopback) Ш§ШіШӘЩҒШ§ШҜЩҮ Щ…ЫҢвҖҢШҙЩҲШҜШҢ Ъ©ЩҮ Ш§Ш¬Ш§ШІЩҮ Щ…ЫҢвҖҢШҜЩҮШҜ ШЁШұЩҶШ§Щ…ЩҮвҖҢЩҮШ§ ШЁШҜЩҲЩҶ ЩҶЫҢШ§ШІ ШЁЩҮ ШҙШЁЪ©ЩҮ Ш®Ш§ШұШ¬ЫҢШҢ ШЁШ§ ШіШұЩҲЫҢШівҖҢЩҮШ§ЫҢ Щ…ШӯЩ„ЫҢ Ш§ШұШӘШЁШ§Ш· ШЁШұЩӮШұШ§Шұ Ъ©ЩҶЩҶШҜ. ШӘЩҲШіШ№ЩҮвҖҢШҜЩҮЩҶШҜЪҜШ§ЩҶ Ш§ШәЩ„ШЁ Ш§ШІ ШўЩҶ ШЁШұШ§ЫҢ ШӘШіШӘ ЩҲШЁвҖҢШіШ§ЫҢШӘвҖҢЩҮШ§ШҢ ШҜЫҢШЁШ§ЪҜ Ш§ЩҫЩ„ЫҢЪ©ЫҢШҙЩҶвҖҢЩҮШ§ ЫҢШ§ ШӯШӘЫҢ Ш§Ш¬ШұШ§ЫҢ ШіШұЩҲЫҢШівҖҢЩҮШ§ЫҢ Щ…ШӯЩ„ЫҢ Щ…Ш§ЩҶЩҶШҜ ЩҲШЁвҖҢШіШұЩҲШұЩҮШ§ ШЁЩҮШұЩҮ Щ…ЫҢвҖҢШЁШұЩҶШҜ. Ш№Щ„Ш§ЩҲЩҮ ШЁШұ Ш§ЫҢЩҶШҢ ШЁШұШ®ЫҢ Ш§ЩҫЩ„ЫҢЪ©ЫҢШҙЩҶвҖҢЩҮШ§ Щ…Ш§ЩҶЩҶШҜ Ш§ШЁШІШ§ШұЩҮШ§ЫҢ Ш§Щ…ЩҶЫҢШӘЫҢ ЫҢШ§ Ш§ШӯШұШ§ШІ ЩҮЩҲЫҢШӘШҢ ШЁШұШ§ЫҢ Ш§ЩҶШ¬Ш§Щ… Ш№Щ…Щ„ЫҢШ§ШӘ Ш®ЩҲШҜ ШЁЩҮ Ш§ШӘШөШ§Щ„ ШЁЩҮ Щ„ЩҲЪ©Ш§Щ„вҖҢЩҮШ§ШіШӘ ЩҶЫҢШ§ШІ ШҜШ§ШұЩҶШҜ.
ЩҫШі Ш§ШІ ЩҶШөШЁ ШЁЩҮвҖҢШұЩҲШІШұШіШ§ЩҶЫҢ KB5066835 (Ъ©ЩҮ ШЁШ®ШҙЫҢ Ш§ШІ Patch Tuesday Ш§Ъ©ШӘШЁШұ Ш§ШіШӘ) ЫҢШ§ KB5065789 (ЩҫЫҢШҙвҖҢЩҶЩ…Ш§ЫҢШҙ ШіЩҫШӘШ§Щ…ШЁШұ)ШҢ Ъ©Ш§ШұШЁШұШ§ЩҶ ЪҜШІШ§ШұШҙ ШҜШ§ШҜЩҮвҖҢШ§ЩҶШҜ Ъ©ЩҮ Ш§ШӘШөШ§Щ„Ш§ШӘ HTTP/2 ШЁЩҮ ЫұЫІЫ·.Ы°.Ы°.Ыұ ШҜЫҢЪҜШұ Ъ©Ш§Шұ ЩҶЩ…ЫҢвҖҢЪ©ЩҶШҜ. Ш§ЫҢЩҶ ЩҫШұЩҲШӘЪ©Щ„ HTTP/2ШҢ Ъ©ЩҮ ЩҶШіШ®ЩҮ ЩҫЫҢШҙШұЩҒШӘЩҮвҖҢШӘШұЫҢ Ш§ШІ HTTP Ш§ШіШӘ ЩҲ ЩҲЫҢЪҳЪҜЫҢвҖҢЩҮШ§ЫҢЫҢ Щ…Ш§ЩҶЩҶШҜ ЪҶЩҶШҜЪҜШ§ЩҶЩҮвҖҢШіШ§ШІЫҢ (multiplexing) ЩҲ ЩҒШҙШұШҜЩҮвҖҢШіШ§ШІЫҢ ЩҮШҜШұЩҮШ§ ШұШ§ Ш§ШұШ§ШҰЩҮ Щ…ЫҢвҖҢШҜЩҮШҜШҢ ШҜШұ Ш§ЫҢЩҶ ШҙШұШ§ЫҢШ· ШЁШ§ Ш®Ш·Ш§ЩҮШ§ЫҢЫҢ Щ…Ш§ЩҶЩҶШҜ вҖңERR_CONNECTION_RESETвҖқ ЫҢШ§ вҖңERR_HTTP2_PROTOCOL_ERRORвҖқ Щ…ЩҲШ§Ш¬ЩҮ Щ…ЫҢвҖҢШҙЩҲШҜ. Ш§ЫҢЩҶ Щ…ШҙЪ©Щ„ ШЁЩҮвҖҢЩҲЫҢЪҳЩҮ ШҜШұ ЩҲЫҢЩҶШҜЩҲШІ ЫұЫұ ЩҶШіШ®ЩҮвҖҢЩҮШ§ЫҢ ЫІЫҙHЫІ ЩҲ ЫІЫөHЫІ Щ…ШҙШ§ЩҮШҜЩҮ ШҙШҜЩҮ Ш§ШіШӘ.

ЪҜШІШ§ШұШҙвҖҢЩҮШ§ЫҢ Щ…ШӘШ№ШҜШҜЫҢ Ш§ШІ Ъ©Ш§ШұШЁШұШ§ЩҶ ШҜШұ ЩҒШұЩҲЩ…вҖҢЩҮШ§ЫҢ Щ…Ш§ЫҢЪ©ШұЩҲШіШ§ЩҒШӘШҢ Stack Exchange ЩҲ Reddit Щ…ЩҶШӘШҙШұ ШҙШҜЩҮ Ъ©ЩҮ ЩҮЩ…ЩҮ ШЁЩҮ Ш№ШҜЩ… Ш§Щ…Ъ©Ш§ЩҶ ШЁШұЩӮШұШ§ШұЫҢ Ш§ШӘШөШ§Щ„ HTTP ШЁЩҮ Щ„ЩҲЪ©Ш§Щ„вҖҢЩҮШ§ШіШӘ Ш§ШҙШ§ШұЩҮ ШҜШ§ШұЩҶШҜ. Ш§ЩҫЩ„ЫҢЪ©ЫҢШҙЩҶвҖҢЩҮШ§ЫҢ Щ…ШӯШЁЩҲШЁ ШӘШӯШӘ ШӘШЈШ«ЫҢШұ ШҙШ§Щ…Щ„ Щ…ЩҲШ§ШұШҜ ШІЫҢШұ ЩҮШіШӘЩҶШҜ:
  • ШҜЫҢШЁШ§ЪҜ ШҜШұ Visual Studio: Ш¬Ш§ЫҢЫҢ Ъ©ЩҮ ШӘЩҲШіШ№ЩҮвҖҢШҜЩҮЩҶШҜЪҜШ§ЩҶ ШЁШұШ§ЫҢ ШӘШіШӘ Ъ©ШҜЩҮШ§ЫҢ ЩҲШЁ ЩҶЫҢШ§ШІ ШЁЩҮ Щ„ЩҲЪ©Ш§Щ„вҖҢЩҮШ§ШіШӘ ШҜШ§ШұЩҶШҜ.
  • Ш§ШӯШұШ§ШІ ЩҮЩҲЫҢШӘ Entra ID ШҜШұ SQL Server Management Studio (SSMS): Ъ©ЩҮ ШЁШұШ§ЫҢ Ш§ШӘШөШ§Щ„ Ш§Щ…ЩҶ ШЁЩҮ ШіШұЩҲЫҢШівҖҢЩҮШ§ЫҢ Щ…ШӯЩ„ЫҢ Ш§ШіШӘЩҒШ§ШҜЩҮ Щ…ЫҢвҖҢШҙЩҲШҜ.

Щ…Ш§ЫҢЪ©ШұЩҲШіШ§ЩҒШӘ ШӘЩҲШ¶ЫҢШӯ ШҜШ§ШҜЩҮ Ъ©ЩҮ Ш§ЫҢЩҶ ШЁШ§ЪҜ ШЁЩҮ ШҜШұШ§ЫҢЩҲШұ HTTP.sys Щ…ШұШЁЩҲШ· Ш§ШіШӘШҢ Ъ©ЩҮ Щ…ШіШҰЩҲЩ„ЫҢШӘ Щ…ШҜЫҢШұЫҢШӘ Ш§ШӘШөШ§Щ„Ш§ШӘ HTTP ШҜШұ ШіШ·Шӯ ШіЫҢШіШӘЩ…вҖҢШ№Ш§Щ…Щ„ ШұШ§ ШЁШұ Ш№ЩҮШҜЩҮ ШҜШ§ШұШҜ. Ш№ЩҲШ§Щ…Щ„ Щ…Ш®ШӘЩ„ЩҒЫҢ Щ…Ш§ЩҶЩҶШҜ Ш§ШӘШөШ§Щ„ Ш§ЫҢЩҶШӘШұЩҶШӘ ШҜШіШӘЪҜШ§ЩҮШҢ ШІЩ…Ш§ЩҶ ЩҶШөШЁ ШЁЩҮвҖҢШұЩҲШІШұШіШ§ЩҶЫҢ ЩҲ ШұЫҢШіШӘШ§ШұШӘвҖҢЩҮШ§ Щ…ЫҢвҖҢШӘЩҲШ§ЩҶЩҶШҜ ШЁШұ ШЁШұЩҲШІ Ш§ЫҢЩҶ Щ…ШҙЪ©Щ„ ШӘШЈШ«ЫҢШұ ШЁЪҜШ°Ш§ШұЩҶШҜШҢ ШЁЩҮ ЩҮЩ…ЫҢЩҶ ШҜЩ„ЫҢЩ„ Щ…Щ…Ъ©ЩҶ Ш§ШіШӘ ШҜШұ ЩҮЩ…ЩҮ Щ…ШӯЫҢШ·вҖҢЩҮШ§ Щ…ШҙШ§ЩҮШҜЩҮ ЩҶШҙЩҲШҜ.

Ш§ЫҢЩҶ ШӘШәЫҢЫҢШұ HTTP/2 ШұШ§ ШәЫҢШұЩҒШ№Ш§Щ„ Щ…ЫҢвҖҢЪ©ЩҶШҜ ЩҲ Ш§ШӘШөШ§Щ„Ш§ШӘ ШұШ§ ШЁЩҮ ЩҶШіШ®ЩҮ ЩӮШҜЫҢЩ…ЫҢвҖҢШӘШұ HTTP ШЁШұЩ…ЫҢвҖҢЪҜШұШҜШ§ЩҶШҜШҢ Ш§Щ…Ш§ ШЁШ§ЫҢШҜ ШЁШ§ Ш§ШӯШӘЫҢШ§Ш· Ш§Ш№Щ…Ш§Щ„ ШҙЩҲШҜ ШІЫҢШұШ§ Щ…Щ…Ъ©ЩҶ Ш§ШіШӘ Ш№Щ…Щ„Ъ©ШұШҜ ШЁШұШ®ЫҢ ШЁШұЩҶШ§Щ…ЩҮвҖҢЩҮШ§ ШұШ§ ШӘШӯШӘ ШӘШЈШ«ЫҢШұ ЩӮШұШ§Шұ ШҜЩҮШҜ. ШЁШұШ®ЫҢ Ъ©Ш§ШұШЁШұШ§ЩҶ ЩҶЫҢШІ ЪҜШІШ§ШұШҙ ШҜШ§ШҜЩҶШҜ Ъ©ЩҮ ЩҶШөШЁ ШўШ®ШұЫҢЩҶ ШЁЩҮвҖҢШұЩҲШІШұШіШ§ЩҶЫҢ ЩҮЩҲШҙ Microsoft Defender Щ…ШҙЪ©Щ„ ШұШ§ ШӯЩ„ Ъ©ШұШҜЩҮШҢ ЩҮШұЪҶЩҶШҜ Ш§ЫҢЩҶ ШұЩҲШҙ ШЁШұШ§ЫҢ ЩҮЩ…ЩҮ Ъ©Ш§ШұШіШ§ШІ ЩҶШЁЩҲШҜЩҮ Ш§ШіШӘ.
ШұШ§ЩҮвҖҢШӯЩ„ Щ…Ш·Щ…ШҰЩҶвҖҢШӘШұ Ш§ЩҲЩ„ЫҢЩҮШҢ ШӯШ°ЩҒ ШЁЩҮвҖҢШұЩҲШІШұШіШ§ЩҶЫҢвҖҢЩҮШ§ЫҢ Щ…ШҙЪ©Щ„вҖҢШіШ§ШІ ШЁЩҲШҜ. Щ…ЫҢвҖҢШӘЩҲШ§ЩҶЫҢШҜ Ш§ШІ ШҜШіШӘЩҲШұШ§ШӘ ШІЫҢШұ ШҜШұ Command Prompt (ШЁШ§ ШӯЩӮЩҲЩӮ Ш§ШҜШ§ШұЫҢ) Ш§ШіШӘЩҒШ§ШҜЩҮ Ъ©ЩҶЫҢШҜ:
wusa /uninstall /kb:5066835
wusa /uninstall /kb:5065789
ЩҫШі Ш§ШІ ШӯШ°ЩҒ ЩҲ ШұЫҢШіШӘШ§ШұШӘ ШҜШіШӘЪҜШ§ЩҮШҢ Ш§ШӘШөШ§Щ„Ш§ШӘ Щ„ЩҲЪ©Ш§Щ„вҖҢЩҮШ§ШіШӘ Щ…Ш№Щ…ЩҲЩ„Ш§ЩӢ ШЁЩҮ ШӯШ§Щ„ШӘ Ш№Ш§ШҜЫҢ ШЁШұЩ…ЫҢвҖҢЪҜШұШҜШҜ.
Ш§Щ…Ш§ Ш®ШЁШұ Ш®ЩҲШЁ Ш§ЫҢЩҶ Ш§ШіШӘ Ъ©ЩҮ Щ…Ш§ЫҢЪ©ШұЩҲШіШ§ЩҒШӘ ШҜШұ ЫұЫ· Ш§Ъ©ШӘШЁШұ ЫІЫ°ЫІЫөШҢ Ш§ЫҢЩҶ Щ…ШҙЪ©Щ„ ШұШ§ ШЁШ§ ЫҢЪ© ШЁЩҮвҖҢШұЩҲШІШұШіШ§ЩҶЫҢ Known Issue Rollback (KIR) вҖ“ Ъ©ЩҮ Щ…Ъ©Ш§ЩҶЫҢШіЩ…ЫҢ ШЁШұШ§ЫҢ Ш№ЩӮШЁвҖҢЩҶШҙЫҢЩҶЫҢ ШіШұЫҢШ№ Щ…ШіШ§ШҰЩ„ ШҙЩҶШ§Ш®ШӘЩҮвҖҢШҙШҜЩҮ Ш§ШіШӘ вҖ“ ШЁШұШ·ШұЩҒ Ъ©ШұШҜЩҮ. Ш·ШЁЩӮ ШЁЩҲЩ„ШӘЩҶ ЩҫШҙШӘЫҢШЁШ§ЩҶЫҢ Щ…Ш§ЫҢЪ©ШұЩҲШіШ§ЩҒШӘШҢ ШЁШұШ§ЫҢ Ш§Ш№Щ…Ш§Щ„ Ш§ЫҢЩҶ ЩҒЫҢЪ©Ші Ъ©Ш§ЩҒЫҢ Ш§ШіШӘ:
  • ШӘЩҶШёЫҢЩ…Ш§ШӘ ЩҲЫҢЩҶШҜЩҲШІ ШұШ§ ШЁШ§ШІ Ъ©ЩҶЫҢШҜ ЩҲ ШЁЩҮ ШЁШ®Шҙ Windows Update ШЁШұЩҲЫҢШҜ (Щ…ЫҢвҖҢШӘЩҲШ§ЩҶЫҢШҜ Ш§ШІ Щ…ЩҶЩҲЫҢ Ш§ШіШӘШ§ШұШӘ Ш¬ШіШӘШ¬ЩҲ Ъ©ЩҶЫҢШҜ: вҖңcheck for updatesвҖқ).
  • ШұЩҲЫҢ вҖңCheck for updatesвҖқ Ъ©Щ„ЫҢЪ© Ъ©ЩҶЫҢШҜ ЩҲ Ш§Ш¬Ш§ШІЩҮ ШҜЩҮЫҢШҜ ЩҮШұ ШЁЩҮвҖҢШұЩҲШІШұШіШ§ЩҶЫҢ Щ…ЩҲШ¬ЩҲШҜ ЩҶШөШЁ ШҙЩҲШҜ.
  • ШҜШіШӘЪҜШ§ЩҮ ШұШ§ ШұЫҢШіШӘШ§ШұШӘ Ъ©ЩҶЫҢШҜ вҖ“ ШӯШӘЫҢ Ш§ЪҜШұ ЩҮЫҢЪҶ ШЁЩҮвҖҢШұЩҲШІШұШіШ§ЩҶЫҢ Ш¬ШҜЫҢШҜЫҢ ЩҶШөШЁ ЩҶШҙШҜЩҮ ШЁШ§ШҙШҜ.
ЩҫШі Ш§ШІ ШұЫҢШіШӘШ§ШұШӘШҢ Щ…ШіШ§ШҰЩ„ Ш§ШӘШөШ§Щ„ Щ„ЩҲЪ©Ш§Щ„вҖҢЩҮШ§ШіШӘ ШЁШ§ЫҢШҜ ШӯЩ„ ШҙЩҲШҜ.


Щ…ШҙШ§ЩҮШҜЩҮ Щ…Ш·Щ„ШЁ Ш§ШөЩ„ЫҢ
Щ…Ш·Ш§Щ„ШЁ Щ…ШҙШ§ШЁЩҮ
    Ш§ШҙШӘШұШ§Ъ©вҖҢЩҮШ§ЩҫШ§ЫҢШ§ЩҶ ЩҫШҙШӘЫҢШЁШ§ЩҶЫҢ Ш§ШІ Windows 10Ш§ШҙШӘШұШ§Ъ©вҖҢЩҮШ§ЩҲЫҢЩҶШҜЩҲШІ ЫұЫұ ШЁЩҮ ЫҢЪ© Щ…ЫҢЩ„ЫҢШ§ШұШҜ Ъ©Ш§ШұШЁШұ ШұШіЫҢШҜШҢ ШіШұЫҢШ№вҖҢШӘШұ Ш§ШІ ЩҲЫҢЩҶШҜЩҲШІ ЫұЫ°Ш§ШҙШӘШұШ§Ъ©вҖҢЩҮШ§Ш§Ш№Щ„Ш§Щ… ШЁЩҮвҖҢШұЩҲШІШұШіШ§ЩҶЫҢ SQL Server Management Studio 22.3 ШЁШ§ ШұЩҒШ№ Ш§ШҙЪ©Ш§Щ„Ш§ШӘ ЩҲ ЩҫШҙШӘЫҢШЁШ§ЩҶЫҢ Ш§ШІ ШҜШіШӘЩҲШұШ§Щ„Ш№Щ…Щ„вҖҢЩҮШ§ЫҢ ЩҫШ§ЫҢЪҜШ§ЩҮ ШҜШ§ШҜЩҮШ§ШҙШӘШұШ§Ъ©вҖҢЩҮШ§Щ…Ш§ЫҢЪ©ШұЩҲШіШ§ЩҒШӘ ШЁЩҮ Щ…ЩҶШ§ШіШЁШӘ ЩҫЩҶШ¬Ш§ЩҮЩ…ЫҢЩҶ ШіШ§Щ„ЪҜШұШҜ Ш®ЩҲШҜШҢ ЩҶШіШ®ЩҮвҖҢШ§ЫҢ Щ…ШӯШҜЩҲШҜ Ш§ШІ Ъ©ЩҒШҙвҖҢЩҮШ§ЫҢ Crocs ШұШ§ ШЁШ§ Ш§Щ„ЩҮШ§Щ… Ш§ШІ ЩҲЫҢЩҶШҜЩҲШІ XP Щ…ЩҶШӘШҙШұ Щ…ЫҢвҖҢЪ©ЩҶШҜШ§ШҙШӘШұШ§Ъ©вҖҢЩҮШ§ЩҫШ§ЫҢШ§ЩҶ ЩҫШҙШӘЫҢШЁШ§ЩҶЫҢ Щ…Ш§ЫҢЪ©ШұЩҲШіШ§ЩҒШӘ Ш§ШІ ЩҲЫҢЩҶШҜЩҲШІ 11 ЩҶШіШ®ЩҮ 24H2ШҢ ШўЩҒЫҢШі 2021 ЩҲ ШіШ§ЫҢШұ Щ…ШӯШөЩҲЩ„Ш§ШӘ ШҜШұ ШіШ§Щ„ 2026Ш§ШҙШӘШұШ§Ъ©вҖҢЩҮШ§Щ…Ш§ЫҢЪ©ШұЩҲШіШ§ЩҒШӘ .NET Framework 3.5 ШұШ§ Ш§ШІ ЩҲЫҢЩҶШҜЩҲШІ ЫұЫұ (Ъ©Ш§ЩҶШ§Щ„ Canary) ШӯШ°ЩҒ Щ…ЫҢвҖҢЪ©ЩҶШҜШ§ШҙШӘШұШ§Ъ©вҖҢЩҮШ§ШұЩҲШҙвҖҢЩҮШ§ЫҢЫҢ ШЁШұШ§ЫҢ Ш¬Щ„ЩҲЪҜЫҢШұЫҢ Ш§ШІ ШӯЩ…Щ„Ш§ШӘ SSRF ШҜШұ ШҜШ§ШӘвҖҢЩҶШӘШ§ШҙШӘШұШ§Ъ©вҖҢЩҮШ§Ш§ЩҶШҜШұЩҲЫҢШҜ ЫұЫ¶ Ш§ШІ ШұШ§ЩҮ ШұШіЫҢШҜ: Щ…ШұЩҲШұЫҢ ШЁШұ ЩҶЩҲШўЩҲШұЫҢвҖҢЩҮШ§ ШЁШұШ§ЫҢ Ъ©Ш§ШұШЁШұШ§ЩҶ ЩҲ ШӘЩҲШіШ№ЩҮвҖҢШҜЩҮЩҶШҜЪҜШ§ЩҶЩ…Ш·Ш§Щ„ШЁ ШҜЩҲШұЩҮвҖҢЩҮШ§Ш§ШұШӘШЁШ§Ш·Ш§ШӘ ШЁЩ„Ш§ШҜШұЩҶЪҜ ЩҲ SignalRЩҶШёШұШ§ШӘ Щ…Ш·Ш§Щ„ШЁЪҶШұШ§ ШҜШұ ШіШ§ШІЩ…Ш§ЩҶвҖҢЩҮШ§ ШЁШұЩҶШ§Щ…ЩҮвҖҢЩҮШ§ЫҢ ЩҲШЁ Ш¬Ш§ЫҢЪҜШІЫҢЩҶ ШЁШұЩҶШ§Щ…ЩҮвҖҢЩҮШ§ЫҢ ШҜШіЪ©ШӘШ§Щҫ ШҙШҜЩҮвҖҢШ§ЩҶШҜ (ЫҢШ§ Щ…ЫҢвҖҢШҙЩҲЩҶШҜ)Шҹ
