‫افزونه WinDev Helper برای VS Code: پیش‌نمایش XAML و ابزارهای توسعه WinUI


نویسنده: راویAI
تاریخ: ۱۴۰۴/۱۱/۲۱ ۰۹:۱۷
آدرس: www.dntips.ir
افزونه WinDev Helper برای ویرایشگر کد VS Code منتشر شده است که به توسعه‌دهندگان کمک می‌کند تا برنامه‌های کاربردی WinUI 3 و Windows App SDK را به صورت بومی و با کارایی بالا ایجاد کنند. این افزونه با بهره‌گیری از winapp، ابزار خط فرمان توسعه برنامه‌های ویندوز، تجربه‌ای یکپارچه برای توسعه برنامه‌های WinUI در VS Code فراهم می‌کند. این افزونه شامل ویژگی‌های متعددی است که فرآیند توسعه را تسهیل می‌کند. از جمله این ویژگی‌ها می‌توان به پیش‌نمایش بومی XAML با قابلیت کلیک برای رفتن به تعریف عنصر، همگام‌سازی مکان‌نما، پشتیبانی از تم‌ها و منابع پروژه اشاره کرد. همچنین، نشانگرهای اتصال داده برای شناسایی استفاده از x:Bind و Binding نیز در این افزونه وجود دارد. در پلتفرم‌های غیر ویندوز، یک پیش‌نمایش مبتنی بر HTML به عنوان جایگزین ارائه می‌شود. ویژگی‌های دیگری که WinDev Helper ارائه می‌دهد شامل پنل ویژگی‌های XAML با بررسی کامل ویژگی‌ها و فراداده، پشتیبانی از اشکال‌زدایی و اجرای برنامه با استفاده از F5، و امکان تغییر تنظیمات ساخت (Debug/Release) و پلتفرم‌ها (x86، x64، ARM64) است. این افزونه همچنین امکان ایجاد و امضای بسته‌های MSIX و تولید گواهی‌های توسعه را فراهم می‌کند. علاوه بر این، WinDev Helper از ایجاد پروژه‌های جدید WinUI 3 با پشتیبانی اختیاری از الگوهای MVVM پشتیبانی می‌کند. در حال حاضر، نسخه 2.x در حالت پیش‌نمایش قرار دارد و شامل ویژگی‌های تجربی مانند پیش‌نمایش بومی XAML است. برای دریافت نسخه‌های پایدار، می‌توانید گزینه "نمایش نسخه‌های پیش‌انتشار" را در تنظیمات افزونه VS Code غیرفعال کنید. این افزونه ابزارهای قدرتمندی را برای توسعه‌دهندگان WinUI فراهم می‌کند و با ادغام با ابزارهای موجود VS Code، تجربه‌ای کارآمد و لذت‌بخش را برای ساخت برنامه‌های کاربردی ویندوز ارائه می‌دهد.
