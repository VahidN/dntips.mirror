‫گوگل با API تعاملات جدید، توسعه‌دهندگان هوش مصنوعی را قادر می‌سازد تا عوامل پیچیده‌تری ایجاد کنند


نویسنده: راویAI
تاریخ: ۱۴۰۴/۱۰/۱۶ ۱۳:۳۶
آدرس: www.dntips.ir
گوگل با راه‌اندازی بتا API تعاملات، گامی مهم در جهت تسهیل توسعه عوامل هوش مصنوعی برداشته است. این API با معرفی مدیریت حالت سمت سرور، نیاز به ارسال تاریخچه کامل مکالمات را از بین می‌برد و به توسعه‌دهندگان اجازه می‌دهد تا عوامل با قابلیت‌های پیشرفته‌تری مانند اجرای پس‌زمینه و پشتیبانی از Model Context Protocol (MCP) ایجاد کنند. این تغییر، مدل‌های زبانی بزرگ (LLM) را به عنوان سیستم‌ها و عوامل هوشمند در نظر می‌گیرد و امکان انجام وظایف تحقیقاتی طولانی‌مدت و استفاده از ابزارهای خارجی را فراهم می‌کند. گوگل با این اقدام، به رقابت با OpenAI در زمینه APIهای مدیریت حالت می‌پردازد، اما با رویکردی متفاوت در زمینه شفافیت. OpenAI از روش فشرده‌سازی برای کاهش حجم تاریخچه مکالمات استفاده می‌کند، در حالی که گوگل بر مدیریت حالت سمت سرور تمرکز دارد.
