What's new in the .NET Framework 4.6.1

The .NET Framework 4.6.1 includes new features in the following areas:

  • Cryptography

  • ADO.NET

  • Windows Presentation Foundation (WPF)

  • Windows Workflow Foundation

  • Profiling

  • NGen
